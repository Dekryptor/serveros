Einrichten Ihrer ersten Übertragung

Sie können Ihre erste Übertragung vorbereiten und in weniger als fünf Minuten auf Sendung gehen.

Die folgenden Anweisungen gelten für das Einrichten einer direkten Eins-zu-Eins-Verbindung (ohne den QuickTime Streaming Server oder andere Streaming-Software) zu einem Computer mit QuickTime-Player.

Sie benötigen dazu die IP-Adresse des empfangenden Computers.

  1. Schließen Sie eine Kamera oder ein anderes Videoeingabegerät an Ihren Computer an.

  2. Öffnen Sie den QuickTime Broadcaster und klicken Sie, falls das erweiterte Fenster geöffnet ist, auf "Details ausblenden".

  3. Wählen Sie aus den Einblendmenüs "Audio" und "Video" die Optionen aus, die auf Ihre Situation am besten zutreffen.

  4. Klicken Sie zuerst auf "Details einblenden" und dann auf "Netzwerk". Wählen Sie "Unicast (manuell)" aus dem Einblendmenü "Übertragung" aus.

  5. Geben Sie in das Feld "Adresse" die IP-Adresse des empfangenden Computers ein.

    Behalten Sie die Standardwerte der anderen Einstellungen bei.

  6. Wählen Sie im QuickTime Broadcaster "Ablage" > "Exportieren" > "SDP".

    Die Datei, die mit diesem Schritt gesichert wird, ist die SDP-Datei (Session Description Protocol). Sie enthält Informationen, die die Zuschauer benötigen, um Ihre Sendung ansehen zu können.

  7. Senden bzw. kopieren Sie die SDP-Datei auf den Computer, auf dem Ihre Sendung gezeigt werden soll.

  8. Starten Sie auf dem sendenden Computer die Übertragung, indem Sie im QuickTime Broadcaster-Fenster auf "Senden" klicken.

Wählen Sie die SDP-Datei auf dem empfangenden Computer durch Doppelklicken aus, um die Sendung anzusehen.
