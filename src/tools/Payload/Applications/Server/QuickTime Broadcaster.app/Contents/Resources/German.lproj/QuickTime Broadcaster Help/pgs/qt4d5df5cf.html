Konfigurieren einer Audiosendung

Sie können den QuickTime Broadcaster verwenden, um Audiomaterial zu senden, wie z. B. ein Konzert, eine Rede oder eine Konferenz.

Der QuickTime Broadcaster funktioniert mit:

  • Audiomaterial von Ihrer DV-Kamera oder einem anderen FireWire-Gerät

  • Einem externen (nicht internen) CD-Player

  • Einem Mikrofon oder einer anderen analogen Audioquelle, die an den Audioeingang des Computers angeschlossen ist, oder einem USB-Audioeingabegerät, wie z. B. Griffin iMic

Gehen Sie wie folgt vor, um eine Audiosendung zu konfigurieren:

  1. Klicken Sie im QuickTime Broadcaster-Fenster zuerst auf "Details einblenden" (falls das erweiterte Fenster nicht geöffnet ist) und dann auf "Audio".

  2. Wählen Sie eine Audioquelle aus dem Einblendmenü "Quelle" aus.

    Wenn Ihre Audioquelle im Menü nicht angezeigt wird, beenden Sie den QuickTime Broadcaster und stellen Sie sicher, dass die Audiogeräte angeschlossen und eingeschaltet sind. Wiederholen Sie anschließend den Vorgang.

  3. Wählen Sie aus dem Einblendmenü "Voreinstellung" eine geeignete Einstellung für die Komprimierung aus oder geben Sie eigene Einstellungen ein.

  4. Klicken Sie auf "Video" und vergewissern Sie sich, dass die Option "Video-Stream aktivieren" deaktiviert ist. (Damit wird der Videodatenstrom deaktiviert.)

Sind Audiorückkopplungen zu hören oder möchten Sie sich das Audiomaterial nicht anhören, können Sie den Ton ausschalten, indem Sie im Bereich "Audio" die Markierung aus dem Feld "Wiedergabe über Lautsprecher" entfernen.
