Verwendung eines Streaming-Servers

Sie benötigen einen Streaming-Server, wenn Sie Ihre Sendung im Internet übertragen oder an mehrere Teilnehmer in einem lokalen Netzwerk senden wollen, das nicht multicast-fähig ist. Der Streaming-Server vervielfältigt den einzelnen Datenstrom vom QuickTime Broadcaster und ermöglicht, dass mehrere QuickTime-Player sich in die Sendung zuschalten können.

Sie können den QuickTime Broadcaster auch ohne einen Streaming-Server verwenden, um Ihre Sendung an einen einzelnen Teilnehmer (mit QuickTime-Player) zu übertragen oder die Multicast-Methode zur Übertragung an mehrere Teilnehmer einzusetzen.
