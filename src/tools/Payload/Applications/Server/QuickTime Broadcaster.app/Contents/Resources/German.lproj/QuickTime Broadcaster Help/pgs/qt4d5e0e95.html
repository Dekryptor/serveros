Auswählen einer Sendemethode

Welche Übertragungsmethode Sie auswählen, hängt von der erwarteten Größe des Publikums ab.

  • Wenn ein Live-Ereignis an nur einen Zuschauer gesendet wird, können Sie QuickTime Broadcaster unter Mac OS X verwenden. Sie benötigen dazu keine Serversoftware. Das Senden einer Übertragung an einen einzelnen Empfänger wird als Unicasting bezeichnet. Unicasting ist die gängigste Übertragungsmethode für allgemeine Internetübertragungen.

    Der Unicast-Empfänger muss einen Computer besitzen, auf dem eines der Programme "QuickTime-Player", "QuickTime Streaming Server" (QTSS) oder "Darwin Streaming Server" installiert ist. Wenn der Empfänger über Serversoftware verfügt, kann Ihre Sendung über diesen Server an mehrere Zuschauer ausgestrahlt werden.

  • Ein einzelner Macintosh-Computer kann für mehrere hundert zugeschaltete Zuschauer sowohl die QuickTime Broadcaster- als auch die QTSS-Software ausführen. Der QuickTime Broadcaster sendet den Datenstrom an den QTSS, der ihn wiederum an eine Gruppe von Clients überträgt.

    Die Anzahl der Clients, die sich in die Sendung einschalten können, hängt von der verfügbaren Bandbreite und der Prozessorkapazität ab. Eine typische Mac Pro-Konfiguration lässt mindestens 200 DSL-Verbindungen oder 400 Wählverbindungen zu. Diese Konfiguration bietet sich für Fernunterricht, Vorträge und andere Situationen mit begrenzter Teilnehmerzahl an.

  • Wenn Ihre Sendung sich an eine große Zuschauerzahl richtet, setzen Sie zwei Computer ein: einen mit dem QuickTime Broadcaster zum Senden des Datenstroms und einen mit dem QuickTime Streaming Server, der den Datenstrom an eine große Zuschauerzahl ausstrahlt.

  • Wenn Sie sich in einem Multicast-fähigen Netzwerk befinden, können Sie eine größere Anzahl von Zuschauern erreichen, indem Sie das Multicast-Verfahren für Ihre Übertragung verwenden.Multicasting ist eine Übertragungsmethode über ein lokales Netzwerk (nicht über das Internet), bei der ein einzelner Stream für alle Zuschauer freigegeben wird. Jeder Zuschauer schaltet sich in ähnlicher Weise in die Übertragung ein wie ein Radio in eine UKW-Radiosendung.
