Astuces pour la sélection de réglages de compression audio

Il est généralement préférable de définir les options audio avant les réglages vidéo. Dans QuickTime Broadcaster, vous pouvez choisir parmi tous les codecs fournis avec QuickTime. Ceci vous permet de sélectionner le meilleur codec pour le type de contenu diffusé et pour les connexions de vos spectateurs.

Pour les débutants, QuickTime Broadcaster inclut des réglages par défaut (préréglages) optimisés pour les types de contenu et de connexion habituels. Par exemple, le réglage par défaut pour la parole est MPEG-4 audio (AAC) à 8 kHz mono. (Bien que le codec MPEG-4 audio soit généralement considéré comme un codec musical, il fonctionne aussi très bien pour la parole.)

Pour l’audio, y compris la musique, le réglage par défaut est MPEG-4 audio (AAC) à 20 kilobits par seconde (kbps) pour les utilisateurs commutés et 128 kbps pour les utilisateurs du réseau local.
