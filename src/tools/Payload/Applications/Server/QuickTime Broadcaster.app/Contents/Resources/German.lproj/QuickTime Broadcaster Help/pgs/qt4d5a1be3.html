Einschränken des Zugriffs auf Ihre Sendung

Wenn Ihre Sendung nur für einen bestimmten Zuschauerkreis bestimmt ist, können Sie den gesendeten Film in eine Webseite integrieren und den Zugriff auf die HTML-Seite über die Sicherheitseinstellungen Ihres Webservers kontrollieren.

Wenn Sie die SDP-Datei oder die Ankündigungsfilmdatei für die Sendung verteilen, kann der Datenstrom von jedem, der die Datei erhält, angesehen werden - außer die Sendung wird innerhalb einer Firewall übertragen. Wenn für die Übertragung die Option "Unicast (manuell)" ausgewählt ist, kann nur der Computer mit der angegebenen IP-Adresse die Verbindung herstellen.
