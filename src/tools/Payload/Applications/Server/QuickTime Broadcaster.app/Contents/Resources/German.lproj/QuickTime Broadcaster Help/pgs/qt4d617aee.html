QuickTime Broadcaster

Mithilfe des QuickTime Broadcaster, einer FireWire DV-Quellen (Digital Video) und Mac OS X können Sie Live-Audio- und Videodateien über das Internet per Streaming übertragen. Zusammen mit QuickTime Streaming Server (QTSS) kann der QuickTime Broadcaster ein Live-Ereignis für beliebig viele Zuschauer (von einem bis zu Millionen von Zuschauern) erstellen.

Das Programm "QuickTime Broadcaster" kommt den Ansprüchen von Neueinsteigern und Profis entgegen. Es bietet voreingestellte Broadcast-Einstellungen, mit denen Sie mühelos eine Sendung produzieren können, und zwar ohne großen Konfigurationsaufwand. Wenn Sie ein größeres Maß an Kontrolle wünschen, können Sie leicht eigene Einstellungen definieren.

Aufbauend auf der QuickTime-Architektur können Sie mit dem QuickTime Broadcaster ein Live-Event unter Verwendung der meisten von QuickTime unterstützten Codecs (Compressor/Decompressor) produzieren, einschließlich H.264.

Bitte den QuickTime Broadcaster öffnen.
