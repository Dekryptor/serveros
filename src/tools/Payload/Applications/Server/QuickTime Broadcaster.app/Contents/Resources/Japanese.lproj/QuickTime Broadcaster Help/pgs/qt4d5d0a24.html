どのようなときにストリーミングサーバを使用するか

ブロードキャストをインターネット経由で送信したり、マルチキャストに対応していないローカルネットワークで複数の視聴者に送信する場合は、ストリーミングサーバを使う必要があります。ストリーミングサーバは「QuickTime Broadcaster」からの単数のストリームを複製して、複数の「QuickTime Player」がブロードキャストに接続できるようにします。

「QuickTime Broadcaster」を使用すれば、「QuickTime Player」を使用している単数の視聴者やマルチキャスト送信方法を利用している複数の視聴者に、ストリーミングサーバを使わずにブロードキャストを送信することができます。
