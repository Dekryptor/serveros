Configuration d’une diffusion audio

Vous pouvez utiliser QuickTime Broadcaster pour diffuser du contenu audio tel qu’un concert, un discours ou une réunion, par exemple.

QuickTime Broadcaster fonctionne avec :

  • L’audio fournit par votre caméra DV ou tout autre périphérique FireWire

  • Un lecteur de CD externe (pas interne)

  • Un microphone ou toute autre source audio analogique connectée aux ports d’entrée audio de votre ordinateur, ou à un périphérique d’entrée audio USB, tel que le Griffin iMic

Pour configurer une diffusion audio :

  1. Dans la fenêtre QuickTime Broadcaster, cliquez sur Afficher les détails (si la fenêtre n’est pas développée), puis sur Audio.

  2. Choisissez une source audio dans le menu local Source.

    Si votre source audio n’apparaît pas dans le menu, quittez QuickTime Broadcaster, assurez-vous que votre équipement est connecté et allumé puis retentez l’opération.

  3. Choisissez un réglage de compression approprié dans le menu local Préréglage ou spécifiez des réglages personnalisés.

  4. Cliquez sur Vidéo et assurez-vous que l’option « Activer le flux vidéo » est désélectionnée (cette étape désactive le flux vidéo).

Si les données audio créent un retour ou si vous ne souhaitez pas les écouter, vous pouvez désactiver l’audio en désélectionnant la case « Haut-parleur » du panneau Audio.
