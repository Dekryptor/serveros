Konfigurieren Ihres Streaming-Servers

Sie können Ihren Streaming-Server konfigurieren, die Konfiguration testen und auf Medien zugreifen, die per Streaming von Ihrem Server übertragen wurden.

Gehen Sie wie folgt vor, um Ihren Streaming-Server einzurichten:

  1. Öffnen Sie das Programm "Server-Admin".

  2. Wählen Sie in der Liste der Dienste den Computer aus, den Sie als QuickTime Streaming Server einrichten wollen.

  3. Klicken Sie in der Symbolleiste auf "Einstellungen" und dann auf den Titel "Dienste".

  4. Wählen Sie "QuickTime-Streaming" aus der Liste der Dienste aus und klicken Sie auf "Sichern".

  5. Wählen Sie "QuickTime-Streaming" aus der Serverliste aus, um die Einstellungen anzuzeigen und zu ändern.

  6. Zum Starten des QuickTime-Streaming-Diensts klicken Sie auf die Taste zum Starten unten links im Server-Admin-Fenster.
