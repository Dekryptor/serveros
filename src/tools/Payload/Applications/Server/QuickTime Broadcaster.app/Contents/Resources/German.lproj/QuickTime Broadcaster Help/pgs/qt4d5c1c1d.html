Hinzufügen von Anmerkungen zu einer Sendung

Sie können Ihre Sendung mit einem Titel, einem Copyright-Vermerk oder anderen Anmerkungen versehen.

Gehen Sie wie folgt vor, um Anmerkungen hinzuzufügen:

  1. Klicken Sie im QuickTime Broadcaster auf "Netzwerk".

    Ist die Taste "Netzwerk" nicht sichtbar, klicken Sie auf "Details einblenden".

  2. Füllen Sie die gewünschten Felder unter "Anmerkungen" aus.

Die Zuschauer können diese Informationen anzeigen, indem sie im QuickTime-Player "Fenster" > "Filminformationen einblenden" auswählen. (Teilnehmer, die QuickTime Pro nicht installiert haben, können nur die ersten drei Anmerkungen sehen.)
