Ändern der Videoquelle

Der QuickTime Broadcaster kann mit DV-Kameras, iSight-Kameras von Apple, IIDC-Kameras (nur in Mac OS X 10.2 oder neuer unterstützt) und anderen DV-Quellen (wie zum Beispiel Analog-zu-Digital-Konverterboxen) eingesetzt werden, die sich über FireWire an Ihren Computer anschließen lassen.

Gehen Sie wie folgt vor, um eine Videoquelle auszuwählen:

  1. Klicken Sie im QuickTime Broadcaster-Fenster zuerst auf "Details einblenden" (falls das erweiterte Fenster nicht geöffnet ist) und dann auf "Video".

  2. Wählen Sie eine Option aus dem Einblendmenü "Quelle" aus.

Wenn eine Videoquelle angeschlossen ist und Daten überträgt, wird sie in der Regel im Menü "Quelle" angezeigt. In der Vorschau sollte ein Bild zu sehen sein (sofern die Vorschaufunktion aktiviert ist). Auch die Videoeinstellungen sollten aktiviert (also nicht grau angezeigt) sein.

Wenn Ihr Eingabegerät im Menü "Quelle" nicht angezeigt wird, beenden Sie den QuickTime Broadcaster und stellen Sie sicher, dass das Eingabegerät angeschlossen und eingeschaltet ist. Öffnen Sie anschließend den QuickTime Broadcaster erneut. Wenn Ihr Eingabegerät weiterhin nicht angezeigt wird, ist es möglicherweise nicht mit dem QuickTime Broadcaster kompatibel.
