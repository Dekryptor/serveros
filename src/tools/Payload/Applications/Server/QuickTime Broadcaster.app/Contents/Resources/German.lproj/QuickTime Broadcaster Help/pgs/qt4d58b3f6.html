Erstellen von Voreinstellungen

Nachdem Sie einen Kompressor für das Audio- oder Videomaterial ausgewählt haben, können Sie eine Feinabstimmung Ihrer Ergebnisse vornehmen, indem Sie noch andere Optionen auswählen. Bei vielen Audiokompressoren können Sie die Sampling-Rate anpassen oder zwischen Mono- und Stereoton wählen. Bei vielen Videokompressoren können Sie die Bildraten und die Basisbild-Einstellungen (Key Frames) festlegen.

Gehen Sie wie folgt vor, um festzustellen, ob Sie Codec-Einstellungen anpassen können:

Klicken Sie im Bereich "Video" im QuickTime Broadcaster auf die Taste "Optionen" neben dem Namen des Kompressors.

Ist diese Taste grau dargestellt, stehen keine zusätzlichen Einstellungen zur Verfügung.
