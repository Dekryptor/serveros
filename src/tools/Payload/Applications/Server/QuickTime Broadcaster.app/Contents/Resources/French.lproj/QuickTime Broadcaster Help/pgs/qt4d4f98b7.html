Configuration de votre serveur de diffusion

Vous pouvez configurer votre serveur de diffusion, tester votre configuration et accéder à vos données diffusées depuis votre serveur.

Pour configurer votre serveur de diffusion

  1. Ouvrez Admin Serveur.

  2. Dans la liste Services, sélectionnez l’ordinateur que vous voulez configurer comme serveur de diffusion QuickTime.

  3. Cliquez sur Réglages Général dans la barre d’outils, puis cliquez sur l’onglet Services.

  4. Sélectionnez QuickTime Streaming dans la liste des services, puis cliquez sur Enregistrer.

  5. Sélectionnez QuickTime Streaming dans la liste Serveurs pour visualiser et modifier les réglages.

  6. Pour démarrer le service QuickTime Streaming, cliquez sur le bouton Lecture situé à côté de l’angle gauche inférieur de la fenêtre Serveur.
