Modification de la source audio

QuickTime Broadcaster gère l’audio de votre caméra DV ou Apple iSight, les ports d’entrée audio de votre ordinateur et les appareils d’entrée audio USB tels que le Griffin iMic.

Pour choisir une source audio :

  1. Dans la fenêtre QuickTime Broadcaster, cliquez sur Afficher les détails (si la fenêtre n’est pas développée), puis sur Audio.

  2. Choisissez une source audio dans le menu local Source.

Si une source audio est connectée et fournit des données, votre ordinateur doit émettre de l’audio (si la case « Haut-parleur » est cochée dans le panneau Audio), et les réglages audio doivent être disponibles (non estompés).

Si votre source audio n’apparaît pas dans le menu local Source, quittez QuickTime Broadcaster, assurez-vous que votre appareil est connecté et retentez l’opération.
