Enregistrement de réglages à l’aide de préréglages

Vous pouvez enregistrer vos réglages audio, vidéo et réseau pour pouvoir les réutiliser ultérieurement.

Pour créer un préréglage :

  1. Dans QuickTime Broadcaster, choisissez les réglages dans les sous-fenêtres Audio, Vidéo et Réseau.

    Si vous ne voyez pas ces sous-fenêtres, cliquez sur Afficher les détails.

  2. Choisissez Enregistrer dans le menu local Préréglages.

  3. Pour réutiliser ces réglages, choisissez un préréglage dans le menu local Préréglage de chacune des sous-fenêtres (Audio, Vidéo et Réseau).

  4. Pour transférer vos préréglages vers un autre ordinateur, choisissez QuickTime Broadcaster > Préférences, puis tapez un chemin dans le champ Répertoire des préréglages (ou cliquez sur Choisir et sélectionnez un emplacement).

    REMARQUE :  les préréglages doivent se trouver dans ~/Bibliothèque/QuickTime Broadcaster. Les tiers qui utilisent vos préréglages doivent placer les fichiers au même endroit sur leur ordinateur.
