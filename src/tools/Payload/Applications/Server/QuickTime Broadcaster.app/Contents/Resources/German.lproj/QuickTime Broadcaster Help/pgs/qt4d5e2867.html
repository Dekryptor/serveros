Was ist eine SDP-Datei?

Eine SDP-Datei (Session Description Protocol) beschreibt die Parameter und Einstellungen für eine Sendung und ermöglicht auf diese Weise den Empfang der Sendung durch den QuickTime-Player. Öffnen Sie die SDP-Datei mit dem QuickTime-Player, um die Sendung anzusehen.

Sie können eine SDP-Datei für jede der folgenden drei Übertragungsoptionen erstellen:

  • Unicast (automatisch mit Ankündigung):  Die SDP-Datei kann automatisch auf einem bestimmten QuickTime Streaming Server erstellt werden. Die "Standortadresse" wird nur in diesem Fall angezeigt.

  • Unicast (manuell):  Die SDP-Datei wird auf einem Streaming-Server abgelegt oder direkt im QuickTime-Player geöffnet.

  • Multicast:  Die SDP-Datei wird manuell im QuickTime-Player geöffnet.

Gehen Sie wie folgt vor, um eine SDP-Datei zu erstellen:

Wählen Sie im QuickTime Broadcaster "Ablage" > "Exportieren" > "SDP".

Sie können die SDP-Datei an einen einzelnen Zuschauer senden oder an einen Streaming-Server, sodass sich viele Zuschauer in die Sendung zuschalten können.
