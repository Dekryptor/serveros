ブロードキャスト設定を保存する

カスタムブロードキャスト設定を作成して書類として保存して、再利用したりほかの人と共有したりできます。

カスタムブロードキャスト設定を保存するには：

  1. オーディオ、ビデオ、およびネットワーク設定を選びます。

  2. 「ファイル」＞「ブロードキャスト設定を別名で保存」と選択します。

  3. ファイル名と保存場所を指定します。

保存されるファイルは、.qtbr の拡張子が付きます。

ファイルが保存されたら、ファイルのアイコンをダブルクリックすると、「QuickTime Broadcaster」がカスタム設定で開きます（インストールされている場合）。 その後、「ブロードキャスト」をクリックしてストリーミングを開始します。
