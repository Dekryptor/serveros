Verwenden von Multicasting

Multicasting ist eine Sendemethode in einem lokalen Netzwerk (nicht im Internet), bei der ein einzelner Datenstrom von allen Zuschauern gemeinsam verwendet wird. Jeder Teilnehmer schaltet sich in eine Sendung ein, und zwar ganz ähnlich wie bei einem Radio, das sich in eine Radiosendung zuschaltet.

Für das Multicasting ist ein Netzwerk mit MBone-Zugang (Multicast Backbone) oder ein Multicast-fähiges Netzwerk erforderlich.

Das Multicasting erfolgt in einem lokalen Netzwerk, sofern dieses multicast-fähig ist. Fragen Sie Ihren Netzwerkadministrator, ob Ihr Netzwerk Übertragungen per Multicasting unterstützt.

Gehen Sie zum Senden über ein lokales Netzwerk mit der Multicast-Methode wie folgt vor:

  1. Klicken Sie im QuickTime Broadcaster auf "Details einblenden", falls das erweiterte Fenster nicht geöffnet ist.

  2. Klicken Sie auf "Netzwerk".

  3. Wählen Sie "Multicast" aus dem Einblendmenü "Übertragung" aus.

  4. Klicken Sie auf "IP-Adresse erstellen".

    Dabei wird die IP-Adresse eingegeben, die Sie verwenden, um die Multicast-Übertragung zu senden. Nun wird eine Adresse in einem Bereich erstellt, der für Multicast-Sendungen reserviert ist. Wenn Ihr Netzwerkadministrator bereits spezifische Multicast-Adressen zugewiesen hat, können Sie eine dieser Adressen auch manuell eingeben.

  5. Legen Sie eine Einstellung für den TTL-Wert (Time To Live) fest oder übernehmen Sie den Standardwert.

    Diese Einstellung gibt an, wie oft ein Mediendatenstrom von einem Router zum nächsten weitergegeben werden kann, bevor die Übertragung eingestellt wird. Sie können einen beliebigen Wert zwischen 1 und 255 wählen. Beim Wert 1 werden Clientcomputer im lokalen Netzwerk erreicht. Je höher der Wert, desto größer ist die Strecke, die die Multicast-Pakete zurücklegen.

  6. Wählen Sie "Ablage" > "Exportieren" > "SDP" bzw. "Ablage" > "Exportieren" > "Film".

    Damit wird eine SDP-Datei (Session Description Protocol) oder eine QuickTime-Filmdatei erstellt, die Sie an andere Benutzer senden können. Zum Anzeigen der Sendung wählen Sie die SDP-Datei durch Doppelklicken aus oder verwenden Sie den QuickTime-Player, um die Filmdatei zu öffnen.
