Managing Mail Service

Use the Mail pane to turn mail service on or off, specify a relay server for outgoing mail, or adjust junk mail and virus filtering.

To configure the Mail pane:

Change the desired settings, using the following information as a guide:

  • On/Off: 
    Indicates the status of mail service, and clicking the On/Off switch turns the service on or off.
  • Relay outgoing mail through ISP: 
    Select this option only if your ISP requires you to relay outgoing email through a specific server. Click Edit to change the address or the authentication for a relay server that’s already set up.
Specifying a Mail Relay Server
  • Reject email from blacklisted servers: 
    Reject incoming email from senders whose IP addresses are listed as a likely sender of junk mail by the DNSBL server zen.spamhaus.org. To specify a different DNSBL server, click Edit.
Rejecting Mail from Blacklisted Senders
  • Enable junk mail and virus filtering: 
    Scan incoming email, delete any that contains known viruses, and mark any that exceeds the junk score set by the slider.
Scanning for Incoming Junk Mail and Viruses
