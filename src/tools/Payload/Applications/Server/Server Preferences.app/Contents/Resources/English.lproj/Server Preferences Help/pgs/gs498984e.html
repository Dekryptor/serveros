If You Can’t Switch to AirPort Management

If you have an AirPort Extreme Base Station (802.11n) or a Time Capsule but you don’t see a “Switch to AirPort management” button or a “Switch to Firewall security” button when security control is turned off in the Security pane of Server Preferences, try these solutions:

  • Make sure your AirPort Extreme Base Station or Time Capsule is configured to share an Internet connection.

  • Make sure your server’s primary network interface (usually the first built-in Ethernet port) is connected to the same local network as your AirPort Extreme Base Station or Time Capsule.
