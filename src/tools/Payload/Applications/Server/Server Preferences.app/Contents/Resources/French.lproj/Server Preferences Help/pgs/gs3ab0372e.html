Gestion du stockage des sauvegardes des utilisateurs

Snow Leopard Server offre la sécurité et les avantages d’une sauvegarde Time Machine aux utilisateurs disposant de Mac OS X 10.6 ou Mac OS X 10.5 et ayant besoin de conserver des sauvegardes. Les utilisateurs de Snow Leopard n’ayant pas sélectionné un disque de sauvegarde dans la sous-fenêtre Time Machine de Préférences du serveur sont automatiquement invités à indiquer s’ils souhaitent utiliser le serveur comme emplacement de stockage. Les autres utilisateurs doivent ouvrir la sous-fenêtre Time Machine dans Préférences du système et modifier le disque utilisé pour stocker les sauvegardes.

La sous-fenêtre Time Machine dans Préférences du serveur permet d’augmenter l’espace disque disponible sur le serveur pour sauvegarder les ordinateurs des utilisateurs ou de modifier le disque utilisé pour stocker les sauvegardes utilisateur.

Pour configurer la sous-fenêtre Time Machine de Préférences du serveur :

Modifiez les réglages selon vos préférences en utilisant les informations suivantes pour vous guider :

  • On/Off : 
    Cela rend le dossier partagé /Shared Items/Backups/ disponible ou non disponible pour stocker les sauvegardes des utilisateurs sur le disque spécifié dans cette sous-fenêtre.
  • Choisir un disque : 
    Cliquez sur ce bouton pour sélectionner un emplacement de stockage à partir d’une liste contenant tous les disques connectés au serveur. Mac OS X Server crée le dossier partagé Sauvegardes sur le disque que vous sélectionnez.
  • Choisir un disque de sauvegarde : 
    Ce bouton apparaît si aucun disque n’est sélectionné pour stocker les sauvegardes utilisateur. Cliquez sur ce bouton pour rendre le dossier partagé Sauvegardes disponible et sélectionner son emplacement sur le disque. Cela revient à activer la sous-fenêtre Time Machine.
  • Ouvrir Partage de fichiers : 
    Ce bouton apparaît si le service de partage de fichiers est désactivé. Cliquez sur le bouton pour ouvrir la sous-fenêtre Partage de fichiers. Le service de partage de fichiers doit être activé pour que les Mac des utilisateurs accèdent au dossier partagé Sauvegardes.

Vous pouvez contrôler l’accès de chaque utilisateur au stockage des sauvegardes Time Machine sur le serveur à partir de la sous-fenêtre Utilisateurs ou afficher le dossier partagé Sauvegardes dans le Finder en cliquant sur les flèches dans le texte sous les informations du disque de sauvegarde.

Si vous changez de disque de sauvegarde, les préférences Time Machine des utilisateurs, configurées afin d’utiliser le serveur pour stocker les sauvegardes, commencent automatiquement à utiliser le dossier Sauvegardes dans son nouvel emplacement. Mac OS X Server ne copie toutefois pas dans le nouveau dossier les données sauvegardées dans l’ancien dossier. Après avoir sélectionné un autre disque de sauvegarde, vous devez informer les utilisateurs que leur première sauvegarde prendra plus de temps car il s’agit d’une sauvegarde complète.
