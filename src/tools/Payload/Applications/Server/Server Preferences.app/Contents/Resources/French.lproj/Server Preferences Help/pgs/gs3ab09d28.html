Remplacement d’un certificat auto-signé

Quand l’autorité de certification vous envoie un certificat signé, vous pouvez utiliser celui-ci pour remplacer votre certificat auto-signé.

Pour remplacer un certificat auto-signé :

  1. Dans le Finder, localisez le fichier du certificat signé envoyé par l’autorité de certification et placez-le dans un emplacement visible lorsque vous utilisez Préférences du serveur, par exemple sur le bureau.

  2. Dans la sous-fenêtre Informations de Préférence du serveur, cliquez sur le bouton Modifier situé à droite de Certificat SSL.

  3. À partir du menu local; choisissez le certificat auto-signé à remplacer.

    « Utiliser un certificat SSL » doit être sélectionné pour que le menu local soit disponible.

  4. À partir du menu local, choisissez Signature de certificat > Remplacer par un certificat signé ou renouvelé.

  5. Faites glisser le fichier contenant le certificat signé au milieu de la zone de dialogue dans Préférences du serveur, puis cliquez sur Remplacer le certificat.
