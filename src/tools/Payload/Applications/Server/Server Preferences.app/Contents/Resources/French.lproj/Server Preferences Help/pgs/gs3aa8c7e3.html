Démarrage ou arrêt du service Carnet d’adresses

Vous pouvez arrêter ou démarrer le service Carnet d’adresses à partir de la sous-fenêtre Carnet d’adresses. Quand le service Carnet d’adresses est arrêté, les utilisateurs ne peuvent accéder aux coordonnées situées dans le carnet d’adresses du serveur.

Pour démarrer ou arrêter le service Carnet d’adresses :

Cliquez sur le bouton Marche/Arrêt de la sous-fenêtre Carnet d’adresses dans Préférences du serveur.

Après avoir démarré le service Carnet d’adresses, en particulier s’il s’agit de la première fois, vous devez dire aux utilisateurs de configurer leur application Carnet d’adresses comme indiqué dans Configuration manuelle des ordinateurs des utilisateurs.
