サーバでアプリケーションファイアウォールが使用されている場合

お使いのサーバを Mac OS X Server v10.5 Leopard からアップグレードした場合で、Snow Leopard Server にアップグレードする前にそのサーバのファイアウォールを「サーバ環境設定」の「ファイアウォール」パネルで有効にした場合、「セキュリティ」パネルではファイアウォールを制御できません。この場合、お使いのサーバでは、アプリケーションファイアウォールと呼ばれる別のファイアウォール技術が使用されており、このファイアウォールは「システム環境設定」の「セキュリティ」パネルで管理できます。「システム環境設定」の「セキュリティ」パネルでファイアウォールを無効にした場合、サーバのファイアウォールは「サーバ環境設定」で管理できます。

サーバのファイアウォールを「サーバ環境設定」で管理するには：

  1. サーバで（「サーバ環境設定」ではなく）「システム環境設定」の「セキュリティ」パネルを開き、「ファイアウォール」をクリックします。

    「システム環境設定」の「セキュリティ」パネルを開く

  2. 「停止」をクリックしてアプリケーションファイアウォールを無効にします。
