Restriction de l’accès à un site web personnalisé

Vous pouvez restreindre l’accès à chaque site web personnalisé ou aux dossiers contenus dans le dossier de ce site. Vous pouvez restreindre l’accès à un seul groupe qui peut afficher l’intégralité du site web ou restreindre l’accès à certains dossiers présents dans le dossier de ce site. Les dossiers d’accès restreint d’un site web sont appelés des royaumes.

Pour contrôler l’accès à un site web :

  1. Dans la sous-fenêtre Web de Préférences du serveur, cliquez sur Sites personnalisés.

  2. Sélectionnez un site web dans la liste, notez son emplacement de dossier, puis cliquez sur le bouton Modifier (crayon).

  3. Pour désigner le groupe pouvant accéder à l’intégralité du site web, choisissez ce groupe dans le menu local Qui peut visualiser ce site.

  4. Pour désigner des groupes pouvant accéder à des dossiers présents dans le site web, sélectionnez Personnaliser dans le menu local Qui peut visualiser ce site.

    Vous pouvez contrôler l’accès uniquement à des dossiers existants. N’hésitez pas à utiliser le Finder pour ajouter des dossiers au dossier du site web dont vous avez noté l’emplacement à l’étape 2.
