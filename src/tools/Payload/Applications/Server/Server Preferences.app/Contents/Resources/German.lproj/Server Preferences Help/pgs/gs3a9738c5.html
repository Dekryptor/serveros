Ändern des Namens oder des Kennworts eines Benutzers

Mit dem Programm "Servereinstellungen" können Sie den Namen und/oder das Kennwort für den Account eines Standardbenutzers oder eines Administrators ändern.

Gehen Sie wie folgt vor, um den Namen und/oder das Kennwort für einen Benutzer zu ändern:

  1. Öffnen Sie die Servereinstellung "Benutzer" und wählen Sie in der Liste links den Account des Standardbenutzers oder Administrators aus, den Sie ändern wollen. Klicken Sie danach auf "Benutzer".

  2. Bearbeiten Sie den aktuellen Eintrag im Feld "Name", wenn Sie den Benutzernamen ändern wollen.

    Der Name kann bis zu 255 Zeichen lang sein (was u. U. nur 85 japanischen Zeichen entspricht). Der Name darf Leerzeichen enthalten.

    Das Ändern des Kurznamens ist mit dem Programm "Servereinstellungen" nicht möglich.

  3. Klicken Sie auf "Kennwort zurücksetzen", wenn Sie das Kennwort für den Benutzer ändern wollen. Geben Sie das gewünschte neue Kennwort in die Felder "Neues Kennwort" und "Bestätigen" ein und klicken Sie abschließend auf "Kennwort ändern".

    Sie können sich beim Festlegen des Kennworts vom Kennwortassistenten unterstützen lassen. Klicken Sie auf das Schlüsselsymbol rechts vom Feld "Neues Kennwort", wenn Sie wissen wollen, wie sicher das von Ihnen gewählte Kennwort ist.

Benutzer können ihre Kennwörter in der Systemeinstellung "Benutzer" auf ihren Computern ändern.

Für Benutzer-Accounts, die einzeln oder als Gruppen von einem Verzeichnisserver importiert wurden, können die Benutzernamen und Kennwörter nur von einem Administrator des Verzeichnisservers geändert werden, auf dem die Account-Informationen gespeichert sind. Wenn auf diesem Verzeichnisserver Mac OS X Server 10.6 installiert ist und Sie Administratorrechte besitzen, können Sie im Programm "Servereinstellungen" u. U. per Fernzugriff die Verbindung zum Server herstellen und die Änderungen über das Netzwerk vornehmen.

Verwenden Sie die Systemeinstellung "Benutzer" auf dem Server, um das Kennwort oder andere Attribute des primären Administrator-Accounts zu ändern, der bei der Erstkonfiguration des Servers erstellt wurde. Weitere Informationen über Änderungen an lokalen Accounts erhalten Sie, wenn Sie die Systemeinstellungen öffnen und deren Hilfe anzeigen.

Möchten Sie das Kennwort oder andere Attribute des Verzeichnis-Administrator-Accounts ändern, der bei der Erstkonfiguration erstellt wurde, verwenden Sie das Programm "Arbeitsgruppenmanager" (im Ordner "/Programme/Server/"). Anleitungen hierzu finden Sie im Menü "Hilfe" im Arbeitsgruppenmanager.

Informationen über das Ändern des Kennworts des Systemadministrator-Accounts (root) finden Sie in der Einführung.
