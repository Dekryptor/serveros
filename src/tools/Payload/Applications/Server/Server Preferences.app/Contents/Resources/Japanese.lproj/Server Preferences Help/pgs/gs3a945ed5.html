サービスの状況を確認する

「サーバ環境設定」では、各サービスの全体的な状況がサービスアイコンの横のインジケータに示されます。

すべてのサービスの状況インジケータを表示するには：

「サーバ環境設定」ウインドウで、「すべてを表示」をクリックします。

インジケータが緑色の場合は、サービスが機能しています。

サービスを有効または無効にするには：

  1. 「サーバ環境設定」ウインドウで、有効または無効にしたいサービスのアイコンをクリックします。

    サービスのアイコンが表示されていない場合は、「サーバ環境設定」ウインドウの「すべてを表示」をクリックします。「表示」メニューを使って、有効または無効にしたいサービスを選択することもできます。

  2. サービスパネルの「入」／「切」スイッチをクリックします。

各サービスパネルの「入」／「切」スイッチは、そのサービスが実行中であるかどうかを示します。

「サーバ状況」ウィジェットを使ってサービスの状況を監視することもできます。
