If You Can’t Change the Short Name of a User Account

You can’t change the short name for a user account after the account has been created. If you need to change the short name, you must delete the account and create a new user account with a new short name. For instructions, see Deleting a User Account and Adding a User Account.

Deleting a user account cancels its group memberships and stops its access to group services and private wikis. Deleting a user account also deletes the user’s mail stored on the server. A deleted user account can no longer access calendars and address book information on the server.

After you create a new account on the server for a user with Mac OS X v10.6 Snow Leopard, the user must go to the Login Options section of Accounts preferences, remove the server from the directory servers list, and add it back. For information about connecting to your server, see Setting Up a Mac by Using Accounts Preferences.
