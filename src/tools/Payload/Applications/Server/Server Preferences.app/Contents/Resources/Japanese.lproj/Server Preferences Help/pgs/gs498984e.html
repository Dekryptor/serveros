AirMac マネージメントに切り替えることができない場合

AirMac Extreme ベースステーション（802.11n）または Time Capsule を使用しており、「サーバ環境設定」の「セキュリティ」パネルでセキュリティ制御が無効になっているときに「代わりに AirMac マネージメントを使用」ボタンまたは「代わりにファイアウォールセキュリティを使用」ボタンが表示されない場合は、以下の解決法を試してください：

  • AirMac Extreme ベースステーションまたは Time Capsule が、インターネット接続を共有するよう設定されていることを確認します。

  • サーバの主要なネットワークインターフェイス（通常は 1 つ目の内蔵 Ethernet ポート）が、AirMac Extreme ベースステーションまたは Time Capsule と同一のローカルネットワークに接続していることを確認します。
