最新のヘルプトピック

コンピュータがインターネットに接続している場合は、アップデートされた新しいヘルプトピックが自動的に取得されます。これらのトピックのいくつかによって、Mac OS X Server に付属の「お使いになる前に」マニュアルのトピックもアップデートされます。

  • 最新のヘルプトピックのリストを見るには、「ホーム」をクリックし、次に「最新のヘルプトピック」というリンクをクリックします。新しいヘルプトピックが公開されているかどうか、定期的に確認してください。

  • アップルの Mac OS X Server サポート Web サイトの最新情報を検索するには、上にある検索フィールドを使います。この検索の有効／無効を切り替えるには、検索フィールドにあるポップアップメニューで「製品サポート情報の検索結果を表示」を選択します。

  • 「お使いになる前に」および Mac OS X Server のオンスクリーンヘルプの最新アップデートが一覧される RSS フィードを表示するには、RSS リーダーアプリケーション（Safari、Mail など）を使って、次の URL にアクセスします：

feed://helposx.apple.com/rss/snowleopard/serverdocupdates.xml
