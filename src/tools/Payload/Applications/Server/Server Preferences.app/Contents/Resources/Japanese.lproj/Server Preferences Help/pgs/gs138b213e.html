Web サイトのカスタム設定を変更する

カスタム Web サイトの IP アドレス、アクセスポート、サイトのフォルダの場所、およびアクセス権限を変更できます。

Web サイトの設定を変更するには：

  1. 「サーバ環境設定」の「Web」パネルで「カスタムサイト」をクリックします。

  2. 変更したい Web サイトを選択し、編集ボタン（鉛筆）をクリックして、以下の情報を参考に設定を変更します：

    • IP アドレス： 
      サーバの IP アドレスが複数存在する場合は、それらのいずれかを選択するか、「任意」（ユーザがそれらのいずれかを使って Web サイトにアクセスできるようになります）を選択します。
    • ポート： 
      アクセスポート番号を入力します。標準のアクセスポート番号は 80 です。別のポート番号を入力する場合は、サーバによって提供される別のサービスと競合していないことを、あらかじめ確認してください。Mac OS X Server で使用されるポートのリストについては、「サーバ管理」（「/アプリケーション/サーバ/」にあります）を開き、「ヘルプ」メニューで「TCP および UDP ポートの参照」を検索します。
    • サイトファイルの保存場所： 
      Web サイトで使用したいファイルが含まれたフォルダを選択します。
    • このサイトを表示できるユーザ： 
      特定のグループのメンバーのみが Web サイトにアクセスできるように、そのグループを選択します。または、「カスタマイズ」を選択して、Web サイトフォルダ内の各フォルダへのアクセスを制限します。操作方法については、「カスタム Web サイトへのアクセスを制限する」を参照してください。
