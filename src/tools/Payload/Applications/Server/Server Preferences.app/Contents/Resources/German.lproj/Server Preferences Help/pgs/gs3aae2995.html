Einrichten von Wikis

Ist der Dienst "Wikis" in der Servereinstellung "Web" aktiviert, können alle Serverbenutzer mit einem Webbrowser Wikis erstellen.

Gehen Sie wie folgt vor, um den Dienst "Wikis" zu aktivieren oder zu deaktivieren:

Klicken Sie in der Servereinstellung "Web" auf "Webdienste" und aktivieren oder deaktivieren Sie dann "Wikis".

Wenn diese Option ausgewählt ist und die Webdienste aktiviert sind, wird durch Klicken auf den als Link definierten Pfeil daneben die Liste aller Wikis geöffnet.

Wenn Wikis verfügbar sind, können Benutzer auf ihre eigenen Wiki-Seiten sowie auf Wikis zugreifen, für die sie die erforderliche Berechtigung haben. Die Berechtigung zur Nutzung eines Wikis kann im Abschnitt "Einstellungen" des Wikis geändert werden. Der Eigentümer des Wikis oder ein Serveradministrator kann Änderungen am Abschnitt "Einstellungen" eines Wikis vornehmen.

Benutzer können auf Wikis zugreifen, indem sie den Zusatz "/groups/" an die Website-Adresse des Servers anhängen. Hier ein Beispiel:

http://server.beispiel.de/groups/

Benutzer mit Accounts in der Systemeinstellung "Benutzer" können nach der Anmeldung an ihrem Wiki-Portal "Meine Seite" neue Wikis erstellen.

Informationen über die Verwendung eines Wikis erhalten Sie, indem Sie auf einer Wiki-Seite auf "Hilfe" klicken.
