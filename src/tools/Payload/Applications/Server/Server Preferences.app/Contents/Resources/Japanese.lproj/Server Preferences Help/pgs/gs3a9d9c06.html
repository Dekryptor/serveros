ホーム
ユーザのコンピュータを手動で設定する

Mac OS X v10.5 Leopard 以前を使用するユーザ、または Windows を実行するユーザは、アプリケーションを手動で構成して、サーバのサービスを使用できます。次の設定を使用できます。斜体のプレースホルダは、サーバの DNS 名とユーザのユーザ名に置き換えてください。通常は、Mac OS X v10.6 Snow Leopard を持っているユーザには、使用するアプリケーションを自動設定するための案内が表示されますが、手動でそれらのアプリケーションを設定することも可能です。案内については、「お使いのネットワークに参加する Mac を設定する」を参照してください。

アプリケーションの設定

Finder またはクライアントを共有する別の SMB または AFP ファイル

afp://myserver.example.com

smb://myserver.example.com

アドレスブック（Mac OS X v10.6 Snow Leopard に付属）または別の CardDAV 対応の連絡先管理アプリケーション

サーバアドレス：myserver.example.com

ユーザ名：サーバ上のユーザアカウントの名前

パスワード：サーバ上のユーザアカウントのパスワード

iCal（Mac OS X v10.6 Snow Leopard に付属）

アカウントの種類：自動（CalDAV）

メールアドレス：usershortname@myserver.example.com

iCal（Mac OS X Server v10.5 Leopard に付属）または別の CalDAV カレンダーアプリケーション

個人用カレンダー照会（読み出しのみ）：http://myserver.example.com:8008/calendars/users/usershortname/calendar/

iChat または別の XMPP インスタント・メッセージ・アプリケーション

アカウントの種類：Jabber

サーバアドレス：myserver.example.com

Jabber ID：usershortname@myserver.example.com

認証：Kerberos v5 を推奨（別の標準方式もサポート）

ポート：5223

Mail または別のメールアプリケーション

アカウントの種類：IMAP または POP

受信メールサーバ：myserver.example.com

送信メールサーバ：myserver.example.com

メールアドレス：usershortname@myserver.example.com

認証：Kerberos v5 を推奨（別の標準方式もサポート）

Safari または別の Web ブラウザ

サーバの Web サイト：http://myserver.example.com

リンクをクリックすると、「マイページ」、Wiki、ブログ、カレンダー、および Web メールが表示されます。

「ネットワーク」環境設定 または別の VPN 接続（Mac OS X v10.4 Tiger の「インターネット接続」など）

「Mac ユーザの VPN 接続を設定する」または「ユーザの VPN 接続を手動で設定する」を参照してください。
