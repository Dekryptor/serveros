Suppression d’un dossier partagé

Vous pouvez utiliser Préférences du serveur pour supprimer les dossiers partagés que vous ne souhaitez plus utiliser pour le partage de fichiers. Lorsque vous supprimez un dossier partagé, le dossier et son contenu restent sur le disque du serveur.

Pour supprimer un dossier partagé :

Dans la sous-fenêtre Partage de fichiers de Préférences du serveur, sélectionnez le dossier partagé à supprimer, puis cliquez sur le bouton Supprimer (–).
