Si le serveur manque d’espace pour sauvegarder les Mac des utilisateurs

Si le serveur manque d’espace pour sauvegarder les Mac des utilisateurs, vous pouvez connecter un autre disque au serveur et l’utiliser comme emplacement de stockage dans la sous-fenêtre Time Machine des Préférences du serveur. Pour obtenir des instructions, consultez Gestion du stockage des sauvegardes des utilisateurs.

Si Time Machine n’est pas activé dans Préférences du système pour certains utilisateurs, ceux-ci sont automatiquement invités à indiquer s’ils souhaitent utiliser le serveur comme emplacement de stockage. Les autres utilisateurs doivent sélectionner le serveur comme disque de sauvegarde dans la sous-fenêtre Time Machine des Préférences du système.
