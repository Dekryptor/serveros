Starting or Stopping iCal Service

You can stop or start iCal calendar service in the iCal pane. While iCal service is off, users can’t access shared calendars.

To start or stop iCal service:

Click the On/Off switch in the iCal pane of Server Preferences.

After starting iCal service, especially if you’re starting it for the first time, you should advise users to set up their calendar applications as described in Setting Up Users’ Computers Manually.
