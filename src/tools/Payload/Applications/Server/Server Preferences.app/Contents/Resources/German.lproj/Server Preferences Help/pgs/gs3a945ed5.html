Prüfen des Status von Diensten

Im Programm "Servereinstellungen" wird der Status der einzelnen Dienste durch eine Anzeige neben dem Dienstsymbol gekennzeichnet.

Gehen Sie wie folgt vor, um die Statusanzeige für alle Dienste anzuzeigen:

Klicken Sie im Fenster "Servereinstellungen" auf "Alle einblenden".

Eine grüne Anzeige deutet darauf hin, dass der Dienst gestartet ist.

Gehen Sie wie folgt vor, um einen Dienst zu aktivieren oder zu deaktivieren:

  1. Klicken Sie im Fenster "Servereinstellungen" auf das Symbol des Diensts, den Sie aktivieren oder deaktivieren möchten.

    Werden die Symbole der Dienste nicht angezeigt, klicken Sie im Fenster "Servereinstellungen" auf "Alle einblenden". Sie können auch das Menü "Darstellung" verwenden, um den Dienst auszuwählen, der aktiviert oder deaktiviert werden soll.

  2. Klicken Sie auf die Taste "Ein/Aus" im Bereich des jeweiligen Diensts.

Die Taste "Ein/Aus" in jedem Dienstbereich gibt an, ob der betreffende Dienst aktiviert ist.

Mithilfe des Widgets "Serverstatus" können Sie den Status des Diensts überwachen.
