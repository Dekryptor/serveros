アドレスブックサービスを開始する／停止する

「アドレスブック」パネルでアドレスブックサービスを開始または停止できます。アドレスブックサービスが無効の間は、ユーザはサーバベースのアドレスブックにある連絡先情報にアクセスできません。

アドレスブックサービスを開始または停止するには：

「サーバ環境設定」の「アドレスブック」パネルで「入」／「切」スイッチをクリックします。

アドレスブックサービスを開始した後、特にこのサービスをはじめて開始した場合は、「ユーザのコンピュータを手動で設定する」で説明している通りに「アドレスブック」アプリケーションを設定するようユーザに通知してください。
