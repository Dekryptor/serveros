Si un utilisateur peut accéder à certains services uniquement

Les réglages d’un compte utilisateur contrôlent l’accès par l’utilisateur aux services fournis par votre serveur.

Pour autoriser un utilisateur à accéder à davantage de services :

  1. Dans la sous-fenêtre Utilisateurs de Préférences du serveur, sélectionnez un utilisateur dans la liste.

  2. Cliquez sur Services et cochez les services auxquels l’utilisateur peut accéder.
