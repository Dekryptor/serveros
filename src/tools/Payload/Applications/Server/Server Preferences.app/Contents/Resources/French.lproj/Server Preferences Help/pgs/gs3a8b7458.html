Outils et applications avancés

En plus des Préférences du serveur et des autres applications d’administration de base, Snow Leopard Server inclut les applications et les outils d’administration avancés décrits dans le tableau suivant. Pour en savoir plus sur ces outils et ces applications, ouvrez Admin Serveur et utilisez le menu Aide ou consultez le site web Ressources Mac OS X Server à l’adresse www.apple.com/fr/server/macosx/resources/ (en anglais).

Important :  si certains de vos outils ou de vos applications proviennent de Mac OS X Server 10.5 ou antérieur, ne les utilisez pas avec Snow Leopard Server.
Applications et outils avancés pour les administrateurs de serveurs

Utilitaire d’annuaire

Configurez des connexions avancées aux serveurs d’annuaire. Pour ouvrir Utilitaire d’annuaire, cliquez sur Modifier (ou sur Se connecter) dans la section « Options d’ouverture de session » de la sous-fenêtre Comptes des Préférences Système.

Capture de Podcast (dans /Applications/Utilitaires/)

Enregistrez des fichiers audio et vidéo de haute qualité à partir d’une caméra locale ou distante, capturez ce qui s’affiche à l’écran ou téléchargez des fichiers QuickTime dans Podcast Producer afin de les encoder et de les distribuer.

Podcast Composer (dans /Applications/Server/)

Suivez un processus structuré et graphique afin de créer des flux de travaux qui contrôlent la génération et la distribution de podcasts par Podcast Producer.

QuickTime Broadcaster (dans /Applications/)

Capturez en direct des fichiers audio et vidéo qui fonctionnent de façon transparente avec QuickTime Streaming Server pour obtenir une diffusion en réseau de haute qualité.

RAID Admin (dans /Applications/Server/)

Configure et surveille le matériel Xserve RAID.

Admin Serveur (dans /Applications/Server/)

Modifiez des réglages de services avancé, configurez des services avancés et gérez des points de partage de fichiers. Surveille l’activité du serveur et affiche les historiques de maintenance détaillés.

Assistant du serveur

Configurez automatiquement plusieurs serveurs à l’aide de profils de configuration automatique enregistrés. Pour ouvrir Assistant du serveur, utilisez le menu Serveur dans Admin Serveur.

Contrôle de serveur (dans /Applications/Server/)

Surveille et gère à distance un ou plusieurs systèmes Xserve.

Utilitaire d’images de système (dans /Applications/Server/)

Créez des images NetBoot et NetInstall pour les ordinateurs Mac OS X et Mac OS X Server.

Préférences Système (dans /Applications/)

Connecte le serveur à un serveur de répertoire de votre organisation.

Gestionnaire de groupe de travail (dans /Applications/Server/)

Gère les utilisateurs, les groupes, les ordinateurs et les groupes d’ordinateurs dans le cadre de déploiements de serveur avancés. Gère les préférences pour les utilisateurs Mac OS X.

Xgrid Admin (dans /Applications/Server/)

Gérez à distance des grappes de serveurs, surveillez l’activité des contrôleurs et des agents et vérifiez l’état des tâches sur la grille.

Utilitaires de ligne de commande

Utilisez des outils UNIX pour installer et configurer le logiciel serveur, administrer des services, gérer des utilisateurs et effectuer bien d’autres opérations.
