Making All Group Members iChat Buddies

You can have the Jabber IDs (screen names) of all users who are members of a group automatically added to each group member’s Jabber buddy list. Group members see their Jabber buddy lists in iChat (or other XMPP instant messaging application), and may add and remove buddies. If the group has external members, they aren’t included in the iChat autobuddy list.

To make all group members Jabber buddies:

  1. In the Groups pane of Server Preferences, select the group you want to change in the list on the left, and then click Services.

  2. Select the checkbox next to iChat Autobuddy List.

  3. Restart iChat service by clicking On/Off twice.

    Changes that affect iChat service take effect once iChat service is restarted.

If you turn off iChat AutoBuddy List, group members are not automatically removed from each other’s buddy lists. Group members can remove buddies that were automatically added to their buddy lists.
