Customizing Welcome and Invitation Email

You can use Server Preferences to customize email messages that welcome new users or invite existing users to use the server. Your server can send all of the following messages if it’s configured to import users from a directory server. Your server sends only the welcome message if your server isn’t connected to a directory server.

Welcome Email

Tells new users about the services your server offers. Automatically sent to new users whose computers are getting mail service from your server. Not sent if your server’s mail service is off.

Invitation Email

Tells newly imported users how to get services from your server. You can send it to users when you import their accounts. You can also send it manually.
