Gestion des informations du serveur

Utilisez la sous-fenêtre Informations de Préférences du serveur pour obtenir des informations sur votre serveur, notamment le matériel et les logiciels installés, les noms et l’adresse réseau, ainsi que le numéro de série. Vous pouvez également modifier le nom d’ordinateur et le numéro de série du serveur, gérer les alertes par courrier électronique et les certificats SSL.

Pour obtenir ou modifier des informations relatives au serveur :

Recherchez les éléments suivants dans la sous-fenêtre Informations :

  • Adresse IP : 
    Adresse numérique unique qui identifie le serveur sur le réseau. Pour en savoir plus, consultez Modification de l’adresse IP de votre serveur.
  • Nom DNS : 
    Nom unique complet attribué à votre serveur par le serveur DNS.
  • Nom de l’ordinateur : 
    Identifie le serveur auprès des utilisateurs qui recherchent des serveurs de fichiers en réseau ou d’autres ressources réseau identifiés par un nom d’ordinateur plutôt que par un nom DNS. Pour en savoir plus consultez Modification du nom de votre serveur.
  • Licence serveur : 
    Mac OS X Server détermine le type de licence à partir du numéro de série du logiciel ou de la licence de site. Cliquez sur Modifier pour modifier le numéro de licence ou les informations de licence du site. Pour obtenir des instructions, consultez Modification du numéro de série ou des détails de la licence de site.
  • Alertes : 
    Adresse électronique à laquelle le serveur envoie des messages relatifs aux événements système importants. Cliquez sur Modifier pour changer l’adresse ou activer ou désactiver certains types d’alertes. Pour obtenir des instructions, consultez Modification des réglages des alertes.
  • Certificat SSL : 
    Créez, utilisez et gérez les identités des services qui utilisent SSL pour leur sécurité. Pour obtenir des instructions, consultez Utilisation d’un certificat SSL.

La sous-fenêtre Informations située à gauche affiche le numéro de version et le numéro de compilation du logiciel Mac OS X Server installé, la vitesse et le type du ou des processeurs et la quantité totale de mémoire installée sur le serveur.

Pour afficher et modifier d’autres informations réseau, utilisez la sous-fenêtre Réseau des Préférences Système sur votre serveur. Pour en savoir plus sur la modification des préférences Réseau, ouvrez les Préférences Système et consultez le menu Aide.

Pour afficher des informations supplémentaires sur le serveur, utilisez Informations Système Apple (situé dans /Applications/Utilitaires/). Pour plus d’informations concernant l’utilisation du Rapport d’informations système, ouvrez l’application, puis utilisez le menu Aide.
