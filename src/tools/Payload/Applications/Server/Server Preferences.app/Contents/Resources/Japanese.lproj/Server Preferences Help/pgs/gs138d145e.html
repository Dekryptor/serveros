証明書を読み込む

SSL 証明書が含まれたファイルと、一致する秘密鍵を持っている場合は、証明書を読み込み、それらを利用してサーバが提供するサービスを保護することができます。

証明書を読み込むには：

  1. 「サーバ環境設定」を使用しているとき、証明書および一致する秘密鍵が含まれたファイルを Finder で探し、ファイルを目に見える任意の場所（デスクトップなど）に置きます。

  2. 「サーバ環境設定」の「情報」パネルで、「SSL 証明書」の右側にある「編集」ボタンをクリックします。

  3. ポップアップメニューで「証明書の読み込み」＞「証明書を読み込む」と選択します。

    「SSL 証明書を使用」が選択されていないと、ポップアップメニューは表示されません。

  4. 証明書と秘密鍵が含まれたファイルを、「サーバ環境設定」にあるダイアログの中央にドラッグして、「読み込む」をクリックします。

  5. 読み込んだ証明書を使うには、ポップアップメニューでそれを選択します。
