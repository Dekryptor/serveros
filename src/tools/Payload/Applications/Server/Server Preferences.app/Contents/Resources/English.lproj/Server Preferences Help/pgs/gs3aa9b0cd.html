Managing VPN Service

Use the VPN pane to turn VPN remote access service on or off, inspect or change the VPN secret, set the IP address range for VPN users, or save a VPN configuration file for Mac OS X users.

To configure the VPN pane:

Change the desired settings, using the following information as a guide:

  • On/Off: 
    Indicates the status of VPN service, and clicking the On/Off switch turns the service on or off.
  • Shared Secret: 
    Enables the server to trust remote computers that have the shared secret, and allows remote computers to trust the server. Click Edit to read or change the shared secret. If you change the secret here, all VPN users must make the same change in their VPN configurations. For instructions, see Changing the VPN Shared Secret.
  • IP Address Range: 
    Remote computers are assigned an IP address from this range when they make a VPN connection. For instructions, see Changing the IP Address Range for VPN.
  • Client VPN configuration file: 
    Click Save As to automatically generate a file that you can give users who want remote access to the server from Macs with Mac OS X v10.3 or later. For instructions, see Creating a VPN Configuration File.
