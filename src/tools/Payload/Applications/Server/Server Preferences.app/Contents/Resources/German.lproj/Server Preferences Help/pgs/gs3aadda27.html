Dienst "iCal"

Mithilfe von iCal-Server, dem Kalenderdienst für Mac OS X Server, können Benutzer ganz einfach Kalender gemeinsam nutzen, Meetings planen und Ereignisse innerhalb einer Arbeitsgruppe, eines kleinen Unternehmens oder einer großen Organisation koordinieren. Kollegen haben die Möglichkeit, gegenseitig ihre Verfügbarkeit zu prüfen, Konferenztermine zu planen und vorzuschlagen, Konferenzräume zu buchen, Projektoren zu reservieren und mehr. iCal-Server sendet die Einladungen, die Informationen wie eine Tagesordnung oder Aufgabenlisten umfassen können, und sortiert die Antworten.

Für einen Benutzercomputer mit Mac OS X 10.6 Leopard kann das Programm "iCal" automatisch so eingerichtet werden, dass es iCal-Server verwendet. Vgl. Automatisches Konfigurieren der Mac-Computer von Benutzern. Informationen zum Planen von Ereignissen, Einladen anderer Benutzer, Empfangen und Antworten auf Einladungen usw. finden Mac OS X-Benutzer im Menü "Hilfe" in iCal.

iCal-Server funktioniert auch mit dem Programm "Kalender" auf dem iPhone, mit Webkalendern in Snow Leopard Server-Wikis und anderen gängigen Kalenderprogrammen, die das standardmäßige CalDAV-Protokoll unterstützen. Vgl. Manuelles Konfigurieren von Benutzercomputern.

Wenn Sie Zugriff auf iCal-Server im Internet gewähren möchten und einen Kabel-Router, DSL-Router oder einen anderen Netzwerk-Router verwenden, muss für diesen Router die Portweiterleitung (Portzuordnung) für iCal-Server konfiguriert sein. Weitere Informationen über die Portweiterleitung finden Sie in der Einführung.

Wenn Sie den Zugriff auf iCal-Server von außerhalb Ihres lokalen Netzwerks erlauben möchten und Ihr lokales Netzwerk über eine separate Firewall verfügt, bitten Sie den Administrator der Firewall, die Firewall für die Ports zu öffnen, die iCal-Server nutzt. Eine Liste der verfügbaren Ports finden Sie im Anhang der Einführung.
