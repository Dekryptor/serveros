iChat サービスについて

iChat サービスで、Macintosh、Windows、Linux のユーザはインスタントメッセージ（IM）を使用できます。チームのメンバーは、部外者による機密情報の傍受を心配することなく、問題の解決方法を話し合ったり、計画を立てたり、URL を交換したり、ファイルを転送したりできます。iChat サービスを使用すると、複数のユーザ間でテキストメッセージを交換できます。また、iChat は、オーディオ、ビデオ、および多方向のオーディオとビデオによるセッションのためのユーザ間の直接接続を容易にします。

ユーザの iChat アカウント情報は、サーバ上に保管されます。ユーザは、どの Mac からでも自分のアカウントにアクセスでき、画面には同じメンバーリストやグループなどが表示されます。

Mac OS v10.6 Snow Leopard を実行するコンピュータでは、サーバの iChat サービスを使用するために「iChat」アプリケーションを自動設定することができます。「ユーザの Mac を自動的に設定する」を参照してください。

iChat サービスは Mac OS X の旧バージョンの「iChat」アプリケーション、Google トーク、および XMPP プロトコルを使用するそのほかのインスタント・メッセージング・ソフトウェア（Windows および Linux コンピュータで使用可能で、Jabber と呼ばれます）にも対応しています。

インターネット上の iChat サービスへのアクセスを許可したい場合で、ケーブル・ルーター、DSL ルーター、またはそのほかのネットワークルーターを持っている場合は、お使いのルーターで iChat サービス用にポートフォワード（ポートマッピング）を設定する必要があります。ポートフォワードの詳細については、「お使いになる前に」マニュアルを参照してください。

ローカルネットワークの外部の iChat サービスへのアクセスを許可したい場合で、お使いのローカルネットワークに独立したファイアウォール装置がある場合は、ファイアウォールの管理者に問い合わせて、iChat サービスで使用するポートのファイアウォールを解放してもらってください。ポートのリストについては、「お使いになる前に」マニュアルの付録を参照してください。
