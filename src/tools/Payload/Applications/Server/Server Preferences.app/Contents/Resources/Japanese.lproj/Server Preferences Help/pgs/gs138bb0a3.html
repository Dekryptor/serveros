カスタム Web サイトへのアクセスを制限する

個々のカスタム Web サイト、またはサイトフォルダ内のフォルダへのアクセスを制限できます。単独のグループが、Web サイト全体、または Web サイトフォルダに含まれる個々のフォルダを参照できるように指定できます。アクセスが制限された Web サイトフォルダを、保護領域と呼びます。

Web サイトのアクセスを制御するには：

  1. 「サーバ環境設定」の「Web」パネルで「カスタムサイト」をクリックします。

  2. リストで Web サイトを選択し、フォルダの場所を確認して、編集（鉛筆）ボタンをクリックします。

  3. Web サイト全体にアクセスできる単独のグループを指定するには、「このサイトを表示できるユーザ」ポップアップメニューからグループを選択します。

  4. Web サイト内のフォルダにアクセスできる複数のグループを指定するには、「このサイトを表示できるユーザ」ポップアップメニューから「カスタマイズ」を選択します。

    既存のフォルダへのアクセスのみを制御できます。Finder を使って Web サイトフォルダ（手順 2 で確認した場所）にフォルダを追加できます。
