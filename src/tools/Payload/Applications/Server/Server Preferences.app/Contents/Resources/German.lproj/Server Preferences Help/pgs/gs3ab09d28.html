Ersetzen eines selbstsignierten Zertifikats

Nachdem Sie ein signiertes Zertifikat von einer Zertifizierungsstelle erhalten haben, können Sie Ihr selbstsigniertes Zertifikat dadurch ersetzen.

Gehen Sie wie folgt vor, um ein selbstsigniertes Zertifikat zu ersetzen:

  1. Wählen Sie im Finder die signierte Zertifikatsdatei aus, die Sie von einer Zertifizierungsstelle erhalten haben, und legen Sie diese so ab, dass Sie sie während der Verwendung des Programms "Servereinstellungen" sehen können, z. B. auf dem Schreibtisch.

  2. Öffnen Sie die Servereinstellung "Informationen" und klicken rechts neben "SSL-Zertifikat" auf die Taste "Bearbeiten".

  3. Wählen Sie aus dem Einblendmenü das selbstsignierte Zertifikat aus, das Sie ersetzen möchten.

    Die Option "SSL-Zertifikat verwenden" muss ausgewählt sein, damit das Einblendmenü verfügbar ist.

  4. Wählen Sie "Zertifikatssignierung" > "Durch signiertes oder erneutes Zertifikat ersetzen" aus dem Einblendmenü aus.

  5. Bewegen Sie die Datei mit dem signierten Zertifikat in die Mitte des Dialogfensters der Servereinstellungen und klicken Sie dann auf "Zertifikat ersetzen".
