Si vous ne pouvez pas modifier le nom abrégé d’un compte utilisateur

Une fois qu’un compte utilisateur est créé, vous ne pouvez plus modifier son nom abrégé. Si vous souhaitez modifier le nom abrégé, vous devez supprimer le compte et en créer un nouveau avec un nouveau nom abrégé. Pour obtenir des instructions, consultez Suppression d’un compte d’utilisateur et Ajout d’un compte utilisateur.

La suppression d’un compte utilisateur entraîne l’annulation de ses adhésions de groupes et met fin à ses accès aux services des groupes et aux wikis privés. Le courrier électronique de l’utilisateur stocké sur le serveur est également supprimé. Un compte utilisateur supprimé ne peut plus accéder aux calendriers ni au carnet d’adresses sur le serveur.

Une fois que vous avez créé un nouveau compte sur le serveur pour un utilisateur Mac OS X 10.6, ce dernier doit accéder à la section Options d’ouverture de session des préférences Comptes, supprimer le serveur de la liste des serveurs de répertoire, puis l’y ajouter à nouveau. Pour obtenir des informations sur la connexion à votre serveur, consultez Configuration d’un Mac à l’aide des préférences Comptes.
