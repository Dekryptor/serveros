ユーザの Mac をバックアップする十分な空き容量がサーバにない場合

ユーザの Mac をバックアップする十分な空き容量がサーバにない場合は、サーバに別のディスクを接続し、「サーバ環境設定」の「Time Machine」パネルでそのディスクを保存場所とします。操作方法については、「ユーザのバックアップの保管場所を管理する」を参照してください。

Snow Leopard ユーザが「システム環境設定」で「Time Machine」を有効にしていない場合は、サーバを保存場所とするかどうかを確認するメッセージが自動的に表示されます。そのほかのユーザは、「システム環境設定」の「Time Machine」パネルで、バックアップ用ディスクとしてサーバを選択する必要があります。
