Anpassen der Diensteinstellungen

Sie können Einstellungen für die unten aufgeführten Dienste ändern. Sie können auch Dienste deaktivieren, die nicht benötigt werden, da sie z. B. bereits von einem anderen Server bereitgestellt werden. Der Server muss z. B. den Dienst "Mail" nicht bereitstellen, wenn Sie den Dienst Ihres Internetanbieters oder Ihres Unternehmens verwenden wollen.

Dateifreigabe

Fügen Sie freigegebene Ordner hinzu und entfernen Sie sie. Ändern Sie die Zugriffsrechte für freigegebene Ordner.

Verwalten des Diensts "Dateifreigabe"
Adressbuch

Stellen Sie für jeden Benutzer ein persönliches Adressbuch oder für alle Benutzer ein gemeinsames Adressbuch bereit. Geben Sie bei Verwendung eines gemeinsamen Adressbuchs einen Namen dafür an.

Verwalten des Diensts "Adressbuch"
iCal

Begrenzen Sie die Größe jedes Kalenderereignisses und die Gesamtgröße der einzelnen Benutzerkalender.

Verwalten des Diensts "iCal"
iChat

Aktivieren Sie Instant-Messaging mit Benutzern anderer Server. Protokollieren und archivieren Sie alle Chats.

Verwalten des Diensts "iChat"
Mail

Konfigurieren Sie einen Relaisserver für ausgehende Mails. Passen Sie die Filterfunktion für Spam-Mail und Viren an. Geben Sie einen Server an, der dafür sorgt, dass E-Mails von Absendern auf schwarzen Listen abgelehnt werden.

Verwalten des Diensts "Mail"
Web

Wählen Sie die Startseite des Servers aus. Aktivieren oder deaktivieren Sie Webdienste - Wikis, Kalender, Blogs und Webmail. Konfigurieren Sie eigene Websites.

Verwalten der Webdienste
VPN

Erlauben Sie, dass entfernte Benutzer über das Internet einen sicheren Zugang zu Ihrem Netzwerk erhalten. Ändern Sie den Schlüssel (Shared Secret) und die IP-Adressen für entfernte Benutzer. Erstellen Sie eine Konfigurationsdatei, um den Fernzugriff für die Mac-Computer der Benutzer einzurichten.

Verwalten des Diensts "VPN"
