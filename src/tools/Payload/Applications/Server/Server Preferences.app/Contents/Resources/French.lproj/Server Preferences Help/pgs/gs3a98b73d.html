Configuration de la gestion des utilisateurs et des groupes

Si votre serveur n’était pas initialement configuré pour gérer ses propres utilisateurs et groupes, le bouton Configurer s’affiche quand vous ouvrez la sous-fenêtre Utilisateurs dans Préférences du serveur.

Vous pouvez configurer un répertoire propre à votre serveur en cliquant sur le bouton Configuration. Cela crée un domaine Open Directory sur votre serveur et fait de votre serveur un maître Open Directory. Pour en savoir plus sur Open Directory, ouvrez Admin Serveur puis utilisez le menu Aide pour rechercher la « Vue d’ensemble de Open Directory » ou consultez le site web Ressources de Mac OS X Server à l’adresse www.apple.com/fr/server/macosx/resources/(en anglais).
