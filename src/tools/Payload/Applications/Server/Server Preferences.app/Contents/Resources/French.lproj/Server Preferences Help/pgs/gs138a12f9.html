Configuration d’un Mac à l’aide des préférences Comptes

Si un Mac avec Snow Leopard n’est pas encore connecté à un serveur et si l’invitation à obtenir les services disponibles sur votre serveur n’apparaît pas automatiquement, l’utilisateur Mac peut se connecter manuellement à votre serveur. L’utilisateur doit se connecter manuellement si le nom ou le nom abrégé de son compte local sur le Mac ne correspond pas à un compte utilisateur sur le serveur, ou si le serveur se trouve sur un réseau local (sous-réseau IP) différent de celui du Mac. Le cas échéant, l’utilisateur doit également se connecter manuellement s’il a antérieurement refusé une invitation à utiliser votre serveur ou s’il a supprimé la connexion du Mac à votre serveur et doit la rétablir.

Pour connecter un Mac à votre serveur :

  1. Sur le Mac, ouvrez la sous-fenêtre Comptes des Préférences Système, puis cliquez sur Options d’ouverture de session.

  2. Cliquez sur le bouton Se connecter, ou cliquez sur le bouton Modifier puis sur le bouton Ajouter (+).

    Le bouton Se connecter s’affiche si le Mac n’est pas encore connecté à un serveur d’annuaire.

    Le bouton Modifier s’affiche si le Mac est déjà connecté à un serveur d’annuaire.

  3. Sélectionnez votre serveur dans le menu local ou saisissez le nom DNS ou l’adresse IP de votre serveur.

  4. Si une zone de dialogue apparaît vous proposant de configurer des services, choisissez si les applications de l’utilisateur doivent être configurées pour obtenir les services disponibles sur votre serveur.

    • Si vous souhaitez configurer les applications pour qu’elles obtiennent les services disponibles sur votre serveur, cliquez sur Configurer des services. Saisissez ensuite le nom et le mot de passe du compte de l’utilisateur sur le serveur.

      Pour obtenir des informations sur les modifications de comptes utilisateur et sur les services auxquels l’utilisateur peut accéder, consultez Configuration automatique des Mac des utilisateurs.

    • Si vous souhaitez seulement que le Mac soit connecté à votre domaine Open Directory, cliquez sur Se connecter uniquement. Les applications de l’utilisateur ne sont pas configurées et le compte local de l’utilisateur n’est pas lié au compte de l’utilisateur sur le serveur.

    La zone de dialogue proposant ces choix n’apparaît pas si aucun service n’est activé dans Préférences du serveur pour votre serveur ou si le serveur spécifié à l’étape 3 n’exécute pas Mac OS X Server 10.6 ou ultérieur.
