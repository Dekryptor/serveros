Qu’est-ce que Préférences du serveur ?

Grâce à Préférences du serveur, vous pouvez gérer divers aspects de Snow Leopard Server, notamment les personnes autorisées à utiliser les services, le type de configuration de ces services et leur état.

Lorsque vous ouvrez Préférences du serveur, les préférences individuelles sont regroupées dans les catégories décrites ci-dessous. Cliquez sur un lien pour trouver d’autres réglages et préférences dans chaque catégorie.
