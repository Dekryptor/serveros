Manuelles Senden einer E-Mail-Einladung

Sie können mit dem Programm "Servereinstellungen" eine E-Mail-Einladung senden, in der Sie den Benutzer über den Server und dessen Dienste informieren. Sofern möglich, sendet der Server automatisch eine Begrüßungsnachricht, wenn Sie einen neuen Benutzer-Account erstellen. Beim Importieren von Benutzer-Accounts haben Sie außerdem die Möglichkeit, eine Einladung zu senden. Sie können auch jederzeit manuell eine Einladung an einen Benutzer senden.

Gehen Sie wie folgt vor, um einem Benutzer per E-Mail eine Einladung zu senden:

  1. Öffnen Sie die Servereinstellung "Benutzer" und wählen Sie den Benutzer-Account aus, an den Sie die Einladung senden möchten.

  2. Klicken Sie auf die Taste des Aktionsmenüs (Zahnradsymbol) und wählen Sie "Einladung schicken an".

  3. Geben Sie die E-Mail-Adresse des Benutzers ein und klicken Sie auf "Einladung schicken".

Sie können Ihren Namen, die E-Mail-Adresse und eine persönliche Nachricht zur Standardeinladung hinzufügen. Anleitungen hierzu finden Sie unter Anpassen der Einladungsnachricht des Servers.
