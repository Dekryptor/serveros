手動でメール案内を送信する

「サーバ環境設定」を使って、サーバおよびサーバが提供するサービスに関する案内メールを送ることができます。ユーザアカウントを新規に作成するとサーバから自動的にウェルカムメッセージがメールで送信され、ユーザを読み込んだときに案内を送信するかどうかのオプションを選択できます。また、ユーザへの案内の送信は、いつでも手動で実行することができます。

ユーザにメール案内を送信するには：

  1. 「サーバ環境設定」の「ユーザ」パネルで、案内を送信するユーザアカウントを選択します。

  2. アクション（ギア）ボタンをクリックして、「__に案内を送信」を選択します。

  3. ユーザのメールアドレスを入力して、「案内を送信」をクリックします。

標準の案内メールには、名前、メールアドレス、および個人メッセージを追加できます。操作方法については、「サーバ案内メールをカスタマイズする」を参照してください。
