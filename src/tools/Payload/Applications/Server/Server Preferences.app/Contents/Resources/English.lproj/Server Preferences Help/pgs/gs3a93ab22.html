Customizing Service Settings

You can change settings for the services listed below. You can also turn off services that you don’t need, perhaps because another server provides them. For example, your server doesn’t need to provide mail service if you want to use mail service provided by your ISP or your organization.

File Sharing

Add and remove shared folders. Change permissions for shared folders.

Managing File Sharing Service
Address Book

Provide a personal address book for each user, or share a single address book among all users and specify the name of the shared address book.

Managing Address Book Service
iCal

Limit the size of each calendar event and each user’s total calendar size.

Managing iCal Service
iChat

Enable instant messaging with users of other servers. Log and archive all chats.

Managing iChat Service
Mail

Set up a relay server for outgoing mail. Adjust filtering of junk mail and viruses. Specify a server for rejecting email from blacklisted senders.

Managing Mail Service
Web

Choose the server’s home page. Turn web services—wikis, calendar, blogs, and webmail—on or off. Configure custom websites.

Managing Web Services
VPN

Allow remote users secure access to your network over the Internet. Change the shared secret and IP addresses for remote users. Create a configuration file to set up remote access on users’ Macs.

Managing VPN Service
