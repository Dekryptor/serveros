iChat サービスを管理する

「iChat」パネルを使用して、iChat インスタント・メッセージ・サービスを有効または無効にしたり、ほかのインスタント・メッセージ・システムでチャットを有効にしたり、すべてのチャットのログとアーカイブを設定したりすることもできます。

「iChat」パネルを構成するには：

  1. 次の情報を参考にして、必要に応じて設定を変更します：

    • 「入」／「切」： 
      iChat サービスの状態を示します。「入」／「切」スイッチをクリックすると、サービスの有効／無効が切り替わります。
    • サーバ間通信を有効にする： 
      ユーザが、「Google トーク」などの、ほかの XMPP インスタント・メッセージ・システムのユーザとチャットできるようにします。操作方法については、「Google トークおよびその他の XMPP サービスのユーザとチャットする」を参照してください。
    • すべてのチャットをログに記録してアーカイブ： 
      すべてのチャット（インスタントメッセージ）の記録をファイルに保存し、その記録を週に 1 回、圧縮ファイルにアーカイブします。 毎週の記録と圧縮されたアーカイブは「/var/jabberd/message_archives/」に保存されます。操作方法については、「インスタントメッセージを保存してアーカイブする」を参照してください。

  2. iChat サービスを再起動するときは、「入」／「切」を 2 回クリックします。

    iChat サービスの設定変更は、iChat サービスが再起動されるまでは有効になりません。
