グループのサービスを変更する

グループサービスの設定を行うには、「サーバ環境設定」の「グループ」パネルにある「グループ」をクリックします。グループサービスには、共有フォルダや iChat 自動メンバーリストがあります。グループ名を使って Wiki を作成したり、既存の Wiki に移動したりもできます。

サービスの設定を変更するには：

  1. 「サーバ環境設定」の「グループ」パネルで左側のリストから、変更するグループを選択します。

  2. 「グループ」をクリックして、グループに割り当てるサービスを選択し、除外するサービスの選択を解除します。

    • ファイル共有フォルダ： 
      グループにファイル共有フォルダを提供します。メンバーは、グループの共有フォルダからファイルを取得したり、ファイルをこのフォルダに置いたりできます。このフォルダはグループのユーザ名を元に名前が付けられ、サーバの起動ディスクの「/グループ/」フォルダに置かれます。「ファイル共有フォルダ」の隣にあるリンクの矢印をクリックすると、そのグループの共有フォルダが Finder で開きます。
    • iChat 自動メンバーリスト： 
      すべてのグループメンバーの Jabber ID（スクリーン名）を、自動的に各メンバーの Jabber メンバーリストに追加します。グループに外部メンバーが存在する場合、そのメンバーは iChat 自動メンバーリストには含まれません。
    • グループ Wiki を作成： 
      このボタンが表示された場合、ボタンをクリックするとお使いの Web ブラウザが開き、このグループ名を使って Wiki を作成するプロセスが開始します。 Wiki を作成する際、視覚テーマを選択し、アクセス権を設定します。Wiki には、デフォルトで Web カレンダー、ブログ、およびメーリングリストが含まれています。作成した Wiki を後からカスタマイズできます。詳細については、任意の Wiki ページの「ヘルプ」リンクをクリックしてください。
    • “__”Wiki へ移動： 
      このリンクが表示された場合、このグループの名前を使って Wiki が作成されています。リンクをクリックすると、お使いのブラウザで Wiki が開きます。

共有フォルダまたはそのグループに使用権限のあるプライベート Wiki にアクセスするために、グループメンバーは自分のユーザアカウント名とパスワードを使用して認証される必要があります。グループサービスが使用可能かどうかは、ファイル共有サービス、iChat サービス、および Web サービスが有効かどうかによります。
