Applications et utilitaires de base

Après avoir configuré Snow Leopard Server, vous pouvez gérer des utilisateurs et des groupes, modifier des réglages essentiels de services et effectuer d’autres tâches d’administration de serveur de base à l’aide des applications et des utilitaires décrits ci-dessous.

Important :  si certains de vos utilitaires ou de vos applications proviennent de Mac OS X Server 10.5 ou antérieur, ne les utilisez pas avec Snow Leopard Server.
Applications et utilitaires pour les administrateurs de serveurs

Utilitaire iCal Server (dans /Applications/Server/)

Créez, modifiez et supprimez les emplacements et les ressources que les utilisateurs peuvent réserver lorsqu’ils programment des réunions dans les calendriers partagés hébergés par iCal Server.

Assistant du serveur

Configurez Mac OS X Server pour la première fois. Installez Mac OS X Server sur un ordinateur distant.

Préférences du serveur (dans /Applications/Server/)

Permet de gérer les utilisateurs et les groupes, de personnaliser les services et les informations système et, enfin, de surveiller l’activité du serveur.

Widget d’état du serveur pour Dashboard

Surveillez l’activité d’un serveur depuis n’importe quel Mac avec Snow Leopard.

Préférences Système (dans /Applications/)

Connecte votre serveur à un serveur de répertoire de votre organisation. Configurez la sauvegarde Time Machine du serveur. Configurez le partage d’une imprimante USB ou FireWire directement connectée.
