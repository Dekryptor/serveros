Configuration de la page d’accueil du serveur
Pour choisir la page d’accueil du serveur :

Dans la sous-fenêtre Web, cliquez sur Services web et sélectionnez-la à partir du menu local Page d’accueil.

Si le menu local est estompé, le service Wikis est désactivé.

Si le menu local ne comporte aucun wiki, vous devez en créer un ou charger quelqu’un disposant d’un compte utilisateur sur le serveur de le faire. Pour obtenir des instructions, cliquez sur Aide dans n’importe quelle page wiki.
