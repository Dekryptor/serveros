Changing a User’s Name or Password

You can use Server Preferences to change the name or password for a standard user account or an administrator account.

To change a user account’s name or password:

  1. In the Users pane of Server Preferences, select a standard or administrator user account you want to change in the list on the left, and then click Account.

  2. If you want to change the name, edit the Name field.

    The name can be up to 255 characters (which can be as few as 85 Japanese characters). It can include spaces.

    You can’t change a user account’s short name using Server Preferences.

  3. If you want to change the password, click Reset Password, enter the new password in the New Password and Verify fields, and then click Change Password.

    You can use Password Assistant to help you choose a new password. Click the Key button to the right of the New Password field to see how secure the new password is.

Users can change their passwords in the Accounts pane of System Preferences on their computers.

If your server has imported user accounts, or user accounts imported automatically from groups, their names or passwords can be changed by an administrator of the directory server where the accounts are stored. If that directory server has Mac OS X Server v10.6 and you are an administrator of it, you may be able to connect Server Preferences to it remotely and then make the changes.

To change the password or other attributes of the primary administrator account that was created during initial server setup, use the Accounts pane of System Preferences on the server. For information about changing local accounts, open System Preferences and then use the Help menu.

To change the password or other attributes of the directory administrator account that was created during initial server setup, use the Workgroup Manager application (located in /Applications//Server/). For instructions, open Workgroup Manager and then use the help menu.

For information about changing the password of the System Administrator (root) account, see the Getting Started manual.
