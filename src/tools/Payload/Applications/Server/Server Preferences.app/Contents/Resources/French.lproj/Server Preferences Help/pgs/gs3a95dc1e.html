Importation d’utilisateurs

Si votre serveur est connecté au serveur d’annuaire de votre organisation, vous pouvez importer des comptes d’utilisateur existants.

Votre serveur attribue à chaque compte utilisateur importé sa propre adresse iChat, un calendrier personnel et un portail de wiki Ma page. Les comptes utilisateur importés peuvent également accéder à des wikis, des blogs et des calendriers web, à un carnet d’adresses basé sur le serveur, aux fichiers partagés du serveur et au stockage de sauvegarde Time Machine. Si le service de messagerie et le service VPN de votre serveur sont activés, les comptes utilisateur importés reçoivent une adresse électronique et peuvent utiliser le service VPN pour accéder à distance au serveur. Pour être disponible, chaque service doit être activé ; vous pouvez contrôler séparément l’accès de chaque compte utilisateur importé aux différents services.

Pour importer un compte utilisateur :

  1. Si vous souhaitez que le serveur envoie un message électronique d’invitation aux utilisateurs importés, vérifiez que l’introduction personnalisée, le nom et l’adresse électronique de l’expéditeur correspondent à vos besoins.

    Pour obtenir des informations, consultez Personnalisation du message électronique d’invitation du serveur.

  2. Dans la sous-fenêtre Utilisateurs de Préférences du serveur, cliquez sur le bouton Ajouter (+) et choisissez « Importer l’utilisateur du répertoire » dans le menu local.

    Si aucun menu local n’apparaît lorsque vous cliquez sur le bouton Ajouter (+), votre serveur n’est pas connecté à un serveur de répertoire de votre organisation. Consultez Connexion de votre serveur à un serveur d’annuaire.

    Si votre organisation ne dispose pas d’un serveur de répertoire (votre serveur excepté), vous pouvez créer des comptes utilisateur. Pour obtenir des instructions, consultez Ajout d’un compte utilisateur.

  3. Saisissez une partie du nom de l’utilisateur ou son nom complet dans le champ de recherche ; lorsque le nom apparaît, sélectionnez-le. Si vous le souhaitez, sélectionnez « Inviter par courrier les nouveaux utilisateurs importés », puis cliquez sur Importer.

  4. Une fois l’importation des comptes utilisateur terminée, cliquez sur Terminé.

Les comptes utilisateur importés via cette procédure apparaissent dans la liste Importés de la sous-fenêtre Utilisateurs.

Vous pouvez importer automatiquement des comptes utilisateur à partir de groupes au lieu de les importer un par un via cette procédure. Pour obtenir des instructions, consultez Importation automatique de groupes d’utilisateurs.
