Ändern der Einstellungen eigener Websites

Sie können die IP-Adresse einer eigenen Website, den Zugangsport, den Speicherort des Website-Ordners und die Zugriffsrechte ändern.

Gehen Sie wie folgt vor, um Website-Einstellungen zu ändern:

  1. Öffnen Sie die Servereinstellung "Web" und klicken Sie auf "Eigene Sites".

  2. Wählen Sie die Website aus, die Sie ändern möchten. Klicken Sie dann auf die Taste "Bearbeiten" (das Stiftsymbol) und ändern Sie Einstellungen unter Berücksichtigung der folgenden Informationen:

    • IP-Adresse: 
      Besitzt der Server mehrere IP-Adressen, wählen Sie eine davon aus oder wählen Sie "Jeder", damit Benutzer mit jeder der IP-Adressen auf die Website zugreifen können.
    • Port: 
      Geben Sie eine Nummer für den Zugangsport ein. Der standardmäßige Zugangsport für Websites ist Port 80. Vergewissern Sie sich vor der Eingabe eines anderen Ports, dass dadurch kein Konflikt mit einem anderen von Ihrem Server bereitgestellten Dienst entsteht. Eine Liste der Ports, die Mac OS X Server verwenden kann, finden Sie im Menü "Hilfe" des Programms "Server-Admin" (im Ordner "Programme/Server"), indem Sie nach "TCP- und UDP-Port-Referenz" suchen.
    • Site-Dateien sichern in: 
      Wählen Sie den Ordner aus, der die Dateien enthält, die Sie für die Website verwenden möchten.
    • Benutzer, die diese Site sehen können: 
      Wählen Sie eine Gruppe aus, um nur deren Mitgliedern Zugriff auf die gesamte Website zu gewähren. Wählen Sie "Anpassen", um den Zugriff auf einzelne Ordner innerhalb des Website-Ordners zu beschränken. Anleitungen hierzu finden Sie unter Beschränken des Zugriffs auf eine eigene Website.
