À propos du service de partage de fichiers

Le service de partage de fichiers de Mac OS X Server permet aux utilisateurs d’accéder aux dossiers partagés et de stocker des fichiers sur le serveur. Ceux-ci peuvent utiliser des ordinateurs Macintosh, Windows ou UNIX pour accéder à leurs fichiers et dossiers partagés sans logiciel spécifique et ce, à l’aide de protocoles de fichiers natifs, tels que les protocoles AFP et SMB. Les utilisateurs de Windows voient les serveurs de fichiers Mac OS X Server dans leurs Favoris réseau, de la même manière que les serveurs de fichiers Windows.

Si vous souhaitez autoriser l’accès au partage de fichiers sur Internet et si vous possédez un routeur câble, un routeur DSL ou tout autre routeur, celui-ci doit être configuré pour la redirection de ports (mappage de ports) utilisés par le partage de fichiers. Pour en savoir plus sur la redirection de port, consultez le manuel Premiers contacts.

Si vous souhaitez autoriser l’accès au partage de fichiers à l’extérieur de votre réseau local et si celui-ci dispose d’un coupe-feu séparé, demandez à l’administrateur du coupe-feu d’ouvrir celui-ci aux ports utilisés par le partage de fichiers. Pour accéder à une liste des ports, consultez l’annexe dans le manuel Premiers contacts.
