Web サービスを開始する／停止する

「Web」パネルで Web サービスを開始または停止できます。Web サービスが無効の間は、ユーザは、その Web メールと「マイページ」、自分のグループの Wiki、ブログ、カレンダー、メーリング・リスト・アーカイブのいずれにもアクセスできません。

Web サービスを開始または停止するには：

「サーバ環境設定」の「Web」パネルで「入」／「切」スイッチをクリックします。

Web サービスを開始した後、特にこれらのサービスをはじめて開始した場合は、「ユーザのコンピュータを手動で設定する」で説明しているアドレスで Web サービスにアクセスできることをユーザに通知してください。
