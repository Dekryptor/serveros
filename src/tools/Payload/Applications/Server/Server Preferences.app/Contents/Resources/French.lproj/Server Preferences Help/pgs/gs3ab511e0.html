Informations complémentaires
Pour en savoir plus, consultez les ressources suivantes

Documents Ouvrez-moi (sur le disque d’installation de Mac OS X Server et le disque Administration Tools)

Obtenez des mises à jour importantes et des informations spéciales.

Site web de Mac OS X Server (www.apple.com/fr/server/macosx, en anglais)

Découvrez, au moyen de cette passerelle, des informations approfondies sur les produits et les technologies.

Site web d’assistance Mac OS X Server (www.apple.com/fr/support/macosxserver, en anglais)

Accédez à des centaines d’articles du service d’assistance Apple.

Site web de formation et de certification Apple (www.apple.com/fr/training/)

Perfectionnez vos compétences d’administration de serveurs en suivant des formations avec enseignant ou des autoformations et différentiez-vous sur le marché grâce aux certifications Apple.

Site web de discussions Apple (discussions.apple.com, en anglais)

Partagez des questions, des connaissances et des conseils avec d’autres administrateurs.

Site web des listes d’envoi d’Apple (www.lists.apple.com/fr, en anglais)

Abonnez-vous aux listes d’envoi pour discuter par courrier électronique avec d’autres administrateurs.
