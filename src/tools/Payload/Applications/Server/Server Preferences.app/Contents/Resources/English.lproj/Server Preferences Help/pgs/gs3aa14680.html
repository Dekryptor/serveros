Managing Groups

In the Groups pane, you create groups, turn group services on or off, add and remove group members, and delete unneeded groups.

Options
Add (+) button

Lets you create a new group. For instructions, see Creating a New Group.

Delete (–) button

Lets you delete a group. For instructions, see Deleting a Group.

Group

Lets you change the group name, see the group’s short name, and set up group services. Group services include a file sharing folder and automatic iChat buddies. For instructions, see Changing a Group’s Name and Changing Group Services.

Members

Lets you add or remove group members that are standard, imported, or automatic users in the Users pane. (To have imported or automatic users, your server must be connected to your organization’s directory server.) For instructions, see Adding or Removing Members of a Group.

External Members

Lets you add or remove group members that have user or group accounts on your organization’s directory server. External members of a group can access the group’s file sharing folder and wikis that the group has permission to use. For instructions, see Adding or Removing External Members of a Group.

If you don’t see an External Members tab, your server is not connected to a directory server in your organization. If your organization has a directory server, you can connect to it. For instructions, see Connecting Your Server to a Directory Server.

Set Up

If you see a Set Up button instead of the options above, your server doesn’t have its own directory domain for users and groups. Clicking this button sets one up. For instructions, see Setting Up Users and Groups Management.
