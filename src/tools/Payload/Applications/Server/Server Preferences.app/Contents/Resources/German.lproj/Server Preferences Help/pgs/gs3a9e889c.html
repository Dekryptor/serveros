Ändern des Namens einer Gruppe

In der Servereinstellung "Gruppen" können Sie den Namen einer Gruppe ändern.

Gehen Sie wie folgt vor, um den Namen einer Gruppe zu ändern:

  1. Öffnen Sie die Servereinstellung "Gruppen" und wählen Sie in der Liste links die Gruppe aus, die Sie umbenennen wollen. Klicken Sie danach auf "Gruppe".

  2. Bearbeiten Sie den Inhalt des Felds "Gruppenname".

    Der Name kann bis zu 255 Zeichen lang sein (was u. U. nur 85 japanischen Zeichen entspricht). Der Name darf Leerzeichen enthalten.

    Der Kurzname einer Gruppe kann mit dem Programm "Servereinstellungen" nicht geändert werden.
