Sécurisation de votre serveur

Par sécurité, créez un compte utilisateur standard une fois la configuration du serveur terminée. Lorsque vous ouvrez une session sur le serveur, prenez l’habitude d’utiliser ce compte standard plutôt qu’un compte administrateur. Réservez ensuite l’utilisation du compte administrateur aux applications qui exigent des autorisations administrateur. Par exemple, utilisez votre nom et mot de passe d’administrateur avec Préférences du serveur lorsque vous devez gérer des utilisateurs, des groupes ou des services.

Pour créer un compte utilisateur standard, utilisez la sous-fenêtre Comptes des Préférences Système sur le serveur. Pour obtenir des informations, ouvrez les Préférences Système, puis utilisez le menu Aide.
