Das Programm "Servereinstellungen"

Mit dem Programm "Servereinstellungen" können Sie verschiedene Verwaltungstätigkeiten für Snow Leopard Server ausführen und u. a. festlegen, welche Benutzer die bereitgestellten Dienste nutzen können, wie die Dienste konfiguriert werden und welchen Status sie haben.

Im Fenster "Servereinstellungen" sind die Optionen und Einstellungen in die unten beschriebenen Kategorien eingeteilt. Klicken Sie auf einen Link, um Näheres über die Einstellungen der einzelnen Kategorien zu erfahren.
