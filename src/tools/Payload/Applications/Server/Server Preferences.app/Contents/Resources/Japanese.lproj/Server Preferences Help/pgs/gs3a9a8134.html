ホーム
ユーザを管理する

「ユーザ」パネルで、このサーバにより提供されるサービスを使用するユーザのアカウントを設定し、それらのユーザがどのサービスにアクセスできるかを制御します。ユーザの連絡先情報をアップデートしたり、グループのメンバーシップを変更したりできます。また、ウェルカムメッセージのメールを編集することもできます。編集したメッセージは新しいユーザに配信されます。

オプション
追加（＋）ボタン

新しいユーザアカウントを作成したり、組織のディレクトリサーバから既存のユーザアカウントを読み込んだりできます。（読み込みができるのは、サーバが組織内の別のディレクトリサーバに接続されている場合に限ります。）操作方法については、「ユーザアカウントを追加する」を参照してください。

削除（－）ボタン

選択したユーザアカウントを削除できます。操作方法については、「ユーザアカウントを削除する」を参照してください。

アクション（ギア）ボタン

グループからのユーザの自動読み込みを管理したり、新規ユーザを歓迎するために送信するメールメッセージを管理したり、新しく読み込んだユーザにサーバを使用するように案内したり、グループの新しい外部メンバーにサービスを使用するように案内したりできます。操作方法については、「ウェルカムメッセージと案内メールをカスタマイズする」および「ユーザのグループを自動的に読み込む」を参照してください。

アカウント

ユーザの名前、パスワード、ピクチャ、または管理者権限を変更できます。操作方法については、「ユーザのアカウント設定を変更する」を参照してください。

コンタクト情報

ユーザの氏名、住所、メールとチャットのアドレス、Web サイトのアドレス、およびブログのアドレスを変更できます。操作方法については、「ユーザの連絡先を変更する」を参照してください。

サービス

個々のサービスへのユーザのアクセスを制御できます。操作方法については、「ユーザによるサービスへのアクセスを制御する」を参照してください。

グループ

ユーザをグループに追加したり、グループからユーザを取り除いたりできます。操作方法については、「ユーザのグループメンバーシップを変更する」を参照してください。

設定

上記のオプションの代わりに「設定」ボタンが表示されている場合、お使いのサーバにはユーザおよびグループ用の独自のディレクトリがありません。 このボタンをクリックすると設定ができます。操作方法については、「ユーザおよびグループの管理を設定する」を参照してください。

ユーザを読み込んだ場合は、サーバが接続しているディレクトリサーバに設定が保管されているため、ユーザの設定項目の一部が淡色表示されます。
