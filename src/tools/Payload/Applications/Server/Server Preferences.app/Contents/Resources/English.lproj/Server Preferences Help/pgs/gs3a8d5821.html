What Is Mac OS X Server?

Mac OS X Server version 10.6 Snow Leopard combines intuitively simple Macintosh ease of use with a mature, stable UNIX foundation. It provides an extensive array of services that support Macintosh, Windows, and UNIX client computers over a network.

With Snow Leopard Server, small organizations and workgroups without an IT department can take full advantage of the benefits of a server. Even a nontechnical user can set up and manage Snow Leopard Server for a group. Other users in the group can automatically configure their Macs to get services from Snow Leopard Server. Snow Leopard Server has advanced configuration options and management tools for IT professionals as well.

Users and Groups

Each person who wants to get services from your server needs a user account. A user account enables a person to authenticate for services, controls which services the person can use, contains the person’s contact information, and specifies the person’s group membership.

When you set up your server for the first time, you specify where your server gets user and group accounts. Your server may have its own users and groups that you create using Server Preferences. Your server may be configured to allow importing users from a directory server in your organization. Instead of having its own users or importing them, your server may be configured to access users and groups residing on your organization’s directory server.

You can also create groups, set up group services, and change group membership.

Services and System Information

Your server offers users a variety of services such as file sharing, address book, calendar, instant messaging, mail, web, and VPN. You can customize service settings and turn services off or on. You can check system information, logs, and graphs. You can also change system information, firewall settings, and backup settings for users’ computers.
