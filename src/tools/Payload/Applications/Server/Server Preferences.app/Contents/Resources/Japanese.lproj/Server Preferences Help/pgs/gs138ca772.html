ユーザがインターネット経由でサービスにアクセスできない場合

ユーザが、ローカルネットワークのサービスにアクセスできてもインターネット経由ではアクセスできない場合は、以下の方法を試してください：

  • 「サーバ環境設定」の「セキュリティ」パネルに「～に表示するサービス」というメッセージが含まれている場合は、目的のサービスが一覧表示されていることを確認します。

  • 「サーバ環境設定」の「セキュリティ」パネルに「AirMac マネージメントは現在“切”です」というメッセージが含まれている場合は、それを入にして、目的のサービスが一覧表示されていることを確認します。

  • サーバのインターネット接続が AirMac ベースステーションまたはほかのインターネットルーターを経由している場合は、サーバのサービスをインターネット上で使用できるようにルーターを構成する必要があります。AirMac Extreme ベースステーション（802.11n）または Time Capsule がある場合、それらは Mac OS X Server で自動的に構成されます。そのほかのモデルの AirMac ベースステーション、またはインターネットルーターについては、「お使いになる前に」マニュアルの第 2 章「Mac OS X Server の準備をする」を参照してください。
