Configuration des Mac des nouveaux utilisateurs importés

Si Mac OS X 10.6 est déjà installé sur les ordinateurs de certains utilisateurs et si vous importez leurs comptes utilisateur à partir d’un serveur d’annuaire, votre serveur peut leur envoyer un message électronique les invitant à se connecter au serveur. Les utilisateurs Snow Leopard peuvent cliquer sur un bouton situé dans le message électronique d’invitation afin que leurs applications se configurent automatiquement pour accéder aux services disponibles sur votre serveur.

Remarque :  pour recevoir un message électronique d’invitation, un utilisateur importé doit avoir une adresse électronique dans son compte sur le serveur d’annuaire.

Si un utilisateur Snow Leopard clique sur le bouton présent dans le message électronique d’invitation, la sous-fenêtre Comptes des Préférences Système s’ouvre. Elle invite l’utilisateur à saisir le nom et le mot de passe du compte utilisateur sur votre serveur.

Pour obtenir des informations sur les modifications de comptes utilisateur et sur les services auxquels l’utilisateur peut accéder, consultez Configuration automatique des Mac des utilisateurs.

Si un utilisateur Snow Leopard n’a pas reçu de message électronique d’invitation, vous pouvez lui en envoyer un manuellement au moyen du bouton Action dans la sous-fenêtre Utilisateurs. Pour obtenir des instructions, ouvrez Préférences du serveur, puis utilisez le menu Aide.

Pour savoir comment ajouter votre nom, votre adresse électronique et une introduction personnelle au message standard généré par le serveur pour le courrier électronique d’invitation, voir Personnalisation du message électronique d’invitation du serveur.
