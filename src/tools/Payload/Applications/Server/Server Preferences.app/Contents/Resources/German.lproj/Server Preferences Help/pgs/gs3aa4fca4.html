Verwalten des Diensts "iChat"

In der Servereinstellung "iChat" können Sie den Chatdienst "iChat" aktivieren und deaktivieren, das Chatten mit anderen Instant-Messaging-Systemen ermöglichen und die Einstellungen für das Protokollieren und Archivieren der Chatdaten festlegen.

Gehen Sie wie folgt vor, um die Einstellungen der Servereinstellung "iChat" zu konfigurieren:

  1. Ändern Sie die Einstellungen wie erforderlich und beachten Sie dabei die folgenden Informationen.

    • Ein/Aus: 
      Gibt den Status des Diensts "iChat" an. Durch Klicken auf die Taste "Ein/Aus" wird der Dienst aktiviert oder deaktiviert.
    • Server-zu-Server-Kommunikation aktivieren: 
      Ermöglicht Benutzern das Chatten mit Benutzern anderer XMPP-Instant-Messaging-Systeme wie Google Talk. Anleitungen hierzu finden Sie unter Kommunizieren mit Benutzern von Google Talk und anderen XMPP-Diensten.
    • Alle Chats protokollieren und archivieren: 
      Sichert ein Transskript aller Chats in einer Datei und archiviert dieses Transskript einmal wöchentlich in einer komprimierten Datei. Das wöchentliche Transkript und die komprimierten Archivdaten werden im Ordner "/var/jabberd/message_archives/" gesichert. Anleitungen hierzu finden Sie unter Speichern und Archivieren von Instant-Messaging-Nachrichten.

  2. Starten Sie den Dienst "iChat" erneut, indem Sie zweimal auf "Ein/Aus" klicken.

    Änderungen an den Einstellungen für den Dienst "iChat" werden erst nach dem Neustart des Diensts wirksam.
