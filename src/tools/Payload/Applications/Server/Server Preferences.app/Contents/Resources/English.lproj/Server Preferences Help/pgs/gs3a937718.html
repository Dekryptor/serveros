Changing Your Server’s Settings

With Server Preferences, you can customize settings for services your server provides, manage users and groups, and view server status. You can use Server Preferences on the server you want to manage, or use it remotely from an administrator computer.

To change server settings:

Open Server Preferences and click the icon for the settings you want to change.

If you’re not sure where a setting is, use the search box to find it. For instructions, see Finding Settings in Server Preferences.

To change settings for the server itself, such as the server’s Network preferences or its Time Machine preferences, open the System Preferences application on the server. For information about using System Preferences, open it and then use the Help menu.
