Hinzufügen und Entfernen von Gruppenmitgliedern

In der Servereinstellung "Gruppen" können Benutzer, die in der Servereinstellung "Benutzer" erstellt oder importiert wurden, als Mitglieder zu Gruppen hinzugefügt oder aus Gruppen entfernt werden. (Für importierte Benutzer muss Ihr Server mit einem Verzeichnisserver verbunden sein.)

Gehen Sie wie folgt vor, um einer Gruppe Mitglieder hinzuzufügen oder Mitglieder zu löschen:

  1. Öffnen Sie die Servereinstellung "Gruppen" und wählen Sie in der Liste links die Gruppe aus, die bearbeitet werden soll.

  2. Klicken Sie auf "Mitglieder" und danach auf "Mitgliedschaft bearbeiten".

  3. Markieren Sie das Feld jedes Benutzers, der Mitglied der Gruppe sein soll. Entfernen Sie die Markierungen aus den Feldern der Benutzer, die nicht Mitglied der Gruppe sein sollen.

  4. Klicken Sie, nachdem Sie die Änderungen abgeschlossen haben, erneut auf "Mitgliedschaft bearbeiten", um eine statische Liste der Gruppenmitglieder anzuzeigen.
