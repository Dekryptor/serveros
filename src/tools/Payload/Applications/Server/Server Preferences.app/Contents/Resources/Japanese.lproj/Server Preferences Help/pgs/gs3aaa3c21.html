ユーザブログを設定する
ユーザブログを有効または無効にするには：

「Web」パネルで、「Web サービス」をクリックして「ブログ」を選択または選択解除します。

このオプションが選択されていて Web サービスが有効になっている場合は、その隣の矢印のリンクをクリックすると、すべてのブログがリストで表示されます。

ユーザが自分のブログにアクセスするには、「/users/」をサーバの Web サイトアドレスの末尾に付加します。たとえば、次のように入力します：

http://server.example.com/users/

ブログの使用方法については、任意の Wiki またはブログのページの「ヘルプ」をクリックしてください。
