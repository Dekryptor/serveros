メールサービスを管理する

「メール」パネルを使って、メールサービスを有効または無効にしたり、送信メール用リレーサーバを指定したり、ジャンクメールとウイルスのフィルタリングを調整したりできます。

「Mail」パネルを構成するには：

次の情報を参考にして、必要に応じて設定を変更します：

  • 「入」／「切」： 
    メールサービスの状態を示します。「入」／「切」スイッチをクリックすると、サービスの有効／無効が切り替わります。
  • ISP を経由して送信メールをリレー： 
    このオプションは、お使いの ISP により特定のサーバ経由での送信メールのリレーが要求される場合にのみ選択します。「編集」をクリックして、すでに設定済みのリレーサーバのアドレスまたは認証を変更します。
メール・リレー・サーバを指定する
  • ブラックリストにあるサーバからのメールを拒否： 
    DNSBL サーバ（zen.spamhaus.org）によって迷惑メール送信者の可能性ありとしてリストに挙げられた IP アドレスを使用している送信者からのメールの受信を拒否します。 別の DNSBL サーバを指定するには、「編集」をクリックします。
ブラックリストに記載された送信者からのメールを拒否する
  • 迷惑メールとウイルスのフィルタを有効にする： 
    受信メールをスキャンして、既知のウイルスが含まれているものを削除し、スライダで設定された迷惑メールの確率を超えるものをマークします。
