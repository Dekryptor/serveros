Gestion des utilisateurs

La sous-fenêtre Utilisateurs vous permet de configurer des comptes pour les personnes qui utilisent les services fournis par ce serveur et de contrôler les services auxquels elles ont accès. Vous pouvez mettre à jour leurs informations de contact et changer leur appartenance à des groupes. Il est également possible de gérer les messages électroniques de bienvenue à envoyer aux nouveaux utilisateurs.

Options
Bouton Ajouter (+)

Vous permet de créer un nouveau compte utilisateur ou d’importer un compte utilisateur existant à partir du serveur d’annuaire de votre organisation. (vous ne pouvez effectuer une importation que si votre serveur est connecté à un autre serveur d’annuaire de votre organisation). Pour obtenir des instructions, consultez Ajout d’un compte utilisateur.

Bouton Supprimer (–)

Vous permet de supprimer le compte utilisateur sélectionné. Pour obtenir des instructions, consultez Suppression d’un compte d’utilisateur.

Bouton Action (engrenage)

Vous permet de gérer l’importation automatique d’utilisateurs provenant de groupes, de gérer les messages électroniques envoyés pour accueillir les nouveaux utilisateurs, d’inviter les utilisateurs qui viennent d’être importés à utiliser le serveur et d’inviter de nouveaux membres extérieurs à un groupe à employer ses services. Pour obtenir des instructions, consultez Personnalisation d’un message électronique de bienvenue ou d’invitation et Importation automatique de groupes d’utilisateurs.

Compte

Vous permet de modifier le nom, le mot de passe, l’image ou le privilège administrateur d’un utilisateur. Pour obtenir des instructions, consultez Modification des réglages d’un compte utilisateur.

Coord. des contacts

Vous permet de modifier le nom et le prénom d’un utilisateur, son adresse, ses adresses électronique et de messagerie instantanée, l’adresse de son site web et l’adresse de son blog Pour obtenir des instructions, consultez Modification des coordonnées d’un utilisateur.

services

Vous permet de contrôler l’accès d’un utilisateur à chaque service. Pour obtenir des instructions, consultez Contrôle de l’accès d’un utilisateur aux différents services.

Groupes

Vous permet d’ajouter un utilisateur à un groupe ou de supprimer un utilisateur d’un groupe. Pour obtenir des instructions, consultez Modification de l’appartenance d’un utilisateur à des groupes.

Configurer

Si le bouton Configurer apparaît au lieu des options ci-dessus, votre serveur ne dispose pas de son propre domaine de répertoire pour les utilisateurs et les groupes. Pour en définir un, cliquez sur ce bouton. Pour obtenir des instructions, consultez Configuration de la gestion des utilisateurs et des groupes.

Si vous avez importé des utilisateurs, certains de leurs réglages sont estompés car ils sont stockés sur le serveur d’annuaire auquel votre serveur est connecté.
