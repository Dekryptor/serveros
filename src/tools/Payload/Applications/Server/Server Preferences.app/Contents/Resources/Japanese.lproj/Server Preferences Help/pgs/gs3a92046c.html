サーバシステムを管理する

一般情報の取得、SSL 証明書の管理、サービスログの確認、サーバ利用状況のグラフの表示、ユーザの Time Machine バックアップの保管場所の設定、およびセキュリティ設定の変更を行います。

情報

サーバのシステムのバージョン、プロセッサ、メモリ、ネットワークのアドレスと名前、およびシリアル番号に関する情報を取得します。サーバのコンピュータ名とシリアル番号を変更します。SSL 証明書を管理します。

サーバ情報を管理する
ログ

システム動作のログと、アドレスブック、ファイル共有、iChat、iCal、メール、Web、VPN、ディレクトリサービスなどのさまざまなサービスのログを表示します。

サーバログを確認する
グラフ

プロセッサ使用率、全体的なネットワークトラフィック、空きディスク領域、ファイル共有トラフィック、および Web トラフィックを示すグラフを表示します。

サーバグラフを監視する
サーバ状況ウィジェットを使用する
Time Machine

サーバのバックアップを作成するため、またはユーザのコンピュータのバックアップを作成するための、「Time Machine」環境設定を指定します。

サーバのバックアップと復元を行う
ユーザのバックアップの保管場所を管理する
セキュリティ

AirMac マネージメントまたはサーバのファイアウォールを使って、ローカルネットワークの外部からのサービスへのアクセスを制御します。

セキュリティの設定を変更する
