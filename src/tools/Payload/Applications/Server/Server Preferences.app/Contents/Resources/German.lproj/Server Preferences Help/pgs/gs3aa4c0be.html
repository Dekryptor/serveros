Starten oder Stoppen des Diensts "iCal"

Sie können den Kalenderdienst "iCal" in der Servereinstellung "iCal" stoppen oder starten. Wenn der Dienst "iCal" deaktiviert ist, haben Benutzer keinen Zugriff auf freigegebene Kalender.

Gehen Sie wie folgt vor, um den Dienst "iCal" zu starten oder zu stoppen:

Klicken Sie auf die Taste "Ein/Aus" in der Servereinstellung "iCal".

Nach dem Starten des Diensts "iCal", und besonders nach dem erstmaligen Starten, sollten Sie die Benutzer darauf hinzweisen, ihr Kalenderprogramm einzurichten. Hinweise dazu finden Sie unter Manuelles Konfigurieren von Benutzercomputern.
