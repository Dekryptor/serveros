Ein Benutzer kann nur auf bestimmte Dienste zugreifen

Mit bestimmten Einstellungen eines Benutzer-Accounts wird der Zugriff des Benutzers auf die von Ihrem Server bereitgestellten Dienste gesteuert.

Gehen Sie wie folgt vor, um einem Benutzer Zugriff auf weitere Dienste zu gewähren:

  1. Wählen Sie in der Servereinstellung "Benutzer" einen Benutzer in der Liste aus.

  2. Klicken Sie auf "Dienste". Markieren Sie danach die Felder der Dienste, die dem Benutzer zur Verfügung gestellt werden sollen.
