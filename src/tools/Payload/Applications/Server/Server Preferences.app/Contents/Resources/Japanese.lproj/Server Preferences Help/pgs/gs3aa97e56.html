Google トークおよびその他の XMPP サービスのユーザとチャットする

サーバのユーザが、「Google トーク」や、XMPP プロトコルを使用するほかのインスタント・メッセージ・システムのユーザとインスタントメッセージを交換できるようにすることができます。

Google トークやその他の XMPP サービスを介してチャットするには：

  1. 「サーバ環境設定」の「iChat」パネルで、「サーバ間通信を有効にする」を選択します。

  2. iChat サービスを再起動するときは、「入」／「切」を 2 回クリックします。

    iChat サービスの設定変更は、iChat サービスの再起動後に有効になります。

ユーザにほかの XMPP メッセージングサービスを使用している人とインスタントメッセージのやりとりをさせたい場合で、お使いのローカルネットワークに独立したファイアウォール装置がある場合は、ファイアウォールの管理者に問い合わせて、iChat サービスで使用する通信ポートのファイアウォールを解放してもらってください。ポートのリストについては、「お使いになる前に」マニュアルの付録を参照してください。

お使いのサーバがケーブル・ルーター、DSL ルーター、またはそのほかのルーターを介してインターネット接続を利用している場合は、そのルーターで、iChat サービスが使用する通信ポートのポートフォワード（ポートマッピング）を設定する必要があります。ポートフォワードの詳細については、「お使いになる前に」マニュアルを参照してください。
