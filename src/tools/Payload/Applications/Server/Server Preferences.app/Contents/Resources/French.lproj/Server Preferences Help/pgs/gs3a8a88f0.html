Dernières rubriques d’aide

Lorsque votre ordinateur est connecté à Internet, il obtient automatiquement de nouvelles rubriques d’aide ou des rubriques mises à jour. Certaines de ces rubriques peuvent également mettre à jour des rubriques dans le manuel Premiers contacts inclus avec Mac OS X Server.

  • Pour afficher la liste des rubriques d’aide les plus récentes, cliquez sur Départ, puis cliquez sur le lien Dernières rubriques d’aide. Vérifiez régulièrement si de nouvelles rubriques sont ajoutées.

  • Pour rechercher les informations les plus récentes du site web d’assistance Mac OS X Server d’Apple, utilisez le champ de recherche ci-dessus. Vous pouvez activer ou désactiver cette recherche en sélectionnant « Inclure les recherches effectuées sur le site de support produits » dans le menu local du champ de recherche.

  • Pour afficher un flux RSS répertoriant les mises à jour les plus récentes de Premiers contacts et de l’aide à l’écran de Mac OS X Server, utilisez un lecteur RSS tel que Safari ou Mail, puis accédez à :

feed://helposx.apple.com/rss/snowleopard/serverdocupdates.xml
