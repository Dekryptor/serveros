À propos du service Carnet d’adresses

Le service de carnet d’adresses pour Mac OS X Server, appelé Serveur Carnet d’adresses, permet aux utilisateurs de conserver des coordonnées sur le serveur.

Les utilisateurs peuvent afficher et ajouter des coordonnées dans un carnet d’adresses basé sur le serveur au moyen de l’application Carnet d’adresses à partir de tout Mac Snow Leopard situé sur le réseau. L’application Carnet d’adresses peut être configurée automatiquement sur un Mac Snow Leopard pour utiliser un carnet d’adresses basé sur le serveur. Consultez Configuration automatique des Mac des utilisateurs.

Serveur Carnet d’adresses fonctionne également avec d’autres applications de gestion de coordonnées qui prennent en charge le standard ouvert CardDAV, telles que l’application open-source multiplateforme Mulberry (www.mulberrymail.com). Consultez Configuration manuelle des ordinateurs des utilisateurs.

Si vous souhaitez autoriser l’accès à Serveur Carnet d’adresses sur Internet et si vous possédez un routeur câble, un routeur DSL ou tout autre routeur, celui-ci doit être configuré pour la redirection de ports (mappage de ports) utilisés par Serveur Carnet d’adresses. Pour en savoir plus sur la redirection de port, consultez le manuel Premiers contacts.

Si vous souhaitez autoriser l’accès à Serveur Carnet d’adresses à l’extérieur de votre réseau local et si celui-ci dispose d’un coupe-feu séparé, demandez à l’administrateur du coupe-feu d’ouvrir celui-ci aux ports et aux protocoles utilisés par Serveur Carnet d’adresses. Pour accéder à une liste des ports, consultez l’annexe dans le manuel Premiers contacts.
