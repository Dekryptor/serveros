iCal サービスを開始する／停止する

「iCal」パネルで iCal カレンダーサービスを開始または停止できます。iCal サービスが無効の間は、ユーザが共有カレンダーにアクセスできません。

iCal サービスを開始または停止するには：

「サーバ環境設定」の「iCal」パネルで「入」／「切」スイッチをクリックします。

iCal サービスを開始した後、特にこのサービスをはじめて開始した場合は、「ユーザのコンピュータを手動で設定する」で説明している通りにカレンダーアプリケーションを設定するようユーザに通知してください。
