Setting Up Blogs
To turn user blogs on or off:

In the Web pane, click Web Services and then select or deselect Blogs.

If this option is selected and web services are on, clicking the link arrow next to it goes to the list of all blogs.

Users can access their blogs by appending /users/ to your serverâ€™s website address. For example:

http://server.example.com/users/

For information about using blogs, click Help on any wiki or blog page.
