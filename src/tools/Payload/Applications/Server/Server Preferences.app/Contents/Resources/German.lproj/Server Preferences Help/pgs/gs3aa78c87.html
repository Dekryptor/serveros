Einrichten des Diensts "Webmail"
Gehen Sie wie folgt vor, um den Dienst "Webmail" zu aktivieren oder zu deaktivieren:

Klicken Sie im Bereich "Web" auf "Webdienste" und aktivieren oder deaktivieren Sie dann "Webmail".

Wenn Sie diese Option aktivieren, können Sie mit dem als Link definierten Pfeil neben der Option die Website für den Dienst "Webmail" öffnen.

Damit Benutzer den vom Server bereitgestellten Dienst "Webmail" verwenden können, müssen sie an die Website-Adresse des Servers den Zusatz "/webmail" anhängen. Hier ein Beispiel:

http://server.beispiel.de/webmail/
