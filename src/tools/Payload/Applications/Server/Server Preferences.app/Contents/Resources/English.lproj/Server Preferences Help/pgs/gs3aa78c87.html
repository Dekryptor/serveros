Setting Up Webmail
To turn webmail service on or off:

In the Web pane, click Web Services and then select or deselect Webmail.

If this option is selected, clicking the link arrow next to it opens the webmail website.

Users can access your server’s webmail by appending /webmail/ to your server’s website address. For example:

http://server.example.com/webmail/
