Qu’est-ce que Mac OS X Server ?

Mac OS X Server 10.6 associe la simplicité et la convivialité du Macintosh à une base UNIX robuste. Il propose une vaste gamme de services prenant en charge des ordinateurs clients Macintosh, Windows et UNIX via un réseau.

Avec Snow Leopard Server, les petites organisations et les groupes de travail qui ne disposent pas d’un service informatique peuvent exploiter au mieux les avantages d’un serveur. Même un utilisateur sans connaissances techniques peut configurer et gérer Snow Leopard Server pour un groupe. Les autres utilisateurs du groupe peuvent configurer automatiquement leur Mac pour obtenir des services auprès de Snow Leopard Server. Snow Leopard Server dispose également d’options de configuration et d’outils de gestion avancés destinés aux informaticiens.

Utilisateurs et groupes

Toute personne souhaitant obtenir des services auprès de votre serveur nécessite un compte utilisateur. Un compte utilisateur permet à une personne de s’authentifier pour bénéficier de services, contrôle les services auxquels la personne peut accéder, contient les informations de contact de la personne et indique les appartenances de groupe de celle-ci.

Lorsque vous configurez votre serveur pour la première fois, vous indiquez l’emplacement où il obtient les comptes utilisateur et les comptes de groupe. Votre serveur peut disposer de ses propres utilisateurs et de ses propres groupes que vous créez au moyen des Préférences du serveur. Vous pouvez configurer votre serveur afin d’autoriser l’importation d’utilisateurs à partir d’un serveur d’annuaire de votre organisation. Vous pouvez également configurer votre serveur pour qu’il accède aux utilisateurs et aux groupes résidant sur le serveur d’annuaire de votre organisation au lieu de disposer de ses propres utilisateurs ou de les importer.

Vous pouvez également créer des groupes, configurer des services de groupe et modifier l’appartenance de groupe.

Informations sur les services et le système

Votre serveur propose aux utilisateurs divers services tels que le partage de fichiers, un carnet d’adresses, un calendrier, une messagerie instantanée, une messagerie électronique, un web et un VPN. Vous pouvez personnaliser les réglages de service et activer ou désactiver les services. Vous pouvez examiner les informations système, les historiques et les graphiques. Vous pouvez également modifier les informations système, les paramètres de coupe-feu et les réglages de sauvegarde des ordinateurs des utilisateurs.
