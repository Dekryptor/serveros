Finding the Server’s Website Address

Users can visit your server’s homepage and web services at the addresses shown in the following table. Replace the italicized placeholder in the addresses with your server’s DNS name, which is shown in the Information pane of Server Preferences.

To see:
Use this address:

Server’s homepage

http://myserver.example.com

My Page

http://myserver.example.com/updates/

User’s calendar

http://myserver.example.com/ical/

All wikis

http://myserver.example.com/groups/

All blogs

http://myserver.example.com/users/

Webmail

http://myserver.example.com/webmail/

If your server homepage is a private wiki, visitors must log in using the name and password of a user who has permission to see the wiki.
