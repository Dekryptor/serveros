ユーザアカウントを削除する

「サーバ環境設定」を使用して、サーバ上で不要になったユーザアカウントを削除できます。

ユーザアカウントを削除するには：

  1. 「サーバ環境設定」の「ユーザ」パネルで、左側のリストから、削除するユーザアカウントを選択します。

  2. 削除（－）ボタンをクリックします。

ユーザアカウントを削除すると、そのアカウントが属していたグループメンバーシップはキャンセルされ、グループサービスおよびプライベート Wiki がアクセス停止になります。ユーザアカウントを削除すると、サーバに保存されているそのユーザのメールデータも削除されます。削除されたユーザアカウントは、サーバ上のカレンダーおよびアドレスブックの情報にアクセスできません。

ユーザアカウントを削除しても、ユーザのバックアップデータは取り除かれません。削除したユーザのコンピュータの「Time Machine」環境設定で、バックアップの保存先にサーバを使用するように設定されていた場合は、「サーバ環境設定」の「Time Machine」パネルで指定されているバックアップディスク上の「/Shared Items/Backups/」に、ユーザのバックアップデータが残ります。
