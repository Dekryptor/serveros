高度なツールとアプリケーション

「サーバ環境設定」およびそのほかの基本管理アプリケーションとは別に、Snow Leopard Server には、以下の表に示すような高度な管理アプリケーションおよびツールが含まれています。これらのツールやアプリケーションの詳細については、「サーバ管理」を開いて「ヘルプ」メニューを参照するか、Mac OS X Server リソースの Web サイト（www.apple.com/jp/server/macosx/resources/）を参照してください。

重要： Mac OS X Server v10.5 Leopard 以前の管理用アプリケーションおよびツールをお持ちの場合、Snow Leopard Server ではそれらを使用しないでください。
高度なサーバ管理者用アプリケーションとツール

ディレクトリユーティリティ

ディレクトリサーバへの高度な接続を設定します。「システム環境設定」の「アカウント」パネルにある「ログインオプション」セクションで、「編集」（または「接続」）をクリックして、「ディレクトリユーティリティ」を開きます。

Podcast キャプチャ（「/アプリケーション/ユーティリティ/」にあります）

高品質のオーディオやビデオをローカルまたはリモートのカメラで収録したり、画面の動きをキャプチャしたり、QuickTime ファイルを Podcast プロデューサーにアップロードしてエンコーディングや配信を行ったりします。

Podcast Composer（「/アプリケーション/サーバ/」にあります）

構造化されたグラフィカルなプロセスに従って、Podcast プロデューサーによる Podcast の生成および配信を制御するワークフローを作成します。

QuickTime Broadcaster（「/アプリケーション/」にあります）

高品質のネットワークブロードキャストを行うために、QuickTime Streaming Server とシームレスに連携するライブオーディオやビデオをキャプチャします。

RAID Admin（「/アプリケーション/サーバ/」にあります）

Xserve RAID ハードウェアを設定し、監視します。

サーバ管理（「/アプリケーション/サーバ/」にあります）

高度なサービス設定の変更、高度なサービスの構成、およびファイル共有ポイントの管理を行います。サーバアクティビティを監視し、詳細なサービスログを表示します。

サーバアシスタント

保存済みの自動設定プロファイルを使って、複数のサーバを自動的に設定します。「サーバ管理」の「サーバ」メニューを使って、「サーバアシスタント」を開きます。

サーバモニタ（「/アプリケーション/サーバ/」にあります）

1 つまたは複数の Xserve システムをリモートで監視し、管理します。

システムイメージユーティリティ（「/アプリケーション/サーバ/」にあります）

Mac OS X および Mac OS X Server コンピュータのための NetBoot および NetInstall イメージを作成します。

システム環境設定（「/アプリケーション/」にあります）

サーバを組織内のディレクトリサーバに接続します。

ワークグループマネージャ（「/アプリケーション/サーバ/」にあります）

高度なサーバ展開によりユーザ、グループ、コンピュータ、コンピュータグループを管理します。Mac OS X ユーザの環境設定を管理します。

Xgrid Admin（「/アプリケーション/サーバ/」にあります）

クラスタをリモート管理し、コントローラやエージェントのアクティビティを監視し、グリッド上のジョブの状況をチェックします。

コマンドラインツール

UNIX ツールを使用して、サーバソフトウェアのインストールと設定、サービスの管理、ユーザの管理などを行います。
