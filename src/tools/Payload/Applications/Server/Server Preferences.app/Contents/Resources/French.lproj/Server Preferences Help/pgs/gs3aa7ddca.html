Démarrage ou arrêt du service VPN

Vous pouvez arrêter ou démarrer le service d’accès distant VPN dans la sous-fenêtre VPN. Quand le service VPN est arrêté, les utilisateurs ne peuvent établir de connexions distantes au serveur via Internet.

Pour démarrer ou arrêter le service VPN :

Cliquez sur le bouton Marche/Arrêt de la sous-fenêtre VPN des Préférences du serveur.

Après avoir démarré le service VPN, en particulier s’il s’agit de la première fois, vous devez dire aux utilisateurs de configurer leur connexion VPN comme indiqué dans les rubriques Configuration d’une connexion VPN pour les utilisateurs Mac et Configuration manuelle de la connexion VPN d’un utilisateur.
