Hinzufügen eines gemeinsamen Ordners

Sie können einen Ordner für die gemeinsame Nutzung hinzufügen, auf den Benutzer zugreifen können, wenn Sie die Verbindung zum Server herstellen, um Daten gemeinsam mit anderen Benutzern zu verwenden. Auf einem Mac-Computer werden gemeinsame Ordner als freigegebene Volumes angezeigt. Unter Windows werden gemeinsame Ordner im Ordner "Netzwerkumgebung" angezeigt. Gemeinsam genutzte Ordner werden auch als Netzwerkordner bezeichnet.

Gehen Sie wie folgt vor, um einen Ordner für die gemeinsame Nutzung hinzuzufügen:

  1. Erstellen und benennen Sie einen Ordner im Finder, wenn Sie einen bisher noch nicht vorhandenen Ordner für die gemeinsame Nutzung freigeben wollen.

  2. Öffnen Sie die Servereinstellung "Dateifreigabe", klicken Sie auf "Hinzufügen" (+), wählen Sie den freizugebenden Ordner aus und klicken Sie auf "Freigeben".

    Wenn Sie das Programm "Servereinstellungen" auf dem Servercomputer ausführen, können Sie alternativ den gewünschten Ordner auch aus dem Finder in die Liste im Bereich "Dateifreigabe" bewegen.

  3. Klicken Sie auf "Zugriffsrechte bearbeiten", wenn es Benutzern möglich sein soll, auch ohne Anmeldung auf diesen Ordner zuzugreifen, bzw. wenn Sie die Liste der Benutzer und Gruppen bearbeiten wollen, die zum Zugriff auf den Ordner berechtigt sind.

    Steuern des Zugriffs auf einen gemeinsamen Ordner

Zum Erstellen eines gemeinsamen Ordners für einen bestimmte Gruppe müssen Sie die Servereinstellung "Gruppen" verwenden. Die gemeinsamen Ordner aller Gruppen werden innerhalb des Ordners "/Groups/" angelegt.
