Importing a Certificate

If you already have files containing an SSL certificate and matching private key, you can import them, and then use the certificate to secure services provided by your server.

To import a certificate:

  1. In the Finder, locate the files containing the certificate and matching private key, and put the files somewhere you’ll be able to see them, such as on the desktop, while using Server Preferences.

  2. In the Information pane of Server Preferences, click the Edit button to the right of SSL Certificate.

  3. Choose Certificate Import > Import Certificate from the pop-up menu.

    “Use SSL certificate” must be selected to make the pop-up menu available.

  4. Drag the files containing the certificate and private key to the middle of the dialog in Server Preferences, and then click Import.

  5. To use the imported certificate, choose it from the pop-up menu.
