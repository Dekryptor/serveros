Modification du nom de votre serveur

Avec Préférences du serveur, vous pouvez personnaliser des réglages de services, gérer des utilisateurs et des groupes et afficher l’état du serveur. Vous pouvez utiliser les Préférences du serveur sur le serveur à gérer ou à distance à partir d’un ordinateur administrateur.

Pour modifier les réglages du serveur :

Ouvrez les Préférences du serveur et cliquez sur l’icône des réglages à modifier.

Si vous ne savez pas où réside un réglage, utilisez la zone de recherche pour le trouver. Pour obtenir des instructions consultez Recherche de réglages dans Préférences du serveur.

Pour modifier des réglages du serveur lui-même, tels que les préférences Réseau du serveur ou ses préférences Time Machine, ouvrez l’application Préférences Système à partir du serveur. Pour plus d’informations concernant l’utilisation des Préférences Système, ouvrez l’application, puis utilisez le menu Aide.
