Envoi manuel d’une invitation par courrier électronique

Vous pouvez utiliser Préférences du serveur pour envoyer une invitation par courrier électronique à un utilisateur lui décrivant votre serveur et les services qu’il apporte. Le serveur tente d’envoyer automatiquement un message de bienvenue lorsque vous créez un nouveau compte utilisateur. Vous pouvez également, si vous le souhaitez, envoyer une invitation lorsque vous importez des utilisateurs. En outre, vous pouvez envoyer manuellement une invitation à un utilisateur à tout instant.

Pour envoyer un courrier électronique d’invitation à un utilisateur :

  1. Dans la sous-fenêtre Utilisateurs de Préférences du serveur, sélectionnez le compte utilisateur auquel vous souhaitez envoyer l’invitation.

  2. Cliquez sur le bouton Action (en forme d’engrenage) et sélectionnez « Envoyer une invitation à __ ».

  3. Saisissez l’adresse électronique de l’utilisateur, puis cliquez sur Envoyer une invitation.

Vous pouvez ajouter votre nom, votre adresse électronique et un message personnel au message d’invitation standard. Pour obtenir des instructions, consultez Personnalisation du message électronique d’invitation du serveur.
