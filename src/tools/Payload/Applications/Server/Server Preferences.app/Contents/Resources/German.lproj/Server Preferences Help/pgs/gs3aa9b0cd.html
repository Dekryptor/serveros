Verwalten des Diensts "VPN"

In der Servereinstellung "VPN" können Sie den Dienst "VPN" für den entfernten Zugriff aktivieren und deaktivieren, den Schlüssel ("Shared Secret") für VPN überprüfen und ändern, den für den VPN-Zugriff verfügbaren IP-Adressbereich festlegen und für Benutzer von Mac OS X eine spezielle VPN-Konfigurationsdatei sichern.

Gehen Sie wie folgt vor, um die Einstellungen der Servereinstellung "VPN" zu konfigurieren:

Ändern Sie die Einstellungen wie erforderlich und beachten Sie dabei die folgenden Informationen.

  • Ein/Aus: 
    Gibt den Status des Diensts "VPN" an. Durch Klicken auf die Taste "Ein/Aus" wird der Dienst aktiviert oder deaktiviert.
  • Schlüssel ("Shared Secret"): 
    Bewirkt, dass der Server entfernte Computer als vertrauenswürdig einstuft, auf denen der Schlüssel installiert ist, und dass der Server für entfernte Computer als vertrauenswürdig gilt. Klicken Sie auf "Bearbeiten", um die Informationen des Schlüssels zu lesen oder zu ändern. Eine Änderung des "Shared Secret"-Schlüssels auf dem Servercomputer muss stets von allen VPN-Benutzern in ihren jeweiligen VPN-Konfigurationen nachvollzogen werden. Anleitungen hierzu finden Sie unter Ändern des "Shared Secret"-Schlüssels für VPN.
  • IP-Adressbereich: 
    Entfernten Computern wird eine IP-Adresse aus diesem Bereich zugewiesen, wenn sie eine VPN-Verbindung herstellen. Anleitungen hierzu finden Sie unter Ändern des IP-Adressbereichs für VPN.
  • VPN-Client-Konfigurationsdatei: 
    Klicken Sie auf "Sichern unter", um automatisch eine Datei zu generieren, die Sie an Benutzer weitergeben können, die von Mac-Computern mit Mac OS X 10.3 (oder neuer) per Fernzugriff auf den Server zugreifen wollen. Anleitungen hierzu finden Sie unter Erstellen einer VPN-Konfigurationsdatei.
