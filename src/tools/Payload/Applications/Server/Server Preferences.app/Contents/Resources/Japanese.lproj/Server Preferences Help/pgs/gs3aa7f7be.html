ファイル共有サービスについて

Mac OS X Server のファイル共有サービスを使用すると、ユーザは共有フォルダにアクセスして、サーバ上にファイルを保管できます。ユーザは Macintosh、Windows、または UNIX コンピュータから、特別なソフトウェアを使用せずに、AFP や SMB などのネイティブのファイルプロトコルを使用して、自分のファイルや共有フォルダにアクセスできます。Windows ユーザには、「マイネットワーク」の中に Mac OS X Server ファイルサーバが、Windows ファイルサーバと同様に表示されます。

インターネット上のファイル共有へのアクセスを許可したい場合で、ケーブル・ルーター、DSL ルーター、またはそのほかのネットワークルーターを持っている場合は、お使いのルーターでファイル共有用にポートフォワード（ポートマッピング）を設定する必要があります。ポートフォワードの詳細については、「お使いになる前に」マニュアルを参照してください。

ローカルネットワークの外部のファイル共有へのアクセスを許可したい場合で、お使いのローカルネットワークに独立したファイアウォール装置がある場合は、ファイアウォールの管理者に問い合わせて、ファイル共有で使用するポートのファイアウォールを解放してもらってください。ポートのリストについては、「お使いになる前に」マニュアルの付録を参照してください。
