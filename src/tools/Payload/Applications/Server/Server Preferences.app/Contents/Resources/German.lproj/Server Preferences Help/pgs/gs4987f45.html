Ihr Server verwendet die Application Firewall

Wenn Sie Ihren Server von Mac OS X Server 10.5 Leopard aktualisiert haben und die Firewall des Servers vor der Aktualisierung auf Snow Leopard Server in der Servereinstellung "Firewall" aktiviert war, können Sie die Firewall nicht im Bereich "Sicherheit" verwalten. In diesem Fall verwendet Ihr Server eine andere Firewall-Technologie, eine so genannte Application Firewall, die in der Systemeinstellung "Sicherheit" verwaltet werden kann. Wenn Sie die Firewall in der Systemeinstellung "Sicherheit" deaktivieren, können Sie die Firewall des Servers im Programm "Servereinstellungen" verwalten.

Gehen Sie wie folgt vor, um die Firewall des Servers im Programm "Servereinstellungen" zu verwalten:

  1. Öffnen Sie die Systemeinstellung (nicht die Servereinstellung) "Sicherheit" auf dem Server und klicken Sie auf "Firewall".

    Systemeinstellung "Sicherheit"

  2. Klicken Sie auf "Stopp", um die Application Firewall zu deaktivieren.
