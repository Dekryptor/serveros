Personnalisation du message électronique d’invitation du serveur

Les Préférences du serveur permettent d’ajouter vos nom et adresse électronique ainsi qu’une présentation personnelle au message électronique standard par lequel votre serveur indique aux utilisateurs comment accéder à ses services. Le message standard indique le nom DNS du serveur et explique quels services sont fournis par le serveur. Les destinataires qui possèdent Mac OS X 10.6 peuvent cliquer sur un bouton dans le message électronique pour configurer automatiquement l’accès de leur Mac aux services du serveur.

Vous pouvez sélectionner une option permettant d’envoyer ce message électronique lorsque vous importez des utilisateurs ou un groupe d’utilisateurs. Le serveur envoie l’invitation aux adresses électroniques figurant dans les comptes utilisateur importés. Le serveur n’envoie pas d’invitation aux comptes utilisateur importés ne contenant pas d’adresse électronique.

Vous pouvez également envoyer une invitation par courrier électronique manuellement. Pour obtenir des informations, consultez Envoi manuel d’une invitation par courrier électronique.

Pour personnaliser le message électronique envoyé aux nouveaux comptes utilisateur importés :

  1. Dans la sous-fenêtre Utilisateurs de Préférences du serveur, cliquez sur le bouton Action (en forme d’engrenage) et choisissez Réglages de message dans le menu local.

  2. Saisissez le nom et l’adresse électronique de l’expéditeur dans les champs « Nom complet de l’admin » et Adresse de l’admin.

  3. Le cas échéant, saisissez un message personnel dans le champ Invitation.

    Si vous ne voyez pas le champ Invitation, votre serveur n’est pas connecté à un serveur de répertoire. Consultez Connexion de votre serveur à un serveur d’annuaire.

    Vous pouvez utiliser le message pour vous présenter, ce qui garantit l’authenticité du message aux destinataires. Par exemple : Bonjour, je suis l’administratrice de notre serveur. Si vous avez besoin d’aide pour accéder à ses services, n’hésitez pas à m’envoyer un message électronique ou à m’appeler au 310-555-4357. —Valérie

Les destinataires pourront voir votre présentation dans un encadré indépendant du message standard généré par le serveur.
