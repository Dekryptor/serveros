Personnalisation d’un message électronique de bienvenue ou d’invitation

Vous pouvez utiliser Préférences du serveur pour personnaliser les messages électroniques qui souhaitent la bienvenue aux nouveaux utilisateurs ou qui invitent des utilisateurs existants à tirer profit du serveur. Votre serveur peut envoyer tous les messages répertoriés ci-dessous s’il est configuré pour importer des utilisateurs provenant d’un serveur d’annuaire. Si votre serveur n’est pas connecté à un serveur d’annuaire, il envoie uniquement le message de bienvenue.

Message d’accueil

Informe les nouveaux utilisateurs sur les services envoyés par votre serveur. Envoyé automatiquement aux nouveaux utilisateurs dont les ordinateurs obtiennent un service de messagerie de la part du serveur. Non envoyé si le service de messagerie du serveur est arrêté.
