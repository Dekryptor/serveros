Configuration des wikis

Si le service Wikis est activé dans la sous-fenêtre Web, tous les utilisateurs du serveur peuvent créer des wikis à l’aide d’un navigateur web.

Pour activer et désactiver le service Wikis :

Dans la sous-fenêtre Web, cliquez sur Services web et sélectionnez ou désélectionnez Wikis.

Si cette option est sélectionnée et si les services web sont activés, le fait de cliquer sur la flèche de lien située à côté ouvre la liste de tous les wikis.

Si les wikis sont disponibles, les utilisateurs peuvent accéder à leur propre page wiki et aux wikis qu’ils sont autorisés à utiliser. Vous pouvez changer l’autorisation d’utiliser un wiki dans la section Réglages du wiki. Le propriétaire du wiki ou l’administrateur du serveur peuvent modifier un wiki à partir de la section Réglages.

Les utilisateurs peuvent accéder aux wikis en ajoutant /groups/ à l’adresse du site web de votre serveur. Par exemple :

http://serveur.exemple.com/groups/

Les utilisateurs ayant un compte listé dans la sous-fenêtre Utilisateurs de Préférences du serveur peuvent créer des wikis après avoir ouvert une session de leur portail wiki Ma page.

Pour obtenir des informations sur l’utilisation d’un wiki, cliquez sur le lien Aide à partir de n’importe quelle page wiki.
