Sie können den Kurznamen eines Benutzer-Accounts nicht ändern

Nach der Erstellung eines Accounts können Sie dessen Kurznamen nicht mehr ändern. Wenn Sie den Kurznamen ändern müssen, löschen Sie den Account und erstellen Sie einen neuen Benutzer-Account mit einem neuen Kurznamen. Anleitungen hierzu finden Sie unter Löschen eines Benutzer-Accounts und Hinzufügen eines Benutzers.

Durch das Löschen eines Benutzer-Accounts werden dessen Gruppenmitgliedschaften storniert und der Zugriff auf Gruppendienste und private Wikis wird unterbunden. Durch das Löschen eines Benutzer-Accounts werden auch die E-Mails des Benutzers auf dem Server gelöscht. Ein gelöschter Benutzer-Account bietet keinen Zugriff mehr auf Kalender und Adressbuchinformationen auf dem Server.

Nachdem Sie für einen Benutzer mit Mac OS X 10.6 Snow Leopard einen neuen Account auf dem Server erstellt haben, muss der Benutzer im Bereich "Anmeldeoptionen" der Systemeinstellung "Benutzer" den Server aus der Liste der Verzeichnisserver entfernen und dann erneut hinzufügen. Informationen zum Herstellen einer Verbindung zu Ihrem Server finden Sie unter Konfigurieren eines Mac-Computers mit der Systemeinstellung "Benutzer".
