Definieren aller Gruppenmitglieder als iChat-Kontakte

Sie können festlegen, dass die Jabber-IDs (Bildschirmnamen) aller Benutzer, die Mitglieder einer Gruppe sind, automatisch der Jabber-Kontaktliste jedes Gruppenmitglieds hinzugefügt werden. Gruppenmitglieder können ihre Jabber-Kontaktlisten im Programm "iChat" (bzw. einem anderen XMPP-Instant-Messaging-Programm) anzeigen und Kontakte hinzufügen und entfernen. Externe Mitglieder der Gruppe werden nicht zur automatisch erstellten iChat-Kontaktliste hinzugefügt.

Gehen Sie wie folgt vor, um alle Gruppenmitglieder als Jabber-Kontakte zu definieren:

  1. Öffnen Sie die Servereinstellung "Gruppen" und wählen Sie in der Liste links die Gruppe aus, die Sie ändern wollen. Klicken Sie danach auf "Dienste".

  2. Markieren Sie das Feld "Automatisches Hinzufügen zur iChat-Kontaktliste".

  3. Starten Sie den Dienst "iChat" erneut, indem Sie zweimal auf "Ein/Aus" klicken.

    Änderungen am Dienst "iChat" werden erst nach dem Neustart des Diensts wirksam.

Wenn Sie das Feld "Automatisches Hinzufügen zur iChat-Kontaktliste" deaktivieren, werden Gruppenmitglieder nicht automatisch aus den Kontaktlisten der jeweils anderen Gruppenmitglieder entfernt. Gruppenmitglieder haben die Möglichkeit, Kontakte zu löschen, die ihrer Kontaktliste automatisch hinzugefügt wurden.
