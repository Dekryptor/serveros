サーバ環境設定とは

「サーバ環境設定」を使用すると、サービスを利用できるユーザ、サービスの構成方法、サービスの状況など、Snow Leopard Server の標準構成またはワークグループ構成のさまざまな設定を管理できます。

「サーバ環境設定」を開くと、個々の環境設定が、次に説明するカテゴリごとにグループ分けされています。リンクをクリックすると、それぞれのカテゴリの設定や環境設定の詳細が表示されます。
