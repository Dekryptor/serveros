Sending an Email Invitation Manually

You can use Server Preferences to email an invitation telling a user about your server and the services it provides. The server tries to email a welcome message automatically when you create a new user account, and you have the option of sending an invitation when you import users. You can also manually send an invitation to a user at any time.

To send an email invitation to a user:

  1. In the Users pane of Server Preferences, select the user account you want to send the invitation to.

  2. Click the Action (gear) button and choose “Send Invitation to __.”

  3. Enter the user’s email address and click Send Invitation.

You can add your name, email address, and personal message to the standard invitation email. For instructions, see Customizing the Server Invitation Email.
