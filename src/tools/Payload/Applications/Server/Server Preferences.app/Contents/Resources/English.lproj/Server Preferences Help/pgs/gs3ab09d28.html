Replacing a Self-Signed Certificate

After you receive a signed certificate from a certificate authority, you can use it to replace your self-signed certificate.

To replace a self-signed certificate:

  1. In the Finder, locate the signed certificate file you received from a certificate authority, and put the file somewhere you’ll be able to see it, such as on the desktop, while using Server Preferences.

  2. In the Information pane of Server Preferences, click the Edit button to the right of SSL Certificate.

  3. Choose the self-signed certificate you want to replace from the pop-up menu.

    “Use SSL certificate” must be selected to make the pop-up menu available.

  4. Choose Certificate Signing > “Replace With Signed or Renewed Certificate” from the pop-up menu.

  5. Drag the file containing the signed certificate to the middle of the dialog in Server Preferences, and then click Replace Certificate.
