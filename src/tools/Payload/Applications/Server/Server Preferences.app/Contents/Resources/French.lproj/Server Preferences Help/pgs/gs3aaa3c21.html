Configuration de blogs
Pour activer et désactiver les blogs d’utilisateur :

Dans la sous-fenêtre Web, cliquez sur Services web, puis sélectionnez ou désélectionnez Blogs.

Si cette option est sélectionnée et les services web activés, cliquez sur la flèche de lien située en regard de cette option pour accéder à la liste de tous les blogs.

Les utilisateurs peuvent accéder à leurs blogs en ajoutant /utilisateurs/ à l’adresse du site web de votre serveur. Par exemple :

http://serveur.exemple.com/users/

Pour obtenir des informations sur l’utilisation des blogs, cliquez sur Aide dans un wiki ou dans une page de blog.
