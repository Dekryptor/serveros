Ändern des persönlichen Bilds eines Benutzers

Mit dem Programm "Servereinstellungen" können Sie das persönliche Bild für den Account eines Standardbenutzers oder eines Administrators ändern. Für Benutzer-Accounts, die einzeln oder als Gruppen von einem Verzeichnisserver importiert wurden, können die persönlichen Bilder nicht geändert werden, wenn sie ursprünglich auf dem Verzeichnisserver Ihres Unternehmens festgelegt wurden.

Gehen Sie wie folgt vor, um das persönliche Bild für einen Benutzer zu ändern:

  1. Öffnen Sie die Servereinstellung "Benutzer" und wählen Sie in der Liste links den Benutzer-Account aus, den Sie ändern wollen. Klicken Sie danach auf "Benutzer".

  2. Wenn Sie eines der vorgefertigten Bilder verwenden wollen, klicken Sie auf das aktuelle Bild und wählen Sie das gewünschte Bild aus dem Einblendmenü aus.

  3. Klicken Sie auf das Bildfeld und wählen Sie "Bild bearbeiten" aus dem Einblendmenü aus, wenn Sie das aktuelle Bild bearbeiten oder ein Bild von einer Kamera oder aus einer Datei laden wollen. Nehmen Sie die gewünschten Änderungen vor und klicken Sie auf "Einstellen". Klicken Sie auf "Abbrechen", wenn Sie Ihre Änderungen nicht verwenden wollen.

    • Zum Auswählen eines der zuletzt verwendeten Bilder klicken Sie auf "Benutzte Bilder".

    • Zum Ändern der Position des Bilds bewegen Sie es per Drag&Drop nach oben, unten oder zur Seite.

    • Zum Beschneiden des Bilds bewegen Sie den Schieberegler.

    • Zum Aufnehmen eines neuen Bilds mit einer an den Computer angeschlossenen Videokamera klicken Sie auf die Taste "Kamera".

    • Zum Anwenden eines visuellen Effekts klicken Sie auf die Taste "Visuelle Effekte" (Strudel), blättern durch die verfügbaren Effekte und wählen Sie den gewünschten Effekt aus.

    • Zum Verwenden einer Bilddatei klicken Sie auf "Wählen".
