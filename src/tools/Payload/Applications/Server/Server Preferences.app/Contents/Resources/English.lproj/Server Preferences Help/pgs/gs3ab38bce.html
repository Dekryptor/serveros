Renewing a Certificate

All certificates have an expiration date and must be replaced periodically.

To renew an expiring certificate:

  1. Request a new certificate from the certificate authority that issued the expiring certificate.

    For instructions, see Obtaining a Signed Certificate.

  2. In the Information pane of Server Preferences, click the Edit button to the right of SSL Certificate.

  3. Choose the certificate you want to replace from the pop-up menu.

    “Use SSL certificate” must be selected to make the pop-up menu available.

  4. Choose Certificate Signing > “Replace With Signed or Renewed Certificate” from the pop-up menu.

  5. Drag the file containing the replacement certificate to the middle of the dialog in Server Preferences, and then click Replace Certificate.
