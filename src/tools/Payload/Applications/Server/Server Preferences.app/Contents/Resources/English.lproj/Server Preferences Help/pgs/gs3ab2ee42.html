Managing Server Information

Use the Information pane of Server Preferences to get information about your server, including the hardware and software installed, network names and address, and serial number. You can also change the server’s computer name and serial number, manage email alerts, and manage SSL certificates.

To get or change server information:

Inspect the Information pane for the following items:

  • IP Address: 
    The unique numeric address that identifies the server on the network. For more information, see Changing Your Server’s IP Address.
  • DNS Name: 
    The full unique name provided by the DNS server that your server uses.
  • Computer Name: 
    Identifies the server to users who are browsing for network file servers or other network resources identified by computer name rather than DNS name. For more information, see Changing Your Server’s Name.
  • Server License: 
    Mac OS X Server determines the type of license from the software serial number or site license. Click Edit to change the serial number or site license information. For instructions, see Changing the Serial Number or Site License Details.
  • Alerts: 
    The email address to which the server sends messages about important system events. Click Edit to change the address or turn some types of alerts on or off. For instructions, see Changing Alert Settings.
  • SSL Certificate: 
    Create, use, and maintain identities for services that use SSL for security. For instructions, see Using an SSL Certificate.

On the left, the Information pane shows the version number and build number of Mac OS X Server currently installed, the processor speed and type, and the total amount of memory installed in the server.

You can see and change other network information in the Network pane of System Preferences on the server. For information about changing Network preferences, open System Preferences and then use the Help menu.

You can get additional information about the server by using Apple System Profiler (located in /Applications/Utilities/). For information about using System Profiler, open it and then use the Help menu.
