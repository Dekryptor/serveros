Adding or Removing Members of a Group

In the Groups pane, you can add or remove group members who are users you’ve created or imported in the Users pane. (To have imported users, your server must be connected to a directory server.)

To add or remove members of a group:

  1. In the Groups pane of Server Preferences, select the group you want to edit in the list on the left.

  2. Click Members, and then click Edit Membership.

  3. Select the checkbox next to each user you want to be a member of the group. Deselect the checkbox next to each user you don’t want to be a member.

  4. When you finish, click Edit Membership again to display a static list of group members.
