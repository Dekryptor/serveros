Entfernen eines gemeinsamen Ordners

Sie können mit dem Programm "Servereinstellungen" gemeinsame Ordner, die nicht länger für die Dateifreigabe zur Verfügung stehen sollen, jederzeit entfernen. Wenn Sie einen Ordner als gemeinsamen Ordner entfernen, verbleiben der Ordner und sein gesamter Inhalt dennoch physisch auf der Festplatte des Servers.

Gehen Sie wie folgt vor, um einen Ordner für die gemeinsame Nutzung zu entfernen:

Öffnen Sie die Servereinstellung "Dateifreigabe", wählen Sie den Ordner aus, den Sie entfernen wollen, und klicken Sie auf "Löschen" (-).
