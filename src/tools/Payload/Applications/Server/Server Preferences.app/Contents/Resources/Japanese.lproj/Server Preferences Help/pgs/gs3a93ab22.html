ホーム
サービス設定をカスタマイズする

次のサービスの設定を変更できます。ほかのサーバが提供しているサービスなどの不要なサービスは、停止することもできます。たとえば、ISP や所属組織が提供するメールサービスを使用する場合は、サーバで提供されるメールサービスは不要です。

ファイル共有

共有フォルダを追加および削除します。共有フォルダのアクセス権を変更します。

ファイル共有サービスを管理する
アドレスブック

各ユーザに個人用アドレスブックを提供するか、全ユーザで共有する単独のアドレスブックの名前を指定します。

アドレスブックサービスを管理する
iCal

各カレンダーイベントのサイズと、各ユーザのカレンダーの合計サイズを制限します。

iCal サービスを管理する
iChat

ほかのサーバのユーザとのインスタントメッセージングを有効にします。すべてのチャットのログとアーカイブを作成します。

iChat サービスを管理する
Mail

送信メールのリレーサーバを設定します。迷惑メールとウイルスのフィルタを調整します。ブラックリストに記載された送信者からのメールを拒否するようにサーバで指定します。

メールサービスを管理する
Web

サーバのホームページを選択します。Web サービス（Wiki、カレンダー、ブログ、および Web メール）の有効／無効を切り替えます。カスタム Web サイトを構成します。

Web サービスを管理する
VPN

リモートユーザがインターネットを経由してネットワークに安全にアクセスできるようにします。リモートユーザの共有シークレットおよび IP アドレスを変更します。 構成ファイルを作成して、ユーザの Mac へのリモートアクセスを設定します。

VPN サービスを管理する
