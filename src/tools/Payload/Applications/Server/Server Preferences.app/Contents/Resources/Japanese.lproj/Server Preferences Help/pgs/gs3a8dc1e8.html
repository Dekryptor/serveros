管理用コンピュータを準備する

管理用コンピュータを使って、別のコンピュータの Mac OS X Server をネットワーク経由でインストールおよび設定できます。管理用コンピュータを使って、設定後のサーバをリモートで管理することもできます。リモートサーバには、ディスプレイは必要ありません。

Mac OS X v10.6 を実行するコンピュータを管理用コンピュータにするには、サーバ管理ソフトウェアをインストールします。 すでに Mac OS X Server v10.6 が設定されているサーバがほかにある場合は、そのサーバも管理用コンピュータとして使用できます。

管理用コンピュータを設定するには：

  1. Mac OS X コンピュータに Mac OS X v10.6 Snow Leopard がインストールされていることを確認します。

  2. Mac OS X Server の「Administration Tools」ディスクを挿入します。

  3. 「インストーラ」フォルダを開きます。

  4. ServerAdministrationSoftware.mpkg をダブルクリックしてインストーラを開き、画面に表示される指示に従います。

管理者用コンピュータの使用方法については、「リモートでインストールする」、「サーバをリモートで設定する」、および「サーバ環境設定をリモートサーバに接続する」を参照してください。
