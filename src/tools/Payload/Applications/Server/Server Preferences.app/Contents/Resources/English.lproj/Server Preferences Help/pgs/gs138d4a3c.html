If the Server Runs Out of Space for Backing Up Users’ Macs

If the server runs out of space for backing up users’ Macs, you can connect another disk to the server and make it the storage location in the Time Machine pane of Server Preferences. For instructions, see Managing Users’ Backup Storage.

Snow Leopard users who don’t have Time Machine turned on in System Preferences will have their Macs automatically ask if they want to use the server as a storage location. Other users will need to select the server as their backup disk in the Time Machine pane of System Preferences.
