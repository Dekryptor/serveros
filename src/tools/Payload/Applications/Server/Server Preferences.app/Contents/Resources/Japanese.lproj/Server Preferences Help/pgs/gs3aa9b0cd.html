VPN サービスを管理する

「VPN」パネルを使用して、VPN リモート・アクセス・サービスを有効または無効にしたり、VPN シークレットを調べて変更したりできます。また、VPN ユーザの IP アドレス範囲の設定や、ユーザの VPN 構成ファイルの保存を実行できます。

「VPN」パネルを構成するには：

次の情報を参考にして、必要に応じて設定を変更します：

  • 「入」／「切」： 
    VPN サービスの状態を示します。「入」／「切」スイッチをクリックすると、サービスの有効／無効が切り替わります。
  • 共有シークレット： 
    サーバが共有シークレットを持つリモートコンピュータを信頼でき、リモートコンピュータがサーバを信頼できるようにします。 「編集」をクリックすると、共有シークレットを読み出したり、変更したりできます。ここでシークレットを変更した場合は、すべての VPN ユーザが自分の VPN 構成を同様に変更する必要があります。操作方法については、「VPN 共有シークレットを変更する」を参照してください。
  • IP アドレスの範囲： 
    リモートコンピュータが VPN 接続を行うときに、この範囲から IP アドレスが割り当てられます。操作方法については、「VPN の IP アドレス範囲を変更する」を参照してください。
  • クライアント VPN 構成ファイル： 
    「別名で保存」をクリックすると、自動的に構成ファイルが作成されます。このファイルを、Mac OS X v10.3 以降で動作する Mac からサーバにリモートアクセスを実行しようとするユーザに渡すことができます。操作方法については、「VPN 構成ファイルを作成する」を参照してください。
