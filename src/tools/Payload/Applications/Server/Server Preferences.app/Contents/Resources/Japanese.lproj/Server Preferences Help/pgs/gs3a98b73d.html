ユーザおよびグループの管理を設定する

サーバの初期設定で、ユーザおよびグループを管理する設定を行わなかった場合は、「サーバ環境設定」の「ユーザ」パネルまたは「グループ」パネルを表示したときに、「設定」ボタンが表示されます。

この「設定」ボタンをクリックすると、サーバに独自のディレクトリが作成されるように設定できます。この設定により、サーバ上に Open Directory ドメインが作成され、サーバが Open Directory のマスターになります。Open Directory の詳細については、「サーバ管理」を開いて「ヘルプ」メニューで「Open Directory の概要」を検索するか、Mac OS X Server リソースの Web サイト（www.apple.com/jp/server/macosx/resources/）を参照してください。
