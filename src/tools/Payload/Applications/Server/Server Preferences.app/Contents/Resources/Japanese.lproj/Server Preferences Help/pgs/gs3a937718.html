サーバの設定を変更する

「サーバ環境設定」を使用して、サーバが提供するサービスの設定をカスタマイズしたり、ユーザとグループを管理したり、サーバの状況を表示したりできます。「サーバ環境設定」は、管理対象のサーバ上で、または管理用コンピュータからリモートで使用できます。

サーバの設定を変更するには：

「サーバ環境設定」を開いて、変更したい設定のアイコンをクリックします。

設定の場所が分からない場合は、検索ボックスを使って検索します。操作方法については、「サーバ環境設定で設定を見つける」を参照してください。

サーバの「ネットワーク」環境設定や「Time Machine」環境設定など、サーバ自体の設定を変更するには、サーバ上で「システム環境設定」アプリケーションを開きます。「システム環境設定」の使用については、「システム環境設定」を開いて「ヘルプ」メニューを参照してください。
