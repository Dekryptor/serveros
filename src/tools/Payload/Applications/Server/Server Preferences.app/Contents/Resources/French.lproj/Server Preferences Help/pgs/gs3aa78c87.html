Configuration de Webmail
Pour activer et désactiver le service Webmail :

Dans la sous-fenêtre Web, cliquez sur Services web et sélectionnez ou désélectionnez Webmail.

Si cette option est sélectionnée, le fait de cliquer sur la flèche gauche située à côté ouvre le site web Webmail.

Les utilisateurs peuvent accéder au Webmail de votre serveur en ajoutant /webmail/ à l’adresse du site web de votre serveur. Par exemple :

http://serveur.exemple.com/webmail/
