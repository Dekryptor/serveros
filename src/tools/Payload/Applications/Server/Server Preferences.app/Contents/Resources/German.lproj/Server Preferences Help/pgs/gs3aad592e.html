Starten oder Stoppen des Diensts "iChat"

Sie können den Dienst "iChat" für das Instant-Messaging in der Servereinstellung "iChat" stoppen oder starten. Wenn der Dienst "iChat" deaktiviert ist, können Benutzer mithilfe der vom Server bereitgestellten Jabber-Routinen keine Instant-Messaging-Nachrichten senden oder empfangen.

Gehen Sie wie folgt vor, um den Dienst "iChat" zu starten oder zu stoppen:

Klicken Sie auf die Taste "Ein/Aus" in der Servereinstellung "iChat".

Nach dem Starten des Diensts "iChat" und besonders nach dem erstmaligen Starten sollten Sie die Benutzer darauf hinweisen, ihr Chat-Programm einzurichten. Hinweise dazu finden Sie unter Manuelles Konfigurieren von Benutzercomputern.
