Ändern der Servereinstellungen

Mit dem Programm "Servereinstellungen" können Sie Einstellungen für die von Ihrem Server bereitgestellten Dienste anpassen, Benutzer und Gruppen verwalten und den Serverstatus anzeigen. Sie können das Programm "Servereinstellungen" auf dem Server verwenden, den Sie verwalten möchten, oder per Fernzugriff auf einem Administratorcomputer.

Gehen Sie wie folgt vor, um die Servereinstellungen zu ändern:

Öffnen Sie das Programm "Servereinstellungen" und klicken Sie auf das Symbol für die Einstellungen, die Sie ändern möchten.

Wenn Sie im Unklaren darüber sind, wo sich eine Einstellung befindet, können Sie sie mithilfe des Suchfelds suchen. Anleitungen hierzu finden Sie unter Suchen nach Einstellungen im Programm "Servereinstellungen".

Öffnen Sie die Systemeinstellungen auf dem Server, um die Einstellungen für den Server selbst zu ändern, z. B. in der Systemeinstellung "Netzwerk" oder "Time Machine" des Servers. Weitere Informationen über die Verwendung der Systemeinstellungen erhalten Sie, wenn Sie die Systemeinstellungen öffnen und dessen Hilfe anzeigen.
