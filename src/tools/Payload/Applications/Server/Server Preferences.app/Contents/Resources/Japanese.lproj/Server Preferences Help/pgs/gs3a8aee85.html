基本アプリケーションとユーティリティ

Snow Leopard Server を設定したら、ユーザやグループを管理したり、重要なサービス設定を変更したり、次に説明するアプリケーションやユーティリティを使ってほかの基本的なサーバ管理作業を実行したりできます。

重要：これらのアプリケーションが Mac OS X Server v10.5 Leopard 以前のバージョンの場合、Snow Leopard Server ではそれらを使用しないでください。
サーバ管理者用アプリケーションとユーティリティ

iCal サーバユーティリティ（「/アプリケーション/サーバ/」にあります）

iCal サーバでホストされるカレンダーでユーザが会議を予約するとき、予約可能な場所とリソースを、作成、編集、および削除します。

サーバアシスタント

Mac OS X Server を新規に設定します。Mac OS X Server をリモートコンピュータにインストールします。

サーバ環境設定（「/アプリケーション/サーバ/」にあります）

ユーザとグループの管理、サービスとシステム情報のカスタマイズ、およびサーバ動作の監視を行います。

「Dashboard」の「サーバ状況」ウィジェット

Snow Leopard を搭載した Mac からサーバアクティビティを監視します。

システム環境設定（「/アプリケーション/」にあります）

サーバを組織内のディレクトリサーバに接続します。サーバの「Time Machine」バックアップを構成します。直接接続された USB または FireWire プリンタの共有を設定します。
