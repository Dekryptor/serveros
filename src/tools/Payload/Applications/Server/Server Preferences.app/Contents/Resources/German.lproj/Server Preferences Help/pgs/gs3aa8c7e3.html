Starten oder Stoppen des Diensts "Adressbuch"

Sie können den Dienst "Adressbuch" in der Servereinstellung "Adressbuch" stoppen oder starten. Während der Dienst "Adressbuch" deaktiviert ist, können Benutzer nicht auf Kontaktinformationen in serverbasierten Adressbüchern zugreifen.

Gehen Sie wie folgt vor, um den Dienst "Adressbuch" zu starten oder zu stoppen:

Klicken Sie auf die Taste "Ein/Aus" in der Servereinstellung "Adressbuch".

Nach dem Starten des Diensts "Adressbuch", besonders nach dem erstmaligen Starten, sollten Sie die Benutzer darauf hinweisen, ihr Programm "Adressbuch" wie unter Manuelles Konfigurieren von Benutzercomputern beschrieben einzurichten.
