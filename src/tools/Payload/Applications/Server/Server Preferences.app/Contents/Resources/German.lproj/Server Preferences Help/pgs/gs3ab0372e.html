Verwalten der Sicherungsvolumes für Benutzerdaten

Snow Leopard Server bietet Benutzern mit Mac OS X 10.6 Snow Leopard oder Mac OS X 10.5 Leopard, die Speicherplatz für ihre Datensicherungen benötigen, die Sicherheit und den Komfort der Time Machine-Sicherungsfunktion. Snow Leopard-Benutzer, die in der Systemeinstellung "Time Machine" kein Sicherungsvolume ausgewählt haben, werden automatisch gefragt, ob sie den Server als Speicherort verwenden möchten. Die anderen Benutzer müssen die Systemeinstellung "Time Machine" öffnen und das für die Speicherung von Sicherungskopien verwendete Volume ändern.

Verwenden Sie die Servereinstellung "Time Machine", um Speicherplatz auf dem Server für Sicherungskopien von Benutzercomputern bereitzustellen oder um das für die Speicherung von Sicherungskopien verwendete Volume zu ändern.

Gehen Sie wie folgt vor, um die Einstellungen der Servereinstellung "Time Machine" zu konfigurieren:

Ändern Sie bei Bedarf die aktuellen Werte der angezeigten Optionen. Beachten Sie dabei die folgenden Informationen.

  • Ein/Aus: 
    Macht den gemeinsamen Ordner "Backups" im Verzeichnis "/Shared Items/Backups/" für die Datensicherung auf dem in diesem Bereich angegeben Volume verfügbar oder nicht verfügbar.
  • Volume auswählen: 
    Klicken Sie, um aus einer Liste mit derzeit an den Server angeschlossenen Volumes ein Sicherungsvolume auszuwählen. Mac OS X Server erstellt den gemeinsam genutzten Ordner "Backups" auf dem gewählten Volume.
  • Backup-Volume auswählen: 
    Diese Taste wird angezeigt, wenn für die Speicherung von Sicherungskopien von Benutzercomputern kein Volume ausgewählt wurde. Klicken Sie auf diese Taste, um den gemeinsamen Ordner "Backups" verfügbar zu machen und das Volume auszuwählen, auf dem er abgelegt werden soll. Dies entspricht der Aktivierung von Time Machine.
  • Dateifreigabe öffnen: 
    Diese Taste wird angezeigt, wenn der Dienst "Dateifreigabe" deaktiviert ist. Klicken Sie auf die Taste, um den Bereich "Dateifreigabe" zu öffnen. Der Dienst "Dateifreigabe" muss aktiviert sein, damit die Mac-Computer der Benutzer auf den gemeinsamen Ordner "Backups" zugreifen können.

Sie können den Zugriff jedes Benutzers auf das Time Machine-Sicherungsvolume auf dem Server in der Servereinstellung "Benutzer" steuern. Außerdem haben Sie die Möglichkeit, den gemeinsamen Ordner "Backups" im Finder anzuzeigen, indem Sie auf die Pfeile im Text unter den Informationen zum Sicherungsvolume klicken.

Wenn Sie das Sicherungsvolume wechseln, werden Benutzerdaten, die aufgrund der Einstellungen für Time Machine auf dem lokalen Computer bisher bereits auf dem Server gesichert wurden, automatisch im gemeinsamen Ordner "Backups" an dessen neuen Speicherort gesichert. Allerdings kopiert Mac OS X Server vorhandene Sicherungsdaten aus dem alten Ordner "Backups" nicht in den neuen Ordner "Backups". Sie sollten nach einer Änderung des Sicherungsvolumes die Benutzer daher informieren, dass die erste Datensicherung nach der Änderung länger dauert, da alle Daten neu gesichert werden (vollständiges Backup).
