Mac OS X Server とは

Mac OS X Server バージョン 10.6 Snow Leopard では、直観的でシンプルな Macintosh の優れた操作性と、技術が成熟し安定性に優れた UNIX が融合されています。 また、ネットワーク上の Macintosh、Windows、UNIX の各クライアントコンピュータをサポートするさまざまな種類のサービスを提供します。

Snow Leopard Server を使用すると、IT 部門を持たない小規模組織やワークグループでも、サーバの機能を十分に利用することができます。技術者以外のユーザでも、グループの Snow Leopard Server の設定や管理を実行できます。グループのほかのユーザの Mac は自動的に構成され、ユーザは Snow Leopard Server のサービスを利用できます。Snow Leopard Server には、IT プロフェッショナル用の高度な構成オプションと管理ツールも用意されています。

ユーザとグループ

サーバのサービスを利用するユーザごとに、ユーザアカウントが必要です。ユーザアカウントによって、ユーザがサービスの認証を受けることが可能になり、ユーザが利用できるサービスが制御されます。ユーザアカウントはユーザの連絡先情報を含み、ユーザのグループメンバーシップを指定します。

サーバを新規に設定するとき、サーバがユーザおよびグループのアカウントをどこから取得するかを指定します。「サーバ環境設定」を使って、サーバに独自のユーザおよびグループを作成することができます。所属する組織のディレクトリサーバからのユーザの読み込みを許可するようにサーバを設定することもできます。独自のユーザを作成したり、ユーザを読み込んだりする代わりに、所属組織のディレクトリサーバに存在するユーザおよびグループにアクセスするようにサーバを設定してもかまいません。

グループを作成して、グループのサービスを設定したり、グループのメンバーシップを変更したりすることもできます。

サービスとシステム情報

サーバでは、ファイル共有、アドレスブック、カレンダー、インスタントメッセージング、メール、Web、VPN などのさまざまなサービスをユーザに提供します。サービスの設定をカスタマイズしたり、サービスを有効または無効にしたりできます。システム情報、ログ、およびグラフを確認できます。 また、システム情報、ファイアウォール設定、およびユーザのコンピュータのバックアップ設定を変更することもできます。
