Setting Up a Group File Sharing Folder

You can set up a shared folder for a group. Group members can get files from the shared folder and put files in it. It’s named using the group’s short name and is located on the server’s startup disk at /Groups/.

To set up a file sharing folder for a group:

  1. In the Groups pane of Server Preferences, select a group in the list on the left or create a new group, and then click Group.

    Creating a New Group

  2. Select the checkbox for File Sharing Folder.

  3. Click Show All, click File Sharing, and make sure file sharing service is on and access to the Groups shared folder is turned on.

When accessing the group’s shared folder, a group member authenticates using the name and password of the user account that belongs to the group.
