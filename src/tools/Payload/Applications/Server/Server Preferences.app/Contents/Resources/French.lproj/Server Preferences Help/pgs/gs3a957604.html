Suppression d’un compte d’utilisateur

Vous pouvez utiliser Préférences du serveur pour supprimer des comptes utilisateur dont votre serveur n’a plus de besoin.

Pour supprimer un compte utilisateur :

  1. Dans la sous-fenêtre Utilisateurs de Préférences du serveur, sélectionnez le compte utilisateur à supprimer dans la liste située à gauche.

  2. Cliquez sur le bouton Supprimer (–).

La suppression d’un compte utilisateur entraîne l’annulation de ses adhésions de groupes et met fin à ses accès aux services des groupes et aux wikis privés. Le courrier électronique de l’utilisateur stocké sur le serveur est également supprimé. Un compte utilisateur supprimé ne peut plus accéder aux calendriers ni au carnet d’adresses sur le serveur.

La suppression d’un compte utilisateur ne supprime pas les données de sauvegarde de l’utilisateur. Si les Préférences Time Machine de l’ordinateur de l’utilisateur supprimé sont réglées pour utiliser le serveur comme espace de stockage pour la sauvegarde, les données sauvegardées de l’utilisateur demeurent dans /Shared Items/Backups/ sur le disque de sauvegarde spécifié dans la sous-fenêtre Time Machine de Préférences du serveur.
