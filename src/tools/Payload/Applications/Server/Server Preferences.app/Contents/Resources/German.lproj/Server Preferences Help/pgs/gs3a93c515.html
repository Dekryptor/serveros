Suchen nach Einstellungen im Programm "Servereinstellungen"

Das Programm "Servereinstellungen" umfasst eine Vielzahl von Optionen und Einstellungen, mit denen Sie das Verhalten und die Funktionalität Ihres Computers anpassen können. Damit Sie gezielt auf bestimmte Einstellungen zugreifen können, unterstützt das Programm "Servereinstellungen" eine spezielle Suchfunktion.

Gehen Sie wie folgt vor, um nach bestimmten Einstellungen zu suchen:

  1. Zeigen Sie einen beliebigen Bereich des Programms "Servereinstellungen" an, mit Ausnahme des Bereichs "Benutzer" oder "Gruppen".

  2. Geben Sie in das Suchfeld oben im Fenster ein Wort oder eine Wortfolge ein, das bzw. die die gesuchte Einstellung beschreibt. Wenn Sie einer Gruppe neue Mitglieder hinzufügen wollen, könnten Sie zum Beispiel "Gruppenmitglieder" eingeben.

    Bereits während der Eingabe wird eine Liste mit Einstellungen angezeigt, die mit Ihrer Eingabe übereinstimmen.

  3. Stoppen Sie die Texteingabe, wenn in der Liste der Einstellungen die gewünschte Einstellung angezeigt wird. Im Fenster "Servereinstellungen" können eine oder mehrere Einstellungen hervorgehoben sein.

  4. Klicken Sie in der Liste auf den Eintrag der Einstellung, die Sie ändern wollen. Daraufhin wird der Bereich angezeigt, in dem Sie die gewählte Einstellung ändern können.
