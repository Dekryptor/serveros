Closing a Server Preferences Connection

For security, you should close a Server Preferences window when you aren’t actively using it to manage the server’s users, groups, services, or system information. Leaving a server connection open on an unattended server makes it easier for an unauthorized person to make changes to users, groups, or services.

Here are ways you can close a Server Preferences connection:

  • Close the Server Preferences window.

  • Choose Connection > Close.

  • Quit Server Preferences.

If you close the only open Server Preferences window, Server Preferences quits automatically.
