Gestion du service VPN

Utilisez la sous-fenêtre VPN pour activer et désactiver le service d’accès à distance VPN, vérifier ou modifier le secret VPN, définir la plage d’adresses IP pour les utilisateurs VPN ou enregistrer un ficher de configuration VPN pour les utilisateurs de Mac OS X.

Pour configurer la sous-fenêtre VPN :

Modifiez les réglages souhaités en utilisant les informations suivantes pour vous guider :

  • On/Off : 
    Cela indique l’état du service VPN et permet d’activer ou de désactiver ce service.
  • Secret partagé : 
    Cela permet au serveur de faire confiance aux ordinateurs distants détenteurs du secret partagé et inversement. Cliquez sur Modifier pour lire ou modifier le secret partagé. Si vous changez leur secret, tous les utilisateurs VPN doivent procéder au même changement dans leur configuration VPN. Pour obtenir des instructions, consultez Modification du secret partagé VPN.
  • Plage d’adresses IP : 
    Lorsqu’ils établissent une connexion VPN, les ordinateurs distants reçoivent une adresse IP de cette plage. Pour obtenir des instructions, consultez Modification de la plage d’adresses IP pour le VPN.
  • Fichier de configuration des clients VPN : 
    Cliquez sur Enregistrer sous pour générer automatiquement un fichier que vous pouvez donner aux utilisateurs souhaitant disposer d’un accès distant au serveur à partir d’ordinateurs Mac fonctionnant sous Mac OS X 10.3 ou ultérieur. Pour obtenir des instructions, consultez Création d’un fichier de configuration VPN.
