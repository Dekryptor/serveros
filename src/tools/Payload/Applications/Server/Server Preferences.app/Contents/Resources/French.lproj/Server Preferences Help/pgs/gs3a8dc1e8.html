Préparation d’un ordinateur administrateur

Vous pouvez utiliser un ordinateur administrateur pour installer et configurer Mac OS X Server sur un autre ordinateur via le réseau. Vous pouvez également utiliser un ordinateur administrateur pour gérer le serveur à distance après configuration. Il n’est pas nécessaire que le serveur distant soit équipé d’un moniteur.

Pour transformer un ordinateur Mac OS X 10.6 en ordinateur administrateur, vous devez y installer un logiciel d’administration de serveur. Si vous disposez d’un autre serveur sur lequel Mac OS X Server 10.6 est déjà configuré, vous pouvez également l’utiliser en tant qu’ordinateur administrateur.

Pour configurer un ordinateur administrateur :

  1. Assurez-vous que Mac OS X 10.6 est installé sur l’ordinateur Mac OS X.

  2. Insérez le disque Outils d’administration de Mac OS X Server.

  3. Ouvrez le dossier du programme d’installation.

  4. Double-cliquez sur ServerAdministrationSoftware.mpkg pour ouvrir le programme d’installation et suivez les instructions à l’écran.

Pour obtenir des informations sur l’utilisation d’un ordinateur administrateur, consultez Installation à distance, Configuration d’un serveur à distance et Connexion de Préférences du serveur à un serveur distant.
