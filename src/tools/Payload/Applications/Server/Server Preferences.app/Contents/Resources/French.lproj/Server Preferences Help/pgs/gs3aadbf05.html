Gestion du service de messagerie

La sous-fenêtre Mail vous permet d’activer ou de désactiver le service de messagerie, de spécifier un serveur relais pour les messages sortants et de régler le filtrage du courrier indésirable et des virus.

Pour configurer la sous-fenêtre Mail :

Modifiez les réglages souhaités en utilisant les informations suivantes pour vous guider :

  • On/Off : 
    Indique l’état du service de messagerie et permet d’activer ou de désactiver ce service.
  • Faire transiter le courrier sortant par le FAI : 
    Sélectionnez cette option uniquement si votre fournisseur d’accès à Internet vous demande de relayer les messages sortants vers un serveur particulier. Cliquez sur Modifier pour modifier l’adresse ou l’authentification pour un serveur de relais déjà configuré.
Choix d’un serveur de relais du courrier électronique
  • Rejeter le courrier provenant de serveurs sur liste noire : 
    Rejetez les messages entrants provenant d’expéditeurs dont l’adresse IP est répertoriée par le serveur zen.spamhaus.org comme l’adresse d’un expéditeur probable de courrier indésirable. Pour spécifier un autre serveur DNSBL, cliquez sur Modifier.
Rejet du courrier électronique provenant d’expéditeurs sur liste noire
  • Activer le filtrage du courrier indésirable et des virus : 
    Analyse les messages entrants, supprime ceux contenant des virus connus et marque les messages dépassant les critères de courrier indésirable minimaux définis par le curseur.
