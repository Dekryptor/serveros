Utilisation du widget d’état du serveur

Vous pouvez utiliser le widget d’état du serveur pour surveiller l’état de Mac OS X Server, soit sur le serveur lui-même, soit à partir d’un autre ordinateur sous Mac OS X Server ou Mac OS X.

Pour utiliser le widget d’état du serveur :

  1. Ouvrez Dashboard et cherchez le widget État du serveur.

    Vous pouvez ouvrir Dashboard en cliquant sur son icône dans le Dock ou en utilisant son raccourci clavier (touche F4 ou F12 en général).

    Ouvrir Dashboard

    Si vous ne voyez pas le widget État du serveur dans Dashboard, cliquez sur le bouton Ouvrir Dashboard (+) et cliquez ensuite sur le widget État du serveur ou faites-le glissez à partir de la barre des widgets.

  2. Si vous voyez les champs Serveur, Nom d’utilisateur et Mot de passe, saisissez le nom du serveur DNS ou son adresse IP suivie du nom et du mot de passe d’un administrateur et cliquez ensuite sur Terminé.

  3. Une fois le widget État du serveur connecté à un serveur, il affiche un graphique ainsi que des informations concernant l’état du serveur et de ses services.

    • Contrôlez l’utilisation du processeur, la charge du réseau ou l’utilisation des disques en cliquant sur l’une des icônes situées sous le graphique.

    • Cliquez sur le graphique pour modifier l’échelle de temps du graphique du processeur ou du réseau (une heure, un jour ou une semaine).

    • Si votre serveur possède plusieurs disques, affichez l’état de chacun d’eux en cliquant sur le graphique de l’utilisation du disque.

    • Vérifiez l’indicateur d’état et les statistiques d’activité pour les services répertoriés. Un voyant vert signifie que le service fonctionne normalement.

    • Connectez-vous à un autre serveur en déplaçant la souris vers l’angle supérieur gauche du widget et en cliquant sur le petit bouton Informations (i).

Vous pouvez ouvrir un autre widget État du serveur pour visualiser en même temps plusieurs aspects de l’état d’un serveur ou pour surveiller un autre serveur du réseau.

Le widget d’état du serveur fonctionne avec Mac OS X Server 10.6 Snow Leopard et Mac OS X 10.6 Snow Leopard.

Pour en savoir plus sur les widgets et sur Dashboard, basculez sur le Finder et consultez le menu Aide.

Il est également possible de visualiser les graphiques d’activité du serveur à l’aide de Préférences du serveur. Pour obtenir des informations, consultez Contrôle des courbes du serveur.
