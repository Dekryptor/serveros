アドレスブックサービスについて

Mac OS X Server、アドレスブックサーバのアドレスブックサービスによって、ユーザは連絡先情報をサーバ上で維持することができます。

ユーザは、ネットワーク上にある、Snow Leopard を実行する任意の Mac で「アドレスブック」アプリケーションを使うことによって、サーバベースのアドレスブックに連絡先情報を表示および追加できます。Snow Leopard を実行する Mac では、「アドレスブック」アプリケーションでサーバベースのアドレスブックが使用されるように、自動的に設定が行われます。「ユーザの Mac を自動的に設定する」を参照してください。

アドレスブックサーバは、オープンソース、クロスプラットフォームの Mulberry（www.mulberrymail.com）などの CardDAV オープンスタンダードをサポートしているほかの連絡先管理アプリケーションにも対応しています。「ユーザのコンピュータを手動で設定する」を参照してください。

インターネット上のアドレスブックサーバへのアクセスを許可したい場合で、ケーブル・ルーター、DSL ルーター、またはそのほかのネットワークルーターを持っている場合は、お使いのルーターでアドレスブックサーバ用にポートフォワード（ポートマッピング）を設定する必要があります。ポートフォワードの詳細については、「お使いになる前に」マニュアルを参照してください。

ローカルネットワークの外部のアドレスブックサーバへのアクセスを許可したい場合で、お使いのローカルネットワークに独立したファイアウォール装置がある場合は、ファイアウォールの管理者に問い合わせて、アドレスブックサーバで使用するポートおよびプロトコルのファイアウォールを解放してもらってください。ポートのリストについては、「お使いになる前に」マニュアルの付録を参照してください。
