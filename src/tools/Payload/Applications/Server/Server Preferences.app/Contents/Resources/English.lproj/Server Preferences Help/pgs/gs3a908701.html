What Is Server Preferences?

With Server Preferences, you can manage various aspects of Snow Leopard Server, such as who can use its services, how its services are configured, and what its status is.

When you open Server Preferences, individual preferences are grouped in the categories described below. Click a link to find out more about settings and preferences in each category.
