サーバ環境設定で設定を見つける

「サーバ環境設定」には数十もの設定が含まれており、これらを変更してコンピュータをカスタマイズできます。「サーバ環境設定」をすばやく検索して、特定の設定を見つけることができます。

「サーバ環境設定」で設定を見つけるには：

  1. 「サーバ環境設定」の任意のパネル（「ユーザ」パネル、「グループ」パネルは除く）を表示します。

  2. ウインドウ上部にある検索ボックスに、変更したい項目を表す単語や語句を入力します。たとえば、グループにメンバーを追加する場合は、「グループ メンバー」と入力します。

    入力に合わせて、その内容に一致する設定のリストが表示されます。

  3. 設定のリストに目的の単語や語句が表示されたら、入力を止めます。「サーバ環境設定」ウインドウでは、1 つ以上の環境設定が強調表示されています。

  4. リストで、変更したい項目に一致する項目をクリックします。開いたパネルで、その項目の設定を変更できます。
