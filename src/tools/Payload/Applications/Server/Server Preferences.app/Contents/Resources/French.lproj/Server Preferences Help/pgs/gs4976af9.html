Après la configuration d’un serveur

Après la configuration d’un serveur, vous pouvez :

  • améliorer la sécurité, l’accessibilité et la fonctionnalité de votre nouveau serveur, en suivant les conseils du document Étapes suivantes de Mac OS X Server créé et placé sur le bureau du serveur après la configuration initiale.

  • prendre quelques mesures supplémentaires pour sécuriser votre serveur. Pour obtenir des informations, consultez Sécurisation de votre serveur et Protection du compte administrateur système (root).

  • Si votre organisation dispose d’un serveur Open Directory, d’un serveur Active Directory ou de tout autre serveur d’annuaire auquel vous n’avez pas connecté (lié) votre nouveau serveur au cours de la configuration initiale, vous pouvez le connecter dès maintenant. Pour obtenir des instructions, consultez Connexion de votre serveur à un serveur d’annuaire.

  • Utilisez « Mise à jour de logiciels » pour installer les mises à jour de Mac OS X Serveur disponibles. Pour obtenir des informations, consultez Mise à jour de Snow Leopard Server.

  • Configurez une borne d’accès AirPort ou un routeur Internet pour protéger votre réseau et autoriser les utilisateurs à accéder à des services sélectionnés sur Internet. Pour en savoir plus, consultez le chapitre 2 « Préparation pour Mac OS X Server » dans le manuel Premiers contacts.

  • Si vous configurez un seul serveur pour une petite organisation ou un serveur pour un groupe de travail dans une organisation moyenne ou grande, utilisez Préférences du serveur pour configurer les utilisateurs et les groupes, personnaliser les services et les informations système, et surveiller l’activité du serveur. Pour obtenir des informations sur ces sous-tâches, consultez Gestion de comptes, Personnalisation des réglages de service et Gestion du système serveur.

    Vous pouvez utiliser conjointement le widget d’état du serveur et Dashboard pour surveiller votre serveur. Pour obtenir des informations, consultez Utilisation du widget d’état du serveur.

  • Modifiez les réglages avancés, configurez les services avancés, changez les réglages d’utilisateurs et de groupes et gérez les ordinateurs des utilisateurs à l’aide d’Admin Serveur, du Gestionnaire de groupe de travail, des autres applications Mac OS X ou des outils de lignes de commande UNIX. Pour obtenir des informations sur ces outils et ces applications, ouvrez Admin Serveur et utilisez le menu Aide ou consultez le site web Ressources Mac OS X Server à l’adresse www.apple.com/fr/server/macosx/resources/ (en anglais).
