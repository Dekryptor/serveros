Preparing an Administrator Computer

You can use an administrator computer to install and set up Mac OS X Server on another computer over the network. You can also use an administrator computer to manage the server remotely after setup. The remote server doesn’t need a display.

You make a computer with Mac OS X v10.6 into an administrator computer by installing server administration software on it. If you have another server with Mac OS X Server v10.6 already set up, you can use it as an administrator computer as well.

To set up an administrator computer:

  1. Make sure the Mac OS X computer has Mac OS X v10.6 Snow Leopard installed.

  2. Insert the Mac OS X Server Administration Tools disc.

  3. Open the Installers folder.

  4. Double-click ServerAdministrationSoftware.mpkg to open the Installer, and then follow the onscreen instructions.

For information about using an administrator computer, see Installing Remotely, Setting Up a Server Remotely, and Connecting Server Preferences to a Remote Server.
