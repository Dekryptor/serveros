Importation d’un certificat

Si vous disposez déjà de fichiers contenant un certificat SSL et une clé privée correspondante, vous pouvez les importer, puis utiliser le certificat pour protéger les services fournis par votre serveur.

Pour importer un certificat :

  1. Dans le Finder, localisez les fichiers contenant le certificat et la clé privée correspondante, puis placez-les à un emplacement visible lorsque vous utilisez Préférences du serveur, par exemple sur le bureau.

  2. Dans la sous-fenêtre Informations de Préférence du serveur, cliquez sur le bouton Modifier situé à droite de Certificat SSL.

  3. Dans le menu local, choisissez Importation de certificat > Importer un certificat.

    « Utiliser un certificat SSL » doit être sélectionné pour que le menu local soit disponible.

  4. Faites glisser les fichiers contenant le certificat et la clé privée au milieu de la zone de dialogue dans Préférences du serveur, puis cliquez sur Importer.

  5. Pour utiliser le certificat importé, choisissez-le dans le menu local.
