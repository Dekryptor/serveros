ホーム
リモートでインストールする

管理者用コンピュータの「サーバ管理」および「サーバアシスタント」を使って、Mac OS X Server をネットワーク経由で別のコンピュータにインストールできます。インストール先のコンピュータにディスプレイは必要ありませんが、Mac OS X Server Install Disc を使うので DVD ドライブは必須です。インストール先のコンピュータに DVD ドライブがない場合は、外付けの FireWire DVD ドライブ、またはターゲット・ディスク・モードで稼動している DVD ドライブ付きの Mac を接続します。

以下のことが実行できます：

  • Mac OS X Server または Mac OS X がまだインストールされていないディスクへの、Mac OS X Server の新規インストール

  • クリーンインストール（ターゲットディスクを消去、フォーマットした後に、Mac OS X Server をインストール）

  • Intel プロセッサおよび Mac OS X Server v10.5 Leopard または Mac OS X Server v10.4.11 Tiger を搭載したサーバのアップグレード（そのほかのアップグレードおよび移行のオプションについては、「サーバ管理」を開いて「ヘルプ」メニューを参照するか、Mac OS X Server リソースの Web サイト（www.apple.com/jp/server/macosx/resources/）を参照してください）

Mac OS X Server をリモートでインストールするには：

  1. ターゲットディスクまたはパーティションを消去する場合は、必ずそのバックアップを作成し、必要ならディスクユーティリティを使ってターゲットディスクを準備します。

    最も一般的なフォーマットである「Mac OS 拡張（ジャーナリング）」でターゲットディスクを消去するだけなら、ディスクユーティリティを使う必要はありません。ディスクユーティリティを使うと、ターゲットディスクをほかのフォーマットで消去したり、サーバのハードディスクのパーティションを作成したり、RAID セットを作成したりできます。

  2. ターゲットサーバを、Mac OS X Server Install Disc から起動します。

    ターゲットサーバに DVD ドライブが組み込まれている場合は、ディスクを挿入して、キーボードの C キーを押しながらコンピュータを再起動します。Apple ロゴが表示されたら、C キーを放します。

    ターゲットサーバに外付けの FireWire DVD ドライブが接続されている場合は、Option キーを押しながらコンピュータを再起動し、Mac OS X Server Install Disc を示すアイコンを選択して、矢印ボタンを押します。

  3. 管理者用コンピュータで「サーバ管理」を開き、左側のリストで「インストールの準備ができました」を選択します。

    「サーバ管理」は「/アプリケーション/サーバ/」にあります。

    「サーバ管理」で、設定済みのサーバに接続するためのパスワードを要求された場合は、「キャンセル」をクリックします。Mac OS X Server をリモートでインストールするのに、管理者アカウントは必要ありません。

  4. 右側にあるターゲットサーバを選択して、「インストール」をクリックします。

    目的のサーバがインストールされない場合は、「リフレッシュ」ボタン（曲がった矢印）をクリックして、ローカルネットワーク上でインストールの準備ができているサーバを、「サーバ管理」でもう一度検索することができます。

    目的のサーバが別のローカルネットワーク上にある場合は、「サーバ」＞「リモートサーバをインストール」と選択します。

  5. 「サーバアシスタント」が開いたら、ターゲットサーバのアドレスを確認し、サーバに組み込まれたハードウェアのシリアル番号の最初の 8 桁をパスワードとして入力し、「続ける」をクリックします。

    「アドレス」フィールドが空白または間違っている場合は、サーバの DNS 名またはその IP アドレスを IPv4 形式で入力します（例：192.0.2.200）。

    「サーバアシスタント」によってローカルネットワーク（IP サブネット）上に検出されたサーバの場合、IP アドレスはネットワーク上の DHCP サーバによって自動的に割り当てられます。DHCP サーバが存在しない場合は、ターゲットサーバでは 169.254.xxx.xxx というアドレス（ローカルネットワーク上のサーバ間で一意）が使用されます。後でサーバを設定するとき、IP アドレスを変更することができます。

    シリアル番号は、サーバに貼付されているラベルを参照してください。シリアル番号は、大文字と小文字の違いをラベルの通りに入力します。

    Intel プロセッサを搭載した Xserve で、メインのロジックボードが交換されているためハードウェアのシリアル番号がないものについては、パスワードとして「System S」と入力します（カギカッコは入力しません）。そのほかのコンピュータで、組み込みのハードウェアのシリアル番号がないものについては、「12345678」をパスワードとして使用します。

  6. Mac OS X Server で使用する言語を選択し、「続ける」をクリックします。

    ここで選択した言語は、ユーザのコンピュータの言語には影響しません。

  7. インストール先のディスクまたはボリューム（パーティション）を選択して、「インストール」をクリックします。

  8. 選択したボリュームにすでに Mac OS X Server または Mac OS X がインストールされている場合は、選択できるオプションを選択して「OK」をクリックします。

    オプションは以下の通りです：

    • 消去： 
      インストール先のボリュームを完全に消去してから、Mac OS X Server を新規にインストールします。
    • アップグレード： 
      このオプションは、インストール先のボリュームに Mac OS X Server v10.5 Leopard または Mac OS X Server v10.4.11 Tiger がインストールされている場合のみ選択できます。 インストール先のボリュームを消去せずにこのボリュームを Snow Leopard Server にアップグレードすることができます。アップグレードおよび移行については、「サーバ管理」を開いて「ヘルプ」メニューを参照するか、Mac OS X Server リソースの Web サイト（www.apple.com/jp/server/macosx/resources/）を参照してください。

  9. インストールが完了するとターゲットサーバが再起動します。「より詳細に」をクリックして、サーバをリモートで設定するか、Mac OS X Server を別のコンピュータにインストールすることができます。

  10. 「より詳細に」をクリックした場合は、「サーバアシスタント」で次に行いたい作業を選択します。

    • Mac OS X Server をリモートでインストール： 
      この手順を（手順 3 はスキップして）繰り返します。
    • Mac OS X Server をリモートで設定： 
      操作方法については、「サーバをリモートで設定する」を参照してください。
    • 自動設定のための情報を準備して保存： 
      設定手順を一通り完了し、設定オプションを選択して設定データを入力し、設定情報を使って今すぐサーバを構成する代わりに、設定情報を自動設定プロファイルとしてリムーバブルドライブまたはディスク上に保存します。 後で、その保存した設定プロファイルを使って、1 つまたは複数のサーバの設定を自動化できます。詳細については、「インストールと設定のワークシート」を参照してください。（これは、Mac OS X Server Install Disc の「Documentation」フォルダ内にあります。）

管理者用コンピュータで「サーバアシスタント」を使う代わりに、Mac OS X v10.5 Leopard または Mac OS X v10.6 Snow Leopard を実行する Mac で画面共有を使用するか、Apple Remote Desktop（別売）を別の Mac で使用して、インストールをリモート制御することができます。詳細については、「サーバ管理」の「ヘルプ」メニューを使って、Apple Remote Desktop の Web サイト（www.apple.com/jp/remotedesktop/）を参照してください。
