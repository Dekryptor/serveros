Ajout d’une vidéo de titre

Par défaut, le film fixe de l’étape d’édition inclut une vidéo de titre. Si tel n’est pas le cas, cliquez sur le bouton d’ajout (+) et sélectionnez Titre pour ajouter la vidéo.

Pour configurer la vidéo de titre :

  1. Dans Podcast Composer, accédez à l’étape de l’édition (choisissez Étapes > Édition ou appuyez sur Commande + 3).

  2. Double-cliquez sur l’icône de la vidéo de titre.

  3. Dans le menu local Titre du générique, choisissez l’un des styles par défaut.

  4. Dans Éléments du logo, si cette option est activée, sélectionnez un style par défaut. Vous pouvez aussi sélectionner Choisir pour ajouter votre propre logo ou faire glisser le logo sur l’aperçu de vidéo.

  5. Sélectionnez les éléments à inclure dans la vidéo du titre :

    • Titre

    • Date

    • Copyright

    • Description

    • Auteur

    • Organisation

  6. Cliquez sur Terminé.

  7. Choisissez Fichier > Enregistrer.
