Enregistrement d’un processus

Pour enregistrer un processus dans Podcast Composer, utilisez la commande Enregistrer ou Enregistrer sous du menu Fichier. Vous pouvez également utiliser le bouton Enregistrer qui apparaît à l’étape du résumé.

Lorsque vous enregistrez un processus, Podcast Composer génère un paquet de processus contenant toutes les informations et ressources nécessaires au traitement de ce processus : images, séquences vidéo, préréglages, modèles, outils et compositions.

Le paquet de processus comprend également des métadonnées, des informations sur le mot de passe, ainsi que le code XML définissant le processus. Il contient aussi vos informations d’authentification et le modèle de tâche Xgrid définissant les tâches à réaliser et celles qui peuvent être exécutées en parallèle.

Pour enregistrer un nouveau processus, procédez comme suit :

  1. Sélectionnez Fichier > Enregistrer (Commande + S) ou appuyez sur le bouton Enregistrer qui apparaît à l’étape du résumé.

  2. Dans la zone de dialogue Enregistrer sous, saisissez le nom du paquet de processus et indiquez l’emplacement où le stocker.

    Le nom que vous saisissez dans ce champ ne correspond pas nécessairement à celui que vous avez indiqué à l’étape des informations.

  3. Cliquez sur Enregistrer.

    Pour enregistrer les modifications apportées à un paquet de processus existant, sélectionnez Fichier > Enregistrer (Commande + S) ou appuyez sur le bouton Enregistrer qui apparaît à l’étape du résumé.

    Pour modifier le nom d’un paquet de processus, sélectionnez Fichier > Enregistrer sous (Commande + Maj + S).

  4. Saisissez le nouveau nom du paquet de processus dans la zone de dialogue Enregistrer sous.
