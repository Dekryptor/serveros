Envoi de contenu vers un processus

Vous pouvez envoyer votre podcast vers un autre processus personnalisé ou une autre chaîne de processus, afin de réaliser des tâches supplémentaires que vous ne pouvez pas configurer dans Podcast Composer.

Vous pouvez ajouter des destinations de processus.

Pour envoyer du contenu vers un processus :

  1. Dans Podcast Composer, accédez à l’étape de publication (choisissez Étapes > Publier ou appuyez sur Commande + 5).

  2. Cliquez sur le bouton d’ajout (+) et sélectionnez Processus.

    Une zone de dialogue apparaît.

  3. Cliquez sur le bouton (+) pour ajouter des fichiers multimédias à la liste de fichiers multimédias à envoyer vers Podcast Producer.

  4. Dans le champ Processus, saisissez le nom du processus que Podcast Producer va utiliser pour traiter le contenu transmis.

  5. Dans les champs Nom d’utilisateur et Mot de passe, saisissez les références de l’utilisateur nécessaires pour envoyer la tâche vers Podcast Producer.

  6. Cliquez sur Terminé.

  7. Choisissez Fichier > Enregistrer.
