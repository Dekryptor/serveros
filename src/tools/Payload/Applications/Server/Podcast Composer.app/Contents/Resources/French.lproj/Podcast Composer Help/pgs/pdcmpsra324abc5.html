Prévisualisation du podcast

L’étape de l’édition propose une fenêtre d’aperçu active dans laquelle vous pouvez prévisualiser la vidéo QuickTime générée à ce stade. Pendant que cette fenêtre est ouverte, les changements que vous apportez à l’étape de l’édition sont répercutés dans l’aperçu actif.

Pour ouvrir la fenêtre d’aperçu active :

  1. Dans Podcast Composer, accédez à l’étape de l’édition (choisissez Étapes > Édition ou appuyez sur Commande + 3).

  2. Cliquez sur le bouton Aperçu.

    Pour afficher une fenêtre d’aperçu active plus grande, tout en maintenant la touche Option enfoncée, cliquez sur le bouton Aperçu.
