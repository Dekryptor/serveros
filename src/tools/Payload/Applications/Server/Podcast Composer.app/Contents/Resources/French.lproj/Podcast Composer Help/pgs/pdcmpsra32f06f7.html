Ajout de formats Compressor

Podcast Composer prend en charge les formats d’encodage créés à l’aide de l’application Compressor. Cependant, vous devez d’abord installer Compressor sur votre ordinateur. Pour en savoir plus sur la façon d’installer Compressor, voir la documentation relative à Compressor.

Pour ajouter des formats d’encodage Compressor lors de l’étape d’exportation :

  1. Dans Podcast Composer, accédez à l’étape de l’exportation (choisissez Étapes > Exporter ou appuyez sur Commande + 4).

  2. Cliquez sur le bouton d’ajout (+).

  3. Dans le menu local, choisissez Compressor.

    Un encodeur Compressor est ajouté à l’étape d’exportation et une zone de dialogue apparaît.

  4. Dans le menu local Encodeur Compressor, choisissez l’encodeur que vous souhaitez utiliser.

    Ce menu affiche les encodages personnalisés que vous avez créés avec Compressor.

  5. Pour utiliser le système de traitement distribué Apple Qmaster afin d’accélérer le processus d’encodage, saisissez les informations requises.

    Apple Qmaster accélère l’encodage en divisant le nombre total d’images d’une tâche en segments plus petits. Chacun des ordinateurs de traitement travaille alors sur un segment différent. Comme les nœuds fonctionnent en parallèle, la tâche se termine plus tôt qu’avec un seul ordinateur.

    Pour en savoir plus sur le système de traitement distribué Apple Qmaster, voir Configuration du processus distribué Apple Qmaster 2 et Compressor 2.

  6. Cliquez sur Terminé.

  7. Choisissez Fichier > Enregistrer.
