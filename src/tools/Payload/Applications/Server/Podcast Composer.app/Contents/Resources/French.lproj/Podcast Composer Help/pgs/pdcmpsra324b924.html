Ajout et configuration d’une vidéo d’introduction

Par défaut, le film fixe de l’étape d’édition inclut une vidéo d’introduction. Si tel n’est pas le cas, cliquez sur le bouton d’ajout (+) et sélectionnez Introduction pour ajouter la vidéo.

Pour supprimer l’icône Introduction du film fixe, sélectionnez-la et appuyez sur la touche Retour arrière.

Pour configurer la vidéo d’introduction :

  1. Dans Podcast Composer, accédez à l’étape de l’édition (choisissez Étapes > Édition ou appuyez sur Commande + 3).

  2. Double-cliquez sur l’icône de la vidéo d’introduction.

  3. Dans le menu local Séquence d’introduction, sélectionnez une vidéo d’introduction par défaut fournie avec Podcast Composer.

    Pour choisir une vidéo personnalisée, sélectionnez Choisir, recherchez la vidéo, sélectionnez-la, puis cliquez sur Ouvrir. Vous pouvez également faire glisser la vidéo vers l’aperçu de la vidéo dans le menu local.

    Vous pouvez également faire glisser une vidéo directement sur l’icône de la vidéo d’introduction sans ouvrir le menu local.

  4. Cliquez sur Terminé.

  5. Choisissez Fichier > Enregistrer.
