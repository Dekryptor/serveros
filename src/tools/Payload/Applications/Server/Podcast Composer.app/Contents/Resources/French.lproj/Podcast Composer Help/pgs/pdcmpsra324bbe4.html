Ajout et configuration d’une vidéo de sortie

Par défaut, le film fixe de l’étape d’édition inclut une vidéo de sortie. Si tel n’est pas le cas, cliquez sur le bouton d’ajout (+) et sélectionnez Quitter pour ajouter la vidéo.

Pour configurer la vidéo de sortie :

  1. Dans Podcast Composer, accédez à l’étape de l’édition (choisissez Étapes > Édition ou appuyez sur Commande + 3).

  2. Double-cliquez sur l’icône de la vidéo de sortie.

  3. Dans le menu local Fermer la séquence, sélectionnez une vidéo de sortie par défaut fournie avec Podcast Composer.

    Pour choisir une vidéo personnalisée, sélectionnez Choisir dans le menu local, recherchez la vidéo, sélectionnez-la, puis cliquez sur Ouvrir. Vous pouvez aussi ajouter une vidéo de sortie en faisant glisser la vidéo dans la zone d’aperçu.

  4. Cliquez sur Terminé.

  5. Choisissez Fichier > Enregistrer.
