Configuration de la source unique

Si vous souhaitez que le processus crée un podcast à partir d’une seule vidéo QuickTime, sélectionnez et configurez l’option Source unique à l’étape de l’importation.

Pour configurer l’option Source unique à l’étape de l’importation :

  1. Dans Podcast Composer, accédez à l’étape de l’importation (choisissez Étapes > Importer ou appuyez sur Commande + 2).

  2. Double-cliquez sur Source unique ou positionnez la souris au-dessus de l’icône Source unique et cliquez sur le bouton Informations dans le coin inférieur droit de l’icône.

    Si l’option Source unique est sélectionnée, vous pouvez appuyer sur Entrée pour afficher la zone de dialogue Informations.

  3. Dans le menu local « Type de source unique », sélectionnez l’une des options suivantes pour spécifier le type d’enregistrement dont les utilisateurs de Transfert de podcast peuvent se servir avec ce processus :

    • Quelconque : la vidéo d’entrée peut être un enregistrement vidéo, un enregistrement d’écran ou un enregistrement strictement audio.

    • Dispositif vidéo seulement : la vidéo d’entrée est un enregistrement vidéo.

    • Enregistreur d’écran seulement : la vidéo d’entrée est un enregistrement d’écran.

    • Audio seulement : la vidéo d’entrée est un enregistrement strictement audio.

  4. Si vous voulez que le processus détecte automatiquement les changements dans la vidéo d’entrée et utilise ces informations pour affecter des chapitres dans le podcast généré, sélectionnez « Génération automatique des chapitres ».

    Dans le cas d’un enregistrement de l’écran, le processus insère un marqueur de chapitre dans la vidéo lorsqu’il détecte une transition entre diapositives ou lorsque aucun changement ne se produit dans la vidéo à un certain moment. Cela s’avère utile si vous enregistrez un écran ou pour toute vidéo qui inclut des présentations.

    L’outil utilisé par le processus pour ajouter des marqueurs de chapitre détecte la stabilité dans une vidéo en comparant les images et une fois que l’outil détecte une vidéo stable, il la considère comme un chapitre. Ensuite, lorsqu’une nouvelle vidéo succède à la première, l’outil considère qu’il s’agit d’une transition.

    L’utilisation de marqueurs de chapitre est recommandée pour les enregistrements d’écran, mais pas pour les enregistrements vidéo et audio. Cependant, il existe des cas où l’ajout de ces marqueurs est utile, notamment lorsque la vidéo source est destinée à un enregistrement d’écran capturé à partir du flux VGA ou DVI d’un écran d’ordinateur. L’enregistrement d’écran à l’aide de Transfert de podcast fonctionne uniquement sur un ordinateur exécutant Mac OS X 10.5 ou ultérieur.

  5. Cliquez sur Terminé.

  6. Choisissez Fichier > Enregistrer.
