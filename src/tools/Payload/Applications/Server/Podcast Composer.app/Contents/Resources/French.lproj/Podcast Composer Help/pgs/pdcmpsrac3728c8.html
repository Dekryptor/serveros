Présentation de Podcast Composer

Podcast Composer permet de simplifier et d’accélérer la création de processus, grâce à une interface utilisateur graphique très simple. Vous n’avez qu’à indiquer les informations sur le processus, sans avoir besoin d’écrire du code XML ni vous soucier de trouver un emplacement de stockage pour les ressources et les informations d’authentification.
