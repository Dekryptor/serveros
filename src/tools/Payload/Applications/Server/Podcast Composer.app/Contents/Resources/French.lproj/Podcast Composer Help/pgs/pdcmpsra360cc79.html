Ouverture d’un processus à distance

Pour apporter des modifications à un processus qui se trouve sur un serveur Podcast Producer, utilisez la commande Fichier > Ouvrir élément distant dans Podcast Composer.

Pour ouvrir un processus à distance dans Podcast Composer, procédez comme suit :

  1. Sélectionnez Fichier > Ouvrir élément distant (Commande + Maj + O).

  2. Dans le champ Serveur de la fenêtre du navigateur de processus, saisissez l’adresse du serveur Podcast Producer, puis cliquez sur Se connecter.

  3. Dans les champs Nom et Mot de passe, saissisez le nom d’utilisateur et le mot de passe qui vous ont été attribués par l’administrateur Podcast Producer.

  4. Cliquez à nouveau sur Se connecter.

  5. Choisissez le processus à modifier dans la liste des processus.
