Adding and Configuring an Exit Movie

By default, the film strip in the Edit stage includes an exit movie. If an exit movie isnâ€™t present, click the Add (+) button and select Exit to add the movie.

To configure the exit movie:

  1. In Podcast Composer, go to the Edit stage (choose Stages > Edit or press Command-3).

  2. Double-click the icon of the exit movie.

  3. From the Exit Movie pop-up menu, choose a default exit movie that comes with Podcast Composer.

    To select a custom movie, select Choose from the pop-up menu, browse for the movie, select it, and click Open. You can also add an exit movie by dragging the movie onto the live preview box.

  4. Click Done.

  5. Choose File > Save.
