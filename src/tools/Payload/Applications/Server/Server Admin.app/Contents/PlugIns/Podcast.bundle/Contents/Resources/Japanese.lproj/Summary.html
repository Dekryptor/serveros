Podcast プロデューサーが構成されて開始しています。ここではその使いかたを紹介します。

Podcast キャプチャ

ビデオ、オーディオ、および両方のソースのエピソードを Podcast キャプチャ（Mac OS X 10.5 以降に付属しています）または Podcast キャプチャ Web アプリケーションで取り込んで送信しましょう。

Podcast ライブラリ

Podcast エピソードをブラウズ、ストリーミング、ダウンロードしましょう。ライブラリ内の Podcast フィードを iTunes で登録すると、自動的に iPod、iPhone、または Apple TV と同期されます。

Podcast Composer

Podcast の外観を独自のウォーターマークやイントロビデオを追加するなどしてカスタマイズしましょう。Podcast Composer を使ってワークフローを iTunes U、Wiki、およびその他の公開先と統合することもできます。
