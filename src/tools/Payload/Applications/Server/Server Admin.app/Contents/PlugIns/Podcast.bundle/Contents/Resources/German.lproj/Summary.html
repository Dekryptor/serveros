Podcast-Produzent wurde konfiguriert und gestartet. Hier finden Sie eine Übersicht.

Podcast-Aufzeichnung

Verwenden Sie Podcast-Aufzeichnung (in Mac OS X 10.5 oder neuer enthalten) oder das Webprogramm „Podcast-Aufzeichnung“ zum Aufnehmen und Senden von Video, Audio oder Episoden mit doppelter Quelle.

Podcast-Mediathek

Durchsuchen, Streaming und Download von Podcast-Episoden. Abonnieren Sie mit iTunes einen beliebigen Podcast-Feed aus der Mediathek und synchronisieren Sie anschließend Ihren iPod, Ihr iPhone oder Ihr Apple TV damit.

Podcast-Composer

Gestalten Sie Ihre Podcasts beispielsweise mit einem individuellen Wasserzeichen oder einem Introvideo. Mit Podcst-Composer können Sie Arbeitsabläufe in iTunes U, Wikis und anderen Publikationsorten integrieren.
