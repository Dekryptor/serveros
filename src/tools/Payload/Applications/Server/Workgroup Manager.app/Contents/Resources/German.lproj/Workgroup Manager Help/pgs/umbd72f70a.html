Konfigurieren von Computern und Computergruppen

Damit Sie einen einzelnen Computer verwalten können, müssen Sie einen Computer-Account einrichten. Zum Verwalten einer Gruppe von Computern erstellen Sie eine Computergruppe, die sich aus Computer-Accounts und anderen Computergruppen zusammensetzt.

Mit dem Arbeitsgruppenmanager können Sie Computer und Computergruppen anzeigen, erstellen, bearbeiten und löschen.
