はじめに

「ワークグループマネージャ」は、クライアントコンピュータを管理するときに使用する、重要なアプリケーションです。「ワークグループマネージャ」を使用してアカウントを作成したり、環境設定を管理したりできます。

管理者のコンピュータとパスワードを構成する

「ワークグループマネージャ」を使用するには、最初に Mac OS X Server の管理ツールをインストールする必要があります。クライアントコンピュータを管理する前に、お使いのコンピュータを管理用コンピュータとして使用できるように構成し、ディレクトリ管理者アカウントを作成する必要があります。

ワークグループマネージャを使用する

Mac OS X Server ソフトウェアをインストールしてディレクトリ管理者アカウントを設定した後に、ユーザを管理するために「ワークグループマネージャ」にアクセスして使用することができます。

ワークグループマネージャを使用する
アカウントを検索する／アカウントの一覧を表示する

「ワークグループマネージャ」には、ユーザアカウント、グループアカウント、コンピュータアカウント、およびコンピュータグループを検索してそれらの一覧を表示する方法がいくつか用意されています。

アカウントを検索する／アカウントの一覧を表示する
アカウントを操作するためのショートカット

「ワークグループマネージャ」には、同じ設定を新しいアカウントまたは既存のアカウントに適用するためのショートカットが用意されています。
