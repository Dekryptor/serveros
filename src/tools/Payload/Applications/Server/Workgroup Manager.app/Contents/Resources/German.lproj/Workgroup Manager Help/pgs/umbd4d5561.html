Einführung

Der Arbeitsgruppenmanager ist das Programm, das Sie überwiegend für die Verwaltung von Client-Computern verwenden werden. Mit dem Arbeitsgruppenmanager können Sie Benutzer-Accounts anlegen und Einstellungen verwalten.

Konfigurieren von Computer und Account des Administrators

Wenn Sie den Arbeitsgruppenmanager verwenden möchten, müssen Sie erst die Verwaltungswerkzeuge für Mac OS X Server installieren. Damit Sie Client-Computer verwalten können, konfigurieren Sie zuerst einen Computer als Administratorcomputer und erstellen einen Account für den Verzeichnis-Administrator.

Konfigurieren von Computer und Account des Administrators
Verwenden des Arbeitsgruppenmanagers

Nach dem Installieren der Mac OS X Server-Software und dem Einrichten des Verzeichnis-Administrator-Accounts können Sie auf den Arbeitsgruppenmanager zugreifen und ihn für die Benutzerverwaltung einsetzen.

Verwenden des Arbeitsgruppenmanagers
Suchen und Auflisten von Accounts

Der Arbeitsgruppenmanager bietet mehrere Methoden zum Suchen und Auflisten von Benutzer-Accounts, Gruppen-Accounts, Computer-Accounts und Computergruppen.
