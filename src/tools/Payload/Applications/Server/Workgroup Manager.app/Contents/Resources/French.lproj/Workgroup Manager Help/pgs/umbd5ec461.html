Configuration des comptes d’utilisateur

Les comptes d’utilisateur permettent de donner à chaque utilisateur une identité propre sur votre réseau et de gérer les utilisateurs.

Gestionnaire de groupe de travail vous permet de consulter, de créer, de modifier et de supprimer des comptes d’utilisateur.

À propos des comptes d’utilisateur
Administration de comptes d’utilisateur

Vous pouvez consulter, créer, modifier et supprimer des comptes d’utilisateur enregistrés dans différents types de domaines de répertoire.

Administration de comptes d’utilisateur
Utilisation des préréglages

Les préréglages sont des modèles vous permettant de définir les attributs automatiquement appliqués aux nouveaux comptes d’utilisateur et de groupe.

Gestion des préréglages
Utilisation des réglages de base

Les réglages élémentaires sont un ensemble d’attributs devant être définis pour tous les utilisateurs.

Gestion des réglages élémentaires
Utilisation des autorisations

Vous pouvez attribuer à un compte d’utilisateur des prérogatives d’administration du domaine complètes ou limitées. Lorsque vous conférez des autorisations d’administration limitées, vous pouvez spécifier quels utilisateurs et quels groupes l’utilisateur est en droit d’administrer, ainsi que les actions d’administration qu’il peut appliquer à ces utilisateurs et à ces groupes.

Utilisation de privilèges
Utilisation des réglages avancés

Les réglages avancés comprennent les réglages d’ouverture de session, les mots-clés, le type de mot de passe et la possibilité de recherche dans les commentaires.

Utilisation de réglages avancés
Utilisation des réglages de groupe

Les réglages de groupe identifient les groupes dont un utilisateur est membre.

Utilisation de réglages de groupe
Utilisation des réglages de départ

Les réglages de départ spécifient les attributs du dossier de départ d’un utilisateur.

Utilisation des réglages de dossier de départ
Utilisation des réglages de courrier électronique

Pour créer un compte de service de messagerie Mac OS X Server pour un utilisateur, spécifiez les réglages de courrier électronique de ce compte d’utilisateur. Pour utiliser le compte, l’utilisateur doit configurer un client de courrier pour l’identification du nom d’utilisateur, du mot de passe, du service et du protocole de courrier que vous avez spécifiés dans les réglages du courrier.

Utilisation des réglages de courrier électronique
Utilisation des réglages de quotas d’impression

Les réglages d’impression associés à un compte d’utilisateur déterminent dans quelle mesure un utilisateur peut imprimer vers les files d’attente d’impression de Mac OS X Server.

Utilisation des réglages de quotas d’impression
Utilisation des réglages informatifs

Les attributs de cette sous-fenêtre Infos comprennent le numéro de téléphone, l’adresse de courrier électronique, ainsi que les adresses URL du journal Web et de la page Web.

Utilisation des réglages informatifs
Utilisation des réglages Windows

Aux utilisateurs de Windows sont également associés des réglages portant sur le dossier de départ Windows, un profil d’utilisateur itinérant et un script d’ouverture de sessions Windows.

Utilisation des réglages Windows
