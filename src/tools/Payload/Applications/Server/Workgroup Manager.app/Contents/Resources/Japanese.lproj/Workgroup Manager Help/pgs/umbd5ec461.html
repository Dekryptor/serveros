ユーザアカウントを設定する

ユーザアカウントによってユーザにネットワーク上の一意の ID が割り当てられるので、ユーザアカウントを使用してユーザを管理することができます。

「ワークグループマネージャ」を使用して、ユーザアカウントを表示、作成、編集、および削除できます。

ユーザアカウントについて

ユーザアカウントについては、以下のヘルプを参照してください。

ユーザアカウントについて
ユーザアカウントを管理する

さまざまな種類のディレクトリドメイン内に保管されているユーザアカウントを表示、作成、編集、および削除できます。

ユーザアカウントを管理する
プリセットを操作する

新規のユーザアカウントまたはグループアカウントに自動的に適用される属性を定義するときには、プリセットと呼ばれるテンプレートを使用します。

プリセットを操作する
基本設定を操作する

基本設定は、すべてのユーザに定義する必要のある属性の集まりです。

基本設定を操作する
アクセス権を操作する

ユーザアカウントに対して、完全なまたは制限付きのドメイン管理権限を割り当てることができます。制限付きの管理権限を割り当てるときに、そのユーザがどのようなユーザとグループを管理できるようにするか、およびそれらのユーザとグループに対してどのような制御を許可するかを選択できます。

アクセス権を操作する
詳細設定を操作する

詳細設定として、ログイン設定、キーワード、パスワードタイプ、および検索可能なコメントなどがあります。

詳細設定を操作する
グループ設定を操作する

グループ設定は、ユーザがメンバーになっているグループを識別します。

グループ設定を操作する
ホーム設定を操作する

ホーム設定では、ユーザのホームフォルダの属性を定義します。

ホーム設定を操作する
メール設定を操作する

ユーザアカウントのメール設定を指定することによって、そのユーザの Mac OS X Server メールサービス・アカウントを作成できます。ユーザがそのアカウントを使用するには、メール設定に指定されているユーザ名、パスワード、メールサービス、およびメールプロトコルを識別するメールクライアントを構成します。

メール設定を操作する
プリントクオータ設定を操作する

プリント設定は、ユーザアカウントに関連付けられていて、アクセス可能な Mac OS X Server プリントキューにユーザがプリントできるかどうかを制御します。

プリントクオータ設定を操作する
情報設定を操作する

「情報」パネルで管理されている属性は、電話番号、メールアドレス、ウェブログの URL、およびホームページの URL です。

情報設定を操作する
Windows 設定を操作する

Windows ユーザのために、Windows ホームフォルダ、ローミング・ユーザ・プロファイル、および Windows ログインスクリプトの設定が用意されています。
