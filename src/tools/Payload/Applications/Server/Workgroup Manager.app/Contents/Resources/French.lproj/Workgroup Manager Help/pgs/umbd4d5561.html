Gestionnaire de groupe de travail est la principale application permettant de gérer les ordinateurs clients. Vous pouvez ainsi vous servir de Gestionnaire de groupe de travail pour créer des comptes et gérer les préférences correspondantes.

Configuration de l’ordinateur et du compte de l’administrateur

Pour pouvoir vous servir de Gestionnaire de groupe de travail, vous devez d’abord utiliser les outils d’administration de Mac OS X Server. Avant de pouvoir gérer les ordinateurs clients, vous devez configurer un ordinateur particulier qui sera l’ordinateur administrateur et créer un compte administrateur de répertoire.

Utilisation du Gestionnaire de groupe de travail

Après avoir installé le logiciel Mac OS X Server et configuré un compte administrateur de répertoire, vous pouvez utiliser Gestionnaire de groupe de travail pour effectuer la gestion des comptes d’utilisateur.

Recherche et listage des comptes

Gestionnaire de groupe de travail vous offre plusieurs méthodes de recherche et de listage des comptes d’utilisateur, des comptes de groupe, des comptes d’ordinateur et des groupes d’ordinateurs.
