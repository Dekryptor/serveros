Getting Started

Workgroup Manager is the primary application for managing client computers. You can use Workgroup Manager to create accounts and manage preferences.

Configuring the Administrator’s Computer and Account

To use Workgroup Manager, you must first install the Mac OS X Server administration tools. Before you can manage client computers, you configure a computer for use as an administrator computer and create a directory administrator account.

Using Workgroup Manager

After you install the Mac OS X Server software and set up a directory administrator account, you can access and use Workgroup Manager for user management.
