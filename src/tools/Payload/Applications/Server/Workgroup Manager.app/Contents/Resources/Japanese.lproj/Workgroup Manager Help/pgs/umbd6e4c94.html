グループアカウントを設定する

グループアカウントを使用すると、類似した必要を持つユーザの集まりを簡単に管理できます。また、グループフォルダを作成すれば、グループメンバーがファイルを簡単に共有することができます。

「ワークグループマネージャ」を使用して、グループアカウントを表示、作成、編集、および削除できます。

グループアカウントについて
