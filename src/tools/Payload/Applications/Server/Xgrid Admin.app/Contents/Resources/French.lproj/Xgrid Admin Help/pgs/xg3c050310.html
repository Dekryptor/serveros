Client

Tout système peut devenir un client Xgrid s'il exécute Mac OS X v10.4 ou supérieur et est connecté à un contrôleur Xgrid via un réseau. En règle générale, le client ne peut se connecter qu'à un seul contrôleur.

En fonction de la configuration du contrôleur, le client doit fournir un mot de passe pour être identifié ou être authentifié via Kerberos (authentification unique) avant de soumettre une tâche à la grille.

Un utilisateur soumet une tâche au contrôleur à partir d'un système qui exécute le logiciel client Xgrid, généralement un outil de type ligne de commande et une application Terminal. La tâche peut spécifier le contrôleur ou utiliser le multicast DNS (mDNS) pour découvrir dynamiquement le premier contrôleur disponible. Une fois la tâche exécutée, le contrôleur notifie le client et celui-ci peut récupérer les résultats de la tâche.
