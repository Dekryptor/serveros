Suppression d'un travail

Vous pouvez supprimer une tâche dans Xgrid Admin.

Pour supprimer une tâche :

  1. Ouvrez Xgrid Admin.

  2. Dans la liste Contrôleurs et grilles, sélectionnez le contrôleur.

  3. Cliquez sur Tâches.

  4. Sélectionnez le travail à supprimer.

  5. Cliquez sur le bouton Supprimer (–) en dessous de la liste des tâches.
