Xgrid エージェントを管理する

エージェントを表示、追加、または削除するときには、「Xgrid Admin」を使用します。また、エージェントの状況を表示するときにも、「Xgrid Admin」の状況インジケータを使用します。詳しくは、以下を参照してください：

  • エージェントのリストを表示する

  • エージェントを追加する

  • エージェントを削除する

  • Xgrid Admin の状況インジケータ

「サーバ管理」には、Xgrid サービスを 1 台のサーバで、または Xserve システムのラックで有効にする簡単なインターフェイスがありますが、Mac OS X v10.3 以降が動作するデスクトップコンピュータで Xgrid を構成することはできません。

ボランティアに頼ってデスクトップエージェントを提供している場合は、「システム環境設定」の「共有」パネルで Xgrid を有効にする手順を送信することができます。

ボランティアが Mac OS X v10.3 を使用している場合は、まず Xgrid Agent for Mac OS X v10.3 をダウンロードしてから、「システム環境設定」の「Xgrid」パネルを使用する必要があります。Xgrid Agent for Mac OS X v10.3 は www.apple.com/jp/server/macosx/technology/xgrid.html でダウンロードできます。

コンピュータのグループを管理していて、Xgrid を使用してそれらのコンピュータをグループに参加させたい場合は、次の方法を使用できます：

  • Apple Remote Desktop

  • SSH

  • NetBoot または NetInstall
