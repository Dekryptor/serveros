Vue d'ensemble de Xgrid Admin

Xgrid Admin est un outil destiné à contrôler les grilles et à gérer des agents et des tâches. Vous pouvez ajouter des contrôleurs et des agents pour contrôler et spécifier les agents qui n'ont pas rejoint de grille. Vous pouvez également utiliser Xgrid Admin pour interrompre, arrêter ou redémarrer des travaux.

Vous pouvez administrer des grilles de calcul à l'aide de Xgrid Admin. Une grille de calcul est un groupe fixe d'agents avec une file d'attente dédiée.

Un contrôleur peut comporter plusieurs grilles, mais un agent ne peut appartenir qu'à une seule grille. Vous ne pouvez pas déplacer un agent d'une grille à l'autre pendant l'exécution d'une tâche ou d'une sous-tâche.

Pour en savoir plus, consultez Utilisation de Xgrid Admin.
