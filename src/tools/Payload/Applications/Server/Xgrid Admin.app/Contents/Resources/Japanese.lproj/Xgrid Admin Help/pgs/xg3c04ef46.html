Xgrid と計算グリッドについて

Xgrid を利用すれば、Mac システムを一時的にグループ化して、低コストのスーパーコンピュータにすることができます。Xgrid は個人で活動している研究者、共同制作者、およびアプリケーション開発者にとって理想的です。たとえば、次のようなことができます：

  • 科学者は、Xserve クラスタシステム上で生物学データベースを検索できます。

  • 技術者は、ワークグループのデスクトップ上で有限要素解析を実行できます。

  • アニメータは、会社のさまざまな場所にある Mac システムを使って、イメージをレンダリングできます。

  • 研究チームは、同僚や一般人に協力を求めてインターネット規模のボランティアグリッドを構成し、長期間の科学計算を実行できます。

  • 大量の CPU 計算を行う必要がある人は、複数のコンピュータ間で 1 つのジョブを同時に実行して、スループットと反応速度を劇的に向上させることができます。

Xgrid の機能を Mac OS X Server に統合させることによって、システム管理者は会社全体の Mac システム上で Xgrid をすばやく有効にし、コストをかけなくても待機状態の CPU サイクルを生産的なクラスタに変えることができます。
