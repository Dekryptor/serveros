Administration des tâches Xgrid

Vous utilisez Xgrid Admin pour gérer les tâches une fois qu'elles sont envoyées par le client.

Remarque :  vous ne pouvez pas déplacer les tâches entre les grilles.

  • Visualisation d'une liste de tâches

  • Interruption d'une tâche

  • Répétition ou redémarrage d'un travail

  • Suppression d'un travail

  • Indicateurs d'état de Xgrid Admin
