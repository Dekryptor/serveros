Xgrid Admin の状況インジケータ

「Xgrid Admin」の状況インジケータは、色付きの小さいランプによってコントローラ、エージェント、およびジョブの状況を示すようになっています。 インジケータの色に応じて、次の状況を示します：

  • 無色 = コントローラまたはエージェントはオフラインです。ジョブは保留中です

  • グレイ = ジョブは送信中です

  • 緑 = コントローラは接続されています。エージェントは動作中です。ジョブは実行中です

  • 黄 = エージェントは使用可能ですが実行中ではありません

  • 赤 = エージェントは使用不可です。ジョブは失敗したかキャンセルされました

  • 青 = ジョブは完了しました
