Ajout d'un contrôleur Xgrid

Vous utilisez Xgrid Admin pour ajouter un contrôleur Xgrid à la liste Contrôleurs et grilles.

Pour ajouter un contrôleur Xgrid à la liste de contrôle :

  1. Ouvrez Xgrid Admin.

  2. Cliquez sur le bouton Ajouter (+) et sélectionnez Ajouter un contrôleur.

  3. Dans le menu local, choisissez un contrôleur ou saisissez son nom et cliquez sur Connexion.

  4. Si nécessaire, sélectionnez l'option d'authentification correcte, saisissez le mot de passe, puis cliquez sur OK.
