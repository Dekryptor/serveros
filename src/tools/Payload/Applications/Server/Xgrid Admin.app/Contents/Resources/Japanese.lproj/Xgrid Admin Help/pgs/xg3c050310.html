ホーム
クライアント

Mac OS X v10.4 以降が動作していて、ネットワークで Xgrid コントローラシステムに接続されていれば、どんなシステムでも Xgrid クライアントとして使用できます。一般に、クライアントは 1 つのコントローラにのみ接続できます。

コントローラの構成方法によっては、クライアントは、グリッドにジョブを送信する前に、パスワードを入力するか、Kerberos（シングルサインオン）で認証を行う必要があります。

ユーザは、Xgrid クライアントソフトウェアが動作するシステムからコントローラにジョブを送信します。通常、Xgrid クライアントソフトウェアは、「ターミナル」アプリケーションを使ってアクセスするコマンドラインツールです。ジョブでは、コントローラを指定するか、マルチキャスト DNS（mDNS）を使用して、最初に利用可能なコントローラを動的に検出できます。ジョブが完了すると、コントローラはクライアントに通知し、クライアントはジョブの結果を取得できます。
