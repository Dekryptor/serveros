Benutzer und Gruppen (Xsan)

Die im Fenster "Benutzer und Gruppen" für einen primären Xsan-Metadaten-Controller gewählte Option gibt an, wie Sie die Benutzer- und Gruppen-Accounts verwalten, mit denen Benutzer von SAN-Computern identifiziert werden und mit denen bestimmt wird, auf welche Dienste sie zugreifen dürfen. Sie verwenden die Benutzer- und Gruppen-Accounts, um den Zugriff auf Dateien und Ordner im SAN zu steuern, indem Sie Benutzern und Gruppen Zugriffsrechte zuweisen.

Optionen zum Verwalten von Benutzern und Gruppen
Benutzer und Gruppen mit Xsan-Admin verwalten

Konfiguriert den primären Metadaten-Controller als Open Directory-Master, sodass er über einen eigenen Verzeichnisdienst für Benutzer- und Gruppen-Accounts verfügt. Sie erstellen und verwalten Benutzer nach der Konfiguration mit Xsan-Admin. Xsan-Admin konfiguriert SAN-Computer automatisch zum Herstellen der Verbindung zum Open Directory-Dienst.

Diese Option ist empfehlenswert, wenn Sie keinen Verzeichnisserver haben und nicht mehr als 20 SAN-Benutzer unterstützen müssen. Diese Benutzer verfügen über lokale Benutzerordner auf ihren Computern, nicht über Netzwerkbenutzerordner auf dem Server.

Wichtig: Wenn Sie diese Option auswählen, verwenden Sie unbedingt Xsan-Admin und nicht das Programm "Servereinstellungen", um Benutzer und Gruppen zu erstellen. Xsan-Admin erstellt einen Benutzerordner für einen neuen Benutzer-Account, das Programm "Servereinstellungen" tut dies nicht.
Wichtig:Sie können diese Option nur auswählen, wenn Sie diesen Konfigurationsassistenten zum Konfigurieren von Mac OS X Server auf dem primären Metadaten-Controller verwenden. Später können Sie diese Option nicht mehr konfigurieren.
Benutzer und Gruppen von einem Open Directory-Server verwenden

Ermöglicht Ihnen, den primären Metadaten-Controller mit einem vorhandenen Open Directory-Server zu verbinden, dessen DNS-Name oder IP-Adresse Sie im nächsten Fenster angeben. Xsan-Admin konfiguriert SAN-Computer automatisch zum Herstellen der Verbindung zum Open Directory-Server. Benutzer und Gruppen werden von einem Administrator des Open Directory-Servers verwaltet.

Später mit einem Verzeichnisserver verbinden

Ermöglicht Ihnen, die Verbindung zu einem vorhandenen Open Directory-Server bis nach der Konfiguration zu verschieben oder nach der Konfiguration eine Verbindung zu einem anderen Typ von Verzeichnisserver wie Active Directory herzustellen. Konfigurieren Sie die Verzeichnisdienstverbindungen auf SAN-Computern unter Snow Leopard im Bereich "Anmeldeoptionen" der Systemeinstellung "Benutzer". Konfigurieren Sie Verzeichnisdienstverbindungen auf SAN-Computern mit Leopard mit dem Programm "Verzeichnisdienste".

Wenn diese Optionen nicht angezeigt werden, konfigurieren Sie keine primären Xsan-Metadaten-Controller. Weitere Informationen hierzu finden Sie unter Benutzer und Gruppen.
