Copie de sauvegarde du client

Utilisez la sous-fenêtre Copie de sauvegarde du client pour libérer de l’espace disque sur le serveur en vue de sauvegarder le contenu des Mac des utilisateurs. Les utilisateurs de Snow Leopard pour lesquels Time Machine n’est pas activé dans les Préférences Système sont automatiquement invités à utiliser le serveur en tant qu’emplacement de stockage. Les autres utilisateurs doivent ouvrir la sous-fenêtre Time Machine dans Préférences Système et modifier le disque utilisé pour stocker les copies de sauvegarde.

Si vous sélectionnez l’option « Autoriser les utilisateurs à sauvegarder sur ce serveur », vous pouvez sélectionner un disque pour les copies de sauvegarde Time Machine des utilisateurs à partir de la liste de disques connectés au serveur.

La sous-fenêtre Copie de sauvegarde du client apparaît lors de la configuration en local et pendant la configuration à distance d’un serveur unique. Elle n’apparaît pas si vous avez désactivé le partage de fichiers dans la sous-fenêtre Services ou si cette dernière n’apparaissait pas car vous configuriez en mode manuel. La sous-fenêtre Copie de sauvegarde du client n’apparaît pas non plus si vous configurez plusieurs serveurs distants en même temps.
