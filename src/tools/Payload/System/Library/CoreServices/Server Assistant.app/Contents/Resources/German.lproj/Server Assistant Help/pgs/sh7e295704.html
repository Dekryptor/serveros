Seriennummer

Geben Sie im Fenster "Seriennummer" eine Seriennummer für jeden Server ein, den Sie jetzt konfigurieren.

Sie müssen eine Mac OS X Server 10.6 Snow Leopard-Seriennummer verwenden, die mit "XSVR-106" beginnt. Die Seriennummer der Serversoftware ist auf den Materialien aufgedruckt, die zusammen mit dem Mac OS X Server-Paket bereitgestellt werden. Sofern Sie keine Standortlizenz besitzen, benötigen Sie für jeden Server eine eindeutige Seriennummer. Sie können nicht die Seriennummer einer früheren Version von Mac OS X Server verwenden, um Snow Leopard Server zu registrieren.

Wenn Sie eine Standortlizenz haben, müssen Sie die registrierten Angaben für Eigentümername und Organisation exakt so eingeben, wie von Ihrem Apple-Repräsentanten vorgegeben.

Wenn Sie mehrere Server konfigurieren, werden im Fenster "Seriennummer" die Server aufgeführt, die gültige Seriennummern haben. Die Anzahl der Server ohne Seriennummern wird unter der Liste angezeigt. Klicken Sie auf "Hinzufügen", um eine Seriennummer hinzuzufügen. Der Serverassistent wendet die Seriennummer auf einen Server an, der derzeit keine Seriennummer besitzt. Sie können auch eine Standortlizenz hinzufügen, die dann vom Serverassistenten auf mehrere Server angewendet wird.
