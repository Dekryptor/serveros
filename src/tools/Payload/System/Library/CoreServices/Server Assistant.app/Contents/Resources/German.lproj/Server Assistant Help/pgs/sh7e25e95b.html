Dienst "Dateifreigabe"

Der Mac OS X Server-Dienst "Dateifreigabe" ermöglicht Ihren Gruppenmitgliedern den Zugriff auf gemeinsam genutzte Ordner und die Speicherung privater Dateien auf dem Server. Der Zugriff auf Dateien und gemeinsam genutzte Ordner ist von Macintosh-, Windows- und UNIX-Computern aus möglich. Er erfolgt mithilfe nativer Dateiprotokolle wie AFP und SMB, sodass keine spezielle Software nötig ist. Dateiserver mit Mac OS X Server werden im Ordner "Netzwerkumgebung" von Windows in gleicher Weise angezeigt wie Windows-Dateiserver.
