Xsan-Controller-Typ

Definieren Sie diesen Server im Fenster "Xsan-Controller-Typ" als primären Metadaten-Controller oder als Ersatz-Metadaten-Controller für Ihr SAN. Die Controller verwalten Volume-Metadaten sowie ein Dateisystemjournal und steuern den gleichzeitigen Zugriff auf Dateien. Die Metadaten enthalten Informationen, darunter solche zum Speicherort von Dateien und zu den neuen Dateien zugewiesenen Teilen des verfügbaren Speichers.

Ein SAN muss einen primären Controller enthalten. Zur Gewährleistung der Volumeverfügbarkeit sollte ein SAN auch einen oder mehr Ersatz-Controller enthalten. Fällt der primäre Controller aus, übernimmt ein Ersatz-Controller seine Aufgabe.
