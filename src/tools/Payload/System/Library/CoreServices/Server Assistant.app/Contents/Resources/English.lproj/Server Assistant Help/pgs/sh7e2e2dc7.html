Time Zone

Use the Time Zone pane to select the local time zone for the server.

Choose a city by clicking the map, typing in the Closest City field, or using the pop-up menu. If the city your server resides in isn’t found, select a city that’s in the same time zone.

To change the network time server, click the Edit button. Select “Use a network time server.” Identify the network time server to use by choosing it from the NTP Server pop-up menu or entering an IP address or DNS name. For reliable operation of your server, Apple recommends that you keep your server’s clock accurate by synchronizing it with a network time server.
