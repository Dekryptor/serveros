Ihren Server übertragen

Wählen Sie im Fenster "Ihren Server übertragen" das Volume aus, das die Daten und Einstellungen enthält, die Sie von einem vorhandenen Server oder einer Time Machine-Sicherung eines Servers übertragen wollen. Das Volume mit den zu migrierenden Informationen muss sich im Server befinden oder an einen FireWire- oder USB-Anschluss des zu konfigurierenden Servers angeschlossen sein. Wenn Sie von einem anderen Server migrieren, muss dieser Server im FireWire-Festplattenmodus gestartet werden und mit einem FireWire-Anschluss des zu konfigurierenden Servers verbunden sein. Wenn das nicht möglich ist, müssen Sie den Serverassistenten beenden (mit der Tastenkombination "Befehl-Q"), das Volume des vorhandenen Servers auf ein Wechsellaufwerk kopieren, das Sie an den zu konfigurierenden Server anschließen können. Starten Sie dann den Server, um die Konfiguration zu wiederholen.

Sie können von einem Server mit Mac OS X Server 10.4.11 oder der letzten Aktualisierung von Mac OS X Server 10.5 migrieren. Sie können auch von einer Time Machine-Sicherung oder einer Standardkonfiguration oder einer Arbeitsgruppenkonfiguration der letzten Aktualisierung von Mac OS X Server 10.5 migrieren.

Der Migrationsprozess beginnt sofort, wenn Sie auf "Übertragen" klicken.
