Ajustement des options IPv6

IPv6 est une nouvelle version du protocole Internet. Actuellement, IPv6 est essentiellement utilisé par des instituts de recherche. La plupart des ordinateurs n’ont pas besoin de configurer ou d’utiliser IPv6.

Le principal avantage d’IPv6 est qu’il fait passer la taille de l’adresse de 32 bits (la norme IPv4 actuelle) à 128 bits. Une adresse de 128 bits est suffisante pour prendre en charge une très grande quantité d’adresses, malgré l’insuffisance de l’affectation d’adresse. Cela autorise un nombre supérieur d’adresses ou de nœuds qu’actuellement. IPv6 propose également davantage de façons de configurer l’adresse et une autoconfiguration simplifiée.

Par défaut, IPv6 est configuré automatiquement et les réglages par défaut suffisent pour la grande majorité des ordinateurs qui ont besoin d’utiliser IPv6. Cependant, si l’administrateur réseau ou le fournisseur d’accès à Internet vous a spécifiquement demandé de configurer IPv6 manuellement ou de le désactiver, suivez les instructions ci-dessous.

Les services qui prennent en charge les adresses IPv6 sont les suivants : DNS, coupe-feu, courrier électronique, partage de fichiers (protocole SMB) et web.

Pour configurer l’adressage IPv6 :

  1. Dans la liste de gauche, sélectionnez le service de connexion réseau Ethernet, AirPort ou FireWire dont vous voulez modifier les réglages IPv6.

  2. Cliquez sur IPv6, choisissez une méthode dans le menu local Configurer IPv6 ; si vous sélectionnez Manuellement, saisissez les informations de configuration à l’aide des données suivantes :

    • Adresse IPv6 :  
      Un numéro à huit sections généralement écrit sous la forme 0000:0000:0000:0000:0000:0000:0000:0000.
    • Routeur :  
      L’adresse IPv6 du routeur ou de la passerelle sur le réseau local du serveur.
    • Longueur du préfixe :  
      Le nombre de bits significatifs de l’adresse IPv6 servant à identifier le réseau.
