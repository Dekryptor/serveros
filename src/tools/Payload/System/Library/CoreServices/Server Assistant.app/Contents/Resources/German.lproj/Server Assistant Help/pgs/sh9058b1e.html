AirPort-Verwaltung

Im Fenster "AirPort-Verwaltung" können Sie die Verwendung von Mac OS X Server zum Verwalten einer AirPort Extreme-Basisstation (802.11n) oder einer Time Capsule aktivieren, sodass Ihr lokales Netzwerk geschützt und gleichzeitig der Zugriff auf einige Dienste aus dem Internet erlaubt wird. Gehen Sie wie folgt vor, um die AirPort-Verwaltung zu aktivieren:

  • Sie müssen das Kennwort für die Basisstation oder Time Capsule (nicht das Kennwort für das drahtlose Netzwerk) eingeben.

  • Wenn Ihr Server eine Verbindung zu mehr als einer AirPort Extreme-Basisstation oder Time Capsule herstellen kann, müssen Sie den Namen der Station oder Time Capsule kennen, die Ihr Server verwalten soll.

Nach der anfänglichen Konfiguration können Sie im Programm "Servereinstellungen" einzelne Dienste angeben, die außerhalb Ihres lokalen Netzwerks zugänglich sein sollen. Mac OS X Server konfiguriert Ihre AirPort Extreme-Basisstation oder Time Capsule so, dass eingehende Anforderungen für diese Dienste an Ihren Server durchgereicht werden.

Wenn Ihr Server über Ihre AirPort Extreme-Basisstation oder Time Capsule Benutzern, die über eine eigene AirPort-Basisstation, eine eigene Time Capsule oder einen anderen für die gemeinsame Nutzung der Internetverbindung konfigurierten Router verfügen, einen VPN-Dienst zur Verfügung stellt, darf die IP-Adresse Ihres Servers nicht mit den gleichen drei Werten beginnen wie die IP-Adressen der VPN-Benutzer, etwa "10.0.1" oder "192.168.1". Weitere Informationen zum Vermeiden von IP-Adresskonflikten finden Sie im Handbuch Einführung, das zum Lieferumfang von Mac OS X Server gehört.

Der Fenster "AirPort-Verwaltung" wird angezeigt, wenn die primäre Netzwerkschnittstelle Ihres Servers über eine AirPort Extreme-Basisstation oder Time Capsule mit dem Internet verbunden ist. Außerdem muss die Option "Verbindung gemeinsam nutzen" Ihrer AirPort Extreme-Basisstation oder Time Capsule auf "Eine öffentl. IP-Adresse nutzen" (d. h. eine Internetverbindung) eingestellt sein.

Das Fenster "AirPort-Verwaltung" wird bei der entfernten Konfiguration mehrerer Server nicht angezeigt.
