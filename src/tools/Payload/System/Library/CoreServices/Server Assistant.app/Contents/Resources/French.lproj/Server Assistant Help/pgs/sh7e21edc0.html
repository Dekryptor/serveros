Installation

La sous-fenêtre Installation s’affiche lors de l’installation de Mac OS X Server sur le serveur distant. Elle vous permet de contrôler l’avancement de la procédure d’installation.

Une fois l’installation terminée, le serveur distant redémarre automatiquement. L’Assistant du serveur attend que vous lui indiquiez ce que vous souhaitez faire.

  • Pour installer Mac OS X Server sur un autre ordinateur distant, cliquez sur Plus d’options pour accéder à la sous-fenêtre Bienvenue.

  • Pour configurer Mac OS X Server à distance, cliquez sur Plus d’options pour accéder à la sous-fenêtre Bienvenue.

  • Pour quitter l’Assistant du serveur, cliquez sur Quitter.
