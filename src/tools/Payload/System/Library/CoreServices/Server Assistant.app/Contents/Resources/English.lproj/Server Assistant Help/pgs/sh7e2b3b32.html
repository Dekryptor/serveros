Services

Use the Services pane to specify which services your server will initially provide to users’ computers. If the server has more than one disk or a partitioned disk with multiple volumes, you can also choose which one the service data will be stored on.

For information about the services, see the following topics:

About Address Book Service
About Calendar Service
About File Sharing Service
About Instant Messaging Service
About Mail Service
About Web Services
Important:  The server’s firewall service isn’t configured to block access to any services from outside your network unless you selected the option to configure the server as an Internet gateway in an earlier pane. (The gateway option appears only if the server has at least two Ethernet ports.) If the server is not protected by a separate Internet router, gateway, or firewall, you should turn on firewall protection in the Firewall pane of Server Preferences or Server Admin after finishing initial setup.

After setup is complete, you can change service settings with the Server Preferences application. If you need to change advanced service settings or set up services that aren’t listed here, you can use the Server Admin application.

The Services pane doesn’t appear if you select Configure Manually in the “Users and Groups” pane and then don’t connect to a directory server and don’t create an Open Directory master in the “Connect to a Directory Server” pane. The Services pane also doesn’t appear if you’re setting up Xsan on a server.
