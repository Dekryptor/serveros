Vorhandenen Server übertragen?

Im Fenster "Vorhandenen Server übertragen?" können Sie Daten und Einstellungen automatisch von einem vorhandenen Server migrieren.

  • Wenn Sie die Migration überspringen wollen, wählen Sie "Neuen Server konfigurieren" und klicken Sie auf "Fortfahren".

  • Wenn Sie die Migration durchführen wollen, wählen Sie "Die Informationen von einem bestehenden Server übertragen" und klicken Sie auf "Fortfahren". Im nächsten Fenster wählen Sie das Volume aus, von dem die Migration durchgeführt werden soll.

Wichtig: Sie können nur bei der Serverkonfiguration eine automatische Migration durchführen. Nach der Konfiguration ist kein Migrationsassistent mehr verfügbar.

Sie können von einem Server mit Mac OS X Server 10.4.11 oder der letzten Aktualisierung von Mac OS X Server 10.5 migrieren. Sie können auch von einer Time Machine-Sicherung einer Standardkonfiguration oder Arbeitsgruppenkonfiguration der letzten Aktualisierung von Mac OS X Server 10.5 migrieren. Die Quelle, von der Sie migrieren, ist von der Migration selbst nicht betroffen.

Der Server, von dem Sie migrieren, muss den gleichen DNS-Namen und die gleiche IP-Adresse haben, die Sie auch für den zu konfigurierenden Server verwenden wollen. Sie können die IP-Adresse des neuen Servers nach der Konfiguration ändern. Hierzu verwenden Sie die Systemeinstellung "Netzwerk".

Das Fenster "Vorhandenen Server übertragen?" wird bei der lokalen und bei der entfernten Konfiguration eines Servers angezeigt. Er wird bei der entfernten Konfiguration mehrerer Server nicht angezeigt.

Weitere Informationen zum Migrieren von einem vorhandenen Server finden Sie auf der Website für Mac OS X Server-Ressourcen unter der Adresse www.apple.com/de/server/macosx/resources/.
