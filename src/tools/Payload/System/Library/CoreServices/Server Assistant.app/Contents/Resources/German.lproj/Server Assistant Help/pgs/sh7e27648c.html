Verbinden mit einem Verzeichnisserver

Geben Sie im Fenster "Mit einem Verzeichnisserver verbinden" den Verzeichnisserver an, von dem Sie Benutzer-Accounts importieren wollen. Sie können vorhandene Benutzer-Accounts vom Open Directory-Server oder Active Directory-Server Ihres Unternehmens importieren. Ihr Server verwendet den Account-Namen, das zugehörige Kennwort und weitere Attribute, die bereits in den importierten Benutzer-Accounts vorhanden sind. Ihr Server kann die vorhandenen Attribute um die für seine Dienste erforderlichen Attribute ergänzen. Zum Beispiel können importierte Benutzer Mitglieder einer Gruppe sein, die Sie auf Ihrem Server erstellen.

Einstellungen zum Importieren von einem Verzeichnisserver
Serveradresse:

Wählen Sie den Verzeichnisserver aus dem Einblendmenü aus oder geben Sie den DNS-Namen des Verzeichnisservers ein. Wenn Sie eine Verbindung zu einem Open Directory-Server herstellen, können Sie dessen IP-Adresse eingeben.

Servertyp

Kann Open Directory oder Active Directory sein.

Wenn Sie jetzt einen Server konfigurieren (kein Profil für die automatische Konfiguration zur späteren Verwendung erstellen), wird der Servertyp automatisch bestimmt.
Benutzer-Account-Name und Kennwort

Diese Felder werden angezeigt, wenn der Verzeichnisserver eine Identifizierung erfordert.

  • Für einen Open Directory-Server können Sie den Kurznamen oder Langnamen und das Kennwort eines Standardbenutzer-Accounts eingeben. Sie müssen keinen Verzeichnisadministrator-Account verwenden.

  • Für einen Active Directory-Server können Sie den Namen und das Kennwort eines Active Directory-Administrator-Accounts oder eines Standardbenutzer-Accounts eingeben, der über das Zugriffsrecht verfügt, Workstations zu Domains hinzuzufügen.
