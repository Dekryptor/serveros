About Instant Messaging Service

Mac OS X Server provides instant messaging (IM) for Macintosh, Windows, and Linux users. Team members can brainstorm solutions, make plans, exchange URLs, or transfer files, without worrying about outsiders intercepting confidential information. Instant messaging service provides text messaging between users or among multiple users. It also facilitates direct connections between users for audio, video, and multiway audio and video sessions.

Instant messaging service works with the iChat application in Mac OS X, Google Talk, and other instant messaging software that uses the XMPP protocol, called Jabber.
