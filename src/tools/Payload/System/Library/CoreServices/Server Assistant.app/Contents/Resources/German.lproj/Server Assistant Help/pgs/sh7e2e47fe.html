Dienst "Adressbuch"

Der Dienst "Adressbuch" ermöglicht Benutzern, Kontaktinformationen in einem Adressbuch auf mehreren Computern ohne die mit LDAP verbundenen Schemabeschränkungen und Sicherheitsprobleme zu suchen. Ein Benutzer, der mehr als einen Snow Leopard-Computer hat, kann die Programme "Mail"und "Adressbuch" auf einem beliebigen dieser Computer verwenden, um Kontakte in einem Adressbuch auf Ihrem Server anzuzeigen und hinzuzufügen. Benutzer anderer Computer können mithilfe von Drittanbieterprogrammen, die zu dem offenen Standard CardDAV kompatibel sind, auf ein Adressbuch auf Ihrem Server zugreifen.
