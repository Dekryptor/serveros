FireWire ネットワーク接続サービスを構成する

サーバや別の Mac コンピュータに FireWire ポートがある場合は、FireWire ケーブルを使ってそれらを接続した後、それらの間にローカルネットワークを設定することができます。

FireWire ネットワーク接続サービスを構成するには：

  1. 左側のリストで「FireWire」を選択します。

  2. 「構成」をクリックし、ネットワーク管理者が推奨する構成方法を選択して、構成の詳細を入力します。

    ほとんどの FireWire ネットワーク環境では、「DHCP サーバを参照」を選択し、自動的に割り当てられた構成の詳細を使用すれば十分です。

    操作方法については、以下のトピックを参照してください：

    DHCP を使用してネットワーク接続サービスを構成する
    DHCP（アドレスは手入力）を使用してネットワーク接続サービスを構成する
    BootP を使用してネットワーク接続サービスを構成する
    ネットワーク接続サービスを手動で構成する

  3. FireWire ネットワークで、詳細な IPv6 アドレス設定を調整する必要がある場合は、「IPv6」ボタンをクリックします。操作方法については、「IPv6 オプションを調整する」を参照してください。

    デフォルトでは、サーバによって自動的に IPv6 アドレスが生成されます。

FireWire がネットワーク接続サービスのリストにない場合は、「アクション」（歯車）ポップアップメニューから「すべてのサービスを表示」を選択します。 それでも FireWire が表示されない場合は、サーバに動作している FireWire ポートが存在しません。
