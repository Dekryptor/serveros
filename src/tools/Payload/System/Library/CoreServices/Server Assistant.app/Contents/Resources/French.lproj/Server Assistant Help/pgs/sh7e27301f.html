Configuration d’un service de connexion réseau PPPoE

Vous devez créer un service de connexion réseau PPPoE si toutes les conditions suivantes sont vérifiées :

  • Vous utilisez un modem câble ou DSL pour vous connecter à Internet.

  • Votre fournisseur d’accès à Internet (FAI) requiert une connexion via PPPoE.

  • Le serveur et les autres ordinateurs du réseau possèdent une adresse IP émise par votre FAI.

Si votre modem câble ou DSL est relié à un routeur Internet, tel qu’une borne d’accès AirPort, et qu’il est configuré de façon à partager une adresse IP unique avec le serveur et les autres ordinateurs du réseau local, ne configurez pas de connexion PPPoE sur le serveur ou les autres ordinateurs, sauf si votre FAI vous l’a expressément demandé. La plupart du temps, vous configurez une connexion PPPoE sur la borne d’accès AirPort ou sur un autre routeur Internet et la connexion est automatiquement partagée.

Si vous configurez un serveur avec Xsan, vous ne pouvez pas créer de service de connexion réseau PPPoE car ce type de service n’est pas adapté à un réseau Xsan.

Pour créer et configurer un service PPPoE :

  1. Dans la liste située sur la gauche, sélectionnez le service de connexion réseau devant faire appel au protocole PPPoE : Ethernet, AirPort ou FireWire.

  2. Cliquez sur Configurer, choisissez « Créer un service PPPoE », puis attribuez un nom descriptif au service PPPoE.

  3. Sélectionnez le service PPPoE dans la liste située sur la gauche, puis saisissez le nom et le mot de passe de votre compte, fournis par votre FAI.

    Ne tapez rien dans le champ « Nom du service PPPoE », sauf en cas d’instruction spécifique de la part de votre FAI.
