Konfigurieren eines PPPoE-Netzwerkverbindungsdiensts

Sie müssen einen PPPoE-Netzwerkverbindungsdienst erstellen, wenn alle der folgenden Bedingungen zutreffen:

  • Sie verwenden ein DSL- oder Kabelmodem, um eine Internetverbindung herzustellen.

  • Ihr Internetanbieter erfordert eine PPPoE-Verbindung.

  • Der Server und die anderen Computer in Ihrem Netzwerk erhalten IP-Adressen von Ihrem Internetanbieter.

Wenn Ihr DSL- oder Kabelmodem mit einem Internet-Router wie einer AirPort-Basisstation verbunden und dafür konfiguriert ist, eine einzelne IP-Adresse mit dem Server und den anderen Computern in Ihrem lokalen Netzwerk gemeinsam zu nutzen, konfigurieren Sie keine PPPoE-Verbindung auf dem Server oder den anderen Computern, sofern Ihr Internetanbieter Sie dazu nicht auffordert. In den meisten Fällen konfigurieren Sie eine PPPoE-Verbindung auf der AirPort-Basisstation oder dem anderen Internet-Router, die dann gemeinsam genutzt wird.

Wenn Sie einen Server mit Xsan konfigurieren, können Sie keinen PPPoE-Netzwerkverbindungsdienst erstellen, weil dieser für ein Xsan-Netzwerk ungeeignet ist.

Gehen Sie wie folgt vor, um den PPPoE-Dienst zu erstellen und zu konfigurieren:

  1. Wählen Sie in der Liste links den Netzwerkverbindungsdienst aus, bei dem PPPoE verwendet werden soll: Ethernet, AirPort oder FireWire.

  2. Klicken Sie auf "Konfigurieren", wählen Sie "PPPoE-Dienst erstellen" und geben Sie dem PPPoE-Dienst einen aussagekräftigen Namen.

  3. Wählen Sie den PPPoE-Dienst in der Liste links aus und geben Sie dann den Account-Namen und das Kennwort ein, die Sie von Ihrem Internetanbieter erhalten haben.

    Geben Sie nichts in das Feld "PPPoE-Dienstname" ein, sofern Ihr Internetanbieter Sie dazu nicht aufgefordert hat.
