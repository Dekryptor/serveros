Lokales Konfigurieren

Hier finden Sie die Hilfethemen für die Fenster des Serverassistenten, die Sie beim lokalen Konfigurieren von Mac OS X Server verwenden können. Sie werden in der Reihenfolge aufgeführt, in der sie angezeigt werden. Manche Fenster werden in Abhängigkeit von der Auswahl, die Sie in den vorherigen Fenstern getroffen haben, und den Eigenschaften des Servers und Netzwerks nicht verwendet. Wenn Sie einen Xsan-Metadaten-Controller konfigurieren, verwenden Sie zusätzliche Fenster (siehe Xsan-Konfigurationsthemen).
