Dienst "Kalender"

Der Dienst "Kalender" ermöglicht und erleichtert Benutzern die gemeinsame Nutzung von Kalendern, die Zeitplanung und die Koordinierung von Terminen in Arbeitsgruppen und Kleinbetrieben ebenso wie in großen Unternehmen. Kollegen haben die Möglichkeit, gegenseitig ihre Verfügbarkeit zu prüfen, Konferenztermine zu planen und vorzuschlagen, Konferenzräume zu buchen, Projektoren zu reservieren und mehr. Der Dienst "Kalender" sendet Meeting-Einladungen, die Informationen wie eine Tagesordnung oder Aufgabenlisten umfassen können, und sortiert die Antworten.

Snow Leopard Server bietet zusätzlich Push-Benachrichtigungen, die Möglichkeit, Einladungen zu Veranstaltungen per E-Mail an Benutzer zu senden, die keine iCal Server-Benutzer sind, Integration mit dem Kalender des iPhone und ein Webprogramm, das Benutzern den Zugriff auf ihre Kalender auf jedem Computer mit einem Webbrowser ermöglicht.

iCal Server arbeitet mit dem Programm "iCal" von Mac OS X 10.6 Snow Leopard, dem Kalender des iPhone, Webkalendern in Snow Leopard Server-Wikis und Kalenderprogrammen von Drittanbietern zusammen, die das Standardprotokoll CalDAV unterstützen.
