カレンダーサービスについて

カレンダーサービスにより、ワークグループ、小規模企業、または大規模な組織内で、カレンダーの共有、会議の予約、イベントの予定が容易になります。社員による相互のスケジュールの確認、会議の企画または提案、会議室の予約、プロジェクタの使用予約などがすばやく簡単に実行可能になります。カレンダーサービスで、会議の議題や To Do 項目と共に会議の参加依頼が送信され、その返信が集計されます。

Snow Leopard Server は、プッシュ通知、「iCal サーバ」のユーザ以外にイベントへの参加依頼をメールで送信する機能、iPhone のカレンダーとの統合、ユーザが Web ブラウザを使ってコンピュータから自分のカレンダーにアクセスするための Web アプリケーションを追加します。

「iCal サーバ」は、Mac OS X v10.6 Snow Leopard の「iCal」アプリケーション、iPhone の「カレンダー」、Snow Leopard Server Wiki の Web カレンダー、および標準の CalDAV プロトコルに対応する他社製のカレンダーアプリケーションと連携して動作します。
