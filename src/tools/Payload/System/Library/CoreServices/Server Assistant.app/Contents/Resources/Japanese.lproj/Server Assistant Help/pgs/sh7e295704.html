シリアル番号

「シリアル番号」パネルでは、ここで設定する各サーバのシリアル番号を入力します。

XSVR-106 で始まる、Mac OS X Server v10.6 Snow Leopard のシリアル番号を使用する必要があります。シリアル番号は、Mac OS X Server パッケージの付属物にプリントされています。サイトライセンスがない場合は、サーバごとに一意のシリアル番号が必要です。Snow Leopard Server を登録するために、前のバージョンの Mac OS X Server のシリアル番号を使用することはできません。

サイトライセンスがある場合は、登録されている所有者名と組織を、アップルの販売代理店から指定された通りに正確に入力する必要があります。

複数のサーバを設定する場合、「シリアル番号」パネルには、有効なシリアル番号を持つサーバのリストが表示されます。シリアル番号のないサーバの数がリストの下に表示されます。シリアル番号を追加するには、「追加」をクリックします。「サーバアシスタント」により、現時点でシリアル番号のないサーバにシリアル番号が付けられます。サイトライセンスを追加することもでき、「サーバアシスタント」によって複数のサーバに適用されます。
