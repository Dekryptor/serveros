Ajout d’un service de connexion réseau

Vous pouvez ajouter, supprimer, réorganiser et afficher ou masquer les services de connexion réseau dans la sous-fenêtre Réseau. Un service de connexion réseau est un ensemble de réglages associés à une interface ou un port réseau spécifiques (un connecteur réseau physique tel que le port Ethernet ou la carte AirPort de l’ordinateur, par exemple).

Pour ajouter un service de connexion réseau :

  1. Si le service de connexion réseau souhaité devient visible une fois que vous avez choisi « Afficher tous les services » dans le menu local Action (image d’engrenage), passez à l’étape 5 et configurez le service de connexion réseau qui s’affiche au lieu d’en ajouter un nouveau.

    Si l’option « Afficher tous les services » ne figure pas dans le menu local Action (image d’engrenage), cela signifie que vous pouvez déjà voir tous les services de connexion réseau. Passez à l’étape suivante si vous avez besoin d’en ajouter un.

  2. Cliquez sur le bouton Ajouter (+) situé au bas de la liste.

  3. Choisissez une interface dans le menu local Type, par exemple Ethernet ou AirPort, puis attribuez un nom descriptif au service de connexion réseau.

  4. Si vous créez un profil de configuration automatique (sans configurer un serveur à ce stade), spécifiez l’interface physique (port) que le service de connexion réseau doit utiliser, en saisissant son nom BSD ou son adresse matérielle (aussi appelée adresse MAC).

    Le premier port Ethernet intégré porte le nom en0. Pour en savoir plus, consultez la rubrique Recherche d’un nom d’interface ou d’une adresse matérielle.

  5. Cliquez sur Créer, sélectionnez le nouveau service dans la liste, puis définissez les réglages appropriés.

    Remarque :  vous ne pouvez pas cliquer sur Créer si le nom du service est identique à celui d’un service existant.

    Pour obtenir des instructions, consultez les rubriques suivantes.

    Configuration d’un service de connexion réseau Ethernet
    Configuration d’un service de connexion réseau AirPort
    Configuration d’un service de connexion réseau FireWire
    Configuration d’un service de connexion réseau PPPoE
    Configuration d’un service de connexion réseau de port LOM
    Détails de la configuration réseau

Pour modifier l’ordre des services de connexion réseau lorsque plusieurs sont répertoriés, faites-les glisser dans la liste. Faites glisser la connexion réseau publique en haut de la liste, puis triez les autres dans l’ordre dans lequel vous voulez que le serveur les utilise.

Pour afficher tous les services de connexion réseau ou uniquement les préférés, utilisez le menu local Action (image d’engrenage).

Pour supprimer un service de connexion réseau, sélectionnez-le dans la liste et cliquez sur le bouton Supprimer (–).
