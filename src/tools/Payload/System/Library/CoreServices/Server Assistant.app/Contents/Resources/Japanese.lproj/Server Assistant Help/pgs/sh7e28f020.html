Ethernet オプションを調整する

一部の Ethernet ネットワークでは、詳細なオプションを設定する必要があります。ほとんどの場合、自動的に構成される設定は正しいものです。Ethernet ポート設定が正しくないと、ネットワークのパフォーマンスに影響したり、ポートが使用できなくなったりすることがあります。ネットワーク管理者またはインターネット・サービス・プロバイダは、詳細な設定を調整する必要がある場合はユーザに指示することができます。

詳細な Ethernet 設定を構成するには：

Ethernet サービスを選択し、「Ethernet」ボタンをクリックし、以下の情報を参考にしてオプションを設定します：

  • スピード： 
    Ethernet ポートが送受信できる最大速度（ビット／秒）を指定します。コンピュータおよび接続している装置はギガビット Ethernet に対応していても、使用している Ethernet ケーブルが対応していない場合は、Ethernet 速度の調整が必要になることがあります。あるいは、ギガビット Ethernet に対応するケーブルを購入してください。
  • 通信方式： 
    入力パケットと出力パケットを同時に転送するか（全二重）、交互に転送するか（半二重）を決定します。
  • MTU： 
    ポートで送受信する最大のパケットのことです。（MTU は、バイト単位で表した最大転送単位を表します。）パケットサイズを大きくするとスループット（ファイル転送など）が向上しますが、パケットを受信する装置（スイッチ、ルーターなど）がそのパケットサイズに対応している必要があります。
