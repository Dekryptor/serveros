Tastatur

Im Fenster "Tastatur" können Sie ein Tastaturlayout für den von Ihnen konfigurierten Server auswählen.

Mac OS X Server unterstützt eine Vielzahl von Tastaturlayouts. Im Serverassistenten werden Tastaturlayouts aufgeführt, die zu der von Ihnen ausgewählten Sprache passen. Wenn Sie ein anderes Tastaturlayout verwenden wollen, wählen Sie "Alle einblenden" und wählen Sie eine Tastatur in der Liste aus.

Die gewählte Tastatur wirkt sich nur auf den Server aus, den Sie konfigurieren. Sie wirkt sich nicht auf die Tastatureinstellungen der Computer von Benutzern aus.
