Destination

Utilisez la sous-fenêtre Destination pour identifier l’ordinateur distant sur lequel vous souhaitez installer Mac OS X Server. Avant de faire appel à cette sous-fenêtre, démarrez l’ordinateur distant depuis le disque d’installation de Mac OS X Server ou à partir d’une image d’installation en réseau fournie par le service NetBoot de Mac OS X Server.

Informations nécessaires pour commencer l’installation à distance
Adresse

Pour installer le logiciel serveur sur un ordinateur du réseau local (sous-réseau IP), choisissez l’ordinateur cible dans le menu local.

Pour spécifier un ordinateur non répertorié, tapez son nom DNS ou son adresse IP au format IPv4 (par exemple, 192.0.2.200).

Pour les serveurs détectés par l’Assistant du serveur sur le réseau local (sous-réseau IP), il est possible qu’une adresse IP soit automatiquement affectée par un serveur DHCP du réseau. S’il n’existe pas de serveur DHCP, le serveur cible utilise une adresse de type 169.254.xxx.xxx unique parmi les serveurs du réseau local. Vous pouvez modifier l’adresse IP ultérieurement, lors de la configuration du serveur.

Numéro de série

Saisissez les huit premiers caractères du numéro de série matériel intégré au serveur en tant que mot de passe. Pour connaître ce numéro de série, cherchez l’étiquette apposée sur le serveur. Lors de la saisie, veillez à respecter l’emploi des majuscules dans le numéro de série.

Pour un Xserve avec processeur Intel dont la carte logique principale a été remplacée et n’ayant pas de numéro de série matériel, saisissez « System S » (sans les guillemets) en tant que mot de passe. Pour tout autre ordinateur ne disposant pas d’un numéro de série matériel intégré, tapez 12345678 comme mot de passe.
