Informasi pemulihan

Dokumen ini berisi informasi mengenai memperbaiki volume komputer Anda, menginstal ulang macOS dengan tetap menyimpan file dan pengaturan pengguna Anda secara utuh, menggunakan Time Machine untuk memulihkan sistem komputer Anda, serta menghapus dan menginstal ulang macOS.

Untuk bantuan lainnya, buka https://www.apple.com/support/id-id.

Menghubungkan ke Internet

Jika Anda perlu menghubungkan ke Internet, Anda dapat menggunakan koneksi jaringan nirkabel atau berkabel. Untuk bergabung dengan jaringan nirkabel:

  1. Klik ikon status Wi-Fi status di bar menu (pojok kanan atas layar), lalu pilih jaringan.
  2. Jika perlu, ketik kata sandi jaringan.

Untuk bergabung dengan jaringan tertutup, pilih Bergabung Dengan Jaringan Lain. Masukkan nama jaringan dan, jika perlu, kata sandi jaringan.

Memperbaiki volume Anda

Jika Anda memiliki masalah dengan komputer Anda atau jika Anda memulai komputer Anda dan melihat panel Utilitas macOS, Anda mungkin harus memperbaiki perangkat penyimpanan internal dan volume komputer Anda.

Catatan:Anda mungkin ingin mencatat instruksi ini sebelum menutup jendela ini atau keluar dari Safari.

  1. Pilih Safari > Tutup Safari, pilih Utilitas Disk di panel Utilitas macOS, lalu klik Lanjutkan.
  2. Pilih perangkat penyimpanan internal di daftar di sebelah kiri, lalu klik Pertolongan Pertama.
  3. Pilih volume mulai di daftar di sebelah kiri, lalu klik Pertolongan Pertama lagi.
  4. Klik Jalankan.

Jika Utilitas Disk tidak dapat memperbaiki disk Anda, cadangkan informasi Anda sebanyak mungkin, lalu ikuti instruksi di “Menghapus dan menginstal ulang macOS” di bagian berikutnya di dokumen ini.

Menginstal ulang macOS

Dalam beberapa kondisi, Anda mungkin perlu menginstal ulang macOS. Anda dapat menginstal ulang macOS dengan tetap menyimpan file dan pengaturan pengguna Anda secara utuh.

Catatan:Anda mungkin ingin mencatat instruksi ini sebelum menutup jendela ini atau keluar dari Safari.

  1. Pilih Safari > Tutup Safari.
  2. Pastikan Anda terhubung ke Internet. (Untuk terhubung ke jaringan nirkabel, ikuti instruksi di “Menghubungkan ke Internet” di bagian sebelumnya di dokumen ini.)
  3. Pilih Instal Ulang macOS, lalu klik Lanjutkan.
  4. Di panel tempat Anda memilih volume, pilih volume macOS Anda saat ini (dalam kebanyakan kasus, hanya satu volume tersebut yang tersedia).
  5. Klik Sesuaikan untuk memilih atau membatalkan pilihan perangkat lunak opsional.
  6. Klik Instal.

Menggunakan Time Machine untuk memulihkan sistem Anda

Jika Anda menghapus dan menginstal ulang macOS dan Anda sebelumnya membuat cadangan Time Machine, gunakan utilitas Pulihkan untuk memulihkan semua yang ada di komputer Anda ke kondisi sebelumnya.

Hanya gunakan cadangan Time Machine Anda untuk memulihkan informasi ke komputer yang merupakan sumber cadangan. Jika Anda ingin mentransfer informasi ke komputer baru, gunakan Asisten Migrasi (di folder Lainnya di Launchpad).

Catatan:Anda mungkin ingin mencatat instruksi ini sebelum menutup jendela ini atau keluar dari Safari.

  1. Pilih Safari > Tutup Safari.
  2. Jika cadangan Anda ada di Time Capsule, pastikan Anda terhubung ke Internet. (Untuk menghubungkan ke jaringan nirkabel, ikuti instruksi di “Menghubungkan ke Internet” di bagian sebelumnya di dokumen ini.)
  3. Pilih Pulihkan dari Cadangan Time Machine, lalu klik Lanjutkan.
  4. Pilih sumber cadangan, pilih volume yang berisi cadangan Time Machine, lalu ikuti instruksi di layar.

Menghapus dan menginstal ulang macOS

Jika Anda menghapus dan menginstal ulang macOS, semuanya (akun pengguna, pengaturan jaringan, dan semua file dan folder Anda) akan dihapus. Sebelum Anda menghapus macOS, cadangkan file yang ingin Anda simpan dengan menyalinnya ke volume lain. Catat pengaturan jaringan Anda dari preferensi Jaringan agar memudahkan Anda untuk terhubung lagi setelah menginstal ulang macOS.

Catatan:Anda mungkin ingin mencatat instruksi ini sebelum menutup jendela ini atau keluar dari Safari.

  1. Pilih Safari > Tutup Safari.
  2. Pastikan Anda terhubung ke Internet. (Untuk terhubung ke jaringan nirkabel, ikuti petunjuk di “Menghubungkan ke Internet” di bagian sebelumnya di dokumen ini.)
  3. Pilih Utilitas Disk, lalu klik Lanjutkan.
  4. Pilih volume Anda di daftar di sebelah kiri, lalu klik Hapus.
  5. Masukkan nama untuk volume Anda, pilih Mac OS Diperluas dari menu pop-up Format, lalu klik Hapus.

    Setelah volume dihapus, pilih Utilitas Disk > Tutup Utilitas Disk. Pilih Instal Ulang macOS, klik Lanjutkan, lalu ikuti instruksi di Penginstal macOS untuk menginstal ulang macOS.
