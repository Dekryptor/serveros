À propos du service iCal

Le service de calendrier de Mac OS X Server, iCal Server, permet aux utilisateurs de partager des calendriers, de programmer des réunions et de coordonner des événements au sein d’un groupe de travail, d’une petite entreprise ou d’une grande organisation, en toute simplicité. Avec rapidité et aisance, les collègues peuvent vérifier la disponibilité des uns et des autres, configurer et proposer des réunions, réserver des salles de conférence et des projets, entre autres. iCal Server envoie les invitations, pouvant comprendre des informations telles qu’un agenda ou une liste de tâches, et classe les réponses.

L’application iCal d’un ordinateur Mac OS X 10.6 peut être configurée automatiquement pour utiliser iCal Server. Consultez Configuration automatique des Mac des utilisateurs. Pour obtenir des informations sur la programmation d’événements, l’invitation d’autres utilisateurs, la réception d’invitations, leur réponse, ainsi que sur d’autres possibilités, les utilisateurs Mac OS X peuvent ouvrir iCal et utiliser le menu Aide.

iCal Server fonctionne également avec l’application Calendrier sur l’iPhone, avec les calendriers web dans les wikis Snow Leopard Server et avec d’autres applications de calendrier répandues qui prennent en charge le protocole CalDAV standard. Consultez Configuration manuelle des ordinateurs des utilisateurs.

Si vous souhaitez autoriser l’accès à iCal Server sur Internet et si vous possédez un routeur câble, un routeur DSL ou tout autre routeur, celui-ci doit être configuré pour la redirection de ports (mappage de ports) utilisés par iCal Server. Pour en savoir plus sur la redirection de port, consultez le manuel Premiers contacts.

Si vous souhaitez autoriser l’accès à iCal Server à l’extérieur de votre réseau local et si celui-ci dispose d’un coupe-feu séparé, demandez à l’administrateur du coupe-feu d’ouvrir celui-ci aux ports utilisés par iCal Server. Pour accéder à une liste des ports, consultez l’annexe dans le manuel Premiers contacts.
