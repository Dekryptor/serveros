Verwalten der Webdienste

Verwenden Sie die Servereinstellung "Web", um den Webserver zu aktivieren oder zu deaktivieren, den Speicherort der Startseite Ihres Servers zu ändern, eigene Websites zu veröffentlichen oder standardmäßige Webdienste (Wikis, Webkalender, Blogs und Webmail) zu aktivieren oder zu deaktivieren.

Gehen Sie wie folgt vor, um die Einstellungen der Servereinstellung "Web" zu konfigurieren:

Ändern Sie die Einstellungen wie erforderlich und beachten Sie dabei die folgenden Informationen.

  • Ein/Aus: 
    Gibt den Status des Webservers an. Durch Klicken auf die Taste "Ein/Aus" wird der Webserver aktiviert oder deaktiviert.
  • Webdienste: 
    Ermöglicht das Auswählen einer Startseite für Ihren Server und das separate Aktivieren und Deaktivieren von Wikis, Webkalendern, Blogs und Webmail-Diensten.
  • Eigene Sites: 
    Ermöglicht das Hinzufügen, Entfernen oder Bearbeiten eigener Websites, die Sie (oder jemand anderes für Sie) mit Website-Entwicklungssoftware erstellt haben.
  • Startseite: 
    Wählen Sie "Server-Startseite", um die standardmäßige Mac OS X Server-Website zu verwenden. Wählen Sie ein Wiki als Website Ihres Servers aus.
  • Wikis: 
    Macht alle Wikis verfügbar. Ist "Wikis" ausgewählt, wird durch Klicken auf den als Link definierten Pfeil neben "Wikis" eine Liste der Wikis geöffnet.
  • Kalender: 
    Fügt einen Kalender zum Portal "Meine Seite" jedes Benutzers hinzu und ermöglicht es den Eigentümern von Wikis, einen Webkalender zu ihrem Wiki hinzuzufügen. Ist "Kalender" ausgewählt, wird durch Klicken auf den als Link definierten Pfeil daneben die Eingabe eines Benutzernamens und Kennworts angefordert. Anschließend wird der Webkalender dieses Benutzers geöffnet.
  • Blogs: 
    Ermöglicht es allen Serverbenutzern, mit einem Webbrowser Blogs anzulegen. Ist "Blogs" ausgewählt, wird durch Klicken auf den als Link definierten Pfeil daneben eine Liste der Blogs geöffnet.
  • Webmail: 
    Ermöglicht es allen Serverbenutzern, mit einem Webbrowser auf ihre E-Mails zuzugreifen. Wenn "Webmail" aktiviert ist, können Sie mit dem als Link definierten Pfeil neben der Option die Website für den Dienst "Webmail" öffnen.
  • Das Wiki erstellen: 
    Diese Taste wird angezeigt, wenn die für Wikis verwendete Site mit dem Programm "Server-Admin" gelöscht wurde. Klicken Sie auf diese Taste, wenn Sie Wikis verfügbar machen möchten.
  • Eigene Site zeigen: 
    Diese Taste wird angezeigt, wenn eine eigene Website mit demselben DNS-Namen und derselben Portnummer für die Kommunikation wie das Wiki erstellt wurde. In der Regel verwendet das Wiki den DNS-Namen des Servers (in der Servereinstellung "Informationen" angezeigt) sowie Port 80. Klicken Sie auf diese Taste, wenn Sie Wikis verfügbar machen möchten.
