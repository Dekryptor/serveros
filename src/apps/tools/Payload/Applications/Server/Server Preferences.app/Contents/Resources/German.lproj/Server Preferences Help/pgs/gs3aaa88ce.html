Speichern und Archivieren von Instant-Messaging-Nachrichten

Sie können den Dienst "iChat" so einrichten, dass ein Transkript aller Instant-Messaging-Nachrichten in Form einer Textdatei gespeichert wird. Der Dienst komprimiert das Transkript und speichert die Archivdaten einmal wöchentlich. Das jeweils neuste Transkript und die komprimierten Archivdaten befinden sich im Ordner "/var/jabberd/message_archives/".

Gehen Sie wie folgt vor, um Instant-Messaging-Nachrichten zu speichern und zu archivieren:

  1. Öffnen Sie die Servereinstellung "iChat" und wählen Sie "Alle Chats protokollieren und archivieren" aus.

  2. Starten Sie den Dienst "iChat" erneut, indem Sie zweimal auf "Ein/Aus" klicken.

    Änderungen an den Einstellungen für den Dienst "iChat" werden erst nach dem Neustart des Diensts wirksam.
