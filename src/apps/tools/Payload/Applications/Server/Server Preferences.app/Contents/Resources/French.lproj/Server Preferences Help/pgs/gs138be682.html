Rendre un site web personnalisé non disponible

Vous pouvez rendre un site web personnalisé non disponible.

Pour rendre un site web personnalisé non disponible :

Dans la sous-fenêtre Web, cliquez sur Sites personnalisés et décochez la case en regard du site web que vous souhaitez rendre non disponible.

Si vous sélectionnez un site web personnalisé et cliquez sur le bouton Supprimer (–), le site devient également non disponible. Toutefois, le dossier du site web n’est pas supprimé, si bien que vous pouvez recréer ce site ultérieurement.
