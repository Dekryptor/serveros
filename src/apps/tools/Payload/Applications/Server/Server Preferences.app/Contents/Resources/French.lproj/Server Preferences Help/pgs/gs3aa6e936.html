Gestion du service iCal

Utilisez la sous-fenêtre iCal pour activer et désactiver le service calendrier iCal, limiter la taille des fichiers joints ou restreindre le volume total de données du calendrier de chaque utilisateur.

Pour configurer la sous-fenêtre iCal :

Modifiez tous les paramètres disponibles en vous référant aux informations suivantes :

  • On/Off : 
    Indique l’état du service iCal et permet d’activer ou de désactiver ce service.
  • Limiter la taille de chaque événement de calendrier à __ Mo : 
    Définit la taille totale maximale d’un événement, d’un élément de tâche ou d’un autre objet de calendrier, y compris la taille totale de tous les fichiers joints. Si l’utilisateur tente d’enregistrer un objet de calendrier plus important, le serveur envoie un message d’erreur à l’application de calendrier de l’utilisateur.
  • Limiter la taille des calendriers de chaque utilisateur à __ Mo : 
    Limite l’espace disque que les événements d’un utilisateur, ses éléments de tâches et d’autres données de calendrier peuvent occuper sur le serveur. Si un utilisateur dépasse cette limite, le serveur envoie un message d’erreur à l’application de calendrier de l’utilisateur.

Utilisez Utilitaire iCal Server (dans /Applications/Server/) pour créer, modifier ou supprimer les emplacements et les ressources que les utilisateurs peuvent réserver lorsqu’ils programment des réunions dans les calendriers partagés hébergés par iCal Server. Les emplacements sont des pièces ou d’autres points de repère qui ne sont pas associés à une personne donnée. Les ressources sont des équipements partagés tels que les imprimantes, les scanneurs ou les projecteurs. Pour en savoir plus, ouvrez Utilitaire iCal Server, puis utilisez le menu Aide.
