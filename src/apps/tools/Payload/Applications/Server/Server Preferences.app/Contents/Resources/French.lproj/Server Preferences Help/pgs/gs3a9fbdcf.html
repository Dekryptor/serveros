Configuration d’un dossier de partage de fichiers d’un groupe

Vous pouvez configurer un dossier partagé pour un groupe. Les membres du groupe peuvent obtenir des fichiers du dossier partager et y mettre des fichiers. Ce dossier porte le nom abrégé du groupe et se trouve sur le disque de démarrage du serveur, dans /Groups/.

Pour configurer un dossier de partage de fichiers pour un groupe :

  1. Dans la sous-fenêtre Groupes de Préférences du serveur, sélectionnez un groupe dans la liste située à gauche ou créez un nouveau groupe, puis cliquez sur Groupe.

    Création d’un nouveau groupe

  2. Cochez la case « Dossier de partage de fichiers ».

  3. Cliquez sur Tout afficher, puis sur Partage de fichiers et assurez-vous que le service de partage de fichiers et l’accès au dossier partagé Groupe sont activés.

Lorsqu’un membre du groupe accède au dossier partagé de ce groupe, il doit s’authentifier au moyen du nom et du mot de passe du compte utilisateur qui appartient à ce groupe.
