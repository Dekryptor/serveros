Snow Leopard Server を最新の状態に保つ

サーバがインターネットに接続されているときは、「ソフトウェア・アップデート」により最新の無料 Snow Leopard Server バージョン、セキュリティアップデート、およびアップルからのそのほかの機能拡張が自動的に取得されます。日ごと、週ごと、月ごとにアップデートをサーバに確認させることができます。 チェックは随時、実行できます。

アップデートがあるかを確認する、または自動アップデートを調整するには：

  1. サーバで「システム環境設定」を開きます。

  2. 「ソフトウェア・アップデート」アイコンをクリックして、画面に表示される指示に従います。

    「システム環境設定」の「ソフトウェア・アップデート」パネルを開く

組織内に Mac OS X Server で動作するほかのサーバがある場合は、アップルの代わりに、そのサーバから「ソフトウェア・アップデート」を取得できます。経験豊富な管理者は、「サーバ管理」で「ソフトウェア・アップデート」サービスが提供されるよう、Mac OS X Server を設定できます。

「ソフトウェア・アップデート」は、アップルのダウンロード用 Web サイトから直接ダウンロードすることもできます：www.apple.com/jp/support/downloads/
