Si vous ne pouvez pas activer la gestion AirPort

Si vous disposez d’une borne d’accès AirPort Extreme (802.11n) ou d’un Time Capsule mais que le bouton « Activer la gestion AirPort » ou « Activer la sécurité du coupe-feu » ne s’affiche pas lorsque le contrôle de sécurité est désactivé dans la sous-fenêtre Sécurité de Préférences du serveur, essayez les solutions suivantes :

  • Vérifiez que votre borne d’accès AirPort Extreme ou votre Time Capsule est configuré pour partager une connexion Internet.

  • Vérifiez que l’interface réseau maître du serveur (généralement le premier port Ethernet intégré) est connectée au même réseau local que votre borne d’accès AirPort Extreme ou que votre Time Capsule.
