サービスを追加する

初期設定の後、「サーバ環境設定」を使って、基本サービスやシステム設定を変更したり、ユーザやグループを追加したりできます。「サーバ管理」を使って、高度な設定を変更したり、高度なサービスを設定したりできます。「ワークグループマネージャ」を使って、高度なユーザ設定およびグループ設定を変更したり、ユーザの環境設定を制御したり、コンピュータの記録を管理したりできます。「サーバ管理」および「ワークグループマネージャ」については、それらのアプリケーションを開いて「ヘルプ」メニューを参照してください。
