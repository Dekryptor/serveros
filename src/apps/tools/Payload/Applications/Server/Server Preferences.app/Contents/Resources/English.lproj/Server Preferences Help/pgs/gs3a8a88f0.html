Latest Help Topics

When your computer is connected to the Internet, it automatically gets new and updated help topics. Some of these may also update topics in the Getting Started manual included with Mac OS X Server.

  • To see a list of the latest help topics, click Home and then click the Latest Help Topics link. Check back regularly to see if new help topics have been published.

  • To search for late-breaking information from Apple’s Mac OS X Server Support website, use the search field above. You can turn this search on or off by choosing Include Product Support Searches from the pop-up menu in the search field.

  • To view an RSS feed listing the latest updates to Getting Started and Mac OS X Server onscreen help, use an RSS reader application such as Safari or Mail and go to:

feed://helposx.apple.com/rss/snowleopard/serverdocupdates.xml
