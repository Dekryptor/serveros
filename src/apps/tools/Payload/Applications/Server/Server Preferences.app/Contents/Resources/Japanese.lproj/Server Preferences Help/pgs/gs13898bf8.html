ユーザアカウントのユーザ名を変更できない場合

ユーザアカウントの作成後にそのユーザ名を変更することはできません。ユーザ名を変更しなければならない場合は、アカウントを削除して新規のユーザ名で新規のユーザアカウントを作成する必要があります。操作方法については、「ユーザアカウントを削除する」および「ユーザアカウントを追加する」を参照してください。

ユーザアカウントを削除すると、そのアカウントが属していたグループメンバーシップはキャンセルされ、グループサービスおよびプライベート Wiki がアクセス停止になります。ユーザアカウントを削除すると、サーバに保存されているそのユーザのメールデータも削除されます。削除されたユーザアカウントは、サーバ上のカレンダーおよびアドレスブックの情報にアクセスできません。

Mac OS X v10.6 Snow Leopard を実行するサーバでユーザ用のアカウントを新規に作成した後、ユーザは「アカウント環境設定」の「ログインオプション」を開き、ディレクトリサーバのリストからサーバを削除して、サーバを追加し直す必要があります。サーバへの接続については、「アカウント環境設定を使用して Mac を設定する」を参照してください。
