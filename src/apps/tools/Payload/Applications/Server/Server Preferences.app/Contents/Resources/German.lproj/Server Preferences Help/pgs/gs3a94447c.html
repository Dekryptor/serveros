Schließen einer Verbindung im Programm "Servereinstellungen"

Aus Sicherheitsgründen sollten Sie es sich zur Angewohnheit machen, das Fenster "Servereinstellungen" zu schließen, wenn Sie es nicht aktiv zum Verwalten der Benutzer, Gruppen, Dienste und Systeminformationen eines Servers nutzen. Ist ein Server unbeaufsichtigt, kann eine geöffnete Serververbindung von Dritten missbraucht werden, um unbefugte Änderungen an Benutzern, Gruppen und Diensten vorzunehmen.

Sie können eine Verbindung im Programm "Servereinstellungen" auf verschiedene Arten schließen:

  • Schließen Sie das Fenster "Servereinstellungen".

  • Wählen Sie "Verbindung" > "Schließen".

  • Beenden Sie das Programm "Servereinstellungen".

Wenn nur ein Fenster "Servereinstellungen" geöffnet ist und Sie dieses Fenster schließen, wird das Programm "Servereinstellungen" automatisch beendet.
