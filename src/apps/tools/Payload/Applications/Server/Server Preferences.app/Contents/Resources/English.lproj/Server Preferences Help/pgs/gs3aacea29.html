Managing Address Book Service

Use the Address Book pane to turn address book service on or off, or to limit each user’s disk space for contact information stored on the server.

To configure the Address Book pane:

Change any available setting, using the following information as a guide:

  • On/Off: 
    Indicates the status of address book service, and clicking the On/Off switch turns the service on or off.
  • Limit each user’s total book size to __ MB: 
    Limits how much disk space each user’s address book can use on the server. If a user exceeds this limit, the server sends an error message to the user’s Address Book application.
