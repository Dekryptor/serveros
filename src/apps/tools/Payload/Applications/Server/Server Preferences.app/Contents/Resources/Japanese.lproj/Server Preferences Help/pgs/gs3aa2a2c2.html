メールサービスについて

メールサービスでユーザは、任意のメールアプリケーションを使用して、ローカルネットワークとインターネットでメールを送受信できます。メールサービスは、迷惑メールやウイルスからユーザを保護するフィルタを備えています。

サーバ上のユーザアカウントを持つすべてのユーザにはメールアドレスが与えられます。Mac OS v10.6 Snow Leopard を実行するコンピュータでは、サーバのメールサービスを使用するために「Mail」アプリケーションを自動設定することができます。「ユーザの Mac を自動的に設定する」を参照してください。

メールサービスは、iPhone の「メール」や、標準の IMAP、POP、および SMTP メールプロトコルを使用するそのほかの一般的なメールアプリケーションにも対応しています。

インターネット上のメールサービスへのアクセスを許可したい場合で、ケーブル・ルーター、DSL ルーター、またはそのほかのネットワークルーターを持っている場合は、お使いのルーターでメールサービス用にポートフォワード（ポートマッピング）を設定する必要があります。ポートフォワードの詳細については、「お使いになる前に」マニュアルを参照してください。

ローカルネットワークの外部のメールサービスへのアクセスを許可したい場合で、お使いのローカルネットワークに独立したファイアウォールデバイスがある場合は、ファイアウォールの管理者に問い合わせて、メールサービスで使用するポートのファイアウォールを解放してもらってください。ポートのリストについては、「お使いになる前に」マニュアルの付録を参照してください。
