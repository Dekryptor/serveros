Verwalten von Gruppen

In der Servereinstellung "Gruppen" können Sie Gruppen erstellen, Gruppendienste aktivieren oder deaktivieren, Gruppenmitglieder hinzufügen und entfernen und nicht mehr benötigte Gruppen löschen.

Optionen
Taste "Hinzufügen" (+)

Ermöglicht das Erstellen einer Gruppe. Anleitungen hierzu finden Sie unter Erstellen einer neuen Gruppe.

Taste "Entfernen" (–)

Ermöglicht das Löschen einer Gruppe. Anleitungen hierzu finden Sie unter Löschen einer Gruppe.

Gruppe

Ermöglicht das Ändern des Gruppennamens, das Anzeigen des Kurznamens der Gruppe und das Einrichten von Gruppendiensten. Gruppendienste umfassen einen gemeinsamen Ordner und automatisch erstellte iChat-Kontaktlisten. Anleitungen hierzu finden Sie unter Ändern des Namens einer Gruppe und Ändern der Dienste von Gruppen.

Mitglieder

Ermöglicht das Hinzufügen oder Entfernen von Gruppenmitgliedern, die als Standardbenutzer, importierte Benutzer oder automatisch erstellte Benutzer in der Servereinstellung "Benutzer" definiert sind. (Damit Ihr Server importierte oder automatisch erstellte Benutzer verwenden kann, muss er mit dem Verzeichnisserver Ihres Unternehmens verbunden sein.) Anleitungen hierzu finden Sie unter Hinzufügen und Entfernen von Gruppenmitgliedern.

Externe Mitglieder

Ermöglicht das Hinzufügen oder Entfernen von Gruppenmitgliedern, für die auf dem Verzeichnisserver Ihres Unternehmens Benutzer- oder Gruppen-Accounts angelegt sind. Externe Mitglieder einer Gruppe können auf den gemeinsamen Ordner der Gruppe sowie auf Wikis zugreifen, die für die Gruppe zur Verfügung stehen. Anleitungen hierzu finden Sie unter Hinzufügen und Entfernen externer Mitglieder einer Gruppe.

Wird der Bereich "Externe Mitglieder" nicht angezeigt, ist Ihr Server nicht mit einem Verzeichnisserver des Unternehmens verbunden. Wenn Ihr Unternehmen einen Verzeichnisserver verwendet, können Sie eine Verbindung dazu herstellen. Anleitungen hierzu finden Sie unter Verbinden Ihres Servers mit einem Verzeichnisserver.

Konfigurieren

Wird anstelle der oben beschriebenen Optionen die Taste "Konfigurieren" angezeigt, besitzt der Server keine eigene Verzeichnis-Domain für Benutzer und Gruppen. Durch Klicken auf diese Taste können Sie eine Verzeichnis-Domain einrichten. Anleitungen hierzu finden Sie unter Einrichten der Verwaltung von Benutzern und Gruppen.
