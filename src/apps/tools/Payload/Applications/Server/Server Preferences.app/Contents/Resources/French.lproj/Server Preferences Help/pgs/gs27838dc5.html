Personnalisation des services à l’aide des applications avancées

Il est facile de personnaliser les réglages de services de base grâce aux Préférences du serveur. Vous pouvez aussi utiliser Admin Serveur et les autres applications et outils avancés répertoriés dans Outils et applications avancés. Les outils et applications avancés vous permettent de personnaliser de nombreux réglages de services additionnels. Vous pouvez aussi activer des services non visibles dans les Préférences du serveur, tels que Podcast Producer et QuickTime Streaming Server. Pour obtenir des informations sur les services avancés, les options et les applications, ouvrez Admin Serveur et allez ensuite dans le menu Aide, ou consultez le site web des ressources de Mac OS X Server à l’adresse www.apple.com/fr/server/macosx/resources/ (en anglais).

Important :  avant d’utiliser Admin Serveur ou d’autres applications avancées pour modifier des réglages de services que vous gérez habituellement à partir des Préférences du serveur, prenez note des réglages actuels au cas où vous auriez besoin de revenir à ces réglages. Par exemple, vous pouvez sauvegarder une capture d’écran de chaque sous-fenêtre et zone de dialogue avant d’en modifier les réglages. (Pour en savoir plus sur les captures d’écran, basculez sur le Finder et consultez le menu Aide.)
