Vérification de l’état du service

Préférences du serveur affiche l’état général de chaque service au moyen d’un indicateur en regard de chaque icône de service.

Pour afficher les indicateurs d’état de tous les services :

Cliquez sur Tout afficher dans la fenêtre Préférences du serveur.

Un voyant vert indique que le service fonctionne.

Pour activer ou désactiver un service :

  1. Dans la fenêtre Préférences du serveur, cliquez sur l’icône du service que vous souhaitez activer ou désactiver.

    Si vous ne voyez pas les icônes des services, cliquez sur Tout afficher dans la fenêtre Préférences du serveur. Vous pouvez également utiliser le menu Présentation pour choisir le service que vous souhaitez activer ou désactiver.

  2. Dans la sous-fenêtre des services, cliquez sur le commutateur Marche/Arrêt.

Le commutateur Marche/Arrêt de chaque sous-fenêtre de service indique si le service est en cours d’exécution.

Vous pouvez également surveiller l’état du service à l’aide du widget État du serveur.
