Rejet du courrier électronique provenant d’expéditeurs sur liste noire

Le service de messagerie de votre serveur peut rejeter le courrier électronique en provenance d’autres serveurs de messagerie qui sont répertoriés sur liste noire en tant que relais ouverts par un serveur DNSBL (DNS blacklist). Votre service de messagerie utilise le serveur DNSBL zen.spamhaus.org, qui est géré par The Spamhaus Project (www.spamhaus.org, en anglais). Vous pouvez spécifier un serveur DNSBL différent. Les serveurs DNSBL sont aussi appelés serveurs Real-time Blacklist (RBL, listes noires en temps réel) ou serveurs black-hole (trous noirs).

Bloquer des courriers électroniques indésirables provenant d’expéditeurs sur liste noire peut ne pas être totalement efficace. Parfois cela empêche de recevoir des courriers électroniques valides à partir de serveurs de messagerie qui ne sont pas configurés correctement.

Pour spécifier le serveur DNSLB à utiliser avec votre service de messagerie :

Dans la sous-fenêtre Mail dans Préférences du serveur, sélectionnez « Rejeter le courrier provenant d’expéditeurs sur liste noire », cliquez sur le bouton Modifier accompagnant cette option et saisissez l’adresse du serveur DNSBL.
