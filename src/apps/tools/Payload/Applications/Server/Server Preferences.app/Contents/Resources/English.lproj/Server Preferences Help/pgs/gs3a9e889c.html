Changing a Group’s Name

You can use Server Preferences to change the name of a group.

To change a group’s name:

  1. In the Groups pane of Server Preferences, select the group you want to rename in the list on the left, and then click Group.

  2. Edit the Group Name field.

    The name can be up to 255 characters (which can be as few as 85 Japanese characters). It can include spaces.

    You can’t change a group’s short name using Server Preferences.
