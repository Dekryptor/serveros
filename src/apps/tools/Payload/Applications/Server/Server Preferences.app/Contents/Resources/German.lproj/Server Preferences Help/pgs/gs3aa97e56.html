Kommunizieren mit Benutzern von Google Talk und anderen XMPP-Diensten

Sie können den auf Ihrem Server definierte Benutzern die Instant-Messaging (IM)-Kommunikation mit Benutzern von Google Talk und anderen IM-Systemen ermöglichen, die das Protokoll XMPP verwenden.

Gehen Sie wie folgt vor, um die Kommunikation mit Google Talk und anderen XMPP-Diensten zu aktivieren:

  1. Öffnen Sie die Servereinstellung "iChat" und wählen Sie "Server-zu-Server-Kommunikation aktivieren" aus.

  2. Starten Sie den Dienst "iChat" erneut, indem Sie zweimal auf "Ein/Aus" klicken.

    Änderungen an den Einstellungen für den Dienst "iChat" werden erst nach dem Neustart des Diensts wirksam.

Wenn Sie Benutzern die Möglichkeit geben möchten, Sofortnachrichten mit Benutzern anderer XMPP-Chatdienste auszutauschen und wenn Ihr Netzwerk über eine separate Firewall verfügt, bitten Sie den Administrator der Firewall, die Firewall für die Ports zu öffnen, die der Dienst "iChat" für die Kommunikation verwendet. Eine Liste der verfügbaren Ports finden Sie im Anhang der Einführung.

Wenn Ihr Server über einen Kabel-Router, DSL-Router oder einen anderen Router mit dem Internet verbunden ist, müssen Sie die Portweiterleitung (Portzuordnung) des Routers für die Ports konfigurieren, die der Dienst "iChat" für die Kommunikation verwendet. Weitere Informationen über die Portweiterleitung finden Sie in der Einführung.
