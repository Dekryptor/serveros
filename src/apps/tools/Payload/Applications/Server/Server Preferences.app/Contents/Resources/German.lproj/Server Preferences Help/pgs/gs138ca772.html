Benutzer können nicht über das Internet auf einen Dienst zugreifen

Wenn Benutzer im lokalen Netzwerk auf einen Dienst zugreifen können, nicht aber über das Internet, versuchen Sie Folgendes:

  • Wenn in der Servereinstellung "Sicherheit" die Meldung "Diese Dienste erreichbar machen auf __" angezeigt wird, prüfen Sie, ob der gewünschte Dienst genannt wird.

  • Wenn in der Servereinstellung "Sicherheit" die Meldung "Die AirPort-Verwaltung ist zurzeit deaktiviert" angezeigt wird, aktivieren Sie sie und prüfen Sie, ob der gewünschte Dienst genannt wird.

  • Wenn Ihr Server über eine AirPort-Basisstation oder einen anderen Internet-Router auf das Internet zugreift, müssen Sie den Router möglicherweise konfigurieren, um die Dienste des Servers im Internet verfügbar zu machen. Wenn Sie eine AirPort Extreme-Basisstation (802.11n) oder Time Capsule verwenden, kann Mac OS X Server das Gerät automatisch konfigurieren. Informationen zum Konfigurieren anderer Modelle einer AirPort-Basisstation oder eines Internet-Routers finden Sie in Kapitel 2 "Vorbereitung für die Installation von Mac OS X Server" der Einführung.
