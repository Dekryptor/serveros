グループを管理する

「グループ」パネルでは、グループの作成、グループサービスの有効／無効の切り替え、グループメンバーの追加と削除、不要なグループの削除を実行します。

オプション
追加（＋）ボタン

新しいグループを作成できます。操作方法については、「新しいグループを作成する」を参照してください。

削除（－）ボタン

グループを削除できます。操作方法については、「グループを削除する」を参照してください。

グループ

グループ名の変更、グループのユーザ名の表示、およびグループサービスの設定ができます。グループサービスには、ファイル共有や iChat の自動メンバーがあります。操作方法については、「グループ名を変更する」および「グループのサービスを変更する」を参照してください。

メンバー

「ユーザ」パネルで標準ユーザ、読み込まれたユーザ、または自動ユーザであるグループメンバーを追加したり取り除いたりできます。（読み込まれたユーザまたは自動ユーザを使用するには、サーバが組織のディレクトリサーバに接続されている必要があります。）操作方法については、「グループのメンバーを追加する／取り除く」を参照してください。

外部メンバー

一覧を表示して、組織のディレクトリサーバにユーザアカウントまたはグループアカウントを持つグループメンバーを追加したり取り除いたりできます。グループの外部メンバーは、グループに使用権限のあるグループのファイル共有フォルダおよび Wiki にアクセスできます。操作方法については、「グループの外部メンバーを追加する／取り除く」を参照してください。

「外部メンバー」タブが表示されない場合は、サーバが組織のディレクトリサーバに接続されていません。所属する組織にディレクトリサーバがある場合は、それに接続できます。操作方法については、「サーバをディレクトリサーバに接続する」を参照してください。

設定

上記のオプションの代わりに「設定」ボタンが表示されている場合、お使いのサーバにはユーザおよびグループ用の独自のディレクトリがありません。 このボタンをクリックすると設定ができます。操作方法については、「ユーザおよびグループの管理を設定する」を参照してください。
