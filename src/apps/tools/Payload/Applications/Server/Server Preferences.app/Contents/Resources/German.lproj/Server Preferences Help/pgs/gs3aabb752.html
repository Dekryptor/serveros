Dateifreigabe-Adresse eines Servers

Benutzer - und gegebenenfalls Gäste - können über die Protokolle AFP und SMB und unter Verwendung der folgenden Adressen auf die gemeinsamen Ordner Ihres Servers zugreifen:

  • smb:///meinserver.beispiel.de

  • afp:///meinserver.beispiel.de

Anstelle des kursiv gesetzten Platzhalters muss der DNS-Name Ihres Servers eingegeben werden, der in der Servereinstellung "Informationen" angezeigt wird.

Damit Benutzer auf gemeinsame Ordner zugreifen können, auf die der Zugriff als Gast nicht möglich ist, müssen sie sich mit den Namen und Kennwörtern ihrer Benutzer-Accounts identifizieren. Nach dem Anmelden können Benutzer auf die gemeinsamen Ordner aller Gruppen zugreifen, denen sie als Mitglied angehören. Wird der Gastzugriff für einen gemeinsamen Ordner eingeräumt, können Benutzer darauf zugreifen, ohne sich anmelden zu müssen.
