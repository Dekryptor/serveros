Unzureichender Speicherplatz auf dem Server für die Datensicherung der Benutzercomputer

Wenn der Speicherplatz auf dem Server für die Datensicherung der Mac-Computer von Benutzern nicht ausreicht, können Sie ein anderes Volume an den Server anschließen und dieses in der Servereinstellung "Time Machine" als Sicherungsvolume definieren. Anleitungen hierzu finden Sie unter Verwalten der Sicherungsvolumes für Benutzerdaten.

Snow Leopard-Benutzer, die Time Machine in den Systemeinstellungen nicht aktiviert haben, werden von ihren Mac-Computern gefragt, ob sie den Server als Speicherort für Datensicherungen verwenden möchten. Andere Benutzer müssen den Server in der Systemeinstellung "Time Machine" als Sicherungsvolume auswählen.
