Modification du nom DNS de votre serveur

Seul l’administrateur de votre service DNS peut modifier le nom DNS du serveur.

  • Si votre service DNS fait partie des services apportés par votre fournisseur d’accès à Internet, contactez celui-ci pour assistance.

  • Si votre service DNS est fourni par un autre serveur de votre réseau, demandez assistance à l’administrateur de ce serveur DNS.

  • Si votre serveur fournit son propre service DNS, utilisez l’application Admin Serveur pour modifier le nom dans l’enregistrement de l’ordinateur de votre serveur. Pour en savoir plus, ouvrez Admin Serveur, puis utilisez le menu Aide pour rechercher la « Vue d’ensemble du service DNS » ou consultez le site web Ressources de Mac OS X Server à l’adresse www.apple.com/fr/server/macosx/resources/ (en anglais).

Évitez de modifier le nom DNS du serveur car cela oblige les utilisateurs d’ordinateurs clients à reconfigurer leurs ordinateurs. Pour obtenir des instructions, consultez Si des utilisateurs ne peuvent pas configurer leur ordinateur pour utiliser le serveur.
