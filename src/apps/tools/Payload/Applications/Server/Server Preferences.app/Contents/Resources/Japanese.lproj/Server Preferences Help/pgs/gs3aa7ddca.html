VPN サービスを開始する／停止する

「VPN」パネルで VPN リモート・アクセス・サービスを開始または停止できます。VPN サービスが無効の間は、ユーザはインターネット経由でサーバにリモート接続できません。

VPN サービスを開始または停止するには：

「サーバ環境設定」の「VPN」パネルで「入」／「切」スイッチをクリックします。

VPN サービスを開始した後、特にこのサービスをはじめて開始した場合は、「Mac ユーザの VPN 接続を設定する」および「ユーザの VPN 接続を手動で設定する」で説明されている通りに VPN 接続を設定するようユーザに通知してください。
