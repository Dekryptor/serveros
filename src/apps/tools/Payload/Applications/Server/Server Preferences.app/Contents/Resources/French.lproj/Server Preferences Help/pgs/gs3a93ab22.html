Personnalisation des réglages de service

Vous pouvez modifier les réglages des services ci-dessous. Vous pouvez aussi désactiver les services inutiles (comme ceux déjà fournis par un autre serveur). Par exemple, vous n’avez pas besoin du service de messagerie du serveur si vous souhaitez utiliser le service de messagerie de votre fournisseur d’accès à Internet ou celui de votre organisation.

Partage de fichiers

Ajoutez et supprimez des dossiers partagés. Modifiez les autorisations des dossiers partagés.

Gestion du service de partage de fichiers
Carnet d’adresses

Il permet à chaque utilisateur de disposer d’un carnet d’adresses personnel ou de partager un carnet d’adresses unique avec tous les autres utilisateurs. Vous pouvez donner un nom à ce carnet d’adresses partagé.

Gestion du service de carnet d’adresses
iCal

Limitez la taille de chaque événement et la taille totale du calendrier de chaque utilisateur.

Gestion du service iCal
iChat

Activez la messagerie instantanée avec les utilisateurs d’autres serveurs. Consignez et archivez toutes les conversations.

Gestion du service iChat
Mail

Configurez un serveur de relais pour les messages sortants. Ajustez le filtrage du courrier indésirable et des virus. Spécifiez un serveur pour rejeter le courrier électronique provenant d’expéditeurs sur liste noire.

Gestion du service de messagerie
Web

Choisissez la page d’accueil du serveur. Activez ou désactivez des services web (les wikis, le calendrier, les blogs et le courrier électronique). Configurez des sites web personnalisés.

Gestion des services web
VPN

Donnez la possibilité aux utilisateurs distants d’accéder à votre réseau via Internet. Modifiez le secret partagé et les adresses IP pour les utilisateurs distants. Créez un fichier de configuration pour définir un accès à distance sur les ordinateurs Mac des utilisateurs.

Gestion du service VPN
