Renouvellement d’un certificat

Chaque certificat a une date d’expiration et doit être remplacé périodiquement.

Pour renouveler un certificat à expiration :

  1. Demandez un nouveau certificat à l’autorité de certification qui vous avait fournit le certificat à expiration.

    Pour obtenir des instructions, consultez Obtention d’un certificat signé.

  2. Dans la sous-fenêtre Informations de Préférence du serveur, cliquez sur le bouton Modifier situé à droite de Certificat SSL.

  3. À partir du menu local; choisissez le certificat à remplacer.

    « Utiliser un certificat SSL » doit être sélectionné pour que le menu local soit disponible.

  4. À partir du menu local, choisissez Signature de certificat > Remplacer par un certificat signé ou renouvelé.

  5. Faites glisser le fichier contenant le certificat de remplacement au milieu de la zone de dialogue de Préférences du serveur, puis cliquez sur Remplacer le certificat.
