ウェルカムメッセージと案内メールをカスタマイズする

「サーバ環境設定」を使って、新規ユーザを歓迎したり、既存のユーザにサーバの使用環境を案内したりするメールメッセージをカスタマイズできます。 ディレクトリサーバからユーザを読み込むよう設定されているサーバは、以下のメッセージをすべて送信することができます。サーバがディレクトリサーバに接続されていない場合、サーバはウェルカムメッセージのみを送信します。

ウェルカムメール

サーバで提供するサービスについて新規ユーザに知らせます。新規ユーザのコンピュータがサーバからメールサービスを受けている場合に、自動的に送信されます。サーバのメールサービスが無効になっている場合は送信されません。

案内メール

新規に読み込まれたユーザに、サーバのサービスの利用方法を知らせます。それらのユーザのアカウントを読み込むときに送信できます。 手動での送信も可能です。
