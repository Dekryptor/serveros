Starten oder Stoppen des Diensts "Mail"

In der Servereinstellung "Mail" können Sie den Dienst "Mail" stoppen oder starten. Wenn der Dienst "Mail" gestoppt ist, können Benutzer mithilfe der vom Server bereitgestellten E-Mail-Adressen keine E-Mails senden oder empfangen.

Gehen Sie wie folgt vor, um den Dienst "Mail" zu starten oder zu stoppen:

Klicken Sie auf die Taste "Ein/Aus" in der Servereinstellung "Mail".

Nach dem Starten des Diensts "Mail" und besonders nach dem erstmaligen Starten sollten Sie die Benutzer darauf hinzweisen, ihr Mail-Programm einzurichten. Hinweise dazu finden Sie unter Manuelles Konfigurieren von Benutzercomputern.
