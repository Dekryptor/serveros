Auswählen der Startseite des Servers
Gehen Sie wie folgt vor, um die Startseite des Servers auszuwählen:

Klicken Sie in der Servereinstellung "Web" auf "Webdienste" und wählen Sie dann die gewünschte Seite aus dem Einblendmenü "Startseite" aus.

Wird das Einblendmenü grau dargestellt, ist der Dienst "Wikis" deaktiviert.

Wenn das Einblendmenü keine Wikis enthält, müssen Sie oder ein anderer Benutzer mit einem Benutzer-Account auf dem Server ein Wiki erstellen. Anleitungen hierzu erhalten Sie, indem Sie auf einer Wiki-Seite auf "Hilfe" klicken.
