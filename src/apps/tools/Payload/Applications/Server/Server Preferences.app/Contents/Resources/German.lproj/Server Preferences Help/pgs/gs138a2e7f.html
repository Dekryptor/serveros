Ein Benutzer erhält keine Einladung zum Zugriff auf Ihren Server

Wenn ein Mac mit Mac OS X 10.6 Snow Leopard auf Ihr Netzwerk zugreift, wird möglicherweise automatisch eine Einladung zur Nutzung von Diensten Ihres Servers angezeigt. Wird diese Einladung nicht angezeigt, kann dies unterschiedliche Ursachen haben.

Die Einladung zum Zugriff auf Ihren Server wird unter folgenden Umständen nicht automatisch angezeigt:

  • Der Mac des Benutzers arbeitet mit einer Mac OS X-Version älter als 10.6 Snow Leopard. Informationen zum Konfigurieren eines solchen Mac-Computers finden Sie unter Manuelles Konfigurieren von Benutzercomputern.

  • Der Benutzer hat zuvor eine Einladung zum Zugriff auf Ihren Server abgelehnt.

  • Der Benutzer war an seinem Mac angemeldet, während Sie den Account des Benutzers auf dem Server erstellt haben. In diesem Fall muss sich der Benutzer abmelden und dann erneut anmelden.

  • Der Benutzer hat sich nicht mit dem standardmäßigen Account an seinem Mac angemeldet.

  • Der Benutzer-Account auf Ihrem Server stimmt nicht mit dem Account auf dem Mac des Benutzers überein. Beide Accounts müssen denselben Namen oder Kurznamen besitzen, um übereinzustimmen. Lauten Account-Name und Kurzname auf dem Server z. B. Gitta Kumar und gkumar, auf dem Mac des Benutzers aber Gitta und gitta, stimmen die Accounts nicht überein.

    Sie beheben das Problem, indem Sie den Namen des Benutzer-Accounts auf dem Server so ändern, dass er dem Namen des Benutzer-Accounts auf dem Mac des Benutzers entspricht. Eine manuelle Verbindung ist auch dann möglich, wenn die Namen nicht übereinstimmen. Anleitungen hierzu finden Sie unter Konfigurieren eines Mac-Computers mit der Systemeinstellung "Benutzer".

  • Der Mac-Computer des Benutzers verfügt bereits über eine Verbindung zu den Verzeichnisdiensten. Der Benutzer kann manuell eine Verbindung zu Ihrem Server hinzufügen. Anleitungen hierzu finden Sie unter Konfigurieren eines Mac-Computers mit der Systemeinstellung "Benutzer".

  • Der Account des Benutzers auf dem Server wird von einem Verzeichnisserver importiert (und nicht auf Ihrem Server erstellt). Der Benutzer erhält per E-Mail eine Einladung mit einer Taste, über die Dienste von Ihrem Server angefordert werden können. Weitere Informationen hierzu finden Sie unter Einrichten der Mac-Computer neu importierter Benutzer.

  • Auf Ihrem Server ist nicht mindestens ein Dienst im Programm "Servereinstellungen" aktiviert. Weitere Informationen hierzu finden Sie unter Prüfen des Status von Diensten.

  • Auf dem Mac Ihres Benutzers oder auf Ihrem Server ist ein Problem mit der Netzwerkverbindung aufgetreten.
