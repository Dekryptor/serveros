Configuration d’une connexion VPN pour les utilisateurs Mac

Vous pouvez utiliser Préférences du serveur pour générer un fichier qu’un utilisateur Mac peut ouvrir afin de créer automatiquement une configuration VPN. L’utilisateur peut ensuite établir une connexion VPN au serveur et à son réseau via Internet. Le fichier de configuration fonctionne avec Mac OS X 10.3 ou ultérieur. Pour obtenir des informations sur la génération du fichier de configuration, consultez Création d’un fichier de configuration VPN.

Lorsque vous donnez aux utilisateurs Mac le fichier de configuration VPN que vous avez généré, vous pouvez également leur fournir les instructions d’utilisation. Pour obtenir ces instructions d’utilisation, consultez Utilisation d’un fichier de configuration VPN.
