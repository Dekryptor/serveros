Verwalten der Serverinformationen

In der Servereinstellung "Informationen" können Sie allgemeine Informationen über Ihren Server abrufen und anzeigen, u. a. Angaben zur installierten Hard- und Software, Namen und Adressen im Netzwerk und die Seriennummer. Sie können auch den Computernamen und die Seriennummer des Servers ändern und E-Mail-Warnhinweise und SSL-Zertifikate verwalten.

Gehen Sie wie folgt vor, um Serverinformationen abzurufen oder zu ändern:

Prüfen Sie die folgenden Angaben in der Servereinstellung "Informationen":

  • IP-Adresse: 
    Die eindeutige numerische Adresse, die den Server im Netzwerk identifiziert. Weitere Information finden Sie unter Ändern der IP-Adresse des Servercomputers.
  • DNS-Name: 
    Der vollständige, eindeutige Name, der vom DNS-Server Ihres Servers bereitgestellt wird.
  • Computername: 
    Identifiziert den Server für Benutzer, die im Netzwerk nach Dateiservern oder anderen, mit Computernamen anstelle von DNS-Namen versehenen Netzwerkressourcen suchen. Weitere Information finden Sie unter Ändern des Namens des Servercomputers.
  • Serverlizenz: 
    Mac OS X Server bestimmt den Typ der Lizenz anhand der Seriennummer der Software oder der standortbezogenen Lizenz. Klicken Sie auf "Bearbeiten", um die Informationen der Seriennummer oder standortbezogenen Lizenz zu ändern. Anleitungen hierzu finden Sie unter Ändern der Seriennummer oder der Angaben für die standortbezogene Lizenz.
  • Warnhinweise: 
    Die E-Mail-Adresse, an die der Server Meldungen zu wichtigen Systemereignissen sendet. Klicken Sie auf "Bearbeiten", um die Adresse zu ändern oder bestimmte Arten von Warnhinweisen zu aktivieren oder zu deaktivieren. Anleitungen hierzu finden Sie unter Ändern der Einstellungen für Warnhinweise.
  • SSL-Zertifikat: 
    Erstellen, verwenden und pflegen Sie Identitäten für Dienste, die zu Sicherheitszwecken SSL verwenden. Anleitungen hierzu finden Sie unter Verwenden eines SSL-Zertifikats.

Im Bereich "Informationen" auf der linken Seite werden Versions- und Build-Nummer der derzeit installierten Mac OS X Server-Software angezeigt, ebenso wie Prozessorgeschwindigkeit und -typ und die Gesamtgröße des installierten Arbeitsspeichers.

Andere Netzwerkinformationen können im Bereich "Netzwerk" der Systemeinstellungen auf dem Server angezeigt und geändert werden. Informationen über Änderungen der Systemeinstellung "Netzwerk" erhalten Sie, wenn Sie die Systemeinstellungen öffnen und deren Hilfe anzeigen.

Wenn Sie Näheres über den Server erfahren möchten, verwenden Sie das Apple-Programm "System-Profiler" (im Ordner "/Programme/Dienstprogramme/"). Weitere Informationen über die Verwendung des Programms "System-Profiler" erhalten Sie, indem Sie das Programm öffnen und dessen Hilfe anzeigen.
