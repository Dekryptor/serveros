iCal サービスについて

Mac OS X Server のカレンダーサービスである「iCal サーバ」を使用すると、ワークグループや中小企業、大規模な組織において、カレンダーの共有、会議の予約、イベントの予定が容易になります。社員による相互のスケジュールの確認、会議の企画または提案、会議室の予約、プロジェクタの使用予約などがすばやく簡単に実行可能になります。「iCal サーバ」で、会議の議題や To Do 項目と共に参加依頼が送信され、その返信が集計されます。

Mac OS X v10.6 Snow Leopard を実行するコンピュータでは、「iCal サーバ」を使用するために「iCal」アプリケーションを自動設定することができます。「ユーザの Mac を自動的に設定する」を参照してください。Mac OS X ユーザの場合は、イベントの予定、ほかのユーザへの参加依頼、参加依頼の受信と返信などについては「iCal」を開いて「ヘルプ」メニューを参照してください。

「iCal サーバ」は iPhone のカレンダー、Snow Leopard Server Wiki の Web カレンダー、および標準 CalDAV プロトコルをサポートしているそのほかの一般的なカレンダーアプリケーションにも対応しています。「ユーザのコンピュータを手動で設定する」を参照してください。

インターネット上の「iCal サーバ」へのアクセスを許可したい場合で、ケーブル・ルーター、DSL ルーター、またはそのほかのネットワークルーターを持っている場合は、お使いのルーターで「iCal サーバ」用にポートフォワード（ポートマッピング）を設定する必要があります。ポートフォワードの詳細については、「お使いになる前に」マニュアルを参照してください。

ローカルネットワークの外部の「iCal サーバ」へのアクセスを許可したい場合で、お使いのローカルネットワークに独立したファイアウォール装置がある場合は、ファイアウォールの管理者に問い合わせて、「iCal サーバ」で使用するポートのファイアウォールを解放してもらってください。ポートのリストについては、「お使いになる前に」マニュアルの付録を参照してください。
