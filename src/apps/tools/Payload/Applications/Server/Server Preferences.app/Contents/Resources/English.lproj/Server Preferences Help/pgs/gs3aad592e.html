Starting or Stopping iChat Service

You can stop or start iChat instant messaging service in the iChat pane. While iChat service is off, users can’t send or receive instant messages using the Jabber handles provided by the server.

To start or stop iChat service:

Click the On/Off switch in the iChat pane of Server Preferences.

After starting iChat service, especially if you’re starting it for the first time, you should advise users to set up their instant messaging applications as described in Setting Up Users’ Computers Manually.
