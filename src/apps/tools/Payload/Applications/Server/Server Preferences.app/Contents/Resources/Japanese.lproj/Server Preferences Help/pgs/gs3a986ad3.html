ユーザによるサービスへのアクセスを制御する

「サーバ環境設定」の「ユーザ」パネルで「サービス」をクリックすると、個々のサービスへのユーザのアクセスを制御できます。

ユーザのサービスアクセス設定を変更するには：

  1. 「サーバ環境設定」の「ユーザ」パネルで左側のリストから、制御するユーザアカウントを選択します。

  2. 「サービス」をクリックして、ユーザにアクセスを許可するサービスを選択します。ユーザにアクセスを許可しないサービスの選択を解除します。

    サーバをユーザの Time Machine バックアップの保管場所にする場合は、ユーザのファイル共有へのアクセスを無効にすると、Time Machine バックアップの保管場所へのアクセスも無効になります。
