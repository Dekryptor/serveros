Erstellen eines selbstsignierten Zertifikats

Wenn Ihr Server kein SSL-Zertifikat besitzt oder Sie ein weiteres Zertifikat benötigen, erstellen Sie als ersten Schritt ein selbstsigniertes Zertifikat.

Gehen Sie wie folgt vor, um ein selbstsigniertes Zertifikat zu erstellen:

  1. Öffnen Sie die Servereinstellung "Informationen" und klicken rechts neben "SSL-Zertifikat" auf die Taste "Bearbeiten".

  2. Wählen Sie "Zertifikatsimport" > "Selbst-signiertes Zertifikat erstellen" aus dem Einblendmenü aus.

    Die Option "SSL-Zertifikat verwenden" muss ausgewählt sein, damit das Einblendmenü verfügbar ist.

  3. Geben Sie in das Feld "Name" des Zertifikatsassistenten einen vollständig qualifizierten DNS-Namen für Ihren Server ein (z. B. meinserver.beispiel.de) und klicken Sie auf "Fortfahren".

    Lassen Sie die anderen Einstellungen unverändert. Für "Identitätstyp" sollte "Root, selbst-signiert" und für "Zertifikatstyp" sollte "SSL-Server" festgelegt sein. Die Option "Standardwerte überschreiben" sollte deaktiviert sein.

Sie können das neue selbstsignierte Zertifikat für den Server auswählen. Weitere Informationen hierzu erhalten Sie unter Verwenden eines SSL-Zertifikats.

Sie können auch das neue, selbstsignierte Zertifikat verwenden, um ein signiertes Zertifikat von einer Zertifizierungsstelle anzufordern. Anleitungen hierzu finden Sie unter Anfordern eines signierten Zertifikats.
