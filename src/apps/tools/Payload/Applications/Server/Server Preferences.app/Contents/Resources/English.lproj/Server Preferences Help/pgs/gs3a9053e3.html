Keeping Snow Leopard Server Up to Date

When your server is connected to the Internet, Software Update can automatically get the latest free Snow Leopard Server version, security updates, and other enhancements from Apple. You can have your server check for updates daily, weekly, or monthly. You can also check now.

To check for updates or adjust automatic updating:

  1. Open System Preferences on the server.

  2. Click the Software Update Icon and follow the onscreen instructions.

    Open the Software Update pane of System Preferences

If your organization has another server with Mac OS X Server, your server may get software updates from it rather than from Apple. An expert administrator can set up Mac OS X Server to provide software update service by using Server Admin.

You can also download software updates directly from the Apple Downloads website:  www.apple.com/support/downloads/
