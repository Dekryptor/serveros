Der Server kann Ihre Firewall-Einstellungen nicht laden

Wenn die Firewall-Einstellungen des Servers mit dem Programm "Server-Admin" geändert wurden, können Sie sie möglicherweise nicht in der Systemeinstellung "Sicherheit" verwalten. Trifft dies auf Ihren Server zu, wird in der Systemeinstellung "Sicherheit" ein entsprechender Hinweis angezeigt.
