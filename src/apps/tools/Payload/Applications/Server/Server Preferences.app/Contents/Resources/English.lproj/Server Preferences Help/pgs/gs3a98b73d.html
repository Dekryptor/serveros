Setting Up Users and Groups Management

If your server wasn’t initially set up to manage its own users and groups, you see a “Set Up” button when you view the Users pane or the Groups pane in Server Preferences.

You can configure your server to have its own directory by clicking this Set Up button. This creates an Open Directory domain on your server and makes your server an Open Directory master. For more information about Open Directory, open Server Admin and then use the Help menu to search for “Open Directory Overview,” or see the Mac OS X Server Resources website at www.apple.com/server/macosx/resources/.
