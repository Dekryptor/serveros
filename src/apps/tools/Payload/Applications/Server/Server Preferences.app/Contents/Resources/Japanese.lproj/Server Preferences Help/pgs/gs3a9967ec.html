ユーザのグループメンバーシップを変更する

「ユーザ」パネルで「グループ」をクリックして、ユーザをグループに追加したりグループから取り除いたりできます。

ユーザのグループメンバーシップを変更するには：

  1. 「サーバ環境設定」の「ユーザ」パネルで左側のリストから、変更するユーザアカウントを選択します。

  2. 「グループ」、「メンバーシップを編集」とクリックします。

  3. ユーザをメンバーにする各グループの隣にあるチェックボックスを選択します。ユーザをメンバーにしない各グループの隣にあるチェックボックスの選択を解除します。

  4. 変更し終わったら、「メンバーシップを編集」を再度クリックします。ユーザが所属するグループの静的なリストが表示されます。

グループメンバーはグループのファイル共有フォルダにアクセスでき、各メンバーは自分の iChat のメンバーリストに相手が自動的に追加されるようにできます。 グループメンバーも、Wiki オーナーによってプライベート Wiki へのアクセスが許可されます。グループメンバーによるこれらのサービスへのアクセスは、アクセスが有効になっているかどうかと、グループがアクセスを許可されているかどうかに依存します。

「ワークグループ」という名前の常設グループからユーザを削除することはできません。このグループには常に、全ユーザがメンバーとして含まれています。
