新規に読み込まれたユーザの Mac を設定する

一部のユーザがすでに Mac OS X v10.6 Leopard の設定を完了しており、そのユーザアカウントがすでにディレクトリサーバから読み込まれている場合は、サーバからこれらのユーザに対して、サーバに接続するよう案内するメールを送信することができます。Snow Leopard のユーザは、案内メールに含まれるボタンをクリックすれば、サーバから提供されるサービスを利用できるようにアプリケーションが自動設定されきます。

注記： 読み込まれたユーザが案内メールを受信するには、ディレクトリサーバ上のそのユーザのアカウントにメールアドレスが含まれている必要があります。

Snow Leopard ユーザが案内メールのボタンをクリックすると、「システム環境設定」の「アカウント」パネルが開きます。サーバのユーザアカウントの名前とパスワードを要求されます。

ユーザアカウントの変更およびユーザがアクセスできるサービスについては、「ユーザの Mac を自動的に設定する」を参照してください。

Snow Leopard ユーザに案内メールが届いていない場合は、「ユーザ」パネルのアクションボタンを使って手動で案内メールを送ることができます。操作方法については、「サーバ環境設定」を開いて「ヘルプ」メニューを参照してください。

案内メール用に自動的に生成される標準メッセージテキストに名前、メールアドレス、および自己紹介を追加する方法については、「サーバ案内メールをカスタマイズする」を参照してください。
