Neuste Hilfethemen

Wenn Ihr Computer mit dem Internet verbunden ist, werden neue und aktualisierte Hilfethemen automatisch geladen. Dabei werden möglicherweise auch Informationen in der Einführung aktualisiert, die im Lieferumfang von Mac OS X Server enthalten ist.

  • Öffnen Sie die Mac-Hilfe, klicken Sie auf "Start" (das Haussymbol) und anschließend auf den Link zu den neusten Hilfethemen, um eine Liste mit den aktuellen Themen anzuzeigen. Sehen Sie hier regelmäßig nach, ob neue Hilfethemen veröffentlicht wurden.

  • Verwenden Sie das Suchfeld oben im Fenster, um auf der Mac OS X Server-Support-Website von Apple nach aktuellen Informationen zu suchen. Sie können die Suche aktivieren oder deaktivieren, indem Sie "Auch im Produkt-Support suchen" aus dem Einblendmenü im Suchfeld auswählen.

  • Wenn Sie einen RSS-Feed mit den neuen Aktualisierungen zur Einführung und der Mac OS X Server-Online-Hilfe anzeigen möchten, verwenden Sie einen RSS-Reader wie Safari oder Mail und besuchen Sie folgende Seite:

feed://helposx.apple.com/rss/snowleopard/serverdocupdates.xml
