Modification de l’adresse IP de votre serveur

L’adresse IP du serveur fait partie des réglages de connexion au réseau de la sous-fenêtre Réseau des Préférences Système sur le serveur. Pour plus d’informations sur la modification de l’adresse IP des préférences Réseau, ouvrez les Préférences Système sur le serveur, puis utilisez le menu Aide.

Si la connexion au réseau de votre serveur est configurée à l’aide de DHCP, ce serveur doit disposer d’une adresse IP statique (toujours la même) attribuée par votre serveur DHCP. Dans ce cas, pour modifier l’adresse IP de votre serveur, le serveur DHCP doit être reconfiguré. Si vous utilisez une borne d’accès AirPort ou un autre routeur proposant un service DHCP, vous pouvez modifier l’adresse IP de votre serveur en reconfigurant la fonctionnalité Réservations DHCP ou mappage statique.

Important :  après avoir modifié l’adresse IP du serveur, vous devez mettre à jour le serveur DNS de votre réseau afin que le nom DNS du serveur pointe vers la nouvelle adresse IP. En outre, une recherche inversée de la nouvelle adresse IP doit pointer vers le nom DNS de votre serveur.

Si votre serveur fournit son propre service DNS, vous pouvez utiliser Admin Serveur pour mettre ce service à jour. Si votre service DNS provient de votre fournisseur d’accès à Internet ou d’un autre serveur du réseau, demandez à votre FAI ou à l’administrateur de ce serveur DNS de mettre à jour les enregistrements DNS de votre serveur.

La modification de l’adresse IP de votre serveur peut perturber les connexions des ordinateurs des utilisateurs Mac OS X 10.6. Dans ce cas, les utilisateurs doivent supprimer le serveur dans leur liste de serveurs d’annuaire, puis l’y remettre. Pour en savoir plus, consultez Configuration d’un Mac à l’aide des préférences Comptes.
