Starten oder Stoppen des Diensts "VPN"

Sie können den Dienst "VPN" für den Fernzugriff in der Servereinstellung "VPN" stoppen oder starten. Wenn der Dienst "VPN" deaktiviert ist, können Benutzer keine entfernten Verbindungen zum Server über das Internet herstellen.

Gehen Sie wie folgt vor, um den Dienst "VPN" zu starten oder zu stoppen:

Klicken Sie auf die Taste "Ein/Aus" in der Servereinstellung "VPN".

Nach dem Starten des Diensts "VPN", besonders nach dem erstmaligen Starten, sollten Sie die Benutzer darauf hinweisen, wie sie ihre VPN-Verbindungen einrichten können. Näheres dazu wird unter Einrichten einer VPN-Verbindung auf Benutzercomputern mittels Konfigurationsdatei und Manuelles Einrichten einer VPN-Verbindung auf Benutzercomputern beschrieben.
