自己署名証明書を置き換える

認証局から署名入り証明書を受領したら、それを自分の自己署名証明書と置き換えることができます。

自己署名証明書を置き換えるには：

  1. 「サーバ環境設定」を使用しているとき、認証局から受領した署名入り証明書ファイルを Finder で探し、そのファイルを目に見える任意の場所（デスクトップなど）に置きます。

  2. 「サーバ環境設定」の「情報」パネルで、「SSL 証明書」の右側にある「編集」ボタンをクリックします。

  3. ポップアップメニューで、置き換えたい自己署名証明書を選択します。

    「SSL 証明書を使用」が選択されていないと、ポップアップメニューは表示されません。

  4. ポップアップメニューで、「証明書署名」＞「署名または更新された証明書に置き換え」と選択します。

  5. 署名入り証明書が含まれたファイルを、「サーバ環境設定」にあるダイアログの中央にドラッグして、「証明書を置き換える」をクリックします。
