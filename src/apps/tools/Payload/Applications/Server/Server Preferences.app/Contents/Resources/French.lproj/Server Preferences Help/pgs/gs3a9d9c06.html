Configuration manuelle des ordinateurs des utilisateurs

Les utilisateurs Windows ou Mac OS X 10.5 ou antérieur peuvent bénéficier des services de votre serveur en configurant manuellement leurs applications. Ils peuvent utiliser les réglages ci-dessous, en remplaçant les paramètres fictifs en italique par le nom DNS de votre serveur et par leur nom abrégé d’utilisateur. Normalement, pour les utilisateurs Mac OS X 10.6, une invitation à configurer automatiquement leurs applications s’affiche mais ils peuvent également configurer leurs applications manuellement. Pour obtenir des informations sur cette invitation, consultez Configuration de Mac qui se connectent à votre réseau.

Réglages des applications

Finder ou autre client de partage de fichiers SMB ou AFP

afp://monserveur.exemple.com

smb://monserveur.exemple.com

Carnet d’adresses (avec Mac OS X 10.6) ou autre application de contacts CardDAV

Adresse du serveur : monserveur.exemple.com

Nom d’utilisateur : le nom du compte utilisateur sur le serveur

Mot de passe : le mot de passe du compte utilisateur sur le serveur

iCal (avec Mac OS X 10.6)

Type de compte : automatique (CalDAV)

Adresse électronique : nom_abrégé_utilisateur@monserveur.exemple.com

iCal (avec Mac OS X Server 10.5) ou autre application de calendrier CalDAV

Abonnement à un calendrier personnel (en lecture seule) : http://monserveur.exemple.com:8008/calendars/users/nom_abrégé_utilisateur/calendar/

iChat ou autre application de messagerie instantanée XMPP

Type de compte :  Jabber

Adresse du serveur : monserveur.exemple.com

Identifiant Jabber : nom_abrégé_utilisateur@monserveur.exemple.com

Authentification : de préférence Kerberos 5 (autres méthodes standard prises en charge)

Port :  5223

Mail ou autre application de courrier électronique

Type de compte :  IMAP ou POP

Serveur de courrier entrant : monserveur.exemple.com

Serveur de courrier sortant : monserveur.exemple.com

Adresse électronique : nom_abrégé_utilisateur@monserveur.exemple.com

Authentification : de préférence Kerberos 5 (autres méthodes standard prises en charge)

Safari ou autre navigateur web

Site web du serveur : http://monserveur.exemple.com

Cliquez sur les liens pour afficher Ma page, les wikis, les blogs, les calendriers et webmail.
