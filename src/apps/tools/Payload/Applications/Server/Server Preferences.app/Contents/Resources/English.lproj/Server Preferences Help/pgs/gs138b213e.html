Changing Custom Website Settings

You can change a custom website’s IP address, access port, site folder location, and access privileges.

To change website settings:

  1. In the Web pane of Server Preferences, click Custom Sites.

  2. Select the website you want to change, click the Edit (pencil) button, and change settings using the following information as a guide:

    • IP Address: 
      If the server has more than one IP address, choose one of them, or choose Any to allow users to access the website using any of them.
    • Port: 
      Enter an access port number. The standard access port for websites is 80. Before entering a different port, make sure it doesn’t conflict with another service provided by your server. For a list of ports Mac OS X Server may use, open Server Admin (in /Applications/Server/) and use the Help menu to search for “TCP and UDP port reference.”
    • Store Site Files in: 
      Choose the folder that contains the files you want to use for the website.
    • Who Can View This Site: 
      Choose a group to allow only its members to access the entire website, or choose Customize to restrict access to individual folders in the website folder. For instructions, see Restricting Access to a Custom Website.
