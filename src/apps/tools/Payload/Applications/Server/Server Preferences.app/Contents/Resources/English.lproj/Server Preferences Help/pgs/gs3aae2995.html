Setting Up Wikis

If the Wikis service is turned on in the Web pane, all server users can create wikis by using a web browser.

To turn the Wikis service on or off:

In the Web pane, click Web Services and then select or deselect Wikis.

If this option is selected and web services are on, clicking the link arrow next to it goes to the list of all wikis.

When wikis are available, users can access their own wiki pages and wikis that they have permission to use. Permission to use a wiki can be changed in the Settings section of the wiki. The wiki owner or a server administrator can make changes in the Settings section of a wiki.

Users can access wikis by appending /groups/ to your serverâ€™s website address. For example:

http://server.example.com/groups/

Users with accounts in the Users pane of Server Preferences can create new wikis after logging in to their My Page wiki portal.

For information about using a wiki, click Help on any wiki page.
