Importieren eines Zertifikats

Wenn Sie bereits Dateien mit einem SSL-Zertifikat und passendem privaten Schlüssel haben, können Sie diese importieren und anschließend mithilfe des Zertifikats die von Ihrem Server bereitgestellten Dienste schützen.

Gehen Sie wie folgt vor, um ein Zertifikat zu importieren:

  1. Suchen Sie im Finder nach den Dateien, die das Zertifikat und den passenden privaten Schlüssel enthalten und legen Sie diese so ab, dass Sie sie während der Verwendung des Programms "Servereinstellungen" sehen können, z. B. auf dem Schreibtisch.

  2. Öffnen Sie die Servereinstellung "Informationen" und klicken rechts neben "SSL-Zertifikat" auf die Taste "Bearbeiten".

  3. Wählen Sie "Zertifikatsimport" > "Zertifikat importieren" aus dem Einblendmenü aus.

    Die Option "SSL-Zertifikat verwenden" muss ausgewählt sein, damit das Einblendmenü verfügbar ist.

  4. Bewegen Sie die Dateien mit dem Zertifikat und dem privaten Schlüssel in die Mitte des Dialogfensters der Servereinstellungen und klicken Sie dann auf "Importieren".

  5. Wählen Sie das importierte Zertifikat aus dem Einblendmenü aus, um es zu verwenden.
