サーバの設定を完了したら

サーバの設定を完了したら、以下のことが可能です：

  • 「Mac OS X Server 次のステップ」マニュアル（初期設定の完了後に生成されサーバのデスクトップに置かれます）のアドバイスに従って、新しいサーバのセキュリティ、アクセシビリティ、および利便性を拡張します。

  • いくつかの手順に従って、サーバの安全性を保ちます。詳細については、「サーバの安全性を保つ」および「System Administrator（root）アカウントを保護する」を参照してください。

  • 所属する組織に、初期設定中に新しいサーバに接続（バインド）しなかった Open Directory サーバ、Active Directory サーバ、またはそのほかのディレクトリサーバがある場合は、すぐに接続できます。操作方法については、「サーバをディレクトリサーバに接続する」を参照してください。

  • 「ソフトウェア・アップデート」を使って、入手可能な Mac OS X Server のアップデートをインストールします。詳細については、「Snow Leopard Server を最新の状態に保つ」を参照してください。

  • ネットワークを保護しながらもインターネット上の特定サービスにはユーザがアクセスできるように、AirMac ベースステーションまたはインターネットルーターを設定します。詳細については、「お使いになる前に」マニュアルの第 2 章「Mac OS X Server の準備をする」を参照してください。

  • 小規模な組織用に、あるいは中規模または大規模な組織のワークグループ用に単独のサーバを設定する場合は、「サーバ環境設定」を使って、ユーザおよびグループを設定し、サービスおよびシステム情報をカスタマイズし、サーバの利用状況を監視します。これらの作業については、「アカウントを管理する」、「サービス設定をカスタマイズする」、および「サーバシステムを管理する」を参照してください。

    「Dashboard」の「サーバ状況」ウィジェットを使って、サーバを監視することもできます。詳細については、「サーバ状況ウィジェットを使用する」を参照してください。

  • 高度な設定を変更したり、高度なサービスを設定したり、高度なユーザ設定およびグループ設定を変更したり、「サーバ管理」、「ワークグループマネージャ」、そのほかの Mac OS X アプリケーション、または UNIX コマンドラインツールを使ってユーザのコンピュータを管理したりします。これらのアプリケーションやツールについては、各アプリケーションを開いて「ヘルプ」メニューを参照するか、「Mac OS X Server ガイド」の Web サイト（www.apple.com/jp/server/macosx/resources/）を参照してください。
