Verwalten des Diensts "Adressbuch"

Verwenden Sie die Servereinstellung "Adressbuch", um den Dienst "Adressbuch" zu aktivieren oder zu deaktivieren oder um den Speicherplatz jedes Benutzers für Kontaktdaten auf der Festplatte einzuschränken.

Gehen Sie wie folgt vor, um den Bereich "Adressbuch" zu konfigurieren:

Ändern Sie die aktuellen Werte der angezeigten Optionen und beachten Sie dabei die folgenden Informationen.

  • Ein/Aus: 
    Gibt den Status des Diensts "Adressbuch" an. Durch Klicken auf die Taste "Ein/Aus" wird der Dienst aktiviert oder deaktiviert.
  • Adressbuchgröße pro Benutzer beschränken auf __ MB: 
    Legt fest, wie viel Speicherplatz das Adressbuch jedes Benutzers auf dem Server belegen darf. Überschreitet ein Benutzer diese Größenvorgabe, sendet der Server eine Fehlermeldung an das Programm "Adressbuch" des betreffenden Benutzers.
