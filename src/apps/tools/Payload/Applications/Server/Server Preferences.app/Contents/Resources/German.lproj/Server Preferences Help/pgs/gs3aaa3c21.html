Einrichten von Blogs
Gehen Sie wie folgt vor, um den Dienst für Blogs zu aktivieren oder zu deaktivieren:

Klicken Sie im Bereich "Web" auf "Webdienste" und aktivieren oder deaktivieren Sie dann "Blogs".

Wenn diese Option ausgewählt ist und die Webdienste aktiviert sind, wird durch Klicken auf den als Link definierten Pfeil daneben die Liste aller Blogs geöffnet.

Benutzer können auf ihre Blogs zugreifen, indem sie den Zusatz "/users/" an die Website-Adresse des Servers anhängen. Hier ein Beispiel:

http://server.beispiel.de/users/

Informationen über die Verwendung von Blogs erhalten Sie, indem Sie auf einer Wiki- oder Blog-Seite auf "Hilfe" klicken.
