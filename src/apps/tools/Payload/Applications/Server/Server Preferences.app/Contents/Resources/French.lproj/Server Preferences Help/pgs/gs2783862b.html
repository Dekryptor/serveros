Installation à distance

En utilisant Admin Serveur et Assistant du serveur à partir d’un ordinateur administrateur, vous pouvez installer Mac OS X Server sur un autre ordinateur via le réseau. L’ordinateur sur lequel vous l’installez n’a pas besoin d’un écran mais doit être équipé d’un lecteur de DVD pour lire le disque d’installation de Mac OS X Server. Si l’ordinateur n’est pas équipé d’un lecteur de DVD intégré, vous pouvez y brancher un lecteur de DVD FireWire externe ou un Mac qui a un lecteur de DVD et fonctionne en mode disque cible.

Vous pouvez effectuer :

  • une nouvelle installation de Mac OS X Server sur un disque où Mac OS X Server ou Mac OS X ne sont pas déjà installés ;

  • une installation spéciale, qui installe Mac OS X Server après avoir effacé et formaté un disque cible ;

  • une mise à niveau d’un serveur avec un processeur Intel et Mac OS X Server 10.5 ou Mac OS X Server 10.4.11 (pour obtenir des informations sur d’autres options de mise à niveau ou de migration, ouvrez Admin Serveur et allez ensuite dans le menu Aide, ou consultez le site web des ressources de Mac OS X Server à l’adresse www.apple.com/fr/server/macosx/resources/ (en anglais).

Pour installer Mac OS X Server à distance :

  1. Si vous prévoyez d’effacer le disque cible ou la partition, assurez vous d’en avoir fait une copie de sauvegarde et, si vous le souhaitez, utilisez Utilitaire de disque pour préparer le disque cible.

    Si vous n’avez besoin d’effacer que le disque cible en vous servant du format le plus courant, Mac OS Étendu (journalisé), vous n’avez pas besoin d’utiliser Utilitaire de disque. Avec Utilitaire de disque, vous pouvez effacer le disque cible en utilisant d’autres formats, partitionner le disque dur du serveur et créer un ensemble RAID.

  2. Démarrez le serveur cible avec le disque d’installation Mac OS X Server.

    Si le serveur cible est équipé d’un lecteur de DVD intégré, insérez le disque et redémarrez ensuite votre ordinateur tout en maintenant la touche C de votre clavier enfoncée. Relâchez la touche C quand vous voyez le logo Apple.

    Si le serveur cible est équipé d’un lecteur de DVD FireWire externe, redémarrez votre ordinateur tout en maintenant la touche Option enfoncée, sélectionnez l’icône représentant le disque d’installation Mac OS X Server et cliquez ensuite sur la flèche.

  3. Sur un ordinateur administrateur, ouvrez Admin Serveur et sélectionnez Prêt pour installer dans la liste à gauche.

    Admin Serveur est situé dans /Applications/Serveur/.

    Si Admin Serveur vous demande un mot de passe pour vous connecter à un serveur qui est déjà configuré sur votre réseau, vous pouvez cliquez sur Annuler. Il n’est pas nécessaire d’avoir un compte administrateur pour installer Mac OS X Server à distance.

  4. Sélectionnez le serveur cible sur la droite et cliquez ensuite sur Installer.

    Si le serveur dont vous avez besoin n’est pas listé, vous pouvez cliquez sur le bouton Actualiser (flèche courbe) pour qu’Admin Serveur cherche de nouveau des serveurs prêts à l’installation sur votre réseau local.

    Si le serveur dont vous avez besoin est sur un réseau local différent, sélectionnez Serveur > Installer un serveur distant.

  5. Quand Assistant du serveur s’ouvre, confirmez l’adresse du serveur cible, saisissez les huit premiers caractères du numéro de série matériel intégré du serveur en tant que mot de passe et cliquez sur Continuer.

    Si le champ d’Adresse est vide ou incorrect, saisissez le nom DNS du serveur ou son adresse IP au format IPv4 (par exemple, 192.0.2.200).

    Un serveur DHCP sur le réseau peut attribuer automatiquement une adresse IP aux serveurs qu’Assistant du serveur trouve sur le réseau local (sous-réseau IP). S’il n’existe pas de serveur DHCP, le serveur cible utilise une adresse au format 169.254.xxx.xxx unique parmi les serveurs du réseau local. Vous pourrez changer l’adresse IP par la suite, quand vous configurerez le serveur.

    Pour trouver le numéro de série, cherchez une étiquette collée sur le serveur. Respectez la casse du numéro de série quand vous le saisissez.

    Pour un Xserve avec processeur Intel dont la carte logique principale a été remplacée et qui n’a pas de numéro de série, saisissez « System S » (sans les guillemets) comme mot de passe. Pour les autres ordinateurs qui n’ont pas de numéro de série matériel intégré, utilisez 12345678 comme mot de passe.

  6. Sélectionnez la langue dans laquelle vous voulez utiliser Mac OS X Server et cliquez sur Continuer.

    La langue que vous choisissez n’a pas d’effet sur la langue des ordinateurs des utilisateurs.

  7. Sélectionnez un disque ou un volume de destination (partition) et cliquez sur Installer.

  8. Si Mac OS X Server ou Mac OS X sont déjà installés sur le volume que vous avez choisi, sélectionnez une option disponible et cliquez ensuite sur OK.

    Vous pouvez choisir entre :

    • Effacer : 
      Cette option efface complètement le volume de destination avant d’installer une nouvelle copie de Mac OS X Server.
    • Mise à niveau : 
      Cette option est disponible seulement si le volume de destination a Mac OS X Server 10.5 Leopard ou Mac OS X Server 10.4.11 Tiger. Vous pouvez mettre ce volume à niveau vers Snow Leopard Server sans effacer le volume de destination. Pour obtenir des informations sur la mise à niveau et la migration, ouvrez Admin Serveur et allez ensuite dans le menu Aide, ou consultez le site web des ressources de Mac OS X Server à l’adresse www.apple.com/fr/server/macosx/resources/ (en anglais).

  9. Une fois l’installation terminée, le serveur cible redémarre et vous pouvez cliquer sur Plus d’options pour configurer le serveur à distance ou pour installer Mac OS X Server sur un autre ordinateur.

  10. Si vous avez cliqué sur Plus d’options, choisissez maintenant ce que vous voulez faire avec Assistant du serveur.

    • Installer Mac OS X Server à distance : 
      Recommencez cette procédure en ignorant l’étape 3.
    • Configurer Mac OS X Server à distance : 
      Pour obtenir des instructions, consultez Configuration d’un serveur à distance.
    • Préparer et enregistrer les informations pour la configuration automatique : 
      Cela vous permet de parcourir la procédure de configuration, en choisissant les options de configuration et en saisissant les données de configuration. Ensuite, au lieu d’utiliser les informations de configuration pour configurer un serveur immédiatement, enregistrez-les en tant que profil de configuration automatique sur un disque ou un lecteur amovible. Le profil de configuration automatique sauvegardé vous permet ensuite d’automatiser la configuration d’un ou plusieurs serveurs. Pour en savoir plus, consultez la feuille d’opérations Installation et configuration. (Elle se trouve dans le dossier Documentation sur le disque d’installation de Mac OS X Server.)

Au lieu d’utiliser Assistant du serveur sur un ordinateur administrateur, vous pouvez contrôler à distance l’installation en utilisant le partage d’écran sur un Mac équipé de Mac OS X 10.5 ou un Mac OS X 10.6. Sur un autre Mac, vous pouvez vous servir d’Apple Remote Desktop (vendu séparément). Pour obtenir des informations, ouvrez Admin Serveur et allez ensuite dans le menu Aide et visitez le site web de Apple Remote Desktop à l’adresse www.apple.com/fr/remotedesktop/.
