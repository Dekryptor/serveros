Starting or Stopping File Sharing Service

You can start or stop file sharing in the File Sharing pane. While file sharing is off, users can’t access shared folders or use the server for Time Machine backup storage.

To start or stop file sharing:

Click the On/Off switch in the File Sharing pane of Server Preferences.

After starting file sharing service, especially if you’re starting it for the first time, you should tell users they can access shared folders by connecting to afp://myserver.example.com or smb://myserver.example.com (replacing the italicized placeholders with your server’s DNS name).
