Démarrage ou arrêt du service Partage de fichiers

Vous pouvez démarrer ou arrêter le partage de fichiers dans la sous-fenêtre Partage de fichiers. Lorsque le partage de fichiers est désactivé, les utilisateurs ne peuvent pas accéder aux dossiers partagés ni utiliser le serveur pour le stockage de sauvegarde Time Machine.

Pour démarrer ou arrêter le partage de fichiers :

Cliquez sur le bouton Marche/Arrêt de la sous-fenêtre Partage de fichiers des Préférences du serveur.

Après avoir démarré le service de partage de fichiers, en particulier s’il s’agit de la première fois, vous devez dire aux utilisateurs qu’ils peuvent accéder aux dossiers partagés en se connectant à afp://monserveur.exemple.com ou à smb://monserveur.exemple.com (en remplaçant les paramètres fictifs en italique par le nom DNS de votre serveur).
