Si le serveur ne peut pas charger les réglages de votre coupe-feu

Si les réglages du coupe-feu du serveur ont été modifiés à l’aide d’Admin Serveur, vous risquez de ne pas pouvoir les gérer dans la sous-fenêtre Sécurité des Préférences Système. Si tel est le cas pour votre serveur, un message à cet effet s’affiche dans la sous-fenêtre Sécurité.
