Mac OS X Server im Überblick

Mac OS X Server 10.6 Snow Leopard kombiniert die Benutzerfreundlichkeit von Mac-Computern mit einer ausgereiften und stabilen UNIX-Grundlage. Mit seiner umfassenden Palette an Diensten unterstützt der Server Client-Computer mit Macintosh-, Windows- und UNIX-Betriebssystemen über das Netzwerk.

Die Snow Leopard Server-Lösung bietet kleinen Unternehmen und Arbeitsgruppen ohne IT-Abteilung die Möglichkeit, alle Vorteile und Vorzüge zu nutzen, die ein Server bietet. Auch ein Benutzer ohne spezifisches technisches Wissen kann Snow Leopard Server für eine Gruppe konfigurieren und verwalten, und andere Benutzer innerhalb der Gruppe können ihre Mac-Computer so konfigurieren, dass sie auf die von Snow Leopard Server bereitgestellten Dienste zugreifen können. Für professionelle IT-Benutzer unterstützt Snow Leopard Server andererseits eine Fülle erweiterter Optionen und Werkzeuge für die Konfiguration und Verwaltung.

Benutzer und Gruppen

Jeder Benutzer, der auf die Dienste Ihres Servers zugreifen möchte, benötigt einen Benutzer-Account. Ein Benutzer-Account erlaubt es einer Person, sich bei Diensten anzumelden. Er steuert, welche Dienste die Person nutzen darf, enthält die Kontaktinformationen der Person und definiert die Gruppenmitgliedschaften der Person.

Wenn Sie Ihren Server das erste Mal einrichten, legen Sie fest, woher der Server Benutzer- und Gruppen-Accounts bezieht. Möglicherweise erstellen Sie mithilfe des Programms "Servereinstellungen" auf dem Server eigene Benutzer- und Gruppen-Accounts. Oder Ihr Server ist so konfiguriert, dass Benutzer-Accounts von einem Verzeichnisserver Ihres Unternehmens importiert werden können. Eine weitere Möglichkeit neben dem Erstellen bzw. Importieren von Benutzer-Accounts besteht darin, dass der Server auf Benutzer und Gruppen zugreift, die sich auf dem Verzeichnisserver Ihres Unternehmens befinden.

Sie können auch Gruppen einrichten, Gruppendienste konfigurieren und Gruppenmitgliedschaften ändern.

Informationen zu Diensten und System

Ihr Server bietet Benutzern zahlreiche Dienste wie Dateifreigabe, Adressbuch, Kalender, Sofortnachrichten, Mail, Web und VPN. Sie können die Servereinstellungen anpassen und Dienste aktivieren und deaktivieren. Zudem können Sie Systeminformationen, Protokolle und Graphen überprüfen. Ebenfalls ist es möglich, Systeminformationen, Firewall-Einstellungen und Einstellungen für die Datensicherung der Benutzercomputer zu ändern.
