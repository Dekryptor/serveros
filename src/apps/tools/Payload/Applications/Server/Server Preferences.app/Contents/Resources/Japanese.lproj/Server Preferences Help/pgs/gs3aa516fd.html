ファイル共有サービスを開始する／停止する

「ファイル共有」パネルでファイル共有を開始または停止できます。ファイル共有が切のときは、ユーザは共有フォルダにアクセスしたり、サーバを Time Machine バックアップの保管場所として使ったりすることはできません。

ファイル共有を開始または停止するには：

「サーバ環境設定」の「ファイル共有」パネルで「入」／「切」スイッチをクリックします。

ファイル共有サービスを開始した後、特にこのサービスをはじめて開始した場合は、afp://myserver.example.com または smb://myserver.example.com（斜体の部分はサーバの DNS 名に置き換えます）に接続することによって共有フォルダにアクセスできることをユーザに通知してください。
