Read Me 書類（Mac OS X Server Install Disc および Administration Tools ディスクに収録されています）

重要なアップデートや特別な情報を記載しています。

Mac OS X Server の Web サイト（www.apple.com/jp/server/macosx/）

製品およびテクノロジーに関するさまざまな情報を入手できます。

Mac OS X Server のサポート Web サイト（www.apple.com/jp/support/macosxserver/）

アップルのサポート部門から寄せられた数百もの記事を利用できます。

アップルのトレーニングと認定資格の Web サイト（www.apple.com/jp/training/）

インストラクターの指導によるトレーニングや自分のペースで進められるトレーニングを通じてサーバ管理のスキルを磨き、資格を取って差を付けましょう。

アップルのディスカッション Web サイト（discussions.info.apple.co.jp）

質問、知識、およびアドバイスをほかの管理者と共有しましょう。

アップルのメーリングリスト Web サイト（www.lists.apple.com）

メーリングリストに登録して、メールを使ってほかの管理者と意見の交換ができます（英語による情報です）。
