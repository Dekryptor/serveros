About File Sharing Service

Mac OS X Server file sharing service lets users access shared folders and store files on the server. They can use Macintosh, Windows, or UNIX computers to access their files and shared folders without special software, using native file protocols such as AFP and SMB. Windows users see Mac OS X Server file servers in their Network Places, just like Windows file servers.

If you want to allow access to file sharing on the Internet and you have a cable router, DSL router, or other network router, your router must have port forwarding (port mapping) configured for file sharing. For more information about port forwarding, see the Getting Started manual.

If you want to allow access to file sharing outside your local network and your local network has a separate firewall device, ask the firewall administrator to open the firewall for the ports that file sharing uses. For a list of ports, see the appendix in the Getting Started manual.
