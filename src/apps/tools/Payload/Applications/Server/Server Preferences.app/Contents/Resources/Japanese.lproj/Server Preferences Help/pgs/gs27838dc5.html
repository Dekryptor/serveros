高度なアプリケーションを使ってサービスをカスタマイズする

「サーバ環境設定」を使えば基本的なサービスを簡単にカスタマイズできますが、「サーバ管理」や、「高度なツールとアプリケーション」に一覧されているそのほかの高度なアプリケーションおよびツールも利用できます。高度なアプリケーションやツールを使って、数多くの追加のサービス設定をカスタマイズすることができます。「サーバ環境設定」に表示されていないサービス（Podcast プロデューサー、QuickTime Streaming Server など）を有効にすることもできます。高度なサービス、オプション、およびアプリケーションについては、「サーバ管理」を開いて「ヘルプ」メニューを参照するか、Mac OS X Server リソースの Web サイト（www.apple.com/jp/server/macosx/resources/）を参照してください。

重要： 通常は「サーバ環境設定」で操作できるサービス設定を、「サーバ管理」またはそのほかの高度なアプリケーションを使って変更する場合は、後で元に戻す場合に備えて、高度なアプリケーションの現在の設定を事前に慎重に確認してください。たとえば、各パネルおよびダイアログの設定を変更する場合は、あらかじめそれらのスクリーンショットを保存しておきます。（スクリーンショットの取得について詳しくは、Finder に切り替えて「ヘルプ」メニューを参照してください。）
