Wiki を設定する

「Web」パネルで Wiki が有効になっている場合は、サーバのユーザ全員が Web ブラウザを使って Wiki を作成できます。

Wiki サービスを有効または無効にするには：

「Web」パネルで、「Web サービス」をクリックして、「Wiki」を選択または選択解除します。

このオプションが選択されていて Web サービスが有効になっている場合は、その隣の矢印のリンクをクリックすると、すべての Wiki がリストで表示されます。

Wiki が使用可能なとき、ユーザは自分の Wiki ページおよび使用権限のある Wiki にアクセスすることができます。Wiki の使用権限は、Wiki の「設定」セクションで変更できます。Wiki オーナーまたはサーバの管理者は、Wiki の「設定」セクションを変更することができます。

ユーザが Wiki にアクセスするには、「/groups/」をサーバの Web サイトアドレスの末尾に付加します。たとえば、次のように入力します：

http://server.example.com/groups/

「サーバ環境設定」の「ユーザ」パネルにアカウントが表示されるユーザは、自分の「マイページ」の Wiki ポータルにログインした後、新しい Wiki を作成できます。

Wiki の使用方法については、任意の Wiki ページの「ヘルプ」をクリックしてください。
