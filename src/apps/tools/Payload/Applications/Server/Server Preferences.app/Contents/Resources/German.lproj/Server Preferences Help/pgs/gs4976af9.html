Nach der Serverkonfiguration

Nachdem Sie einen Server eingerichtet haben, haben Sie folgende Möglichkeiten:

  • Optimieren Sie Sicherheit, Verfügbarkeit und Nutzen Ihres neuen Servers, indem Sie die Hinweise im Dokument "Mac OS X Server Nächste Schritte" befolgen, das nach der Erstkonfiguration des Servercomputers generiert und auf dem Schreibtisch platziert wird.

  • Sie können einige zusätzliche Schritte ausführen, die zur Sicherheit des Servers beitragen. Informationen hierzu finden Sie unter Gewährleisten der Serversicherheit und Schützen des Accounts für den Systemadministrator (root).

  • Wenn Ihr Unternehmen einen Open Directory-Server, Active Directory-Server oder einen anderen Verzeichnisserver verwendet, mit dem Sie Ihren Server bei der Erstkonfiguration nicht verbunden haben, können Sie dies jetzt nachholen. Anleitungen hierzu finden Sie unter Verbinden Ihres Servers mit einem Verzeichnisserver.

  • Verwenden Sie die Funktion "Softwareaktualisierung", um verfügbare Mac OS X Server-Aktualisierungen zu installieren. Informationen hierzu finden Sie unter Aktualisieren von Snow Leopard Server.

  • Konfigurieren Sie eine AirPort-Basisstation oder einen Internet-Router, um Ihr Netzwerk zu schützen und Benutzern gleichzeitig die Möglichkeit zu geben, über das Internet auf ausgewählte Dienste zuzugreifen. Weitere Informationen finden Sie in Kapitel 2 "Vorbereitung für die Installation von Mac OS X Server" der Einführung.

  • Wenn Sie einen einzelnen Server für ein kleines Unternehmen oder einen Server für eine Arbeitsgruppe in einem mittleren oder großen Unternehmen einrichten, verwenden Sie das Programm "Servereinstellungen", um Benutzer und Gruppen zu konfigurieren, Dienste und Systeminformationen anzupassen und die Serveraktivität zu überwachen. Informationen zu diesen Aufgaben finden Sie unter Verwalten von Accounts, Anpassen der Diensteinstellungen und Verwalten des Serversystems.

    Zum Überwachen der Serveraktivitäten steht Ihnen außerdem das Dashboard-Widget "Serverstatus" zur Verfügung. Informationen hierzu finden Sie unter Widget "Serverstatus".

  • Ändern Sie erweiterte Einstellungen, konfigurieren Sie erweiterte Dienste, ändern Sie erweiterte Benutzer- und Gruppeneinstellungen und verwalten Sie Benutzercomputer mit den Programmen "Server-Admin", "Arbeitsgruppenmanager", anderen Mac OS X-Programmen oder UNIX-Befehlszeilenprogrammen. Informationen über diese Programme finden Sie im Menü "Hilfe" im jeweiligen Programm oder auf der Mac OS X Server-Ressourcen-Website unter www.apple.com/de/server/macosx/resources/.
