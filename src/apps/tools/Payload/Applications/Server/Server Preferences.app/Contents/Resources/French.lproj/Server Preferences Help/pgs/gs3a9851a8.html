Ajout d’un compte utilisateur

Vous pouvez ajouter des comptes utilisateur pour toute personne utilisant les services fournis par votre serveur. Votre serveur attribue à chaque compte utilisateur sa propre adresse électronique, une adresse iChat, un calendrier personnel et un portail de wiki Ma page. Les comptes utilisateur peuvent aussi avoir accès à des wikis, des blogs et des calendriers web, à un carnet d’adresses basé sur le serveur, aux fichiers partagés du serveur et au stockage de sauvegarde Time Machine. Le VPN leur permet d’accéder au serveur à distance. Pour être disponibles, les services doivent être activés. Il est possible de contrôler individuellement l’accès de chaque compte utilisateur aux différents services.

Pour ajouter un nouveau compte utilisateur :

  1. Cliquez sur le bouton Ajouter (+) de la sous-fenêtre Utilisateurs de Préférences du serveur.

  2. Si un menu local s’affiche, choisissez l’une des commandes suivantes :

    • Importer un utilisateur à partir de l’annuaire : 
      Vous pouvez importer des comptes utilisateur existants depuis le serveur d’annuaire de votre organisation. Pour obtenir des instructions, consultez Importation d’utilisateurs.
    • Créer un utilisateur : 
      Lorsque cela est possible, vous ajoutez généralement de nouveaux utilisateurs à partir du serveur d’annuaire de votre organisation. Vous pouvez toutefois créer des comptes utilisateur dans l’annuaire de votre serveur. Pour réaliser cette opération, passez à l’étape 3.

    Si aucun menu local n’apparaît lorsque vous cliquez sur le bouton Ajouter (+), votre serveur n’est pas connecté à un serveur de répertoire. Passez à l’étape 3 pour créer un compte utilisateur dans le répertoire de votre serveur.

  3. Saisissez le nom de l’utilisateur.

    Le nom peut comporter jusqu’à 255 caractères (et se réduit à seulement 85 caractères japonais). Ce nom peut comporter des espaces.

  4. Si vous ne souhaitez pas utiliser le nom abrégé généré, tapez un nom abrégé différent.

    Vous ne pouvez pas modifier ce nom abrégé une fois le compte créé.

    Le nom abrégé se limite généralement à huit caractères, mais il peut comporter jusqu’à 255 caractères latins. Utilisez uniquement les caractères a à z, A à Z, 0 à 9, . (point), _ (souligné) et - (tiret).

    Remarque :   si un utilisateur possède déjà un nom abrégé dans Mac, essayez d’utiliser le même nom abrégé pour le compte utilisateur sur le serveur. Cela lui facilite l’accès aux services.

  5. Saisissez le mot de passe utilisateur dans les champs Mot de passe et Confirmer.

    Vous pouvez utiliser « Assistant mot de passe » pour vous aider à choisir un mot de passe. Cliquez sur le bouton de clé à droite du champ Mot de passe pour vérifier la confidentialité du mot de passe. L’utilisateur peut modifier son mot de passe via la sous-fenêtre Comptes dans Préférences du serveur à partir de son ordinateur.

  6. Sélectionnez « Autoriser l’utilisateur à administrer l’ordinateur » si ce compte utilisateur a besoin de créer d’autres comptes utilisateur ou des groupes, d’installer des logiciels sur le serveur ou de modifier des réglages du serveur.
