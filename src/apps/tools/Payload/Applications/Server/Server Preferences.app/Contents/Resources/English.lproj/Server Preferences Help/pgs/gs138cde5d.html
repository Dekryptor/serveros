Changing Your Server’s DNS Name

The server’s DNS name can be changed only by the administrator of your DNS service.

  • If your ISP provides your DNS service, ask your ISP for help.

  • If another server on your network provides DNS service, ask the DNS server administrator for help.

  • If your server provides its own DNS service, use the Server Admin application to change the name in your server’s machine record. For more information, open Server Admin and then use the Help menu to search for “DNS service overview,” or see the Mac OS X Server Resources website at www.apple.com/server/macosx/resources/.

Avoid having the server’s DNS name changed, because changing it requires client computer users to reconfigure their computers. For instructions, see If Users Can’t Configure Their Computers to Use the Server.
