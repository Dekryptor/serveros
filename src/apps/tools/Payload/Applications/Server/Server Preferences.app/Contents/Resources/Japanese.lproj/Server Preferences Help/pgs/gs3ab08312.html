自己署名証明書を作成する

お使いのサーバに SSL 証明書がない場合、または別の証明書が必要な場合は、まず自己署名証明書を作成します。

自己署名証明書を作成するには：

  1. 「サーバ環境設定」の「情報」パネルで、「SSL 証明書」の右側にある「編集」ボタンをクリックします。

  2. ポップアップメニューで、「証明書の読み込み」＞「自己署名証明書を作成」と選択します。

    「SSL 証明書を使用」が選択されていないと、ポップアップメニューは表示されません。

  3. 証明書アシスタントの「名前」フィールドで、サーバの完全に修飾された DNS 名（myserver.example.com など）を入力し、「続ける」をクリックします。

    そのほかの設定は変更しないでおきます。「固有名のタイプ」は「自己署名ルート」とし、「証明書のタイプ」は「SSL サーバ」とし、「デフォルトを無効化」は選択しません。

サーバ用に新しい自己署名証明書を選択することができます。詳細については、「SSL 証明書を使う」を参照してください。

新しい自己署名証明書を使って、既知の認証局に署名入り証明書を申請することもできます。操作方法については、「署名入り証明書を入手する」を参照してください。
