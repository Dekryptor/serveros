Web サービスを管理する

「サーバ環境設定」の「Web」パネルを使って、「Web サーバ」の有効／無効を切り替えたり、サーバの Web ホームページの場所を変更したり、カスタム Web サイトを公開したり、標準の Web サービス（Wiki、Web カレンダー、ブログ、および Web メール）の有効／無効を切り替えたりします。

「Web」パネルを構成するには：

次の情報を参考にして、必要に応じて設定を変更します：

  • 「入」／「切」： 
    「Web サーバ」の状態を示します。「入」／「切」スイッチをクリックすると、「Web サーバ」の有効／無効が切り替わります。
  • Web サービス： 
    サーバ用のホームページを選択したり、Wiki、Web カレンダー、ブログ、および Web メールサービスの有効／無効を個別に切り替えたりできます。
  • カスタムサイト： 
    Web サイト開発ソフトウェアで（自分または自分以外の人が）作成したカスタム Web サイトを追加、削除、または編集することができます。
  • ホームページ： 
    デフォルトの Mac OS X Server Web サイトを使うには、「サーバのホームページ」を選択します。Wiki を選択して、それをお使いのサーバの Web サイトにします。
  • Wiki： 
    すべての Wiki を使用可能にします。「Wiki」が選択されている場合は、その隣の矢印のリンクをクリックすると、Wiki のリストが開きます。
  • カレンダー： 
    各ユーザの「マイページ」ポータルにカレンダーを追加し、Wiki の作成者が各 Wiki に Web カレンダーを含めることができるようにします。「カレンダー」が選択されている場合、その隣の矢印のリンクをクリックすると、ユーザ名とパスワードが要求され、続いてそのユーザの Web カレンダーが開きます。
  • ブログ： 
    すべてのサーバユーザが Web ブラウザを使用してブログを作成できます。「ブログ」が選択されている場合は、その隣の矢印のリンクをクリックすると、ブログのリストが開きます。
  • Web メール： 
    すべてのサーバユーザが Web ブラウザを使って自分のメールにアクセスできるようにします。「Web メール」が選択されている場合は、その隣の矢印のリンクをクリックすると、Web メールの Web サイトが開きます。
  • Wiki を作成： 
    このボタンは、Wiki に使用するサイトが「サーバ管理」を使って削除されている場合に表示されます。Wiki を使用可能にするには、ボタンをクリックしてください。
  • カスタムサイトを表示： 
    このボタンは、カスタム Web サイトが Wiki と同一の DNS 名および通信ポート番号を使用して作成されている場合に表示されます。 Wiki では通常、サーバの DNS 名（「サーバ環境設定」の「情報」パネルに示されます）と、ポート 80 を使用します。Wiki を使用可能にするには、ボタンをクリックしてください。
