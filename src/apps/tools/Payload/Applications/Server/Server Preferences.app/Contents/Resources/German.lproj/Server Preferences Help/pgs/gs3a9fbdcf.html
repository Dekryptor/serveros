Einrichten des gemeinsamen Ordners für eine Gruppe

Sie können einen gemeinsamen Ordner für eine Gruppe einrichten. Alle Mitglieder einer Gruppe können auf Dateien im gemeinsamen Ordner zugreifen und Dateien in diesen Ordner legen. Als Name für den Ordner wird der Kurzname der Gruppe verwendet. Der Ordner wird auf dem Startvolume des Servers im Ordner "/Groups/" angelegt.

Gehen Sie wie folgt vor, um für eine Gruppe den gemeinsam genutzten Ordner einzurichten:

  1. Öffnen Sie die Servereinstellung "Gruppen" und wählen Sie in der Liste links eine Gruppe aus oder erstellen Sie eine neue Gruppe. Klicken Sie danach auf "Gruppe".

    Erstellen einer neuen Gruppe

  2. Markieren Sie das Feld "Dateifreigabe-Ordner".

  3. Klicken Sie zunächst auf "Alle einblenden" und danach auf "Dateifreigabe" und vergewissern Sie sich, dass der Dienst "Dateifreigabe" und der Zugriff auf den gemeinsam genutzten Ordner einer Gruppe aktiviert sind.

Wenn Sie auf den gemeinsamen Ordner der Gruppe zugreifen, meldet sich ein Gruppenmitglied mit dem Namen und Kennwort des Benutzer-Accounts der Gruppe an.
