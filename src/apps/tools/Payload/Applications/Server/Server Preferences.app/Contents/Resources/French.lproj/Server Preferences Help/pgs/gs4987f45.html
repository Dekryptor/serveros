Si votre serveur utilise le coupe-feu d’application

Si vous avez mis votre serveur à niveau à partir de Mac OS X Server 10.5 et si le coupe-feu du serveur était activé dans la sous-fenêtre Coupe-feu de Préférences du serveur avant la mise à niveau à Snow Leopard Server, vous ne pouvez pas gérer le coupe-feu dans la sous-fenêtre Sécurité. Dans ce cas, votre serveur utilise une autre technologie de coupe-feu appelée un coupe-feu d’application et vous pouvez gérer ce coupe-feu dans la sous-fenêtre Sécurité des Préférences Système. Si vous désactivez le coupe-feu dans la sous-fenêtre Sécurité des Préférences Système, vous pouvez alors gérer le coupe-feu du serveur dans Préférences du serveur.

Pour gérer le coupe-feu du serveur dans Préférences du serveur :

  1. Sur le serveur, ouvrez la sous-fenêtre des Préférences Système (et non Préférences du serveur), puis cliquez sur Coupe-feu.

    Ouvrir la sous-fenêtre Sécurité des Préférences Système

  2. Cliquez sur Arrêter pour désactiver le coupe-feu d’application.
