Conversion des membres de groupe en contacts iChat

Les identifiants Jabber (noms d’écran) de tous les utilisateurs membres d’un groupe peuvent être ajoutés automatiquement à la liste de contacts Jabber de chaque membre. Les membres de groupe voient leur liste de contacts Jabber dans iChat (ou dans une autre application de messagerie instantanée XMPP). Ils peuvent ajouter ou supprimer des contacts. Si le groupe compte des membres externes, ceux-ci ne figurent pas dans la liste de contacts iChat automatique.

Pour ajouter tous les membres de groupe à la liste de contacts Jabber :

  1. Dans la sous-fenêtre Groupes de Préférences du serveur, sélectionnez le groupe à modifier dans la liste de gauche et cliquez ensuite sur Services.

  2. Cochez la case à côté de la liste de contacts iChat automatique.

  3. Redémarrez le service iChat en cliquant deux fois sur Marche/Arrêt.

    Les modifications apportées au service iChat prennent effet une fois ce dernier redémarré.

Si vous désactivez la liste de contacts iChat automatique, les membres du groupe ne sont pas supprimés automatiquement des listes de contacts. Chaque membre de groupe peut supprimer les contacts qui ont été ajoutés automatiquement à sa liste de contacts.
