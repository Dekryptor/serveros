Ändern der Einstellungen eines Benutzers

Die Servereinstellung "Benutzer" enthält den Bereich "Account", in dem Sie den Namen, das Kennwort und das persönliche Bild des Benutzers ändern und festlegen können, ob der Benutzer Administratorrechte erhalten soll.

Gehen Sie wie folgt vor, um die Account-Einstellungen für einen Benutzer zu ändern:

  1. Öffnen Sie die Servereinstellung "Benutzer" und wählen Sie in der Liste links den gewünschten Benutzer-Account aus.

  2. Klicken Sie auf "Account" und nehmen Sie die erforderlichen Änderungen an den angezeigten Einstellungen vor. Berücksichtigen Sie dabei die folgenden Informationen:

    • Name: 
      Geben Sie den Benutzernamen ein. Der Name wird in Verbindung mit dem Kennwort für die Identifizierung des Benutzers gegenüber den bereitgestellten Diensten verwendet. Anleitungen hierzu finden Sie unter Ändern des Namens oder des Kennworts eines Benutzers.
    • Kurzname: 
      Dies ist ein anstelle des Benutzernamens einsetzbarer kurzer Name, der für die E-Mail- und die iChat-Adresse des Benutzers verwendet wird. Auch er kann in Verbindung mit dem Kennwort für die Identifizierung des Benutzers gegenüber den bereitgestellten Diensten verwendet werden. Nach dem Erstellen eines Accounts kann der Kurzname nicht mehr geändert werden.
    • Kennwort zurücksetzen: 
      Klicken Sie auf diese Taste, wenn Sie das Kennwort ändern wollen. Für einen Benutzer, dessen Account vom Verzeichnisserver importiert wurde, kann das Kennwort nicht geändert werden.
    • Bild: 
      Klicken Sie auf diese Taste, wenn Sie für den Benutzer ein Bild auswählen, ein Bild mit einer angeschlossenen Kamera aufnehmen oder einen visuellen Effekt verwenden wollen. Anleitungen hierzu finden Sie unter Ändern des persönlichen Bilds eines Benutzers.
    • Der Benutzer darf diesen Server verwalten: 
      Wenn diese Option ausgewählt ist, ist der Benutzer berechtigt, mit dem Programm "Servereinstellungen" Benutzer, Gruppen und Dienste zu verwalten. Anleitungen hierzu finden Sie unter Administrator-Accounts.

Wenn im Bereich "Account" neben dem Kurznamen weitere Einstellungen grau dargestellt werden, so bedeutet dies, dass die betreffenden Informationen auf dem Verzeichnisserver gespeichert sind, mit dem Ihr Server verbunden ist, und Sie keine Änderungen daran vornehmen können.
