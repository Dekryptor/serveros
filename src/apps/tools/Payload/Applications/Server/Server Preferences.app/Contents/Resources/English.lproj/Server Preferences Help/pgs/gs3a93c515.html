Finding Settings in Server Preferences

Server Preferences contains dozens of settings you can change to customize your computer. You can quickly search Server Preferences to find the specific setting you want.

To find settings in Server Preferences:

  1. View any Server Preferences pane except the Users pane or the Groups pane.

  2. In the search box at the top of the window, type a word or phrase that describes what you want to change. For example, if you want to add members to a group, type “group members.”

    As you type, you see a list of settings that match your typing.

  3. When you see what you’re looking for in the settings list, stop typing. One or more preferences are spotlighted in the Server Preferences window.

  4. Click the item in the list that matches what you want to change. In the pane that opens, you can change the settings for that item.
