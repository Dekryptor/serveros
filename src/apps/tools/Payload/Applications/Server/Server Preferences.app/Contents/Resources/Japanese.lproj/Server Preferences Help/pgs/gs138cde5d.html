サーバの DNS 名を変更する

サーバの DNS 名を変更できるのは、DNS サービスの管理者だけです。

  • 利用している ISP プロバイダが DNS サービスを提供している場合は、その ISP に相談してください。

  • ネットワーク上の別のサーバが DNS サービスを提供している場合は、DNS サーバの管理者に相談してください。

  • お使いのサーバが独自の DNS サービスを提供している場合は、サーバ管理アプリケーションを使ってサーバのマシンレコードの名前を変更します。詳細については、「サーバ管理」を開いて「ヘルプ」メニューで「DNS サービスの概要」を検索するか、Mac OS X Server リソースの Web サイト（www.apple.com/jp/server/macosx/resources/）を参照してください。

サーバの DNS 名を変更することは避けてください。変更すると、クライアントコンピュータのユーザはコンピュータを構成し直す必要があります。 操作方法については、「ユーザが自分のコンピュータでサーバを使用するよう設定できない場合」を参照してください。
