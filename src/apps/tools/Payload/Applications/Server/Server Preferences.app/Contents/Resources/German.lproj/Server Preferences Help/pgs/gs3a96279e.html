Anpassen der Begrüßungs- und Einladungsnachricht

Mit dem Programm "Servereinstellungen" können Sie E-Mails anpassen, mit denen neue Benutzer begrüßt oder vorhandene Benutzer zur Nutzung der Dienste des Servers eingeladen werden. Ihr Server kann alle der folgenden Nachrichten senden, wenn er für den Import von Benutzern von einem Verzeichnisserver konfiguriert ist. Wenn der Server keine Verbindung zu einem Verzeichnisserver hat, wird nur die Begrüßungsnachricht gesendet.

Begrüßungsnachricht

Diese E-Mail informiert neue Benutzer über die Dienste, die Ihr Server anbietet. Sie wird automatisch an neue Benutzer gesendet, deren Computer den Dienst "Mail" Ihres Servers nutzen. Sie wird nicht gesendet, wenn der Dienst "Mail" Ihres Servers deaktiviert ist.

Einladungsnachricht

Diese E-Mail informiert neu importierte Benutzer darüber, wie sie Dienste Ihres Servers nutzen können. Sie können sie an die Benutzer senden, deren Benutzer-Accounts Sie importieren. Das manuelle Senden ist ebenfalls möglich.

Einladungsnachricht für Gruppenmitglieder

Diese E-Mail informiert neue externe Mitglieder einer Gruppe darüber, wie sie die Dienste dieser Gruppe nutzen können. Sie wird an neue Gruppenmitglieder gesendet, deren externe Benutzer-Account bereits E-Mail-Adressen enthalten.
