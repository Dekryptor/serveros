Création d’un certificat auto-signé

Si votre serveur ne possède pas de certificat SSL ou si vous en avez besoin d’un autre, commencez par créer un certificat auto-signé.

Pour créer un certificat auto-signé :

  1. Dans la sous-fenêtre Informations de Préférence du serveur, cliquez sur le bouton Modifier situé à droite de Certificat SSL.

  2. Dans le menu local, choisissez Importation de certificat > Créer un certificat auto-signé.

    « Utiliser un certificat SSL » doit être sélectionné pour que le menu local soit disponible.

  3. Dans le champ Nom de l’Assistant de certification, saisissez un nom de DNS correct pour votre serveur (par exemple monserveur.exemple.com) et cliquez sur Continuer.

    Ne modifiez pas les autres réglages. Le Type d’identité doit être Racine auto-signée, le Type de certificat doit être Serveur SSL et « Me laisser ignorer les réglages par défaut » doit être désélectionné.

Vous pouvez choisir le nouveau certificat auto-signé pour le serveur. Pour en savoir plus, consultez Utilisation d’un certificat SSL.

Vous pouvez également utiliser le nouveau certificat auto-signé pour demander un certificat signé par une autorité de certification. Pour obtenir des instructions, consultez Obtention d’un certificat signé.
