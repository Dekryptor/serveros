Deleting a Group

You can use Server Preferences to delete groups that are no longer needed.

To delete a group:

  1. In the Groups pane of Server Preferences, select the group you want to delete in the list on the left.

  2. Click the Delete (–) button.

After you delete a group, the group’s shared folder remains on the server’s startup disk in the /Groups/ folder. You can keep the group’s shared folder or drag it to the Trash.
