サーバ情報を管理する

「サーバ環境設定」の「情報」パネルを使って、取り付けられているハードウェアとインストールされているソフトウェア、ネットワーク名とアドレス、シリアル番号など、サーバに関する情報を取得します。サーバのコンピュータ名やシリアル番号を変更したり、メールによる通知を管理したり、SSL 証明書を管理したりすることもできます。

サーバ情報を取得または変更するには：

「情報」パネルで次の項目を調べます：

  • IP アドレス： 
    ネットワーク上でサーバを識別する一意の数字アドレス。詳細については、「サーバの IP アドレスを変更する」を参照してください。
  • DNS 名： 
    サーバが使用する DNS サーバにより提供される一意の完全な名前。
  • コンピュータ名： 
    DNS 名ではなく、コンピュータ名で識別されるネットワーク・ファイル・サーバ、またはそのほかのネットワークリソースをブラウズするユーザのサーバを識別します。詳細については、「サーバの名前を変更する」を参照してください。
  • サーバ使用許諾契約： 
    Mac OS X Server で、ソフトウェアのシリアル番号またはサイトライセンスを基にライセンスのタイプが判断されます。「編集」をクリックすると、シリアル番号またはサイトライセンス情報を変更できます。操作方法については、「シリアル番号またはサイトライセンスの詳細を変更する」を参照してください。
  • 通知： 
    サーバが送信する重要なシステムイベントに関するメッセージの宛先となるメールアドレス。「編集」をクリックすると、アドレスを変更したり、特定のタイプの通知を有効または無効にしたりできます。操作方法については、「通知設定を変更する」を参照してください。
  • SSL 証明書： 
    セキュリティに SSL を使用するサービスの識別情報を作成、使用、および管理します。操作方法については、「SSL 証明書を使う」を参照してください。

「情報」パネルの左側には、現在インストールされている Mac OS X Server のバージョン番号とビルド番号、プロセッサの処理速度とタイプ、サーバにインストールされているメモリの合計サイズが表示されます。

サーバの「システム環境設定」の「ネットワーク」パネルで、ほかのネットワーク情報を表示または変更できます。「ネットワーク」環境設定の変更については、「システム環境設定」を開いて「ヘルプ」メニューを参照してください。

「システムプロファイラ」（「/アプリケーション/ユーティリティ/」にあります）を使用すると、サーバに関する詳しい情報を見ることができます。 「システム・プロフィール」の使用については、「システム環境設定」を開いて「ヘルプ」メニューを参照してください。
