Si un utilisateur n’est pas invité à se connecter à votre serveur

Lorsqu’un Mac exécutant Mac OS X 10.6 se connecte à votre réseau, il affiche généralement une invitation à obtenir les services disponibles sur votre serveur. Éventuellement, cette invitation ne s’affiche pas, et cela pour plusieurs raisons.

L’invitation à se connecter à votre serveur n’apparaît pas automatiquement si :

  • La version Mac OS X du Mac de l’utilisateur est antérieure à 10.6. Pour obtenir des informations sur la configuration de ce Mac, consultez Configuration manuelle des ordinateurs des utilisateurs.

  • L’utilisateur a antérieurement refusé une invitation à se connecter à votre serveur.

  • L’utilisateur avait ouvert une session sur son Mac lorsque vous avez créé le compte de cet utilisateur sur le serveur. L’utilisateur doit fermer sa session puis l’ouvrir à nouveau.

  • L’utilisateur a ouvert une session sur son Mac via un autre compte que son compte habituel.

  • Le compte de l’utilisateur sur votre serveur ne correspond pas au compte de l’utilisateur sur son Mac. Les deux comptes doivent avoir le même nom et le même nom abrégé. Par exemple, si le nom et le nom abrégé du compte sont Gita Kumar et gkumar sur le serveur mais Gita et gita sur le Mac de l’utilisateur, les comptes ne correspondent pas.

    Pour y remédier, modifiez le nom du compte utilisateur sur le serveur afin qu’il corresponde au nom du compte utilisateur sur le Mac de celui-ci. L’utilisateur peut également se connecter manuellement, même si les noms ne correspondent pas. Pour obtenir des instructions, consultez Configuration d’un Mac à l’aide des préférences Comptes.

  • Le Mac de l’utilisateur dispose déjà d’une connexion de services d’annuaire. L’utilisateur peut ajouter manuellement une connexion à votre serveur. Pour obtenir des instructions, consultez Configuration d’un Mac à l’aide des préférences Comptes.

  • Le compte de l’utilisateur sur le serveur est importé à partir d’un serveur d’annuaire (il n’a pas été créé sur votre serveur). L’utilisateur reçoit une invitation par courrier électronique comportant un bouton pour obtenir les services disponibles sur votre serveur. Pour en savoir plus, consultez Configuration des Mac des nouveaux utilisateurs importés.

  • Aucun service n’est activé dans Préférences du serveur sur votre serveur. Pour en savoir plus, consultez Vérification de l’état du service.

  • Le Mac de l’utilisateur ou votre serveur présente un problème de connexion réseau.
