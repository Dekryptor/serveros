Ändern des DNS-Namens des Servercomputers

Der DNS-Name Ihres Servers kann nur von einem Benutzer geändert werden, der über Administratorrechte für den Dienst "DNS" verfügt.

  • Wenn Ihr Internetanbieter den Dienst "DNS" bereitstellt, bitten Sie ihn um Unterstützung.

  • Stellt ein anderer Server in Ihrem Netzwerk den Dienst bereit, bitten Sie den Administrator des DNS-Servers um Unterstützung.

  • Wenn Ihr Server den Dienst "DNS" selbst bereitstellt, verwenden Sie das Programm "Server-Admin", um den Namen im Systemeintrag Ihres Servers zu ändern. Weitere Informationen erhalten Sie, indem Sie das Programm "Server-Admin" öffnen und anschließend im Menü "Hilfe" nach "Überblick über den Dienst "DNS"" suchen oder die Website zu Mac OS X Server-Ressourcen unter www.apple.com/de/server/macosx/resources/ besuchen.

Vermeiden Sie Änderungen des DNS-Namens des Servers, da die Benutzer von Client-Computern sonst ihre Computer neu konfigurieren müssen. Anleitungen hierzu finden Sie unter Benutzer können ihre Computer nicht für die Nutzung des Servers konfigurieren.
