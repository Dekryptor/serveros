Bereitstellen weiterer Dienste

Nach der Erstkonfiguration können Sie mithilfe des Programms "Servereinstellungen" grundlegende Dienst- und Systemeinstellungen ändern und Benutzer und Gruppen hinzufügen. Sie können die erweiterten Einstellungen ändern und erweiterte Dienste mit dem Programm "Server-Admin" konfigurieren. Sie können das Programm "Arbeitsgruppenmanager" verwenden, um erweiterte Einstellungen für Benutzer und Gruppen zu ändern, Benutzereinstellungen zu steuern und Computereinträge zu verwalten. Informationen zu den Programmen "Server-Admin" und "Arbeitsgruppenmanager" finden Sie im Menü "Hilfe" des jeweiligen Programms.

In den nachfolgenden Tabellen wird gezeigt, für welche Konfigurationsschritte die Programme "Servereinstellungen", "Arbeitsgruppenmanager" und "Server-Admin" verwendet werden können. Die Tabellen enthalten auch Informationen darüber, was während der Erstkonfiguration konfiguriert werden kann.
