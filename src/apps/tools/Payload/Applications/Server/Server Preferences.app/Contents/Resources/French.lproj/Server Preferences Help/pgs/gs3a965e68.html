Modification de la photo d’un utilisateur

Vous pouvez utiliser Préférences du serveur pour modifier la photo d’un compte utilisateur standard ou d’un compte administrateur. Si votre serveur contient des comptes utilisateur importés ou des comptes utilisateur importés automatiquement à partir de groupes, vous ne pouvez pas modifier leur image si celle-ci a été définie sur le serveur d’annuaire de votre organisation.

Pour changer la photo d’un compte utilisateur :

  1. Dans la sous-fenêtre Utilisateurs de Préférences du serveur, sélectionnez le compte utilisateur à modifier dans la liste de gauche, puis cliquez sur Compte.

  2. Pour utiliser une photo de la liste, cliquez sur le champ de photo et choisissez une photo dans le menu local.

  3. Pour modifier l’image ou pour en utiliser une provenant d’un appareil photo ou d’un fichier, cliquez sur le champ d’image, choisissez Modifier l’image dans le menu local, modifiez l’image comme souhaité, puis cliquez sur Définir ou sur Annuler.

    • Pour choisir une image que vous avez utilisée récemment, cliquez sur Photos récentes.

    • Pour déplacer l’image, faites-la glisser vers le haut, vers le bas ou vers les côtés.

    • Pour rogner l’image, faites glisser le curseur.

    • Pour capturer une nouvelle photo à l’aide d’un appareil connecté à l’ordinateur, cliquez sur le bouton Appareil photo.

    • Pour appliquer un effet visuel, cliquez sur le bouton Effets visuels (en forme de spirale), faites défiler les effets disponibles, puis sélectionnez l’effet souhaité.

    • Pour utiliser un fichier d’image, cliquez sur Choisir.
