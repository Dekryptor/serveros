Starting or Stopping Web Services

You can stop or start web services in the Web pane. While web services are off, users can’t access their webmail and My Page webpages, or their wikis, blogs, web calendars, and mailing list archives.

To start or stop web services:

Click the On/Off switch in the Web pane of Server Preferences.

After starting web services, especially if you’re starting them for the first time, you should tell users they can access web services at the addresses shown in Setting Up Users’ Computers Manually.
