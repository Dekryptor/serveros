If Users Can’t Access a Service via the Internet

If users can access a service on the local network but not via the Internet, try these solutions:

  • If the Security pane of Server Preferences contains the message “Expose these services on __,” make sure the desired service is listed.

  • If the Security pane of Server Preferences contains the message “AirPort management is currently off,” turn it on and make sure the desired service is listed.

  • If your server gets its Internet connection through an AirPort Base Station or other Internet router, you may need to configure the router to make your server’s services available on the Internet. If you have an AirPort Extreme Base Station (802.11n) or Time Capsule, Mac OS X Server can configure it automatically. For information about configuring other AirPort Base Station models or an Internet router, see Chapter 2, “Getting Ready for Mac OS X Server,” in the Getting Started manual.
