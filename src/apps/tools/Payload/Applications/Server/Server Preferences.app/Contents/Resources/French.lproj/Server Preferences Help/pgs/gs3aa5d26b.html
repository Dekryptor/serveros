Démarrage ou arrêt des services Web

Vous pouvez arrêter ou démarrer les services web dans la sous-fenêtre Web. Lorsque les services web sont arrêtés, les utilisateurs ne peuvent pas accéder à leur webmail et à leurs pages web Ma page ni à leurs wikis, blogs, calendriers web et archives de liste d’envoi.

Pour démarrer ou arrêter les services Web :

Cliquez sur le bouton Marche/Arrêt de la sous-fenêtre Web des Préférences du serveur.

Après avoir démarré les services web, en particulier s’il s’agit de la première fois, vous devez dire aux utilisateurs qu’ils peuvent accéder aux services web aux adresses indiquées dans la Configuration manuelle des ordinateurs des utilisateurs.
