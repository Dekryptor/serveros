Starting or Stopping Address Book Service

You can stop or start address book service in the Address Book pane. While address book service is off, users can’t access contact information in server-based address books.

To start or stop Address Book service:

Click the On/Off switch in the Address Book pane of Server Preferences.

After starting address book service, especially if you’re starting it for the first time, you should advise users to set up their Address Book application as described in Setting Up Users’ Computers Manually.
