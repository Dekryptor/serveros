Aufheben der Verfügbarkeit einer eigenen Website

Sie können die Verfügbarkeit einer eigenen Website aufheben.

Gehen Sie wie folgt vor, um die Verfügbarkeit einer eigenen Website aufzuheben:

Klicken Sie in der Servereinstellung "Web" auf "Eigene Sites" und deaktivieren Sie das Markierungsfeld neben der Website, deren Verfügbarkeit Sie aufheben möchten.

Wird eine eigene Website entfernt, indem Sie sie auswählen und auf die Taste "Löschen" (–) klicken, steht die Website ebenfalls nicht mehr zur Verfügung. Der Ordner der Website wird nicht entfernt, sodass Sie die Website zu einem späteren Zeitpunkt erneut erstellen können.
