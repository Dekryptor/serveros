Gestion du service de partage de fichiers

Utilisez la sous-fenêtre Partage de fichiers pour activer et désactiver le service de partage de fichiers et pour contrôler l’accès aux dossiers partagés Utilisateurs, Groupes et Public. Vous pouvez aussi ajouter ou supprimer vos propres dossiers partagés, également connus sous le nom de points de partage

Pour configurer la sous-fenêtre Partage de fichiers :

  • Cliquez sur le bouton Marche/Arrêt pour activer ou désactiver le service de partage de fichiers. Cette commande indique également si le partage de fichiers est activé ou désactivé.

  • Cliquez sur les cases de la liste Dossiers partagés pour activer ou désactiver l’accès aux dossiers Sauvegardes, Groupes, Public et Utilisateurs ainsi qu’à chaque dossier partagé que vous ajoutez.

  • Cliquez sur le bouton Ajouter (+) pour ajouter un nouveau dossier partagé.

    Ajout d’un dossier partagé

  • Cliquez sur le bouton Supprimer (–) pour ne plus partager un dossier et le rendre inaccessible aux utilisateurs du partage de fichiers.

    Suppression d’un dossier partagé

  • Cliquez sur Modifier les autorisations pour choisir d’activer ou de désactiver l’accès invité et pour modifier l’accès en lecture/écriture des utilisateurs à un dossier partagé.

    Contrôle de l’accès au dossier partagé

Le dossier partagé Sauvegardes contient les données des copies de sauvegarde des utilisateurs ayant configuré les préférences Time Machine pour stocker ces copies sur le serveur. Il n’est pas possible d’utiliser le bouton Modifier les autorisations ou le bouton Supprimer (–) avec le dossier partagé Copies de sauvegarde.

Le dossier partagé Groupes contient un dossier pour chaque groupe possédant un dossier de partage de fichiers dans la sous-fenêtre Groupes. Le bouton Supprimer (–) ne vous permet pas de rendre le dossier Groupes inaccessible aux utilisateurs de partage de fichiers.

Le dossier partagé Public contient des fichiers et des dossiers ajoutés par les utilisateurs ayant un compte listé dans la sous-fenêtre Utilisateurs de Préférences du serveur.

Le dossier partagé Utilisateurs contient des dossiers de départ pour les comptes utilisateur listés dans la sous-fenêtre Comptes de Préférences du système sur le serveur.
