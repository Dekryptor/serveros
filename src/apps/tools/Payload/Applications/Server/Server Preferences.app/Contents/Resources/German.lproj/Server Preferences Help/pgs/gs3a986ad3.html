Bereitstellen von Diensten für einen Benutzer

Sie können den Zugriff eines Benutzers auf einzelne Dienste steuern, indem Sie im Bereich "Benutzer" der Servereinstellungen auf "Dienste" klicken.

Gehen Sie wie folgt vor, um festzulegen, auf welche Dienste ein Benutzer zugreifen kann:

  1. Öffnen Sie die Servereinstellung "Benutzer" und wählen Sie in der Liste links den gewünschten Benutzer-Account aus.

  2. Klicken Sie auf "Dienste". Markieren Sie danach die Felder der Dienste, die dem Benutzer zur Verfügung gestellt werden sollen. Entfernen Sie die Markierung aus den Feldern der Dienste, auf die der Benutzer keinen Zugriff haben soll.

    Wenn der Server Speicherplatz für die Time Machine-Datensicherungen von Benutzern bereitstellt, wird durch Deaktivieren des Diensts "Dateifreigabe" für einen Benutzer auch der Zugriff auf das Time Machine-Sicherungsvolume deaktiviert.
