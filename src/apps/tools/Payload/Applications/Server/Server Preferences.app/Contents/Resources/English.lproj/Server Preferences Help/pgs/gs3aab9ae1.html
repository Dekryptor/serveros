Starting or Stopping Mail Service

You can stop or start mail service in the Mail pane. While mail service is stopped, users can’t send or receive email using the email addresses provided by the server.

To start or stop mail service:

Click the On/Off switch in the Mail pane of Server Preferences.

After starting mail service, especially if you’re starting it for the first time, you should advise users to set up their email applications as described in Setting Up Users’ Computers Manually.
