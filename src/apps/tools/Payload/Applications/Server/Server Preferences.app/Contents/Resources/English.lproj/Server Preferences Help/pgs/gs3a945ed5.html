Checking Service Status

Server Preferences shows the overall status of each service with an indicator next to the service icon.

To see status indicators for all services:

Click Show All in the Server Preferences window.

If the indicator is green, the service is running.

To turn a service on or off:

  1. In the Server Preferences window, click the icon of the service you want to turn on or off.

    If you donâ€™t see the service icons, click Show All in the Server Preferences window. You can also use the View menu to choose the service you want to turn on or off.

  2. Click the On/Off switch in the service pane.

The On/Off switch in each service pane indicates whether the service is running.

You can also monitor service status using the Server Status widget.
