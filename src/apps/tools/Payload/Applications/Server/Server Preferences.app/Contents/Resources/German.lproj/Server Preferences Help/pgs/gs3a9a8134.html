Verwalten von Benutzern

In der Servereinstellung "Benutzer" können Sie Accounts für die Personen einrichten, die die Möglichkeit haben sollen, die vom Server bereitgestellten Dienste zu nutzen, und bestimmen, welche Benutzer auf welche Dienste zugreifen können. Sie können die Kontaktinformationen für Benutzer-Accounts aktualisieren und ihre Gruppenzugehörigkeit ändern. Des weiteren können Sie Begrüßungsnachrichten verwalten, die an neue Benutzer gesendet werden sollen.

Optionen
Taste "Hinzufügen" (+)

Ermöglicht das Erstellen eines neuen Benutzer-Accounts oder das Importieren eines vorhandenen Benutzer-Accounts vom Verzeichnisserver Ihres Unternehmens. (Sie können angeben, dass der Import nur dann erfolgen soll, wenn Ihr Server mit einem anderen Verzeichnisserver in Ihrem Unternehmen verbunden ist.) Anleitungen finden Sie unter Hinzufügen eines Benutzers.

Taste "Entfernen" (–)

Ermöglicht das Löschen des ausgewählten Benutzer-Accounts. Anleitungen finden Sie unter Löschen eines Benutzer-Accounts.

Taste für das Aktionsmenü (Zahnradsymbol)

Ermöglicht das Verwalten des automatischen Benutzerimports aus Gruppen oder das Verwalten der E-Mails, die gesendet werden, um neue Benutzer zu begrüßen, um neu importierte Benutzer zur Verwendung des Servers einzuladen und um neue externe Mitglieder einer Gruppe zur Verwendung der Dienste einzuladen. Anleitungen hierzu finden Sie unter Anpassen der Begrüßungs- und Einladungsnachricht und Automatisches Importieren von Benutzern einer Gruppe.

Account

Ermöglicht das Ändern des Namens, Kennworts, Bilds oder der Administratorrechte eines Benutzers. Anleitungen hierzu finden Sie unter Ändern der Einstellungen eines Benutzers.

Kontaktinfo

Ermöglicht das Ändern der Kontaktinformationen - Vorname, Nachname, Adresse, E-Mail- und Chat-Adressen, Website-Adresse und Blog-Adresse - eines Benutzers. Anleitungen hierzu finden Sie unter Ändern der Kontaktinformationen eines Benutzers.

Dienste

Ermöglicht das Steuern des Zugriffs eines Benutzers auf einzelne Dienste. Anleitungen hierzu finden Sie unter Bereitstellen von Diensten für einen Benutzer.

Gruppen

Ermöglicht Ihnen, einen Benutzer zu einer Gruppe hinzuzufügen oder aus einer Gruppe zu löschen. Anleitungen hierzu finden Sie unter Ändern der Gruppenmitgliedschaften eines Benutzers.

Konfigurieren

Wird anstelle der oben beschriebenen Optionen die Taste "Konfigurieren" angezeigt, besitzt der Server keine eigene Verzeichnis-Domain für Benutzer und Gruppen. Durch Klicken auf diese Taste können Sie eine Verzeichnis-Domain einrichten. Anleitungen hierzu finden Sie unter Einrichten der Verwaltung von Benutzern und Gruppen.

Wenn Sie Benutzer-Accounts importiert haben, werden einige ihrer Einstellungen grau dargestellt, da sie auf dem Verzeichnisserver gespeichert sind, mit dem Ihr Server verbunden ist.
