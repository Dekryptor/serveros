Sie können nicht zur AirPort-Verwaltung wechseln

Wenn Sie eine AirPort Extreme-Basisstation (802.11n) oder eine Time Capsule verwenden, die Taste für den Wechsel zur AirPort-Verwaltung oder die Taste für den Wechsel zur Firewall aber nicht angezeigt wird, wenn die Sicherheitseinstellungen in der Servereinstellung "Sicherheit" deaktiviert sind, versuchen Sie Folgendes:

  • Vergewissern Sie sich, dass Ihre AirPort Extreme-Basisstation oder Time Capsule für die Freigabe einer Internetverbindung konfiguriert ist.

  • Prüfen Sie, ob die primäre Netzwerkschnittstelle Ihres Servers (meist der erste integrierte Ethernet-Port) mit demselben lokalen Netzwerk verbunden ist wie Ihre AirPort Extreme-Basisstation oder Time Capsule.
