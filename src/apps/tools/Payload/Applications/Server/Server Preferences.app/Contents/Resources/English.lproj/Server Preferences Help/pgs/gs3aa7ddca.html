Starting or Stopping VPN Service

You can stop or start VPN remote access service in the VPN pane. While VPN service is off, users can’t make remote connections to the server via the Internet.

To start or stop VPN service:

Click the On/Off switch in the VPN pane of Server Preferences.

After starting VPN service, especially if you’re starting it for the first time, you should advise users to set up their VPN connections as described in Setting Up a Mac User’s VPN Connection and Setting Up a User’s VPN Connection Manually.
