Modification des réglages d’un site web personnalisé

Vous pouvez modifier l’adresse IP, le port d’accès, l’emplacement du dossier du site et les autorisations d’accès d’un site web personnalisé.

Pour modifier les réglages d’un site web :

  1. Dans la sous-fenêtre Web de Préférences du serveur, cliquez sur Sites personnalisés.

  2. Sélectionnez le site web que vous souhaitez modifier, cliquez sur le bouton Modifier (crayon), puis modifiez les réglages en vous guidant grâce aux informations suivantes :

    • Adresse IP : 
      Si le serveur comporte plusieurs adresses IP, choisissez l’une d’entre elles ou sélectionnez Toutes pour autoriser les utilisateurs à accéder au site web via n’importe quelle adresse.
    • Port : 
      Saisissez un numéro de port d’accès. Le port d’accès standard pour les sites web est 80. Avant de saisir un port différent, vérifiez qu’il n’est pas en conflit avec un autre service disponible sur votre serveur. Pour obtenir la liste des ports utilisés par Mac OS X Server, ouvrez Admin Serveur (dans /Applications/Server/) et utilisez le menu Aide pour rechercher « Référence du port UDP et TCP ».
    • Stocker les fichiers du site dans : 
      Sélectionnez le dossier contenant les fichiers que vous souhaitez utiliser pour le site web.
    • Qui peut visualiser ce site : 
      Choisissez un groupe, afin que seuls ses membres puissent accéder à l’intégralité du site web ou sélectionnez Personnaliser pour restreindre l’accès à des dossiers distincts présents dans le dossier du site web. Pour obtenir des instructions, consultez Restriction de l’accès à un site web personnalisé.
