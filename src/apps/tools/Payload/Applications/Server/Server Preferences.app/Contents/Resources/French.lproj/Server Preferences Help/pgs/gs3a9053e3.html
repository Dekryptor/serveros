Mise à jour de Snow Leopard Server

Si votre serveur est connecté à Internet, Mise à jour de logiciels peut récupérer automatiquement la dernière version gratuite de Snow Leopard Server, des mises à jour de sécurité et d’autres améliorations conçues par Apple. Vous pouvez régler votre serveur pour qu’il recherche les mises à jour tous les jours, toutes les semaines ou tous les mois. Vous pouvez également rechercher immédiatement des mises à jour

Pour rechercher des mises à jour ou régler la mise à jour automatique :

  1. Ouvrez les Préférences Système sur le serveur.

  2. Cliquez sur l’icône Mise à jour de logiciels et suivez les instructions à l’écran.

    Ouvrez la sous-fenêtre Mise à jour de logiciels dans les Préférences Système

Si votre organisation dispose d’un autre serveur équipé de Mac OS X Server, votre serveur peut passer par celui-ci plutôt que par le site d’Apple pour récupérer des mises à jour de logiciels. Un administrateur qualifié peut configurer Mac OS X Server de façon à ce qu’il propose un service de mise à jour de logiciels à l’aide d’Admin Serveur.

Vous pouvez aussi télécharger les mises à jour de logiciels directement depuis le site web de téléchargements d’Apple : www.apple.com/fr/support/downloads/
