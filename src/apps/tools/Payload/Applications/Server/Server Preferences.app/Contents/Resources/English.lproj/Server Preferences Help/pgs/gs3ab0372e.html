Managing Users’ Backup Storage

Snow Leopard Server offers the security and convenience of Time Machine backup to users with Mac OS X v10.6 Snow Leopard or Mac OS X v10.5 Leopard who need backup storage. Snow Leopard users who haven’t selected a backup disk in the Time Machine pane of System Preferences will be automatically asked if they want to use the server as a storage location. Other users will need to open the Time Machine pane of System Preferences and change the disk used for storing backups.

Use the Time Machine pane of Server Preferences to make server disk space available for backing up users’ computers or to change the disk used for storing user backups.

To configure the Time Machine pane of Server Preferences:

Change the settings as desired, using the following information as a guide:

  • On/Off: 
    Makes the Backups shared folder at /Shared Items/Backups/ available or unavailable for users’ backup storage on the disk specified in this pane.
  • Select Disk: 
    Click to select a storage location from a list of disks currently attached to the server. Mac OS X Server creates the Backups shared folder on the disk you select.
  • Select Backup Disk: 
    This button appears if no disk has been selected for storing user backups. Click this button to make the Backups shared folder available and select the disk where it will be located. This is equivalent to turning on the Time Machine pane.
  • Open File Sharing: 
    This button appears if file sharing service is turned off. Click the button to open the File Sharing pane. File sharing service must be on in order for users’ Macs to access the Backups shared folder.

You can control each user’s access to the server’s Time Machine backup storage in the Users pane, or reveal the Backup shared folder in the Finder, by clicking the arrows in the text below the backup disk information.

If you change the backup disk, users’ Time Machine preferences that were set to use the server for backup storage will automatically begin using the Backups folder in its new location. However, Mac OS X Server doesn’t copy users’ backup data from the old Backups folder to the new Backups folder. After selecting a different backup disk, you should advise users that their first backup will take longer because it’s a full backup.
