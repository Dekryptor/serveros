アドレスブックサービスを管理する

「アドレスブック」パネルを使用して、アドレスブックサービスを有効または無効にしたり、サーバに保存される連絡先情報用の各ユーザのディスク領域を制限したりします。

「アドレスブック」パネルを構成するには：

次の情報を参考にして、使用可能な設定を変更します：

  • 「入」／「切」： 
    アドレスブックサービスの状態を示します。「入」／「切」スイッチをクリックすると、サービスの有効／無効が切り替わります。
  • 「各ユーザのブックの合計サイズの上限：__ MB」： 
    各ユーザのアドレスブックがサーバで使用できるディスク領域を制限します。ユーザがこの制限を超えて使用すると、サーバがユーザの「アドレスブック」アプリケーションにエラーメッセージを送信します。
