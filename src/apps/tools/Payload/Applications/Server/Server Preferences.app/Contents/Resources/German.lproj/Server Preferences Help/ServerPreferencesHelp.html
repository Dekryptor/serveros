Servereinstellungen-Hilfe

Verwalten von Benutzern

Erstellen Sie Accounts, steuern Sie den Zugriff auf Dienste, aktualisieren Sie Kontaktinformationen und ändern Sie Gruppenmitgliedschaften.

Verwalten von Gruppen

Erstellen Sie Gruppen, fügen Sie Gruppenmitgliedschaften hinzu und wählen Sie Gruppendienste aus.

Anpassen der Diensteinstellungen

Definieren Sie die Einstellungen für Dateifreigabe, Adressbuch, Kalender, Instant-Messaging, Mail, Web und den Fernzugriff neu.

Verwalten des Serversystems

Rufen Sie Systeminformationen ab, prüfen Sie Protokolle, zeigen Sie Diagramme (Graphen) an, legen Sie Einstellungen für Sicherungskopien von Benutzercomputern fest und steuern Sie die Sicherheitseinstellungen.
