If Your Server is Using the Application Firewall

If you upgraded your server from Mac OS X Server v10.5 Leopard and the server’s firewall was turned on in the Firewall pane of Server Preferences before you upgraded to Snow Leopard Server, you can’t manage the firewall in the Security pane. In this case, your server is using a different firewall technology called an application firewall, and you can manage this firewall in the Security pane of System Preferences. If you turn off the firewall in the Security pane of System Preferences, then you can manage the server’s firewall in Server Preferences.

To manage the server’s firewall in Server Preferences:

  1. Open the Security pane of System Preferences (not Server Preferences) on the server, and then click Firewall.

    Open the Security pane of System Preferences

  2. Click Stop to turn off the application firewall.
