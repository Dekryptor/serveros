証明書を更新する

すべての証明書には有効期限があるため、定期的に更新する必要があります。

有効期限が近づいている証明書を更新するには：

  1. 有効期限が近づいているその証明書を発行した認証局に、新しい証明書を申請します。

    操作方法については、「署名入り証明書を入手する」を参照してください。

  2. 「サーバ環境設定」の「情報」パネルで、「SSL 証明書」の右側にある「編集」ボタンをクリックします。

  3. ポップアップメニューで、置き換えたい証明書を選択します。

    「SSL 証明書を使用」が選択されていないと、ポップアップメニューは表示されません。

  4. ポップアップメニューで、「証明書署名」＞「署名または更新された証明書に置き換え」と選択します。

  5. 更新用証明書が含まれたファイルを、「サーバ環境設定」にあるダイアログの中央にドラッグして、「証明書を置き換える」をクリックします。
