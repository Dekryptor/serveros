Verwalten des Diensts "Mail"

Verwenden Sie die Servereinstellung "Mail", um den Dienst "Mail" zu aktivieren oder zu deaktivieren, einen Relaisserver für ausgehende E-Mails festzulegen oder die Filtereinstellungen für Junk-/Spam-Nachrichten und Viren anzupassen.

Gehen Sie wie folgt vor, um die Einstellungen der Servereinstellung "Mail" zu konfigurieren:

Ändern Sie die Einstellungen wie erforderlich und beachten Sie dabei die folgenden Informationen.

  • Ein/Aus: 
    Gibt den Status des Diensts "Mail" an. Durch Klicken auf die Taste "Ein/Aus" wird der Dienst aktiviert oder deaktiviert.
  • Ausgehende Mails über ISP weiterleiten: 
    Wählen Sie diese Option nur, wenn Ihr Internetanbieter die Weiterleitung ausgehender E-Mails über einen bestimmten Server erfordert. Klicken Sie auf "Bearbeiten", um die Adresse oder die Identifizierung für einen Relaisserver zu ändern, der bereits konfiguriert ist.
Festlegen eines Relaisservers für den Postausgang
  • E-Mails von Servern auf Blacklist-Listen ablehnen: 
    Lehnen Sie eingehende E-Mails von Absendern ab, deren IP-Adressen vom DNSBL-Server "zen.spamhaus.org" als potentielle Absender von Junk-/Spam-Nachrichten genannt werden. Klicken Sie auf "Bearbeiten", um einen anderen DNSBL-Server anzugeben.
Ablehnen von E-Mails von Absendern auf schwarzen Listen
  • Filter für Junk-Mail und Viren aktivieren: 
    Prüfen Sie eingehende E-Mails, löschen Sie E-Mails mit bekannten Viren und kennzeichnen Sie E-Mails, die gemäß der Einstellung des Schiebereglers als Junk-/Spam-Nachrichten gelten.
Überprüfen empfangener E-Mail-Nachrichten auf Junk-/Spam-Elemente und Computerviren
