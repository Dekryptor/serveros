Löschen einer Gruppe

Nicht mehr benötigte Gruppen können in der Servereinstellung "Gruppen" gelöscht werden.

Gehen Sie wie folgt vor, um eine Gruppe zu löschen:

  1. Öffnen Sie die Servereinstellung "Gruppen" und wählen Sie in der Liste links die Gruppe aus, die gelöscht werden soll.

  2. Klicken Sie auf die Taste "Löschen" (-).

Auch nach dem Löschen einer Gruppe verbleibt der gemeinsame Ordner der Gruppe auf dem Startvolume des Servers im Ordner "/Groups/". Sie können diesen Ordner beibehalten oder in den Papierkorb bewegen.
