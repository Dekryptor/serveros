Démarrage ou arrêt du service de messagerie

Vous pouvez arrêter ou démarrer le service de messagerie dans la sous-fenêtre Mail. Quand le service de messagerie est arrêté, les utilisateurs ne peuvent ni envoyer, ni recevoir de messages à l’aide des adresses électroniques fournies par le serveur.

Pour lancer ou arrêter le service de courrier :

Cliquez sur le bouton Marche/Arrêt de la sous-fenêtre Mail des Préférences du serveur.

Après avoir démarré le service de messagerie, en particulier s’il s’agit de la première fois, vous devez dire aux utilisateurs de configurer leur application de messagerie comme indiqué dans la Configuration manuelle des ordinateurs des utilisateurs.
