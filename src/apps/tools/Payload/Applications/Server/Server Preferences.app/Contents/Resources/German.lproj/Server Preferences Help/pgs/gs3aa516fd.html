Starten oder Stoppen des Diensts "Dateifreigabe"

Der Dienst "Dateifreigabe" kann in der Servereinstellung "Dateifreigabe" gestartet oder gestoppt werden. Ist der Dienst deaktiviert, können Benutzer nicht auf freigegebene Ordner zugreifen oder den Server für Time Machine-Sicherungskopien verwenden.

Gehen Sie wie folgt vor, um den Dienst "Dateifreigabe" zu starten oder zu stoppen:

Klicken Sie auf die Taste "Ein/Aus" in der Servereinstellung "Dateifreigabe".

Nach dem Starten des Diensts "Dateifreigabe", besonders nach dem erstmaligen Starten, sollten Sie die Benutzer darauf hinweisen, wie sie durch Herstellen einer Verbindung zu afp://meinserver.beispiel.de oder smb://meinserver.beispiel.de auf freigegebene Ordner zugreifen können. (Die kursiv dargestellten Platzhalter sind durch den DNS-Namen Ihres Servers zu ersetzen.)
