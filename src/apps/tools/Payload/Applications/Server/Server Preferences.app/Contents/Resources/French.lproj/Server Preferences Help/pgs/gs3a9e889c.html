Modification du nom d’un groupe

Vous pouvez utiliser Préférences du serveur pour modifier le nom d’un groupe.

Pour modifier le nom d’un groupe :

  1. Dans la sous-fenêtre Groupes de Préférences du serveur, sélectionnez le groupe à renommer dans la liste située à gauche, puis cliquez sur Groupe.

  2. Modifiez le champ Nom du groupe.

    Le nom peut comporter jusqu’à 255 caractères (et se réduit à seulement 85 caractères japonais). Ce nom peut comporter des espaces.

    Vous ne pouvez pas changer le nom abrégé d’un groupe à l’aide de Préférences du serveur.
