Starten oder Stoppen von Webdiensten

In der Servereinstellung "Web" können Sie Webdienste stoppen oder starten. Sind Webdienste deaktiviert, können Benutzer nicht auf ihre Webmail-Nachrichten und ihr Portal "Meine Seite" bzw. ihre Wikis, Blogs, Webkalender und archivierten Mailing-Listen zugreifen.

Gehen Sie wie folgt vor, um Webdienste zu starten oder zu stoppen:

Klicken Sie auf die Taste "Ein/Aus" in der Servereinstellung "Web".

Nach dem Starten der Webdienste, besonders nach dem erstmaligen Starten, sollten Sie die Benutzer darauf hinweisen, wie sie auf die Webdienste zugreifen können, auf die in den folgenden Informationen hingewiesen wird Manuelles Konfigurieren von Benutzercomputern.
