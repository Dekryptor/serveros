Gestion des services web

Utilisez la sous-fenêtre Web de Préférences du serveur pour activer ou désactiver Serveur web, modifier l’emplacement de la page d’accueil web de votre serveur, publier des sites web personnalisés ou activer ou désactiver les services web standard (wikis, calendriers web, blogs et webmail).

Pour configurer la sous-fenêtre Web :

Modifiez les réglages souhaités en utilisant les informations suivantes pour vous guider :

  • On/Off : 
    Indique l’état de Serveur web ; vous pouvez cliquer sur ce réglage pour activer ou désactiver Serveur web.
  • Services web : 
    Vous permet de choisir une page d’accueil pour votre serveur et d’activer ou de désactiver séparément les services de wikis, de calendriers web, de blogs et de webmail.
  • Sites personnalisés : 
    Vous permet d’ajouter, de supprimer ou de modifier des sites web personnalisés que vous avez créés (ou fait créer) à l’aide d’un logiciel de développement de site web.
  • Page d’accueil : 
    Sélectionnez Page d’accueil du serveur pour utiliser le site web par défaut de Mac OS X Server. Choisissez un wiki pour en faire le site web de votre serveur.
  • Wikis : 
    Rend tous les wikis disponibles. Si vous sélectionnez Wikis, vous pouvez ouvrir une liste de wikis en cliquant sur la flèche de lien située en regard de ce réglage.
  • Calendrier : 
    Ajoute un calendrier à chaque portail Ma page des utilisateurs et autorise les créateurs de wikis à inclure un calendrier web dans chaque wiki. Si vous sélectionnez Calendrier, vous pouvez cliquer sur la flèche de lien située en regard de ce réglage et rentrer votre nom d’utilisateur et votre mot de passe pour ouvrir le calendrier web de cet utilisateur.
  • Blogs : 
    Permet à tous les utilisateurs du serveur de créer des blogs au moyen d’un navigateur web. Si vous sélectionnez Blogs, vous pouvez cliquer sur la flèche de lien située en regard de ce réglage pour ouvrir une liste de blogs.
  • Webmail : 
    Permet à tous les utilisateurs du serveur d’accéder à leur courrier électronique au moyen d’un navigateur web. Si « Webmail » est sélectionné, le fait de cliquer sur la flèche gauche située à côté ouvre le site web Webmail.
  • Créer le wiki : 
    Ce bouton apparaît si le site utilisé pour les wikis a été supprimé au moyen d’Admin Serveur. Cliquez sur ce bouton si vous souhaitez que les wikis deviennent disponibles.
  • Révéler le site personnalisé : 
    Ce bouton apparaît si un site web personnalisé a été créé en utilisant le même nom DNS et le même numéro de port de communication que le wiki. En général, un wiki utilise le nom DNS du serveur (affiché dans la sous-fenêtre Informations de Préférences du serveur) et le port 80. Cliquez sur ce bouton si vous souhaitez que les wikis deviennent disponibles.
