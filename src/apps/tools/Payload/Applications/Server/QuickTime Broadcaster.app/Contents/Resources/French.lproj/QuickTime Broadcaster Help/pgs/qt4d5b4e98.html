Configuration de votre premier flux

Configurez votre premier flux et diffusez-le en moins de cinq minutes.

Les instructions qui suivent sont destinées à vous permettre de configurer une connexion un à un directe (sans utiliser QTSS ni aucun autre logiciel d’enchaînement) vers un ordinateur équipé de QuickTime Player.

Vous devez connaître l’adresse IP de l’ordinateur de réception.

  1. Connectez une caméra ou tout autre périphérique d’entrée vidéo à votre ordinateur.

  2. Ouvrez QuickTime Broadcaster et, si la fenêtre développée est ouverte, cliquez sur Masquer les détails.

  3. Dans les menus locaux Audio et Vidéo, choisissez les réglages qui vous conviennent le mieux.

  4. Cliquez sur Afficher les détails, puis sur l’onglet Réseau et choisissez Monodiffusion manuelle dans le menu local Transmission.

  5. Tapez l’adresse IP de l’ordinateur de réception dans le champ Adresse

    Conservez les valeurs par défaut des autres réglages.

  6. Dans QuickTime Broadcaster, choisissez Fichier > Exporter > SDP.

    Le fichier enregistré dans cette étape est un fichier Session Description Protocol (SDP) contenant les informations dont les spectateurs auront besoin pour pouvoir visionner la diffusion.

  7. Envoyez ou copiez le fichier SDP vers l’ordinateur de réception des données.

  8. Sur l’ordinateur d’envoi, lancez la diffusion en cliquant sur Diffuser dans la fenêtre QuickTime Broadcaster.

Pour visionner la diffusion, double-cliquez sur le fichier SDP sur l’ordinateur de réception.
