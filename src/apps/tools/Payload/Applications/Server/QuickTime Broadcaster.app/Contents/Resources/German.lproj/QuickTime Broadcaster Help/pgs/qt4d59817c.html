Firewalls

Netzwerk-Firewalls, die interne Netzwerke vor Fremdzugriffen schützen, können das Senden und Empfangen von Sendungen beeinträchtigen.

Wenn Sie eine Sendung ausstrahlen wollen, vergewissern Sie sich bei Ihrem Netzwerkadministrator, dass die Ports für ausgehende Sendungen geöffnet sind. Viele Internetanbieter und Unternehmensnetzwerke sperren diese Ports, um den Verbrauch an Bandbreite einzuschränken und die Sicherheit zu erhöhen.

Für das Übertragen einer Sendung verwendet der QuickTime Broadcaster standardmäßig die folgenden Ports:

  • Unicast (automatisch mit Ankündigung): 554

  • Unicast (manuell): Geben Sie die Information für den Port im Bereich "Netzwerk" ein.

Fügen Sie zum Ändern einer Portnummer einen Doppelpunkt und die gewünschte Portnummer am Ende der IP-Adresse hinzu. Beispiel: 127.0.0.1:7070.

Teilnehmer hinter einer Firewall können Ihre Sendung in einigen Fällen ansehen, wenn die Netzwerkeinstellungen in den QuickTime-Einstellungen angepasst werden. Alternativ kann auch die Verwendung eines Proxy-Servers in der Systemeinstellung "Netzwerk" konfiguriert werden.
