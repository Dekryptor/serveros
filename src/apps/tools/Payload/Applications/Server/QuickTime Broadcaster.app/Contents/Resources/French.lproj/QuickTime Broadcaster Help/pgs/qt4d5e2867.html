Qu’est-ce qu’un fichier SDP ?

Un fichier SDP (Session Description Protocol) décrit les paramètres et les réglages d’une diffusion, permettant ainsi à QuickTime Player de recevoir la diffusion. Pour visionner une diffusion, ouvrez le fichier SDP avec QuickTime Player.

Vous pouvez créer un fichier SDP pour chacune des trois options de transport :

  • Monodiffusion automatique (Annonce) :  le fichier SDP peut être créé automatiquement sur le serveur QuickTime Streaming Server spécifié. L’adresse de l’emplacement n’apparaît que dans ce cas.

  • Monodiffusion manuelle :  le fichier SDP est placé sur un serveur d’enchaînement ou ouvert directement dans QuickTime Player.

  • Multidiffusion :  le fichier SDP est ouvert manuellement dans QuickTime Player.

Pour créer un fichier SDP :

Dans QuickTime Broadcaster, choisissez Fichier > Exporter > SDP.

Vous pouvez envoyer le fichier SDP à un seul spectateur ou à un serveur d’enchaînement qui permettra à de nombreux spectateurs de se connecter à la diffusion.
