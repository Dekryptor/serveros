Das Videobild sieht verschwommen aus, ruckelt oder friert ein

Wenn die Bildschirmanzeige bei der Übertragung verschwommen aussieht, ruckelt oder einfriert, ist vielleicht an der Kamera nicht die korrekte Schärfe eingestellt, das Netzwerk überlastet oder die Einstellungen für die Komprimierung sind nicht korrekt.

Führen Sie einen oder mehrere der folgenden Schritte aus, um die Sendequalität zu verbessern:

  • Stellen Sie die Kamera scharf.

  • Reduzieren Sie die Anzahl gleichzeitiger Zuschauer (im Bereich "QTSS" von Server-Admin).

  • Kommt es beim Empfänger zu einem ruckelnden Videobild, passen Sie die Datenrate der Sendung an.

  • Stellen Sie sicher, dass Sie mit einem Video-Codec senden, der für das Streaming optimiert ist, wie z. B. H.264 oder MPEG-4.

  • Reduzieren Sie Bildrate und Bildgröße für das Video.

Testen Sie verschiedene Einstellungen vor dem Übertragen, um eine hochwertige Sendung bereitzustellen. Auf diese Weise können Sie die optimalen Einstellungen für Ihren besonderen Inhalt und für die Verbindungsgeschwindigkeit der Teilnehmer herausfinden.
