Modification de la source vidéo

QuickTime Broadcaster est compatible avec les caméras DV, Apple iSight et IIDC (gérées seulement par Mac OS X 10.2 ou ultérieur), ainsi qu’avec d’autres sources DV (les convertisseurs analogiques-DV par exemple) qu’il est possible de connecter à un ordinateur via un port FireWire.

Pour choisir une source vidéo :

  1. Dans la fenêtre QuickTime Broadcaster, cliquez sur Afficher les détails (si la fenêtre n’est pas développée), puis sur Vidéo.

  2. Sélectionnez une option dans le menu local Source.

Si une source vidéo est connectée et transmet des données, elle apparaît dans le menu Source et une image d’aperçu s’affiche (à condition que la fonction d’aperçu soit activée). Les réglages vidéo doivent également être disponibles (non estompés).

Si votre périphérique d’entrée ne figure pas dans le menu Source, quittez QuickTime Broadcaster, assurez-vous que votre appareil est connecté et allumé, puis ouvrez à nouveau QuickTime Broadcaster. Si l’appareil n’apparaît toujours pas, c’est qu’il est peut-être incompatible avec QuickTime Broadcaster.
