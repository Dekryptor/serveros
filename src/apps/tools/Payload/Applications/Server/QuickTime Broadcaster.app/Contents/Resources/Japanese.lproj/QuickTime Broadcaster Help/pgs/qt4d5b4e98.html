最初のストリームをセットアップする

最初のストリームをセットアップすれば、それを 5 分以内にオンエアすることができます。

ここでは、「QuickTime Player」がインストールされたコンピュータに、1 対 1 接続を使って直接セットアップする場合について説明します。「QTSS」などのストリーミングソフトウェアは使用しません。

受信側コンピュータの IP アドレスが必要です。

  1. カメラなどのビデオ入力装置をコンピュータに接続します。

  2. 「QuickTime Broadcaster」を開き、拡張ウインドウが開いている場合は「詳細を隠す」をクリックします。

  3. 「オーディオ」および「ビデオ」ポップアップメニューから、自分の使用環境に最も適したオプションを選択します。

  4. 「詳細を表示」をクリックし、「ネットワーク」をクリックして、「転送」ポップアップメニューから「手動ユニキャスト」を選択します。

  5. 「アドレス」フィールドに受信側コンピュータの IP アドレスを入力します。

    ほかの設定はデフォルトのまま変更しないでください。

  6. 「QuickTime Broadcaster」で、「ファイル」＞「書き出す」＞「SDP」と選択します。

    この手順で保存される Session Description Protocol（SDP）ファイルには、視聴者がブロードキャストを視聴するために必要な情報が含まれています。

  7. この SDP ファイルを、ブロードキャストの視聴に使用されるコンピュータに送信またはコピーします。

  8. 送信側コンピュータで、「QuickTime Broadcaster」ウインドウの「ブロードキャスト」をクリックしてブロードキャストを開始します。

ブロードキャストを表示するには、受信側コンピュータで SDP ファイルをダブルクリックします。
