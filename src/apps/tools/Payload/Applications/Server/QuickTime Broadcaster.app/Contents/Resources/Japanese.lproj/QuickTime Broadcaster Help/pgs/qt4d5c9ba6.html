自動ユニキャスト（アナウンス）を使用する

「QuickTime Broadcaster」の自動ユニキャスト（アナウンス）転送方式を使って、「QTSS」にブロードキャストすることができます。そのためには、次のものが必要です：

  • ブロードキャスターのユーザアカウント。「QuickTime Broadcaster」では、ストリーミングサーバ上に SDP ファイルを作成できるように、アカウントのユーザ名とパスワードを要求します。「QuickTime Broadcaster」と「QTSS」を同じコンピュータ上で使用している場合、ブロードキャスターのユーザアカウントは不要です。

  • QTSS 4.0 以降がストリーミングサーバにインストールされていること。

自動ユニキャストのブロードキャストを開始するには：

  1. 「QuickTime Broadcaster」で、（拡張ウインドウが開いていない場合は「詳細を表示」をクリックしてから）「ネットワーク」をクリックします。

  2. 「転送」ポップアップメニューから「自動ユニキャスト（アナウンス）」を選択します。

  3. 受信側サーバの IP アドレスかホスト名、ブロードキャストファイルの名前、ストリーミングサーバのユーザアカウント用のユーザ名とパスワード（必要な場合）、およびバッファ遅延（あるいはデフォルトを使用）を入力します。

  4. 「ブロードキャスト」をクリックします。

    すべての情報がサーバに送信されます。SDP ファイルを作成する必要はありません。
