マルチキャストを使用する

マルチキャスティングは、ブロードキャストのすべての視聴者が共有する単一のストリームを使って、（インターネットではなく）ローカルネットワークを介してブロードキャストする方法です。それぞれの視聴者は、ラジオを FM 放送に合わせるのと同じようにブロードキャストに合わせます。

マルチキャスティングでは、マルチキャストバックボーン（MBone）にアクセスできるネットワークかマルチキャスト対応のネットワークが必要です。

マルチキャストが有効になっているローカルネットワーク経由でマルチキャストすることができます。使用しているネットワークでマルチキャストがサポートされているかどうかは、担当のネットワーク管理者にお問い合わせください。

マルチキャスト方式を使用して、ローカルネットワーク経由でブロードキャストするには：

  1. 「QuickTime Broadcaster」で、拡張ウインドウが開いていない場合は「詳細を表示」をクリックします。

  2. 「ネットワーク」をクリックします。

  3. 「転送」ポップアップメニューから「マルチキャスト」を選びます。

  4. 「IP アドレスの生成」をクリックします。

    これにより、マルチキャストの送信に使用する IP アドレスが入力されます。アドレスは、マルチキャストブロードキャスト用に確保されている範囲内で作成されます。 ネットワーク管理者が特定のマルチキャストアドレスを割り当てている場合は、手動でアドレスを入力できます。

  5. Time To Live（TTL）設定を指定します（または、デフォルトをそのまま使用します）。

    この設定は、ストリームが転送されなくなるまでに、あるルーターから別のルーターにメディアストリームを何回渡すことができるかを表しています。1 〜 255 の任意の値を使用してください。値が 1 の場合は、ローカルネットワーク内のクライアントコンピュータが範囲となります。 数値が大きくなればなるほど、マルチキャストパケットの移動距離は長くなります。

  6. 「ファイル」＞「書き出す」＞「SDP」、または「ファイル」＞「書き出す」＞「ムービー」と選択します。

    これにより、ほかの人に送信できる SDP（Session Description Protocol）ファイルまたは QuickTime ムービーファイルが作成されます。ブロードキャストを視聴するには、SDP ファイルをダブルクリックするか、「QuickTime Player」を使用してムービーファイルを開きます。
