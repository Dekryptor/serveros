ログファイルを使ってプレイリストのブロードキャストを監視する

ロギングを有効にすると、ログファイルを使って、ブロードキャスト中に発生する問題を解決することができます。発生する可能性のある問題を解決するためのヒントを次に示します：

  • プレイリスト内のメディアがブロードキャストされない場合は、「サーバ管理」を使って、ストリーミングサーバが動作していることを確認します。

  • ストリーミングサーバが動作している場合は、サーバコンピュータ上で PlaylistBroadcaster というプロセスが実行中であることを確認します。実行中の場合は、ブロードキャストを停止し、ストリーミングサーバのメディアディレクトリからブロードキャスト用の SDP ファイルを削除して、ブロードキャストを再開します。ブロードキャストを再開するときに、新しい SDP ファイルが生成されます。
