オーディオのみのブロードキャストをセットアップする

「QuickTime Broadcaster」を使用して、コンサート、スピーチ、会議などのオーディオコンテンツをブロードキャストできます。

「QuickTime Broadcaster」は、以下のもので使用できます：

  • DV カメラやその他の FireWire 装置から出力されるオーディオ

  • （内蔵型ではない）外付け型の CD プレーヤー

  • コンピュータのオーディオ入力ポートや USB オーディオ入力装置（Grifin iMic など）に接続されたマイクロホンやその他のアナログ・オーディオ・ソース

オーディオのブロードキャストをセットアップするには：

  1. 「QuickTime Broadcaster」ウインドウで、（拡張ウインドウが開いていない場合は「詳細を表示」をクリックしてから）「オーディオ」をクリックします。

  2. 「入力源」ポップアップメニューからオーディオソースを選択します。

    メニューにオーディオソースが表示されない場合は「QuickTime Broadcaster」を終了し、装置が接続されて電源が入っているかどうか確認してからもう一度同じことを試みてください。

  3. 「プリセット」ポップアップメニューから適切な圧縮設定を選択するか、カスタム設定を指定します。

  4. 「ビデオ」をクリックし、「ビデオストリームを有効にする」が選択されていないことを確認します（これにより、ビデオストリームが無効になります）。

オーディオからフィードバックが作成されているか、オーディオを聞きたくない場合は、「オーディオ」パネルの「スピーカーで再生」チェックボックスの選択を解除してオーディオをオフにしてください。
