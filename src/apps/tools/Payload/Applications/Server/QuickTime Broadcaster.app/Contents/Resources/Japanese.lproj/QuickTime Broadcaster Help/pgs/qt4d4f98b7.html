ストリーミングサーバをセットアップする

ストリーミングサーバをセットアップし、セットアップをテストして、サーバからストリーミングされたメディアにアクセスすることができます。

ストリーミングサーバをセットアップするには：

  1. 「サーバ管理」を開きます。

  2. 「サービス」リストで QuickTime ストリーミングサーバとしてセットアップしたいコンピュータを選択します。

  3. ツールバーの「一般設定」をクリックし、「サービス」タブをクリックします。

  4. サービスリストの「QuickTime Streaming」を選択してから、「保存」をクリックします。

  5. 「サーバ」リストの「QuickTime Streaming」を選択して、設定を表示および変更します。

  6. QuickTime Streaming サービスを開始するには、「サーバ管理」ウインドウの左下隅の近くにある「再生」ボタンをクリックします。
