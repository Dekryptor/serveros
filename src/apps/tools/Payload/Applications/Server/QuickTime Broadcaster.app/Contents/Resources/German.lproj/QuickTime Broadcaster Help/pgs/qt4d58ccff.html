Erstellen einer Steuerspur für die Filmdatei

Sie können eine komprimierte und mit einer Steuerspur ausgestattete Kopie Ihrer Sendung als Filmdatei sichern. Der Film wird auf dem Computer gesichert, den Sie für die Übertragung verwenden. Er kann jetzt an einen QuickTime Streaming Server gesendet werden, sodass sich die Zuschauer eine Wiederholung des Events ansehen können.

Gehen Sie wie folg vor, um Ihre Sendung aufzuzeichnen und eine Steuerspur für die Filmdatei zu erstellen:

  1. Wählen Sie "QuickTime Broadcaster" > "Einstellungen".

  2. Geben Sie in das Feld "Aufnahme sichern unter" einen Pfad ein, unter dem Ihre Sendedatei gesichert werden soll (oder klicken Sie auf "Wählen", um einen Speicherort auszuwählen).

  3. Wählen Sie "Steuerspur für Streaming Server".

    Diese Option fügt eine Steuerspur mit Informationen hinzu, die für das Streaming Ihres Films erforderlich sind.

  4. Geben Sie einen Namen für Ihren Film ein und klicken Sie auf "OK".

  5. Wählen Sie im QuickTime Broadcaster-Fenster die Option "Auf Medium aufnehmen".

  6. Klicken Sie auf "Senden".

    Hinweis:  Archivierte Sendungen nehmen unter Umständen viel Festplattenspeicher ein. Umfasst die Sendung mehrere Stunden, werden ggf. mehrere .mov-Dateien erstellt. In diesem Fall werden die Dateien aufsteigend nummeriert (z. B. "mymovie.mov", "mymovie2.mov" etc.).
