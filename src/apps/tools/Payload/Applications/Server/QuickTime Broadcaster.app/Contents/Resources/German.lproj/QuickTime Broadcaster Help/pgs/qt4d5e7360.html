Sichern von Einstellungen mithilfe von Voreinstellungen

Sie können Ihre Audio-, Video- und Netzwerkeinstellungen sichern und somit zu einem späteren Zeitpunkt erneut verwenden.

Gehen Sie wie folgt vor, um eine Voreinstellung zu erstellen:

  1. Wählen Sie im QuickTime Broadcaster die gewünschten Einstellungen in den Bereichen "Audio", "Video" und "Netzwerk" aus.

    Wenn diese Bereiche nicht angezeigt werden, klicken Sie auf "Details einblenden".

  2. Wählen Sie "Voreinstellung sichern" aus dem Einblendmenü "Voreinstellung" aus.

  3. Wenn Sie diese Einstellungen erneut verwenden wollen, wählen Sie im jeweiligen Bereich ("Audio", "Video" oder "Netzwerk") Ihre Voreinstellung aus dem Einblendmenü "Voreinstellung" aus.

  4. Wählen Sie "QuickTime Broadcaster" > "Einstellungen" und geben Sie einen neuen Pfad in das Feld "Verzeichnis für Voreinstellungen" ein (oder klicken Sie auf "Wählen" und wählen Sie ein Ziel aus), um Ihre Voreinstellungen auf einen anderen Computer zu übertragen.

    Hinweis: Voreinstellungen müssen im Ordner "~/Library/QuickTime Broadcaster" abgelegt sein. Andere Benutzer, die Ihre Voreinstellungen verwenden, müssen ihre eigenen Dateien am selben Speicherort auf ihren Computern ablegen.
