オーディオ圧縮設定の選択のヒント

一般に、オーディオ・オプションを設定してからビデオ設定を行うのがベストです。「QuickTime Broadcaster」では、QuickTime に含まれている任意のコーデックを使用できます。これにより、ブロードキャストされているコンテンツのタイプと視聴者の接続に最も適したコーデックが選択できます。

初心者のために、「QuickTime Broadcaster」には一般によく使用されるコンテンツの種類と視聴者の接続に最適化されたデフォルト設定（プリセット）が用意されています。たとえば、スピーチのデフォルト設定は、8 kHz モノラルの MPEG-4 オーディオ（AAC）です。（MPEG-4 オーディオは一般に音楽用コーデックと考えられていますが、スピーチコンテンツにも十分適しています。）

音楽を含むオーディオのデフォルト設定は、ダイヤルアップ接続ユーザの場合は 20 キロビット／秒（Kbps）、LAN 接続ユーザの場合は 128 Kbps の MPEG-4 オーディオ（AAC）です。
