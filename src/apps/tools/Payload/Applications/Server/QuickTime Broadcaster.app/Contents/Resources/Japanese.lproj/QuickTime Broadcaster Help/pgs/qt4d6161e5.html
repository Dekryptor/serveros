ビデオソースを変更する

「QuickTime Broadcaster」は、DV カメラ、アップルの iSight カメラ、IIDC カメラ（Mac OS X v10.2 以上でサポートされています）、FireWire でコンピュータに接続できるその他の DV ソース（アナログ－ DV コンバータボックスなど）で使用できます。

ビデオソースを選択するには：

  1. 「QuickTime Broadcaster」ウインドウで、（拡張ウインドウが開いていない場合は「詳細を表示」をクリックしてから）「ビデオ」をクリックします。

  2. 「入力源」ポップアップメニューからオプションを選択します。

ビデオソースが接続されていて入力を提供中の場合は、ビデオソースが「入力源」メニューに表示され、さらにプレビューイメージが表示されるはずです（プレビュー機能がオンの場合）。ビデオ設定も使用可能な状態（薄く表示されていない）になっているはずです。

「入力源」メニューに入力装置が表示されない場合は「QuickTime Broadcaster」を終了し、入力装置が接続されていて電源が入っているかどうか確認してから「QuickTime Broadcaster」をもう一度開いてください。それでも入力装置が表示されない場合は、入力装置が「QuickTime Broadcaster」と互換性がない可能性があります。
