Ändern der Audioquelle

Der QuickTime Broadcaster arbeitet mit Audiodaten von einer DV-Kamera oder einer iSight-Kamera von Apple, den Audioeingängen des Computers sowie mit USB-Audioeingabegeräten wie Griffin iMic.

Gehen Sie wie folgt vor, um eine Audioquelle auszuwählen:

  1. Klicken Sie im QuickTime Broadcaster-Fenster zuerst auf "Details einblenden" (falls das erweiterte Fenster nicht geöffnet ist) und dann auf "Audio".

  2. Wählen Sie eine Audioquelle aus dem Einblendmenü "Quelle" aus.

Wenn eine Audioquelle angeschlossen ist und Daten überträgt, sollte Ton über den Computer ausgegeben werden (sofern "Wiedergabe über Lautsprecher" im Bereich "Audio" markiert ist). Die Audioeinstellungen sollten verfügbar (also nicht grau angezeigt) sein.

Wenn Ihre Audioquelle im Einblendmenü "Quelle" nicht angezeigt wird, beenden Sie den QuickTime Broadcaster und stellen Sie sicher, dass die Audiogeräte angeschlossen sind. Wiederholen Sie anschließend den Vorgang.
