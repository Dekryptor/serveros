Verwenden der Übertragungsmethode "Unicast (manuell)"

Die Option "Unicast (manuell)" ist eine Übertragungsmethode, die voraussetzt, dass Sie eine SDP-Datei vom QuickTime Broadcaster an einen Empfangscomputer senden, der durch seine IP-Adresse identifiziert wird. Das empfangende Programm kann ein QuickTime-Player, ein QuickTime Streaming Server (QTSS) oder ein Darwin Streaming Server (DSS) sein. Handelt es sich beim empfangenden Programm um QTSS oder DSS, wird die Sendung vervielfältigt und ist nicht mehr auf eine einzige IP-Adresse beschränkt.

Gehen Sie wie folgt vor, um über die Methode "Unicast (manuell)" zu senden:

  1. Klicken Sie im QuickTime Broadcaster auf "Details einblenden", falls das erweiterte Fenster nicht geöffnet ist.

  2. Klicken Sie auf "Netzwerk".

  3. Wählen Sie eine Option aus dem Einblendmenü "Voreinstellung" aus.

  4. Wählen Sie "Unicast (manuell)" aus dem Einblendmenü "Übertragung" aus.

  5. Geben Sie die IP-Adresse des empfangenden Computers ein.

    Zum Anzeigen der IP-Adresse eines Mac OS X-Computers öffnen Sie die Systemeinstellungen, klicken auf "Netzwerk" und wählen eine Option aus dem Einblendmenü "Anzeigen" aus. Auf einem Windows-Computer öffnen Sie in der Systemsteuerung den Bereich "Netzwerkverbindungen".

  6. Geben Sie die Nummern für den Audio- und den Videoport ein und legen Sie die Pufferverzögerung fest (oder akzeptieren Sie im Falle einer Eins-zu-Eins-Übertragung jeweils die Standardeinstellung).

  7. Auf Wunsch können Sie im Abschnitt "Anmerkungen" Informationen eingeben.

  8. Klicken Sie auf "Senden".

  9. Wählen Sie "Ablage" > "Exportieren" > "SDP" bzw. "Ablage" > "Exportieren" > "Film".

  10. Geben Sie einen Namen für die SDP- oder Filmdatei an und klicken Sie auf "Exportieren".

    Wenn die Datei direkt in QuickTime-Player exportiert wird, kann der Benutzer die Datei öffnen, indem er "Ablage" > "Datei öffnen" wählt.

    Wenn die Datei in QTSS oder DSS exportiert wird, wählt der Benutzer "Ablage" > "URL öffnen". Wenden Sie sich an Ihren QTSS- oder DSS-Administrator, um die korrekte URL-Adresse zu erfahren.
