SDP ファイルとは

Session Description Protocol（SDP）ファイルにはブロードキャストのパラメータと設定が記述されており、これによって「QuickTime Player」はブロードキャストを受信できます。ブロードキャストを表示するには、「QuickTime Player」で SDP ファイルを開きます。

SDP ファイルは、次の 3 つの転送方式で作成することができます：

  • 自動ユニキャスト（アナウンス）： 指定した「QuickTime Streaming Server」上に SDP ファイルを自動的に作成できます。ファイルのアドレスはこの場合にだけ表示されます。

  • 手動ユニキャスト： SDP ファイルは、ストリーミングサーバ上に配置されるか、「QuickTime Player」で直接開かれます。

  • マルチキャスト： SDP ファイルは「QuickTime Player」で手動で開かれます。

SDP ファイルを作成するには：

「QuickTime Broadcaster」で、「ファイル」＞「書き出す」＞「SDP」と選択します。

これで、SDP ファイルを単数の視聴者に送信するか、複数の視聴者がブロードキャストに接続できるようにするストリーミングサーバに送信できます。
