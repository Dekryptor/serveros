使用可能な帯域幅を最大限活用する

ブロードキャストを構成するときは、次の点を考慮してください：

  • ブロードキャスト時のアップロード速度。ほとんどのインターネット・サービス・プロバイダでは、アップロード速度を 128 ～ 256 キロビット／秒（Kbps）に制限しています。ブロードキャスト（オーディオおよびビデオ）の合計データレートがアップロード速度を超えないようにする必要があります。 実際には、ネットワークの混雑を考慮して、アップロード速度よりかなり低めに合計データレートを設定する必要があります。

  • 視聴者のネットワーク接続の速度。たとえば、視聴者が 56K モデムを使って接続している場合、ブロードキャストの全体的なデータレートを 33 Kbps に制限する必要があるでしょう。より高速な接続を使用している視聴者は、より高いレートを維持することができますが、T1 インターネットアクセスを使用している場合でも、ブロードキャストのデータレートを 410 Kbps に制限する必要があります。（どちらの数字も控えめな見積もり値です。）

  • ブロードキャストまたはストリーミングのサーバコンピュータがネットワークに接続する方法と、予想される視聴者の数。ストリーミングサーバコンピュータが T1 回線に接続されている場合、56K ダイヤルアップモデムを使用して、最高 25 人程度の視聴者がブロードキャストに接続できます。
