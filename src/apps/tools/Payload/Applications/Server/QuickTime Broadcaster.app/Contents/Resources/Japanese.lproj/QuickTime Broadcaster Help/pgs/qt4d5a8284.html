手動ユニキャストを使用する

手動ユニキャストは、「QuickTime Broadcaster」から IP アドレスで特定される受信側のコンピュータに、SDP ファイルを書き出さなくてはならないブロードキャスト方法です。受信側アプリケーションは、1 つの「QuickTime Player」、「QTSS」または「Darwin Streaming Server（DSS）」です。受信側アプリケーションが QTSS や DSS の場合、ブロードキャストが単一の IP アドレスに限定されることはなくなります。

手動ユニキャストを使ってブロードキャストするには：

  1. 「QuickTime Broadcaster」で、拡張ウインドウが開いていない場合は「詳細を表示」をクリックします。

  2. 「ネットワーク」をクリックします。

  3. 「プリセット」ポップアップメニューからオプションを選択します。

  4. 「転送」ポップアップメニューから「手動ユニキャスト」を選択します。

  5. 受信側コンピュータの IP アドレスを入力します。

    Mac OS X コンピュータの IP アドレスを確認するには、「システム環境設定」を開き、「ネットワーク」をクリックし、「表示」ポップアップメニューからオプションを選択します。Windows コンピュータの場合は、「コントロールパネル」を開き、「ネットワーク接続」を開きます。

  6. オーディオポート番号とビデオポート番号を入力し（1 対 1 のブロードキャストの場合はデフォルトをそのまま使用します）、バッファ遅延を入力します（または、デフォルトをそのまま使用します）。

  7. 必要に応じて、「注釈」セクションに情報を入力します。

  8. 「ブロードキャスト」をクリックします。

  9. 「ファイル」＞「書き出す」＞「SDP」、または「ファイル」＞「書き出す」＞「ムービー」と選択します。

  10. SDP ファイルまたはムービーファイルの名前を指定し、「書き出し」をクリックします。

    ファイルが「QuickTime Player」に書き出される場合は、ユーザは「ファイル」＞「ファイルを開く」と選択することによってファイルを開きます。

    ファイルが「QTSS」または「DSS」に書き出される場合は、ユーザは「ファイル」＞「URL を開く」と選択します。QTSS、DSS の管理者と相談して URL を決めてください。
