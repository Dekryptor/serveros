Utilisation de la multidiffusion

La multidiffusion est une méthode de diffusion via un réseau local (et non via Internet) qui n’utilise qu’un seul flux, partagé par tous les spectateurs de la diffusion. Chaque spectateur se règle sur la diffusion comme une radio se règle sur une diffusion FM.

La multidiffusion suppose un réseau ayant accès à un « multicast backbone » (MBone) ou apte à la multidiffusion.

Vous pouvez réaliser une multidiffusion via un réseau local dans la mesure où il y est habilité. Contactez votre administrateur de réseau afin de savoir si votre réseau gère la multidiffusion.

Pour réaliser une multidiffusion via un réseau local :

  1. Dans QuickTime Broadcaster, si la fenêtre développée n’est pas ouverte, cliquez sur Afficher les détails.

  2. Cliquez sur Réseau.

  3. Choisissez Multidiffusion dans le menu local Transmission.

  4. Cliquez sur « Créer une adresse IP ».

    Ceci saisit l’adresse IP que vous utilisez pour envoyer la multidiffusion. Une adresse est créée dans une plage réservée aux multidiffusions. Si votre administrateur de réseau a attribué des adresses de multidiffusion spécifiques, vous pouvez taper une adresse manuellement.

  5. Spécifiez un réglage Time To Live (TTL) (ou acceptez la valeur par défaut).

    Ce réglage spécifie le nombre de fois qu’un flux de données peut être transmis d’un routeur à l’autre avant de ne plus être transmis. Il peut s’agir de tout nombre compris entre 1 et 255. La valeur 1 atteint les ordinateurs clients sur le réseau local. Plus le nombre est élevé, plus les paquets de multidiffusion peuvent aller loin.

  6. Choisissez Fichier > Exporter > SDP ou Fichier > Exporter > Séquence.

    Ceci crée un fichier Session Description Protocol (SDP) ou un fichier de séquence QuickTime que vous pouvez envoyer à d’autres personnes. Pour visionner la diffusion, double-cliquez sur le fichier SDP ou utilisez QuickTime Player pour ouvrir le fichier de séquence.
