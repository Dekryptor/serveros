Sichern von Broadcast-Einstellungen

Mit dem QuickTime Broadcaster können Sie eigene Broadcast-Einstellungen festlegen und als Dokumente für die erneute Verwendung sichern und mit anderen Benutzern gemeinsam verwenden.

Gehen Sie wie folgt vor, um eine eigene Broadcast-Einstellung zu sichern:

  1. Wählen Sie Ihre Einstellungen für Audio, Video und Netzwerk aus.

  2. Wählen Sie "Ablage" > "Broadcast-Einstellungen sichern unter".

  3. Geben Sie einen Dateinamen und den Speicherort an.

Die Datei wird mit der Erweiterung ".qtbr" gesichert.

Nachdem die Datei gesichert wurde, wird durch Doppelklicken auf das Dateisymbol der QuickTime Broadcaster (sofern installiert) mit Ihren Einstellungen geöffnet. Sie können dann auf "Senden" klicken, um den Streaming-Vorgang zu starten.
