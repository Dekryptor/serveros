プリセットを使用して設定を保存する

オーディオ、ビデオ、およびネットワークの設定は、後で再利用できるように保存することができます。

プリセットを作成するには：

  1. 「QuickTime Broadcaster」の「オーディオ」パネル、「ビデオ」パネル、または「ネットワーク」パネルで、設定を選択します。

    これらのパネルが表示されていない場合は、「詳細を表示」をクリックします。

  2. 「プリセット」ポップアップメニューから「プリセットを保存」を選択します。

  3. これらの設定を再利用するには、各パネル（「オーディオ」、「ビデオ」、または「ネットワーク」）にある「プリセット」ポップアップメニューからプリセットを選択します。

  4. プリセットを別のコンピュータに転送するには、「QuickTime Broadcaster」＞「環境設定」と選択し、「プリセットディレクトリ」フィールドにパスを入力します（または、「選択」をクリックし、転送先を選択します）。

    注記： プリセットは、「~/ライブラリ/QuickTime Broadcaster」に配置する必要があります。ほかの人がプリセットを使う場合は、自分のコンピュータのこれと同じ場所に自分のプリセットファイルを入れる必要があります。
