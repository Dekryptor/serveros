Auswählen des Packetizer

In den meisten Fällen müssen die Standardeinstellungen des Packetizer für die einzelnen Kompressoren nicht geändert werden. Wenn Sie RTP kennen und eine Feineinstellung des Packetizer vornehmen möchten, können Sie die Einstellungen ändern. Klicken Sie hierzu auf die Taste "Packetizer" in den Bereichen "Audio" und "Video" und geben Sie dann Werte in die angezeigten Dialogfenster ein.

  • Wenn Sie AAC an Mobiltelefone senden, stellen Sie den Packetizer auf "MPEG-4 LATM Audio" ein.

  • Wenn Sie H.263 an Mobiltelefone senden, stellen Sie den Video-Packetizer auf "H.263 3GPP" ein.
