Verwenden von Protokolldateien zum Überwachen der Übertragung von Wiedergabelisten

Wenn Sie die Protokollierung aktivieren, können Sie mithilfe der Protokolldatei Fehler beheben, die während einer Übertragung auftreten. Hier finden Sie Tipps zum Beseitigen potenzieller Probleme:

  • Werden die in der Wiedergabeliste enthaltenen Medien nicht übertragen, vergewissern Sie sich mithilfe des Programms "Server-Admin", dass der Streaming-Server in Betrieb ist.

  • Ist der Streaming-Server in Betrieb, stellen Sie sicher, dass ein Prozess mit dem Titel "PlaylistBroadcaster" ausgeführt wird. Ist das der Fall, müssen Sie die Übertragung stoppen, die entsprechende SDP-Datei aus dem Medienordner des Streaming-Servers löschen und die Übertragung anschließend neu starten. Wenn Sie die Übertragung erneut starten, wird eine neue SDP-Datei generiert.
