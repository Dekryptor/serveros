Utilisation de la Monodiffusion automatique (Annonce)

Vous pouvez utiliser QuickTime Broadcaster pour diffuser sur QTSS à l’aide de la méthode de transmission Monodiffusion automatique (annonce). Pour ce faire, vous devez disposer :

  • D’un compte d’utilisateur de diffusion. QuickTime Broadcaster va demander le nom d’utilisateur et le mot de passe du compte d’utilisateur de diffuseur pour autoriser la création d’un fichier SDP sur le serveur d’enchaînement. Si vous utilisez QuickTime Broadcaster et QTSS sur le même ordinateur, vous n’avez pas besoin de compte d’utilisateur de diffuseur.

  • De QTSS 4.0 ou ultérieur installé sur le serveur d’enchaînement.

Pour lancer une monodiffusion automatique :

  1. Dans QuickTime Broadcaster, cliquez sur Afficher les détails (si la fenêtre n’est pas développée), puis sur Réseau.

  2. Choisissez Monodiffusion automatique (Annonce) dans le menu local Transmission.

  3. Tapez l’adresse IP ou le nom d’hôte du serveur de réception, le nom du fichier de diffusion, le nom et le mot de passe du compte d’utilisateur du serveur d’enchaînement (si nécessaire) et le délai de mise en tampon (ou acceptez le délai par défaut).

  4. Cliquez sur Diffuser.

    Toutes les informations sont envoyées sur le serveur. Il est inutile de créer un fichier SDP.
