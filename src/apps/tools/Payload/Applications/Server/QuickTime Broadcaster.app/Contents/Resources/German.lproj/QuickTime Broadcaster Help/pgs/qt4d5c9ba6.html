Verwenden der Übertragungsmethode "Unicast (automatisch mit Ankündigung)"

Mit dem QuickTime Broadcaster können Sie für die Übertragung auf einen QuickTime Streaming Server (QTSS) die Übertragungsmethode "Unicast (automatisch mit Ankündigung)" verwenden. Dazu benötigen Sie Folgendes:

  • Einen Broadcaster-Benutzer-Account. Sie werden im QuickTime Broadcaster aufgefordert, den Benutzernamen und das Kennwort für den Account einzugeben, damit eine SDP-Datei auf dem Streaming-Server erstellt werden kann. Wenn Sie den QuickTime Broadcaster und QTSS auf demselben Computer verwenden, ist kein Broadcaster-Benutzer-Account erforderlich.

  • QTSS 4.0 (oder neuer) muss auf dem Streaming-Server installiert sein.

Gehen Sie wie folgt vor, um eine automatische Unicast-Sendung zu starten:

  1. Klicken Sie im QuickTime Broadcaster zuerst auf "Details einblenden" (falls das erweiterte Fenster nicht geöffnet ist) und dann auf "Netzwerk".

  2. Wählen Sie "Unicast (automatisch mit Ankündigung)" aus dem Einblendmenü "Übertragung" aus.

  3. Geben Sie die IP-Adresse oder den Host-Namen des empfangenden Servers ein, den Namen der Sendedatei, falls erforderlich den Benutzernamen und das Kennwort für den Benutzer-Account auf dem Streaming-Server sowie die Pufferverzögerung (oder akzeptieren Sie den Standardwert).

  4. Klicken Sie auf "Senden".

    Alle Informationen werden an den Server gesendet. Sie müssen hierzu keine SDP-Datei erstellen.
