AppleScript で QuickTime Broadcaster を自動化する

AppleScript を使って「QuickTime Broadcaster」を制御することができます。たとえば、スクリプトを使って、使用するプリセットを選択したり、ブロードキャストの状態を決定したり、ストリームの開始、停止、一時停止を行うことができます。

「QuickTime Broadcaster」がサポートする AppleScript の機能について詳しくは、「/アプリケーション/AppleScript/」にある「スクリプトエディタ」アイコンに「QuickTime Broadcaster」アイコンをドラッグしてください。
