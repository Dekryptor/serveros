Limitation de l’accès à votre diffusion

Pour limiter l’accès à votre diffusion, vous pouvez incorporer la séquence à diffuser dans une page Web et contrôler l’accès à la page HTML à l’aide des réglages de sécurité du serveur web.

Si vous distribuez le fichier SDP ou la séquence d’annonce de diffusion, toute personne qui reçoit le fichier sera en mesure de visionner le flux, à moins que la diffusion ne se fasse à partir d’un coupe-feu d’entreprise. Dans le cas d’une monodiffusion manuelle, seul l’ordinateur avec l’adresse IP spécifiée pourra se connecter.
