Si on vous demande un mot de passe

Lorsque vous diffusez vers QTSS ou DSS à l’aide de la méthode de diffusion Monodiffusion automatique (Annonce), vous devez fournir un nom d’utilisateur et un mot de passe afin qu’un compte d’utilisateur et un fichier de diffusion puissent être créés sur le serveur d’enchaînement. Pour obtenir des détails sur la manière de créer le compte d’utilisateur spécial, reportez-vous à l’Aide QTSS ou DSS.

Le nom d’utilisateur et le mot de passe requis pour la monodiffusion automatique sont différents de ceux utilisés pour l’utilitaire d’administration web du serveur d’enchaînement.
