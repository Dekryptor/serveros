ムービーファイルをヒントする

ブロードキャストのコピーを圧縮およびヒントし、ムービーファイルとして保存することができます。ムービーはブロードキャスト元のコンピュータに保存され、視聴者がイベントの再放送を見られるように、QuickTime ストリーミングサーバへ送信する準備ができた状態になります。

ブロードキャストを記録し、ムービーファイルをヒントするには：

  1. 「QuickTime Broadcaster」＞「環境設定」と選択します。

  2. 「記録ディレクトリ」フィールドに、ブロードキャストファイルを保存するパスを指定します（または「選択」をクリックして場所を選択します）。

  3. 「Streaming Server 用ヒント」を選択します。

    このオプションは、ムービーをストリーミングするために必要な情報を含んだヒント・トラックを追加します。

  4. ムービーの名前を入力して「OK」をクリックします。

  5. 「QuickTime Broadcaster」ウインドウで、「ディスクに記録」を選択します。

  6. 「ブロードキャスト」をクリックします。

    注記： アーカイブされたブロードキャストは、大量のディスク領域を使用することがあります。ブロードキャストが数時間を超える場合には、複数の .mov ファイルが作成されることがあります。そのような場合、ファイルには連続した番号が付きます（「mymovie.mov」、「mymovie2.mov」など）。
