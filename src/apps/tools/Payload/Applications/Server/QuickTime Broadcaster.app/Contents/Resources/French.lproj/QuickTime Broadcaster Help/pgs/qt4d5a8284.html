Utilisation de Monodiffusion manuelle

La monodiffusion manuelle est une méthode de diffusion qui nécessite que vous exportiez un fichier SDP de QuickTime Broadcaster à un ordinateur récepteur identifié par son adresse IP. L’application réceptrice peut être un lecteur QuickTime unique, QTSS ou Darwin Streaming Server (DSS). Si l’application réceptrice est QTSS ou DSS, la diffusion est renvoyée et n’est donc plus limitée à une seule adresse IP.

Pour diffuser via Monodiffusion manuelle :

  1. Dans QuickTime Broadcaster, si la fenêtre développée n’est pas ouverte, cliquez sur Afficher les détails.

  2. Cliquez sur Réseau.

  3. Choisissez une option dans le menu local Préréglage.

  4. Choisissez Monodiffusion manuelle dans le menu local Transmission.

  5. Entrez l’adresse IP de l’ordinateur récepteur.

    Pour voir l’adresse IP d’un ordinateur sous Mac OS X, ouvrez Préférences Système, cliquez sur Réseau, puis choisissez une option dans le menu local Afficher. Sur un ordinateur sous Windows, ouvrez le Panneau de configuration, puis Connexions réseau.

  6. Tapez les numéros des ports audio et vidéo (ou acceptez les valeurs par défaut si vous diffusez de un à un) et la durée de la mémoire tampon (ou acceptez la valeur par défaut).

  7. Si vous le souhaitez, tapez des informations dans la section Annotation.

  8. Cliquez sur Diffuser.

  9. Choisissez Fichier > Exporter > SDP ou Fichier > Exporter > Séquence.

  10. Donnez un nom au fichier SDP ou de film et cliquez sur Exporter.

    Si le fichier est exporté directement vers QuickTime Player, l’utilisateur ouvre le fichier en choisissant Fichier > Ouvrir un fichier.

    Si le fichier est exporté vers QTSS ou DSS, l’utilisateur choisit Fichier > Ouvrir l’URL. Adressez-vous à votre administrateur QTSS ou DSS pour connaître l’URL correcte.
