Ajout d’annotations à votre diffusion

Vous pouvez ajouter à votre diffusion un titre, un copyright ou toute autre annotation.

Pour ajouter des annotations :

  1. Dans QuickTime Broadcaster, cliquez sur Réseau.

    Si le bouton Réseau n’apparaît pas, cliquez sur le bouton Afficher les détails.

  2. Sous Notes, complétez les champs souhaités.

Votre public verra ces informations en choisissant Fenêtre > « Information sur la séquence » dans QuickTime Player. (Les spectateurs qui ne disposent pas de QuickTime Pro ne verront que les trois premières annotations).
