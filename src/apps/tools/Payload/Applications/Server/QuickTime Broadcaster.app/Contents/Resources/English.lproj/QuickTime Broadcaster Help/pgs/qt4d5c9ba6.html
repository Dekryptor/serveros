Using Automatic Unicast (Announce)

You can use QuickTime Broadcaster to broadcast to QTSS using the Automatic Unicast (Announce) transmission method. To do so, you need:

  • A broadcaster user account. QuickTime Broadcaster will ask for the account’s user name and password, to allow an SDP file to be created on the streaming server. If you’re using QuickTime Broadcaster and QTSS on the same computer, you don’t need a broadcaster user account.

  • QTSS 4.0 or later installed on the streaming server.

To start an Automatic Unicast broadcast:

  1. In QuickTime Broadcaster, click Show Details (if the expanded window isn’t open) and click Network.

  2. Choose Automatic Unicast (Announce) from the Transmission pop-up menu.

  3. Enter the IP address or host name of the receiving server, the name of the broadcast file, the user name and password for the streaming server user account (if needed), and the buffer delay (or accept the default).

  4. Click Broadcast.

    All information is sent to the server. You do not need to create an SDP file.
