QuickTime Broadcaster

You can use QuickTime Broadcaster to stream live audio and video over the Internet using a FireWire DV (Digital Video) source and MacþOSþX. When teamed with QuickTime Streaming Server (QTSS), QuickTime Broadcaster can produce a live event for delivery to an audience of any size, from an individual to a large global audience.

QuickTime Broadcaster meets the needs of beginners and professionals. It provides preset broadcast settings so you can easily set up a broadcast without spending time configuring. For more control, you can easily create custom settings.

Built using the QuickTime architecture, QuickTime Broadcaster enables you to produce a live event using most codecs that QuickTime supports, including H.264.
