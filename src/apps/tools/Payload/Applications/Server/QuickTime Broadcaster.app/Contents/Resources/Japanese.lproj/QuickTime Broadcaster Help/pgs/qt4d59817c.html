ファイアウォールについて

内部ネットワークを外部から保護するネットワークファイアウォールは、ブロードキャストの送受信を妨害することがあります。

ブロードキャストする場合は、ネットワーク管理者に連絡し、送信ポートがブロードキャスト用に開放されていることを確認してください。多くの ISP や企業ネットワークは、これらのポートをブロックし、帯域幅の使用を制限してセキュリティを強化します。

ブロードキャストを送信するときに、「QuickTime Broadcaster」はデフォルトで次のポートを使用します：

  • 自動ユニキャスト（アナウンス）： 554。

  • 手動ユニキャスト： 「ネットワーク」パネルでポート情報を入力します。

ポート番号を変更するには、IP アドレスの末尾にコロンとポート番号を追加します。たとえば、127.0.0.1:7070 のようにします。

ファイアウォールの外側にいる視聴者は、QuickTime 環境設定のネットワーク設定を調整するか、「ネットワーク」環境設定でプロキシサーバの使用を構成することで、ブロードキャストを視聴できる場合があります。
