Utilisation des fichiers d’historique pour contrôler la liste de lecture des diffusions

Si vous activez le listage, vous pouvez utiliser le fichier d’historique pour résoudre les problèmes survenant durant une diffusion. Voici quelques astuces pour résoudre d’éventuels problèmes :

  • Si les données multimédia faisant partie de la liste de lecture ne sont pas diffusées, utilisez Admin Serveur pour vous assurer que le serveur de diffusion est en cours d’exécution.

  • Si tel est le cas, assurez-vous qu’un processus appelé PlaylistBroadcaster est exécuté sur l’ordinateur serveur. Stoppez la diffusion, supprimez le fichier SDP depuis le serveur de diffusion, puis redémarrez celle-ci. Un nouveau fichier SDP est généré lors de la reprise de la diffusion.
