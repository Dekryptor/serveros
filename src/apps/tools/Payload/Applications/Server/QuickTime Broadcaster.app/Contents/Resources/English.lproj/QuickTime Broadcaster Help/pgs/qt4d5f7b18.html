I am Being Asked for a Password

When you broadcast to QTSS or DSS using Automatic Unicast (Announce), you must provide a user name and password so that a user account and a broadcast file can be created on the streaming server computer. For details on how to create the special user account, see QTSS or DSS Help.

The user name and password required for automatic unicast are not the same as those used for the streaming server web admin utility.
