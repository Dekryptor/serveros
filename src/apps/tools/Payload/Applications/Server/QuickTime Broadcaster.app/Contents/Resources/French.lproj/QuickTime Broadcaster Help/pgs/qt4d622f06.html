Automatisation de QuickTime Broadcaster avec AppleScript

Vous pouvez utiliser AppleScript pour contrôler QuickTime Broadcaster. Par exemple, vous pouvez utiliser un script pour sélectionner quels préréglages utiliser, déterminer l’état d’une diffusion et démarrer, arrêter ou mettre en pause un flux.

Pour obtenir des informations détaillées à propos des fonctionnalités AppleScript prises en charge par QuickTime Broadcaster, faites glisser l’icône de QuickTime Broadcaster sur l’icône Éditeur de script qui se trouve dans le dossier Applications/AppleScript/.
