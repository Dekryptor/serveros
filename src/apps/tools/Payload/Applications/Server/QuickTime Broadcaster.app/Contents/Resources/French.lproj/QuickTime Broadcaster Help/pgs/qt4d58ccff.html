Indices du fichier de séquence

Vous pouvez enregistrer une copie de votre diffusion compressée et contenant des indices comme un fichier de séquence. La séquence est enregistrée sur l’ordinateur à partir duquel vous diffusez et est prêt à être posté sur un serveur d’enchaînement QuickTime afin que les spectateurs puissent visualiser une rediffusion de l’événement.

Pour enregistrer votre diffusion et ajouter des indices à un fichier de séquence :

  1. Choisissez QuickTime Broadcaster > Préférences.

  2. Dans le champ Enregistrement, spécifiez le chemin (ou cliquez sur Choisir pour choisir un emplacement) dans lequel vous souhaitez enregistrer le fichier de diffusion.

  3. Sélectionnez « Indice pour le serveur d’enchaînement ».

    Cette option ajoute une piste d’indices contenant les informations requises pour diffuser votre séquence en continu.

  4. Tapez le nom de votre séquence, puis cliquez sur OK.

  5. Dans la fenêtre QuickTime Broadcaster, sélectionnez « Enregistrer sur le disque ».

  6. Cliquez sur Diffuser.

    REMARQUE :  une diffusion archivée peut utiliser beaucoup d’espace disque. Si la diffusion dure plus que quelques heures, plusieurs fichiers .mov risquent d’être créés. Si tel est le cas, les fichiers sont numérotés de manière séquentielle (par exemple, « maséquence.mov », « maséquence2.mov », etc.).
