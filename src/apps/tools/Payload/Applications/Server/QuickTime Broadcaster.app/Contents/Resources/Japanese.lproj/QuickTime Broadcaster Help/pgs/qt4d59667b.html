オーディオソースを変更する

「QuickTime Broadcaster」は、DV カメラまたはアップルの iSight カメラ、コンピュータのオーディオ入力ポート、Grifin iMic などの USB オーディオ入力装置などから提供されるオーディオを処理します。

オーディオソースを選択するには：

  1. 「QuickTime Broadcaster」ウインドウで、（拡張ウインドウが開いていない場合は「詳細を表示」をクリックしてから）「オーディオ」をクリックします。

  2. 「入力源」ポップアップメニューからオーディオソースを選択します。

オーディオソースが接続されていて入力を提供中であれば、コンピュータを介してオーディオが聞こえ（「オーディオ」パネルの「スピーカーで再生」が選択されている場合）、さらにオーディオ設定が使用可能な状態（薄く表示されていない）になっているはずです。

「入力源」ポップアップメニューにオーディオソースが表示されない場合は、「QuickTime Broadcaster」を終了して装置が接続されているかどうか確認してから、もう一度同じことを試みてください。
