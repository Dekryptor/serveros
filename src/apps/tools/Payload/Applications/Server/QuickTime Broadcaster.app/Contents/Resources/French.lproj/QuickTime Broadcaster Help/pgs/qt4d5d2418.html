Enregistrements de réglages de diffusion

Grâce à QuickTime Broadcaster, vous pouvez créer vos réglages de diffusion personnalisés et les enregistrer comme documents pour les réutiliser et les partager.

Pour enregistrer un réglage de diffusion personnalisé :

  1. Choisissez vos réglages audio, vidéo et réseau.

  2. Choisissez Fichier > Enregistrer les réglages de diffusion sous.

  3. Entrez un nom pour le fichier et spécifiez un emplacement pour l’enregistrement.

Le fichier enregistré portera l’extension .qtbr.

Une fois le fichier enregistré, le fait de double-cliquer sur son icône ouvrira QuickTime Broadcaster (s’il est installé) avec les réglages personnalisés. Vous pouvez ensuite cliquer sur Diffuser pour commencer la diffusion.
