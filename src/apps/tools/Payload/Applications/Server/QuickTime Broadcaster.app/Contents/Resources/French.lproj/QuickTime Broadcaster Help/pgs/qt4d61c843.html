Aperçu de votre diffusion

Vous pouvez avoir un aperçu de votre diffusion avec ou sans compression dans la fenêtre QuickTime Broadcaster. Il est également possible de désactiver la fonction d’aperçu. Choisissez l’une des options suivantes :

  • Pour obtenir un aperçu de votre diffusion à partir de la source, choisissez Source dans le menu local Aperçu (au-dessus de la zone d’aperçu vidéo).

  • Pour prévisualiser votre diffusion avec compression, choisissez Compresser dans le menu local Aperçu, puis sélectionnez des préréglages dans les menus locaux Audio et Vidéo. (S’il n’y a pas de menu local sous la zone d’aperçu, cliquez sur Masquer les détails.)

    Vous pouvez aussi cliquer sur Afficher les détails. Dans les panneaux Audio et Vidéo, choisissez des préréglages dans le menu local Préréglage ou spécifiez des réglages de compression personnalisés. Dans le panneau Vidéo, ajustez la réglette Qualité afin d’obtenir la meilleure image possible.

  • Pour désactiver la fonction d’aperçu, choisissez Aucun dans le menu local Aperçu.

    L’aperçu vous montre la manière dont votre flux apparaît dans des circonstances idéales. Il ne prend pas en compte les retards potentiels du réseau ou les limitations de la bande passante.
