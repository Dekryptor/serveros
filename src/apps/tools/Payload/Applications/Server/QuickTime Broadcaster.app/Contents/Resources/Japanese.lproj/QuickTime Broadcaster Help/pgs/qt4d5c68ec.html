ビデオがぼやけたり、コマ落ちしたり、フリーズしたりする場合

ブロードキャストがぼやけたり、コマ落ちしたり、フリーズする場合、カメラの焦点が合っていない、ネットワークが混雑している、圧縮の設定が不適切などの原因が考えられます。

ブロードキャストの品質を改善するには、次にあげる対策を 1 つ以上試みてください：

  • カメラの焦点を合わせる。

  • （「サーバ管理」の「QTSS」パネルを使用して）同時視聴者の数を減らす。

  • 受信者側で再生中にコマ落ちが発生する場合は、ブロードキャストのデータレートを調整する。

  • H.264 や MPEG-4 など、ストリーミング用に最適化されたビデオコーデックを使ってブロードキャストを行っていることを確認する。

  • ビデオのフレームレートとフレームサイズを小さくする。

高品質のブロードキャストを実現する最善の方法は、ブロードキャストを配信する前にさまざまな設定をテストすることです。これにより、コンテンツの種類やユーザの接続速度に応じた、最適な設定の組み合わせを決定することができます。
