Automatisieren des QuickTime Broadcaster mit AppleScript

Sie können AppleScript-Skripte zur Steuerung des QuickTime Broadcaster verwenden. So können Sie beispielsweise ein Skript zur Auswahl der gewünschten Voreinstellungen, zum Festlegen des Status der Sendung sowie zum Starten, Beenden oder Anhalten eines Datenstroms einsetzen.

Wenn Sie weitere Informationen über die AppleScript-Funktionen wünschen, die vom QuickTime Broadcaster unterstützt werden, bewegen Sie das QuickTime Broadcaster-Symbol auf das AppleScript-Editors-Symbol im Ordner "Programme/AppleScript".
