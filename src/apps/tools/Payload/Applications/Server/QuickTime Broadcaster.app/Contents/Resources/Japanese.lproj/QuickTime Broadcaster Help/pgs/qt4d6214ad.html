ブロードキャストイメージのサイズを変更する

視聴者が表示するイメージのサイズを変更できます。フレームサイズは大きい方が望ましいのですが、より広い帯域幅が必要になり、コンピュータの速度が遅い場合はブロードキャストと受信の両方で、過度の負担となります。コンサートや講演など、ブロードキャストでビデオよりもサウンドの方が重要な場合は、フレームサイズを小さくして、オーディオに割り当てられる帯域幅を広くすることを検討してください。

視聴者が受信するビデオフレームのサイズを変更するには：

  1. 「QuickTime Broadcaster」ウインドウで、（拡張ウインドウが開いていない場合は「詳細を表示」をクリックしてから）「ビデオ」をクリックします。

  2. 「圧縮」領域で、「幅」および「高さ」フィールドに新しい数値を入力します。

この手順では、「QuickTime Broadcaster」の「プレビュー」イメージのサイズは変わりません。
