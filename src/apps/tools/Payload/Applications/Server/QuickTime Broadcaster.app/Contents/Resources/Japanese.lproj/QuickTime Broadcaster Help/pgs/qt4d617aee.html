QuickTime Broadcaster

「QuickTime Broadcaster」を使えば、FireWire DV（デジタルビデオ）ソースと Mac OS X を使って、インターネット経由でライブオーディオやライブビデオをストリーミングすることができます。「QTSS（QuickTime Streaming Server）」と組み合わせた場合、「QuickTime Broadcaster」では、個人から大規模でグローバルな視聴者にいたるまでのあらゆる規模の視聴者への配信に合わせたライブイベントを製作できます。

「QuickTime Broadcaster」は、初心者とプロフェッショナルのニーズを満たしています。プリセットのブロードキャスト設定が用意されているので、時間をかけず簡単にブロードキャストを設定できます。詳細な制御を行う場合は、カスタム設定を簡単に作成できます。

「QuickTime Broadcaster」は QuickTime アーキテクチャを使用して構築されているので、H.264 など、QuickTime がサポートする大部分のコーデックを使ってライブイベントを製作できます。
