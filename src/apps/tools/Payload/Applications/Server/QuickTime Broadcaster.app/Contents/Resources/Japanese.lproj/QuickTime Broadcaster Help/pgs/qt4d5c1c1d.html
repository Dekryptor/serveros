ブロードキャストに注釈を追加する

ブロードキャストにタイトルや著作権情報などの注釈を追加できます。

注釈を追加するには：

  1. 「QuickTime Broadcaster」で「ネットワーク」をクリックします。

    「ネットワーク」ボタンが表示されない場合は、「詳細を表示」をクリックしてください。

  2. 「注釈」で、必要なフィールドに入力します。

視聴者は、「QuickTime Player」で「ウインドウ」＞「ムービーの情報を表示」と選択することによって、この情報を見ることができます。（「QuickTime Pro」を持っていない視聴者には、最初の 3 つの注釈だけが表示されます。）
