Configuration des ordinateurs et des groupes d’ordinateurs

Pour gérer un ordinateur, vous devez créer un compte d’ordinateur. Pour gérer un groupe d’ordinateurs, vous devez créer un groupe d’ordinateurs composé de comptes d’ordinateurs ou d’autres groupes d’ordinateurs.

Vous pouvez utiliser Gestionnaire de groupe de travail pour consulter, créer, modifier et supprimer des ordinateurs et des groupes d’ordinateurs.
