Einrichten von Benutzer-Accounts

Benutzer-Accounts sorgen für die eindeutige Identität der Benutzer Ihres Netzwerks und erlauben es Ihnen, diese Benutzer zu verwalten.

Mit dem Arbeitsgruppenmanager können Sie Benutzer-Accounts anzeigen, erstellen, bearbeiten und löschen.

Informationen zu Benutzer-Accounts

Hier erfahren Sie mehr über Benutzer-Accounts.

Informationen zu Benutzer-Accounts
Verwalten von Benutzer-Accounts

Sie können Benutzer-Accounts anzeigen, erstellen, bearbeiten und löschen, die in verschiedenen Arten von Verzeichnis-Domains gespeichert sind.

Verwalten von Benutzer-Accounts
Arbeiten mit Vorgaben

Vorgaben sind Vorlagen, mit denen Sie Attribute definieren, die automatisch für neue Benutzer- oder Gruppen-Accounts gelten.

Arbeiten mit Vorgaben
Arbeiten mit Basiseinstellungen

Als Basiseinstellungen bezeichnet man eine Sammlung von Attributen, die für alle Benutzer definiert werden müssen.

Arbeiten mit Basiseinstellungen
Arbeiten mit Zugriffsrechten

Sie können einem Benutzer-Account umfassende oder eingeschränkte Administratorrechte für eine Domain zuweisen. Bei der Erteilung eingeschränkter Administratorrechte können Sie festlegen, welche Benutzer und Gruppen der betreffende Benutzer auf welche Weise verwalten kann.

Arbeiten mit Zugriffsrechten
Arbeiten mit erweiterten Einstellungen

Zu den erweiterten Einstellungen gehören Anmeldeeinstellungen, Schlüsselwörter, Kennworttyp und durchsuchbare Kommentare.

Arbeiten mit erweiterten Einstellungen
Arbeiten mit Gruppeneinstellungen

Gruppeneinstellungen identifizieren die Gruppen, zu denen ein Benutzer gehört.

Arbeiten mit Gruppeneinstellungen
Arbeiten mit privaten Einstellungen

Die privaten Einstellungen beschreiben die Attribute eines Benutzerordners.

Arbeiten mit privaten Einstellungen
Arbeiten mit Mail-Einstellungen

Sie können einen Account im Mac OS X Server Mail-Dienst anlegen, indem Sie die E-Mail-Einstellungen im Account des betreffenden Benutzers festlegen. Um mit dem Account arbeiten zu können, konfiguriert der Benutzer einen Mail-Client so, dass die von Ihnen in den E-Mail-Einstellungen festgelegten Angaben für Benutzernamen, Kennwort und Mail-Protokoll übernommen werden.

Arbeiten mit Mail-Einstellungen
Arbeiten mit Einstellungen für Druckkontingente

Die Druckeinstellungen eines Benutzer-Accounts definieren, wie ein Benutzer Druckaufträge an ihm zugängliche Mac OS X Server-Druckwartelisten senden kann.

Arbeiten mit Einstellungen für Druckkontingente
Arbeiten mit Informationseinstellungen

Zu den Attributen, die im Info-Bereich aufgenommen werden, gehören Telefonnummer, E-Mail-Adresse, Weblog URL und URL der Startseite (Homepage).

Arbeiten mit Informationseinstellungen
Arbeiten mit Windows-Einstellungen

Mit diesen Einstellungen kann für Windows-Benutzer ein Benutzerordner, ein Profil und ein Windows-Anmeldeskript eingerichtet werden.

Arbeiten mit Windows-Einstellungen
