Einrichten von Gruppen-Accounts

Ein Gruppen-Account bietet eine einfache Möglichkeit für die Verwaltung mehrerer Benutzer mit ähnlichen Anforderungen. Sie können auch Gruppenordner anlegen. Diese bieten eine einfache Möglichkeit für die Gruppenmitglieder, Dateien untereinander auszutauschen bzw. gemeinsam zu nutzen.

Mit dem Arbeitsgruppenmanager können Sie Gruppen-Accounts anzeigen, erstellen, bearbeiten und löschen.

Arbeiten mit Einstellungen für Gruppenordner

Ein Gruppenordner bietet eine Möglichkeit, Dokumente und Programme zu verwalten und an Gruppenmitglieder zu verteilen. Die Gruppenmitglieder erhalten dadurch auch die Möglichkeit, Dateien untereinander auszutauschen bzw. gemeinsam zu nutzen.
