Rubriques d’aide les plus récentes

Votre ordinateur doit être connecté à Internet pour que vous puissiez bénéficier des nouvelles rubriques d’aide.

Cliquez sur Accueil, puis cliquez sur le lien « Rubriques d’aide les plus récentes » pour obtenir la liste des dernières rubriques ajoutées. Procédez ainsi régulièrement pour voir si de nouvelles rubriques d’aide ont été publiées.

Vous pouvez également rechercher sur le site web d’assistance de Mac OS X Server les informations les plus récentes en tapant dans le champ de recherche ci-dessus.
