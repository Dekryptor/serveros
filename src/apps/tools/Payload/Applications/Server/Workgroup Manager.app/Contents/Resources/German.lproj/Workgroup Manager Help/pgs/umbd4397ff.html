Neuste Hilfethemen

Ihr Computer muss über einen Internetzugang verfügen, damit Sie auf die aktuellsten Hilfethemen zugreifen können.

Öffnen Sie die Mac-Hilfe, klicken Sie auf "Start" (das Haussymbol) und anschließend auf den Link zu den neusten Hilfethemen, um eine Liste mit den aktuellen Themen anzuzeigen. Sehen Sie hier regelmäßig nach, ob neue Hilfethemen veröffentlicht wurden.

Sie können auch die Mac OS X Server Support-Website nach aktuellen Informationen durchsuchen. Geben Sie dazu den gewünschten Suchbegriff in das obige Suchfeld ein.
