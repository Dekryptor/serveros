Verwalten von Einstellungen

Durch das Verwalten von Einstellungen für Benutzer, Arbeitsgruppen, Computer und Computergruppen können Sie die Arbeitsoberflächen der Benutzer anpassen und den Zugriff der Benutzer auf von Ihnen ausgewählte Programme einschränken.

Zum Verwalten von Einstellungen verwenden Sie den Bereich "Einstellungen" im Arbeitsgruppenmanager.
