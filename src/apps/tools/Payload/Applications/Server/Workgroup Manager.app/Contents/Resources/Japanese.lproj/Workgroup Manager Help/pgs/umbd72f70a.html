コンピュータとコンピュータグループを設定する

個々のコンピュータを管理するには、コンピュータアカウントを作成する必要があります。コンピュータのグループを管理するには、コンピュータアカウントまたは別のコンピュータグループで構成されるコンピュータグループを作成する必要があります。

コンピュータとコンピュータグループを表示、作成、編集、および削除するときは、「ワークグループマネージャ」を使用します。
