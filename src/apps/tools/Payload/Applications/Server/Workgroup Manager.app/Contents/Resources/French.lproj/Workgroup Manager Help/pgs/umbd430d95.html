Prise en charge de fiche améliorée et augmentée

Les fiches d’utilisateur augmentées sont des comptes d’utilisateur importés qui vous permettent d’activer des réglages spécifiques, tels que la photo d’ouverture de session de l’utilisateur, ou de gérer les préférences.

Ces réglages sont spécifiques à la fiche augmentée. Vous n’avez pas besoin de modifier les comptes stockés dans le domaine de répertoire original. Les réglages que vous ne pouvez pas modifier sont synchronisés avec le domaine de répertoire depuis lequel ils ont été importés.

Vous pouvez utiliser des fiches d’utilisateur augmentées pour gérer les préférences des comptes stockés dans un domaine de répertoire actif ou dans un domaine Open Directory pour lequel vous ne possédez pas d’autorisations de modification.

Prise en charge au format MS-DOS (FAT) des comptes externes.

Les comptes externes sont pris en charge sur les disques externes utilisant le format MS-DOS (FAT).

Un compte externe est un compte mobile dont le dossier de départ local est stocké sur le volume d’un disque externe. Le répertoire de départ portable est créé à partir du dossier de départ local stocké sur ce disque externe et du dossier de départ réseau de l’utilisateur.

Les comptes externes prennent désormais en charge les volumes éjectables formatés au format Mac OS X étendu (HFS Plus) ou MS-DOS (FAT).
