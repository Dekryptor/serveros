ワークグループマネージャとは

Mac OS X Server に含まれている「ワークグループマネージャ」は、アカウントを作成および管理するために使用する、ユーザ管理ツールです。

アカウントを管理するときには、名前、パスワード、ホームフォルダの場所、グループメンバーシップなどの基本的なアカウント設定を定義できます。 また、環境設定を管理することによって、ユーザの操作環境をカスタマイズしたり、各ユーザのコンピュータの設定やネットワークリソースへのアクセスを許可または制限したりすることもできます。

「ワークグループマネージャ」は、ディレクトリドメインに密接に関連付けられます。ディレクトリドメインは、データベースに似ていますが、特にアカウント情報の保存や認証の処理のために設計されています。
