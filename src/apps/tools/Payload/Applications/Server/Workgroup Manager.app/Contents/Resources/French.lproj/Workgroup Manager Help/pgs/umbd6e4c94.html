Configuration des comptes de groupe

Un compte de groupe permet de gérer facilement un ensemble utilisateur aux besoins similaires. Vous avez également la possibilité de créer des dossiers de groupe, qui constituent un moyen simple de partage de fichiers entre les membres d’un groupe.

Gestionnaire de groupe de travail vous permet de consulter, de créer, de modifier et de supprimer des comptes de groupe.

Utilisation des réglages de membre des groupes
