オーギュメントレコードのサポートの向上

オーギュメント・ユーザ・レコードは読み込んだアカウントであり、ユーザのログインピクチャなどの特定の設定を有効にしたり、環境設定を管理したりできます。

これらの設定はオーギュメントレコードに固有です。元のディレクトリドメインに保管されているアカウントを編集する必要はありません。編集が許可されていない設定は、読み込み元のディレクトリドメインと同期されます。

オーギュメント・ユーザ・レコードを使用して、Active Directory ドメインまたは Open Directory ドメインに保管された、編集のアクセス権を持っていないアカウントの環境設定を管理することができます。

オーギュメント・ユーザ・レコードを作成する
外部アカウントは MS-DOS（FAT）フォーマットに対応します。

外部アカウントは、MS-DOS（FAT）フォーマットを使用する外部ドライブでサポートされます。

外部アカウントは、外部ドライブ上のボリュームにローカル・ホームフォルダが保管されているモバイルアカウントのことです。ポータブル・ホームディレクトリは、その外部ドライブに保管されているローカル・ホームフォルダとユーザのネットワーク・ホームフォルダから作成されます。

外部アカウントは、Mac OS X 拡張フォーマット（HFS Plus）または MS-DOS フォーマット（FAT）としてフォーマットされた外部ボリュームまたは取り出し可能ボリュームに対応するようになりました。
