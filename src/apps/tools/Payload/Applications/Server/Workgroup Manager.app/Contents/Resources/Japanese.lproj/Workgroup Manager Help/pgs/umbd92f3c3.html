環境設定を管理する

ユーザ、ワークグループ、コンピュータ、およびコンピュータグループの環境設定を管理することにより、ユーザの操作環境をカスタマイズしたり、選択したアプリケーションやネットワークリソースだけにユーザアクセスを制限したりできます。

環境設定を管理するときは、「ワークグループマネージャ」の「環境設定」パネルを使用します。
