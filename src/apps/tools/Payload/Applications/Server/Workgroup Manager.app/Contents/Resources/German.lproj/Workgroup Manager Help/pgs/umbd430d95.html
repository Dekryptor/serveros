Verbesserte Unterstützung für erweiterte Benutzereinträge

Bei den erweiterten Benutzereinträgen handelt es sich um importierte Benutzer-Accounts, mit denen Sie bestimmte Einstellungen z.B. ein Benutzerbild für die Anmeldung festlegen oder verwalten können.

Diese Einstellungen sind nur für erweiterte Benutzereinträge verfügbar. Es ist nicht erforderlich, die in der Originalverzeichnis-Domain gespeicherten Accounts zu ändern. Die Einstellungen, die Sie nicht ändern dürfen, werden mit der Verzeichnis-Domain synchronisiert, von der sie importiert werden.

Sie können erweiterte Benutzereinträge zum Verwalten der Einstellungen für Accounts verwenden, die in einer Active Directory-Domain oder in einer Open Directory-Domain gespeichert sind, für die Sie keine Schreibrechte besitzen.

Unterstützung für externe Accounts auf Volumes im MS-DOS (FAT)-Format.

Externe Accounts werden auf externen Laufwerken im Format "MS-DOS (FAT)" unterstützt.

Ein externer Account ist ein mobiler Account mit einem lokalen Benutzerordner, der auf einem Volume eines externen Laufwerks gespeichert ist. Der mobile Netzwerkordner wird aus dem lokalen Benutzerordner, der auf diesem externen Laufwerk gespeichert ist, und dem Netzwerk-Benutzerordner erstellt.

Externe Accounts unterstützen jetzt externe und auswerfbare Volumes, die im Format "Mac OS X Extended" (HFS Plus) oder "MS-DOS" (FAT) formatiert wurden.
