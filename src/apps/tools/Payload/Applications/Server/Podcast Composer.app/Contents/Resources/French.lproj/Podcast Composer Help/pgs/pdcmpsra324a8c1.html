Configuration des transitions

Vous pouvez configurer des transitions entre les vidéos lors de l’étape d’édition.

Pour configurer les transitions :

  1. Dans Podcast Composer, accédez à l’étape de l’édition (choisissez Étapes > Édition ou appuyez sur Commande + 3).

  2. Double-cliquez sur l’icône de la transition.

  3. Choisissez une transition dans le menu local Transition.

    Si vous n’avez pas besoin d’une transition, sélectionnez Aucune.

  4. Si le menu local Direction est activé, choisissez la direction de la transition.

  5. Si le champ Durée est activé, indiquez la durée de la transition en secondes.

  6. Cliquez sur Terminé.

  7. Choisissez Fichier > Enregistrer.
