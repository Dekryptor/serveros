Envoi de contenu à l’aide de protocoles de transfert de fichiers

Podcast Composer vous permet d’envoyer du contenu vers des systèmes de fichiers de sauvegarde à l’aide des protocoles de transfert suivants :

  • FTP

  • FTPS

  • SFTP

  • WebDav

  • WebDavs

Vous pouvez ajouter des destinations de transfert de fichiers.

Pour envoyer un podcast vers un système de fichiers de sauvegarde :

  1. Dans Podcast Composer, accédez à l’étape de publication (choisissez Étapes > Publier ou appuyez sur Commande + 5).

  2. Cliquez sur le bouton d’ajout (+) et sélectionnez Transfert de fichier.

    Une zone de dialogue apparaît.

  3. Dans le menu local, sélectionnez le format de podcast pour sauvegarder le système de fichiers.

  4. Dans le menu local Protocole, choisissez le protocole de transfert de fichiers à utiliser.

  5. Dans le champ Serveur, saisissez l’adresse du serveur vers lequel envoyer les fichiers.

  6. Dans le champ Chemin, indiquez l’emplacement où le podcast doit être stocké.

  7. Dans les champs Nom d’utilisateur et Mot de passe, saisissez les références de l’administrateur pour accéder au système de fichiers.

  8. Cliquez sur Terminé.

  9. Choisissez Fichier > Enregistrer.
