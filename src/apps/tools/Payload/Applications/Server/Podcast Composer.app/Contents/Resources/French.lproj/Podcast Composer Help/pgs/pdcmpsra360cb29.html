Vérification d’un processus

Avant de déployer un processus complet, il est conseillé de procéder à sa vérification, même si Podcast Composer réalise cette vérification avant de réaliser le déploiement. Le fait de vérifier le processus vous permet de le déployer très rapidement.

Pour vérifier un processus dans Podcast Composer, procédez comme suit :

  1. Sélectionnez Fichier > Vérifier le processus (Commande + Y).

    Si le processus n’est pas correctement configuré, un message s’affiche avec la description du problème au niveau du premier élément de processus pour lequel il manque des réglages. Une flèche rouge pointant vers l’élément en question s’affiche dans Podcast Composer.

  2. Si un message d’erreur s’affiche, procédez comme suit :

    1. Cliquez sur OK pour fermer la zone de dialogue.

    2. Résolvez le problème.

    3. Vérifiez à nouveau le processus.

    Il se peut que vous ayez à répéter plusieurs fois ces étapes pour que le processus passe avec succès l’étape de vérification.

  3. Une fois le processus correctement configuré, lorsqu’un message apparaît indiquant que la configuration du processus a abouti, cliquez sur OK.
