Previewing the Podcast

The Edit stage provides a live preview window where you can preview the QuickTime movie that is generated by the Edit stage. While the window is open, changes you make in the Edit stage are reflected in the live preview.

To open the live preview window:

  1. In Podcast Composer, go to the Edit stage (choose Stages > Edit or press Command-3).

  2. Click the Preview button.

    To display a bigger live preview window, alt-click the Preview button.
