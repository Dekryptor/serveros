Ajout de formats d’encodage QuickTime

Vous pouvez ajouter jusqu’à quatre formats d’encodage QuickTime lors de l’étape d’exportation.

Pour ajouter un format d’encodage QuickTime lors de l’étape d’exportation :

  1. Dans Podcast Composer, accédez à l’étape de l’exportation (choisissez Étapes > Exporter ou appuyez sur Commande + 4).

  2. Cliquez sur le bouton d’ajout (+).

    Si le module Compressor est installé, un menu local apparaît.

  3. Choisissez QuickTime.

    Un encodeur QuickTime est ajouté à l’étape d’exportation et une zone de dialogue apparaît.

  4. Dans le menu local Encodeur QuickTime, choisissez l’encodeur que vous souhaitez utiliser.

  5. Cliquez sur Terminé.

  6. Choisissez Fichier > Enregistrer.
