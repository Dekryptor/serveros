Envoi de contenu vers la bibliothèque de podcasts

La bibliothèque de podcasts est l’emplacement dans lequel vous stockez du contenu. Elle permet de publier le contenu par l’intermédiaire de flux RSS ou Atom, et de suivre automatiquement votre contenu.

Pour ajouter la bibliothèque de podcasts à l’étape de la publication :

  1. Dans Podcast Composer, accédez à l’étape de publication (choisissez Étapes > Publier ou appuyez sur Commande + 5).

    Par défaut, la bibliothèque de podcasts est ajoutée à l’étape de la publication.

  2. Si l’élément de bibliothèque de podcasts est présent, double-cliquez dessus ; dans le cas contraire, cliquez sur le bouton d’ajout (+) et sélectionnez Bibliothèque de podcasts.

    Une zone de dialogue apparaît.

  3. Sous « Publier les fichiers multimédias dans la bibliothèque de podcasts », sélectionnez le fichier multimédia à publier dans la bibliothèque de podcasts :

    • Sources d’origine : sélectionnez cette option pour archiver les fichiers source d’origine ayant servi à créer le podcast.

    • Fichiers d’aperçu : sélectionnez cette option pour archiver les fichiers d’aperçu générés par Podcast Composer.

    • Modèle édité : sélectionnez cette option pour archiver le fichier de modèle édité à partir duquel le contenu encodé est généré. Il s’agit d’une vidéo brute de haute qualité dont vous pouvez avoir besoin ultérieurement si vous souhaitez ré-encoder le podcast dans un autre format.

    • Fichiers multimédias générés : sélectionnez cette option pour publier les fichiers multimédias générés par le processus.

  4. (Facultatif) Dans le champ Balises, ajoutez les balises, séparées par des virgules, afin de retrouver plus facilement le contenu.

    La bibliothèque de podcasts ajoute ces balises à tous les fichiers multimédias publiés. En outre, la bibliothèque de podcasts crée un flux pour chaque balise.

    Les balises sont un moyen approprié et facile de créer plusieurs flux pour les mêmes processus. Par exemple, vous pouvez si vous le souhaitez rendre les processus accessibles selon l’année de la classe et le nom du département.

  5. Cliquez sur Terminé.

  6. Choisissez Fichier > Enregistrer.
