Ajout de notifications par courrier électronique

Lors de l’étape de notification, vous pouvez ajouter une ou plusieurs notifications par courrier électronique.

Pour ajouter une notification par courrier électronique, procédez comme suit :

  1. Dans Podcast Composer, accédez à l’étape de notification (choisissez Étapes > Notifier, ou appuyez sur Commande + 6).

  2. Cliquez sur le bouton d’ajout (+) et sélectionnez Courrier électronique.

    Une zone de dialogue s’affiche.

  3. Sélectionnez les destinataires du courrier :

    • Administrateur Podcast Producer : sélectionnez cette option pour envoyer une notification par courrier électronique à l’administrateur du serveur Podcast Producer.

    • Utilisateur à l’origine de la soumission : sélectionnez cette option pour envoyer une notification par courrier électronique à l’utilisateur qui a soumis le processus.

    • Autres : si vous sélectionnez cette option, saisissez les adresses électroniques (séparées par des virgules) des destinataires de la notification.

  4. Dans le champ Adresse électronique, saisissez l’adresse de la personne qui envoie le courrier électronique.

  5. Dans le champ Adresse du serveur, saisissez l’adresse du serveur utilisé pour envoyer le courrier électronique.

  6. Si le courrier nécessite une protection SSL, sélectionnez l’option Utiliser SSL (Secure Socket Layer).

  7. Dans le menu Authentification, choisissez le type d’authentification.

  8. Dans les champs Nom d’utilisateur et Mot de passe, saisissez les références de l’administrateur auquel envoyer le courrier électronique.

  9. Cliquez sur Terminé.

  10. Sélectionnez Fichier > Enregistrer.
