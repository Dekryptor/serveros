Envoi de contenu vers un serveur wiki Apple

L’une des façons de publier vos podcasts consiste à les mettre sur un blog Apple. Vous pouvez ajouter des destinations de blogs Apple.

Pour publier un podcast sur un wiki Apple :

  1. Dans Podcast Composer, accédez à l’étape de publication (choisissez Étapes > Publier ou appuyez sur Commande + 5).

  2. Cliquez sur le bouton d’ajout (+) et sélectionnez Wiki Apple.

    Une zone de dialogue apparaît.

  3. Dans le menu local, sélectionnez le format de podcast pour la publication sur le blog.

  4. Dans le champ Serveur, saisissez l’adresse du blog.

  5. Si le wiki nécessite une protection de type SSL (Secure Socket Layer), sélectionnez « Utiliser SSL (Secure Socket Layer) ».

  6. Dans les champs Nom d’utilisateur et Mot de passe, saisissez les références de l’administrateur pour accéder au blog.

  7. Pour spécifier la destination du podcast sur le blog, sélectionnez l’une des options suivantes :

    • Soumission du blog de l’utilisateur : le blog de l’utilisateur qui a soumis le podcast.

    • Blog d’utilisateur personnalisé : le blog de l’utilisateur spécifié dans le champ situé à droite.

    • Blog de groupe personnalisé : le blog du groupe spécifié dans le champ situé à droite.

  8. Cliquez sur Terminé.

  9. Choisissez Fichier > Enregistrer.
