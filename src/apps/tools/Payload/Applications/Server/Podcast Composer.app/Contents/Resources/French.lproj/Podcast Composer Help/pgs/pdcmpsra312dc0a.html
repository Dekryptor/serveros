Configuration du montage

Si vous souhaitez que le processus crée un podcast à partir de documents, sélectionnez et configurez l’option Montage à l’étape de l’importation. Podcast Composer traite les pages de ces documents comme des images. Il combine toutes les pages de tous les documents en une seule présentation avec diapositives et génère une vidéo QuickTime de cette dernière.

Par exemple, si en entrée vous disposez d’un document PDF et d’un document Word, Podcast Composer traite chacune de leurs pages comme une image. Ensuite, Podcast Composer assemble ces images dans une seule présentation et utilise les réglages que vous spécifiez au niveau de l’option Montage pour générer une vidéo QuickTime d’un diaporama contenant ces images.

Si un document n’est pas en mode horizontal et que vous affichez une page en entier pour chaque diapositive, le texte risque d’être difficilement lisible, car la page ne peut pas occuper toute la largeur de la diapositive. Pour résoudre ce problème, Podcast Composer divise chaque page du document en deux parties. L’image de la partie supérieure d’une page apparaît sur une diapositive et la partie inférieure s’affiche sur la diapositive suivante. En outre, Podcast Composer ajoute une transition entre deux diapositives pour fournir un effet de défilement.

En entrée pour l’option Montage, vous pouvez utiliser des documents de type PDF, Word, PowerPoint, Pages, Keynote, image et vidéo, qui peuvent être rendus à l’aide de Coup d’œil.

Les documents sont concaténés et ajoutés à la vidéo selon leur ordre alphanumérique.

Pour configurer l’option Montage à l’étape de l’importation :

  1. Dans Podcast Composer, accédez à l’étape de l’importation (choisissez Étapes > Importer ou appuyez sur Commande + 2).

  2. Double-cliquez sur Montage ou positionnez la souris au-dessus de l’icône Montage et cliquez sur le bouton Informations dans le coin inférieur droit.

    La zone de dialogue qui s’affiche contient les réglages que vous pouvez configurer, ainsi qu’un aperçu de vidéo actif qui reflète vos choix.

  3. Dans le champ « Durée de l’image », saisissez le nombre de secondes pendant lequel chaque diapositive est affichée.

  4. Pour spécifier un effet de transition :

    • Dans le menu local Effet de transition, choisissez un effet.

    • Dans le menu local Direction, choisissez la direction de l’effet (le cas échéant).

    • Dans le champ Durée, spécifiez la durée de la transition en secondes.

  5. Si vous voulez que Podcast Composer détecte automatiquement les changements dans les vidéos d’entrée et utilise ces informations pour affecter des chapitres dans le podcast généré, sélectionnez « Génération automatique des chapitres ».

    Par exemple, l’utilisation de chapitres dans le podcast peut être souhaitable pour marquer chaque transition vers une nouvelle diapositive.

    Dans le cas des documents, chaque page est considérée comme un chapitre. Le processus utilise Coup d’œil pour extraire un titre à partir de ces pages afin d’affecter un titre significatif aux chapitres.

  6. Cliquez sur Terminé.

  7. Choisissez Fichier > Enregistrer.
