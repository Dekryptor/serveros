Ajout de notifications iChat

Lors de l’étape de notification, vous pouvez ajouter des notifications iChat.

Pour que les notifications iChat fonctionnent, vous devez avoir accès à un serveur iChat exécuté sur Mac OS X Server 10.6. Si ce n’est pas le cas, vous devez configurer votre propre serveur iChat. Toute notification envoyée via un serveur iChat exécuté sur Mac OS X Server 10.5 ou version antérieure échouera.

Néanmoins, si vous souhaitez envoyer les notifications via un serveur iChat exécuté sur la version 10.5, vous pouvez configurer un serveur iChat sur un ordinateur sur lequel se trouve la version 10.6 et le fédérer ensuite sur un serveur iChat exécuté sur un ordinateur sur lequel la version 10.5 est installée. Les notifications passent d’abord par le serveur iChat exécuté sur l’ordinateur sur lequel est installée la version 10.6, puis par le serveur iChat exécuté sur le serveur 10.5.

Le serveur iChat n’a pas besoin d’être exécuté sur le serveur Podcast Producer.

Pour en savoir plus sur la configuration d’un serveur iChat, reportez-vous au guide d’administration du service iChat.

Pour ajouter une notification iChat, procédez comme suit :

  1. Dans Podcast Composer, accédez à l’étape de notification (choisissez Étapes > Notifier, ou appuyez sur Commande + 6).

  2. Cliquez sur le bouton d’ajout (+) et sélectionnez iChat.

    Une zone de dialogue s’affiche.

  3. Sélectionnez les destinataires iChat :

    • Administrateur Podcast Producer : sélectionnez cette option pour envoyer un message de notification iChat à l’administrateur du serveur Podcast Producer.

    • Utilisateur à l’origine de la soumission : sélectionnez cette option pour envoyer un message de notification iChat à l’utilisateur qui a soumis le processus.

    • Autres : si vous sélectionnez cette option, saisissez les adresses iChat (séparées par des virgules) des destinataires de la notification.

  4. Dans les champs Identifiant Jabber et Mot de passe, saisissez les références de l’administrateur auquel envoyer le message iChat.

    Les notifications iChat fonctionnent uniquement avec les comptes Jabber.

  5. Dans le champ Message iChat, saisissez le texte du message de notification à envoyer via iChat.

    Vous pouvez ajouter à ce message des variables $$KEY$$, qui seront remplacées par Podcast Producer.

  6. Pour ajouter des informations de publication au message iChat, sélectionnez l’option « Afficher des informations supplémentaires ».

  7. Cliquez sur Terminé.

  8. Sélectionnez Fichier > Enregistrer.
