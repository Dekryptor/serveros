Ajout d’un filigrane et d’une superposition de l’introduction

Vous pouvez utiliser l’étape de l’édition pour ajouter un filigrane et une superposition de l’introduction à la vidéo d’entrée.

Un filigrane peut être une image, une vidéo ou une composition Quartz.

Une superposition de l’introduction est un titre qui apparaît au-dessus de la vidéo, à son début, pendant un laps de temps défini.

Pour ajouter un filigrane à la vidéo d’entrée :

  1. Dans Podcast Composer, accédez à l’étape de l’édition (choisissez Étapes > Édition ou appuyez sur Commande + 3).

  2. Double-cliquez sur l’icône de la vidéo d’entrée.

  3. Dans le menu local Filigrane, choisissez un style de filigrane par défaut.

    Pour utiliser un filigrane personnalisé, sélectionnez l’option Choisir, puis un filigrane. Vous pouvez aussi faire glisser le filigrane vers l’aperçu de la vidéo.

    Vous pouvez également faire glisser un filigrane sur l’icône de filigrane et de superposition de l’introduction sans ouvrir le menu local.

  4. Pour spécifier l’opacité du filigrane, déplacez le curseur Opacité vers la gauche (plus transparent) ou vers la droite (moins transparent).

  5. Pour spécifier la taille du filigrane, déplacez le curseur Taille vers la gauche (plus petit) ou vers la droite (plus grand).

  6. Pour spécifier la position du filigrane, sélectionnez l’option dans le menu local Position.

  7. Pour ajouter une superposition de l’introduction sur la vidéo d’entrée et sous le filigrane, choisissez un style de superposition dans le menu local « Superposition de l’introduction ».

    Sélectionnez les informations à afficher dans la superposition en cochant les cases. Selon le style de superposition que vous choisissez, il est possible que certaines cases soient décochées.

    Vous pouvez également faire glisser une vidéo sur l’icône de filigrane et de superposition de l’introduction sans ouvrir le menu local.

  8. Cliquez sur Terminé.

  9. Choisissez Fichier > Enregistrer.
