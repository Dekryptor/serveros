Envoi de contenu vers le dossier surveillé de Final Cut Server

Lors de l’étape de publication dans Podcast Composer, vous pouvez choisir d’envoyer le contenu vers le dossier surveillé de Final Cut Server en vue d’un traitement additionnel.

Vous pouvez ajouter des destinations Final Cut Server.

Pour envoyer du contenu vers Final Cut Server :

  1. Dans Podcast Composer, accédez à l’étape de publication (choisissez Étapes > Publier ou appuyez sur Commande + 5).

  2. Cliquez sur le bouton d’ajout (+) et sélectionnez Final Cut Server.

    Une zone de dialogue apparaît.

  3. Dans le menu local, sélectionnez le format de podcast pour l’envoi vers Final Cut Server.

  4. Dans le champ « Chemin d’accès au dossier surveillé par Final Cut Server », saisissez le chemin vers le dossier surveillé.

  5. Cliquez sur Terminé.

  6. Choisissez Fichier > Enregistrer.
