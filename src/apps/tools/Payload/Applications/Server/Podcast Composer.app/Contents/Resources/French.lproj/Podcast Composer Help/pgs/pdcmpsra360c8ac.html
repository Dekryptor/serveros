Déploiement d’un processus

Une fois achevée la création d’un processus, vous pouvez utiliser Podcast Composer pour télécharger le paquet de processus vers un serveur Podcast Producer. Podcast Producer rend le processus accessible aux clients de l’application de transfert de podcast, comme défini par l’administrateur Podcast Producer.

Pour déployer un processus sur un serveur Podcast Producer, procédez comme suit :

  1. Dans Podcast Composer, sélectionnez Fichier > Déploiement vers le serveur (Commande + U) ou appuyez sur le bouton Déployer qui apparaît à l’étape du résumé.

    Si votre processus est incomplet, une zone de dialogue s’affiche avec la description du problème au niveau du premier réglage incomplet du processus. Une flèche rouge pointant vers l’élément dont le réglage est incomplet s’affiche dans Podcast Composer.

  2. Si une zone de dialogue s’affiche indiquant que le processus est incomplet, cliquez sur le bouton OK pour la fermer, résolvez le problème, puis essayez à nouveau de déployer le processus.

    Il se peut que vous ayez à répéter plusieurs fois l’opération pour que le processus passe avec succès l’étape de vérification.

  3. Dans le champ Serveur de la zone de dialogue Déploiement, saisissez l’adresse du serveur Podcast Producer ou sélectionnez un serveur dans le menu local.

  4. Cliquez sur Déployer.

  5. Dans les champs Nom et Mot de passe, saissisez le nom d’utilisateur et le mot de passe d’administrateur qui vous ont été attribués par l’administrateur Podcast Producer.

  6. Cliquez à nouveau sur Déployer.

    Une zone de dialogue s’affiche, indiquant que le téléchargement du processus vers le serveur Podcast Producer a abouti.
