Ajout de notifications de répertoires de podcasts iTunes

Lors de l’étape de notification, vous ne pouvez ajouter qu’une seule notification de répertoire de podcasts iTunes. Pour que cette notification fonctionne, vous devez d’abord enregistrer vos flux dans iTunes. Lorsque cette notification est envoyée à iTunes, le répertoire de podcasts iTunes se synchronise dès que possible sur votre bibliothèque de podcasts afin de répercuter toutes les modifications. Il s’agit d’une action asynchrone.

Pour en savoir plus sur la bibliothèque de podcasts, reportez-vous au guide d’administration de Podcast Producer.

Pour ajouter une notification de répertoire de podcasts iTunes, procédez comme suit :

  1. Dans Podcast Composer, accédez à l’étape de notification (choisissez Étapes > Notifier, ou appuyez sur Commande + 6).

  2. Cliquez sur le bouton d’ajout (+) et sélectionnez Répertoire des podcasts iTunes.

  3. Sélectionnez Fichier > Enregistrer.
