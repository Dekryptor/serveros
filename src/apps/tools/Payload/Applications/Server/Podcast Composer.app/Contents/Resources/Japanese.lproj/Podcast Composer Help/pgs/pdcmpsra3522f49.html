Adding Email Notifications

In the Notify stage, you can add one or more email notifications.

To add an Email notification:

  1. In Podcast Composer, go to the Notify stage (choose Stages > Notify, or press Command-6).

  2. Click the Add (+) button and choose Email.

    A dialog appears.

  3. Select the mail recipients:

    • Podcast Producer Administrator—Select this option to send an email notification to the administrator of the Podcast Producer server.

    • Submitting User—Select this option to send an email notification to the user who submitted the workflow.

    • Others—Select this option and enter the email addresses (separated by commas) of the notification recipients.

  4. In the Email Address field, enter the address of the person sending the email.

  5. In the Server Address field, enter the address of the server used for sending the email.

  6. If the email requires Secure Socket Layer protection, select Use Secure Socket Layer (SSL).

  7. From the Authentication menu, choose the authentication type.

  8. In the User Name and Password fields, enter the administrator credentials to send the email.

  9. Click Done.

  10. Choose File > Save.
