Repérage des fichiers de configuration ODBC

Les fichiers de configuration ODBC du système se trouvent dans le dossier racine /Library/ODBC/. Les fichiers de configuration de l'utilisateur se trouvent dans le dossier de départ Library/ODBC/.

Les fichiers odbc.ini contiennent des informations relatives aux données sources (DSN), tandis que les fichiers odbcinst.ini concernent les gestionnaires.
