Das Programm "ODBC-Administrator"

Mit dem Programm "ODBC-Administrator" können Sie Datenbanken mit Programmen verbinden, die dem ODBC-Standard (Open Database Connectivity) entsprechen. ODBC ist ein offener Standard für Programmierschnittstellen (API), über die Sie von einem einzelnen Programm aus auf mehrere Datenbanken zugreifen können.

Das Programm "ODBC-Administrator" befindet sich im Ordner "Dienstprogramme" innerhalb des Ordners "Programme". Das Programm wurde dafür konzipiert, die Datenbankanbindung für einen Computer zu konfigurieren. Damit Sie den ODBC-Administrator verwenden können, benötigen Sie:

■

ein ODBC-konformes Programm

■

eine ODBC-konforme Datenbank

■

einen ODBC-Treiber für diese Datenbank

Für ODBC-Verbindungen benötigen Sie ein Programm für die Verwaltung von Treibern oder für die Konfiguration (wie zum Beispiel den ODBC-Administrator), einen Treiber für die Datenbank oder das Datenbankverwaltungssystem (DBMS) sowie eine Datenbank.

Weitere Informationen über ODBC sowie Links zu Treibern anderer Hersteller und Anweisungen für die Konfiguration der Datenbank-Anbindung finden Sie im Internet unter: iODBC.org.
