ODBC アドミニストレータについて

「ODBC アドミニストレータ」を使って、ODBC（Open Database Connectivity）規格に準拠したアプリケーションからデータベースに接続できます。ODBC は、オープンスタンダードの API（アプリケーション・プログラミング・インターフェイス）の 1 つで、1 つのアプリケーションから複数のデータベースにアクセスするときに使います。

「アプリケーション」フォルダの「ユーティリティ」フォルダにある「ODBC アドミニストレータ」を使って、コンピュータのデータベース接続を設定できます。「ODBC アドミニストレータ」を使うときには、次のものが必要です：

■

ODBC 対応アプリケーション

■

ODBC 対応データベース

■

特定のデータベースの ODBC ドライバ

ODBC 接続には、アプリケーション、ドライバマネージャまたは構成ツール（「ODBC アドミニストレータ」など）、データベースまたはデータベース管理システム（DBMS）用のドライバ、およびデータベースが必要です。

データベース接続の設定に必要な他社製のドライバや説明書の入手方法など、ODBC について詳しくは、iODBC.org にアクセスしてください。
