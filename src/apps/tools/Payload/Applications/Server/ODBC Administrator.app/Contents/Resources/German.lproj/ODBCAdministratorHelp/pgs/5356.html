Konfigurieren von Treibern für die Datenbank-Anbindung

ODBC-Treiber (Open Database Connectivity) sammeln mit SQL (Structured Query Language) Daten, die in einer Datenbank gespeichert sind, und übertragen sie anschließend an ein Client-Programm. Mithilfe des Programms "ODBC-Administrator", das sich im Ordner "Dienstprogramme" innerhalb des Ordners "Programme" befindet, können Sie installierte Treiber konfigurieren.

Gehen Sie wie folgt vor, um installierte ODBC-Treiber zu konfigurieren:

Öffnen Sie den ODBC-Administrator und klicken Sie dann auf den Titel "Treiber".

Klicken Sie auf "Hinzufügen".

Geben Sie in das Feld "Beschreibung" einen Namen für den Treiber ein. Wählen Sie einen Namen aus, durch den sich der zugehörige Treiber leicht identifizieren lässt.

Klicken Sie neben dem Feld "Treiberdatei" auf "Auswählen" und wählen Sie einen Treiber aus.

Wenn Sie eine Konfigurationsdatei angeben müssen, klicken Sie neben dem Feld "Konfigurationsdatei" auf "Auswählen" und wählen Sie die Datei aus.

Wählen Sie "Benutzer" oder "System" aus dem Einblendmenü aus. So legen Sie fest, ob die betreffende Treiberkonfiguration nur für den Benutzer angezeigt wird, der sich gerade bei Mac OS X angemeldet hat, oder allen Benutzern des Computers zur Verfügung steht.

Klicken Sie auf die Taste "Hinzufügen", um Platzhalter in den Spalten "Schlagwort" und "Wert" zu erstellen.

Wählen Sie die Platzhalter durch Doppelklicken aus, um Ihre eigenen Schlagwörter und Werte einzugeben.

Klicken Sie auf "OK".

Klicken Sie auf "Anwenden", um eventuelle Änderungen vor dem Schließen des Programms "ODBC-Adminstrator" zu sichern.

Wenn die Einstellungen grau dargestellt werden und ein geschlossenes Vorhängeschloss angezeigt wird, klicken Sie auf dieses Symbol. Geben Sie den Namen und das Kennwort eines Administrators ein.

Damit Sie zu einem späteren Zeitpunkt Änderungen an der Treiberkonfiguration vornehmen können, klicken Sie im Bereich "Treiber" auf den betreffenden Treiber und dann auf die Taste "Konfigurieren".
