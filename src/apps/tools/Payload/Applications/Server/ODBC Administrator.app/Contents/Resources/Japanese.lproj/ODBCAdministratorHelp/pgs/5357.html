ODBC 接続のパラメータを設定する

特定のデータベース用に ODBC ドライバを構成したら、クエリーの種類に応じてパラメータ（データソースと呼ばれます）を設定できます。たとえば、照会先のデータベースや取得する情報の種類を指定できます。

これらのデータソースは、「アプリケーション」フォルダの「ユーティリティ」フォルダにある「ODBC アドミニストレータ」を使って設定できます。

データソースを設定するには：

「ODBC アドミニストレータ」を開きます。

このコンピュータのすべてのユーザがこのデータソースを利用できるようにする場合は、「システム DSN」をクリックします。このコンピュータに現在ログインしているユーザだけがこのデータソースを利用できるようにする場合は、「ユーザ DSN」をクリックします。

「追加」ボタンをクリックし、構成済みのドライバのリストからドライバを選んで、「OK」をクリックします。

表示されたウインドウで、このデータソースの名前と説明を入力します。

選択したドライバに独自のデータソース構成パネルがある場合は、パネルに表示される指示に従ってください。

「追加」ボタンをクリックして、「キー」および「値」列に ODBC ドライバ用のキーと値を入力する領域を作成します。

作成した領域をダブルクリックして、データソース用のキーと値を入力し、「OK」をクリックします。

ウインドウを閉じる前に変更を保存するには、「適用」をクリックします。

設定が淡色で表示され、ロックされたカギのアイコンが表示されている場合は、アイコンをクリックし、管理者ユーザの名前とパスワードを入力します。
