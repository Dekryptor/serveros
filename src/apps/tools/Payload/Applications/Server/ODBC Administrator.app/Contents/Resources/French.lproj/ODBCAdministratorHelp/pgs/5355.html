À propos d'Administrateur ODBC

Avec Administrateur ODBC, vous pouvez établir des connexions entre des bases de données et des applications conformes à la norme ODBC (Open Database Connectivity). Open Database Connectivity (ODBC) est une interface de programmation d'application standard ouverte (API) permettant l'accès à de multiples bases de données à partir d'une seule application.

L'application Administrateur ODBC, située dans le dossier /Applications/Utilitaires, vous permet de configurer la connexion à la base de données sur un ordinateur. Pour l'utiliser, il vous faut :

■

une application conforme à ODBC ;

■

une base de données conforme à ODBC ;

■

un « gestionnaire » ODBC pour cette base de données particulière.

Les connexions ODBC impliquent une application, un pilote de gestion ou un outil de configuration (comme Administrateur ODBC), un gestionnaire pour la base de données ou un système de gestion de base de données (DBMS), ainsi qu'une base de données.

Pour obtenir plus d'information sur l'ODBC, y compris des liens vers des gestionnaires tiers et des instructions sur la configuration de la connexion à la base de données, visitez le site iODBC.org.
