Suchen von ODBC-Konfigurationsdateien

Systemweite ODBC-Konfigurationsdateien werden auf der obersten Ebene des Ordners "/Library/ODBC/" gespeichert. Benutzerspezifische ODBC-Konfigurationsdateien finden Sie in Ihrem Benutzerordner "/Library/ODBC".

Dateien mit dem Namen "odbc.ini" enthalten Informationen über Datenquellen (DSN), während Dateien mit dem Namen "odbcinst.ini" Informationen über die Treiber enthalten.
