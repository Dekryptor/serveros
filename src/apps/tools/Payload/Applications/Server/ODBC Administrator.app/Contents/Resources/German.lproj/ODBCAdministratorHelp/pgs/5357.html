Konfigurieren von Parametern für eine ODBC-Verbindung

Nachdem Sie ODBC-Treiber für eine bestimmte Datenbank konfiguriert haben, können Sie die Parameter (auch "Datenquellen" genannt) für unterschiedliche Typen von Abfragen konfigurieren. Sie können beispielsweise festlegen, welche Datenbank abgefragt werden soll und welche Informationstypen dort abgerufen werden.

Mithilfe des Programms "ODBC-Administrator", das sich im Ordner "Dienstprogramme" innerhalb des Ordners "Programme" befindet, können Sie diese Datenquellen konfigurieren.

Gehen Sie wie folgt vor, um eine Datenquelle zu konfigurieren:

Öffnen Sie den ODBC-Administrator.

Klicken Sie auf "System-DSN", wenn diese Datenquelle für alle Benutzer des Computers verfügbar sein soll. Mit der Option "Benutzer-DSN" können Sie festlegen, dass ausschließlich der Benutzer auf die Datenquelle zugreifen darf, der gerade am Computer angemeldet ist.

Klicken Sie auf die Taste "Hinzufügen". Wählen Sie einen Treiber aus der Liste der konfigurierten Treiber aus und klicken Sie dann auf "OK".

Geben Sie in das nächste Fenster einen Namen und eine Beschreibung für diese Datenquelle ein.

Falls der gewählte Treiber einen eigenen Konfigurationsbereich für Datenquellen besitzt, befolgen Sie die Anweisungen in diesem Bereich.

Klicken Sie auf die Taste "Hinzufügen" und erstellen Sie Platzhalter in den Spalten "Schlagwort" und "Wert".

Wählen Sie die Platzhalter durch Doppelklicken aus. Geben Sie Ihre eigenen Schlagwörter und Werte ein und klicken Sie anschließend auf "OK".

Klicken Sie auf "Anwenden", um Änderungen vor dem Schließen des Fensters zu sichern.

Wenn die Einstellungen grau dargestellt werden und ein geschlossenes Vorhängeschloss angezeigt wird, klicken Sie auf dieses Symbol. Geben Sie den Namen und das Kennwort eines Administrators ein.
