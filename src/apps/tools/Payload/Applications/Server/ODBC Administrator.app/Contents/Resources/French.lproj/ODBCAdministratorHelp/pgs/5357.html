Configuration des paramètres pour une connexion ODBC

Après avoir configuré les gestionnaires ODBC pour une base de données spécifique, vous pouvez configurer les paramètres (appelés données sources) de différents types de demandes. Vous pouvez par exemple spécifier la base de données que vous souhaitez interroger et les types d'informations à récupérer.

L'application Administrateur ODBC, située dans le dossier /Applications/Utilitaires, vous permet de configurer ces données source.

Pour configurer des données source :

Ouvrir Administrateur ODBC.

Cliquez sur « DNS Système » si vous souhaitez rendre ces données source disponibles à tous les utilisateurs de cet ordinateur. Cliquez sur « DNS Utilisat. » pour rendre ces données source uniquement disponibles à l'utilisateur actuellement connecté à cet ordinateur.

Cliquez sur le bouton Ajouter et choisissez un gestionnaire dans la liste des gestionnaires configurés, puis cliquez sur OK.

Dans la fenêtre qui s'affiche, tapez un nom et une description pour ces données source.

Si le gestionnaire que vous avez sélectionné possède son propre panneau de configuration de données source, suivez les instructions figurant dans celui-ci.

Cliquez sur le bouton Ajouter pour créer des marques de réservation dans les colonnes Mot de passe et Valeur.

Double-cliquez sur les paramètres fictifs pour saisir vos propres mots de passe et valeurs, puis cliquez sur OK.

Pour enregistrer toutes les modifications avant de fermer la fenêtre, cliquez sur Appliquer.

Si les réglages sont estompés et l'icône de cadenas verrouillée, cliquez sur l'icône puis tapez les nom et mot de passe d'un utilisateur administrateur.
