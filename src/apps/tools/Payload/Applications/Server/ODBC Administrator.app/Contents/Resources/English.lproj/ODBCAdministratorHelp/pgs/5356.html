Configuring drivers for database connectivity

Open Database Connectivity (ODBC) drivers use Structured Query Language (SQL) to gather data stored in a database and then deliver the data to a client application. You can use ODBC Administrator, located in the Utilities folder of the Applications folder, to configure any installed drivers.

To configure installed ODBC drivers:

Open ODBC Administrator, and then click the Drivers tab.

Click Add.

In the Description box, type a name for the driver that will easily identify it.

Click Choose next to the “Driver file” field, and select a driver.

If you need to specify a set up file, click Choose next to the “Setup file” field and select it.

Choose User or System from the pop-up menu to designate whether this driver configuration is visible only to the user currently logged in to Mac OS X, or is visible to all users of the computer.

To create placeholders in the Keyword and Value columns, click the Add button.

Double-click the placeholders to type your own keys and values.

Click OK.

To save any changes before quitting ODBC Administrator, click Apply.

If the settings are dimmed and the lock icon is locked, click the icon and type the name and password of an administrator user.

To make changes to the driver configuration later, click the driver in the Drivers pane and then click Configure.
