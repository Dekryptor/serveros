データベース接続のドライバを構成する

ODBC（Open Database Connectivity）ドライバは、SQL（Structured Query Language）を使って、データベースに格納されたデータを取得し、クライアントアプリケーションに送信します。「アプリケーション」フォルダの「ユーティリティ」フォルダにある「ODBC アドミニストレータ」を使って、インストール済みのドライバを構成できます。

インストール済みの ODBC ドライバを構成するには：

「ODBC アドミニストレータ」を開き、「ドライバ」タブをクリックします。

「追加」をクリックします。

「説明」ボックスに、ドライバを簡単に識別できる名前を入力します。

「ドライバファイル」フィールドの横にある「選択」をクリックして、ドライバを選択します。

設定ファイルを指定する必要がある場合は、「設定ファイル」の横にある「選択」をクリックして選択します。

ポップアップメニューから「ユーザ」または「システム」を選んで、このドライバの構成の表示方法（Mac OS X に現在ログインしているユーザだけに表示するか、コンピュータのすべてのユーザに表示するか）を指定します。

「キーワード」および「値」列に ODBC ドライバ用のキーと値を入力する領域を作成するには、「追加」ボタンをクリックします。

作成した領域をダブルクリックして、データソース用のキーと値を入力します。

「OK」をクリックします。

「ODBC アドミニストレータ」を終了する前に変更を保存するには、「適用」をクリックします。

設定が淡色で表示され、ロックされたカギのアイコンが表示されている場合は、アイコンをクリックし、管理者ユーザの名前とパスワードを入力します。

ドライバ構成を後で変更するときは、「ドライバ」パネルで目的のドライバをクリックし、「構成」をクリックします。
