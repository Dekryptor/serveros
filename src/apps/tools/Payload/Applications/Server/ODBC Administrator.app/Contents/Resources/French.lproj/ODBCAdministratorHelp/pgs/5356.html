Configuration des gestionnaires pour la connexion à une base de données

Les gestionnaires Open Database Connectivity (ODBC) utilisent le langage SQL (Structured Query Language) pour collecter les données stockées dans une base de données et les transmettre ensuite à une application client. L'application Administrateur ODBC, située dans le dossier /Applications/Utilitaires, vous permet de configurer tous les gestionnaires installés.

Pour configurer les gestionnaires ODBC installés :

Ouvrez Administrateur ODBC, cliquez sur l'onglet Gestionnaires.

Cliquez sur Ajouter.

Dans la zone Description, tapez un nom de gestionnaire afin de pouvoir l'identifier aisément.

Cliquez sur Sélectionner à côté du champ « Fichier gestionnaire », puis sélectionner un gestionnaire.

Si vous devez spécifier un fichier de configuration, cliquez sur Sélectionner à côté du champ « Fichier de configuration » et sélectionnez-le.

Choisissez Utilisateur ou Système dans le menu local pour spécifier si la configuration de ce gestionnaire doit être visible uniquement pour l'utilisateur actuellement connecté sur Mac OS X, ou si elle apparaît pour tous les utilisateurs de cet ordinateur.

Cliquez sur le bouton Ajouter pour créer des espaces réservés dans les colonnes Clé et Valeur.

Double-cliquez sur les espaces réservés pour saisir vos propres clés et valeurs, puis cliquez sur OK.

Cliquez sur OK.

Pour enregistrer toutes les modifications avant de quitter Administrateur ODBC, cliquez sur Appliquer.

Si les réglages sont estompés et l'icône de cadenas verrouillée, cliquez sur l'icône puis tapez les nom et mot de passe d'un utilisateur administrateur.

Pour modifier ultérieurement la configuration du gestionnaire, cliquez dessus dans la fenêtre Gestionnaires puis sur Configurer.
