Aufträge

Ein Auftrag ist eine Sammlung von Ausführungsanweisungen, die Daten und ausführbare Dateien enthalten kann. Xgrid kann Skripts, Dienstprogramme und angepasste Software ausführen (generell alle Komponenten, für die keine Benutzerinteraktion erforderlich ist).

Ein Client sendet einen Auftrag an das Grid. Der Controller akzeptiert den Auftrag und die dazugehörigen Dateien, unterteilt den Auftrag in Aufgaben und verteilt die Aufgaben anschließend an die Agenten. Die Agenten akzeptieren die Aufgaben, führen die Berechnungen aus und senden die Ergebnisse zurück an den Controller. Dort werden sie gesammelt und an die Clients zurückgegeben.
