Administration des agents Xgrid

Utilisez Xgrid Admin pour visualiser, ajouter ou supprimer des agents. Xgrid Admin utilise également des indicateurs d'état pour afficher l'état des agents. Pour en savoir plus, consultez les éléments suivants :

  • Visualisation d'une liste d'agents

  • Ajout d'un agent

  • Suppression d'un agent

  • Indicateurs d'état de Xgrid Admin

Bien qu’Admin Serveur fournisse une interface simple pour activer les services Xgrid sur un serveur ou sur un rack de systèmes Xserve, il n'offre pas le moyen de configurer Xgrid sur des ordinateurs de bureau qui exécutent Mac OS X v10.3 ou ultérieur.

Si vous comptez sur des volontaires pour fournir des agents d'ordinateurs de bureau, vous pouvez envoyer les instructions d'activation d'Xgrid depuis la sous-fenêtre Partage des Préférences Système.

Si les volontaires utilisent Mac OS X v10.3, vous devez préalablement télécharger l'agent Xgrid pour Mac OS X v10.3 et utiliser la sous-fenêtre Xgrid des Préférences Système. Vous pouvez télécharger l'agent Xgrid pour Mac OS X v10.3 à partir de l'adresse www.apple.com/fr/server/macosx/technology/xgrid.html

Si vous administrez un groupe d'ordinateurs et que vous souhaitez qu'ils participent à une grille à l'aide d'Xgrid, vous pouvez utiliser l'une des méthodes suivantes :

  • Apple Remote Desktop

  • SSH

  • NetBoot ou NetInstall
