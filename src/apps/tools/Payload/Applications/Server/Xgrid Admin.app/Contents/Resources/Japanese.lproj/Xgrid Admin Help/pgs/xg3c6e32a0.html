Xgrid Admin の概要

「Xgrid Admin」は、グリッドを監視し、エージェントとジョブを管理するためのツールです。コントローラやエージェントをモニタに追加し、グリッドに参加していないエージェントを指定します。「Xgrid Admin」を使用して、ジョブを一時停止、停止、または再起動することもできます。

「Xgrid Admin」を使って計算グリッドを管理できます。計算グリッドは、専用のキューを持つエージェントの固定したグループです。

1 つのコントローラに複数のグリッドを割り当てることができますが、1 つのエージェントが所属できるのは 1 つのグリッドだけです。ジョブ（またはタスク）の実行中は、グリッド間でエージェントを移動することはできません。

詳しくは、「Xgrid Admin を使用する」を参照してください。
