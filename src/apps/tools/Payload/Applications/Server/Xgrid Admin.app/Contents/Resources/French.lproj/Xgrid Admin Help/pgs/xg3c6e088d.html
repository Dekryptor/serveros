Suppression d'un agent

Vous pouvez supprimer l'agent d'un contrôleur dans Xgrid Admin.

Pour supprimer un agent :

  1. Ouvrez Xgrid Admin.

  2. Dans la liste Contrôleurs et grilles, sélectionnez le contrôleur.

  3. Cliquez sur Agents.

  4. Cliquez sur le bouton Supprimer (–) sous la liste des agents.

    Remarque :  si vous supprimez un agent que vous savez se trouver sur le sous-réseau local et qui est configuré pour être associé à ce contrôleur, attendez un moment et il réapparaîtra dans la liste. Si l'agent ne réapparaît pas, utilisez le bouton Ajouter (+) puis saisissez son nom pour le récupérer.
