Verwalten von Xgrid-Aufträgen

Mithilfe des Programms "Xgrid-Admin" verwalten Sie Aufträge, nachdem diese von einem Client übertragen wurden.

Hinweis:  Das Verlagern von Aufträgen zwischen Grids ist nicht möglich.

  • Anzeigen einer Auftragsliste

  • Stoppen eines Auftrags

  • Wiederholen oder Neustarten eines Auftrags

  • Löschen eines Auftrags

  • Statusanzeigen im Programm "Xgrid-Admin"
