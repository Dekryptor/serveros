Types courants de grilles et de styles de grilles de calcul

Xgrid peut aussi bien être utilisé au sein de grappes de serveurs étroitement interconnectés que dans des grilles mondiales, et tout ce qui se trouve entre ces deux options. Cette extraordinaire flexibilité vous permet de déployer des grilles de pratiquement n'importe quelle nature. Trois topologies principales sont couramment utilisées lors des déploiements d'Xgrid, présentées comme suit :

  • Grappes de serveurs Xgrid

  • Grilles locales

  • Grilles réparties
