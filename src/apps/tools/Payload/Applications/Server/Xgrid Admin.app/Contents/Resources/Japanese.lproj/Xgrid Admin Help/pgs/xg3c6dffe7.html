Xgrid Admin を使用する

「Xgrid Admin」は、グリッドを監視し、エージェントとジョブを管理するためのツールです。

「Xgrid Admin」では、次のことができます：

  • 動作中のエージェントと空きエージェントの数、使用中の処理能力と空き処理能力、実行中のジョブと保留中のジョブの数など、グリッドとその動作の状況を確認します

  • 管理するコントローラとグリッドを追加または削除します

  • グリッド内のエージェントのリストと、各エージェントの空き CPU 能力と使用中の CPU 能力を表示します

  • グリッド内のエージェントを追加または削除します

  • グリッド内のジョブのリスト、各ジョブの送信日時、ジョブの進行状況、およびジョブの動作中の CPU 能力を表示します

  • グリッド内のジョブを削除します

  • 進行中のジョブを停止します

  • 停止中のジョブまたは完了したジョブを再起動します

「Xgrid Admin」では、すべてのオプションを視覚的なインターフェイスとメニューコマンドで制御できます。

これらのタスクは、Xgrid コマンドラインツールを使用して実行することもできます。
