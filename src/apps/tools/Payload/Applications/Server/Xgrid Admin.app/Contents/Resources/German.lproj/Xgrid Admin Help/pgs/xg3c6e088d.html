Löschen eines Agenten

Sie können in Xgrid-Admin einen Agenten für einen Xgrid-Controller löschen.

Gehen Sie wie folgt vor, um einen Agenten zu löschen:

  1. Öffnen Sie das Programm "Xgrid-Admin".

  2. Wählen Sie den Controller in der Liste "Controller und Grids" aus.

  3. Klicken Sie auf "Agenten".

  4. Klicken Sie auf die Taste "Löschen" (-) unter der Liste der Agenten.

    Hinweis:  Wenn Sie versehentlich einen Agenten löschen, der sich im lokalen Teilnetzwerk befindet und der für eine Verbindung mit diesem Controller konfiguriert ist, warten Sie einen Moment. Der Agent sollte dann wieder in der Liste angezeigt werden. Wird der Agent nicht wieder angezeigt, können Sie ihn wiederherstellen. Klicken Sie dazu auf die Taste "Hinzufügen" (+) und geben Sie den Namen des Agenten ein.
