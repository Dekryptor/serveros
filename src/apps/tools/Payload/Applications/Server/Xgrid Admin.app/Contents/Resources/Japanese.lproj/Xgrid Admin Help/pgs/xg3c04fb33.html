コントローラ

Xgrid コントローラは、グリッドの計算用のリソース間の通信を管理します。コントローラには、Mac OS X Server v10.4 以降が必要です。コントローラは、クライアントとエージェントからのネットワーク接続を受け入れます。クライアントからのジョブ送信を受信し、ジョブをタスクに分割し、タスクをエージェントに送り、結果をクライアントに返します。

サブネットで複数の Xgrid コントローラを実行することはできますが、論理グリッドごとのコントローラは 1 つだけです。

各コントローラには任意の数のエージェントを接続できますが、アップルでテスト済みのエージェント数はコントローラ 1 つあたり 128 です。ただし、エージェント数に対するソフトウェアの制限はありません。また、Xgrid のユーザは、コントローラシステムで利用可能なソケット数を理論上の上限として、自己の責任でコントローラ 1 つあたり 128 を超えるエージェントを接続できます。
