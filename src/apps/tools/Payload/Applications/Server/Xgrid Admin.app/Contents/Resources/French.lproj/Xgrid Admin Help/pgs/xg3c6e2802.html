Répétition ou redémarrage d'un travail

Vous pouvez répéter une tâche ou redémarrer une tâche arrêtée dans Xgrid Admin.

Pour répéter ou redémarrer une tâche :

  1. Ouvrez Xgrid Admin.

  2. Dans la liste Contrôleurs et grilles, sélectionnez le contrôleur.

  3. Cliquez sur Tâches.

  4. Sélectionnez le travail à répéter ou redémarrer.

  5. Cliquez sur Démarrer en dessous de la liste des tâches.
