Tâches

Une tâche est une collection d'instructions d'exécution qui inclut les données et les fichiers exécutables. Xgrid peut exécuter des scripts, des utilitaires et des logiciels personnalisés (tout ce qui ne nécessite aucune intervention de l'utilisateur).

Un client soumet une tâche à la grille. Le contrôleur accepte la tâche et ses fichiers associés, divise la tâche en sous-tâches, puis répartit celles-ci entre les différents agents. Les agents acceptent les sous-tâches, effectuent les calculs puis retournent le résultat au contrôleur, qui les regroupe avant de les renvoyer aux clients.
