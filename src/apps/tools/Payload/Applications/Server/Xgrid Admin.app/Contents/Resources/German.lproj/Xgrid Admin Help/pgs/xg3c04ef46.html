Xgrid und Computer-Grids

Xgrid macht es leicht, eine Gruppe von Mac-Systemen in einen kostengünstigen Supercomputer zu verwandeln. Xgrid ist eine ideale Lösung für einzelne Forscher, spezialisierte Projektmitarbeiter und Programmentwickler. Hier ein Beispiel:

  • Wissenschaftler können Biologiedatenbanken in einem Cluster aus Xserve-Systemen durchsuchen.

  • Ingenieure können finite Elementanalysen auf den Desktop-Computern ihrer Arbeitsgruppe ausführen.

  • Autoren von Animationen können Bilder standortübergreifend auf Mac-Systemen rendern.

  • Forschungsgruppen können Kollegen und interessierte Laien zur Mitarbeit in Internet-weiten Freiwilligen-Grids gewinnen, um wissenschaftliche Langzeitberechnungen durchzuführen.

  • Jeder Benutzer, der prozessorintensive Berechnungen durchzuführen hat, kann einen einzelnen Auftrag gleichzeitig auf mehreren Computern ausführen und so Durchsatz und Antwortverhalten drastisch verbessern.

Dank der in Mac OS X Server integrierten Xgrid-Funktionalität können Systemadministratoren Xgrid unternehmensweit aktivieren und damit ohne zusätzliche Kosten freie CPU-Zyklen in ein produktives Cluster verwandeln.
