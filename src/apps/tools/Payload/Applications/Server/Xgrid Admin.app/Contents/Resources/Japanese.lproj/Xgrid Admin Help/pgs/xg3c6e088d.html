エージェントを削除する

「Xgrid Admin」で Xgrid コントローラのエージェントを削除できます。

エージェントを削除するには：

  1. 「Xgrid Admin」を開きます。

  2. 「コントローラとグリッド」リストでコントローラを選択します。

  3. 「エージェント」をクリックします。

  4. エージェントのリストの下にある削除（－）ボタンをクリックします。

    注記： ローカルサブネット上にあることが分かっており、そのコントローラに追加するように構成されているエージェントを削除した場合は、しばらく待っていると、リストに再度表示されます。エージェントが再度表示されない場合は、追加（＋）ボタンを使用してその名前を入力すると、そのエージェントは戻ります。
