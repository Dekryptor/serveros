Wiederholen oder Neustarten eines Auftrags

Sie können einen Auftrag in Xgrid-Admin wiederholen oder einen unterbrochenen Auftrag erneut starten.

Gehen Sie wie folgt vor, um einen Auftrag zu wiederholen oder neu zu starten:

  1. Öffnen Sie das Programm "Xgrid-Admin".

  2. Wählen Sie den Controller in der Liste "Controller und Grids" aus.

  3. Klicken Sie auf "Aufträge".

  4. Wählen Sie den Auftrag aus, den Sie wiederholen oder neu starten möchten.

  5. Klicken Sie auf die Taste "Start" unter der Auftragsliste.
