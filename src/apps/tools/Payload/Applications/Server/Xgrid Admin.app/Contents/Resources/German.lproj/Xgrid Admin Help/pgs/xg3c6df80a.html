Verwalten von Xgrid-Controllern

Im Allgemeinen verwalten Sie den Xgrid-Controller wie jeden anderen Dienst von Mac OS X Server. Dazu verwenden Sie das Programm "Server-Admin" (zum Verwalten von laufenden Prozessen) oder Xgrid-Admin (zum Verwalten von Agenten- und Auftragswartelisten des Controllers).

Der erforderliche Verwaltungsaufwand hängt auch davon ab, wie viele Wartelisten vorhanden sind und wie viele Benutzer welche Arten von Aufträgen übermitteln.

Xgrid verwendet eine einfache FIFO-Warteliste (First-in, first-out) für die Ablaufplanung der einzelnen Grids. Das bedeutet, als Administrator müssen Sie sich auf die Mitarbeit Ihrer Kollegen verlassen, um sicherzustellen, dass die Ressourcen korrekt auf die Anzahl der Benutzer verteilt werden.
