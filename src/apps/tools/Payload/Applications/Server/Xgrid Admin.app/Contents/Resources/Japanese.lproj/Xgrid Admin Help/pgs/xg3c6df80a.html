Xgrid コントローラを管理する

一般に、Xgrid コントローラは、Mac OS X Server 上で動作するほかのサービスと同じように管理します。「サーバ管理」を使用して実行するプロセスを管理し、「Xgrid Admin」を使用してコントローラ上のエージェントとジョブのキューを管理します。

必要な管理作業の量は、存在するキューの数、およびジョブを送信するユーザの数（および性質）によっても決まります。

Xgrid では、個々のグリッドのスケジューリングに簡単なファーストイン・ファーストアウト（FIFO）のキューを使用します。これは、リソースが複数のユーザに正しく割り当てられるためには管理者が同僚の協力を得なければならないという意味です。
