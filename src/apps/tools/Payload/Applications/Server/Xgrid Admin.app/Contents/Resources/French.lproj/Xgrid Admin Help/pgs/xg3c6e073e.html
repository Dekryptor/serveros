Suppression d'une grille

Vous utilisez Xgrid Admin pour supprimer la grille d'un contrôleur Xgrid dans la liste Contrôleurs et grilles.

Pour supprimer une grille :

  1. Ouvrez Xgrid Admin.

  2. Dans la liste Contrôleurs et grilles, sélectionnez la grille.

  3. Cliquez sur le menu local Action sous la liste Contrôleur et grilles, puis sélectionnez Supprimer la grille.

  4. Cliquez sur OK.
