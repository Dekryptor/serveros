Administration des contrôleurs Xgrid

En règle générale, vous administrez le contrôleur Xgrid comme n'importe quel autre service s'exécutant sous Mac OS X Server, à l'aide d’Admin Serveur qui gère les processus en cours d'exécution et Xgrid Admin qui gère l'agent et les files d'attente de tâches sur le contrôleur.

L'importance de l'administration requise dépend également du nombre de files d'attente dont vous disposez et du nombre (et du caractère) des utilisateurs qui soumettent des tâches.

Xgrid utilise une simple file d'attente premier entré-premier sorti (FIFO) pour planifier chaque grille, ce qui signifie qu'en tant qu'administrateur, vous devez obtenir la collaboration de vos collègues pour vous assurer que les ressources sont affectées correctement entre plusieurs utilisateurs.
