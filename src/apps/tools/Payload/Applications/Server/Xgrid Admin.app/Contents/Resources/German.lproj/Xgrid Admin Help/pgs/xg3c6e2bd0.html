Hinzufügen eines Agenten

Sie können in Xgrid-Admin einen Agenten für einen Controller hinzufügen. Sie haben auch die Möglichkeit, Agenten hinzuzufügen, die offline sind. Die Agenten sind für den Controller dann verfügbar, wenn die Computer online sind oder wenn der Controller-Administrator diese Agenten aktiviert.

Gehen Sie wie folgt vor, um einen Agenten hinzuzufügen:

  1. Öffnen Sie das Programm "Xgrid-Admin".

  2. Wählen Sie den Controller in der Liste "Controller und Grids" aus.

  3. Klicken Sie auf "Agenten".

  4. Klicken Sie auf die Taste "Hinzufügen" (+) unter der Liste der Agenten.

  5. Geben Sie einen Namen für den Agenten ein und klicken Sie auf "OK".

Der Agent wird zur Liste hinzugefügt. Der farbige Punkt links neben dem Namen zeigt den Status des Agenten an. Näheres hierzu finden Sie unter: Statusanzeigen im Programm "Xgrid-Admin".
