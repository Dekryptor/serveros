グリッドの動作を監視する

グリッドの動作は、「Xgrid Admin」ですぐに確認できます。「Xgrid Admin」を使用して、エージェントとジョブの動作を表示することもできます。詳しくは、「エージェントのリストを表示する」および「ジョブのリストを表示する」を参照してください。

グリッドの動作を監視するには：

  1. 「Xgrid Admin」を開きます。

  2. 「コントローラとグリッド」リストで Xgrid コントローラを選択します。

  3. エージェントの数、使用可能なプロセッサ電力と使用済み処理能力、および実行中のジョブと保留中のジョブの数を表示するときは、「概要」をクリックします。
