Xgrid-Cluster

Bei Computer-Clustern handelt es sich um Systeme gleichen Typs, die ausschließlich für Computing-Aufgaben verwendet werden. In einem Cluster sind Systeme typischerweise in einem Rack untergebracht, über Gigabit-Ethernet oder ein anderes leistungsfähiges Netzwerk miteinander verbunden und unterliegen strengen Verwaltungsstrukturen, um eine größtmögliche Leistung bereitzustellen.

Cluster-Systeme bestehen häufig aus vollkommen identischen Geräten mit den gleichen Betriebssystemversionen, gleichen Software-Komponenten und den gleichen Prozessor-, Festplatten- und Arbeitsspeicher-Konfigurationen.

Xgrid ermöglicht Administratoren, die Verwaltung der verteilten Ressourcen des Clusters mühelos zu konfigurieren. Auf jedem Server im System wird die Agenten-Software ausgeführt und auf dem Hauptknoten im Cluster die Controller-Software.

Xgrid verteilt Aufgaben über die Computer im Cluster. In Clustern ist die Ausfallrate meist sehr niedrig. Die Systeme sind selten, wenn überhaupt, offline und ihre Ressourcen werden nicht auch für allgemeine Benutzeraufgaben verwendet. Cluster stellen die effizienteste jedoch auch teuerste Methode für die Verteilung von Rechenanforderungen dar.
