Herstellen der Verbindung zu einem Xgrid-Controller

Sie verwenden das Programm "Xgrid-Admin", um die Verbindung zu einem Xgrid-Controller herzustellen. Der Controller muss in jedem Netzwerk von dem Administrator-Computer aus erreichbar sein, auf dem Xgrid-Admin ausgeführt wird.

Nach dem Verbinden von Xgrid-Admin mit dem Controller können Sie den Status des entsprechenden Grids anzeigen und die dazugehörigen Agenten und Aufträge verwalten.

Gehen Sie wie folgt vor, um eine Verbindung zu einem Xgrid-Controller herzustellen:

  1. Öffnen Sie das Programm "Xgrid-Admin" und führen Sie einen der folgenden Schritte aus:

    • Wählen Sie einen Controller aus dem Einblendmenü aus oder geben Sie dessen Namen ein. Klicken Sie anschließend auf "Verbinden".

    • Wählen Sie in der Liste "Controller und Grids" den Namen des Controllers aus. Öffnen Sie dann das Aktionsmenü und wählen Sie "Verbinden" aus.

  2. Wählen Sie, falls erforderlich, die Ihre Identifizierungsoption aus und geben Sie ein Kennwort ein. Klicken Sie dann auf "OK".
