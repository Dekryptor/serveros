Entfernen eines Xgrid-Controllers

Sie verwenden das Programm "Xgrid-Admin", um die Verbindung zu einem Xgrid-Controller in der Liste "Controller und Grids" zu trennen.

Gehen Sie wie folgt vor, um einen Xgrid-Controller zu entfernen:

  1. Öffnen Sie das Programm "Xgrid-Admin".

  2. Wählen Sie einen Controller in der Liste "Controller und Grids" aus.

  3. Klicken Sie auf das Aktionsmenü und wählen Sie "Controller entfernen".
