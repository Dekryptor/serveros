Anforderungen und Kapazitäten

Xgrid eignet sich sowohl für kleine Cluster aus wenigen Computern als auch für umfangreiche, organisationsweite Grids. Xgrid unterstützt bis zu 128 Agenten, eine beliebige Anzahl von Aufträgen mit bis zu 100.000 Aufgaben in der Warteliste, bis zu 128 MB gesendete Daten pro Auftrag und bis 128 MB an Ergebnisdaten pro Auftrag.

Hierbei handelt es sich um empfohlene Maximalwerte, die von der Software nicht erzwungen werden. Sie können diese Maximalwerte auf eigenes Risiko überschreiten.
