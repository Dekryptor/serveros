エージェントを追加する

「Xgrid Admin」で、コントローラにエージェントを追加できます。オフラインになっているエージェントを追加できます。エージェントは、コンピュータがオンラインのとき、またはコントローラの管理者がエージェントを有効にしたときに、コンピュータで利用可能になります。

エージェントを追加するには：

  1. 「Xgrid Admin」を開きます。

  2. 「コントローラとグリッド」リストでコントローラを選択します。

  3. 「エージェント」をクリックします。

  4. エージェントのリストの下にある追加（＋）ボタンをクリックします。

  5. エージェントの名前を入力し、「OK」をクリックします。

エージェントがリストに追加されます。エージェント名の左にある色付きのランプは、エージェントの状況を示します。詳しくは、「Xgrid Admin の状況インジケータ」を参照してください。
