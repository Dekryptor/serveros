Hinzufügen eines Xgrid-Controllers

Sie verwenden das Programm "Xgrid-Admin", um einen Xgrid-Controller zur Liste "Controller und Grids" hinzuzufügen.

Gehen Sie wie folgt vor, um einen Xgrid-Controller zur Überwachungsliste hinzuzufügen:

  1. Öffnen Sie das Programm "Xgrid-Admin".

  2. Klicken Sie auf die Taste "Hinzufügen" (+) und wählen Sie "Controller hinzufügen".

  3. Wählen Sie einen Controller aus dem Einblendmenü aus oder geben Sie dessen Namen ein. Klicken Sie anschließend auf "Verbinden".

  4. Wählen Sie, falls erforderlich, die richtige Identifizierungsoption aus und geben Sie ein Kennwort ein. Klicken Sie dann auf "OK".
