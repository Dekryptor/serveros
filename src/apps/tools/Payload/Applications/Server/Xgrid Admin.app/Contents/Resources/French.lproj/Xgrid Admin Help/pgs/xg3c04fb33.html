Contrôleur

Le contrôleur Xgrid gère les communications entre les différentes ressources de traitement d'une grille. Le contrôleur nécessite Mac OS X Server 10.4 ou ultérieur. Le contrôleur accepte les connexions réseau depuis des clients et des agents. Il reçoit les soumissions de tâches des clients, les divise en sous-tâches, puis répartit ces sous-tâches entre les agents et retourne les résultats aux différents clients.

Même si plusieurs contrôleurs Xgrid peuvent s'exécuter sur un sous-réseau, un seul contrôleur peut s'exécuter par grille logique.

Chaque contrôleur peut disposer d'un nombre arbitraire d'agents connectés, mais Apple a testé une configuration avec 128 agents par contrôleur. Cependant, il n'existe aucune limite logicielle en ce qui concerne le nombre d'agents, et les utilisateurs d'Xgrid peuvent décider à leurs propres risques de dépasser le nombre de 128 agents sur un contrôleur, le chiffre maximum théorique étant égal au nombre de prises disponibles sur le système du contrôleur.
