Connexion à un contrôleur Xgrid

Vous pouvez utiliser Xgrid Admin pour vous connecter à un contrôleur Xgrid. Le contrôleur doit être accessible sur un réseau quelconque par l'ordinateur d'administration exécutant Xgrid Admin.

Une fois Xgrid Admin connecté au contrôleur, vous pouvez visualiser l'état de ses grilles et administrer ses agents et ses tâches.

Pour se connecter à un contrôleur Xgrid :

  1. Ouvrez Xgrid Admin et effectuez l'une des opérations suivantes :

    • Dans le menu local, choisissez le contrôleur ou saisissez son nom et cliquez sur Connexion.

    • Dans la liste Contrôleurs et grilles, sélectionnez le nom du contrôleur, cliquez sur le menu local Action et choisissez Connexion.

  2. Si nécessaire, sélectionnez votre option d'authentification, saisissez le mot de passe, puis cliquez sur OK.
