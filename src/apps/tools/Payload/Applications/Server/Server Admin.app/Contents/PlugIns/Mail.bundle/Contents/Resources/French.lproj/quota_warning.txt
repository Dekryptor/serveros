Objet : [Avertissement] Quota du serveur de courrier électronique presque atteint

Nous vous informons que vous approchez la limite de stockage maximale sur le serveur de courrier électronique. Les prochains courriers risquent d’être refusés si vous dépassez votre quota.

Veuillez déplacer les messages dont vous n’avez plus besoin sur le serveur vers vos dossiers de courrier locaux. Pensez également à vérifier vos dossiers « Messages envoyés », « Corbeille » et « Éléments supprimés » pour supprimer les messages qui ne sont plus utiles sur le serveur.

Merci de votre attention.
