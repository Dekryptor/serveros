IPv6 オプションを調整する

IPv6 は、新しいバージョンの IP（インターネットプロトコル）です。現在のところ、IPv6 は主に研究機関で使用されています。ほとんどのコンピュータで、IPv6 を設定したり使用したりする必要はありません。

IPv6 の主な利点は、アドレスサイズが 32 ビット（現在の IPv4 規格）から 128 ビットに増えるという点です。 128 ビットのアドレスサイズは、アドレス割り当ての効率は悪くなりますが、膨大な数のアドレスに対応するには十分な大きさです。 これにより、現在よりも多くのアドレスやノードを使用できるようになります。また、IPv6 にはより多くのアドレス設定方法があり、自動構成がより簡単です。

デフォルトでは IPv6 は自動的に構成され、IPv6 を使用する大多数のコンピュータには、デフォルト設定で十分です。ただし、ネットワーク管理者またはインターネット・サービス・プロバイダから IPv6 を手動で構成したり、無効にするよう具体的な指示があった場合は、下の手順に従ってください。

IPv6 アドレスに対応するサービスには、DNS、ファイアウォール、メール、ファイル共有（SMB プロトコル）、および Web があります。

IPv6 アドレスを構成するには：

  1. 左側のリストで、IPv6 アドレス設定を変更したい Ethernet、AirMac、または FireWire のネットワーク接続サービスを選択します。

  2. 「IPv6」をクリックし、「IPv6 を構成」ポップアップメニューから方法を選択します。「手入力」を選択した場合は、以下の情報を参考にして構成の詳細を入力します：

    • IPv6 アドレス： 
      8 つの部分から成る番号。一般に、0000:0000:0000:0000:0000:0000:0000:0000 の形式で記述されます。
    • ルーター： 
      サーバのローカルネットワーク上のルーターまたはゲートウェイの IPv6 アドレス。
    • プレフィックス長： 
      ネットワークの識別に使用する IPv6 アドレスの有効ビット数。
