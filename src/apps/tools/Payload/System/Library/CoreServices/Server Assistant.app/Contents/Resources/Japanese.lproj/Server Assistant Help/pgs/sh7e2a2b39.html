ネットワーク接続サービスを追加する

「ネットワーク」パネルで、ネットワーク接続サービスの追加、削除、配置変更、表示／非表示の切り替えを行うことができます。ネットワーク接続サービスとは、特定のネットワークポートやインターフェイス（コンピュータの Ethernet ポートや AirMac カードのような物理的ネットワークコネクタ）用の設定のグループのことです。

ネットワーク接続サービスを追加するには：

  1. 「アクション」（歯車）ポップアップメニューから「すべてのサービスを表示」を選択した後で、目的のネットワーク接続サービスが表示された場合は、新たなネットワーク接続サービスを追加するのではなく、手順 5 に進んで、表示されたネットワーク接続サービスを構成します。

    「アクション」（歯車）ポップアップメニューに「すべてのサービスを表示」が表示されない場合は、すでにすべてのネットワーク接続サービスが表示されています。新たにネットワーク接続サービスを追加する必要がある場合は、続けて次の手順を実行します。

  2. リストの下部にある「追加」（＋）ボタンをクリックします。

  3. 「タイプ」ポップアップメニューからインターフェイス（Ethernet や AirMac など）を選択し、新しいネットワーク接続サービスに説明的な名前を付けます。

  4. この時点でサーバを設定しないで、自動設定プロファイルを作成する場合は、BSD 名またはハードウェア・アドレス（MAC アドレスとも呼ばれます）のいずれかを入力することによって、ネットワーク接続サービスが使用する物理インターフェイス（ポート）を指定します。

    最初の内蔵 Ethernet ポートの名前は en0 です。詳しくは、「インターフェイス名またはハードウェア・アドレスを見つける」を参照してください。

  5. 「作成」をクリックして、リストで新しいサービスを選択し、それに適した設定を入力します。

    注記： サービス名が既存のサービス名と同じ場合、「作成」をクリックすることはできません。

    操作方法については、以下のトピックを参照してください。

    Ethernet ネットワーク接続サービスを構成する
    AirMac ネットワーク接続サービスを構成する
    FireWire ネットワーク接続サービスを構成する
    PPPoE ネットワーク接続サービスを構成する
    LOM ポートのネットワーク接続サービスを構成する
    ネットワーク構成の詳細

複数のネットワーク接続サービスがリストされているときに、それらの順序を変更するには、リスト内でドラッグします。パブリックネットワーク接続をリストの一番上にドラッグし、そのほかはサーバで使用したい順序に並べ替えます。

すべてのネットワーク接続サービスを表示したり、よく使うネットワーク接続サービスのみを表示したりするには、「アクション」（歯車）ポップアップメニューを使います。

ネットワーク接続サービスを削除するには、リストで選択し、「削除」（－）ボタンをクリックします。
