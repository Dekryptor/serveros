サーバの転送

「サーバの転送」パネルでは、既存のサーバまたはサーバの Time Machine バックアップから転送したいデータと設定が含まれているディスクを選択します。移行する情報が含まれているディスクは、サーバ内にあるか、設定しているサーバの FireWire または USB ポートに接続されている必要があります。別のサーバから移行する場合、そのサーバはターゲットディスクモードで起動され、設定しているサーバの FireWire ポートに接続されている必要があります。これができない場合は、「サーバアシスタント」を終了し（コマンド＋ Q キーを押して）、既存のサーバのディスクを、設定しているサーバに接続できるリムーバブルディスクにコピーしてから、新しいサーバを起動して再度設定を始める必要があります。

Mac OS X Server v10.4.11、または Mac OS X Server v10.5 の最新アップデートがあるサーバから移行することができます。また、Mac OS X Server v10.5 の最新アップデートの標準構成またはワークグループ構成の Time Machine バックアップから移行することもできます。

移行プロセスは、「転送」をクリックするとすぐに始まります。
