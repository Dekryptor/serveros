Konfigurieren eines FireWire-Netzwerkverbindungsdiensts

Wenn Ihr Server und ein weiterer Mac-Computer über FireWire-Anschlüsse verfügen, können Sie sie mit einem FireWire-Kabel miteinander verbinden und darüber dann ein lokales Netzwerk konfigurieren.

Gehen Sie wie folgt vor, um einen FireWire-Netzwerkverbindungsdienst zu konfigurieren:

  1. Wählen Sie "FireWire" in der Liste auf der linken Seite aus.

  2. Klicken Sie auf "Konfigurieren", wählen Sie die von Ihrem Netzwerkadministrator empfohlene Konfigurationsmethode und geben Sie die Konfigurationsdetails ein.

    In den meisten FireWire-Netzwerkumgebungen genügt die Auswahl von "DHCP" und die Verwendung der automatisch zugeordneten Konfigurationsdetails.

    Entsprechende Anweisungen finden Sie in den folgenden Hilfethemen:

    Konfigurieren eines Netzwerkverbindungsdiensts mit DHCP
    Konfigurieren eines Netzwerkverbindungsdiensts mit DHCP und manueller Adresse
    Konfigurieren eines Netzwerkverbindungsdiensts mit BootP
    Manuelles Konfigurieren eines Netzwerkverbindungsdiensts

  3. Wenn Ihr FireWire-Netzwerk die Anpassung erweiterter IPv6-Einstellungen erfordert, klicken Sie auf die Taste "IPv6". Entsprechende Anweisungen finden Sie unter Anpassen von IPv6-Optionen.

    Standardmäßig generiert der Server automatisch eine IPv6-Adresse.

Wenn FireWire in der Liste der Netzwerkverbindungsdienste fehlt, wählen Sie "Alle Dienste einblenden" aus dem Aktionsmenü (Zahnrad) aus. Wenn FireWire weiterhin nicht aufgeführt wird, besitzt Ihr Server keinen funktionsfähigen FireWire-Anschluss.
