Install Xsan

Use the Install Xsan pane to begin installing the Xsan software on the server now, including the Xsan file system and Xsan Admin application, or defer installing Xsan until later. The Install Xsan pane appears if you chose to set up the server as an Xsan metadata controller in an earlier pane.

Before installing Xsan on the server, you should have the SAN storage and networks set up, and you should install Xsan on each client computer. Before installing Xsan on the server that is the primary metadata controller, you should first install Xsan on the servers that are secondary metadata controllers.
