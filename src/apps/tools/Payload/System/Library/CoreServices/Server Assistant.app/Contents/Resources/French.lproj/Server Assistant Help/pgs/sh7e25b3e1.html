Utilisez la sous-fenêtre Revoir pour indiquer ce que vous souhaitez faire au niveau des réglages définis.

Voici les opérations possibles :

  • Cliquez sur Revenir pour passer en revue les données des sous-fenêtres précédentes et, si nécessaire, y apporter des modifications.

  • Cliquez sur Configurer pour finaliser la configuration du serveur interactif.

    L’Assistant du serveur configure le serveur en fonction des réglages effectués. Il génère également le document Étapes suivantes de Mac OS X Server et le place sur le bureau du serveur. Ce document est spécialement adapté à votre serveur et votre réseau. Il vous indique les modifications supplémentaires à apporter en vue de renforcer la sécurité, d’améliorer l’accessibilité et d’optimiser le fonctionnement général de votre nouveau serveur.

  • Cliquez sur Détails pour connaître les détails de configuration ou enregistrer les réglages effectués.

    Vous pouvez enregistrer un résumé sous forme de fichier texte, puis sauvegarder les réglages dans un profil de configuration automatique. Pour en savoir plus, consultez la rubrique Enregistrement d’un profil de configuration automatique.
