À propos du service Carnet d’adresses

Le service Carnet d’adresses permet aux utilisateurs d’accéder aux informations de contact contenues dans un carnet d’adresses sur plusieurs ordinateurs, sans les limites de schéma et les problèmes de sécurité associés à LDAP. Un utilisateur qui dispose de plusieurs ordinateurs Snow Leopard peut utiliser les applications Mail et Carnet d’adresses sur ces derniers pour afficher et ajouter des contacts à un carnet d’adresses situé sur le serveur. Les utilisateurs des autres ordinateurs peuvent accéder à un carnet d’adresses situé sur le serveur à l’aide d’applications de tierce partie compatibles avec la norme ouverte CardDAV.
