Numéro de série

La sous-fenêtre Numéro de série vous permet d’indiquer un numéro de série pour chaque serveur en cours de configuration.

Vous devez utiliser un numéro de série Mac OS X Server 10.6 Snow Leopard, qui commence par XSVR-106. Celui-ci est imprimé sur les documents fournis avec le paquet Mac OS X Server. À moins que vous ne disposiez d’une licence de site, vous devez avoir un numéro de série unique pour chaque serveur. Il n’est pas possible d’utiliser un numéro de série provenant d’une ancienne version de Mac OS X Server pour enregistrer Snow Leopard Server.

Si vous possédez une licence de site, vous devez indiquer le nom d’un propriétaire enregistré et d’une organisation exactement tels qu’ils vous ont été fournis par le représentant Apple.

Si vous configurez plusieurs serveurs, la sous-fenêtre Numéro de série répertorie les serveurs disposant d’un numéro de série valide. Le nombre de serveurs sans numéro de série est précisé au bas de la liste. Pour ajouter un numéro de série, cliquez sur Ajouter. L’Assistant du serveur applique alors le numéro de série à un serveur qui n’en a pas. Vous pouvez également ajouter une licence de site, que l’Assistant du serveur appliquera à plusieurs serveurs.
