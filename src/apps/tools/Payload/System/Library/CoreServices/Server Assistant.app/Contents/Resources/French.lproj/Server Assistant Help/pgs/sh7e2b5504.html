À propos du service de messagerie instantanée

Mac OS X Server propose la messagerie instantanée aux utilisateurs Macintosh, Windows et Linux. Les membres d’une équipe peuvent réfléchir à des solutions, établir des plans, échanger des adresses URL ou transférer des fichiers, sans craindre de voir des personnes extérieures intercepter des informations confidentielles. Le service de messagerie instantanée propose l’échange de messages de texte entre deux ou plusieurs utilisateurs. Il facilite aussi les connexions directes entre les utilisateurs pour des sessions audio, vidéo ou alliant les deux à la fois.

Le service de messagerie instantanée fonctionne avec l’application iChat de Mac OS X, Google Talk et d’autres logiciels de messagerie instantanée, et utilise le protocole XMPP, appelé Jabber.
