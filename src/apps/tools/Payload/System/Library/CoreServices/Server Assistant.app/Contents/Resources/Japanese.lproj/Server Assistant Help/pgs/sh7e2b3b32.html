サービス

「サービス」パネルでは、サーバがユーザのコンピュータに最初に提供するサービスを指定します。サーバに複数のディスクがある場合、または複数のボリュームでパーティションが作成されたディスクがある場合は、サービスデータを保存するディスクを選択することもできます。

重要： 前のパネルでサーバをインターネットゲートウェイとして構成するオプションを選択した場合を除き、サーバのファイアウォールサービスは、ネットワークの外からのサービスへのアクセスをブロックするようには構成されていません。（ゲートウェイオプションが表示されるのは、サーバに少なくとも 2 つの Ethernet ポートがある場合のみです。）サーバが別個のインターネットルーター、ゲートウェイ、またはファイアウォールによって保護されていない場合は、初期設定の終了後に「サーバ環境設定」の「ファイアウォール」パネルまたは「サーバ管理」でファイアウォール保護を有効にする必要があります。

設定の完了後に、「サーバ環境設定」アプリケーションでサービス設定を変更することができます。詳細なサービス設定の変更や、ここにリストされていないサービスの設定が必要な場合は、「サーバ管理」アプリケーションを使用できます。

「ユーザとグループ」パネルで「手動で設定」を選択した後、ディレクトリサーバに接続して「ディレクトリサーバに接続」パネルで Open Directory のマスターを作成しないと、「サービス」パネルは表示されません。また、サーバ上に Xsan を設定している場合も「サービス」パネルは表示されません。
