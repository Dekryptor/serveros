メールオプション

「メール」オプションパネルでは、メール・リレー・サーバと、新規ユーザに送信されるウェルカムメッセージを指定します。「メールオプション」パネルは、以前に「サービス」パネルでメールサービスを選択した場合に表示されます。

メールオプション
__ を経由して送信メールをリレー

ISP または組織によっては、すべての送信メールをリレーサーバ経由で転送することが必要な場合があります。お使いのネットワークがこれに該当する場合は、リレーサーバの DNS 名を指定する必要があります。たとえば、ISP または組織の DNS サービスがサーバ（または、AirMac ベースステーションやその他のルーター）のパブリック IP アドレスをサーバの DNS 名に解決できない場合は、リレーサーバを使用する必要があります。 これは逆引き参照と呼ばれています。

また、設定後に「サーバ環境設定」アプリケーションの「メール」パネルで、メールリレー（一部のリレーサーバで必要な認証を含む）を構成することもできます。

重要： ISP または組織で指定されたリレーサーバのみを使用してください。許可がないのに別のサーバを介してメールをリレーさせようとすると、サーバが迷惑メールの送信者であるように見える可能性があります。
SMTP リレー認証を有効にする

ISP または組織で、メール送信前にサーバによる認証が要求される場合は、このオプションを選択し、ISP または組織から提供されたユーザ名とパスワードを入力します。

新規ユーザにウェルカムメールを送信

サーバの新規ユーザに対してそのサービスについて説明するメールを送信する場合は、このオプションを選択します。標準メッセージには、サーバの DNS 名と、サーバから提供されるサービスの説明が記載されます。
