Ziellaufwerk

Wählen Sie im Fenster "Ziellaufwerk" die Festplatte oder das Volume einer partitionierten Festplatte aus, auf der bzw. auf dem Sie Mac OS X Server installieren wollen. Wenn auf der Zielfestplatte oder dem Zielvolume eine qualifizierte Version von Mac OS X Server installiert ist, können Sie diese aktualisieren. Wenn auf der Zielfestplatte oder dem Zielvolume eine andere Version von Mac OS X Server oder Mac OS X installiert ist, können Sie die Platte oder das Volume vor der Installation löschen und umbenennen. In der folgenden Tabelle wird erläutert, welche Aktionen Sie ausführen können, je nachdem, was bereits auf der Zielplatte oder dem Zielvolume installiert ist.

ACHTUNG: Kopieren Sie vor dem Löschen einer Festplatte oder eines Volumes alle Dateien, die Sie behalten wollen, auf eine andere Platte. Durch Löschen einer Festplatte oder eines Volumes werden alle Informationen darauf zerstört. Durch Löschen eines Volumes auf einer partitionierten Festplatte die Informationen auf anderen Volumes auf derselben Platte nicht gelöscht.
