Installing

The Installing pane appears while Mac OS X Server is being installed on the remote server. You can monitor installation progress using this pane.

When installation is complete, the remote server restarts automatically. Server Assistant waits for you to indicate what you want to do.

  • If you want to install Mac OS X Server on another remote computer, click More Options to go to the Welcome pane.

  • If you want to set up Mac OS X Server remotely, click More Options to go to the Welcome pane.

  • If you want to quit Server Assistant, click Quit.
