Adding a Network Connection Service

You can add, remove, rearrange, and show or hide network connection services in the Network pane. A network connection service is a group of settings for a specific network port or interface (a physical network connector, such as your computer’s Ethernet port or AirPort Card, for example).

To add a network connection service:

  1. If the desired network connection service becomes visible after you choose Show All Services from the Action (gear) pop-up menu, skip to step 5 and configure the revealed network connection service instead of adding a new one.

    If the Action (gear) pop-up menu doesn’t include Show All Services, you can already see all network connection services. Continue with the next step if you need to add a new one.

  2. Click the Add (+) button at the bottom of the list.

  3. Choose an interface from the Type pop-up menu, such as Ethernet or AirPort, and give the new network connection service a descriptive name.

  4. If you’re creating an auto setup profile (not setting up a server now), specify which physical interface (port) the network connection service will use, either by entering its BSD name or hardware address (also called MAC address).

    The first built-in Ethernet port has the name en0. For more information, see Finding an Interface Name or Hardware Address.

  5. Click Create, select the new service in the list, and enter settings appropriate to it.

    Note:  You can’t click Create if the service name is the same as an existing service name.

    For instructions, see the following topics.

    Configuring an Ethernet Network Connection Service
    Configuring an AirPort Network Connection Service
    Configuring a FireWire Network Connection Service
    Configuring a PPPoE Network Connection Service
    Configuring an LOM Port Network Connection Service
    Network Configuration Details

To change the order of network connection services when more than one is listed, drag them in the list. Drag the public network connection to the top of the list, and arrange the others in the order you want the server to use them.

To show all network connection services or only preferred ones, use the Action (gear) pop-up menu.

To remove a network connection service, select it in the list and click the Delete (–) button.
