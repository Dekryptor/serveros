Willkommen

Mit dem Serverassistenten können Sie die Mac OS X Server-Software auf einem weiteren Computer installieren und konfigurieren (siehe dazu die folgende Tabelle). Sie können den Serverassistenten eines bereits konfigurierten Servers verwenden oder den eines Mac OS X-Computers, auf dem die Verwaltungssoftware für Mac OS X Server installiert ist.

Nach dem Konfigurieren eines Servers verwalten Sie Benutzer und Gruppen und passen die Einstellungen einzelner Dienste mithilfe des Programms "Servereinstellungen" oder erweiterter Serververwaltungsprogramme wie dem Programm "Server-Admin" und dem Arbeitsgruppenmanager an. Weitere Informationen zum Konfigurieren und Verwalten von Mac OS X Server finden Sie im Handbuch Einführung, das zum Lieferumfang von Mac OS X Server gehört.

Sie verwenden den Serverassistenten für folgende Schritte:
Entferntes Installieren von Mac OS X Server

Installieren von Serversoftware auf einem Computer im Netzwerk. Sie haben folgende Möglichkeiten:

  • Installieren auf einem Volume, auf dem Mac OS X Server oder Mac OS X noch nicht installiert ist

  • Aktualisieren von Mac OS X Server Version 10.4.11 oder die letzte Aktualisierung von Mac OS X Server 10.5

Der entfernte Computer muss von der Installations-DVD für Mac OS X Server gestartet werden, wobei vom Installationsprogramm die Sprachauswahl angezeigt wird.

Konfigurieren eines entfernten Servers

Konfigurieren Sie einen Server im Netzwerk. Für den entfernten Server muss Folgendes zutreffen:

  • Es handelt sich um einen neuen Server oder einen Computer mit neu installiertem Mac OS X Server 10.6.

  • Er wurde mit dem Serverassistenten gestartet, dessen Begrüßungsbildschirm angezeigt wird.

Sie können mehrere Server gleichzeitig konfigurieren. Diese müssen dieselbe Hardwarekonfiguration haben und dieselben Konfigurationsdaten verwenden, mit Ausnahme der Mac OS X Server-Seriennummern, Netzwerkverbindungsdienste und Netzwerknamen. Wenn Sie die AirPort-Verwaltung auf einem Server konfigurieren möchten, konfigurieren Sie den Server separat oder aktivieren Sie die AirPort-Verwaltung nach der Konfiguration im Programm "Servereinstellungen".

Erstellen eines Profils für die automatische Konfiguration

Führen Sie den Konfigurationsprozess aus. Wählen Sie Konfigurationsoptionen aus und geben Sie Konfigurationsdaten ein. Konfigurieren Sie dann einen Server nicht sofort mithilfe der Konfigurationsinformationen, sondern sichern Sie diese als Profil für die automatische Konfiguration auf einem Wechselspeicherlaufwerk oder einer CD/DVD. Später können Sie mit dem gesicherten Profil für die automatische Konfiguration die Konfiguration von einem oder mehr Servern automatisieren.

Sie können Profile für die automatische Konfiguration verwenden, um mehrere Server gleichzeitig zu konfigurieren oder Server nach der erneuten Installation der Serversoftware zu testen.

Sie können ein allgemeines Profil für die automatische Konfiguration erstellen, um mehrere Server zu konfigurieren. Sie können ein einzelnes Profil für die automatische Konfiguration aber auch für einen bestimmten Server erstellen. Beim Konfigurieren eines allgemeinen Profils für die automatische Konfiguration führen Sie die folgenden Schritte aus:

  • Sie geben den Namen und das Kennwort für einen Administrator-Account an, der von allen Servern verwendet wird. Sie können das Administratorkennwort eines Servers nach der Konfiguration ändern. Hierzu verwenden Sie die Systemeinstellung "Benutzer".

  • Sie können einen Netzwerkverbindungsdienst für jede Hardwareadresse (auch als Ethernet-ID oder MAC-Adresse bezeichnet) oder einen allgemeinen Netzwerkverbindungsdienst für einen Schnittstellennamen (BSD-Portname) wie en0 oder en1 angeben und Ihren DHCP-Server so konfigurieren lassen, dass er jedem Server eine feste (statische) IP-Adresse gemäß der Hardwareadresse zuordnet. Diese Funktion wird als statische Zuordnung oder DHCP-Reservierungen bezeichnet.

  • Sie können die Netzwerknamen frei lassen und Ihren DNS-Dienst so konfigurieren lassen, dass er jedem Server einen Namen gemäß der IP-Adresse zuordnet.

Sie können das Profil für die automatische Konfiguration aus Sicherheitsgründen in verschlüsselter Form sichern.
