Se connecter à un serveur d’annuaire

Utilisez la sous-fenêtre « Se connecter à un serveur d’annuaire » pour indiquer le serveur d’annuaire à partir duquel vous prévoyez d’importer des comptes d’utilisateur. Vous pouvez importer des comptes existants à partir du serveur Open Directory ou Active Directory de votre organisation. Votre serveur utilise le nom, le mot de passe et les autres attributs de compte qui sont déjà définis dans les comptes d’utilisateur importés. Votre serveur peut compléter les attributs existants avec d’autres attributs nécessaires pour ses services. Par exemple, les utilisateurs importés peuvent être membres d’un groupe créé sur votre serveur.

Réglages relatifs à l’importation depuis un serveur d’annuaire
Adresse du serveur

Sélectionnez le serveur d’annuaire dans le menu local ou tapez directement le nom DNS du serveur d’annuaire. Si vous vous connectez à un serveur Open Directory, vous pouvez saisir son adresse IP.

Type de serveur

Il peut s’agir d’un serveur Open Directory ou Active Directory.

Si vous configurez un serveur dès à présent (sans créer un profil de configuration automatique à utiliser ultérieurement), le type de serveur est automatiquement déterminé.
Nom et mot de passe du compte d’utilisateur

Ces champs s’affichent si le serveur d’annuaire permet ou exige l’authentification.

  • Pour un serveur Open Directory, vous pouvez saisir le nom abrégé et le mot de passe d’un compte d’utilisateur standard, il n’est pas nécessaire d’utiliser un compte d’administrateur d’annuaire.

  • Pour un serveur Active Directory, vous pouvez saisir le nom et le mot de passe d’un compte d’administrateur Active Directory ou d’un compte d’utilisateur standard doté du privilège « Ajouter des stations de travail au domaine ».
