Anpassen von IPv6-Optionen

IPv6 ist eine neue IP-Version (Internet Protocol). Derzeit wird IPv6 vor allem in Forschungseinrichtungen verwendet. Für die meisten Computer muss IPv6 weder konfiguriert noch verwendet werden.

Der Hauptvorteil von IPv6 besteht darin, dass es die Adressgröße von 32 Bit (der aktuelle IPv4-Standard) auf 128 Bit vergrößert. Eine Adressgröße von 128 Bit ist groß genug, um eine große Anzahl von Adressen zu unterstützen, sogar mit einer ineffizienten Adresszuweisung. Dadurch sind mehr Adressen oder Knoten möglich als bisher. IPv6 bietet darüber hinaus mehr Möglichkeiten zum Konfigurieren von Adressen sowie eine einfachere automatische Konfiguration.

Standardmäßig wird IPv6 automatisch konfiguriert, wobei die Standardeinstellungen für die meisten Computer, für die IPv6 verwendet werden muss, ausreichend sind. Wenn Ihr Netzwerkadministrator oder Internetdienstanbieter Sie jedoch ausdrücklich angewiesen hat, IPv6 manuell zu konfigurieren oder zu deaktivieren, befolgen Sie die Anweisungen unten.

Dienste, die IPv6-Adressen unterstützen, sind DNS, Firewall, Mail, Dateifreigabe (SMB-Protokoll) und Web.

Gehen Sie wie folgt vor, um die IPv6-Adressierung zu konfigurieren:

  1. Wählen Sie in der Liste links den Ethernet-, AirPort- oder FireWire-Netzwerkverbindungsdienst aus, dessen IPv6-Adressierungseinstellungen Sie ändern wollen.

  2. Klicken Sie auf "IPv6" und wählen Sie eine Methode aus dem Einblendmenü "IPv6 konfigurieren". Wenn Sie "Manuell" wählen, geben Sie die Konfigurationsdetails anhand der folgenden Informationen ein:

    • IPv6-Adresse:  
      Eine achtteilige Nummer, die im Allgemeinen im Format "0000:0000:0000:0000:0000:0000:0000:0000" angegeben wird.
    • Router:  
      Die IPv6-Adresse des Routers oder Gateways im lokalen Netzwerk des Servers.
    • Präfixlänge:  
      Die Anzahl signifikanter Bits in der IPv6-Adresse, die zum Identifizieren des Netzwerks verwendet werden.
