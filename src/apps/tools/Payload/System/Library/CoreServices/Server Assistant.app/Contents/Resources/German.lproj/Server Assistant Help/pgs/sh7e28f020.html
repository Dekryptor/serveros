Anpassen von Ethernetoptionen

Manche Ethernetnetzwerke erfordern die Festlegung erweiterter Optionen. In den meisten Fällen sind die automatisch konfigurierten Einstellungen richtig. Fehlerhafte Ethernetanschlusseinstellungen können die Netzwerkleistung beeinträchtigen oder einen Anschluss sogar ganz unbrauchbar machen. Ihr Netzwerkadministrator oder Internetanbieter informiert Sie darüber, ob Sie erweiterte Einstellungen anpassen müssen.

Gehen Sie wie folgt vor, um erweiterte Etherneteinstellungen zu konfigurieren:

Wählen Sie einen Ethernetdienst aus, klicken Sie auf die Taste "Ethernet" und legen Sie die Optionen anhand der folgenden Informationen fest:

  • Geschwindigkeit:  
    Gibt die maximale Anzahl von Bit pro Sekunde an, die der Ethernetanschluss senden oder empfangen kann. Wenn Ihr Computer und das Gerät, zu dem Sie eine Verbindung herstellen, Gigabit-Ethernet unterstützen, das verwendete Ethernetkabel hingegen nicht, müssen Sie evtl. die Ethernetgeschwindigkeit anpassen. Alternativ können Sie ein Kabel kaufen, das Gigabit Ethernet unterstützt.
  • Duplex:  
    Gibt an, ob Eingangs- und Ausgangspakete gleichzeitig (vollduplex) oder im Wechsel (halbduplex) übertragen werden.
  • MTU:  
    Das größte Paket, das der Anschluss senden oder empfangen kann. (MTU steht für maximale Übertragungseinheit, ausgedrückt in Byte.) Die Erhöhung der Paketgröße steigert den Durchsatz (z. B. für Dateiübertragungen), allerdings müssen die Geräte, die diese Pakete empfangen (Switches, Router usw.) die betreffende Paketgröße auch unterstützen.
