ホーム
宛先

「宛先」パネルでは、Mac OS X Server をインストールしたいリモートコンピュータを識別します。このパネルを使用する前に、Mac OS X Server Install Disc から、または Mac OS X Server の NetBoot サービスにより提供されるネットワーク・インストール・イメージから、リモートコンピュータを起動してください。

リモートインストールを開始するために必要な情報
アドレス

サーバソフトウェアをローカルネットワーク（IP サブネット）上のコンピュータにインストールするには、ポップアップメニューから対象のコンピュータを選択します。

リストにないコンピュータを指定するには、その DNS 名または IP アドレス（IPv4 フォーマット、たとえば 192.0.2.200）を入力します。

「サーバアシスタント」によってローカルネットワーク（IP サブネット）上で見つかるサーバの場合、IP アドレスはネットワーク上の DHCP サーバによって自動的に割り当てられることがあります。DHCP サーバが存在しない場合、ターゲットサーバはローカルネットワーク上のサーバ全体で一意の 169.254.xxx.xxx というアドレスを使用します。IP アドレスは、後でサーバを設定するときに変更できます。

シリアル番号

サーバの内蔵ハードウェアシリアル番号の最初の 8 文字を、パスワードとして入力します。シリアル番号を見つけるには、サーバ上のラベルを探してください。シリアル番号を入力するときは、大文字と小文字をラベルに記載されている通りに入力してください。

メイン・ロジック・ボードが交換されて、ハードウェアシリアル番号がない Intel プロセッサを搭載した Xserve の場合は、パスワードとして「System S」と入力してください（カッコは入力しないでください）。内蔵ハードウェアシリアル番号がない別のコンピュータでは、パスワードとして 12345678 を使用してください。
