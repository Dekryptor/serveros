Server

Verwenden Sie das Fenster "Server", um den oder die Server anzugeben, die Sie konfigurieren wollen. Sie können mehrere Server gleichzeitig konfigurieren. Diese müssen dieselbe Hardwarekonfiguration haben und dieselben Konfigurationsdaten verwenden, mit Ausnahme der Mac OS X Server-Seriennummern, Netzwerkverbindungsdienste und Netzwerknamen.

In diesem Fenster werden Server aufgeführt, die Sie im Programm "Server-Admin" zur Konfiguration ausgewählt haben. Sie können weitere Server einfügen oder aufgeführte Server löschen.

Sie müssen sich anmelden, um einen Server entfernt zu konfigurieren. Wenn Sie mehrere Server konfigurieren, müssen Sie sich an jedem Server einzeln anmelden.

Das Fenster "Server" wird nur angezeigt, wenn Sie einen entfernten Server konfigurieren (nicht den Computer, an dem Sie arbeiten).

Gehen Sie wie folgt vor, um sich an einem aufgeführten Server für die entfernte Konfiguration anzumelden:

Wählen Sie einen Server in der Liste aus, klicken Sie auf "Identifizieren" und geben Sie das Standardkennwort des Servers ein.

  • Geben Sie für eine neue Installation von Mac OS X Server die ersten acht Zeichen der Hardwareseriennummer des Servers ein. Achten Sie bei der Eingabe der Seriennummer auf die Groß- und Kleinschreibung der Zeichen. Diese Seriennummer finden Sie auf dem Etikett, das außen auf dem Servergehäuse angebracht ist.

    Wenn der entfernte Server ein Xserve mit Intel-Prozessor ist, der keine Hardwareseriennummer hat, weil dessen Hauptplatine ersetzt wurde, geben Sie "System S" (ohne Anführungszeichen) als Kennwort ein.

    Geben Sie für einen weiteren Computer ohne integrierte Hardwareseriennummer die Ziffernfolge "12345678" als Kennwort ein.

  • Geben Sie für einen aktualisierten Server das Kennwort des Benutzer-Accounts "root" ein.

Gehen Sie wie folgt vor, um einen Server für die entfernte Konfiguration hinzuzufügen:

Klicken Sie auf "Hinzufügen". Geben Sie die IP-Adresse oder den DNS-Namen des entfernten Servers und dann ein Kennwort ein, wie in der Aufgabe oben beschrieben.
