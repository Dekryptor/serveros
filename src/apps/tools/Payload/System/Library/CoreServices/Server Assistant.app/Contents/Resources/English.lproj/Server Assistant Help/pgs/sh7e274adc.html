Configuring a FireWire Network Connection Service

If your server and another Mac computer have FireWire ports, you can connect them using a FireWire cable, and then set up a local network between them.

To configure a FireWire network connection service:

  1. Select FireWire in the list on the left.

  2. Click Configure, choose the configuration method recommended by your network administrator, and enter the configuration details.

    In most FireWire network environments, choosing Using DHCP and using the automatically assigned configuration details is sufficient.

    For instructions, see the following topics:

    Configuring a Network Connection Service Using DHCP
    Configuring a Network Connection Service Using DHCP with Manual Address
    Configuring a Network Connection Service Using BootP
    Configuring a Network Connection Service Manually

  3. If your FireWire network requires adjusting advanced IPv6 addressing settings, click the IPv6 button. For instructions, see Adjusting IPv6 Options.

    By default, the server generates an IPv6 address automatically.

If FireWire isn’t in the list of network connection services, choose Show All Services from the Action (gear) pop-up menu. If FireWire still isn’t listed, your server doesn’t have a functional FireWire port.
