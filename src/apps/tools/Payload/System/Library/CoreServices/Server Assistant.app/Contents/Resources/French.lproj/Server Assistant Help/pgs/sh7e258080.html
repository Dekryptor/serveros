Se connecter à un serveur d’annuaire

Utilisez la sous-fenêtre « Se connecter à un serveur d’annuaire » pour configurer manuellement une connexion vers un serveur d’annuaire de votre entreprise. Si vous préférez ne pas configurer la connexion au serveur d’annuaire de suite, vous pourrez vous connecter à un serveur d’annuaire après avoir terminé la configuration du serveur. Pour connecter des serveurs d’annuaire après avoir configuré le serveur, utilisez la rubrique Options dans la sous-fenêtre Comptes des Préférences Système.

Réglages de connexion du serveur d’annuaire
Se connecter à un serveur d’annuaire externe

Sélectionnez cette option pour que le serveur se connecte à un système d’annuaire existant.

Si vous configurez Xsan sur votre serveur, cette option n’est pas disponible. Si vous préférez ne pas vous connecter à un serveur d’annuaire dès à présent, cliquez sur Revenir et sélectionnez « Se connecter à un serveur d’annuaire ultérieurement » dans la sous-fenêtre Utilisateurs et groupes.

Adresse du serveur

Sélectionnez le serveur d’annuaire dans le menu local ou tapez directement le nom DNS du serveur d’annuaire. Si vous vous connectez à un serveur Open Directory, vous pouvez saisir son adresse IP.

Type de serveur

Il peut s’agir d’un serveur Open Directory ou Active Directory.

Si vous configurez un serveur dès à présent (sans créer un profil de configuration automatique à utiliser ultérieurement), le type de serveur est automatiquement déterminé.

Si vous devez vous connecter à un autre type de serveur d’annuaire, désélectionnez l’option « Se connecter à un serveur d’annuaire externe », puis paramétrez la connexion après avoir terminé la configuration du serveur.

Si vous configurez Xsan sur votre serveur, cette sous-fenêtre vous permet uniquement de vous connecter à un serveur Open Directory. Pour connecter un serveur doté de Xsan à un autre type de serveur d’annuaire, cliquez sur Revenir et sélectionnez « Se connecter à un serveur d’annuaire ultérieurement » dans la sous-fenêtre Utilisateurs et groupes.

Nom et mot de passe du compte d’utilisateur

Ces champs s’affichent si le serveur d’annuaire permet ou exige l’authentification.

  • Pour Open Directory, vous pouvez saisir le nom abrégé ou le nom entier et le mot de passe d’un compte d’utilisateur standard, il n’est pas nécessaire d’utiliser un compte d’administrateur d’annuaire.

  • Pour Active Directory, vous pouvez saisir le nom et le mot de passe d’un compte d’administrateur Active Directory ou d’un compte d’utilisateur standard doté du privilège « Ajouter des stations de travail au domaine ».
