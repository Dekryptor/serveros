Dienste

Geben Sie im Fenster "Dienste" an, welche Dienste Ihr Server den Computern der Benutzern anfänglich zur Verfügung stellt. Wenn der Server über mehrere Festplatten oder eine partitionierte Festplatte mit mehreren Volumes verfügt, können Sie auch wählen, wo die Dienstdaten gespeichert werden.

Weitere Informationen zu den Diensten finden Sie in den folgenden Hilfethemen:

Dienst "Adressbuch"
Dienst "iCal"
Dienst "Dateifreigabe"
Dienst "iChat"
Dienst "Mail"
Webdienste
Wichtig: Der Firewall-Dienst des Servers ist nicht dafür konfiguriert, den Zugriff auf Dienste außerhalb Ihres Netzwerks zu blockieren, es sei denn, Sie haben die Option zum Konfigurieren des Servers als Internet-Gateway in einem früheren Fenster ausgewählt. (Die Gateway-Option wird nur angezeigt, wenn der Server mindestens zwei Ethernetanschlüsse besitzt.) Wenn der Server nicht durch separate Internet-Router, -Gateways oder -Firewalls geschützt wird, sollten Sie nach Abschluss der anfänglichen Konfiguration den Firewall-Schutz im Fenster "Firewall" des Programms "Servereinstellungen" oder "Server-Admin" aktivieren.

Nachdem die Konfiguration beendet ist, können Sie die Diensteinstellungen mit dem Programm "Servereinstellungen" ändern. Wenn Sie erweiterte Diensteinstellungen ändern oder Dienste konfigurieren müssen, die hier nicht aufgeführt sind, können Sie dazu das Programm "Server-Admin" verwenden.

Das Fenster "Dienste" wird nicht angezeigt, wenn Sie "Manuell konfigurieren" im Fenster "Benutzer und Gruppen" ausgewählt haben, keine Verbindung zu einem Verzeichnisserver herstellen und auch keinen Open Directory-Master im Fenster "Mit einem Verzeichnisserver verbinden" erstellen. Das Fenster "Dienste" wird ebenfalls nicht angezeigt, wenn Sie Xsan auf einem Server konfigurieren.
