Dienst "Mail"

Der Dienst "Mail" ermöglicht es Benutzern, mit einem E-Mail-Programm ihrer Wahl E-Mail-Nachrichten über das lokale Netzwerk und das Internet zu senden und zu empfangen. Der Dienst "Mail" beinhaltet Filter, die Benutzer vor unerwünschter Werbung und Viren schützen.

Der Dienst "Mail" arbeitet mit dem Programm "Mail" in Mac OS X, dem Programm "Mail" auf dem iPhone und anderen beliebten E-Mail-Programmen zusammen, die Standard-E-Mail-Protokolle verwenden. Der E-Mail-Empfang kann über das Protokoll IMAP oder POP erfolgen. Der E-Mail-Versand erfolgt über das Protokoll SMTP.
