Voulez-vous transférer un serveur déjà en place ?

La sous-fenêtre « Voulez-vous transférer vers un serveur déjà en place ? » permet de procéder automatiquement à la migration des données et des réglages à partir d’un serveur existant.

  • Pour ignorer la migration, sélectionnez l’option « Configurer un nouveau serveur » et cliquez sur Continuer.

  • Pour procéder à la migration, sélectionnez l’option « Transférer les données à partir d’un serveur existant » et cliquez sur Continuer. Vous pourrez choisir le disque à partir duquel procéder à la migration dans la sous-fenêtre suivante.

Important :  la migration automatique peut s’effectuer uniquement pendant la configuration du serveur. Aucun assistant de migration n’est disponible après la configuration.

Vous pouvez procéder à la migration à partir d’un serveur sur lequel est installé Mac OS X Server 10.4.11 ou la dernière mise à jour de Mac OS X Server 10.5. La migration peut également être réalisée à partir d’une sauvegarde Time Machine d’une configuration standard ou d’une configuration de groupe de travail de la dernière mise à jour de Mac OS X Server 10.5. La source à partir de laquelle la migration est réalisée n’est pas affectée par la migration.

Le serveur à partir duquel la migration est réalisée doit avoir le même nom DNS et la même adresse IP que le serveur en cours de configuration. Vous pourrez modifier l’adresse IP du nouveau serveur une fois la configuration terminée, dans la sous-fenêtre Réseau des Préférences Système.

La sous-fenêtre « Voulez-vous transférer un serveur déjà en place ? » s’affiche pendant la configuration d’un serveur au niveau local et à distance. Elle n’apparaît pas pendant la configuration à distance de plusieurs serveurs.

Pour en savoir plus sur la migration à partir d’un serveur déjà en place, consultez le site Web Mac OS X Server Resources, à l’adresse www.apple.com/fr/server/macosx/resources/ (en anglais).
