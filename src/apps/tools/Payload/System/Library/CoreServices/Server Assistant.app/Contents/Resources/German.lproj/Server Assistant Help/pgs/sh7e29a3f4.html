Mail-Optionen

Geben Sie im Fenster "Mail-Optionen" einen Mail-Relaisserver und eine Einführung zu der Begrüßungsmeldung an, die an neue Benutzer gesendet wird. Der Fenster "Mail-Optionen" wird angezeigt, wenn Sie vorher den Mail-Dienst im Fenster "Dienste" ausgewählt haben.

Mail-Optionen
E-Mail versenden über

Manche Internetanbieter oder Unternehmen erfordern die Weiterleitung aller ausgehenden E-Mail-Nachrichten über einen Relaisserver. Wenn dies in Ihrem Netzwerk der Fall ist, müssen Sie den DNS-Namen des Relaisservers angeben. Sie müssen beispielsweise einen Relaisserver verwenden, wenn die öffentliche IP-Adresse Ihres Servers (oder Ihrer AirPort-Basisstation oder eines anderen Routers) vom DNS-Dienst für Ihren Internetanbieter oder Ihr Unternehmen nicht in den DNS-Namen Ihres Servers aufgelöst werden kann. Dies wird als Reverse-Lookup bezeichnet.

Sie können die E-Mail-Weiterleitung, einschließlich der für manche Relaisserver erforderlichen Identifikation, auch nach der Konfiguration im Fenster "Mail" des Programms "Servereinstellungen" konfigurieren.

Wichtig: Verwenden Sie nur einen von Ihrem Internetanbieter oder Ihrem Unternehmen angegebenen Relaisserver. Der Versuch, E-Mails ohne Erlaubnis über einen anderen Server weiterzuleiten, kann dazu führen, dass Ihr Server als Absender unerwünschter Werbung (Junk-Mails) klassifiziert wird.
SMTP-Weiterleitungsidentifizierung aktivieren

Wählen Sie diese Option aus, wenn Ihr Internetanbieter oder Ihr Unternehmen es erfordert, dass sich Ihr Server vor dem Senden von E-Mails identifiziert, und geben Sie dann den Benutzernamen und das Kennwort von Ihrem Internetanbieter oder Unternehmen ein.

Begrüßungsnachricht an neue Benutzer senden

Wählen Sie diese Option aus, wenn Sie wollen, dass der Server eine E-Mail sendet, die neue Benutzer über seine Dienste informiert. In der Standardnachricht sind der DNS-Name des Servers und eine Erklärung zu den vom Server angebotenen Diensten enthalten.

Sie können eine eigene Einführung zu der Standardnachricht eingeben. Hier einige Beispiele:

Hallo, ich bin der Administrator für unseren Server. Wenn Sie bei der Konfiguration Ihres Computers für die Nutzung der unten genannten Dienste Unterstützung benötigen, stehe ich Ihnen gerne per E-Mail oder unter folgender Nummer zur Verfügung: 310-555-4357. - Markus
