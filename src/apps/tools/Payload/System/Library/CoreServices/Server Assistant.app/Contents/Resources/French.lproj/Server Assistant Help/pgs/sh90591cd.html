Transfert de votre serveur

La sous-fenêtre Transfert de votre serveur vous permet de sélectionner le disque sur lequel se trouvent les données et les réglages que vous souhaitez transférer depuis la copie de sauvegarde serveur ou Time Machine d’un serveur. Le disque sur lequel se trouvent les informations à migrer doit être sur le serveur ou connecté à un FireWire ou un port USB du serveur que vous êtes en train de configurer. Si vous effectuez une migration à partir d’un autre serveur, celui-ci doit être démarré en mode disque cible et connecté à un port FireWire du serveur que vous êtes en train de configurer. Si cela ne s’avère pas possible, vous devez quitter l’Assistant du serveur (en appuyant sur Commande + Q), copier le disque du serveur existant sur un disque amovible pouvant être connecté au serveur que vous êtes en train de configurer, puis démarrer le nouveau serveur pour reprendre la configuration.

Vous pouvez procéder à la migration à partir d’un serveur sur lequel est installé Mac OS X Server 10.4.11 ou la dernière mise à jour de Mac OS X Server 10.5. La migration peut également être réalisée à partir d’une sauvegarde Time Machine d’une configuration standard ou d’une configuration de groupe de travail de la dernière mise à jour de Mac OS X Server 10.5.

Le processus de migration démarre dès que vous appuyez sur Transférer.
