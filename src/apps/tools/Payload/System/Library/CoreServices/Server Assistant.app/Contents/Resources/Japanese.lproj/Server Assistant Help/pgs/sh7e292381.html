BootP を使用してネットワーク接続サービスを構成する

BootP サーバで IP アドレスおよびその他のネットワーク接続設定を割り当てる場合は、「IPv4 の構成」ポップアップメニューから「BootP サーバを参照」を選択します。BootP では、常に同じ IP アドレスが特定のネットワークインターフェイスに割り当てられます。これは主に、NetBoot イメージから起動されるコンピュータ用に使用されます。

Xsan を使用するサーバを設定する場合、Xsan ネットワークには適していないため、「BootP サーバを参照」を選択することはできません。

BootP を使用してネットワーク接続サービスを構成するには：

ネットワーク接続サービスを選択し、「構成」ポップアップメニューから「BootP サーバを参照」を選択した後、以下の情報を参考にして BootP 構成の詳細を入力します：

  • DNS サーバ： 
    DNS サーバの IP アドレス（たとえば、192.0.2.12）、または DNS サーバの IP アドレスのリスト。DNS サーバは、server.example.com のような DNS 名を IP アドレスに変換したり、その逆を行います。
  • 検索ドメイン： 
    完全修飾されていない DNS 名に自動的に付加する名前を指定します。たとえば、検索ドメインとして campus.univ.edu を指定する場合は、「Finder」の「サーバへ接続」ダイアログで「server1」と入力して server1.campus.univ.edu に接続することができます。
