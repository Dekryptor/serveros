Options de messagerie

Utilisez la sous-fenêtre Options de messagerie pour spécifier un serveur relais de messagerie et ajouter une introduction au message d’accueil envoyé aux nouveaux utilisateurs. La sous-fenêtre Options de messagerie apparaît si vous avez sélectionné le service de messagerie dans la sous-fenêtre Services.

Options de messagerie
Relayer le courrier sortant via __

Certains fournisseurs d’accès à Internet ou certaines entreprises font systématiquement transiter les messages sortants par un serveur relais. Si cela s’applique aussi à votre réseau, vous devez spécifier le nom DNS du serveur relais. Par exemple, vous devez utiliser un serveur relais si le service DNS du fournisseur d’accès à Internet ou de l’entreprise ne peut pas résoudre l’adresse IP publique du serveur (ou bien la borne d’accès AirPort ou un autre routeur) et son nom DNS. Il s’agit d’une recherche inversée.

Vous pouvez aussi configurer le relais du courrier, y compris l’authentification requise par certains serveurs relais, dans la sous-fenêtre Courrier de l’application Préférences du serveur une fois l’installation effectuée.

Important :  utilisez uniquement un serveur relais spécifié par le fournisseur d’accès à Internet ou l’entreprise. Si vous tentez de faire transiter des messages par un autre serveur sans autorisation, votre serveur risque d’être considéré comme un expéditeur de courrier indésirable.
Activer l’authentification du relais SMTP

Sélectionnez cette option si le fournisseur d’accès à Internet ou l’entreprise nécessite l’authentification du serveur avant l’envoi de courrier électronique, puis saisissez le nom d’utilisateur et le mot de passe du fournisseur d’accès à Internet ou de l’entreprise.

Envoyer un message de bienvenue aux nouveaux utilisateurs

Sélectionnez cette option si vous souhaitez que le serveur envoie un courrier électronique décrivant les services aux nouveaux utilisateurs. Le message standard spécifie le nom DNS du serveur et détaille les services proposés par le serveur.

Vous pouvez ajouter une introduction personnalisée au message standard. Par exemple :

Bonjour, je suis l’administrateur de votre serveur. Si vous avez besoin d’aide pour accéder à ses services, n’hésitez pas à m’envoyer un courrier électronique ou à me joindre au 310-555-4357. — Paul
