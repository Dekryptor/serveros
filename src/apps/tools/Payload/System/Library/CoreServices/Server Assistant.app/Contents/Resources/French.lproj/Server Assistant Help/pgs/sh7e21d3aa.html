Disque cible

Utilisez la sous-fenêtre Disque cible pour sélectionner le disque ou le volume d’un disque partitionné sur lequel vous souhaitez installer Mac OS X Server. Si une version qualifiée de Mac OS X Server est déjà installée sur le disque ou le volume cible, vous pourrez procéder à sa mettre à niveau. Si une version quelconque de Mac OS X Server ou Mac OS X est installée sur le disque ou le volume cible, vous pourrez effacer et renommer le disque ou le volume avant de réaliser l’installation. Le tableau ci-dessous décrit les actions possibles en fonction des applications déjà installées sur le disque ou le volume cible.

ATTENTION :  avant d’effacer le contenu d’un disque ou d’un volume, veillez à copier sur un autre disque tous les fichiers que vous souhaitez conserver. En effaçant un disque ou un volume, vous détruisez toutes les informations qu’il contient. Si vous effacez un volume d’un disque partitionné, les données figurant sur les autres volumes du même disque ne seront pas effacées.
