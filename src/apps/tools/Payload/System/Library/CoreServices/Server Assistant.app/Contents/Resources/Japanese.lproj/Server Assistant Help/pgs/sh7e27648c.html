ディレクトリサーバに接続

「ディレクトリサーバに接続」パネルでは、ユーザアカウントを読み込むディレクトリサーバを指定します。組織の Open Directory サーバまたは Active Directory サーバから既存のユーザアカウントを読み込むことができます。サーバは、読み込まれたユーザアカウント内にすでに存在するアカウント名、パスワード、およびその他の属性を使用します。サーバは、そのサービスに必要な属性で、既存の属性を補強することができます。たとえば、読み込まれたユーザを、サーバ上で作成するグループのメンバーにすることができます。

ディレクトリサーバから読み込むための設定
サーバアドレス

ポップアップメニューからディレクトリサーバを選択するか、ディレクトリサーバの DNS 名を入力します。Open Directory サーバに接続する場合は、その IP アドレスを入力することができます。

サーバのタイプ

Open Directory または Active Directory を選択できます。

この時点でサーバを設定し、後で使用するための自動設定プロファイルは作成しない場合、「サーバのタイプ」は自動的に決定されます。
ユーザアカウント名とパスワード

これらのフィールドは、ディレクトリサーバが認証を許可または必要としている場合に表示されます。

  • Open Directory サーバの場合は、標準のユーザアカウントのユーザ名とパスワードを入力できるため、ディレクトリ管理者アカウントを使用する必要はありません。

  • Active Directory サーバの場合は、Active Directory 管理者アカウント、または「ワークステーションをドメインに追加する」権限を持つ標準のユーザアカウントの名前とパスワードを入力できます。
