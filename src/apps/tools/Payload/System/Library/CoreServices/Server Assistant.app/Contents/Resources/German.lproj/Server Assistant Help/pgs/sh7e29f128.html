Konfigurieren eines Netzwerkverbindungsdiensts mit DHCP

Wenn Sie alle Netzwerkverbindungseinstellungen, einschließlich der IP-Adresse des Servers, durch den DHCP-Server zuweisen lassen wollen, wählen Sie "DHCP" aus dem Einblendmenü "IPv4 konfigurieren" im Fenster "Netzwerk" aus.

Hinweis: Der DHCP-Server sollte so konfiguriert werden, dass er Ihrem Server eine statische IP-Adresse (immer dieselbe IP-Adresse) zuweist, und der DHCP-Dienst muss aktiv sein.

Wenn Sie einen Server mit Xsan konfigurieren, können Sie "DHCP" nicht wählen, da es für ein Xsan-Netzwerk ungeeignet ist.

Gehen Sie wie folgt vor, um einen Netzwerkverbindungsdienst mit DHCP zu konfigurieren:

Geben Sie nach Auswählen eines Netzwerkverbindungsdiensts und der Option "DHCP" aus dem Einblendmenü "Konfigurieren" die DHCP-Konfigurationsdetails ein. Verwenden Sie hierzu die folgenden Informationen:

  • DNS-Server  
    Die IP-Adresse eines DNS-Servers (z. B. 192.0.2.12) oder eine Liste mit IP-Adressen von DNS-Servern. Gewöhnlich stellt der DHCP-Server die DNS-Serveradressen automatisch bereit. Ein DNS-Server übersetzt DNS-Namen wie server.example.com in IP-Adressen und umgekehrt.
  • Such-Domains  
    Können automatisch von Ihrem DHCP-Dienst bereitgestellt werden. Sie können auch den letzten Teil von einem oder mehr Namen eingeben, der automatisch an unvollständige DNS-Namen angehängt werden soll, die Sie bei Verwendung des Servers eingeben. Wenn Sie z. B. "campus.univ.edu" als Such-Domain angeben, können Sie "server1" in das Finder-Dialogfenster "Mit Server verbinden" eingeben, um die Verbindung zu server1.campus.univ.edu herzustellen.
  • DHCP-Client-ID:  
    Geben Sie den Client-ID-Code ein, wenn Sie einen solchen von Ihrem Internetanbieter oder DHCP-Serveradministrator erhalten haben. Die Client-ID gibt den Server an (oder einen Netzwerkanschluss, wenn der Server mehr als einen Netzwerkanschluss hat). Geben Sie keine DHCP-Client-ID an, wenn Sie den Serverassistenten zum Konfigurieren eines entfernten Servers verwenden. Geben Sie stattdessen nach der Konfiguration eine DHCP-Client-ID in der Systemeinstellung "Netzwerk" auf dem Server ein.
