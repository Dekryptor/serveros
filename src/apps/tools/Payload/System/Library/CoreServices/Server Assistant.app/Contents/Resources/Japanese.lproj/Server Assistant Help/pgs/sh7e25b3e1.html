確認

「確認」パネルでは、設定した内容によって何を実行したいかを指定します。

実行できる操作は以下の通りです：

  • 前の方のパネルを表示したり変更したりするには、「戻る」をクリックします。

  • 画面の指示に従ってサーバ設定を実行するには、「設定」をクリックします。

    「サーバアシスタント」により、設定した内容でサーバが構成されます。また、「Mac OS X Server - 次のステップ」という書類も生成され、サーバのデスクトップ上に配置されます。この書類は、お使いのサーバとネットワークに特有の情報が含まれています。これには、新しいサーバのセキュリティ、ユーザ補助、および全体的な利便性を強化するために行う必要のある追加の変更について記載されています。

  • 構成の詳細を表示するか、行った設定を保存するには、「詳細」をクリックします。

    要約はテキストファイルとして保存でき、設定は自動設定プロファイルとして保存できます。詳しくは、「自動設定プロファイルを保存する」を参照してください。
