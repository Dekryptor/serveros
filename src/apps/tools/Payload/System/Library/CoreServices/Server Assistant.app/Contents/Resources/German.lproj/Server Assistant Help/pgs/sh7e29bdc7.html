Konfigurieren eines Ethernet-Netzwerkverbindungsdiensts

Wenn Sie Ethernet für die Verbindung zu einem lokalen Netzwerk oder ein DSL- oder Kabelmodem für den Internetzugang verwenden, müssen Sie die Verbindungseinstellungen für Ihren Server eingeben. Wenn Sie diese nicht kennen, fragen Sie Ihren Netzwerkadministrator oder Internetanbieter.

Wenn Sie eine Verbindung zu einem lokalen Netzwerk herstellen, stellen Sie sicher, dass Ihr Server über ein Ethernetkabel mit dem Ethernetgerät (wie ein Switch oder Router) im Netzwerk verbunden ist.

Wenn Sie eine Verbindung zu einem DSL- oder Kabelmodem herstellen, stellen Sie sicher, dass es über ein Ethernetkabel mit dem Ethernetanschluss Ihres Servers verbunden ist. Ein DSL-Modem muss auch mit dem zugehörigen Kabel an die Telefonbuchse angeschlossen werden. Ein Kabelmodem muss auch mit dem vom Kabelunternehmen installierten Koaxialkabel verbunden werden. Schalten Sie das DSL- oder Kabelmodem ein und warten Sie ab, bis es vollständig gestartet wurde.

Gehen Sie wie folgt vor, um Ethernet zu konfigurieren:

  1. Wählen Sie einen Ethernetdienst in der Liste links aus.

  2. Klicken Sie auf "Konfigurieren", wählen Sie die von Ihrem Netzwerkadministrator oder Internetanbieter empfohlene Konfigurationsmethode und geben Sie die Konfigurationsdetails ein.

    Entsprechende Anweisungen finden Sie in den folgenden Hilfethemen:

    Konfigurieren eines Netzwerkverbindungsdiensts mit DHCP
    Konfigurieren eines Netzwerkverbindungsdiensts mit DHCP und manueller Adresse
    Konfigurieren eines Netzwerkverbindungsdiensts mit BootP
    Manuelles Konfigurieren eines Netzwerkverbindungsdiensts

  3. Wenn Ihr lokales Netzwerk oder Ihr Internetanbieter eine Anpassung der erweiterten Etherneteinstellungen oder der erweiterten IPv6-Adresseinstellungen erfordert, klicken Sie auf die entsprechende Taste unter dem Feld "Such-Domains". Näheres finden Sie in den folgenden Hilfethemen:

    Anpassen von Ethernetoptionen
    Anpassen von IPv6-Optionen

    Standardmäßig konfiguriert der Server automatisch den Ethernetanschluss und generiert eine IPv6-Adresse.
