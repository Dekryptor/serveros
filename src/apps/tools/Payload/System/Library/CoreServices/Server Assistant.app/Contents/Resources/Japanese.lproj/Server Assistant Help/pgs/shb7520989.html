ホーム
インターフェイス名またはハードウェア・アドレスを見つける

自動サーバ設定プロファイルを作成していて、「ネットワーク」パネルでネットワーク接続サービスを追加したいが、ネットワークに接続されているサーバのインターフェイス名（BSD ポート）またはハードウェア・アドレス（MAC アドレス）が分からない場合は、コマンドラインツールを使用して情報を取得することができます。リモートサーバの IP アドレスまたは DNS 名と、そのハードウェアのシリアル番号が分かっている必要があります。ssh を使用してコマンドラインからリモートサーバにログインし、次に networksetup を使用してネットワークポート情報を取得します。

リモートサーバのネットワークインターフェイス情報を取得するには：

  1. リモートサーバがすでに起動していて、ネットワークに接続されていることを確認します。

    Mac OS X Server がまだインストールされていない場合は、Mac OS X Server Install Disc を使ってリモートサーバを起動できます。

  2. お使いの Mac で「ターミナル」を開き、次のコマンドを入力してから Return キーを押してリモートサーバにログインします：

    ssh root@serveraddress

    serveraddress はリモートサーバの IP アドレスまたは DNS 名と置き換えてください。

  3. リモートサーバの真正性を証明できない場合でも接続を続けたいかどうか確認された場合は、「yes」と入力します。

    これは、リモートサーバにはじめてログインするときに確認されます。

  4. パスワードの入力を求められたら、リモートサーバのシリアル番号の最初の 8 文字を入力します。

  5. 次のコマンドを入力して、Return キーを押します：

    networksetup -listallhardwareports

    このコマンドの出力は、インターフェイス名を「Device」、ハードウェア・アドレスを「Ethernet Address」として識別します。
