Adjusting Ethernet Options

Some Ethernet networks require setting advanced options. In most cases, the settings that are configured automatically are correct. Incorrect Ethernet port settings can affect network performance or render a port unusable. Your network administrator or Internet service provider should be able to tell you if you need to adjust advanced settings.

To configure advanced Ethernet settings:

Select an Ethernet service, click the Ethernet button, and set the options using the following information:

  • Speed:  
    Specifies the maximum number of bits per second that the Ethernet port can send or receive. If your computer and the device you are connecting to support Gigabit Ethernet but the Ethernet cable you are using does not, you may need to adjust the Ethernet speed. Alternatively, you can purchase a cable that supports Gigabit Ethernet.
  • Duplex:  
    Determines whether input and output packets are transmitted at the same time (full-duplex) or alternately (half-duplex).
  • MTU:  
    Is the largest packet the port will send or receive. (MTU stands for maximum transmission unit, expressed in bytes.) Increasing the packet size improves throughput (for example, for file transfers), but the devices that receive the packet (switches, routers, and so forth) must support the packet size.
