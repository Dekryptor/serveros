Région

Utilisez la sous-fenêtre Région pour sélectionner la zone géographique dont le serveur doit adopter l’heure, la date et les formats numériques. Les formats régionaux ont une incidence sur les fenêtres du Finder et sur les applications du serveur.

Par défaut, seules les régions associées à la langue sélectionnée lors de l’installation sont affichées. Si la région souhaitée n’est pas disponible, sélectionnez Tout afficher.

Après la configuration, vous pourrez changer de région dans la sous-fenêtre International de l’application Préférences Système.

La sous-fenêtre Région ne s’affiche pas en cas de configuration locale.
