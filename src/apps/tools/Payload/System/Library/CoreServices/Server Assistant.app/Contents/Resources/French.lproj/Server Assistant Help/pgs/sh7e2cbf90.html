Configuration d’un service de connexion réseau de port LOM

Les systèmes Xserve équipés de processeurs Intel incluent un processeur LOM (Lights-Out Management) distinct qui vous permet de communiquer avec le serveur même lorsque celui-ci est éteint. Le processeur LOM dispose de ses propres réglages réseau pour les ports Ethernet intégrés du serveur.

Pour configurer un service de connexion réseau de port LOM :

  1. Sélectionnez un service de port LOM dans la liste de gauche.

  2. Si vous y êtes invité, saisissez le nom et le mot de passe de l’administrateur LOM.

    Le compte d’administrateur LOM est différent du compte d’administrateur Mac OS X Server, mais vous pouvez utiliser les mêmes nom et mot de passe, si vous le souhaitez.

  3. Cliquez sur Configurer, choisissez une méthode de configuration, puis saisissez les informations de configuration à l’aide des données ci-dessous.

    • Adresse IP :  
      Une adresse numérique qui identifie de façon unique le serveur sur le réseau. Elle se présente sous la forme de quatre nombres séparés par des points (par exemple, 192.0.2.123).

      Attribuez une adresse IP spécifique à l’interface LOM. N’utilisez pas la même adresse que celle affectée aux ports Ethernet 1 ou Ethernet 2 intégrés, bien que le même connecteur physique soit utilisé (port 1 ou 2) à l’arrière de Xserve.

    • Masque de sous-réseau :  
      Un groupe de quatre nombres qui identifie la partie réseau de l’adresse IP du serveur ; le reste de l’adresse IP identifie de façon unique le serveur sur ce réseau.
    • Routeur :  
      L’adresse IP du routeur pour le réseau local du serveur (par exemple, 192.168.99.1 ou 10.0.7.1). Le routeur est un périphérique qui réexpédie les paquets de données du réseau local vers leur destination sur d’autres réseaux ou sur Internet. Si vous ne disposez pas de routeur sur votre réseau, saisissez l’adresse IP du serveur.

Vous pouvez modifier le nom ou le mot de passe de l’administrateur LOM en cliquant sur Compte admin.

Après avoir configuré le serveur, vous pouvez changer les réglages réseau du port LOM et ajouter ou modifier les comptes d’administrateur LOM à l’aide de Contrôle de serveur. Pour obtenir des instructions, ouvrez Contrôle de serveur (dans /Applications/Serveur/), puis utilisez le menu Aide. Vous pouvez aussi modifier ces réglages à partir de la ligne de commande en utilisant IPMITool dans Terminal. Pour en savoir plus, consultez la rubrique man IPMITool.
