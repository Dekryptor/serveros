Serial Number

Use the Serial Number pane to enter a serial number for each server you’re setting up now.

You must use a Mac OS X Server v10.6 Snow Leopard serial number, which begins with XSVR-106. You’ll find the serial number printed on the materials provided with the Mac OS X Server package. Unless you have a site license, you need a unique serial number for each server. You can’t use a serial number from an earlier version of Mac OS X Server to register Snow Leopard Server.

If you have a site license, you must enter a registered owner name and organization exactly as specified by your Apple representative.

If you’re setting up multiple servers, the Serial Number pane lists the servers that have valid serial numbers. The number of servers without serial numbers is displayed below the list. To add a serial number, click Add. Server Assistant applies the serial number to a server that currently doesn’t have one. You can also add a site license, and Server Assistant will apply it to multiple servers.
