AirMac マネージメント

「AirMac マネージメント」パネルでは、ローカルネットワークを保護しつつ、インターネットから一部のサービスにアクセスできるように、Mac OS X Server を使用して AirMac Extreme ベースステーション（802.11n）または Time Capsule を管理できるようにします。AirMac マネージメントを有効にするには：

  • ベースステーションまたは Time Capsule のパスワードを入力する必要があります（ワイヤレスネットワークのパスワードではありません）。

  • サーバが AirMac Extreme ベースステーションまたは Time Capsule のほかにも接続できる場合は、サーバが管理する対象の名前が分かっている必要があります。

初期設定の後、「サーバ環境設定」を使用して、ローカルネットワークの外からアクセスできるようにしたい個々のサービスを指定します。Mac OS X Server は、AirMac Extreme ベースステーションまたは Time Capsule で、それらのサービスの着信要求をサーバに渡すことができるように構成します。

自分の AirMac ベースステーション、Time Capsule、またはその他のルーターがインターネット接続を共有するように構成しているユーザに、サーバが AirMac Extreme ベースステーションまたは Time Capsule 経由で VPN サービスを提供する場合は、サーバの IP アドレスを VPN ユーザの IP アドレス（10.0.1 や 192.168.1 など）と同じ 3 つの番号で始めることはできません。IP アドレスの競合の回避については、Mac OS X Server に付属のマニュアル「お使いになる前に」を参照してください。

「AirMac マネージメント」パネルは、サーバのプライマリ・ネットワーク・インターフェイスが AirMac ベースステーションまたは Time Capsule 経由でインターネットに接続されている場合に表示されます。さらに、AirMac Extreme ベースステーションまたは Time Capsule の「接続共有」オプションが「パブリック IP アドレスを共有」（つまり、インターネット接続）に設定されている必要があります。

「AirMac マネージメント」パネルは、複数のサーバのリモート設定中は表示されません。
