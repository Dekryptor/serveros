Xsan をインストール

「Xsan をインストール」パネルでは、サーバ上への「Xsan」ソフトウェア（Xsan ファイルシステムと「Xsan Admin」アプリケーションを含む）のインストールを今すぐ開始するか、後からインストールするかを選択できます。「Xsan をインストール」パネルは、前のパネルでサーバを Xsan メタデータコントローラとして設定するよう選択した場合に表示されます。

「Xsan」をサーバにインストールする前に、SAN ストレージとネットワークをあらかじめ設定し、「Xsan」を各クライアントコンピュータにインストールする必要があります。プライマリ・メタデータ・コントローラであるサーバに「Xsan」をインストールする前には、最初にセカンダリー・メタデータ・コントローラであるサーバに「Xsan」をインストールする必要があります。
