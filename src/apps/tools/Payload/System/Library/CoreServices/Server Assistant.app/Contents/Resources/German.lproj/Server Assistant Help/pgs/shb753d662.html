Konfiguration sichern

Sichern Sie im Fenster "Konfiguration sichern" ein Profil für die automatische Konfiguration, das die von Ihnen eingegebenen Konfigurationsdaten enthält. Sie können es auf einem beliebigen Volume sichern, auf das der Server Zugriff hat. Sie können es beispielsweise auf einem Wechsellaufwerk oder einer CD sichern. Wenn der Server startet und der Begrüßungsbildschirm angezeigt wird, können Sie das Laufwerk anschließen oder die CD einlegen, um mit der automatischen Konfiguration zu beginnen.

Wenn Sie auf "Sichern" klicken, können Sie für das Profil einen beliebigen Dateinamen wählen, dieser muss jedoch die Erweiterung .plist haben. Sie können eine Variante des Profils sichern, indem Sie auf "Zurück" klicken, um einige Einstellungen zu ändern, und dann auf "Fortfahren" klicken, um zu diesem Fenster zurückzukehren, und erneut auf "Sichern" klicken.

Einstellungen für ein Profil für die automatische Konfiguration
Nutzung auf bestimmte Computer einschränken

Wählen Sie diese Option aus und klicken Sie auf "Bearbeiten", um Bedingungen anzugeben, die festlegen, welche Server automatisch mit diesem Profil für die automatische Konfiguration konfiguriert werden können.

Sie können beliebige der folgenden Bedingungen angeben und dabei jeweils mehrere Bedingungen eines Typs hinzufügen:

  • Seriennummer: Eine Hardwareseriennummer.

  • Hostname: Der vollständige oder teilweise DNS-Name.

  • IP-Adresse

  • MAC-Adresse: Die Hardwareadresse (Ethernet-ID) einer Netzwerkschnittstelle.

Sie können auch wählen, wie eine Bedingung mit Attributen eines berechtigten Computers verglichen wird. Die Vergleiche bedürfen keiner Erklärung, mit Ausnahme der folgenden:

  • Ist: Das Attribut eines berechtigten Computers muss diese Bedingung genau erfüllen.

  • Ist ähnlich:  Bedeutet, dass "?" und "*" Platzhalterzeichen in der Bedingung sind, wobei "?" mit einem beliebigen einzelnen Zeichen und "*" mit null oder mehr Zeichen im Attribut eines berechtigten Computers übereinstimmen.

  • Entspricht: Das Attribut eines berechtigten Servers muss die Wörter oder Zeichenmuster enthalten, die durch Auswertung der Bedingung als regulärer Ausdruck angegeben werden (weitere Informationen hierzu finden Sie im Abschnitt zu regulären Ausdrücken auf der Website mit dem ICU-Benutzerhandbuch unter der Adresse www.icu-project.org/userguide/regexp.html).

Wenn Sie mehrere Bedingungen angeben, wird das Profil auf einen Computer angewendet, der beliebige dieser Bedingungen erfüllt.

Wenn ein berechtigter Computer mehrere DNS-Namen, IP-Adressen oder MAC-Adressen hat, werden alle mit anwendbaren Bedingungen verglichen. Zum Beispiel wird eine Bedingung, die eine Hardwareadresse angibt, mit allen Netzwerkanschlüssen eines Computers verglichen.

Verschlüsselung

Wählen Sie, ob das Profil für die automatische Serverkonfiguration in verschlüsselter Form oder als Klartext gesichert werden soll.

  • Wenn Sie die Kennwortverschlüsselung wählen, geben Sie ein Kennwort für die Ver- und Entschlüsselung des Profils für die automatische Konfiguration ein.

    Sie können das Kennwort erneut eingeben, wenn die Konfiguration beginnt. Sie können aber auch eine Textdatei mit dem Kennwort in demselben Ordner wie das Profil für die automatische Konfiguration sichern und der Kennwortdatei denselben Namen wie das Profil für die automatische Konfiguration geben, allerdings mit der Erweiterung .pass.

  • Wenn Sie "Ohne" wählen, wird das Profil im Klartext gespeichert.

Wichtig: Wählen Sie nicht "Ohne", wenn das Profil öffentlich zugänglich sein könnte. Das Profil enthält das Kennwort des Administrator-Accounts des Servers, das dazu missbraucht werden könnte, die Kontrolle über einen Server zu übernehmen, der mithilfe des Profils konfiguriert wird.

Möchten Sie ein gesichertes Profil verwenden, kopieren Sie es in einen Ordner namens "Auto Server Setup" auf der obersten Ebene eines beliebigen mit dem Computer, den Sie automatisch konfigurieren wollen, verbundenen Volumes. Das Volume kann sich auf der Festplatte des Servers befinden oder ein iPod-, ein CD- oder DVD-Laufwerk, ein FireWire-Laufwerk, ein USB-Laufwerk oder jedes andere an den Computer angeschlossene Speichergerät sein.

Wenn Sie einen Computer starten, auf dem Mac OS X Server installiert, aber nicht konfiguriert ist, sucht der Serverassistent automatisch auf allen verfügbaren Volumes in Ordnern mit dem Namen "Auto Server Setup" nach gültigen Profilen für die automatische Konfiguration. Wenn mehr als ein Profil gefunden wird, wird das Profil verwendet, das mit dem konfigurierten Computer am genauesten übereinstimmt. Zur Ermittlung der genauesten Übereinstimmung werden die Bedingungen in folgender Reihenfolge verglichen: Hardwareseriennummer, MAC-Adresse der Netzwerkschnittstelle (Hardwareadresse), IP-Adresse, DNS-Name, Computername, keine Bedingungen. Wenn ein Profil mehrere Bedingungen enthält, wird es auf einen Computer angewendet, der beliebige dieser Bedingungen erfüllt.
