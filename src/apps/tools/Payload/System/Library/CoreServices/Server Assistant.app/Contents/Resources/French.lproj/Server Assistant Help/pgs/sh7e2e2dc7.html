Fuseau horaire

Utilisez la sous-fenêtre Fuseau horaire afin de sélectionner le fuseau horaire local du serveur.

Choisissez une ville en cliquant sur la carte, en renseignant le champ « Ville la plus proche » ou en utilisant le menu local. Si la ville dans laquelle est hébergé le serveur est introuvable, sélectionnez une ville du même fuseau horaire.

Pour modifier le serveur d’horloge de réseau, cliquez sur le bouton Modifier. Sélectionnez « Utiliser un serveur d’horloge de réseau ». Identifiez le serveur d’horloge de réseau à utiliser en le choisissant dans le menu local du serveur NTP ou en saisissant l’adresse IP ou le nom DNS. Pour le bon fonctionnement du serveur, Apple recommande de s’assurer que son horloge est exacte en la synchronisant avec un serveur d’horloge de réseau.
