Installationssprache

Verwenden Sie das Fenster "Installationssprache", um die Sprache auszuwählen, die von Mac OS X Server verwendet werden soll.

Die ausgewählte Sprache wirkt sich auf die Serververwaltungsprogramme und die vom AFP-Dienst (Apple Filing Protocol) verwendete Standardcodierung aus. Sie wirkt sich nicht auf die Sprache der Computer von Benutzern aus. Zum Beispiel können Sie einen Server auf Englisch verwalten, der Benutzer unterstützt, deren Computer Japanisch verwenden.
