Installation de Xsan

Utilisez la sous-fenêtre Installation de Xsan pour lancer maintenant l’installation du logiciel Xsan sur le serveur, y compris le système de fichiers Xsan et l’application Xsan Admin, ou remettez l’installation de Xsan à plus tard. La sous-fenêtre Installation de Xsan s’affiche si vous avez choisi de configurer le serveur comme contrôleur de métadonnées Xsan dans une sous-fenêtre précédente.

Avant de procéder à l’installation de Xsan sur le serveur, le stockage et les réseaux du SAN doivent être configurés et Xsan doit être installé sur tous les ordinateurs clients. Vous devez installer Xsan sur les serveurs qui font office de contrôleurs de métadonnées secondaires avant de procéder à son installation sur le serveur faisant office de contrôleur de métadonnées principal.
