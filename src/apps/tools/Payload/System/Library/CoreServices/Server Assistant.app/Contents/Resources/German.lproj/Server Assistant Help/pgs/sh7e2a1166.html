Benutzer und Gruppen

Die im Fenster "Benutzer und Gruppen" gewählte Option gibt an, wie Sie die Benutzer- und Gruppen-Accounts verwalten, mit denen der Server Benutzer identifiziert und bestimmt, auf welche Dienste sie zugreifen dürfen.

Optionen zum Verwalten von Benutzern und Gruppen
Benutzer und Gruppen erstellen

Konfiguriert einen unabhängigen Server für ein kleines Unternehmen ohne einen vorhandenen Verzeichnisserver. Der Server stellt einen eigenen Verzeichnisdienst als Open Directory-Master bereit, sodass der Server über eigene Benutzer- und Gruppen-Accounts verfügt. Im nächsten Fenster können Sie grundlegende Dienste auswählen, die konfiguriert werden sollen.

Benutzer und Gruppen importieren

Konfiguriert einen Server für eine Arbeitsgruppe in einem Unternehmen mit einem vorhandenen Verzeichnisserver. Der Server kann Benutzer von diesem Verzeichnisserver importieren. Er kann Benutzer einzeln importieren, und er kann automatisch Mitglieder von Gruppen importieren. Sie verbinden den Server im nächsten Fenster mit dem Verzeichnisserver. Der Server stellt auch einen eigenen Verzeichnisdienst als Open Directory-Master bereit und kann so über eigene Benutzer- und Gruppen-Acounts verfügen. Später können Sie grundlegende Dienste auswählen, die konfiguriert werden sollen.

Manuell konfigurieren

Erfahrene Systemadministratoren können die Dienstkonfiguration vollständig steuern, um so einer breiten Palette von Unternehmensanforderungen gerecht zu werden.

Im nächsten Fenster können Sie Informationen eingeben, um den Server mit einem Verzeichnisserver zu verbinden oder einen Open Directory-Master zu erstellen und diesen als Verzeichnisserver zu verwenden. Wenn Sie eine dieser Optionen wählen, können Sie auch grundlegende Dienste auswählen, die konfiguriert werden sollen. Wenn Sie keine Verbindung zu einem Verzeichnisserver herstellen oder keinen Open Directory-Master erstellen, konfigurieren Sie Verzeichnisdienste und alle anderen Dienste nach der Konfiguration selbst.

Hinweis: Wenn der Server nicht mit einem Netzwerk verbunden ist, wählen Sie "Manuell konfigurieren". Mit den anderen zwei Optionen werden die Verzeichnisdienste des Servers konfiguriert, die eine Netzwerkverbindung erfordern.

Wenn diese Optionen im Fenster "Benutzer und Gruppen" nicht angezeigt werden, konfigurieren Sie Mac OS X Server für einen primären Xsan-Metadaten-Controller. Weitere Informationen hierzu finden Sie unter Benutzer und Gruppen (Xsan).
