Hinzufügen eines Netzwerkverbindungsdiensts

Sie können Netzwerkverbindungsdienste im Fenster "Netzwerk" hinzufügen, löschen, anders anordnen sowie ein- oder ausblenden. Ein Netzwerkverbindungsdienst ist eine Gruppe von Einstellungen für einen bestimmten Netzwerkanschluss oder eine bestimmte Netzwerkschnittstelle (ein physischer Netzwerkanschluss wie z. B. der Ethernetanschluss oder die AirPort-Karte Ihres Computers).

Gehen Sie wie folgt vor, um einen Netzwerkverbindungsdienst hinzuzufügen:

  1. Wenn der gewünschte Netzwerkverbindungsdienst angezeigt wird, nachdem Sie "Alle Dienste einblenden" aus dem Aktionsmenü (Zahnrad) ausgewählt haben, fahren Sie mit Schritt 5 fort und konfigurieren Sie den angezeigten Netzwerkverbindungsdienst, statt einen neuen hinzuzufügen.

    Wenn das Aktionsmenü (Zahnrad) "Alle Dienste einblenden" nicht enthält, werden bereits alle Netzwerkverbindungsdienste angezeigt. Fahren Sie mit dem folgenden Schritt fort, wenn Sie einen neuen Dienst hinzufügen müssen.

  2. Klicken Sie auf die Taste "Hinzufügen" (+) unten in der Liste.

  3. Wählen Sie eine Schnittstelle aus dem Einblendmenü "Typ" aus, etwa Ethernet oder AirPort, und geben Sie dem neuen Netzwerkverbindungsdienst einen aussagekräftigen Namen.

  4. Wenn Sie ein Profil für die automatische Konfiguration erstellen (also jetzt keinen Server konfigurieren), geben Sie die vom Netzwerkverbindungsdienst verwendete physische Schnittstelle (Anschluss) an. Geben Sie dazu entweder den BSD-Namen oder die Hardwareadresse (auch MAC-Adresse genannt) ein.

    Der erste integrierte Ethernetanschluss hat den Namen en0. Weitere Informationen hierzu finden Sie unter Ermitteln eines Schnittstellennamens oder einer Hardwareadresse.

  5. Klicken Sie auf "Erstellen", wählen Sie den neuen Dienst in der Liste aus und geben Sie geeignete Einstellungen dafür ein.

    Hinweis: Sie können nicht auf "Erstellen" klicken, wenn der Dienstname mit einem vorhandenen Dienstnamen identisch ist.

    Entsprechende Anweisungen finden Sie in den folgenden Hilfethemen.

Möchten Sie die Reihenfolge der Netzwerkverbindungsdienste ändern, wenn mehr als einer aufgeführt wird, bewegen Sie sie in der Liste. Bewegen Sie die öffentliche Netzwerkverbindung in der Liste nach oben und ordnen Sie die anderen in der Reihenfolge an, in der sie vom Server verwendet werden sollen.

Sollen alle Netzwerkverbindungsdienste oder nur bevorzugte angezeigt werden, verwenden Sie das Aktionsmenü (Zahnrad).

Möchten Sie einen Netzwerkverbindungsdienst löschen, wählen Sie ihn in der Liste aus und klicken Sie auf die Taste "Löschen" (-).
