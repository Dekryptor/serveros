Bienvenue

Le logiciel Mac OS X Server est installé sur cet ordinateur et l’Assistant du serveur est prêt à vous guider à travers la procédure de configuration initiale visant à faire fonctionner le serveur.

Si l’ordinateur dispose d’une nouvelle installation de Mac OS X Server, vous pouvez la configurer de façon interactive ou bien lancer une configuration automatique à l’aide d’un profil de configuration enregistré.

Si vous avez mis à niveau l’ordinateur à partir d’une ancienne version de Mac OS X Server, il vous suffit juste d’indiquer un numéro de série pour Mac OS X 10.6 Snow Leopard. Tous les autres réglages de serveur restent inchangés. L’Assistant du serveur migre toutes les données système nécessaires aux services mis à niveau.

Pour configurer un nouveau serveur de façon interactive :

Sélectionnez la zone géographique dont le serveur doit adopter l’heure, la date et les formats numériques, puis cliquez sur Continuer.

Par défaut, seules les régions associées à la langue sélectionnée lors de l’installation sont affichées. Si la région souhaitée n’est pas disponible, sélectionnez Tout afficher.

Les formats régionaux ont une incidence sur les fenêtres du Finder et sur les applications du serveur. Après la configuration, vous pourrez changer de région dans la sous-fenêtre International de l’application Préférences Système.

Pour configurer un nouveau serveur de façon automatique :

Si vous préférez utiliser un profil de configuration automatique du serveur créé précédemment et enregistré sur un disque ou un lecteur amovible, connectez ce lecteur au serveur ou insérez le disque dans le serveur.

L’Assistant du serveur recherche un profil dans un dossier nommé « Configuration de serveur automatique », à la racine de tous les disques disponibles. Si le profil est associé à des conditions restrictives, l’Assistant du serveur l’utilise si le serveur répond à l’une de ces conditions. Si l’Assistant détecte plusieurs profils pour lesquels le serveur répond aux conditions définies, il opte pour celui qui correspond le mieux. Pour en savoir plus sur ces conditions, consultez la rubrique Enregistrement d’un profil de configuration automatique.

Pour en savoir plus sur la configuration automatique du serveur, reportez-vous à la Feuille d’opération pour l’installation et la configuration(celle-ci se trouve dans le dossier Documentation du disque d’installation de Mac OS X Server).

Une fois la configuration interactive ou automatique du serveur terminée, il est possible que vous deviez effectuer quelques tâches de configuration complémentaires. Recherchez le document Étapes suivantes de Mac OS X Server, qui est généré et placé sur le bureau du serveur après la configuration, puis lisez-le pour obtenir des conseils spécifiques.

Avant d’utiliser le serveur, vous pouvez également configurer certains services supplémentaires, modifier des réglages de service ou gérer les utilisateurs et les groupes. Pour cela, vous pouvez faire appel aux outils Préférences du serveur, Admin Serveur, Gestionnaire de groupe de travail ou à toute autre application disponible dans le dossier /Applications/Server/. Vous pouvez également utiliser les outils de ligne de commande UNIX. Pour en savoir plus sur les applications et les outils du serveur, consultez le manuel Premiers contacts fourni avec Mac OS X Server.
