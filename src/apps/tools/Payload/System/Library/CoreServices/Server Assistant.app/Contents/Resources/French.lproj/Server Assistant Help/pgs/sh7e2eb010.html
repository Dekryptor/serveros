Xsan

La sous-fenêtre Xsan apparaît si le serveur que vous configurez est connecté à un réseau Fibre Channel. Si le réseau Fibre Channel est configuré pour un réseau de stockage local (SAN) et que vous disposez du disque d’installation de Xsan, vous pouvez choisir de configurer le serveur maintenant en tant que contrôleur de métadonnées Xsan. Dans la sous-fenêtre suivante, vous sélectionnez le type de contrôleur de métadonnées Xsan que vous souhaitez configurer, principal ou secondaire. Ensuite, une fois la configuration de Mac OS X Server terminée, vous êtes invité à insérer le disque d’installation de Xsan. Si le logiciel Xsan est déjà installé sur le serveur, vous n’aurez pas à insérer le disque d’installation Xsan ultérieurement.

Xsan est un système de fichiers SAN haute performance facile à utiliser conçu pour Mac OS X et Mac OS X Server. Xsan vous permet de consolider les périphériques de stockage et offre aux clients un accès Fibre Channel simultané aux fichiers situés sur des volumes partagés. Xsan inclut des fonctions avancées qui optimisent la capacité de stockage du réseau, notamment le basculement du contrôleur de métadonnées, le multi-acheminement Fibre Channel et la journalisation du système de fichiers. Vous pouvez utiliser l’application Xsan Admin pour configurer, contrôler et administrer votre SAN. Pour en savoir plus, consultez le Guide de configuration et le Guide de l’administrateur disponibles dans le dossier Documentation du disque d’installation Xsan.
