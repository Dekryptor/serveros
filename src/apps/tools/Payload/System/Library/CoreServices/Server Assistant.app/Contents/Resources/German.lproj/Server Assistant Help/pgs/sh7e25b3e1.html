Prüfen

Geben Sie im Fenster "Prüfen" an, was mit den Einstellungen geschehen soll, die Sie vorgenommen haben.

Sie haben folgende Möglichkeiten:

  • Klicken Sie auf "Zurück", um vorherige Fenster anzuzeigen oder Änderungen darin vorzunehmen.

  • Klicken Sie auf "Konfigurieren", um die interaktive Serverkonfiguration zu beenden.

    Der Serverassistent konfiguriert den Server mit den Einstellungen, die Sie vorgenommen haben. Er generiert auch das Mac OS X Server-Dokument "Weitere Schritte" und bewegt es auf den Schreibtisch des Servers. Dieses Dokument ist auf Ihren Server und Ihr Netzwerk zugeschnitten. Es informiert Sie über zusätzliche Änderungen, die Sie vornehmen sollten, um die Sicherheit, Bedienung und Leistungsfähigkeit Ihres neuen Servers zu verbessern.

  • Klicken Sie auf "Details", um Konfigurationsdetails anzuzeigen oder die vorgenommenen Einstellungen zu sichern.

    Sie können eine Zusammenfassung als Textdatei sichern. Zudem können Sie die Einstellungen als Profil für die automatische Konfiguration sichern. Weitere Informationen hierzu finden Sie unter Sichern eines Profils für die automatische Konfiguration.
