Willkommen

Auf diesem Computer ist die Mac OS X Server-Software installiert, und der Serverassistent führt Sie jetzt durch die anfängliche Konfiguration, um den Server in Betrieb zu nehmen.

Wenn der Computer über eine neue Installation von Mac OS X Server verfügt, können Sie diese interaktiv konfigurieren oder die automatische Konfiguration mit einem gesicherten Konfigurationsprofil starten.

Wenn Sie den Computer von einer früheren Version von Mac OS X Server aktualisiert haben, müssen Sie nur eine Seriennummer für Mac OS X Version 10.6 Snow Leopard eingeben. Alle übrigen Servereinstellungen bleiben unverändert. Der Serverassistent führt die gesamte Migration von Systemdaten durch, die für aktualisierte Dienste erforderlich ist.

Gehen Sie wie folgt vor, um einen neuen Server interaktiv zu konfigurieren:

Wählen Sie die Region aus, deren Zeit-, Datums- und Zahlenformate der Server verwenden soll, und klicken Sie dann auf "Fortfahren".

Standardmäßig werden nur Regionen für die bei der Installation ausgewählte Sprache angezeigt. Wenn die gewünschte Region nicht angezeigt wird, wählen Sie "Alle einblenden".

Die Regionsformate wirken sich auf Finder-Fenster und Programme auf dem Server aus. Nach der Konfiguration können Sie die Region in der Systemeinstellung "Sprache & Text" ändern.

Gehen Sie wie folgt vor, um einen neuen Server automatisch zu konfigurieren:

Wenn Sie ein Profil für die automatische Serverkonfiguration verwenden wollen, das Sie zuvor erstellt und auf einem Wechsellaufwerk oder einer CD/DVD gesichert haben, schließen Sie jetzt das Laufwerk an den Server an oder legen Sie die CD/DVD in den Server ein.

Der Serverassistent sucht in einem Ordner mit Namen "Automatische Serverkonfiguration" auf der obersten Ebene aller verfügbaren Volumes nach Profilen. Wenn das Profil einschränkende Bedingungen enthält, wird es vom Serverassistenten verwendet, wenn der Server beliebige dieser Bedingungen erfüllt. Wenn der Serverassistent mehrere Profile findet, deren Bedingungen der Server erfüllt, verwendet er das Profil, das am genauesten passt. Weitere Informationen zu diesen Bedingungen finden Sie unter Sichern eines Profils für die automatische Konfiguration.

Weitere Informationen zur automatischen Serverkonfiguration finden Sie im Dokument Arbeitsblatt für Installation & Konfiguration.(Dieses befindet sich im Ordner "Dokumentation" auf der DVD Mac OS X Server Installation).

Nach Beendigung der interaktiven oder automatischen Serverkonfiguration können zusätzliche Konfigurationsaufgaben erforderlich sein. Schauen Sie im Mac OS X Server-Dokument "Weitere Schritte" nach, das nach der Konfiguration auf dem Schreibtisch des Servers generiert wird. In diesem Dokument werden die weiteren erforderlichen Schritte genannt.

Bevor Sie den Server verwenden, können Sie zusätzliche Dienste konfigurieren, bestimmte Diensteinstellungen ändern oder Benutzer und Gruppen verwalten. Sie können die Programme "Servereinstellungen", "Server-Admin", "Arbeitsgruppenmanager" und weitere Programme im Ordner "/Programme/Server/" verwenden. Sie können auch UNIX-Befehlszeilenprogramme verwenden. Weitere Informationen zu den Serverprogrammen und -werkzeugen finden Sie im Handbuch Einführung, das zum Lieferumfang von Mac OS X Server gehört.
