Configuring an Ethernet Network Connection Service

If you use Ethernet to connect to a local network or to a DSL or cable modem that connects to the Internet, you need to enter the connection settings for your server. If you’re not sure what they are, ask your network administrator or Internet service provider (ISP).

If you’re connecting to a local network, make sure your server is connected to the network Ethernet device (such as a switch or router) with an Ethernet cable.

If you’re connecting to a DSL or cable modem, make sure it’s connected to your server’s Ethernet port with an Ethernet cable. A DSL modem must also be connected to the phone wall jack with the cable that came with the modem. A cable modem must also be connected to the coaxial cable installed by the cable company. Turn the DSL or cable modem on and let it start up completely.

To configure Ethernet:

  1. Select an Ethernet service in the list on the left.

  2. Click Configure, choose the configuration method recommended by your network administrator or ISP, and enter the configuration details.

    For instructions, see the following topics:

    Configuring a Network Connection Service Using DHCP
    Configuring a Network Connection Service Using DHCP with Manual Address
    Configuring a Network Connection Service Using BootP
    Configuring a Network Connection Service Manually

  3. If your local network or Internet service provider requires adjusting advanced Ethernet settings or advanced IPv6 addressing settings, click the appropriate button below the Search Domains field. For instructions, see the following topics:

    Adjusting Ethernet Options
    Adjusting IPv6 Options

    By default, the server configures the Ethernet port and generates an IPv6 address automatically.
