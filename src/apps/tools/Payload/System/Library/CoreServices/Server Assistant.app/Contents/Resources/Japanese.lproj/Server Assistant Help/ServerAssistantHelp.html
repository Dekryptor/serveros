サーバアシスタントヘルプ

ローカルで設定

このコンピュータ上ではじめて Mac OS X Server を設定します。

リモートでインストール

このコンピュータを使用して Mac OS X Server を別のコンピュータ上にインストールします。

リモートで設定

このコンピュータを使用して別のコンピュータ上の Mac OS X Server を設定します。

自動設定プロファイルを作成

設定情報をファイルに保存し、それを使ってサーバを自動的に設定します。

その他の製品ガイド

インストールと設定の準備については、Mac OS X Server に付属のマニュアル「お使いになる前に」を参照してください。


インストールと設定に必要な情報を収集するには、Mac OS X Server Install Disc の「Documentation」フォルダにある「インストールと設定のワークシート」を使用してください。
