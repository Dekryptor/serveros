Making a Network Connection Service Inactive

If you want to disconnect a network connection service, so that the server doesn’t use it, you can make it inactive.

To turn off a network connection service:

  1. In the list on the left, select the service you want to turn off.

  2. Choose Make Service Inactive from the Action (gear) pop-up menu.

    You can also choose Off from the Configure IPv4 pop-up menu.
