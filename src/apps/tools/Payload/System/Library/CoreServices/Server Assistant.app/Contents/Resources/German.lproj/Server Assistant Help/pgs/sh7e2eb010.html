Xsan

Das Fenster "Xsan" wird angezeigt, wenn der konfigurierte Server mit einem Fibre-Channel-Netzwerk verbunden ist. Wenn das Fibre-Channel-Netzwerk für ein Speicherbereichsnetzwerk (SAN) konfiguriert ist und Sie über die Xsan-Installations-DVD verfügen, können Sie den Server jetzt als Xsan-Metadaten-Controller konfigurieren. Im nächsten Fenster wählen Sie den Typ des Xsan-Metadaten-Controllers, den Sie konfigurieren wollen, d. h. primär oder sekundär. Später, wenn die Mac OS X Server-Konfiguration beendet ist, werden Sie aufgefordert, die Xsan-Installations-DVD einzulegen. Wenn Sie die Xsan-Software bereits auf diesem Server installiert haben, wird die Xsan-Installations-DVD später nicht angefordert.

Xsan ist ein leistungsfähiges, benutzerfreundliches SAN-Dateisystem für Mac OS X und Mac OS X Server. Mit Xsan können Sie Speichergeräte konsolidieren und Clients gleichzeitigen Fibre-Channel-Zugriff auf Dateien auf gemeinsam genutzten Volumes geben. Xsan enthält erweiterte Funktionen für maximale Speichernetzwerkverfügbarkeit. Dazu zählen Metadaten-Controller-Failover, Fibre-Channel-Multipathing und eine Journalfunktion für das Dateisystem. Mithilfe des Programms "Xsan-Admin" können Sie Ihr SAN konfigurieren, überwachen und verwalten. Weitere Informationen hierzu finden Sie im Konfigurationshandbuch und im Administratorhandbuch im Ordner "Dokumentation" auf der Xsan-Installations-DVD.
