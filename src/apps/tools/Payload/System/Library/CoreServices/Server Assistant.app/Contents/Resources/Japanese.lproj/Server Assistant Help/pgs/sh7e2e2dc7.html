ホーム
時間帯

「時間帯」パネルでは、サーバのローカル・タイム・ゾーンを選択します。

マップをクリックするか、「最も近い都市」フィールドに入力するか、ポップアップメニューを使って、都市を選択します。サーバが置かれている都市が見つからない場合は、同じ時間帯の都市を選択します。

ネットワーク・タイム・サーバを変更するには、「編集」ボタンをクリックします。「ネットワーク・タイム・サーバを使用」を選択します。「ネットワーク・タイム・サーバ」ポップアップメニューから選択するか、IP アドレスまたは DNS 名を入力することによって、使用するネットワーク・タイム・サーバを識別します。サーバ操作の信頼性を高めるため、アップルでは、ネットワーク・タイム・サーバと同期することによってサーバのクロックを正確に保つことを推奨しています。
