Configuration d’un service de connexion réseau à l’aide du protocole DHCP

Si vous voulez qu’un serveur DHCP définisse tous les réglages de la connexion réseau, y compris l’adresse IP du serveur, sélectionnez Utilisation de DHCP dans le menu local Configurer IPv4 de la sous-fenêtre Réseau.

Remarque :  configurez le serveur DHCP de manière qu’il attribue une adresse IP statique à votre serveur (la même adresse IP à tout moment) ; le service DHCP doit être en cours d’exécution.

Si vous configurez un serveur avec Xsan, vous ne pouvez pas sélectionner l’option Via DHCP, car cela n’est pas compatible avec un réseau Xsan.

Pour configurer un service de connexion réseau à l’aide du protocole DHCP :

Après avoir choisi un service de connexion réseau et sélectionné l’option Via DHCP dans le menu local Configurer, saisissez les informations de configuration DHCP à l’aide des données suivantes :

  • Serveur DNS :  
    L’adresse IP d’un serveur DNS (par exemple, 192.0.2.12) ou une liste d’adresses IP de serveurs DNS. En règle générale, le serveur DHCP fournit automatiquement l’adresse des serveurs DNS. Un serveur DNS traduit les noms DNS tels que serveur.exemple.com en adresses IP et vice versa.
  • Domaines de recherche :  
    Ils peuvent être fournis automatiquement par votre service DHCP. Vous pouvez aussi saisir la dernière partie des noms que vous souhaitez ajouter automatiquement aux noms DNS partiels que vous saisissez lorsque vous utilisez le serveur. Par exemple, si vous spécifiez le domaine de recherche campus.univ.edu, vous pouvez saisir serveur1 dans la zone de dialogue Connexion au serveur du Finder pour établir la connexion avec serveur1.campus.univ.edu.
  • Id. du client DHCP :  
    Saisissez le code identifiant client si le fournisseur d’accès à Internet ou l’administrateur de serveur DHCP réseau vous en a fourni un. L’identifiant client identifie le serveur (ou un port réseau, si le serveur en dispose de plusieurs). Ne spécifiez pas d’ID client DHCP lorsque vous utilisez l’Assistant du serveur pour configurer un serveur distant. À la place, une fois la configuration terminée, saisissez un ID client DHCP dans la sous-fenêtre Réseau des Préférences Système sur le serveur.
