Installieren

Das Fenster "Installieren" wird angezeigt, während Mac OS X Server auf dem entfernten Server installiert wird. Sie können den Installationsfortschritt in diesem Fenster überwachen.

Der entfernte Server wird automatisch neu gestartet, nachdem die Installation beendet wurde. Der Serverassistent wartet anschließend auf neue Eingaben.

  • Wenn Sie Mac OS X Server auf einem weiteren entfernten Computer installieren wollen, klicken Sie auf "Weitere Optionen", um zum Begrüßungsbildschirm zu gelangen.

  • Wenn Sie Mac OS X Server entfernt konfigurieren wollen, klicken Sie auf "Weitere Optionen", um zum Begrüßungsbildschirm zu gelangen.

  • Wenn Sie den Serverassistenten beenden wollen, klicken Sie auf "Beenden".
