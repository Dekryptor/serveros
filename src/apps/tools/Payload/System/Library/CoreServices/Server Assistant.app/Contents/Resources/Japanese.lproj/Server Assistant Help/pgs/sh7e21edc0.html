インストール中

「インストール中」パネルは、リモートサーバに Mac OS X Server をインストールしているときに表示されます。このパネルを使って、インストールの進行状況を監視できます。

インストールが完了すると、リモートサーバが自動的に再起動します。「サーバアシスタント」は、実行したい操作について指示されるまで待機します。

  • Mac OS X Server を別のリモートコンピュータ上にインストールしたい場合は、「より詳細に」をクリックして「ようこそ」パネルに移動します。

  • Mac OS X Server をリモートで設定したい場合は、「より詳細に」をクリックして「ようこそ」パネルに移動します。

  • 「サーバアシスタント」を終了したい場合は、「終了」をクリックします。
