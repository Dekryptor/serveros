Verbinden mit einem Verzeichnisserver

Im Fenster "Mit einem Verzeichnisserver verbinden" können Sie eine Verbindung zu einem vorhandenen Verzeichnisserver in Ihrem Unternehmen manuell konfigurieren. Wenn Sie wählen, dass Sie jetzt keine Verzeichnisserververbindung konfigurieren wollen, können Sie nach Beendigung der Serverkonfiguration die Verbindung zu einem Verzeichnisserver herstellen. Möchten Sie nach der Serverkonfiguration eine Verbindung zu einem oder mehreren Verzeichnisservern herstellen, verwenden Sie den Abschnitt "Anmeldeoptionen" in der Systemeinstellung "Benutzer".

Einstellungen für die Verbindung zum Verzeichnisserver
Verbinden mit einem externen Verzeichnisserver

Wählen Sie diese Option aus, wenn der Server eine Verbindung zu einem vorhandenen Verzeichnissystem herstellen soll.

Wenn Sie Xsan auf Ihrem Server konfigurieren, kann diese Option nicht ausgewählt werden. Wenn Sie jetzt keine Verbindung zu einem Verzeichnisserver herstellen wollen, klicken Sie auf "Zurück" und wählen Sie "Später mit einem Verzeichnisserver verbinden" im Fenster "Benutzer und Gruppen" aus.

Serveradresse

Wählen Sie den Verzeichnisserver aus dem Einblendmenü aus oder geben Sie den DNS-Namen des Verzeichnisservers ein. Wenn Sie eine Verbindung zu einem Open Directory-Server herstellen, können Sie dessen IP-Adresse eingeben.

Servertyp

Kann Open Directory oder Active Directory sein.

Wenn Sie jetzt einen Server konfigurieren (kein Profil für die automatische Konfiguration zur späteren Verwendung erstellen), wird der Servertyp automatisch bestimmt.

Wenn Sie eine Verbindung zu einem anderen Typ von Verzeichnisserver herstellen müssen, deaktivieren Sie die Option "Verbinden mit einem externen Verzeichnisserver" und konfigurieren Sie die Verbindung nach Beendigung der Serverkonfiguration.

Wenn Sie Xsan auf Ihrem Server konfigurieren, können Sie in diesem Fenster nur eine Verbindung zu einem Open Directory-Server herstellen. Zum Herstellen einer Verbindung zwischen einem Server mit Xsan und einem anderen Typ von Verzeichnisserver klicken Sie auf "Zurück" und wählen Sie "Später mit einem Verzeichnisserver verbinden" im Fenster "Benutzer und Gruppen" aus.

Benutzer-Account-Name und Kennwort

Diese Felder werden angezeigt, wenn der Verzeichnisserver eine Identifizierung erfordert.

  • Für Open Directory können Sie den Kurznamen oder Langnamen und das Kennwort eines Standardbenutzer-Accounts eingeben. Sie müssen keinen Verzeichnisadministrator-Account verwenden.

  • Für Active Directory können Sie den Namen und das Kennwort eines Active Directory-Administrator-Accounts oder eines Standardbenutzer-Accounts eingeben, der über das Zugriffsrecht verfügt, Workstations zu Domains hinzuzufügen.
