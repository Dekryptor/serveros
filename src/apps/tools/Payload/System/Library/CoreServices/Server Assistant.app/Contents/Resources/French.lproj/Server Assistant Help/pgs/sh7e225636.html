Langue d’installation

Utilisez la sous-fenêtre Langue d’installation pour sélectionner la langue principale devant être utilisée par Mac OS X Server.

La langue sélectionnée a un impact sur les applications d’administration du serveur et sur l’encodage par défaut utilisé par le service AFP (Apple Filing Protocol). Elle n’affecte pas la langue utilisée par les ordinateurs des utilisateurs. Ainsi, vous pouvez administrer un serveur en anglais même s’il prend en charge des utilisateurs dont les ordinateurs sont en japonais.
