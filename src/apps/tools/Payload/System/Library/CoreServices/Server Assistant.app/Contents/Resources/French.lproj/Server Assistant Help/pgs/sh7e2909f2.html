Configuration d’un service de connexion réseau AirPort

Votre serveur peut être relié à un réseau sans fil AirPort s’il dispose d’une carte AirPort installée et qu’il se trouve dans le rayon de détection de ce réseau sans fil.

Pour configurer AirPort :

  1. Sélectionnez AirPort dans la liste de gauche.

  2. Choisissez un réseau sans fil dans le menu local Nom du réseau et, si nécessaire, tapez le mot de passe associé.

  3. Cliquez sur Configurer, choisissez la méthode de configuration recommandée par votre administrateur de réseau AirPort, puis indiquez les détails de configuration appropriés.

    Pour obtenir des instructions, consultez les rubriques suivantes :

    Configuration d’un service de connexion réseau à l’aide du protocole DHCP
    Configuration d’un service de connexion réseau à l’aide du protocole DHCP avec une adresse manuelle
    Configuration d’un service de connexion réseau à l’aide du protocole BootP
    Configuration manuelle d’un service de connexion réseau

  4. Si votre réseau local ou votre fournisseur d’accès à Internet requiert l’ajustement des réglages avancés d’adressage IPv6, cliquez sur le bouton IPv6. Pour obtenir des instructions, consultez la rubrique Ajustement des options IPv6.

    Par défaut, le serveur génère automatiquement une adresse IPv6.

Si l’option AirPort ne figure pas dans la liste des services de connexion réseau, sélectionnez « Afficher tous les services » dans le menu local Action (image d’un engrenage). Si l’option AirPort n’est toujours pas disponible, cela signifie que votre serveur ne possède pas de carte AirPort fonctionnelle.

Pour en savoir plus sur la configuration d’une borne d’accès AirPort permettant la connexion à Internet, ouvrez l’application Utilitaire AirPort (située dans /Applications/Utilitaires/), puis servez-vous du menu Aide.
