ホーム
Xsan

「Xsan」パネルは、設定しているサーバが Fibre Channel ネットワークに接続されている場合に表示されます。Fibre Channel ネットワークが SAN（ストレージ・エリア・ネットワーク）用に構成されていて、「Xsan」のインストールディスクを持っている場合は、ここでサーバを Xsan メタデータコントローラとして設定するように選択できます。次のパネルで、設定したい Xsan メタデータコントローラのタイプ（プライマリまたはセカンダリー）を選択します。後で Mac OS X Server の設定が完了したときに、「Xsan」のインストールディスクを挿入するよう求められます。 このサーバ上にすでに「Xsan」ソフトウェアがインストールされている場合は、後で「Xsan」のインストールディスクの挿入が求められることはありません。

「Xsan」は、Mac OS X および Mac OS X Server 用の高パフォーマンスで使いやすい SAN ファイルシステムです。「Xsan」を使用すると、記憶装置を統合して、共有ボリューム上のファイルへのファイバーチャネルによる同時アクセスをクライアントに提供することができます。「Xsan」には、メタデータコントローラのフェイルオーバー、ファイバーチャネルのマルチパス方式、ファイルシステムのジャーナリングなど、ストレージネットワークの可用性を最大限に高める高度な機能があります。 「Xsan Admin」アプリケーションを使用して、SAN を設定、監視、管理することができます。詳しくは、「Xsan」のインストールディスクの「Documentation」フォルダにある設定ガイドと管理者ガイドを参照してください。
