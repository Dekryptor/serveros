Enregistrement de la configuration

La sous-fenêtre Enregistrement de la configuration permet d’enregistrer un profil de configuration automatique contenant les informations que vous avez saisies. Ce profil peut être enregistré sur n’importe quel disque auquel le serveur a accès. Vous pouvez par exemple l’enregistrer sur un lecteur amovible ou sur un CD. Lorsque le serveur démarre et que la sous-fenêtre Bienvenue s’affiche, vous pouvez connecter le lecteur ou insérer le CD pour lancer la configuration automatique.

Si vous cliquez sur Enregistrer, vous pouvez attribuer n’importe quel nom de fichier au profil, avec l’extension .plist. Pour enregistrer une variante du profil, cliquez sur Revenir pour modifier certains réglages, cliquez sur Continuer pour revenir à cette sous-fenêtre, puis à nouveau sur Enregistrer.

Réglages d’un profil de configuration automatique
Restreindre l’utilisation sur certains ordinateurs

Sélectionnez cette option et cliquez sur Modifier pour spécifier les conditions permettant de définir les serveurs qui peuvent faire l’objet d’une configuration automatique à l’aide de ce profil.

Vous pouvez inclure n’importe laquelle des conditions suivantes et avez la possibilité d’en ajouter plusieurs de chaque :

  • Numéro de série :  numéro de série matériel

  • Nom d’hôte :  tout ou partie d’un nom DNS

  • Adresse IP

  • Adresse MAC : adresse matérielle (identifiant Ethernet) d’une interface réseau

Vous pouvez également choisir le mode de comparaison d’une condition par rapport aux attributs d’un ordinateur éligible. Les comparaisons s’expliquent d’elles-mêmes, à l’exception des suivantes :

  • est :  l’attribut d’un ordinateur éligible doit correspondre exactement à la condition

  • est comme :  signifie que les symboles ? et * font office de caractères de remplacement dans la condition, où ? correspond à n’importe quel caractère et * correspond à zéro ou à au moins un caractère dans l’attribut d’un ordinateur éligible

  • correspond à :  l’attribut d’un serveur éligible doit contenir les motifs de mots ou de caractères indiqués par l’interprétation de la condition comme expression rationnelle (pour en savoir plus, consultez la section relative aux expressions rationnelles sur le site Web ICU User Guide, à l’adresse www.icu-project.org/userguide/regexp.html (en anglais)).

Si vous indiquez plusieurs conditions, le profil s’applique à un ordinateur qui correspond à n’importe laquelle.

Si un ordinateur éligible dispose de plusieurs noms DNS, adresses IP ou adresses MAC, tous ces éléments sont comparés aux conditions applicables. Par exemple, une condition indiquant une adresse matérielle sera comparée à tous les ports réseau d’un ordinateur.

Chiffrement

Indiquez si vous souhaitez que le profil de configuration de serveur automatique soit enregistré sous forme chiffrée ou au format Texte.

  • Si vous optez pour le chiffrement du mot de passe, saisissez une phrase clé permettant de chiffrer et de déchiffrer le profil de configuration automatique.

    Vous pourrez saisir à nouveau cette phrase clé au moment du démarrage de la configuration, ou placer un fichier texte contenant cette phrase clé dans le même dossier que celui où se trouve le profil de configuration automatique et attribuer à ce fichier le même nom que celui du profil, avec l’extension .pass.

  • Si vous choisissez Aucun, le profil est enregistré au format Texte.

Important :  ne choisissez pas Aucun si le profil peut être accessible par tous, car il contient le mot de passe du compte d’administrateur du serveur, que n’importe qui peut utiliser pour prendre le contrôle d’un serveur configuré pour utiliser ce profil.

Si vous souhaitez utiliser un profil enregistré, placez-le dans un dossier nommé Configuration de serveur automatique au niveau supérieur (root) de n’importe quel volume connecté à l’ordinateur à configurer automatiquement. Le volume peut se trouver sur le disque dur de l’ordinateur ou sur un iPod, un DVD, un CD, un lecteur FireWire, un périphérique USB ou tout autre périphérique de stockage branché sur l’ordinateur.

Lorsque vous démarrez un ordinateur sur lequel Mac OS X Server est installé mais qui n’est pas configuré, l’Assistant du serveur recherche automatiquement des profils de configuration automatique valides dans les dossiers Configuration de serveur automatique, sur tous les disques disponibles. Si plusieurs profils sont trouvés, c’est celui qui correspond le plus précisément à l’ordinateur en cours de configuration qui est utilisé. Pour déterminer le profil qui correspond le mieux, la comparaison des conditions s’effectue dans l’ordre suivant : numéro de série matériel, interface réseau, adresse MAC (adresse matérielle), adresse IP, nom DNS, nom de l’ordinateur, aucune condition. Si un profil dispose de plusieurs conditions, il s’appliquera à un ordinateur qui correspond à n’importe laquelle.
