Ajustement des options Ethernet

Certains réseaux Ethernet requièrent le réglage de diverses options avancées. Dans la plupart des cas, les réglages configurés automatiquement sont corrects. Si les réglages de port Ethernet sont incorrects, cela peut nuire aux performances du réseau ou rendre un port inutilisable. Votre administrateur réseau ou votre fournisseur d’accès à Internet doit être en mesure de vous dire si vous devez ajuster les réglages avancés.

Pour configurer les réglages Ethernet avancés :

Sélectionnez un service Ethernet, cliquez sur le bouton Ethernet, puis réglez les options en fonction des informations suivantes :

  • Vitesse :  
    Spécifie le nombre maximal de bits par seconde pouvant être envoyés ou reçus par le port Ethernet. Si votre ordinateur et le périphérique auquel vous vous connectez prennent en charge la technologie Gigabit Ethernet mais que ce n’est pas le cas de votre câble Ethernet, vous devrez probablement ajuster la vitesse Ethernet. Vous pouvez également acheter un câble prenant en charge le Gigabit Ethernet.
  • Duplex :  
    Détermine si les paquets d’entrée et de sortie sont transmis en même temps (duplex intégral) ou successivement (semi-duplex).
  • MTU :  
    Correspond au paquet le plus volumineux que le port peut envoyer ou recevoir. Les initiales MTU signifient Maximum Transmission Unit, ou unité de transmission maximale, dont la valeur est exprimée en octets. Augmenter la taille de paquet permet d’améliorer le débit (pour les transferts de fichiers, par exemple), mais les périphériques qui reçoivent ces paquets (commutateurs, routeurs, etc.) doivent être capables de prendre en charge cette taille.
