PPPoE ネットワーク接続サービスを構成する

以下のすべての条件を満たしている場合は、PPPoE ネットワーク接続サービスを作成する必要があります：

  • DSL モデムまたはケーブルモデムを使用してインターネットに接続している場合。

  • ISP（インターネット・サービス・プロバイダ）が PPPoE 経由での接続を要求している場合。

  • ネットワーク上のサーバとその他のコンピュータが、IP アドレスを ISP から取得する場合。

DSL モデムまたはケーブルモデムが、AirMac ベースステーションなどのインターネットルーターに接続されていて、1 つのアドレスをサーバやローカルネットワーク上のその他のコンピュータと共有するように設定されている場合は、ISP から特に指示がない限りは、サーバまたはその他のコンピュータ上に PPPoE 接続を構成しないでください。たいていの場合は、AirMac ベースステーションまたはその他のインターネットルーター上に PPPoE 接続を構成し、それによって接続が共有されます。

Xsan を使用するサーバを設定する場合、PPPoE ネットワーク接続サービスは Xsan ネットワークには適していないため作成することはできません。

PPPoE サービスを作成および構成するには：

  1. 左側のリストで、PPPoEを使用するネットワーク接続サービス（Ethernet、AirMac、または FireWire）を選択します。

  2. 「構成」をクリックし、「PPPoE サービスを作成」を選択して、PPPoE サービスに説明的な名前を付けます。

  3. 左側のリストで PPPoE サービスを選択し、ISP から受け取ったアカウント名とパスワードを入力します。

    ISP からの特別な指示がない限り、「PPPoE サービス名」フィールドには何も入力しないでください。
