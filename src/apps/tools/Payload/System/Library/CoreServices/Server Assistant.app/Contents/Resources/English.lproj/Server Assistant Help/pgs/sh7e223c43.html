Region

Use the Region pane to select the geographic region whose time, date, and number formats you want the server to use. The regional formats affect Finder windows and applications on the server.

By default, only regions for the language selected during installation are shown. If you don’t see the region you want, select Show All.

After setup, you can change the region in the International pane of System Preferences.

The Region pane doesn’t appear if you’re setting up locally.
