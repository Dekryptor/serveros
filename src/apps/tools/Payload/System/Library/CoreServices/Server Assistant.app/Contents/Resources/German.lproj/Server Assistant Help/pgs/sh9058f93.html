Verzeichnisdienste

Definieren Sie Ihren Server im Fenster "Verzeichnisdienste" als Open Directory-Master, damit er Netzwerk-Accounts für Benutzer, Gruppen und Computer bereitstellen kann. Ihr Server stellt den Open Directory-Dienst zusätzlich zu den vorhandenen Verzeichnisservern bereit, mit denen Sie Ihren Server im vorherigen Fenster verbunden haben. Wenn Sie Ihren Server jetzt nicht als Open Directory-Master definieren, können Sie dies mit dem Programm "Server-Admin" nach Beendigung der Serverkonfiguration nachholen.

Einstellungen für einen Open Directory-Master
Name

Der vollständige Name, den Sie für den Verzeichnisadministrator-Account verwenden wollen. Der Standardname ist "Verzeichnisadministrator".
Kurzname

Dies ist ein anstelle des vollständigen Namens verwendbarer kurzer Name, Der Standardkurzname ist "diradmin".
UID

Die Benutzer-ID, die für Ordner- und Dateizugriffsrechte verwendet wird. Die Standard-UID ist 1000.
Kennwort

Wird automatisch auf dasselbe Kennwort wie beim zuvor konfigurierten Serveradministrator-Account eingestellt.
Zugriff auf Dienste für einzelne Benutzer und Gruppen einschränken

Durch Auswahl dieser Option wird der entfernte Zugriff auf Adressbuch, Dateifreigabe, iCal-Kalender, iChat-Instant-Messaging, E-Mail und VPN beschränkt. Im Programm "Servereinstellungen" hinzugefügte Benutzer-Accounts erhalten Zugriff auf alle diese Dienste. Sie können den Benutzer- und Gruppenzugriff auf Dienste mit den Programmen "Servereinstellungen" und "Server-Admin" ändern.

Wenn Sie jetzt keinen Open Directory-Master erstellen und im vorherigen Fenster keine Verbindung zu einem Verzeichnisserver hergestellt haben, konfiguriert der Serverassistent keine Dienste für Sie. Nachdem der Serverassistent beendet wurde, können Sie das Programm "Server-Admin" oder Befehlszeilenprogramme verwenden, um die Dienste zu konfigurieren, die dieser Server bereitstellen soll.
