Serveurs

Utilisez la sous-fenêtre Serveurs pour identifier le ou les serveurs à configurer. Vous avez la possibilité de configurer plusieurs serveurs à la fois. Ceux-ci doivent avoir la même configuration matérielle et utiliser les mêmes données de configuration, à l’exception des numéros de série Mac OS X Server, des services de connexion réseau et des noms de réseau.

Initialement, cette sous-fenêtre contient la liste des serveurs que vous avez sélectionnés dans Admin Serveur en vue de leur configuration. Vous pouvez y ajouter d’autres serveurs ou supprimer certains serveurs répertoriés.

Pour configurer un serveur à distance, vous devez l’authentifier. Si vous configurez plusieurs serveurs, vous devez authentifier un serveur à la fois.

La sous-fenêtre Serveurs n’apparaît que si vous configurez un serveur distant (et non l’ordinateur sur lequel vous travaillez).

Pour authentifier un serveur de la liste en vue de sa configuration à distance :

Sélectionnez un serveur dans la liste, cliquez sur Authentifier, puis saisissez le mot de passe par défaut du serveur.

  • En cas de nouvelle installation de Mac OS X Server, tapez les huit premiers caractères du numéro de série matériel du serveur. Lors de la saisie, veillez à respecter l’emploi des majuscules dans le numéro de série. Pour connaître ce numéro de série, cherchez l’étiquette apposée sur le serveur.

    Si le serveur distant est un Xserve avec processeur Intel ne possédant pas de numéro de série parce que sa carte logique principale a été remplacée, saisissez « System S » (sans les guillemets) en tant que mot de passe.

    Pour tout autre ordinateur ne disposant pas d’un numéro de série matériel intégré, tapez 12345678 comme mot de passe.

  • En cas de serveur mis à niveau, saisissez le mot de passe du compte utilisateur racine.

Pour ajouter un serveur en vue de sa configuration à distance :

Cliquez sur Ajouter, tapez le nom DNS ou l’adresse IP du serveur distant, puis saisissez un mot de passe comme expliqué à l’étape ci-dessus.
