Ethernet ネットワーク接続サービスを構成する

Ethernet を使用して、ローカルネットワーク、またはインターネットに接続している DSL モデムまたはケーブルモデムに接続する場合は、サーバの接続設定を入力する必要があります。設定内容が分からない場合は、ネットワーク管理者または ISP（インターネット・サービス・プロバイダ）に確認してください。

ローカルネットワークに接続する場合は、サーバが Ethernet ケーブルを使ってネットワークの Ethernet 装置（スイッチやルーターなど）に接続されていることを確認してください。

DSL モデムまたはケーブルモデムに接続する場合は、サーバが Ethernet ケーブルを使ってサーバの Ethernet ポートに接続されていることを確認してください。 また、DSL モデムが、モデムに付属のケーブルを使って電話用のモジュラージャックに接続されている必要もあります。さらに、ケーブルモデムが、ケーブル会社が設置した同軸ケーブルに接続されている必要もあります。DSL モデムまたはケーブルモデムに電源を入れて、完全に作動するようにします。

Ehternet を構成するには：

  1. 左側のリストで Ethernet サービスを選択します。

  2. 「構成」をクリックし、ネットワーク管理者または ISP が推奨する構成方法を選択して、構成の詳細を入力します。

    操作方法については、以下のトピックを参照してください：

    DHCP を使用してネットワーク接続サービスを構成する
    DHCP（アドレスは手入力）を使用してネットワーク接続サービスを構成する
    BootP を使用してネットワーク接続サービスを構成する
    ネットワーク接続サービスを手動で構成する

  3. ローカルネットワークまたはインターネット・サービス・プロバイダが、詳細な Ethernet 設定または詳細な IPv6 アドレス設定の調整を必要としている場合は、「検索ドメイン」フィールドの下にある適切なボタンをクリックします。操作方法については、以下のトピックを参照してください：

    Ethernet オプションを調整する
    IPv6 オプションを調整する

    デフォルトでは、サーバにより Ethernet ポートが構成され、IPv6 アドレスが自動的に生成されます。
