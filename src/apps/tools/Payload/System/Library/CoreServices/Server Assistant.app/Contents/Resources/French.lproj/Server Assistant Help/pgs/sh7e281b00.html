Date et heure

Utilisez la sous-fenêtre Date et heure pour régler manuellement la date et l’heure de votre serveur. Cette sous-fenêtre ne s’affiche que si vous avez désélectionné l’option de serveur d’horloge de réseau dans la sous-fenêtre précédente, Fuseau horaire.

Pour un fonctionnement optimal de votre serveur, Apple vous recommande de maintenir l’horloge à l’heure en la synchronisant avec un serveur d’horloge de réseau. Vous pouvez spécifier un serveur d’horloge de réseau dans la sous-fenêtre Fuseau horaire, en cliquant sur Revenir.
