Xsan

The Xsan pane appears if the server you’re setting up is connected to a Fibre Channel network. If the Fibre Channel network is configured for a storage area network (SAN), you can choose to set up the server now as an Xsan metadata controller. In the next pane, you’ll choose the type of Xsan metadata controller you want to set up, primary or secondary. A later pane asks if you want to insert the Xsan installation disk and install the Xsan software. (This pane doesn’t appear if Xsan is already installed on the server.) If you don’t have the disc, you can install Xsan after setup.

Xsan is a high-performance, easy-to-use SAN file system for Mac OS X and Mac OS X Server. Xsan lets you consolidate storage devices and give clients concurrent Fibre Channel access to files on shared volumes. Xsan includes advanced features that maximize storage network availability, such as metadata controller failover, Fibre Channel multipathing, and file system journaling. You can use the Xsan Admin application to set up, monitor, and manage your SAN. For more information, see the setup guide and administrator’s guide in the Documentation folder on the Xsan installation disc.
