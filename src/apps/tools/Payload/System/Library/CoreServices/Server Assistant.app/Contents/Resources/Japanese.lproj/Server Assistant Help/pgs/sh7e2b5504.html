インスタント・メッセージング・サービスについて

Mac OS X Server は、Macintosh、Windows、Linux のユーザに、セキュリティ保護されたインスタントメッセージ（IM）を提供します。チームのメンバーは、部外者による機密情報の傍受を心配することなく、問題の解決方法を話し合ったり、計画を立てたり、URL を交換したり、ファイルを転送したりできます。インスタント・メッセージング・サービスを使用すると、複数のユーザ間でテキストメッセージを交換できます。また、iChat は、オーディオ、ビデオ、および多方向のオーディオとビデオによるセッションのためのユーザ間の直接接続を容易にします。

インスタント・メッセージング・サービスは、Mac OS X の「iChat」アプリケーション、Google Talk、および XMPP プロトコル（Jabber と呼ばれています）を使用するほかのインスタント・メッセージング・ソフトウェアと連携して動作します。
