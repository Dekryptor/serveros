ホーム
サーバ

「サーバ」パネルでは、設定するサーバ（複数可）を識別します。複数のサーバを同時に設定することができます。 これらのサーバは、同じハードウェア構成を持ち、Mac OS X Server のシリアル番号、ネットワーク接続サービス、およびネットワーク名以外は同じ設定データを使用する必要があります。

このパネルには、「サーバ管理」で設定のために選択したサーバのリストが最初に表示されます。サーバをさらに追加することも、リストにあるサーバを削除することもできます。

サーバをリモートで設定するには、認証する必要があります。複数のサーバを設定する場合は、一度に 1 台のサーバを認証する必要があります。

「サーバ」パネルが表示されるのは、（操作しているコンピュータではなく）リモートサーバを設定する場合のみです。

リストにあるリモートで設定するサーバを認証するには：

リストにあるサーバを選択し、「認証」をクリックして、サーバのデフォルトのパスワードを入力します。

  • Mac OS X Server の新しいインストールの場合は、サーバのハードウェアシリアル番号の最初の 8 文字を入力します。シリアル番号を入力するときは、大文字と小文字をラベルに記載されている通りに入力してください。シリアル番号を見つけるには、サーバ上のラベルを探してください。

    メイン・ロジック・ボードが交換されて、ハードウェアシリアル番号がない Intel プロセッサを搭載した Xserve の場合は、パスワードとして「System S」と入力してください（カッコは入力しないでください）。

    内蔵ハードウェアシリアル番号がない別のコンピュータでは、パスワードとして「12345678」を入力します。

  • アップグレードしたサーバの場合は、root ユーザアカウントのパスワードを入力します。

リモートで設定するサーバを追加するには：

「追加」をクリックし、リモートサーバの IP アドレスまたは DNS 名を入力し、上のタスクで説明したパスワードを入力します。
