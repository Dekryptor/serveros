À propos du service de partage de fichiers

Le service de partage de fichiers de Mac OS X Server permet aux membres d’un groupe d’accéder aux dossiers partagés et de stocker des fichiers personnels sur le serveur. Ils peuvent faire appel à des ordinateurs Macintosh, Windows ou UNIX pour accéder à leurs fichiers et dossiers partagés, sans logiciel spécifique, en utilisant les protocoles de fichier natif tels que AFP et SMB. Pour les utilisateurs Windows, les serveurs de fichiers Mac OS X Server sont visibles dans leurs emplacements réseaux, au même titre que des serveurs de fichiers Windows.
