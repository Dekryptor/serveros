Dienst "iChat" (Instant Messaging)

Mac OS X Server stellt Instant-Messaging (IM) für Macintosh-, Windows- und Linux-Benutzer zur Verfügung. Teammitglieder können damit an einem Brainstorming teilnehmen, Pläne machen, URL-Adressen austauschen oder Dateien übertragen, ohne unerlaubten Zugriff auf die ausgetauschten Daten befürchten zu müssen. Der Dienst "Instant-Messaging" stellt Instant-Messaging-Funktionen für den Nachrichtenaustausch zwischen zwei oder mehr Benutzern zur Verfügung. Der Dienst unterstützt darüber hinaus Direktverbindungen zwischen zwei Benutzern für Audio- und Videodaten sowie Mehrwegsitzungen für Audio- und Videodaten.

Der Dienst "Instant-Messaging" arbeitet mit dem Programm "iChat" in Mac OS X, mit Google Talk und weiterer Instant-Messaging-Software zusammen, die das als Jabber bezeichnete XMPP-Protokoll unterstützt.
