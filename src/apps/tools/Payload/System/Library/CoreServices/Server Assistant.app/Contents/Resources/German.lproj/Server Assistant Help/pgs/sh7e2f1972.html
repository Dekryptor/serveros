Xsan installieren

Beginnen Sie jetzt im Fenster "Xsan installieren" mit der Installation der Xsan-Software auf dem Server, einschließlich des Xsan-Dateisystems und des Programms "Xsan-Admin", oder verschieben Sie die Installation von Xsan auf später. Das Fenster "Xsan installieren" wird angezeigt, wenn Sie in einem vorherigen Fenster gewählt haben, dass Sie den Server als Xsan-Metadaten-Controller konfigurieren wollen.

Bevor Sie Xsan auf dem Server installieren, sollten Sie den SAN-Speicher und die SAN-Netzwerke konfiguriert und Xsan auf jedem Clientcomputer installiert haben. Bevor Sie Xsan auf dem Server installieren, der als primärer Metadaten-Controller dient, sollten Sie zuerst Xsan auf den Servern installieren, die als sekundäre Metadaten-Controller dienen.
