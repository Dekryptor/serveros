Services

Utilisez la sous-fenêtre Services pour spécifier les services que le serveur fournit initialement aux ordinateurs des utilisateurs. Si le serveur dispose de plusieurs disques ou d’un disque partitionné en plusieurs volumes, vous pouvez aussi choisir celui sur lequel les données de service sont stockées.

Pour en savoir plus sur les services, consultez les rubriques suivantes :

Important :  le service de coupe-feu du serveur n’est pas défini de façon à bloquer l’accès aux services provenant de l’extérieur de votre réseau, sauf si vous avez sélectionné l’option qui configure le serveur en tant que passerelle Internet dans une sous-fenêtre antérieure. (L’option de passerelle apparaît uniquement si le serveur dispose d’au moins deux ports Ethernet.) Si le serveur n’est pas protégé par un routeur, une passerelle ou un coupe-feu Internet distincts, activez la protection par coupe-feu au niveau de la sous-fenêtre Coupe-feu dans Préférences du serveur ou Admin Serveur une fois l’installation initiale terminée.

Une fois la configuration terminée, vous pouvez modifier les réglages de service à l’aide de l’application Préférences du serveur. Si vous devez modifier des réglages de service avancés ou configurer des services qui ne sont pas répertoriés ici, vous pouvez utiliser l’application Admin Serveur.

La sous-fenêtre Services n’apparaît pas si vous sélectionnez Configurer manuellement dans la sous-fenêtre Utilisateurs et groupes, puis si vous ne vous connectez pas à un serveur d’annuaire et ne créez pas de maître Open Directory dans la sous-fenêtre « Se connecter à un serveur d’annuaire ». La sous-fenêtre Services n’apparaît pas non plus si vous installez Xsan sur un serveur.
