Review

Use the Review pane to indicate what you want to do with the settings you’ve made.

Here are actions you can take:

  • Click Go Back to view or make changes in earlier panes.

  • Click Set Up to complete interactive server setup.

    Server Assistant configures the server with the settings you’ve made. It also generates the Mac OS X Server Next Steps document and places it on the server’s desktop. This document is specifically tailored to your server and your network. It tells you about additional changes you should make to enhance the security, accessibility, and overall usefulness of your new server.

  • Click Details to view configuration details or save the settings you made.

    You can save a summary as a text file, and you can save the settings as an auto setup profile. For more information, see Saving an Auto Setup Profile.
