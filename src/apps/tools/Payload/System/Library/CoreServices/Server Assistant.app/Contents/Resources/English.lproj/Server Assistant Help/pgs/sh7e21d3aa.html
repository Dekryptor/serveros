Target Disk

Use the Target Disk pane to select the disk or the volume of a partitioned disk that you want to install Mac OS X Server on. If the target disk or volume has a qualified version of Mac OS X Server installed, you will be able to upgrade it. If the target disk or volume has any version of Mac OS X Server or Mac OS X installed, you will be able to erase and rename the disk or volume before installing. The table below explains what actions you can take depending on what’s already installed on the target disk or volume.

WARNING:  Before erasing a disk or volume, be sure to copy files you want to save to another disk. Erasing a disk or volume will destroy all information on it. Erasing a volume on a partitioned disk will not erase information on other volumes on the same disk.
