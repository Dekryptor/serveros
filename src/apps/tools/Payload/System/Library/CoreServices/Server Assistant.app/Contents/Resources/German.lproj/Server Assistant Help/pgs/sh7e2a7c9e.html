Client-Backup

Im Fenster "Client-Backup" können Sie Serverfestplattenspeicher für die Sicherung der Mac-Computer von Benutzern bereitstellen. Snow Leopard-Benutzer, die Time Machine in den Systemeinstellungen nicht aktiviert haben, werden automatisch gefragt, ob sie den Server als Speicherort verwenden wollen. Andere Benutzer müssen die Systemeinstellung "Time Machine" öffnen und das zum Speichern von Sicherungen verwendete Volume ändern.

Wenn Sie die Option "Benutzer dürfen Backups auf diesem Server sichern" auswählen, können Sie ein Volume für die Time Machine-Sicherungen der Benutzer aus der Liste der derzeit mit dem Server verbundenen Volumes auswählen.

Das Fenster "Client-Backup" wird bei der lokalen und bei der entfernten Konfiguration eines einzelnen Servers angezeigt. Es wird nicht angezeigt, wenn Sie die Dateifreigabe im Fenster "Dienste" zuvor deaktiviert haben oder wenn das Fenster "Dienste" nicht angezeigt wurde, weil Sie eine manuelle Konfiguration vornehmen. Das Fenster "Client-Backup" wird auch nicht angezeigt, wenn Sie mehrere entfernten Server gleichzeitig konfigurieren.
