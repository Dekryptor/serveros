既存のサーバを転送しますか？

「既存のサーバを転送しますか？」パネルでは、既存のサーバからデータと設定を自動的に移行します。

  • 移行をスキップしたい場合は、「新しいサーバを設定」を選択して「続ける」をクリックします。

  • 移行したい場合は、「既存のサーバから情報を転送」を選択して「続ける」をクリックします。次のパネルで移行元のディスクを選択します。

重要： 自動的に移行できるのは、サーバの設定中のみです。移行アシスタントは、設定後は使用できません。

Mac OS X Server v10.4.11、または Mac OS X Server v10.5 の最新アップデートがあるサーバから移行することができます。また、Mac OS X Server v10.5 の最新アップデートの標準構成またはワークグループ構成の Time Machine バックアップから移行することもできます。移行元のソースは、移行には影響されません。

移行元のサーバの DNS 名 と IP アドレスは、設定しているサーバで使用する予定のものと同じである必要があります。新しいサーバの IP アドレスは、「システム環境設定」の「ネットワーク」パネルを使って設定後に変更することができます。

「既存のサーバを転送しますか？」パネルは、1 台のサーバのローカル設定とリモート設定の際に表示されます。複数のサーバのリモート設定中は表示されません。

既存のサーバからの移行について詳しくは、Mac OS X Server ガイドの Web サイト（www.apple.com/jp/server/macosx/resources/）を参照してください。
