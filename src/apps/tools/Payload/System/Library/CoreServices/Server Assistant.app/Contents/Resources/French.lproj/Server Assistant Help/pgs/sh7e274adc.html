Configuration d’un service de connexion réseau FireWire

Si votre serveur et un autre ordinateur Mac sont équipés de ports FireWire, vous pouvez les relier à l’aide d’un câble FireWire, puis configurer un réseau local entre eux.

Pour configurer un service de connexion réseau FireWire :

  1. Sélectionnez FireWire dans la liste de gauche.

  2. Cliquez sur Configurer, choisissez la méthode de configuration recommandée par votre administrateur réseau, puis indiquez les détails de configuration appropriés.

    Dans la plupart des environnements réseau FireWire, il est suffisant de choisir l’option Via DHCP et d’utiliser les détails de configuration attribués automatiquement.

    Pour obtenir des instructions, consultez les rubriques suivantes :

    Configuration d’un service de connexion réseau à l’aide du protocole DHCP
    Configuration d’un service de connexion réseau à l’aide du protocole DHCP avec une adresse manuelle
    Configuration d’un service de connexion réseau à l’aide du protocole BootP
    Configuration manuelle d’un service de connexion réseau

  3. Si votre réseau FireWire requiert l’ajustement des réglages avancés d’adressage IPv6, cliquez sur le bouton IPv6. Pour obtenir des instructions, consultez la rubrique Ajustement des options IPv6.

    Par défaut, le serveur génère automatiquement une adresse IPv6.

Si l’option FireWire ne figure pas dans la liste des services de connexion réseau, sélectionnez « Afficher tous les services » dans le menu local Action (image d’un engrenage). Si l’option FireWire n’est toujours pas disponible, cela signifie que votre serveur ne possède pas de port FireWire fonctionnel.
