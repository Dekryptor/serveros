About Address Book Service

Address book service allows users to find contact information in an address book across multiple computers without the schema limitations and security issues associated with LDAP. A user who has more than one Snow Leopard computer can use the Mail and Address Book applications on any of those computers to view and add contacts in an address book on your server. Users of other computers can access an address book on your server using third-party applications that are compatible with the CardDAV open standard.
