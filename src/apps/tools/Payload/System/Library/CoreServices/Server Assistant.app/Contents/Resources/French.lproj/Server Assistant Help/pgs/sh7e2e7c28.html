Enregistrement d’un profil de configuration automatique

Une fois que vous avez terminé de saisir les informations de configuration, vous pouvez les enregistrer sous la forme d’un profil de configuration automatique sur un disque accessible au serveur. Par exemple, vous pouvez les enregistrer sur un lecteur amovible ou un CD. Lorsque le serveur démarre et que la sous-fenêtre Bienvenue apparaît, vous pouvez connecter le lecteur ou insérer le CD pour commencer la configuration automatique.

Lorsque vous enregistrez un profil de configuration automatique, vous pouvez le nommer comme vous le souhaitez, à condition que son extension soit .plist.

Réglages d’un profil de configuration automatique
Appliquer ce profil à n’importe quel serveur

Sélectionnez cette option afin que le profil soit appliqué sans condition, indépendamment du numéro de série du serveur, du nom DNS, de l’adresse IP ou de l’adresse matérielle.

Appliquer ce profil si au moins l’une des conditions suivantes est remplie

Sélectionnez cette option pour spécifier les conditions qui déterminent les serveurs pouvant être configurés avec ce profil de configuration automatique.

Vous pouvez inclure l’une des conditions suivantes, ainsi qu’en ajouter plusieurs de chaque :

  • Numéro de série : un numéro de série matériel.

  • Nom d’hôte :  tout ou partie d’un nom DNS.

  • Adresse IP

  • Adresse Mac : adresse matérielle (ID Ethernet) d’une interface réseau.

Vous pouvez aussi choisir la façon dont une condition est comparée aux attributs d’un ordinateur éligible. Les comparaisons se passent d’explication, à l’exception des suivantes :

  • est : un attribut d’ordinateur éligible doit respecter exactement la condition.

  • est comme : signifie que ? et * sont des caractères génériques de remplacement dans la condition, où ? correspond à n’importe quel caractère et * correspond à zéro caractère ou plus dans un attribut d’ordinateur éligible.

  • correspond à : un attribut de serveur éligible doit contenir les mots ou les schémas de caractères spécifiés en interprétant la condition comme une expression régulière (pour en savoir plus, consultez la section Regular Expressions du site ICU User Guide à l’adresse www.icu-project.org/userguide/regexp.html).

Si vous spécifiez plusieurs conditions, le profil s’applique à un ordinateur qui respecte l’une d’elles.

Si un ordinateur éligible dispose de plusieurs noms DNS, adresses IP ou adresses MAC, tous ces éléments sont comparés aux conditions applicables. Par exemple, une condition qui spécifie une adresse matérielle est comparée à tous les ports réseau d’un ordinateur.

Chiffrement

Indiquez si vous souhaitez enregistrer le profil de configuration du serveur sous forme chiffrée ou en format Texte.

  • Si vous choisissez « Chiffrement du mot de passe », saisissez une phrase clé pour chiffrer et déchiffrer le profil de configuration automatique.

    Vous pouvez saisir à nouveau la phrase clé lorsque la configuration commence ou placer un fichier texte contenant la phrase clé dans le même dossier que le profil de configuration automatique, puis attribuer au fichier de phrase clé le même nom que le profil de configuration automatique, mais avec une extension .pass.

  • Si vous choisissez l’option Aucun, le profil est enregistré au format Texte.

Important :  ne sélectionnez pas l’option Aucun si le profil est accessible publiquement. Le profil contient le mot de passe du compte d’administrateur du serveur, que quelqu’un est susceptible d’utiliser pour prendre le contrôle d’un serveur configuré à l’aide de ce profil.

Pour utiliser un profil enregistré, placez-le dans un dossier nommé « Configuration de serveur automatique » au niveau le plus élevé (racine) de tous les volumes connectés à l’ordinateur que vous voulez configurer automatiquement. Ce volume peut se trouver sur le disque dur de l’ordinateur ou sur un iPod, un DVD, un CD, un disque FireWire, un disque USB ou un autre périphérique de stockage connecté à l’ordinateur.

Lorsque vous démarrez un ordinateur sur lequel Mac OS X Server est installé mais pas configuré, l’Assistant du serveur parcourt automatiquement tous les disques disponibles pour rechercher des profils de configuration automatique valides dans les dossiers « Configuration de serveur automatique ». Si l’Assistant détecte plusieurs profils, il utilise celui qui correspond le mieux à l’ordinateur à configurer. Pour déterminer le profil le plus adapté, il compare les conditions dans l’ordre suivant : numéro de série matériel, adresse MAC de l’interface réseau (adresse matérielle), adresse IP, nom DNS, nom de l’ordinateur, absence de conditions. Si un profil présente plusieurs conditions, il s’applique à un ordinateur qui respecte l’une d’elles.
