Services d’annuaire

La sous-fenêtre Services d’annuaire permet d’utiliser votre serveur comme maître Open Directory, de manière à ce qu’il fournisse des comptes réseau aux utilisateurs, aux groupes et aux ordinateurs. Votre serveur fournit le service Open Directory en plus de tout serveur d’annuaire existant auquel vous avez connecté votre serveur dans la sous-fenêtre précédente. Si vous choisissez de ne pas utiliser votre serveur comme maître Open Directory immédiatement, vous pourrez le faire avec Admin Serveur une fois la configuration du serveur terminée.

Réglages d’un maître Open Directory
Nom

Nom complet que vous souhaitez utiliser pour le compte d’administrateur d’annuaire. Le nom par défaut est Administrateur d’annuaire.
Nom abrégé

Il s’agit de l’abréviation du nom complet. Le nom abrégé par défaut est admin_ann.
Identifiant d’utilisateur

Numéro de l’identifiant d’utilisateur, servant pour les autorisations de dossier et de fichier. L’identifiant d’utilisateur par défaut est 1000.
Mot de passe

Défini automatiquement sur le même mot de passe que celui du compte de l’administrateur du serveur configuré précédemment.
Restreindre l’accès des utilisateurs individuels et des groupes aux services

Cette option restreint l’accès au carnet d’adresses, au partage de fichiers, au calendrier iCal, à la messagerie instantanée iChat, au courrier électronique et aux services d’accès à distance au VPN. Les comptes d’utilisateur ajoutés aux Préférences du serveur peuvent accéder à tous ces services. Vous pouvez modifier l’accès des utilisateurs et des groupes à ces services à l’aide de l’option Admin Serveur des Préférences du serveur.

Si vous choisissez de ne pas créer de maître Open Directory immédiatement et si vous ne vous êtes pas connecté à un serveur d’annuaire dans la sous-fenêtre précédente, l’Assistant du serveur ne configure aucun service. Une fois que l’Assistant du serveur a terminé, vous pouvez utiliser Admin Serveur ou les outils de ligne de commande pour configurer les services que ce serveur doit fournir.
