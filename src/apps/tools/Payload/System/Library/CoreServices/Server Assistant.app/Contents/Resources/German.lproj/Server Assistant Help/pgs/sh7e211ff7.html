Start
Ziel

Geben Sie im Fenster "Ziel" einen entfernten Computer an, auf dem Sie Mac OS X Server installieren wollen. Starten Sie vorher den entfernten Computer von der Installations-DVD für Mac OS X Server oder einer Image-Datei für die Netzwerkinstallation, die vom NetBoot-Dienst von Mac OS X Server bereitgestellt wird.

Für eine entfernte Installation erforderliche Informationen
Adresse

Möchten Sie die Serversoftware auf einem Computer in Ihrem lokalen Netzwerk (IP-Teilnetzwerk) installieren, wählen Sie den Zielcomputer aus dem Einblendmenü aus.

Möchten Sie einen nicht aufgeführten Computer angeben, geben Sie seinen DNS-Namen oder seine IP-Adresse im IPv4-Format (zum Beispiel 192.0.2.200) ein.

Bei Servern, die der Serverassistent im lokalen Netzwerk (IP-Teilnetzwerk) findet, werden die IP-Adresse u. U. automatisch durch einen DHCP-Server im Netzwerk zugeordnet. Falls im Netzwerk kein DHCP-Server vorhanden ist, verwendet der Zielcomputer eine Adresse im Format "169.254.xxx.xxx", anhand deren er eindeutig von anderen Servern im lokalen Netzwerk unterschieden werden kann. Bei der nachfolgenden Serverkonfiguration können Sie diese IP-Adresse bei Bedarf ändern.

Seriennummer

Geben Sie die ersten acht Zeichen der integrierten Hardwareseriennummer des Servers als Kennwort ein. Diese Seriennummer finden Sie auf dem Etikett, das außen auf dem Servergehäuse angebracht ist. Achten Sie bei der Eingabe der Seriennummer auf die Groß- und Kleinschreibung der Zeichen.

Für einen Xserve mit Intel-Prozessor, dessen Hauptplatine ersetzt wurde und der keine Hardwareseriennummer hat, geben Sie "System S" (ohne Anführungszeichen) als Kennwort ein. Verwenden Sie für einen weiteren Computer ohne integrierte Hardwareseriennummer die Ziffernfolge "12345678" als Kennwort.
