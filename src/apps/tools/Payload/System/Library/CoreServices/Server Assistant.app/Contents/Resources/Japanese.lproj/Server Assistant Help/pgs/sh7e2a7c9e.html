クライアントのバックアップ作成

「クライアントのバックアップ作成」パネルでは、サーバディスク領域をユーザの Mac のバックアップ作成に使用できるようにできます。「システム環境設定」で「Time Machine」を有効にしていない Snow Leopard ユーザは、サーバを保存場所として使用したいかどうかが自動的に確認されます。 ほかのユーザは、「システム環境設定」の「Time Machine」パネルを開いて、バックアップの保存に使用するディスクを変更する必要があります。

オプション「ユーザがこのサーバにバックアップを作成することを許可」を選択する場合は、現在サーバに接続されているディスクのリストから、ユーザの Time Machine バックアップ用のディスクを選択します。

「クライアントのバックアップ作成」パネルは、ローカル設定の際と、1 つのサーバのリモート設定の際に表示されます。以前に「サービス」パネルでファイル共有を無効にした場合、または手動で構成するために「サービス」パネルが表示されなかった場合は、表示されません。 また、同時に複数のリモートサーバを設定する場合も、「クライアントのバックアップ作成」パネルは表示されません。
