Saving an Auto Setup Profile

When you finish entering setup information, you can save it as an auto setup profile on a disk that the server will be able to access. For example, you can save it on a removable drive or a CD. When the server starts up and the Welcome pane appears, you can connect the drive or insert the CD to begin automatic setup.

When you save an auto setup profile, you can give it any filename you like as long as it has a .plist extension.

Settings for an auto setup profile
Apply this profile to any server

Select this option to have the profile apply unconditionally, regardless of the server’s serial number, DNS name, IP address, or hardware address.

Apply this profile only if any of the following conditions are true

Select this option to specify conditions that determine which servers can be set up automatically with this auto setup profile.

You can include any of the following conditions, and you can add more than one of each:

  • Serial number:  A hardware serial number

  • Hostname:  All or part of a DNS name

  • IP address

  • MAC address: The hardware address (Ethernet ID) of a network interface, with colons separating the number pairs (for example, 11:22:33:44:55:66)

You can also choose how a condition will be compared to an eligible computer’s attributes. The comparisons are self-explanatory, except for these:

  • is:  An eligible computer’s attribute must exactly match the condition

  • is like:  Means ? and * are wildcard characters in the condition, where ? matches any one character and * matches zero or more characters in an eligible computer’s attribute

  • matches:  An eligible server’s attribute must contain the words or character patterns specified by interpreting the condition as a regular expression (for more information, see the Regular Expressions section of the ICU User Guide website at www.icu-project.org/userguide/regexp.html).

If you use an “is” comparison with a hardware serial number, use the full serial number. Don’t just the first eight characters as you would when authenticating for an interactive remote installation or setup.

If you specify multiple conditions, the profile applies to a computer that satisfies any of them.

If an eligible computer has more than one DNS name, IP address, or MAC address, all are compared to applicable conditions. For example, a condition that specifies a hardware address is compared to all of a computer’s network ports.

Encryption

Choose whether to have the auto server setup profile saved in encrypted form or in plain text.

  • If you choose Password Encryption, enter a passphrase for encrypting and decrypting the auto setup profile.

    You can reenter the passphrase when setup begins or put a text file containing the passphrase in the same folder as the auto setup profile, and give the passphrase file the same name as the auto setup profile but with a .pass extension.

  • If you choose None, the profile is saved as plain text.

Important:  Don’t choose None if the profile might be publicly accessible. The profile contains the password of the server’s administrator account, which someone could use to take control of a server that’s set up using the profile.

To use a saved profile, put it in a folder named Auto Server Setup at the top (root) level of any volume connected to the computer you want set up automatically. The volume can be on the computer’s hard drive or an iPod, DVD, CD, FireWire drive, USB drive, or other storage device plugged into the computer.

When you start up a computer with Mac OS X Server installed but not set up, Server Assistant automatically looks on all available disks for valid auto setup profiles in Auto Server Setup folders. If it finds more than one profile, it uses the one that most specifically matches the computer being set up. To determine the most specific match, it compares conditions in this order: hardware serial number, network interface MAC address (hardware address), IP address, DNS name, computer name, no conditions. If a profile has multiple conditions, it applies to a computer that satisfies any of them.
