Webdienste

Die Webdienste können Wikis, Blogs, Webkalender, Webzugriff auf E-Mail und eigene Websites bereitstellen.

Wenn die Webdienste aktiv sind, erhält jeder Benutzer mit einem Account auf dem Server ein Wiki-Portal, "Meine Seite" genannt, zum Anzeigen und Erstellen von Wikis und Blogs, zum Verwenden von Webkalendern, Verfolgen von Wiki-Aktualisierungen und Zugreifen auf Webmail.

Alle Benutzer können Wikis ohne großen Aufwand erstellen und steuern, wer ihre Wiki-Inhalte anzeigen, durchsuchen und bearbeiten darf. Mithilfe vordefinierter (oder selbst erstellter) Vorlagen können Benutzer Wiki-Inhalte hinzufügen, bearbeiten, löschen und formatieren - ganz ohne Kenntnisse der Textauszeichnungscodes oder der speziellen Syntax. Einige wenige Klicks mit der Maus (oder alternativ das Bewegen und Ablegen) genügen, um Dateien und Bilder als Anhänge/Anlagen hinzuzufügen, Podcasts zu publizieren, Schlüssel- und Schlagwörter zuzuordnen und Links zu anderen Wiki-Seiten oder Websites einzufügen. Die Benutzer haben außerdem die Möglichkeit, die komplette Änderungschronik einer Wiki-Sammlung einzusehen und gegebenenfalls eine Seite auf eine frühere Version zurückzusetzen.

Mit Webkalendern können Termine, Meetings und sonstige Veranstaltungen in beliebigen Webbrowsern verfolgt werden. Benutzer können Einladungen zu Veranstaltungen senden und empfangen. Beim Einladen von Personen zu einer Veranstaltung können Benutzer die Verfügbarkeit jeder Person im Kalender sehen. Das Portal "Meine Seite" eines jeden Benutzers enthält einen persönlichen Webkalender, und auch jedes Wiki kann einen Kalender umfassen, den jeder Benutzer verwenden kann, der Zugriff auf das Wiki hat. Benutzer können mehrere Kalender in ihren persönlichen Webkalendern und den Wiki-Gruppenkalendern erstellen, auf die sie Zugriff haben. Benutzer mit Snow Leopard können auch das Programm "iCal" verwenden, um Veranstaltungen anzuzeigen, zu ändern und zu erstellen, die in Webkalendern angezeigt werden.

Blogs bieten Benutzern die Möglichkeit, für Kolleginnen und Kollegen neuste Informationen über Projekte und aktuell bearbeitete Dateien sowie Bilder oder Podcasts bereitzustellen, ohne dass sie sich dafür spezielle technische Kenntnisse aneignen müssen. Zum Publizieren eines eigenen Blogs genügt es, in der Palette der integrierten professionellen Vorlagen eine Vorlage zu wählen und per Drag&Drop zu übernehmen.

Mit dem Dienst "Webmail" können Benutzer mithilfe eines Webbrowsers an einem beliebigen Standort im Internet E-Mail-Nachrichten senden und empfangen. Sie können dabei auf ihre E-Mail-Nachrichten in der gleichen Weise wie mit dem Programm "Mail" oder einem anderen Mail-Programm auf ihrem Mac OS X-Computer zugreifen.

Neben diesen Webdiensten können Sie auch eigene Websites publizieren, die Sie erstellt haben (oder die jemand für Sie erstellt hat). Hierzu verwenden Sie Website-Entwicklungssoftware. Sie können den Zugriff auf jede Website auf eine bestimmte Gruppe beschränken oder Teile der Website auf bestimmte Gruppen beschränken. Sie können auch die IP-Adresse jeder Website, einen Zugriffsport und den Ordner angeben, in dem die Website-Dateien auf dem Server gespeichert sind. Eine eigene Website wird auch virtueller Host genannt.
