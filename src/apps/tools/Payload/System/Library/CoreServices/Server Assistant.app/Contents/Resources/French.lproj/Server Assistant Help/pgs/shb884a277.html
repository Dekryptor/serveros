Configuration en local

Vous trouverez ci-dessous les différentes rubriques d’aide relatives aux sous-fenêtres de l’Assistant du serveur qui permettent de configurer un serveur en local (indiquées dans l’ordre de leur apparition). Il se peut que vous n’ayez pas à utiliser toutes les sous-fenêtres, cela dépend des choix que vous indiquez dans les sous-fenêtres précédentes ainsi que des caractéristiques du serveur et du réseau. Pour configurer un contrôleur de métadonnées Xsan, vous utiliserez des sous-fenêtres complémentaires (consultez la rubrique intitulée Rubriques relatives à la configuration de Xsan).
