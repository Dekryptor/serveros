アドレスブックサービスについて

アドレスブックサービスを使用すると、LDAP に関連するスキーマ制約とセキュリティの問題が生じることなく、複数のコンピュータ内のアドレスブックにある連絡先情報を検索することができます。複数の Snow Leopard コンピュータを持っているユーザは、それらのコンピュータのいずれかにある「Mail」アプリケーションと「アドレスブック」アプリケーションを使用して、サーバ上のアドレスブック内の連絡先を表示したり追加したりできます。ほかのコンピュータのユーザは、CardDAV 標準規格と互換性のある他社製のアプリケーションを使用して、サーバ上のアドレスブックにアクセスすることができます。
