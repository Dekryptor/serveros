Configuring an AirPort Network Connection Service

Your server can join an AirPort wireless network if your server has an AirPort card installed and is in range of the wireless network.

To configure AirPort:

  1. Select AirPort in the list on the left.

  2. Choose a wireless network from the Network Name pop-up menu, and enter the network password if requested.

  3. Click Configure, choose the configuration method recommended by your AirPort network administrator, and enter the configuration details.

    For instructions, see the following topics:

    Configuring a Network Connection Service Using DHCP
    Configuring a Network Connection Service Using DHCP with Manual Address
    Configuring a Network Connection Service Using BootP
    Configuring a Network Connection Service Manually

  4. If your local network or Internet service provider requires adjusting advanced IPv6 addressing settings, click the IPv6 button. For instructions, see Adjusting IPv6 Options.

    By default, the server generates an IPv6 address automatically.

If AirPort isn’t in the list of network connection services, choose Show All Services from the Action (gear) pop-up menu. If AirPort still isn’t listed, your server doesn’t have a functional AirPort card.

For information about setting up an AirPort Base Station to connect to the Internet, open AirPort Utility (located in /Applications/Utilities/) and then use the Help menu.
