Configuration d’un service de connexion réseau Ethernet

Si vous utilisez Ethernet pour vous connecter à un réseau local ou à un modem câble ou DSL relié à Internet, vous devez saisir les réglages de connexion de votre serveur. En cas de doute sur les réglages requis, adressez-vous à l’administrateur réseau ou contactez le fournisseur d’accès à Internet (FAI).

Si vous vous connectez à un réseau local, assurez-vous que le serveur est relié au périphérique réseau Ethernet (tel qu’un commutateur ou un routeur) à l’aide d’un câble Ethernet.

Si vous vous connectez à un modem câble ou DSL, assurez-vous que ce dernier est relié au port Ethernet du serveur à l’aide d’un câble Ethernet. Un modem DSL doit aussi être connecté à la prise téléphonique murale au moyen du câble fourni. Vous devez également relier un modem câble au câble coaxial installé par le câblo-opérateur. Activez le modem câble ou DSL et laissez le processus de démarrage s’effectuer jusqu’au bout.

Pour configurer Ethernet :

  1. Sélectionnez un service Ethernet dans la liste de gauche.

  2. Cliquez sur Configurer, sélectionnez la méthode de configuration recommandée par l’administrateur réseau ou le fournisseur d’accès à Internet, puis saisissez les informations de configuration.

    Pour obtenir des instructions, consultez les rubriques suivantes :

    Configuration d’un service de connexion réseau à l’aide du protocole DHCP
    Configuration d’un service de connexion réseau à l’aide du protocole DHCP avec une adresse manuelle
    Configuration d’un service de connexion réseau à l’aide du protocole BootP
    Configuration manuelle d’un service de connexion réseau

  3. Si votre réseau local ou votre fournisseur d’accès à Internet nécessite d’ajuster les réglages Ethernet avancés ou les réglages d’adressage IPv6 avancés, cliquez sur le bouton approprié sous le champ Domaines de recherche. Pour obtenir des instructions, consultez les rubriques suivantes :

    Ajustement des options Ethernet
    Ajustement des options IPv6

    Par défaut, le serveur configure le port Ethernet et génère automatiquement une adresse IPv6.
