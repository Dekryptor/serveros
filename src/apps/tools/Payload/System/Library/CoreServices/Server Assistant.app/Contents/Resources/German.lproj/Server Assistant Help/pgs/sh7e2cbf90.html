Konfigurieren eines LOM Port-Netzwerkverbindungsdiensts

Xserve-Systeme mit Intel-Prozessoren enthalten einen separaten LOM-Prozessor (Lights-Out Management) für die Verwaltung inaktiver Server, mithilfe dessen Sie auch mit einem Server kommunizieren können, wenn dieser nicht eingeschaltet ist. Der LOM-Prozessor verfügt über eigene Netzwerkeinstellungen für die integrierten Ethernetanschlüsse des Servers.

Gehen Sie wie folgt vor, um einen LOM Port-Netzwerkverbindungsdienst zu konfigurieren:

  1. Wählen Sie einen LOM Port-Dienst in der Liste links aus.

  2. Geben Sie bei Aufforderung den Namen und das Kennwort des LOM-Administrators ein.

    Der LOM Administrator-Account ist nicht identisch mit dem Mac OS X Server-Administrator-Account, Sie können jedoch den gleichen Namen und das gleiche Kennwort verwenden, wenn Sie dies möchten.

  3. Klicken Sie auf "Konfigurieren", wählen Sie die von Ihrem Netzwerkadministrator empfohlene Konfigurationsmethode und geben Sie die Konfigurationsdetails anhand der folgenden Informationen ein.

    • IP-Adresse:  
      Eine numerische Adresse, die den Server im Netzwerk eindeutig identifiziert. Sie wird in Form von vier durch Punkte getrennten Zahlen angegeben (z. B. 192.0.2.123).

      Weisen Sie der LOM-Schnittstelle eine eigene IP-Adresse zu. Verwenden Sie nicht dieselbe Adresse, die Sie dem integrierten Ethernet 1 oder Ethernet 2 zugewiesen haben, auch wenn derselbe physische Stecker (Anschluss 1 oder 2) auf der Rückseite des Xserve verwendet wird.

    • Teilnetzmaske:  
      Eine Gruppe von vier Zahlen, die den Netzwerkteil der IP-Adresse des Servers angibt. Der Rest der IP-Adresse identifiziert den Server in diesem Netzwerk eindeutig.
    • Router:  
      Die IP-Adresse des Routers für das lokale Netzwerk des Servers (z. B. 192.168.99.1 oder 10.0.7.1). Der Router ist ein Gerät, das Datenpakete aus dem lokalen Netzwerk an ihre Ziele in anderen Netzwerken oder im Internet weiterleitet. Wenn Sie keinen Router in Ihrem Netzwerk haben, geben Sie stattdessen die IP-Adresse des Servers ein.

Sie können den Namen und das Kennwort des LOM-Administrators ändern. Klicken Sie dazu auf "Administratoraccount".

Nach Konfigurieren des Servers können Sie die Netzwerkeinstellungen für den LOM-Anschluss ändern und LOM-Administrator-Accounts mithilfe des Programms "Servermonitor" hinzufügen oder ändern. Öffnen Sie das Programm "Servermonitor" (in "/Programme/Server/") und verwenden Sie dann das Menü "Hilfe", um entsprechende Anweisungen zu erhalten. Sie haben auch die Möglichkeit, diese Einstellungen mit dem Befehl IPMITool im Programm "Terminal" zu ändern. Weitere Informationen dazu finden Sie auf der man-Seite IPMITool.
