DHCP（アドレスは手入力）を使用してネットワーク接続サービスを構成する

静的 IP アドレスを入力して、DHCP サーバを使用してほかのネットワーク接続設定を割り当てたい場合は、「ネットワーク」パネルの「IPv4 の構成」ポップアップメニューから「DHCP サーバを使用（アドレスは手入力）」を選択します。

「DHCP（アドレスは手入力）」を使用してネットワーク接続サービスを構成するには：

ネットワーク接続サービスを選択し、「構成」ポップアップメニューから「DHCP サーバを使用（アドレスは手入力）」を選択した後、以下の情報を参考にして構成の詳細を入力します：

  • IP アドレス： 
    ネットワーク上でサーバを一意に識別する数字アドレス。ピリオドで区切られた 4 つの数字として表されます（たとえば、192.0.2.123）。ローカルネットワーク上でインターネット接続を共有している AirMac ベースステーションまたはその他のインターネットルーターがある場合、サーバの IP アドレスは、ベースステーションまたはルーターの IP アドレス(10.0.7.2 や 192.168.99.2 など）と同じ IP サブネット上にある必要があります。
  • DNS サーバ： 
    DNS サーバの IP アドレス（たとえば、192.0.2.12）、または DNS サーバの IP アドレスのリスト。通常は、DHCP サーバによって DNS サーバに自動的にアドレスが提供されます。DNS サーバは、server.example.com のような DNS 名を IP アドレスに変換したり、その逆を行います。
  • 検索ドメイン： 
    DHCP サービスによって自動的に提供される場合があります。1 つまたは複数の名前の最後の部分で、サーバを使用しているときに入力する部分的な DNS 名に自動的に付加されるようにしたい部分を入力することもできます。たとえば、検索ドメインとして campus.univ.edu を指定する場合は、「Finder」の「サーバへ接続」ダイアログで「server1」と入力して server1.campus.univ.edu に接続することができます。
