Bienvenue

L’application Assistant du serveur vous permet d’installer et de configurer le logiciel Mac OS X Server sur un autre ordinateur, comme décrit dans le tableau suivant. Vous pouvez utiliser l’Assistant du serveur depuis un serveur déjà configuré ou depuis un ordinateur Mac OS X sur lequel le logiciel d’administration Mac OS X Server est déjà installé.

Une fois le serveur configuré, vous pouvez gérer les utilisateurs et les groupes et personnaliser les réglages de chaque service à l’aide de l’application Préférences du serveur ou des outils avancés d’administration serveur (Admin Serveur et Gestionnaire de groupe de travail, notamment). Pour en savoir plus sur la configuration et l’administration de Mac OS X Server, voir le manuel Premiers contacts fourni avec Mac OS X Server.

L’Assistant du serveur vous permet d’effectuer les opérations suivantes :
Installation de Mac OS X Server à distance

Installez le logiciel serveur sur un ordinateur du réseau. Vous pouvez :

  • procéder à l’installation sur un disque où Mac OS X Server ou Mac OS X n’est pas déjà installé ;

  • mettre à niveau Mac OS X Server 10.4.11 ou la dernière mise à jour de Mac OS X Server 10.5.

L’ordinateur distant doit être démarré à partir du disque d’installation de Mac OS X Server ou équivalent, et le programme d’installation doit afficher le sélecteur de langues.

Configuration d’un serveur distant

Configurez un serveur sur le réseau. Le serveur distant doit :

  • correspondre à un nouveau serveur ou à un ordinateur sur lequel Mac OS X Server 10.6 a été récemment installé ;

  • être démarré à l’aide de l’Assistant du serveur affichant la sous-fenêtre Bienvenue.

Vous avez la possibilité de configurer plusieurs serveurs à la fois. Ceux-ci doivent avoir la même configuration matérielle et utiliser les mêmes données de configuration, à l’exception des numéros de série Mac OS X Server, des services de connexion réseau et des noms de réseau. Si vous souhaitez configurer la gestion AirPort sur un serveur, configurez ce dernier séparément ou activez la gestion AirPort dans les Préférences du serveur une fois la configuration effectuée.

Création d’un profil pour la configuration automatique

Effectuez la procédure de configuration en sélectionnant les options appropriées et en entrant les données de configuration adéquates puis, au lieu d’utiliser les informations de configuration pour paramétrer de suite un serveur, enregistrez ces données sous la forme d’un profil de configuration automatique sur un lecteur ou un disque amovible. Par la suite, vous pourrez vous servir de ce profil enregistré pour automatiser la configuration d’un ou de plusieurs serveurs.

Vous pouvez utiliser les profils de configuration automatique pour configurer plusieurs serveurs à la fois ou pour configurer des serveurs de test après avoir réinstallé le logiciel serveur.

Vous pouvez créer un profil de configuration automatique générique en vue de configurer plusieurs serveurs ou bien créer un profil individuel pour un serveur spécifique. Lorsque vous paramétrez un profil de configuration automatique générique :

  • Vous devez préciser le nom et le mot de passe d’un compte administrateur qui sera utilisé par tous les serveurs. Vous pouvez modifier le mot de passe d’administrateur d’un serveur après avoir effectué la configuration en utilisant la sous-fenêtre Comptes des Préférences Système.

  • Vous pouvez définir un service de connexion réseau pour chaque adresse matérielle (également appelée identifiant Ethernet ou adresse MAC) ou bien un service de connexion réseau générique pour un nom d’interface (nom de port BSD), tel que en0 ou en1, puis configurer votre serveur DHCP de façon à attribuer une adresse IP fixe (statique) à chaque serveur, en fonction de son adresse matérielle. Cette fonctionnalité est appelée mappage statique ou réservations DHCP.

  • Vous pouvez laisser les noms de réseau vides et configurer votre service DNS de façon à attribuer un nom à chaque serveur, en fonction de son adresse IP.

À des fins de sécurité, vous pouvez enregistrer le profil de configuration automatique sous forme chiffrée.
