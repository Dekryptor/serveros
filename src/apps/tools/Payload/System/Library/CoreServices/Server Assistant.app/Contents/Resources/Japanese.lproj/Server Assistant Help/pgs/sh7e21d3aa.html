ターゲットディスク

「ターゲットディスク」パネルでは、Mac OS X Server をインストールする、ディスクまたはパーティションが作成されたディスクのボリュームを選択します。 ターゲットディスクまたはボリュームに正規の Mac OS X Server がインストールされている場合は、アップグレードすることができます。ターゲットディスクまたはボリュームにいずれかのバージョンの Mac OS X Server または Mac OS X がインストールされている場合は、インストールの前のディスクまたはボリュームを消去するか、名前を変更することができます。下の表で、ターゲットディスクまたはボリューム上にすでにインストールされているものに応じて、どのような処置をとることができるかを説明します。

警告： ディスクやボリュームを消去する前に、残しておきたいファイルを必ず別のディスクにコピーしてください。ディスクやボリュームを消去すると、そこにあるすべての情報が破壊されます。 パーティションが作成されたディスク上のボリュームを消去しても、同じディスク上のほかのボリュームにある情報は消去されません。
