About Calendar Service

Calendar service makes it easy for users to share calendars, schedule meetings, and coordinate events within a workgroup, a small business, or a large organization. Colleagues can quickly and easily check each other’s availability, set up and propose meetings, book conference rooms, reserve projectors, and more. The calendar service sends meeting invitations, which can include information such as an agenda or to-do list, and tabulates replies.

Snow Leopard Server adds push notifications, the ability to email event invitations to non-iCal Server users, integration with Calendar on iPhone, and a web application that lets users access their calendars from any computer with a web browser.

iCal Server works with the iCal application in Mac OS X v10.6 Snow Leopard, Calendar on iPhone, web calendars in Snow Leopard Server wikis, and third-party calendar applications that support the standard CalDAV protocol.
