Servers

Use the Servers pane to identify the server or servers you want to set up. You can set up multiple servers at the same time. They must have the same hardware configuration and use the same setup data except for Mac OS X Server serial numbers, network connection services, and network names.

This pane initially lists servers you selected for setup in Server Admin. You can include more servers or remove listed servers.

You must authenticate to set up a server remotely. If you’re setting up multiple servers, you must authenticate one server at a time.

The Servers pane appears only if you’re setting up a remote server (not the computer you’re working on).

To authenticate a listed server for remote setup:

Select a server in the list, click Authenticate, and enter the server’s default password.

  • For a new installation of Mac OS X Server, enter the first 8 characters of the server’s hardware serial number. Match the capitalization of the serial number when you type it. To find the serial number, look for a label on the server.

    If the remote server is an Xserve with Intel processor that has no serial number because its main logic board was replaced, enter “System S” (don’t enter the quotation marks) as the password.

    For another computer that has no built-in hardware serial number, enter 12345678 as the password.

  • For an upgraded server, enter the password of the root user account.

To add a server for remote setup:

Click Add, enter the remote server’s IP address or DNS name, and enter a password as described in the task above.
