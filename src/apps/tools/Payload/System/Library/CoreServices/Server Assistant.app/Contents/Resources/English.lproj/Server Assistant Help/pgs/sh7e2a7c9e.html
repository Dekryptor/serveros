Client Backup

Use the Client Backup pane to make server disk space available for backing up users’ Macs. Snow Leopard users who don’t have Time Machine turned on in System Preferences will be automatically asked if they want to use the server as a storage location. Other users will need to open the Time Machine pane of System Preferences and change the disk used for storing backups.

If you select the option “Allow users to back up to this server,” you can select a disk for users’ Time Machine backups from the list of disks currently connected to the server.

The Client Backup pane appears during local setup and during remote setup of a single server. It doesn’t appear if you turned off file sharing in the Services pane earlier, or if the Services pane didn’t appear because you’re configuring manually. The Client Backup pane also doesn’t appear if you’re setting up multiple remote servers at the same time.
