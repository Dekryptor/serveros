このコンピュータには Mac OS X Server ソフトウェアがインストールされていて、サーバを起動して実行するために「サーバアシスタント」で初期設定プロセスを実行できる状態になっています。

コンピュータに Mac OS X Server の新しいインストールがある場合は、画面の指示に従って設定するか、保存した設定プロファイルによる自動設定を開始することができます。

コンピュータを前のバージョンの Mac OS X Server からアップグレードした場合、入力する必要があるのは Mac OS X バージョン 10.6 Snow Leopard のシリアル番号だけです。ほかのすべてのサーバ設定は、変更されません。「サーバアシスタント」により、アップグレードされたサービスに必要なシステムデータの移行が実行されます。

新しいサーバを画面の指示に従って設定するには：

サーバで使用する時刻、日付、および数値の書式を持つ地域を選択してから、「続ける」をクリックします。

デフォルトでは、インストール時に選択した言語の地域のみが表示されます。目的の地域が表示されない場合は、「すべてを表示」を選択します。

地域の書式は、サーバ上の「Finder」のウインドウとアプリケーションに影響します。地域は、設定後に「システム環境設定」の「言語環境」で変更できます。

新しいサーバを自動的に設定するには：

以前に作成して、リムーバブルドライブまたはディスク上に保存した自動サーバ設定プロファイルを使用する場合は、ドライブをサーバに接続するか、この時点でサーバにディスクを挿入します。

使用可能なすべてのディスクの最上位（ルート）レベルにある「自動サーバ設定」という名前のフォルダ内で、プロファイルが検索されます。プロファイルに制約条件がある場合は、サーバが条件のいずれかを満たしていれば、そのプロファイルが使用されます。サーバが条件を満たしているプロファイルが複数見つかった場合は、最も正確に一致するプロファイルが使用されます。 これらの条件について詳しくは、「自動設定プロファイルを保存する」を参照してください。

自動サーバ設定について詳しくは、「インストールと設定のワークシート」を参照してください。（このワークシートは、Mac OS X Server Install Disc の「Documentation」フォルダに入っています。）

画面の指示に従ったサーバ設定または自動サーバ設定が完了した後、追加の設定作業が必要になる場合があります。設定後に生成されてサーバのデスクトップ上に配置される「Mac OS X Server - 次のステップ」という書類を探し、これを読んで具体的なアドバイスを確認してください。

サーバを使用する前に、追加のサービスの設定、一部のサービス設定の変更、またはユーザとグループの管理を行うこともできます。「サーバ環境設定」、「サーバ管理」、「ワークグループマネージャ」、および「/アプリケーション/サーバ/」フォルダにあるその他のアプリケーションを使用できます。UNIX コマンドラインツールも使用できます。サーバアプリケーションとツールについて詳しくは、Mac OS X Server に付属のマニュアル「お使いになる前に」を参照してください。
