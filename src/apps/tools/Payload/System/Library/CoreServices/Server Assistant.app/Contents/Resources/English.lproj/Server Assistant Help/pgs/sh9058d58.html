Transfer an Existing Server?

Use the “Transfer to an Existing Server?” pane to automatically migrate data and settings from an existing server.

  • If you want to skip migration, select “Set up a new server” and click Continue.

  • If you want to migrate, select “Transfer the information from an existing server” and click Continue. You select the disk to migrate from in the next pane.

Important:  You can only migrate automatically during server setup. A migration assistant isn’t available after setup.

You can migrate from a server with Mac OS X Server v10.4.11, Mac OS X Server v10.5, or Mac OS X Server v10.6. You can also migrate from a Time Machine backup of Mac OS X Server v10.6 or of a standard configuration or workgroup configuration of Mac OS X Server v10.5. The source you migrate from isn’t affected by migration.

The server you’re migrating from needs to have the same DNS name and IP address that you’re planning to use for the server you’re setting up. You can change the new server’s IP address after setup by using the Network pane of System Preferences.

The “Transfer to an Existing Server?” pane appears during local setup and remote setup of one server. It doesn’t appear during remote setup of multiple servers.

For more information about migrating from an existing server, see the Mac OS X Server Resources website at www.apple.com/server/macosx/resources/.
