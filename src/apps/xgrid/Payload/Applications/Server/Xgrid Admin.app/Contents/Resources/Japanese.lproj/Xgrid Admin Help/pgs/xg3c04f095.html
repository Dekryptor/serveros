エージェント

Xgrid エージェントは、ジョブの計算タスクを実行します。Mac OS X Server では、エージェントはデフォルトで切になっています。エージェントが入になり、起動時に使用可能になると、コントローラに登録されます。（エージェントは、一度に 1 つのコントローラにのみ接続できます。）コントローラは、コントローラのジョブに必要なエージェントに命令とデータを送ります。エージェントは、コントローラから命令を受け取ると、割り当てられたタスクを実行し、結果をコントローラに送り返します。

デフォルトでは、エージェントは、LAN で最初に利用可能なコントローラにバインドしようとします。または、特定のコントローラにバインドするように指定することもできます。

エージェントを常に利用可能にするか、コンピュータが待機状態のときだけ利用可能にするかを指定することもできます。マウスまたはキーボードによる入力がない場合、コンピュータは待機状態とみなされ、CPU とネットワークの動作が無視されます。ユーザがグリッドタスクを実行しているコンピュータに戻ると、そのコンピュータはタスクが完了するまで実行を続行します。

デフォルトでは、Mac OS X Server のエージェントは常に利用可能であり、Mac OS X コンピュータ（サーバではありません）のエージェントは 15 分間ユーザ入力がないとタスクを受け入れるように構成されています。
