Indicateurs d'état de Xgrid Admin

Xgrid Admin fournit des indicateurs d'état, c'est-à-dire, des petites bulles colorées qui indiquent l'état des contrôleurs, des agents et des tâches. Les indicateurs colorés sont les suivants :

  • Sans couleur = le contrôleur ou l'agent est déconnecté, la tâche est en attente

  • Gris = le travail est en cours de soumission.

  • Vert = le contrôleur est activé, l'agent fonctionne et la tâche est en cours d'exécution

  • Jaune = l'agent est disponible mais n'est pas en cours d'exécution.

  • Rouge = l'agent n'est pas disponible, la tâche a échoué ou est annulée

  • Bleu = le travail est terminé.
