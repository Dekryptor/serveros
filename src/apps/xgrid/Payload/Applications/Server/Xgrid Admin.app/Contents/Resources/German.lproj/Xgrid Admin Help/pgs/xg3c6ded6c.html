Anzeigen einer Auftragsliste

Sie können in Xgrid-Admin eine Liste der Aufträge anzeigen.

Gehen Sie wie folgt vor, um eine Auftragsliste anzuzeigen:

  1. Öffnen Sie das Programm "Xgrid-Admin".

  2. Wählen Sie den Controller in der Liste "Controller und Grids" aus.

  3. Klicken Sie auf "Aufträge".

  4. Wählen Sie einen Auftrag aus, um Details zu diesem Auftrag anzuzeigen.
