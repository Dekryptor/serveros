Anzeigen einer Liste mit Agenten

Sie können in Xgrid-Admin eine Liste der Agenten für einen Controller anzeigen.

Gehen Sie wie folgt vor, um eine Liste der Agenten für einen Xgrid-Controller anzuzeigen:

  1. Öffnen Sie das Programm "Xgrid-Admin".

  2. Wählen Sie das Grid in der Liste "Controller und Grids" aus.

  3. Klicken Sie auf "Agenten".

  4. Wählen Sie einen Agenten in der Liste aus, um Informationen über die vom Agenten benötigte CPU-Leistung und die verwendeten Prozessoren anzuzeigen.

    Der farbige Punkt links neben dem Namen zeigt den Status des Agenten an. Näheres hierzu finden Sie unter: Statusanzeigen im Programm "Xgrid-Admin"..
