Ajout d'un agent

Vous pouvez ajouter un agent à un contrôleur dans Xgrid Admin. Vous pouvez ajouter des agents qui sont déconnectés. Les agents sont disponibles pour le contrôleur lorsque les ordinateurs sont connectés ou que l'administrateur de contrôleur active les agents.

Pour ajouter un agent :

  1. Ouvrez Xgrid Admin.

  2. Dans la liste Contrôleurs et grilles, sélectionnez le contrôleur.

  3. Cliquez sur Agents.

  4. Cliquez sur le bouton Ajouter (+) sous la liste des agents.

  5. Attribuez un nom à l'agent puis cliquez sur OK.

L'agent est ajouté à la liste. La bulle colorée à gauche du nom indique l'état de l'agent. Pour en savoir plus, consultez la rubrique Indicateurs d'état de Xgrid Admin.
