Verwalten von Xgrid-Agenten

Verwenden Sie das Programm "Xgrid-Admin", um Agenten anzuzeigen, hinzuzufügen oder zu entfernen. Xgrid-Admin verwendet Statusanzeigen zum Anzeigen des Status der Agenten. Weitere Informationen finden Sie unter:

  • Anzeigen einer Liste mit Agenten

  • Hinzufügen eines Agenten

  • Löschen eines Agenten

  • Statusanzeigen im Programm "Xgrid-Admin"

Das Programm "Server-Admin" bietet zwar eine einfache Oberfläche für die Aktivierung von Xgrid-Diensten auf einem Server oder einem Xserve-System-Rack, es enthält jedoch keine Möglichkeit, Xgrid auf Desktop-Computern mit Mac OS X 10.3 (oder neuer) konfigurieren.

Wenn Sie sich darauf verlassen, dass Benutzer freiwillig Desktop-Agenten bereitstellen, können Sie diesen Anleitungen zur Aktivierung von Xgrid in der Systemeinstellung "Freigaben" senden.

Wenn die Benutzer mit Mac OS X 10.3 arbeiten, müssen Sie zuerst den Xgrid-Agenten für Mac OS X 10.3 laden und anschließend den Bereich "Xgrid" der Systemeinstellungen verwenden. Sie können den Xgrid-Agenten für Mac OS X 10.3 von dieser Website laden: www.apple.com/de/server/macosx/technology/xgrid.html

Wenn Sie eine Gruppe mit Computern verwalten und diese mit Xgrid zu einem Grid kombinieren möchten, stehen Ihnen folgende Methoden zur Verfügung:

  • Apple Remote Desktop

  • SSH

  • NetBoot oder NetInstall
