Visualisation d'une liste d'agents

Vous pouvez visualiser une liste d'agents d'un contrôleur dans Xgrid Admin.

Pour afficher une liste d'agents d'un contrôleur Xgrid :

  1. Ouvrez Xgrid Admin.

  2. Dans la liste Contrôleurs et grilles, sélectionnez la grille.

  3. Cliquez sur Agents.

  4. Sélectionnez l'agent dans la liste pour afficher les informations relatives à la puissance du CPU et aux processeurs qu'il utilise.

    La bulle colorée à gauche du nom indique l'état de chaque agent. Consultez la rubrique Indicateurs d'état de Xgrid Admin pour en savoir plus.
