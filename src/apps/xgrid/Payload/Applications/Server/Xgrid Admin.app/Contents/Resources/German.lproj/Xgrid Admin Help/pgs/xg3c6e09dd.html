Hinzufügen eines Grids

Sie verwenden das Programm "Xgrid-Admin", um ein Grid in der Liste "Controller und Grids" zu einem Xgrid-Controller hinzuzufügen.

Gehen Sie wie folgt vor, um ein Grid hinzuzufügen:

  1. Öffnen Sie das Programm "Xgrid-Admin".

  2. Wählen Sie den Xgrid-Controller aus, der zum Grid hinzugefügt werden soll.

  3. Klicken Sie auf die Taste "Hinzufügen" (+) unter der Liste "Controller und Grids" und wählen Sie "Grid hinzufügen" aus.

  4. Wählen Sie aus dem Einblendmenü einen Namen für das neue Grid aus und klicken Sie auf "OK".
