Composants d'Xgrid

L'architecture à trois niveaux d'Xgrid simplifie la répartition des sous-tâches complexes. Ses clients utilisateurs, contrôleurs de grille et agents de calcul collaborant pour simplifier le processus d'assemblage des nœuds, de soumission des tâches et de récupération des résultats.

Les principaux composants d'une grille de calcul effectuent les fonctions suivantes :

  • Un agent exécute une seule sous­tâche à la fois par CPU ; ainsi, un ordinateur multiprocesseur peut exécuter plusieurs sous­tâches simultanément.

  • Une contrôleur place les sous-tâches en file d'attente, répartit les sous-tâches entre les agents et gère la réaffectation des sous-tâches.

  • Un client soumet les tâches au contrôleur Xgrid sous forme de plusieurs sous-tâches. (Un client peut être un ordinateur quelconque exécutant Mac OS X v10.4 ou supérieur ou Mac OS X Server 10.4 ou supérieur.)

En principe, l'agent, le contrôleur et le client peuvent s'exécuter sur le même serveur, mais il est souvent plus efficace de disposer d'un nœud contrôleur dédié.
