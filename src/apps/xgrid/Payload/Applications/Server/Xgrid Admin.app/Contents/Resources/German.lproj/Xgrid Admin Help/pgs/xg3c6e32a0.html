Xgrid-Admin - Überblick

Mit dem Programm "Xgrid-Admin" können Sie Grids überwachen und Agenten und Aufträge verwalten. Sie können Controller und Agenten hinzufügen, um Agenten zu überwachen und anzugeben, die noch nicht an einem Grid beteiligt sind. Sie können Xgrid-Admin auch verwenden, um Aufträge anzuhalten, zu stoppen oder neu zu starten.

Verwenden Sie zum Verwalten von Computer-Grids das Programm "Xgrid-Admin". Ein Computer-Grid ist eine feste Gruppe von Agenten mit einer dedizierten Warteliste.

Es können mehrere Grids pro Controller vorhanden sein, aber ein Agent kann nur zu einem einzigen Grid gehören. Sie können Agenten nicht zwischen Grids bewegen, während ein Auftrag (oder eine Aufgabe) ausgeführt wird.
