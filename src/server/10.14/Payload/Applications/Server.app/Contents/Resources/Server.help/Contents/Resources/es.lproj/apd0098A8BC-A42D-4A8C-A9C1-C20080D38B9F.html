Seleccionar un Mac

Puede utilizar la app Server en el ordenador que usará como servidor o puede utilizarla para gestionar remotamente un ordenador con macOS Server.

Si selecciona la opción “Este Mac” cuando aparezca el cuadro de diálogo “Seleccione un Mac”, la app Server convertirá el Mac en un servidor. El resto de ordenadores Mac de su red local que tengan macOS Server activado aparecerán en este cuadro de diálogo. También puede añadir otros servidores mediante su dirección IP o nombre de host.

Solo puede gestionar ordenadores Mac remotos que tengan instalado macOS Server si está activada la opción “Permitir administración remota mediante Server” en el ordenador remoto. Esta opción se encuentra en el panel Ajustes de la aplicación Server. Es necesario activar esta opción localmente en el servidor remoto antes de gestionarlo desde otro Mac.

Si no ve el servidor que esperaba, compruebe su conexión de red o la conexión de red del otro servidor.
