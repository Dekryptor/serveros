登入伺服器

針對您想要透過 Server App 來連接的伺服器，登入管理者帳號。在您連接伺服器後，其名稱會出現在側邊欄的「硬體」區段中。

主機名稱或 IP 位址

Server App 會自動填入您正在新增之伺服器的主機名稱（如果您在對話框出現前已選擇伺服器）。

否則，請執行下列其中一項操作：

  • 伺服器的主機名稱：可以是完全合格的 Internet 網域名稱，如 macserver.example.com；完全合格的內部網路網域名稱，如 miniserver.example.private；或伺服器的本機主機名稱，如 macserver.local

  • IP 位址：伺服器的數值 IP 位址，如 192.168.1.100

管理者名稱

為您正在新增的伺服器輸入管理者名稱。如果您正在使用的 Mac 與您正在加入的伺服器為不同的電腦，每個皆有其自己的管理者帳號，而管理者名稱和密碼可能是不同的。

管理者密碼

為您上述輸入名稱的管理者帳號來輸入密碼。
