註冊伺服器的 Internet 主機名稱

允許使用者在 Internet 上使用主機名稱來連接伺服器，您必須註冊伺服器的主機名稱。

  1. 獲得 Internet 網域名稱，如 example.com。

    如果您還沒有網域名稱，可以從公共網域名稱註冊商來購買網域名稱。如需有關網域名稱註冊商的資訊，請搜尋網頁。

  2. 使用您的網域名稱註冊商為此伺服器（例如 server.example.com）註冊唯一的主機名稱。

    如果您的組織有電腦支援群組，請向其申請一個主機名稱。或者，使用與您取得網域名稱的網域名稱註冊商合作，來指定一個主機名稱。

  3. 為此伺服器的 DNS 託管服務增加記錄到其 DNS 伺服器中。

    如果您的組織有電腦支援群組，詢問其是否有託管 DNS 伺服器。或者，您的 DNS 註冊商可能會提供 DNS 託管服務，或您可以搜尋網頁來尋找其他提供者。
