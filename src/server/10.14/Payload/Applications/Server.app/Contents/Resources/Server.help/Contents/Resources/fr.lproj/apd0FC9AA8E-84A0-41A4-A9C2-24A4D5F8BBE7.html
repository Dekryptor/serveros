Enregistrer le nom d’hôte Internet du serveur

Pour permettre aux utilisateurs d’accéder au serveur en utilisant son nom d’hôte sur Internet, vous devez enregistrer le nom d’hôte du serveur.

  1. Obtenez un nom de domaine Internet comme « exemple.com ».

    Si vous ne disposez d’aucun nom de domaine, vous pouvez en acheter un auprès d’un bureau d’enregistrement de noms de domaine public. Pour obtenir des informations sur les bureaux d’enregistrement de noms de domaine, faites des recherches sur le web.

  2. Enregistrez un nom d’hôte unique pour ce serveur (tel que serveur.example.com) auprès de votre bureau d’enregistrement de noms de domaine.

    Si votre entreprise dispose d’un service d’assistance informatique, demandez un nom d’hôte. Sinon, utilisez le bureau d’enregistrement de noms de domaine (par le biais duquel vous avez obtenu votre nom de domaine) pour attribuer un nom d’hôte.

  3. Demandez à un service d’hébergement DNS d’ajouter à ses serveurs DNS des enregistrements pour ce serveur.

    Si votre entreprise dispose d’un service d’assistance informatique, demandez si des serveurs DNS sont hébergés. Sinon, votre bureau d’enregistrement de noms de domaine peut peut-être fournir un service d’hébergement DNS. Vous pouvez également rechercher un fournisseur sur le web.
