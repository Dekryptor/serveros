Assistent zum Ändern von Hostnamen

Verwenden Sie den Assistenten zum Ändern von Hostnamen, um den Hostnamen, den Computernamen, die IP-Adresse und die Netzwerkverbindung zu ändern, sofern Ihr Server über mehrere Ethernetverbindungen verfügt.

Wenn Sie Änderungen mit dem Assistenten vornehmen, aktualisiert dieser automatisch die vom Hostnamen oder der IP-Adresse abhängenden Dienste. DNS-Einträge werden vom Assistenten nicht aktualisiert. Sie müssen diese Einträge im Bereich „DNS“ der Server-App aktualisieren.

Sie können einen Domain-Namen für Ihren Server bei einer öffentlichen Domain-Name-Registrierungsstelle anfordern. Weitere Informationen finden Sie unter Registrieren des Internet-Hostnamens für den Server.
