Een Mac kiezen

U kunt het programma Server gebruiken op de computer die uw server wordt. U kunt het programma ook gebruiken om een computer met macOS Server op afstand te beheren.

Wanneer u het dialoogvenster 'Kies een Mac' ziet en u in dat venster de optie 'Deze Mac' kiest, wordt uw Mac als server geconfigureerd. Andere Mac-computers in het lokale netwerk waarop macOS Server is ingeschakeld, worden ook in dit dialoogvenster weergegeven. U kunt ook andere servers toevoegen via het IP-adres of de hostnaam.

U kunt Mac-computers waarop macOS Server is geïnstalleerd alleen op afstand beheren als de optie 'Sta beheer op afstand toe met Server' is ingeschakeld op de computer op afstand. U vindt deze optie in het paneel 'Instellingen' van het programma Server. U moet deze optie lokaal inschakelen op de server op afstand voordat u deze kunt beheren vanaf een andere Mac.

Als de server die u zoekt niet verschijnt, controleert u uw netwerkverbinding of de netwerkverbinding van de andere server.
