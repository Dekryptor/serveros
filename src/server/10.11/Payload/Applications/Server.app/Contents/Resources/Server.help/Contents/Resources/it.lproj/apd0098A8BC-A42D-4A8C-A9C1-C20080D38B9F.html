Scegliere un Mac

Puoi utilizzare l'app Server sul computer che sarà il tuo server oppure puoi utilizzarla per gestire in modalità remota un computer con OS X Server installato.

Se scegli “Questo Mac” quando viene visualizzata la finestra di dialogo “Scegli un Mac”, l'app Server designerà il tuo Mac come server. In questa finestra di dialogo appaiono anche gli altri computer Mac presenti sul network locale su cui è stato abilitato OS X Server. Inoltre, puoi aggiungere altri server in base al loro indirizzo IP o nome host.

Puoi gestire in modalità remota i computer Mac con OS X Server installato solo se l'opzione “Consenti amministrazione remota usando Server” è selezionata sul computer remoto. Questa opzione si trova nel pannello Impostazioni di Server app. Devi abilitare questa opzione localmente sul server remoto prima di gestirlo da un altro Mac.

Se il server che ti aspetti di vedere non viene mostrato, controlla la tua connessione network o la connessione network dell'altro server.
