Mac を選択する

Server アプリケーションは、サーバにするコンピュータ上で使用することも、OS X Server の稼働するコンピュータのリモート管理に使用することもできます。

「Mac を選択」ダイアログが表示されたときに、「この Mac」オプションを選択すると、お使いの Mac がサーバに昇格します。このダイアログには、ローカルネットワーク上の OS X Server に対応しているほかの Mac コンピュータも表示されます。ほかのサーバ（IP アドレスまたはホスト名）を追加することもできます。

OS X Server がインストールされているリモート Mac コンピュータを管理できます（リモートコンピュータの「Server によるリモート管理を許可」オプションにチェックマークが付いている場合のみ）。このオプションは、Server アプリケーションの「設定」パネルにあります。別の Mac から管理する前に、リモートサーバでこのオプションをローカルに有効にする必要があります。

期待通りのサーバが表示されない場合は、お使いの Mac のネットワーク接続またはほかのサーバのネットワーク接続を確認してください。
