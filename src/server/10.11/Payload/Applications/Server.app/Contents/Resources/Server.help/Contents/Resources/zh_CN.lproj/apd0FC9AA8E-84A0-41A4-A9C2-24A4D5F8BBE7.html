注册服务器的互联网主机名

若要让用户使用服务器在互联网上的主机名来访问它，则必须注册服务器的主机名。

  1. 获得互联网域名，如 example.com。

    如果没有域名，可以通过公共域名注册商购买一个。有关域名注册商的信息，请上网搜索。

  2. 通过域名注册商为此服务器注册唯一的主机名，如 server.example.com。

    如果您的组织拥有电脑支持组，请向他们申请一个主机名。如果没有，请向提供域名给您的域名注册商要求分配一个主机名称。

  3. 让 DNS 托管服务将此服务器的记录添加到其 DNS 服务器。

    如果您的组织拥有电脑支持组，请询问他们是否托管有 DNS 服务器。如果没有，您的 DNS 注册商可能会提供 DNS 托管服务，或者您也可以上网搜索，寻找其他提供商。
