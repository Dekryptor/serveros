Mac 선택

사용자의 서버인 컴퓨터에서 Server App을 사용하거나 이를 사용하여 OS X Server가 설치된 컴퓨터를 원격으로 관리할 수 있습니다.

‘Mac 선택’ 대화상자가 표시될 때 ‘이 Mac’에 대한 옵션을 선택한 경우 Server App이 서버가 될 사용자의 Mac을 승격합니다. OS X Server가 활성화되어 있는 로컬 네트워크 상의 다른 Mac 컴퓨터가 이 대화상자에 나타납니다. 또한 IP 주소 또는 호스트 이름으로 다른 서버를 추가할 수도 있습니다.

원격 컴퓨터에서 ‘Server를 사용하여 원격 관리 허용’ 옵션이 선택된 경우에만 OS X Server가 설치된 원격 Mac 컴퓨터를 관리할 수 있습니다. 이 옵션은 Server App의 설정 패널에 있습니다. 다른 Mac에서 원격 서버를 관리하기 전에 이 옵션을 로컬로 활성화해야 합니다.

예상대로 서버가 나타나지 않을 경우 네트워크 연결이나 다른 서버의 네트워크 연결을 확인하십시오.
