選擇 Mac

您可以在要用來做為伺服器的電腦上使用 Server App，或者您可以用它來遠端管理安裝了 OS X Server 的電腦。

當顯示「選擇 Mac」的對話框時，若您選擇了「這台 Mac」選項，Server App 會將您的 Mac 設定為伺服器。區域網路上其他啟用了 OS X Server 功能的 Mac 電腦會顯示在此對於中。您也可以透過 IP 位址或主機名稱來加入其他伺服器。

如果在遠端電腦上勾選了「允許使用 Server 進行遠端管理」的選項，您只能管理已安裝 OS X Server 的遠端 Mac 電腦。此選項位於 Server App 的「設定」面板。在您從另一部 Mac 對其進行管理之前，必須在遠端伺服器的本機上啟用此選項。

如果您沒有看到預期的伺服器，請檢查您的網路連線或其他伺服器的網路連線。
