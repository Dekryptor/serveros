Auswählen eines Mac

Sie können die Server-App auf dem Computer verwenden, den Sie als Server nutzen wollen, oder Sie können die Server-App für die Fernverwaltung eines Computers mit OS X Server einsetzen.

Wird im Dialogfenster „Mac auswählen“ die Option „Dieser Mac“ ausgewählt, wird Ihr Mac von der Server-App in einen Server umgewandelt. Andere Mac-Computer mit aktiviertem OS X Server in Ihrem lokalen Netzwerk werden in diesem Dialogfenster angezeigt. Sie können weitere Server mithilfe ihrer IP-Adresse oder ihres Hostnamens hinzufügen.

Sie können entfernte Mac-Computer mit installiertem OS X Server nur verwalten, wenn die Option „Entfernte Administration mit Server erlauben“ auf dem entfernten Computer aktiviert ist. Diese Option finden Sie im Bereich „Einstellungen“ der Server-App. Sie müssen diese Option lokal auf dem entfernten Server aktivieren, damit Sie diesen von einem anderen Mac aus verwalten können.

Wenn der erwartete Server nicht angezeigt wird, überprüfen Sie Ihre Netzwerkverbindung oder die Netzwerkverbindung des anderen Servers.
