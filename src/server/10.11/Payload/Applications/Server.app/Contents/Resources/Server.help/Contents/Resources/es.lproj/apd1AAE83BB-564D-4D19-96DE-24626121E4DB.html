Asistente de cambio de nombre de host

Utilice el asistente de cambio de nombre de host para cambiar el nombre del host, el nombre del ordenador, la dirección IP y, si su servidor cuenta con múltiples conexiones Ethernet, la conexión de red.

Al realizar cambios dentro del asistente, este actualiza automáticamente los servicios de actualizaciones que dependen del nombre de host o de la dirección IP. No actualiza los registros DNS; deberá actualizarlos en el panel DNS de la aplicación Server.

Puede obtener un nombre de dominio para su servidor desde un registrador de nombres de dominios públicos. Para obtener más información, consulte Registrar el nombre de host de Internet del servidor.
