Gestione AirPort

Il pannello “Gestione AirPort” appare se l'interfaccia network primaria del server è connessa a Internet tramite una base AirPort Extreme o AirPort Time Capsule. Inoltre, una base AirPort Extreme o AirPort Time Capsule, deve avere l'opzione “Condivisione connessione” impostata su “Condividi indir. IP pubbl.” (cioè una connessione Internet).

Utilizza il pannello “Gestione AirPort” per autorizzare OS X Server a gestire una base AirPort Extreme (802.11n) o AirPort Time Capsule.

Per autorizzare la gestione AirPort, inserisci la password del dispositivo AirPort (non la password del network Wi-Fi).

Server app configura il server DHCP del dispositivo AirPort per prenotare l'indirizzo IP del server e per assegnare sempre tale indirizzo IP al server.

Puoi utilizzare Server app per scegliere servizi pubblici ai quali è possibile accedere da computer su Internet. OS X Server configura il tuo dispositivo AirPort per esporre tali servizi pubblici su Internet. Il processo di esposizione di servizi singoli su Internet viene chiamato mappatura porta o ridirezionamento porta.

Puoi anche usare Server app per consentire agli utenti di accedere al tuo network wireless usando il loro nome utente e password, invece della password del network Wi-Fi. In questo caso, il server fornisce il protocollo RADIUS (Remote Authentication Dial In User Service) per il dispositivo AirPort e autorizza tutti gli account utente ad accedere al network wireless.

Se desideri esporre i servizi pubblici su Internet tramite un dispositivo AirPort o altri router e il server VPN fornisce il servizio agli utenti che hanno il loro dispositivo AirPort o altri router configurati per condividere una connessione Internet, l'indirizzo IP del server non può iniziare con gli stessi tre numeri degli indirizzi IP degli utenti VPN, come ad esempio 10.0.1 o 192.168.1.

Per gestire il dispositivo AirPort dopo aver installato l'app Server, seleziona il dispositivo AirPort nella barra laterale dell'app Server, accanto alla parte superiore della sezione Hardware.
