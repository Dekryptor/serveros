De internethostnaam van de server registreren

Als u wilt dat internetgebruikers via de hostnaam van de server toegang tot de server kunnen krijgen, moet u de hostnaam van de server registreren.

  1. Verkrijg een internetdomeinnaam zoals voorbeeld.com.

    Als u nog geen domeinnaam hebt, kunt u er een aanschaffen bij een openbare domeinnaamregistrar. Zoek op het internet naar informatie over domeinnaamregistrars.

  2. Registreer een unieke hostnaam voor deze server, zoals server.example.com, bij uw domeinnaamregistrar.

    Als uw organisatie een computerondersteuningsgroep heeft, kunt u daar een hostnaam aanvragen. Zo niet, dan kunt u door de domeinnaamregistrar waar u uw domeinnaam hebt aangeschaft, een hostnaam laten toewijzen.

  3. Laat een aanbieder van een DNS-hostingvoorziening records voor deze server toevoegen aan zijn DNS-servers.

    Als uw organisatie een computerondersteuningsgroep heeft, informeert u of zij DNS-servers hosten. Zo niet, is het mogelijk dat uw DNS-registrar een DNS-hostingvoorziening aanbiedt, maar u kunt ook op het internet zoeken naar een aanbieder.
