Register the server’s Internet host name

To let users access the server by using its host name on the Internet, you must register the server’s host name.

  1. Obtain an Internet domain name, such as example.com.

    If you don’t already have a domain name, you can purchase one from a public domain name registrar. For information about domain name registrars, search the web.

  2. Register a unique host name for this server—such as server.example.com—with your domain name registrar.

    If your organization has a computer support group, request a host name from them. Otherwise, work with the domain name registrar where you obtained your domain name in order to assign a host name.

  3. Have a DNS hosting service add records for this server to its DNS servers.

    If your organization has a computer support group, ask if they host DNS servers. Otherwise, your DNS registrar might provide DNS hosting service, or you can search the web for a provider.
