Gestion AirPort

Gérez un appareil AirPort pour donner à d’autres ordinateurs sur Internet un accès à des services, et permettre aux utilisateurs de se connecter à votre réseau sans fil avec leur nom et leur mot de passe.

La sous-fenêtre Gestion AirPort s’affiche si l’interface réseau principale de votre serveur est connectée à Internet par l’intermédiaire de votre borne d’accès AirPort Extreme ou de votre Time Capsule. De plus, l’option Partage de connexion de votre borne d’accès AirPort Extreme ou de votre Time Capsule doit être définie sur « Partager une adresse IP publique » (c’est-à-dire une connexion Internet).

Utilisez la sous-fenêtre Gestion AirPort pour autoriser OS X à gérer un appareil AirPort. OS X Server peut gérer une borne d’accès AirPort Extreme (802.11n) ou une Time Capsule.

Pour autoriser la gestion AirPort, saisissez le mot de passe de la borne d’accès ou de la Time Capsule (et non le mot de passe du réseau sans fil).

L’app Server configure le serveur DHCP de votre appareil AirPort pour réserver l’adresse IP de votre serveur et pour toujours attribuer cette adresse IP à votre serveur.

Vous pouvez utiliser l’app Server pour désigner les services publics accessibles depuis les ordinateurs, via Internet. OS X Server configure votre appareil AirPort pour exposer ces services publics sur Internet. Le processus d’exposition de services individuels sur Internet se nomme « mappage de ports » ou « redirection de ports ».

Vous pouvez également utiliser l’app Server pour laisser les utilisateurs se connecter à votre réseau sans fil avec leur nom d’utilisateur et leur mot de passe au lieu d’utiliser le mot de passe du réseau Wi-Fi. Dans ce cas, votre serveur fournit le service RADIUS (Remote Authentication Dial In User Service) pour votre appareil AirPort et autorise tous les comptes utilisateur du serveur à accéder à votre réseau sans fil.

Si vous souhaitez exposer des services publics sur Internet par le biais d’un appareil AirPort ou un autre routeur, et si votre serveur fournit un service VPN aux utilisateurs ayant leur propre appareil AirPort ou un autre routeur configuré afin de partager une connexion Internet, l’adresse IP de votre serveur ne peut pas commencer par les trois mêmes nombres que les adresses IP des utilisateurs du VPN, par exemple 10.0.1 ou 192.168.1.

Pour gérer votre appareil AirPort après l’installation de l’app Server, sélectionnez votre appareil AirPort dans la barre latérale. Il se trouve dans la partie supérieure de la barre latérale, dans la section Matériel.
