Hostnaam wijzigen

Informatie over de assistent 'Hostnaam wijzigen'.

Met de assistent 'Hostnaam wijzigen' kunt u de volgende onderdelen wijzigen: de hostnaam, de computernaam, het IP-adres en de netwerkverbinding (alleen als de server meerdere Ethernet-verbindingen heeft).

Wanneer u wijzigingen aanbrengt met de assistent, worden voorzieningen die afhankelijk zijn van de hostnaam of het IP-adres automatisch bijgewerkt. DNS-records worden niet automatisch bijgewerkt. Deze moet u bijwerken in het paneel 'DNS' van het programma Server.

U kunt een domeinnaam voor uw server aanvragen bij een openbare domeinnaamregistrar. Zie De internethostnaam van de server registreren voor meer informatie.
