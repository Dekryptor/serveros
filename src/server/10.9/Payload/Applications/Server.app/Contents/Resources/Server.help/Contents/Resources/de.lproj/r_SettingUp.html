Konfigurieren von Diensten

Überwachen Sie die Statusinformationen, während die Server-App Ihren Computer als Server konfiguriert.

Verwenden Sie das Fenster „Dienste konfigurieren“, um die Aktionen zu überwachen, die der Serverassistent ausführt, um Ihren Server zu konfigurieren und zu testen, ob die Dienste und die Netzwerkverbindung funktionieren.
