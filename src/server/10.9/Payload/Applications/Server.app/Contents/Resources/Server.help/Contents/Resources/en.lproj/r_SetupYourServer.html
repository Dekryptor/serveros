Getting Started

Before you set up your server, you need to agree to the end user license agreement (EULA). You are also prompted to enter the password for an administrator account on your Mac.

If you want to review the contents of the EULA at a later time, it is available online at this site:

www.apple.com/legal/sla/

For more information about entering your administrator password, see this help topic: If you're asked for an administrator name and password
