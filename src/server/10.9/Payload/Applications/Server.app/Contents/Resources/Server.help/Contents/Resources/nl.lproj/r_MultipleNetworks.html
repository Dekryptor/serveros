Meerdere netwerken gevonden

Het paneel 'Meerdere netwerken gevonden' wordt weergegeven als uw server is verbonden met meerdere actieve Ethernet-netwerken. Kies de primaire netwerkverbinding van de server uit het venstermenu.

Wanneer u in dit paneel op 'Ga door' klikt, wordt aan de hand van het gekozen netwerk de netwerkidentiteit, inclusief het IP-adres en de hostnaam, van de server vastgesteld. Er wordt geprobeerd het IP-adres van de server op te vragen bij een DHCP-server in het netwerk. Vervolgens wordt op basis van dit IP-adres de hostnaam van de server opgevraagd bij een DNS-server in het netwerk. Indien nodig kunt u in een later paneel de netwerkconfiguratie en de identiteit van de server wijzigen.

Als u na de serverconfiguratie een AirPort-apparaat wilt configureren als gateway, kiest u de interface die verbinding maakt met uw AirPort-apparaat (meestal Wi-Fi). U kunt op een later moment de gateway configureren met het programma Server.
