连接服务器

设置用于标识网络上您的服务器的地址和名称。

使用“连接服务器”面板来设置其他人用于连接您的服务器的名称。

选项描述

电脑名称

这会显示在其他 Mac 电脑的 Finder 边栏中的“AirDrop”或“共享”部分。其他电脑可以使用电脑名称来连接您的服务器，从而进行文件共享或屏幕共享（前提是这些服务已开启）。

主机名称

服务器设置助理会尝试从网络上可用的 DNS 服务器为您确定主机名。

如果服务器设置助理无法确定您服务器的主机名，可能有以下原因：
  • 助理无法在内部网或互联网上找到针对您的服务器配置正确的 DNS 服务器。
  • 您的服务器从 DHCP 服务器（可能是您的互联网路由器）接收到了错误的网络连接设置。
  • 您的内部网没有 DHCP 服务器，因此您必须手动输入服务器的网络连接设置。
有关 DNS 服务器必须如何配置的信息，请参阅下面的选项。如果您的服务器从 DHCP 服务器接收到错误的网络连接设置，或者您的内部网没有 DHCP 服务器，那么您可以设置一个，或者可以在“更改网络”面板中更改您服务器的网络连接设置。

如果您输入主机名，请遵循以下原则：

本地网络的主机名：

若要让用户仅通过内部网访问服务器，请输入以 .local 结尾的主机名，如 server.local。这是一个本地网络名称，也称为本地主机名。

只有在您的本地 IP 子网上的电脑可以使用您服务器的本地网络名称访问您的服务器。通常情况下，同一个子网上的电脑的 IP 地址会以相同的三组数字开头，如 192.168.1。Windows 电脑必须安装了用于 Windows 的 Bonjour 打印服务。它可在 support.apple.com/kb/DL999?viewlocale=zh_CN 上获得。

如果服务器的主机名称是它的本地网络名称，则该服务器不支持 Kerberos 或单点登录鉴定。

专用网络的主机名：

若要让用户从互联网，使用服务器在内部网上的主机名以及使用 VPN 连接来访问此服务器，请输入以 .private 结尾的主机名，如 server.example.private。

如果您的内部网有 DNS 服务器，它必须经过配置以使用服务器的主机名来查找它的 IP 地址以及使用它的 IP 地址来查找主机名。这意味着 DNS 服务器必须有用于正向查找的记录（A 记录）以及用于反向查找的记录（PTR 记录）。请要求您的 DNS 服务器管理员为您配置这些 DNS 记录。

如果您的内部网没有 DNS 服务器，服务器设置助理会配置您的服务器以提供最小化 DNS 服务，以便用户可以使用您的服务器在内部网上的主机名称。

互联网的主机名：

若要让用户使用服务器在互联网和内部网上的同一主机名访问服务器，请输入您通过域名注册商注册的主机名。

您用于互联网的 DNS 服务器必须经过配置以使用服务器的主机名来查找它的 IP 地址以及使用它的 IP 地址来查找主机名。这意味着 DNS 服务器必须有用于正向查找的记录（A 记录）以及用于反向查找的记录（PTR 记录）。请要求您的 ISP 或 DNS 托管服务为您配置这些 DNS 记录。

如果您的内部网具有 DNS 服务器，它也必须使用记录进行配置以用于正向查找和反向查找。请要求您的内部网 DNS 服务器管理员为您配置这些 DNS 记录。

如果您的内部网没有 DNS 服务器，服务器设置助理会配置您的服务器以提供最小化 DNS 服务，以便用户可以使用您的服务器在内部网上的主机名称。

注： 请小心选择主机名。事后更改主机名会造成干扰，尤其是对用户而言。例如，用户必须重新配置它们的电脑并通知其他用户他们的电子邮件地址已更改。

网络地址

在网络上唯一标识您的服务器的数字 IP 地址。网络地址可能被初始设定为您的服务器从网络上的 DHCP 服务器获取的 IP 地址。点按“更改网络”来更改网络地址。

您的服务器应该总是具有相同的 IP 地址，以便其他用户能够可靠地连接到它。如果服务器的 IP 地址发生改变，用户可能无法连接到它。

以下是防止服务器的 IP 地址发生改变的方法：

  • 如果服务器是使用 DHCP 从路由器获得互联网连接的，如 AirPort 基站，则您可以配置路由器的 DHCP 以总是将相同的 IP 地址分配给服务器。此功能也称为“静态映射”或“DHCP 保留”。如果您的网络拥有不同的 DHCP 服务器，请要求管理员为您的服务器配置静态映射。
  • 如果您的服务器有直接互联网连接，例如通过 DSL 调制解调器或线缆调制解调器而不是通过路由器连接到互联网，请从您的 ISP 获取静态（固定）IP 地址。如果需要，您通常可以付费升级到一个网络地址。
  • 如果您的服务器未连接到互联网且您的内部网没有 DHCP 服务器，可以点按“更改网络”来手动输入一个静态 IP 地址。

更改网络

点按以更改服务器的网络地址或其他网络连接设置。

您的 Mac 也有本地主机名，例如 server.local，它不会在此面板中显示且未使用任何 DNS 服务器进行注册。与服务器在同一个内部网（IP 子网）的电脑可以使用服务器的本地主机名（而不是注册主机名）连接到您的服务器。本地主机名默认来源于电脑名称。若要更改它，请在完成服务器设置后使用 Server 应用程序。（选择您的服务器，然后选取“网络”面板。）
