Ouverture de session Server

Ouvrez une session à l’aide d’un compte administrateur du serveur auquel vous souhaitez vous connecter via l’app Server. Lorsque vous êtes connecté à un serveur, son nom s’affiche dans la section Matériel de la barre latérale.
OptionDescription

Nom de l’hôte ou Adresse IP

L’app Server renseigne automatiquement le nom d’hôte du serveur que vous ajoutez, si vous avez sélectionné le serveur avant l’affichage de cette zone de dialogue.

Dans le cas contraire, saisissez l’un des éléments suivants :

  • Le nom d’hôte du serveur : Il peut s’agir d’un nom de domaine Internet complet, tel que serveurmac.exemple.com, d’un nom de domaine intranet complet, tel que miniserveur.exemple.private, ou du nom d’hôte local du serveur, tel que serveurmac.local.
  • Adresse IP : L’adresse IP numérique du serveur, telle que 192.168.1.100.

Nom d’administrateur

Saisissez le nom d’un compte administrateur du serveur que vous ajoutez. Si le Mac que vous utilisez et le serveur que vous ajoutez correspondent à des ordinateurs distincts, chacun d’eux dispose de son propre compte administrateur, et il est possible que les noms et mots de passe administrateur soient différents.

Mot de passe administrateur

Saisissez le mot de passe correspondant au compte administrateur dont vous avez saisi le nom au-dessus.
