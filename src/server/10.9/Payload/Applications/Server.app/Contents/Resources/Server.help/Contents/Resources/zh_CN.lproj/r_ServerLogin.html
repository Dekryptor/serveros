服务器登录

使用 Server 应用程序和要连接到的服务器的管理员帐户进行登录。 在连接到服务器后，其名称会显示在边栏的“硬件”部分中。
选项描述

主机名或 IP 地址

如果在此对话框出现之前选择了要添加的服务器，则 Server 应用程序会自动填写该服务器的主机名。

否则，请输入以下一项：

  • 服务器的主机名：可以是完全合格的互联网域名称，如 macserver.example.com；完全合格的内部网域名称，如 miniserver.example.private；或者服务器的本地主机名，如 macserver.local。
  • IP 地址：服务器的数字 IP 地址，如 192.168.1.100。

管理员名称

输入所添加服务器的管理员帐户的名称。如果使用的 Mac 和添加的服务器是不同的电脑，则它们具有单独的管理员帐户，而管理员名称和密码可能不同。

管理员密码
