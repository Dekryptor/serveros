Gestione AirPort

Gestire un dispositivo AirPort per fornire accesso ai servizi selezionati ai computer su Internet e per consentire agli utenti di eseguire il login al network con il loro nome e la loro password.

Il pannello “Gestione AirPort” appare se l'interfaccia network primaria del server è connessa a Internet tramite AirPort Extreme Base Station o Time Capsule. Inoltre, AirPort Extreme Base Station o Time Capsule, devono avere l'opzione “Condivisione connessione” deve essere impostata su “Condividi indir. IP pubbl.” (cioè una connessione Internet).

Usa il pannello “Gestione AirPort” per autorizzare OS X Server alla gestione di un dispositivo AirPort. OS X Server può gestire una base AirPort Extreme (802.11n) o un dispositivo Time Capsule.

Per autorizzare la gestione AirPort, inserisci la password per la base station o per Time Capsule (non la password del network Wi-Fi).

Server app configura il server DHCP del dispositivo AirPort per prenotare l'indirizzo IP del server e per assegnare sempre tale indirizzo IP al server.

Puoi utilizzare Server app per scegliere servizi pubblici ai quali è possibile accedere da computer su Internet. OS X Server configura il tuo dispositivo AirPort per esporre tali servizi pubblici su Internet. Il processo di esposizione di servizi singoli su Internet viene chiamato “mappatura porta” o “ridirezionamento porta”.

Puoi anche usare Server app per consentire agli utenti di accedere al tuo network wireless usando il loro nome utente e password, invece della password del network Wi-Fi. In questo caso, il server fornisce un RADIUS (Remote Authentication Dial In User Service) per il dispositivo AirPort e autorizza tutti gli account utente sul server ad accedere al network wireless .

Se desideri esporre i servizi pubblici su Internet tramite un dispositivo AirPort o altri router e il server VPN fornisce il servizio agli utenti che hanno il loro dispositivo AirPort o altri router configurati per condividere una connessione Internet, l'indirizzo IP del server non può iniziare con gli stessi tre numeri degli indirizzi IP degli utenti VPN, come ad esempio 10.0.1 o 192.168.1.

Per gestire il dispositivo AirPort dopo aver installato Server app, seleziona il dispositivo AirPort nella barra laterale. Sarà vicino alla parte superiore della barra laterale nella sezione Hardware.
