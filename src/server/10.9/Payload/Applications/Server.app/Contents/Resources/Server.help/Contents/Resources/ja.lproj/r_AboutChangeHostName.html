ホスト名を変更する

「ホスト名を変更アシスタント」について説明します。

「ホスト名変更アシスタント」を使用すると、ホスト名、コンピュータ名、IP アドレスの変更、およびサーバに複数の Ethernet 接続がある場合はネットワーク接続の変更を行えます。

このアシスタント内で行った変更に従って、該当するホスト名または IP アドレスを使用するサービスが自動的にアップデートされます。DNS レコードはアップデートされません。DNS レコードは、Server アプリケーションの「DNS」パネルでアップデートする必要があります。

サーバのドメイン名は、パブリックドメイン名登録業者から入手できます。詳しくは、「サーバのインターネットホスト名を登録する」を参照してください。
