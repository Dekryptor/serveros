Se han detectado varias redes

El panel “Se han detectado varias redes” aparece si el servidor está conectado a más de una red Ethernet activa. Elija la conexión de red principal del servidor desde el menú desplegable.

Al hacer clic en Continuar en este panel, el asistente de configuración del servidor usará la red que elija para determinar la identidad de la red del servidor, incluida su dirección IP y nombre de host. El asistente de configuración del servidor trata de obtener la dirección IP del servidor de un servidor DHCP de la red y usa esta dirección IP para obtener el nombre de host del servidor de un servidor DNS de la red. En caso necesario, puede cambiar la configuración de red del servidor y la identidad en un panel posterior.

Si va a configurar un dispositivo AirPort como puerta de enlace después de terminar la configuración del servidor, elija la interfaz que se conecta a su dispositivo AirPort (normalmente Wi-Fi). Más tarde puede usar la aplicación Server para configurar la puerta de enlace.

Nota: Si está controlando el asistente de configuración del servidor desde otro ordenador mediante el uso de la función de compartir pantalla y cambia el servicio de red, la sesión de compartir pantalla podría finalizar. Pruebe a realizar una conexión de compartir pantalla usando el nuevo servicio de red o vaya al servidor y use su teclado y pantalla para continuar la configuración localmente.
