Più network rilevati

Il pannello “Più network rilevati” appare se il server è connesso a più di un network Ethernet attivo. Scegli la connessione network primaria del server dal menu a comparsa.

Quando si fa clic su Continua in questo pannello, l'assistente di configurazione del server utilizza la rete che hai scelto per determinare l'identità network del server, compresi l'indirizzo IP e il nome host. L'assistente di configurazione server tenta di ottenere l'indirizzo IP del server da un server DHCP sul network e utilizza questo indirizzo IP per ottenere il nome host del server da un server DNS sul network. Se necessario, puoi modificare la configurazione e l'identità network del server in un pannello successivo.

Se hai intenzione di configurare un dispositivo AirPort come gateway dopo aver completato la configurazione del server, scegli l'interfaccia che si connette al dispositivo AirPort (normalmente Wi-Fi). Puoi in seguito utilizzare Server app per configurare il gateway.

Nota: Se stai controllando l'assistente di configurazione del server da un altro computer tramite la condivisione dello schermo e modifichi il servizio network, la sessione di condivisione dello schermo potrebbe terminare. Prova ad effettuare una connessione di condivisione schermo utilizzando il nuovo servizio network oppure vai al server e utilizza la sua tastiera e il suo monitor per continuare l'installazione da locale.
