AirMac マネージメント

AirMac デバイスを管理して、インターネット上のほかのコンピュータがサービスにアクセスしたり、ユーザがユーザ名とパスワードを入力してワイヤレスネットワークにログインしたりできるように設定できます。

サーバのプライマリ・ネットワーク・インターフェイスが AirMac Extreme ベースステーションまたは Time Capsule 経由でインターネットに接続されている場合は、「AirMac マネージメント」パネルが表示されます。このパネルを表示するには、AirMac Extreme ベースステーションまたは Time Capsule で「接続共有」オプションを「パブリック IP アドレスを共有」（つまり、インターネット接続を共有）に設定する必要もあります。

「AirMac マネージメント」パネルでは、OS X Server から AirMac デバイスを管理することを許可できます。OS X Server では、AirMac Extreme ベースステーション（802.11n）または Time Capsule を管理できます。

AirMac の管理を許可するには、ベースステーションまたは Time Capsule のパスワード（Wi-Fi ネットワークのパスワードとは異なります）を入力します。

Server アプリケーションによって、サーバの IP アドレスを予約してその IP アドレスを常にこのサーバに割り当てるように、AirMac デバイスの DHCP サーバが構成されます。

Server アプリケーションを使用することで、インターネット上のコンピュータからアクセスできる公開サービスを指定できます。これらのサービスをインターネットに公開するように AirMac デバイスが構成されます。サービスをインターネットに公開する処理は、「ポートマッピング」または「ポートフォワード」と呼ばれます。

Server アプリケーションを使用して、ユーザが Wi-Fi ネットワークパスワードではなくユーザ名とパスワードを使用してワイヤレスネットワークにログインできるようにすることもできます。この場合、RADIUS（Remote Authentication Dial In User Service）を AirMac デバイスに提供することで、サーバ上のすべてのユーザアカウントがワイヤレスネットワークにアクセスすることが承認されます。

AirMac デバイスまたはその他のルーターを介してサービスをインターネットに公開する場合、サーバが VPN サービスを提供していて、VPN ユーザが自分の AirMac デバイスまたはその他のルーターでインターネット接続を共有しているときは、サーバの IP アドレスと VPN ユーザの IP アドレスの最初の 3 桁の数字が重ならないようにしてください（10.0.1 と 192.168.1 にするなど）。

Server アプリケーションのインストール後に AirMac デバイスを管理する場合は、サイドバーで AirMac デバイスを選択します。AirMac デバイスは、サイドバー上部の「ハードウェア」セクションに表示されます。
