호스트 이름 변경하기

호스트 이름 변경 지원에 관해 배웁니다.

호스트 이름 변경 지원을 사용하여 호스트 이름, 컴퓨터 이름, IP 주소와 서버가 여러 이더넷에 연결되는 경우 네트워크 연결을 변경할 수 있습니다.

지원되는 범위에서 변경하면 호스트 이름 또는 IP 주소에 따라 자동으로 서비스를 업데이트합니다. DNS 레코드를 업데이트하지 않습니다. Server App의 DNS 패널에서 업데이트해야 합니다.

일반 도메인 이름 등록업체에서 서버의 도메인 이름을 얻을 수 있습니다. 자세한 정보는 서버의 인터넷 호스트 이름 등록하기를 참조하십시오.
