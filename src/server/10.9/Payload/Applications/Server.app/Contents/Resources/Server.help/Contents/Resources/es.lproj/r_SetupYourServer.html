Primeros pasos

Antes de configurar su servidor, debe aceptar el contrato de licencia del usuario final (CLUF). También se le pedirá que introduzca la contraseña para una cuenta de administrador del Mac.

Si desea revisar el contenido del CLUF más tarde, está disponible en línea en este sitio:

www.apple.com/es/legal/sla/
