Plusieurs réseaux détectés

La sous-fenêtre Plusieurs réseaux détectés s’affiche si votre serveur est connecté à plusieurs réseaux Ethernet actifs. Choisissez la connexion réseau principale du serveur dans le menu local.

Lorsque vous cliquez sur Continuer dans cette sous-fenêtre, l’assistant réglages du serveur utilise le serveur que vous indiquez pour déterminer l’identité réseau du serveur, notamment ses adresse IP et nom d’hôte. L’assistant réglages du serveur tente d’obtenir l’adresse IP du serveur depuis un serveur DHCP sur le réseau et utilise cette même adresse pour obtenir le nom d’hôte depuis un serveur DNS sur le réseau. Au besoin, modifiez la configuration réseau du serveur et son identité dans la sous-fenêtre suivante.

Si vous envisagez de configurer un périphérique AirPort en tant que passerelle une fois la configuration du serveur terminée, choisissez l’interface se connectant à votre périphérique AirPort (généralement Wi-Fi). Vous pourrez utiliser l’app Server ultérieurement pour configurer la passerelle.

Remarque : Si vous gérez l’assistant réglages du serveur depuis un autre ordinateur à l’aide du partage d’écran et que vous modifiez le service réseau, la session de partage peut être interrompue. Essayez d’établir une nouvelle connexion de partage d’écran à l’aide du nouveau service réseau, ou rendez-vous sur le serveur et poursuivez la configuration localement à l’aide de son clavier et de son écran.
