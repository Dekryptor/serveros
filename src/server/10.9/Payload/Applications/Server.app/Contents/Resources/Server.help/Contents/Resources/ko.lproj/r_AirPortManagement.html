AirPort 관리하기

AirPort 장비를 관리하여 인터넷에 있는 다른 컴퓨터에 서비스 접근을 제공하고 다른 사용자가 자신의 이름 및 암호를 사용하여 무선 네트워크에 로그인할 수 있도록 합니다.

AirPort Extreme 베이스 스테이션 또는 Time Capsule을 통해 서버의 주 네트워크 인터페이스가 인터넷에 연결되면 AirPort 관리 패널이 나타납니다. 또한 AirPort Extreme 베이스 스테이션 또는 Time Capsule의 연결 공유 옵션 설정이 ‘공용 IP 주소 공유’(즉, 인터넷 연결)로 설정되어 있어야 합니다.

AirPort 관리 패널을 사용하여 OS X Server로 AirPort 장비를 관리할 수 있도록 인증하십시오. OS X Server는 AirPort Extreme 베이스 스테이션(802.11n) 또는 Time Capsule을 관리할 수 있습니다.

AirPort 관리를 인증하려면 베이스 스테이션 또는 Time Capsule 암호(Wi-Fi 네트워크 암호 아님)를 입력하십시오.

Server App이 사용자 서버의 IP 주소를 예약하고 사용자의 서버에 항상 해당 IP 주소를 할당하도록 AirPort 장비의 DHCP 서버를 구성합니다.

Server App을 사용하여 인터넷에 있는 컴퓨터에서 접근할 수 있는 공용 서비스를 지정할 수 있습니다. OS X Server는 인터넷에서 공용 서비스를 공개하도록 AirPort 장비를 구성합니다. 개별 서비스를 인터넷에 공개하는 과정은 ‘포트 매핑’ 또는 ‘포트 전달’이라고 합니다.

Server App을 사용하여 Wi-Fi 네트워크 암호 대신에 사용자의 이름 및 암호를 사용하여 무선 네트워크에 로그인하도록 할 수도 있습니다. 이런 경우 사용자의 서버가 AirPort 장비에 RADIUS(Remote Authentication Dial In User Service)를 제공하여 무선 네트워크에 접근할 수 있도록 서버에 있는 모든 사용자 계정을 인증합니다.

AirPort 장비 또는 다른 라우터를 통해 인터넷에 공용 서비스를 공개하고 사용자의 서버가 인터넷 연결을 공유하도록 구성된 개인 AirPort 장비 또는 기타 라우터를 소유하고 있는 사용자에게 VPN 서비스를 제공하도록 하려면 사용자 서버의 IP 주소는 10.0.1 또는192.168.1과 같이 VPN 사용자의 IP 주소와 동일한 세 숫자로 시작해서는 안됩니다.

Server App을 설치한 후에 사이드바에서 사용자의 AirPort 장비를 선택하고 AirPort 장비를 관리하십시오. 사용자의 AirPort 장비는 하드웨어 섹션의 사이드바 상단 근처에 있습니다.
