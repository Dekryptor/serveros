連接您的伺服器

設定在網路上識別您伺服器的位址和名稱。

使用「連接您的伺服器」面板來設定其他用來連接您伺服器的名稱。

選項說明

電腦名稱

這會顯示在其他 Mac 電腦上 Finder 側邊欄的 AirDrop 或「共享」區段中。其他電腦可使用此電腦名稱來連接您的伺服器，進行檔案共享或螢幕共享（若其已啟用這些服務）。

主機名稱

伺服器設定輔助程式會為您嘗試從網路上的可用 DNS 伺服器來判斷主機名稱。

有時候伺服器設定輔助程式可能無法判斷您伺服器的主機名稱，部分原因如下：
  • 輔助程式在您的內部網路或已為伺服器正確設定的 Internet 上，找不到 DNS 伺服器。
  • 您的伺服器已從 DHCP 伺服器（可能是您的 Internet 路由器）接收到錯誤的網路連線設定。
  • 您的內部網路沒有 DHCP 伺服器，因此您必須手動輸入伺服器的網路連線設定。
如需 DNS 伺服器所需設定的相關資訊，請參閱下方的選項。如果您的伺服器已從 DHCP 伺服器接收到錯誤的網路連線設定，或者如果您的內部網路沒有 DHCP 伺服器，您可以設定一個，或者在「更改網路」面板中更改您伺服器的網路連線設定。

如果您要輸入主機名稱，請依照下列準則：

區域網路的主機名稱：

若只想要讓使用者從內部網路連接您的伺服器，請輸入結尾為 .local 的主機名稱，如 server.local。這是區域網路名稱，也稱為本機主機名稱。

只有在您區域 IP 子網路上的電腦可使用伺服器的區域網路名稱來進行連接。通常，電腦若位於相同的子網路上，其 IP 位址會有相同的前三組數字，例如 192.168.1。Windows 電腦必須已安裝「Windows 版 Bonjour 列印服務」。其可在 support.apple.com/kb/DL999?viewlocale=zh_TW 取得。

如果伺服器的主機名稱是其區域網路名稱，伺服器便不支援 Kerberos 或單一登入認證。

專用網路的主機名稱：

若要讓使用者在您的內部網路上使用伺服器的主機名稱和從 Internet 使用 VPN 連線來進行連接，請輸入以 .private 結尾的主機名稱，如 server.example.private。

如果您的內部網路有 DNS 伺服器，其必須設定為使用伺服器的主機名稱來查詢其 IP 位址，反之亦然。這表示 DNS 伺服器必須有向前查詢（A 記錄）和反向查詢（PTR 記錄）的記錄。請您的 DNS 伺服器管理者為伺服器設定這些 DNS 記錄。

如果您的內部網路沒有 DNS 伺服器，伺服器設定輔助程式會設定您的伺服器以提供最精簡的 DNS 服務，讓使用者可在您的內部網路上使用伺服器的主機名稱。

Internet 的主機名稱：

若要讓使用者在 Internet 和內部網路上使用相同的主機名稱來連接您的伺服器，請輸入您向網域名稱註冊商進行註冊的主機名稱。

您用於 Internet 的 DNS 伺服器必須設定為使用伺服器的主機名稱來查詢其 IP 位址，反之亦然。這表示 DNS 伺服器必須有向前查詢（A 記錄）和反向查詢（PTR 記錄）的記錄。請您的 ISP 或 DNS 託管服務為您設定這些 DNS 記錄。

如果您的內部網路有 DNS 伺服器，其也必須使用向前查詢和反向查詢的記錄來加以設定。請您的內部網路 DNS 伺服器管理者為您設定這些 DNS 記錄。

如果您的內部網路沒有 DNS 伺服器，伺服器設定輔助程式會設定您的伺服器以提供最精簡的 DNS 服務，讓使用者可在您的內部網路上使用伺服器的主機名稱。

註: 請謹慎選擇主機名稱。日後若更改主機名稱，可能會有很大的影響（尤其對使用者而言）。例如，使用者必須重新設定其電腦，並告訴其他人其電子郵件位址已更改。

網路位址

數值 IP 位址，專門用來識別您在網路上的伺服器。網路位址一開始可設定為您伺服器從網路上 DHCP 伺服器所接收的 IP 位址。按一下「更改網路」來更改網路位址。

您的伺服器應一律使用相同的 IP 位址，以便其他使用者可順利連接。如果您伺服器的 IP 位址有所更動，使用者可能無法與其連接。

以下是讓您伺服器的 IP 位址免於更改的方式：

  • 如果您的伺服器從路由器透過 DHCP 取得其 Internet 連線，如「AirPort 基地台」，您可以將路由器的 DHCP 設為一律對您的伺服器指定相同 IP 位址。此功能也稱為「靜態對應」（static mapping）或「DHCP 保留」。如果您的網路有不同的 DHCP 伺服器，請管理者為您的伺服器設定靜態對應。
  • 如果您的伺服器直接連接到 Internet，例如透過 DSL 數據機或非路由器的纜線數據機，請從您的 ISP 取得靜態（固定）IP 位址。如有需要，您通常可付費升級為固定 IP 位址。
  • 如果您的伺服器並未連接 Internet 且您的內部網路沒有 DHCP 伺服器，請按一下「更改網路」來手動輸入靜態 IP 位址。

更改網路

按一下來更改伺服器的網路位址或其他網路連線設定。

您的 Mac 也有本機主機名稱，如 server.local，這不會顯示在此面板中且不會向任何 DNS 伺服器註冊。與您伺服器位於相同內部網路（IP 子網路）上的電腦可使用其本機主機名稱來連接您的伺服器，而非使用註冊的主機名稱。依照預設，本機主機名稱是從電腦名稱衍生而來。若要更改，請在完成伺服器設定後使用 Server App 進行更改。（選擇您的伺服器，然後選擇「網路」面板。）
