Vorbereitende Schritte

Bevor Sie Ihren Server konfigurieren, müssen Sie der Endbenutzer-Lizenzvereinbarung (EULA) zustimmen. Darüber hinaus werden Sie aufgefordert, das Kennwort eines Administratoraccounts auf Ihrem Mac einzugeben.

Wenn Sie den Inhalt der EULA zu einem späteren Zeit nochmals lesen möchten, finden Sie diese online unter:

www.apple.com/legal/sla/

Weitere Informationen zum Eingeben Ihres Administratorkennworts finden Sie unter: Aufforderung zur Eingabe eines Administratornamens und des zugehörigen Kennworts
