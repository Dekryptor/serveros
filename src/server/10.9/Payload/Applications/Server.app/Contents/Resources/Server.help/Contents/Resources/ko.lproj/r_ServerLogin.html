서버 로그인

Server App을 사용하여 연결하려는 서버에 대한 관리자 계정을 사용하여 로그인하십시오. 서버에 연결한 후 해당 이름이 사이드바의 하드웨어 부분에 나타납니다.
옵션설명

호스트 이름 또는 IP 주소

이 대화상자가 나타나기 전에 서버를 선택한 경우 Server App은 추가하고 있는 서버의 호스트 이름을 자동으로 입력합니다.

그 외에는 다음 중 하나를 수행하십시오.

  • 서버 호스트 이름: macserver.example.com과 같은 FQDN(Fully Qualified Domain Name), miniserver.example.private와 같은 인트라넷 FQDN 또는 macserver.local과 같은 서버 로컬 호스트 이름이 될 수 있습니다.
  • IP 주소: 192.168.1.100과 같은 서버의 숫자로된 IP 주소입니다.

관리자 이름

추가하고 있는 서버에 대한 관리자 계정 이름을 입력하십시오. 사용하고 있는 Mac과 추가하고 있는 서버가 다른 컴퓨터인 경우 각각은 자신만의 관리자 계정이 있으며 관리자 이름 및 암호는 다를 수 있습니다.

관리자 암호

위에서 입력한 이름의 관리자 계정에 대한 암호를 입력하십시오.
