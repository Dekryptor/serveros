AirPort-Verwaltung

Verwalten Sie ein AirPort-Gerät, damit andere Computer im Internet auf bestimmte Dienste zugreifen können und damit sich Benutzer mit ihren Namen und Kennwörtern an Ihrem drahtlosen Netzwerk anmelden können.

Das Fenster „AirPort-Verwaltung“ wird angezeigt, wenn die primäre Netzwerkschnittstelle des Servers über Ihre AirPort Extreme-Basisstation oder ein Time Capsule-System mit dem Internet verbunden wird. Darüber hinaus muss die Option „Verbindung gemeinsam nutzen“ Ihrer AirPort Extreme-Basisstation oder des Time Capsule-Systems auf „Eine öffentl. IP-Adresse nutzen“ (d. h. eine Internetverbindung) eingestellt sein.

Verwenden Sie das Fenster „AirPort-Verwaltung“, um OS X Server zur Verwaltung eines AirPort-Geräts zu autorisieren. OS X Server kann eine AirPort Extreme Basisstation (802.11n) oder ein Time Capsule-System verwalten.

Zur Autorisierung der AirPort-Verwaltung geben Sie das Kennwort der Basisstation bzw. der Time Capsule ein (nicht das Kennwort für das WLAN).

Die Server-App konfiguriert den DHCP-Server Ihres AirPort-Geräts, um die IP-Adresse Ihres Servers zu reservieren, damit diese IP-Adresse Ihrem Server immer zugewiesen wird.

Sie können in der Server-App die öffentlichen Dienste bestimmen, auf die Computer über das Internet zugreifen können. OS X Server konfiguriert Ihr AirPort-Gerät automatisch so, dass die betreffenden öffentlichen Dienste im Internet zur Verfügung stehen. Der Vorgang, bei dem bestimmte Dienste über das Internet öffentlich zugänglich gemacht werden, wird als „Port Mapping“ oder „Portumleitung“ bezeichnet.

Sie können die Server-App auch verwenden, um Ihr Drahtlosnetzwerk so einzurichten, dass sich Benutzer mit ihrem Benutzernamen und ihrem Kennwort (anstelle des WLAN-Kennworts) anmelden können. In diesem Fall aktiviert Ihr Server den RADIUS-Dienst (Remote Authentication Dial In User Service) für Ihr AirPort-Gerät und berechtigt alle Benutzer, deren Accounts auf dem Server definiert sind, auf das Drahtlosnetzwerk zuzugreifen.

Wenn Sie öffentliche Dienste im Internet über ein AirPort-Gerät oder einen anderen Router zugänglich machen möchten und Ihr Server den VPN-Dienst für Benutzer bereitstellt, die ihr eigenes AirPort-Gerät oder einen anderen Router für die gemeinsame Nutzung einer Internetverbindung konfiguriert haben, darf die IP-Adresse Ihres Servers nicht mit den gleichen drei Ziffern beginnen wie die IP-Adressen der VPN-Benutzer (z. B. 10.0.1 oder 192.168.1.

Zum Verwalten Ihres AirPort-Geräts nach der Installation der Server-App wählen Sie Ihr AirPort-Gerät in der Seitenleiste aus. Es befindet sich relativ weit oben in der Seitenleiste im Abschnitt für die Hardware.
