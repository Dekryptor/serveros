Gestión de AirPort

Puede gestionar un dispositivo AirPort para conceder a otros ordenadores situados en Internet acceso a los servicios y permitir que los usuarios inicien sesión en la red inalámbrica con su nombre y contraseña.

El panel “Gestión de AirPort” aparece si la interfaz de red primaria de su servidor está conectada a Internet a través de su estación base AirPort Extreme o Time Capsule. Además, su estación base AirPort Extreme o Time Capsule debe tener la opción “Compartir conexión” establecida en “Compartir una dirección IP pública” (es decir, una conexión a Internet).

Use el panel “Gestión de AirPort” para autorizar a OS X Server a gestionar un dispositivo AirPort. OS X Server permite gestionar una estación base AirPort Extreme (802.11n) o una Time Capsule.

Para autorizar la gestión de AirPort, introduzca la contraseña de la estación base o de la Time Capsule (no la contraseña de la red Wi-Fi).

La aplicación Server configura el servidor DHCP de su servidor AirPort para reservar la dirección IP de su servidor y asignar siempre dicha dirección IP a su servidor.

También es posible utilizar la aplicación Server para especificar servicios públicos accesibles a los ordenadores situados en Internet. OS X Server configura el dispositivo AirPort para que exponga estos servicios públicos a Internet. El proceso de exposición de servicios individuales a Internet se conoce como “asignación de puertos” o “redireccionamiento de puertos”.

También puede usar la aplicación Server para permitir que los usuarios inicien sesión en la red inalámbrica con su nombre de usuario y contraseña en vez de hacerlo con la contraseña de la red Wi-Fi. En este caso, su servidor proporciona el servicio RADIUS (Remote Authentication Dial In User Service) para el dispositivo AirPort y autoriza a todas las cuentas de usuario del servidor a acceder a la red inalámbrica.

Si desea exponer servicios públicos en Internet a través de un dispositivo AirPort u otro router y su servidor proporciona el servicio VPN a los usuarios que tienen su propio dispositivo AirPort u otro router configurado para compartir una conexión de Internet, la dirección IP de su servidor no puede comenzar por los tres mismos números que las direcciones IP de los usuarios de VPN, como 10.0.1 o 192.168.1.

Para gestionar su dispositivo AirPort después de instalar la aplicación Server, seleccione su dispositivo AirPort en la barra lateral. Estará situada cerca de la parte posterior de la barra lateral, en la sección Hardware.
