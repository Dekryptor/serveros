Cambiar nombres de host

Conozca el asistente “Cambiar nombre de host”.

Utilice el asistente de cambiar nombre de host para cambiar el nombre de host, el nombre del ordenador, la dirección IP y, si su servidor cuenta con múltiples conexiones Ethernet, la conexión de red.

Al realizar cambios dentro del asistente, este actualiza automáticamente los servicios de actualizaciones dependientes del nombre de host o de la dirección IP. No actualiza los registros DNS; deberá actualizarlos en el panel DNS de la aplicación Server.

Puede obtener un nombre de dominio para su servidor desde un registrador de nombres de dominios públicos. Para más información, consulte Registrar el nombre de host de Internet del servidor.
