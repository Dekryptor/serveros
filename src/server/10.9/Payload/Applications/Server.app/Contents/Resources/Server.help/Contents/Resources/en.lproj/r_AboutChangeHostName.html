Change Host Name

Learn about Change Host Name assistant.

Use the change host name assistant to change the host name, computer name, IP address and, if your server has multiple Ethernet connections, the network connection.

When you make changes within the assistant, it automatically updates services dependent on the host name or IP address. It does not update DNS records; you must update those in the DNS pane of the Server app.
