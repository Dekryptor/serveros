Modificare il nome host

Informazioni sull'assistente di modifica del nome host.

Utilizza l'assistente “Modifica nome host” per modificare il nome host, il nome del computer, l'indirizzo IP e, se il server dispone di più connessioni Ethernet, la connessione network.

Quando apporti modifiche all'interno dell'assistente, questo aggiorna automaticamente i servizi che dipendono da nome host e indirizzo IP. Non aggiorna i record DNS; devi aggiornarli nel pannello DNS di Server app.

Puoi ottenere un nome di dominio per il server da un registro pubblico di nomi dominio. Per informazioni, consulta Registrare il nome host Internet del server
