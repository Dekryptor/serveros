选取 Mac

您可以在将充当服务器的电脑上使用 Server 应用，或者可以使用它来远程管理装有 macOS Server 的电脑。

当出现“选取 Mac”对话框时，如果选取“此 Mac”选项，Server 应用会将您的 Mac 升级为服务器。本地网络上其他启用 macOS Server 的 Mac 电脑会显示在此对话框中。您也可以按照 IP 地址或主机名来添加其他服务器。

只有在远程电脑上选中“允许使用 Server 进行远程管理”选项时，您才能管理安装有 macOS Server 的远程 Mac 电脑。此选项位于 Server 应用的“设置”面板中。在从其他 Mac 进行管理之前，您需要在远程服务器上本地启用此选项。

如果您没有看到期望的服务器，请检查网络连接或其他服务器的网络连接。
