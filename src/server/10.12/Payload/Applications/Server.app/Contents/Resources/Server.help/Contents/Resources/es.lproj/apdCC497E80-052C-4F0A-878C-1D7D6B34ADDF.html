Gestión de AirPort

El panel “Gestión de AirPort” aparece si la interfaz de red primaria de su servidor está conectada a Internet a través de una estación base AirPort Extreme o AirPort Time Capsule. Además, una estación base AirPort Extreme o AirPort Time Capsule debe tener la opción “Compartir conexión” establecida en “Compartir una dirección IP pública” (es decir, una conexión a Internet).

Utilice el panel “Gestión de AirPort” para autorizar a macOS Server la gestión de una estación base AirPort Extreme (802.11n) o una AirPort Time Capsule.

Para autorizar la gestión de AirPort, introduzca la contraseña del dispositivo AirPort (no la contraseña de la red Wi-Fi).

La aplicación Server configura el servidor DHCP de su dispositivo AirPort para reservar la dirección IP de su servidor y asignar siempre dicha dirección IP a su servidor.

La app Server se puede utilizar para especificar servicios públicos accesibles a los ordenadores situados en Internet. macOS Server configura el dispositivo AirPort para que exponga dichos servicios públicos a Internet. El proceso de exposición de servicios individuales a Internet se conoce como asignación de puertos o redireccionamiento de puertos.

También puede usar la aplicación Server para permitir que los usuarios inicien sesión en la red inalámbrica con su nombre de usuario y contraseña en vez de hacerlo con la contraseña de la red Wi-Fi. En este caso, su servidor proporciona el servicio RADIUS (Remote Authentication Dial-In User Service) para el dispositivo AirPort y autoriza a todas las cuentas de usuario del servidor a acceder a la red inalámbrica.

Si desea exponer servicios públicos en Internet a través de un dispositivo AirPort u otro router y su servidor proporciona el servicio VPN a los usuarios que tienen su propio dispositivo AirPort u otro router configurado para compartir una conexión de Internet, la dirección IP de su servidor no puede comenzar por los tres mismos números que las direcciones IP de los usuarios de VPN, como 10.0.1 o 192.168.1.

Para gestionar su dispositivo AirPort después de instalar la app Server, selecciónelo en la barra lateral de la app Server, cerca del extremo superior de la sección Hardware.
