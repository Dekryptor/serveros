サーバにログインする

Server アプリケーションで、接続したいサーバの管理者アカウントにログインします。サーバに接続すると、サイドバーの「ハードウェア」セクションにその名前が表示されます。

ホスト名または IP アドレス

このダイアログが表示される前にサーバを選択した場合は、追加しているサーバのホスト名が自動的に表示されます。

そうでない場合は、以下のいずれかを入力します：

  • サーバのホスト名：完全修飾されたインターネットドメイン名（macserver.example.com など）、完全修飾されたイントラネットドメイン名（miniserver.example.private など）、またはサーバのローカルホスト名（macserver.local など）を入力できます

  • IP アドレス：サーバの数値 IP アドレス（192.168.1.100 など）

管理者名

追加しているサーバの管理者アカウントの名前を入力します。使用している Mac と追加しているサーバが別のコンピュータの場合は、それぞれの管理者アカウントは固有で、管理者名とパスワードも異なる場合があります。

管理者パスワード

上で名前を入力した管理者アカウントのパスワードを入力します。
