Assistant de modification du nom d’hôte

Utilisez Assistant de modification du nom d’hôte pour modifier le nom d’hôte, le nom de l’ordinateur, l’adresse IP et, si votre serveur dispose de plusieurs connexions Ethernet, la connexion réseau.

Lorsque vous effectuez des modifications dans l’assistant, il met à jour automatiquement les services qui dépendent du nom d’hôte ou de l’adresse IP. L’assistant ne met pas à jour les enregistrements DNS ; vous devez les mettre à jour dans la sous-fenêtre DNS de l’application Server.

Vous pouvez obtenir un nom de domaine pour votre serveur auprès d’un revendeur de noms de domaines publics. Pour en savoir plus, consultez la rubrique Enregistrer le nom d’hôte Internet du serveur.
