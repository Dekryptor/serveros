서버의 인터넷 호스트 이름 등록하기

사용자가 인터넷의 호스트 이름을 사용하여 서버에 접근할 수 있도록 허용하려면 해당 서버의 호스트 이름을 등록해야 합니다.

  1. example.com과 같은 인터넷 도메인 이름을 얻으십시오.

    도메인 이름이 아직 없는 경우 일반 도메인 이름 등록업체에서 구입할 수 있습니다. 도메인 이름 등록업체에 관한 정보는 웹을 검색하십시오.

  2. 도메인 이름 등록업체를 사용하여 이 서버에 대한 특정 호스트 이름(예: server.example.com)을 등록하십시오.

    조직에 컴퓨터 지원 부서가 있는 경우 컴퓨터 지원 부서에 호스트 이름을 요청하십시오. 그렇지 않은 경우 호스트 이름을 할당하려면 도메인 이름을 받은 도메인 이름 등록업체에 요청하십시오.

  3. DNS 호스팅 서비스가 이 서버에 대한 레코드를 해당 DNS 서버에 추가하도록 합니다.

    조직에 컴퓨터 지원 부서가 있는 경우 DNS 서버를 호스팅하는지 알아보십시오. 그렇지 않은 경우 DNS 등록업체가 DNS 호스팅 서비스를 제공하거나, 제공업체에 대해 웹을 검색해 볼 수 있습니다.
