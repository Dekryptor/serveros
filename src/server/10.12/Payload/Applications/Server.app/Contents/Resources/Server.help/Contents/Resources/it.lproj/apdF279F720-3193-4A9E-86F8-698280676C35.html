Accedere al server

Esegui il login a un account amministratore per il server a cui desideri connetterti utilizzando Server app. Dopo la connessione al server, il suo nome appare nella sezione Hardware della barra laterale.

Nome host o Indirizzo IP

Server app completa automaticamente il nome host del server che stai aggiungendo, se hai selezionato il server prima che questa finestra sia apparsa.

Altrimenti, esegui una delle seguenti operazioni:

  • Indirizzo IP: L'indirizzo IP numerico del server, per esempio 192.168.1.100

Nome amministratore

Inserisci il nome di un account amministratore per il server che stai aggiungendo. Se il computer Mac che stai usando e il server che stai aggiungendo sono computer diversi, ognuno di essi ha il proprio account amministratore e quindi nomi e password amministratore potrebbero essere diversi.

Password amministratore

inserisci la password per l'account amministratore relativa al nome che hai inserito precedentemente.
