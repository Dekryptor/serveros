Registrieren des Internet-Hostnamens für den Server

Damit Benutzer den Hostnamen Ihres Servers verwenden können, um über das Internet auf Ihren Server zuzugreifen, müssen Sie diesen Hostnamen registrieren lassen.

  1. Lassen Sie für sich einen Internet-Domain-Namen (z. B. „example.com“) eintragen.

    Wenn Sie noch keinen Domain-Namen haben, erwerben Sie einen solchen bei einer öffentlichen Registrierungsstelle für Domain-Namen. Weitere Informationen über Registrierungsstellen für Domain-Namen finden Sie im Internet.

  2. Lassen Sie von der Registrierungsstelle einen eindeutigen Hostnamen für Ihren Server eintragen (z. B. „server.example.com“).

    Wenn Ihre Organisation eine Computer-Support-Gruppe hat, fordern Sie dort einen Hostnamen an. Arbeiten Sie andernfalls mit der Registrierungsstelle für Domain-Namen zusammen, bei der Sie Ihren Domain-Namen erworben haben, um einen Hostnamen zuzuordnen.

  3. Beauftragen Sie einen Dienstleister für DNS-Hosting damit, seinen DNS-Servern den Eintrag für Ihren Server hinzuzufügen.

    Wenn Ihre Organisation eine Computer-Support-Gruppe hat, fragen Sie, ob dort DNS-Server gehostet werden. Möglicherweise bietet die DNS-Registrierungsstelle selbst den DNS-Hosting-Dienst an. Ist dies nicht der Fall, können Sie einen entsprechenden Anbieter im Internet finden.
