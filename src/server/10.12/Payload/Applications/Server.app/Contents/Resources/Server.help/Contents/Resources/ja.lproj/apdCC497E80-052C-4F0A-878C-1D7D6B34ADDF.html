AirMac マネージメント

サーバのプライマリ・ネットワーク・インターフェイスが AirMac Extreme ベースステーションまたは AirMac Time Capsule 経由でインターネットに接続されている場合は、「AirMac マネージメント」パネルが表示されます。このパネルを表示するには、AirMac Extreme ベースステーションまたは AirMac Time Capsule で「接続共有」オプションを「パブリック IP アドレスを共有」（つまり、インターネット接続を共有）に設定する必要もあります。

「AirMac マネージメント」パネルを使用して、macOS Server による AirMac Extreme ベースステーション（802.11n）または AirMac Time Capsule の管理を許可します。

AirMac の管理を許可するには、AirMac 装置のパスワード（Wi-Fi ネットワークのパスワードとは異なります）を入力します。

Server アプリケーションによって、サーバの IP アドレスを予約してその IP アドレスを常にこのサーバに割り当てるように、AirMac 装置の DHCP サーバが構成されます。

Server アプリケーションを使用することで、インターネット上のコンピュータからアクセスできる公開サービスを指定できます。macOS Server で、これらの公開サービスをインターネットに公開するように AirMac デバイスが構成されます。サービスをインターネットに公開する処理は、ポートマッピングまたはポートフォワードと呼ばれます。

Server アプリケーションを使用して、ユーザが Wi-Fi ネットワークパスワードではなくユーザ名とパスワードを使用してワイヤレスネットワークにログインできるようにすることもできます。この場合、RADIUS（Remote Authentication Dial-In User Service）を AirMac 装置に提供することで、サーバ上のすべてのユーザアカウントがワイヤレスネットワークにアクセスすることが承認されます。

AirMac 装置またはその他のルーターを介してサービスをインターネットに公開する場合、サーバが VPN サービスを提供していて、VPN ユーザが自分の AirMac 装置またはその他のルーターでインターネット接続を共有しているときは、サーバの IP アドレスと VPN ユーザの IP アドレスの最初の 3 桁の数字が重ならないようにしてください（10.0.1 と 192.168.1 にするなど）。

Server アプリケーションのインストール後に AirMac 装置を管理する場合は、Server アプリケーションのサイドバーの上部にある「ハードウェア」セクションで AirMac 装置を選択します。
