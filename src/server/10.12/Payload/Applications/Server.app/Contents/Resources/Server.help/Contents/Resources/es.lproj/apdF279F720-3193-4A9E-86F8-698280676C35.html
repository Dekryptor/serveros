Iniciar sesión en el servidor

Inicie sesión en una cuenta de administrador del servidor al que desea conectarse mediante la aplicación Server. Tras conectarse a un servidor, el nombre de este aparece en la sección Hardware de la barra lateral.

Nombre de host o dirección IP

La aplicación Server rellena automáticamente el nombre de host del servidor que va a añadir, si ha seleccionado el servidor antes de que se abriera este cuadro de diálogo.

En caso contrario, introduzca uno de los datos siguientes:

  • El nombre de host del servidor: Puede ser un nombre de dominio de Internet totalmente cualificado, como servidormac.ejemplo.com; un nombre de dominio de intranet totalmente cualificado, como miniservidor.ejemplo.privado; o el nombre del servidor local, como servidormac.local

  • Dirección IP: La dirección IP numérica del servidor, como 192.168.1.100

Nombre de administrador

Introduzca el nombre de una cuenta de administrador para el servidor que va a añadir. Si el Mac que está utilizando y el servidor que va a añadir son ordenadores diferentes, cada uno de ellos tiene su propia cuenta de administrador y los nombres y contraseñas de administrador pueden ser diferentes.

Contraseña de administrador

Introduzca la contraseña de la cuenta de administrador cuyo nombre ha introducido anteriormente.
