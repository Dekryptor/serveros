Registrare il nome host Internet del server

Per consentire agli utenti di accedere al server mediante il nome host su Internet, devi registrare il nome host del server.

  • Ottieni un nome di dominio Internet, come esempio.com.

    Se non disponi ancora di un nome dominio, puoi acquistarne uno da un sito di registrazione di nomi domino pubblico. Per informazioni sui siti di registrazione di nomi dominio, esegui una ricerca su Internet.

  • Registra un solo nome host per il server, come ad esempio server.esempio.com, con il sito di registrazione di nomi dominio scelto.

    Se la tua società dispone di un gruppo di supporto computer, richiedi al gruppo un nome host. Altrimenti, lavora con il sito di registrazione di nomi dominio dove hai ottenuto il tuo nome dominio per assegnare il nome host.

  • Fai in modo che un servizio hosting DNS aggiunga record per questo server ai relativi server DNS.

    Se la tua società dispone di un gruppo di supporto computer, richiedi al gruppo se ospita server DNS. Altrimenti, il sito di registrazione DNS potrebbe fornire un servizio di hosting DNS oppure puoi ricercare un fornitore su Internet.
