サーバのインターネットホスト名を登録する

インターネット上のホスト名を使用してユーザがサーバにアクセスできるようにするには、サーバのホスト名を登録する必要があります。

  • インターネットドメイン名を取得します（example.com など）。

    まだドメイン名を持っていない場合、インターネットドメイン名はパブリックドメイン名登録業者から購入できます。ドメイン名登録業者については、Web を検索してください。

  • 任意のドメイン名登録業者にこのサーバの一意のホスト名を登録します（server.example.com など）。

    組織にコンピュータ・サポート・グループがある場合は、そこからホスト名を依頼してください。ない場合は、ドメイン名を取得したドメイン名登録業者と協力して、ホスト名を割り当ててください。

  • DNS ホスティングサービスに、このサーバのレコードを DNS サーバに追加してもらいます。

    組織にコンピュータ・サポート・グループがある場合は、そこで DNS サーバをホストするかどうかを確認してください。ない場合は、DNS 登録業者で DNS ホスティングサービスを提供している可能性があるほか、Web でプロバイダを探すことができます。
