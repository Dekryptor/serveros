Erreichbarkeit

Durch das Überprüfen der Erreichbarkeit können Sie feststellen, wie Ihr Server im Internet verfügbar ist.

Möglicherweise stehen Clients im Internet alle aktivieren Dienste nur dann zur Verfügung, wenn Ihr Server über eine öffentliche IP-Adresse verfügt und mit dem Internet ohne Router oder Gateway verbunden ist. Das Überprüfen der Erreichbarkeit zeigt, welche Dienste für solche Clients verfügbar sind.

Funktionsweise der Erreichbarkeitsprüfung

Die Server-App stellt zu Apple eine sichre Verbindung her und fordert die automatisierten Server bei Apple auf, für jeden aktivierten Dienst eine Verbindung zu Ihrem Server herzustellen. Da sich Apple-Server nicht in Ihrem lokalen Netzwerk befinden, können Sie externe Clients simulieren, die versuchen, eine Verbindung zu Ihrem Server herzustellen. Die Ergebnisse werden in der Server-App angezeigt.

Die Apple-Server prüfen, ob Folgendes vorhanden ist:

  • Eine extern zugängliche IP-Adresse.

  • Ein öffentlicher Hostname.

  • Ein Handshake zu jedem auf dem Server aktivieren Dienst. Nur Informationen über die Verbindungsverfügbarkeit und keine weiteren Informationen werden ausgetauscht.

  • Eine korrekt konfigurierte Portweiterleitung, falls erforderlich.

Apple speichert keine persönlichen Daten und übermittelt nur so viele Informationen über Ihre Konfiguration, damit die Apple-Server feststellen können, welche Art von Verbindung geprüft werden soll.

Vorteile der Erreichbarkeitsprüfung

Dieser Dienst bietet eine zuverlässige Möglichkeit für externe Quellen, Ihre Netzwerkkonnektivität zu überprüfen. Es ist frustrierend, nur zu glauben, dass alles korrekt konfiguriert ist, aber keine Möglichkeit zum Überprüfen von Verbindungen zu haben. Dieser Dienst gibt Ihnen sofort Aufschluss darüber, ob Ihren Konfiguration externe Zugriffsmöglichkeiten bereitstellt.
