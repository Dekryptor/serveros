AirPort 管理

如果您伺服器的主要網路介面已透過「AirPort Extreme 基地台」或 AirPort Time Capsule 連接 Internet，便會顯示「AirPort 管理」面板。此外，「AirPort Extreme 基地台」或 AirPort Time Capsule 必須將其「連線共享」選項設為「共享一個公用 IP 位址」（即 Internet 連線）。

使用「AirPort 管理」面板來授權 OS X Server 管理 AirPort 裝置。OS X Server 可以管理「AirPort Extreme 基地台」（802.11n）或 AirPort Time Capsule。

若要授權 AirPort 管理，請輸入 AirPort 裝置密碼（非 Wi-Fi 網路密碼）。

Server App 會設定您 AirPort 裝置的 DHCP 伺服器來預留您伺服器的 IP 位址，並一律將該 IP 位址指定給您的伺服器。

您可以使用 Server App 來指定 Internet 上電腦可取用的公用服務。OS X Server 會設定您的 AirPort 裝置以將那些公用服務呈現在 Internet 上。對 Internet 發佈個別服務的處理程序即稱為傳輸埠對應或傳輸埠轉送。

您也可以使用 Server App 讓使用者以其使用者名稱和密碼（而非 Wi-Fi 網路密碼）來登入您的無線網路。在此情況下，您的伺服器會為 AirPort 裝置提供「遠端撥入驗證服務」（RADIUS），並授權伺服器上的所有使用者帳號來連接您的無線網路。

如果您想要透過 AirPort 裝置或其他路由器來將公用服務呈現在 Internet 上，且您的伺服器提供 VPN 服務給本身具有 AirPort 裝置或其他路由器（已設為共享 Internet 連線）的使用者，則您伺服器的 IP 位址便無法使用與 VPN 使用者相同的 IP 位址前三個數字開頭，如 10.0.1 或 192.168.1。

若要在安裝 Server App 後管理您的 AirPort 裝置，請在側邊欄中選擇您的 AirPort 裝置。它靠近側邊欄的最上方，位於「硬體」區域。
