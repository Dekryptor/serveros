여러 네트워크가 발견됨

사용자의 서버가 하나 이상의 활성 네트워크에 연결된 경우 여러 네트워크가 발견됨 패널이 나타납니다. 팝업 메뉴에서 서버의 주요 네트워크 연결을 선택하십시오.

이 패널에서 계속을 클릭하면 서버 설정 지원이 선택한 네트워크를 사용하여 IP 주소 및 호스트 이름을 포함하여 해당 서버의 네트워크 신원을 결정합니다. 서버 설정 지원이 네트워크 상의 DHCP 서버에서 해당 서버의 IP 주소를 얻으려고 하며 이 IP 주소를 사용하여 네트워크의 DNS 서버에서 해당 서버의 호스트 이름을 얻습니다. 필요한 경우 다음 패널에서 해당 서버의 네트워크 구성 및 신원을 변경할 수 있습니다.

서버 설정을 완료한 후 게이트웨이로 AirPort 장비를 구성하려고 하는 경우 사용자의 AirPort 장비에 연결하는 인터페이스(일반적으로 Wi-Fi)를 선택하십시오. 나중에 Server App을 사용하여 게이트웨이를 설정할 수도 있습니다.

참고:  화면 공유를 사용하여 다른 컴퓨터에서 서버 설정 지원을 제어하고 있으며 네트워크 서비스를 변경하는 경우, 화면 공유 세션이 종료될 수 있습니다. 새로운 네트워크 서비스를 사용하여 화면 공유 연결을 시도하거나 설정을 로컬로 계속하려면 해당 서버로 이동하여 해당 키보드 및 디스플레이를 사용해 보십시오.
