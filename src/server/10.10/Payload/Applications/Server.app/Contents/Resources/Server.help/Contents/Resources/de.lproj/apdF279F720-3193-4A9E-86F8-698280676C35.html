Anmelden am Server

Für die Anmeldung an einem Server, zu dem Sie in der Server-App eine Verbindung herstellen wollen, müssen Sie dessen Administratoraccount verwenden. Nachdem die Verbindung zu einem Server hergestellt wurde, wird dessen Name im Abschnitt „Hardware“ der Seitenleiste angezeigt.

Hostname oder IP-Adresse

Die Server-App trägt den Hostnamen des Servers, den Sie hinzufügen, automatisch ein, wenn Sie den Server schon vor der Anzeige dieses Dialogfensters ausgewählt haben.

Führen Sie anderenfalls einen der folgenden Schritte aus:

  • Hostname des Servers: Dies kann der vollständig qualifizierte Name einer Internet-Domain sein (z. B. „macserver.example.com“), der vollständig qualifizierte Name einer Intranet-Domain (z. B. „miniserver.example.private“ oder der lokale Hostname des Servers (z. B. „macserver.local“).

  • IP-Adresse: Dies ist die numerische IP-Adresse des Servers (z. B. „192.168.1.100“).

Administratorname

Geben Sie den Benutzernamen eines Administratoraccounts für den Server ein, den Sie hinzufügen wollen. Wenn der Mac-Server, auf dem Sie die Server-App ausführen, nicht identisch ist mit dem Server, den Sie hinzufügen, verfügt jeder der beiden Computer über einen eigenen Administratoraccount. Das bedeutet, dass die Administratornamen und die Passwörter möglicherweise nicht übereinstimmen.

Administratorpasswort

Geben Sie das Passwort des Administratoraccounts ein, dessen Benutzernamen Sie eingegeben haben.
