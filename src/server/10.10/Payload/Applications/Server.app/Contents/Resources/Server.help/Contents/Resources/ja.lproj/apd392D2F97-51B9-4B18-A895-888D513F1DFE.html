複数のネットワークが検出されました

サーバが複数の動作中のネットワークに接続されている場合は、「複数のネットワークが検出されました」パネルが表示されます。ポップアップメニューからサーバのプライマリネットワーク接続を選択します。

このパネルで「続ける」をクリックすると、サーバ設定アシスタントが選択したネットワークを使って、IP アドレスやホスト名などのサーバのネットワーク識別情報を判別します。サーバ設定アシスタントは、ネットワーク上の DHCP サーバからサーバの IP アドレスの取得を試み、この IP アドレスを使用して、ネットワーク上の DNS サーバからサーバのホスト名を取得します。必要に応じて、次のパネルでサーバのネットワーク構成と識別情報を変更することができます。

サーバ設定の終了後に AirMac 装置をゲートウェイとして構成する予定の場合は、AirMac 装置に接続するインターフェイス（通常は Wi-Fi）を選択します。後で Server アプリケーションを使って、ゲートウェイを設定することができます。

参考： 画面共有を使って別のコンピュータからサーバ設定アシスタントを制御しているときに、ネットワークサービスを変更すると、画面共有セッションが終了することがあります。新しいネットワークサービスを使って、画面共有で接続してみてください。または、サーバに移動して、サーバのキーボードとディスプレイを使ってローカルで設定を続けてください。
