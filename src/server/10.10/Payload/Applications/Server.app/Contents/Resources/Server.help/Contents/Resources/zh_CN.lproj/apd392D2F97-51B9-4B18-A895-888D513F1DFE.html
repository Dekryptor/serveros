检测到多个网络

如果您的服务器连接到多个活跃的网络，则“检测到多个网络”面板会出现。从弹出式菜单中选取服务器的首选网络连接。

当您在此面板中点按“继续”时，服务器设置助理会使用您选取的网络来确定服务器的网络身份，包括它的 IP 地址和主机名。服务器设置助理会尝试从网络上的 DHCP 服务器获取服务器的 IP 地址，并使用该 IP 地址从网络上的 DNS 服务器获取服务器的主机名。如果需要，可以在后续面板中更改服务器的网络配置和身份。

在完成服务器设置后，如果您计划将 AirPort 设备配置为网关，请选取连接到 AirPort 设备的接口（通常为 Wi-Fi）。您可以稍后使用 Server 应用程序设置网关。

【注】如果您正在从另一台电脑通过屏幕共享控制服务器设置助理，并且您更改了网络服务，则屏幕共享会话可能会结束。尝试使用新的网络服务进行屏幕共享连接，或者前往该服务器并使用它的键盘和显示器继续本地设置。
