Inloggen op de server

Log in met een beheerdersaccount voor de server waarmee u verbinding wilt maken met behulp van het programma Server. Nadat er verbinding is gemaakt met een server, wordt de servernaam weergegeven in het gedeelte 'Hardware' van de navigatiekolom.

Hostnaam of IP-adres

Als u de server die u wilt toevoegen al vóór het verschijnen van dit dialoogvenster hebt geselecteerd, wordt de hostnaam van de server automatisch ingevuld.

Voer anders een van de volgende waarden in:

  • De hostnaam van de server: Dit kan een volledig gekwalificeerde internetdomeinnaam zijn (zoals macserver.voorbeeld.com), een volledig gekwalificeerde intranetdomeinnaam (zoals miniserver.voorbeeld.private) of de lokale hostnaam van de server (zoals macserver.local).

  • IP-adres: Dit is het numerieke IP-adres van de server, zoals 192.168.1.100.

Naam beheerder

Voer de naam in van een beheerdersaccount voor de server die u toevoegt. Als de Mac die u gebruikt en de server die u toevoegt verschillende computers zijn, heeft elke computer een eigen beheerdersaccount en kunnen de beheerdersnamen en wachtwoorden verschillen.

Wachtwoord beheerder

Voer het wachtwoord in voor de beheerdersaccount waarvan u de naam hierboven hebt opgegeven.
