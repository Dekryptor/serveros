已偵測到多個網路

如果您的伺服器連接到多個使用中的網路連線，便會顯示「已偵測到多個網路」面板。請從彈出式選單中選擇伺服器的主要網路連線。

當您在此面板中按一下「繼續」時，伺服器設定輔助程式會使用您選擇的網路來判斷伺服器的網路身分，包含其 IP 位址和主機名稱。伺服器設定輔助程式會嘗試從網路上的 DHCP 伺服器取得伺服器的 IP 位址，並使用此 IP 位址來從網路上的 DNS 伺服器取得伺服器的主機名稱。如有需要，您可以在後續的面板中更改伺服器的網路設定和身分。

如果您計劃在完成伺服器設定後，將 AirPort 裝置設定為閘道，請選擇會連接到您 AirPort 裝置的介面（通常為 Wi-Fi）。您之後可以使用 Server App 來設定閘道。

【注意】如果您使用螢幕共享從另一部電腦來控制伺服器設定輔助程式，而您更改網路服務，螢幕共享作業可能會結束。請嘗試使用新的網路服務來進行螢幕共享連線，或者移至伺服器並使用其鍵盤和顯示器來繼續本機端的設定。
