Choisissez un Mac

Vous pouvez utiliser l’app Server sur l’ordinateur qui fera office de serveur ou vous en servir pour gérer à distance un ordinateur exécutant OS X Server.

Dans la zone de dialogue « Choisissez un Mac », si vous sélectionnez l’option « Ce Mac », Server transforme votre Mac en serveur. Les autres ordinateurs Mac de votre réseau local sur lesquels OS X Server est activé s’affichent dans cette zone de dialogue. Vous pouvez également ajouter d’autres serveurs à partir de leur adresse IP ou de leur nom d’hôte.

Vous ne pouvez gérer des ordinateurs Mac distants sur lesquels OS X Server est installé que si l’option « Autoriser l’administration à distance via Server » est activée sur l’ordinateur distant. Celle-ci se trouve dans la sous-fenêtre Réglages de l’app Server. Activez l’option localement sur le serveur distant pour pouvoir la gérer depuis un autre Mac.

Si le serveur recherché ne s’affiche pas, vérifiez votre connexion réseau ou celle de l’autre serveur.
