Registrar el nombre de host de Internet del servidor

Si desea que los usuarios tengan acceso al servidor con su nombre de host en Internet, registre antes el nombre de host del servidor.

  • Obtenga un nombre de dominio de Internet, como ejemplo.com.

    Si aún no tiene nombre de dominio, puede comprar uno a un registrador de nombres de dominio públicos. Si desea más información sobre los registradores de nombres de dominio, realice una búsqueda en Internet.

  • Registre un nombre de host único para el servidor, como servidor.ejemplo.com, en su registrador de nombres de dominio.

    Si su organización cuenta con un grupo de soporte informático, solicítele a ellos un nombre de host. En caso contrario, trabaje con el registrador de nombres de dominio del que obtuvo su nombre de dominio para asignar un nombre de host.

  • Utilice un servicio de alojamiento DNS para que añada los registros correspondientes a este servidor a sus servidores DNS.

    Si su organización cuenta con un grupo de soporte informático, pregúnteles si alojan servidores DNS. Si no es así, es posible que su registrador de DNS le ofrezca un servicio de alojamiento de DNS, pero también puede buscar un proveedor en Internet.
