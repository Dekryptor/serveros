Log into your server

Log in to an administrator account for the server you want to connect to using the Server app. After you connect to a server, its name appears in the Hardware section of the sidebar.

Host Name or IP Address

The Server app automatically fills in the host name of the server you’re adding, if you selected the server before this dialog appeared.

Otherwise, enter one of the following:

  • The server’s host name: Can be a fully qualified Internet domain name, such as macserver.example.com; a fully qualified intranet domain name, such as miniserver.example.private; or the server’s local hostname, such as macserver.local

  • IP address: The server’s numeric IP address, such as 192.168.1.100

Administrator Name

Enter the name of an administrator account for the server you’re adding. If the Mac you’re using and the server you’re adding are different computers, each has its own administrator account, and the administrator names and passwords may be different.

Administrator Password

Enter the password for the administrator account whose name you entered above.
