Vorbereitende Schritte

Bevor Sie Ihren Server konfigurieren, müssen Sie der Endbenutzer-Lizenzvereinbarung (EULA) zustimmen. Darüber hinaus werden Sie aufgefordert, das Passwort eines Administratoraccounts auf Ihrem Mac einzugeben.

Wenn Sie den Inhalt der EULA zu einem späteren Zeit nochmals lesen möchten, finden Sie diese online unter Softwarelizenz.
