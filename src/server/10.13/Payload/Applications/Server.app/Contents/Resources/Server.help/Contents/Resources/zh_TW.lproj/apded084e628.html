可連接性測試

可達性測試可讓您瞭解伺服器連接到 Internet 的可行性。

Internet 上的部分用戶端可能會無法使用所有作用中的服務，除非您的伺服器包含 IP 位址且未透過路由器或閘道連接 Internet。可達性測試會顯示此類用戶端可使用的服務。

可達性測試如何運作？

Server App 會安全地連接到 Apple，並要求 Apple 的自動化伺服器針對已啟用的服務嘗試連接到您的伺服器。因為 Apple 伺服器在您的區域網路外，它們可以模擬嘗試連接您伺服器的外部用戶端。結果會顯示在 Server App 中。

Apple 伺服器會檢查：

  • 可外部連接的 IP 位址：

  • 公用主機名稱。

  • 與伺服器上啟用的每個服務之間的通訊。不會交換連線可用性以外的資訊。

  • 如有需要，請正確設定連接埠轉送。

Apple 不會保留您的任何私人資訊，且只會傳送關於設定的資訊，足以讓 Apple 伺服器瞭解要檢查的連線類型。

為何我需要使用可達性測試？

這項服務提供可靠的方式來讓外部來源檢查您的網路連線能力。當您認為一切都已正確設定，卻沒有方法測試連線時，肯定很沮喪。這項服務可讓您得到反饋，得知您的設定是否確實提供外部連接性。

我該如何停用此功能？

若要停用可達性測試，請參閱：停用可達性測試。
