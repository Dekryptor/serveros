AirPort 管理

如果服务器的首选网络接口通过 AirPort Extreme 基站或 AirPort Time Capsule 接入互联网，“AirPort 管理”面板会出现。另外，AirPort Extreme 基站或 AirPort Time Capsule 必须将其“连接共享”选项设定为“共享公共 IP 地址”（即互联网连接）。

使用“AirPort 管理”面板来授权 macOS Server 管理 AirPort Extreme 基站 (802.11n) 或 AirPort Time Capsule。

若要授权 AirPort 管理，请输入 AirPort 设备的密码（不是 Wi-Fi 网络密码）。

Server 应用会将 AirPort 设备的 DHCP 服务器配置成保留您服务器的 IP 地址，并始终将该 IP 地址指定给您的服务器。

您可以使用 Server 应用来指定公共服务，这些服务可被互联网上的电脑访问。macOS Server 会配置您的 AirPort 设备以在互联网上开放这些公共服务。将个别服务公开到互联网的过程称为端口映射或端口转发。

如果您想要通过 AirPort 设备或其他路由器将公共服务在互联网上开放，并且您的服务器向拥有他们自己的 AirPort 设备或其他路由器（已配置为共享互联网连接）提供 VPN 服务，则您的服务器的 IP 地址不能以 VPN 用户的 IP 地址（如 10.0.1 或 192.168.1）中的那三组相同数字开头。

若要在安装 Server 应用后管理 AirPort 设备，请在 Server 应用边栏中选择 AirPort 设备（靠近顶部的“硬件”部分中）。
