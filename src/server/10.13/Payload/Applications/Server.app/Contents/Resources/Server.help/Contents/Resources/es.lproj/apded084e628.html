Comprobación de accesibilidad

La comprobación de accesibilidad le permite saber el grado de accesibilidad de su servidor en Internet.

A menos que su servidor disponga de una dirección IP pública y esté conectado a Internet sin ningún router o puerta de enlace, es posible que no todos los servicios activos estén disponible para los clientes en Internet. La comprobación de accesibilidad muestra qué servicios se encuentran disponibles para dichos clientes.

¿Cómo funciona la comprobación de accesibilidad?

La app Server se conecta a Apple de manera segura y solicita a los servidores automatizados de Apple que intenten conectarse a su servidor para cada servicio activado. Como los servidores de Apple se encuentran fuera de su red local, pueden simular que son clientes externos que están intentando conectarse a su servidor. Los resultados se muestran en la aplicación Server.

Los servidores de Apple comprueban lo siguiente:

  • Una dirección IP accesible de forma externa.

  • Un nombre de host público.

  • Un protocolo de enlace a cada servicio activado en el servidor. No se intercambia ninguna información aparte de la disponibilidad de conexión.

  • Redireccionamiento de puertos correctamente configurado, si es necesario.

Apple no conserva ninguno de sus datos personales y solo transmite la suficiente información sobre su configuración como para que los servidores de Apple sepan qué tipos de conexión deben comprobar.

¿Por qué debería utilizarse la comprobación de accesibilidad?

Este servicio ofrece una forma fiable de comprobar su conectividad de red por parte de una fuente externa. Es frustrante creer que todo está configurado correctamente, pero no tener ningún modo de comprobar las conexiones. Este servicio le ofrece información instantánea sobre si su configuración realmente ofrece accesibilidad externa.

¿Cómo se desactiva?

Para desactivar las pruebas de accesibilidad, consulte Desactivar las pruebas de accesibilidad.
