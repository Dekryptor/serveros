AirMac 管理

サーバのプライマリ・ネットワーク・インターフェイスが AirMac Extreme ベースステーションまたは AirMac Time Capsule 経由でインターネットに接続されている場合は、「AirMac 管理」パネルが表示されます。このパネルを表示するには、AirMac Extreme ベースステーションまたは AirMac Time Capsule で「接続共有」オプションを「パブリック IP アドレスを共有」（つまり、インターネット接続を共有）に設定する必要もあります。

「AirMac 管理」パネルを使用して、macOS Server による AirMac Extreme ベースステーション（802.11n）または AirMac Time Capsule の管理を許可します。

AirMac の管理を許可するには、AirMac デバイスのパスワード（Wi-Fi ネットワークのパスワードとは異なります）を入力します。

Server アプリケーションによって、サーバの IP アドレスを予約してその IP アドレスを常にこのサーバに割り当てるように、AirMac デバイスの DHCP サーバが構成されます。

Server アプリケーションを使用することで、インターネット上のコンピュータからアクセスできる公開サービスを指定できます。macOS Server で、これらの公開サービスをインターネットに公開するように AirMac デバイスが構成されます。サービスをインターネットに公開する処理は、ポートマッピングまたはポートフォワードと呼ばれます。

AirMac デバイスまたはその他のルーターを介してサービスをインターネットに公開する場合、サーバが VPN サービスを提供していて、VPN ユーザが自分の AirMac デバイスまたはその他のルーターでインターネット接続を共有しているときは、サーバの IP アドレスと VPN ユーザの IP アドレスの最初の 3 桁の数字が重ならないようにしてください（10.0.1 と 192.168.1 にするなど）。

Server アプリケーションのインストール後に AirMac デバイスを管理する場合は、Server アプリケーションのサイドバーの上部にある「ハードウェア」セクションで AirMac デバイスを選択します。
