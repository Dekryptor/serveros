恢復資訊

此文件包含關於修復電腦卷宗、在保留檔案和使用者設定的情況下重新安裝 macOS、使用 Time Machine 回復電腦系統，以及清除和重新安裝 macOS 的資訊。

如需更多輔助說明，請前往 https://www.apple.com/tw/support 網站。

連接 Internet

若需要連接 Internet，您可以使用無線或有線網路連線。若要加入無線網路：

  1. 按一下選單列上的 Wi-Fi 狀態圖像（位於螢幕右上方），然後選擇網路。
  2. 如有需要，請輸入網路密碼。

若要加入封閉網路，請選擇「加入其他網路⋯」。輸入網路名稱，如有需要，請輸入網路密碼。

修復卷宗

若電腦發生問題或開機後看見「macOS 工具程式」面板，您可能需要修復電腦的內置儲存裝置和卷宗。

注意：關閉此視窗或結束 Safari 前，您可能需要記下這些指示。

  1. 選擇 Safari >「結束 Safari」，在「macOS 工具程式」面板中選擇「磁碟工具程式」，然後按一下「繼續」。
  2. 在左側列表中選擇內置儲存裝置，然後按一下「修理工具」。
  3. 在左側列表中選擇啟動卷宗，然後再按一下「修理工具」。
  4. 按一下「執行」。

如果「磁碟工具程式」無法修復磁碟，請儘可能地備份您的資訊，然後依照此文件下方「清除並重新安裝 macOS」中的指示來執行。

重新安裝 macOS

在部分情況下，可能需要重新安裝 macOS。重新安裝 macOS 時，您仍可以保留原有的檔案和使用者設定。

注意：關閉此視窗或結束 Safari 前，您可能需要記下這些指示。

  1. 選擇 Safari >「結束 Safari」。
  2. 確定已連接 Internet。（若要連接無線網路，請依照此文件上方「連接 Internet」的指示操作。）
  3. 選擇「重新安裝 macOS」，然後按一下「繼續」。
  4. 在選擇卷宗的面板中選擇現用的 macOS 卷宗（在大部分的情況下，這是唯一可用的卷宗）。
  5. 按一下「自定」來選擇或取消選取選用的軟體。
  6. 按一下「安裝」。

使用 Time Machine 回復系統

若清除或重新安裝 macOS 且之前已建立 Time Machine 備份，則可使用「回復」工具程式來將電腦上的所有資料回復到先前的狀態。

Time Machine 備份只能將資訊回復到備份來源的電腦。若要將資訊傳送到新電腦，請使用「系統移轉輔助程式」（位於 Launchpad 的「其他」檔案夾）。

注意：關閉此視窗或結束 Safari 前，您可能需要記下這些指示。

  1. 選擇 Safari >「結束 Safari」。
  2. 如果備份位於 Time Capsule，請確定已連接 Internet（若要連接無線網路，請依照此文件上方「連接 Internet」的指示操作。）
  3. 從「Time Machine 備份」選擇「回復」，然後按一下「繼續」。
  4. 選擇備份來源，選擇包含 Time Machine 備份的卷宗，然後依照螢幕上的指示操作。

清除並重新安裝 macOS

若您清除並重新安裝 macOS，電腦上的所有資料（您的使用者帳號、網路設定、所有檔案和檔案夾）都將刪除。清除 macOS 前請先將想要保留的檔案拷貝到其他卷宗來進行備份。請記下「網路」偏好設定中的網路設定，以便重新安裝 macOS 後可以再次輕鬆上網。

注意：關閉此視窗或結束 Safari 前，您可能需要記下這些指示。

  1. 選擇 Safari >「結束 Safari」。
  2. 確定已連接 Internet（若要連接無線網路，請依照此文件上方「連接 Internet」的指示操作。）
  3. 選擇「磁碟工具程式」，然後按一下「繼續」。
  4. 在左側列表中選擇卷宗，然後按一下「清除」。
  5. 輸入卷宗名稱，從「格式」彈出式選單中選擇「Mac OS 擴充格式（日誌式）」，然後按一下「清除」。

    清除卷宗後，請選擇「磁碟工具程式」>「結束磁碟工具程式」。選擇「重新安裝 macOS」，按一下「繼續」，然後依照「macOS 安裝程式」的指示來重新安裝 macOS。
