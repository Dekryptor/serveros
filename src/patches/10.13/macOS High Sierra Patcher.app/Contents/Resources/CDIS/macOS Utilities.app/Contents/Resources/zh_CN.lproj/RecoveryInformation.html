恢复信息

本文稿包含以下信息：修复电脑的宗卷、保持文件和用户设置不变的情况下重新安装 macOS、使用 Time Machine 恢复电脑系统以及抹掉并重新安装 macOS。

有关更多帮助，请对 https://www.apple.com/cn/support 进行访问。

接入互联网

若要接入互联网，您可以使用无线或有线网络连接。若要加入无线网络：

  1. 点按菜单栏中的 Wi-Fi 状态图标（位于屏幕右上角），然后选择网络。
  2. 如果需要，请键入网络的密码。

若要加入封闭网络，请选择“加入其他网络”。输入网络的名称和密码（如果需要）。

修复宗卷

如果您的电脑出现了问题，或者在启动电脑时看到了“macOS 实用工具”面板，则可能需要修复电脑的内部存储设备和宗卷。

注：关闭此窗口或退出 Safari 之前，不妨先记下这些说明。

  1. 选择“Safari”>“退出 Safari”，在“macOS 实用工具”面板中选择“磁盘工具”，然后点按“继续”。
  2. 选择左侧列表中的内部存储设备，然后点按“急救”。
  3. 选择左侧列表中的启动宗卷，然后再次点按“急救”。
  4. 点按“运行”。

如果“磁盘工具”无法修复您的磁盘，请备份尽可能多的信息，然后按照本文稿后半部分“抹掉并重新安装 macOS”中的说明操作。

重新安装 macOS

某些情况下，您可能需要重新安装 macOS。您可以在保持文件和用户设置不变的情况下重新安装 macOS。

注：关闭此窗口或退出 Safari 之前，不妨先记下这些说明。

  1. 选择“Safari”>“退出 Safari”。
  2. 确定您的电脑已接入互联网。（若要接入无线网络，请按照本文稿前半部分“接入互联网”中的说明操作。）
  3. 选择“重新安装 macOS”，然后点按“继续”。
  4. 在您选择宗卷的面板中，选择您当前的 macOS 宗卷（大多数情况下，它是唯一可用的宗卷）。
  5. 点按“自定”以选择或取消选择一些可选软件。
  6. 点按“安装”。

使用 Time Machine 恢复您的系统

如果您抹掉并重新安装了 macOS，而且在此之前已经创建了 Time Machine 备份，那么您可以使用“恢复”实用工具将电脑上所有的内容都恢复到其先前的状态。

您的 Time Machine 备份只能将信息恢复到作为备份来源的电脑上。如果您想要将信息传输到新电脑，请使用“迁移助理”（在 Launchpad 的“其他”文件夹中）。

注：关闭此窗口或退出 Safari 之前，不妨先记下这些说明。

  1. 选择“Safari”>“退出 Safari”。
  2. 如果您的备份在 Time Capsule 上，请确定您的电脑已接入互联网。（若要接入无线网络，请按照本文稿前半部分“接入互联网”中的说明操作。）
  3. 从“Time Machine 备份”中选择“恢复”，然后点按“继续”。
  4. 选择备份来源，选择含有 Time Machine 备份的宗卷，然后按照屏幕说明操作。

抹掉并重新安装 macOS

如果抹掉并重新安装 macOS，则会删除电脑上的所有内容（用户帐户、网络设置以及所有文件和文件夹）。抹掉 macOS 之前，请将要保留的任何文件拷贝到其他宗卷上进行备份。记下“网络”偏好设置中您网络的设置，以便在重新安装 macOS 后能轻松地重新接入网络。

注：关闭此窗口或退出 Safari 之前，不妨先记下这些说明。

  1. 选择“Safari”>“退出 Safari”。
  2. 确定您的电脑已接入互联网。（若要接入无线网络，请按照本文稿前半部分“接入互联网”中的说明操作。）
  3. 选择“磁盘工具”，然后点按“继续”。
  4. 选择左侧列表中的宗卷，然后点按“抹掉”。
  5. 输入宗卷的名称，从“格式”弹出式菜单中选择“Mac OS 扩展”，然后点按“抹掉”。在抹掉宗卷后，选择“磁盘工具”>“退出磁盘工具”。选择“重新安装 macOS”，点按“继续”，然后按照“macOS 安装器”中的说明来重新安装 macOS。
