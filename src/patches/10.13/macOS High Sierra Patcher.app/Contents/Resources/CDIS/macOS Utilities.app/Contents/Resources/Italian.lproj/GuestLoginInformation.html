Connettiti a Internet

Per potere navigare sul web, devi disporre di una connessione network wireless o cablata. Per accedere a un network wireless:

  1. Fai clic sull'icona di stato Wi-Fi nella barra dei menu (nell'angolo superiore destro dello schermo), quindi seleziona un network.
  2. Se necessario, digita la password del network.

Per accedere a un network chiuso, scegli “Accedi a un altro network”. Inserisci il nome del network e, se necessario, la password.

Naviga sul web

Una volta connesso a Internet, puoi navigare sul web da questa finestra di Safari.
