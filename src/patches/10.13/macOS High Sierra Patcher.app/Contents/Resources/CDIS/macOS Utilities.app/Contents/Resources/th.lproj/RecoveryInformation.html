ข้อมูลการกู้คืน

เอกสารนี้ประกอบด้วยข้อมูลเกี่ยวกับการซ่อมแซมดิสก์โวลุ่มของคอมพิวเตอร์ของคุณ การติดตั้ง macOS อีกครั้งในขณะที่ทำให้ไฟล์และการตั้งค่าผู้ใช้ของคุณคงเดิมอยู่เสมอ การใช้ Time Machine เพื่อกู้คืนระบบคอมพิวเตอร์ของคุณ และการลบและติดตั้ง macOS อีกครั้ง

สำหรับวิธีใช้เพิ่มเติม โปรดไปที่ https://www.apple.com/th/support/

การเชื่อมต่ออินเทอร์เน็ต

ถ้าคุณต้องเชื่อมต่ออินเทอร์เน็ต คุณสามารถใช้การเชื่อมต่อเครือข่ายแบบไร้สายหรือใช้สายได้ หากต้องการเข้าร่วมเครือข่ายไร้สาย:

  1. คลิกไอคอนสถานะ Wi-Fi ในแถบเมนู (ตรงมุมขวาบนของหน้าจอ) แล้วเลือกเครือข่าย
  2. ถ้าจำเป็น ให้ป้อนรหัสผ่านของเครือข่าย

ถ้าต้องการเข้าร่วมเครือข่ายปิด ให้เลือกเข้าร่วมเครือข่ายอื่น ป้อนชื่อของเครือข่าย และถ้าจำเป็น ให้ป้อนรหัสผ่านของเครือข่าย

ซ่อมแซมดิสก์โวลุ่มของคุณ

ถ้าคุณมีปัญหาเกี่ยวกับคอมพิวเตอร์ของคุณ หรือถ้าคุณเปิดคอมพิวเตอร์แล้วเห็นบานหน้าต่างยูทิลิตี้ macOS แสดงว่าคุณอาจต้องซ่อมแซมอุปกรณ์จัดเก็บข้อมูลภายในและดิสก์โวลุ่มของคอมพิวเตอร์ของคุณ

หมายเหตุ:คุณอาจต้องการจดขั้นตอนเหล่านี้ก่อนที่จะปิดหน้าต่างนี้หรือออกจาก Safari

  1. เลือก Safari > ออกจาก Safari แล้วเลือกยูทิลิตี้ดิสก์ในบานหน้าต่างยูทิลิตี้ macOS จากนั้นคลิกดำเนินการต่อ
  2. เลือกอุปกรณ์จัดเก็บข้อมูลภายในในรายการทางด้านซ้าย แล้วคลิกการช่วยเหลือเบื้องต้น
  3. เลือกดิสก์โวลุ่มเริ่มต้นในรายการทางด้านซ้าย แล้วคลิกการช่วยเหลือเบื้องต้นอีกครั้ง
  4. คลิกสั่งทำงาน

ถ้ายูทิลิตี้ดิสก์ไม่สามารถซ่อมแซมดิสก์ของคุณได้ ให้สำรองข้อมูลของคุณให้มากที่สุดเท่าที่ทำได้ แล้วทำตามขั้นตอนใน “ลบแล้วติดตั้ง macOS อีกครั้ง” ในภายหลังในเอกสารนี้

ติดตั้ง macOS อีกครั้ง

ในบางสถานการณ์ คุณอาจต้องติดตั้ง macOS อีกครั้ง คุณสามารถติดตั้ง macOS อีกครั้งได้ในขณะที่ทำให้ไฟล์และการตั้งค่าผู้ใช้ของคุณคงเดิมอยู่เสมอ

หมายเหตุ:คุณอาจต้องการจดขั้นตอนเหล่านี้ก่อนที่จะปิดหน้าต่างนี้หรือออกจาก Safari

  1. เลือก Safari > ออกจาก Safari
  2. ตรวจสอบให้แน่ใจว่าคุณเชื่อมต่ออินเทอร์เน็ตอยู่ (หากต้องการเชื่อมต่อกับเครือข่ายไร้สาย ให้ทำตามขั้นตอนใน “เชื่อมต่ออินเทอร์เน็ต” ซึ่งปรากฏก่อนหน้าในเอกสารนี้)
  3. เลือกติดตั้ง macOS อีกครั้ง แล้วคลิกดำเนินการต่อ
  4. ในบานหน้าต่างที่คุณเลือกดิสก์โวลุ่ม ให้เลือกดิสก์โวลุ่ม macOS ปัจจุบันของคุณ (ในกรณีส่วนมาก จะมีดิสก์โวลุ่มให้เลือกเพียงดิสก์เดียว)
  5. คลิกกำหนดเองเพื่อเลือกหรือเลิกเลือกซอฟต์แวร์เสริม
  6. คลิกติดตั้ง

ใช้ Time Machine เพื่อกู้คืนระบบของคุณ

หากคุณลบและติดตั้ง macOS อีกครั้งและคุณเคยสร้างข้อมูลสำรอง Time Machine ไว้ก่อนหน้านี้ ให้ใช้ยูทิลิตี้กู้คืนเพื่อกู้คืนทุกสิ่งทุกอย่างในคอมพิวเตอร์ของคุณให้กลับไปเป็นสภาพเดิม

ใช้เฉพาะข้อมูลสำรอง Time Machine ของคุณเพื่อกู้คืนข้อมูลกลับไปยังคอมพิวเตอร์ที่เป็นที่มาของข้อมูลสำรองนั้นเท่านั้น ถ้าคุณต้องการถ่ายโอนข้อมูลไปยังคอมพิวเตอร์เครื่องใหม่ ให้ใช้ผู้ช่วยการโยกย้าย (ในโฟลเดอร์อื่นๆ ใน Launchpad)

หมายเหตุ:คุณอาจต้องการจดขั้นตอนเหล่านี้ก่อนที่จะปิดหน้าต่างนี้หรือออกจาก Safari

  1. เลือก Safari > ออกจาก Safari
  2. ถ้าข้อมูลสำรองของคุณอยู่ใน Time Capsule โปรดตรวจสอบให้แน่ใจว่าคุณเชื่อมต่ออินเทอร์เน็ตอยู่ (หากต้องการเชื่อมต่อกับเครือข่ายไร้สาย ให้ทำตามขั้นตอนใน “เชื่อมต่ออินเทอร์เน็ต” ที่ปรากฏก่อนหน้าในเอกสารนี้)
  3. เลือกกู้คืนจากการสำรองข้อมูล Time Machine แล้วคลิกดำเนินการต่อ
  4. เลือกแหล่งข้อมูลสำรอง แล้วเลือกดิสก์โวลุ่มที่มีข้อมูลสำรอง Time Machine จากนั้นทำตามขั้นตอนบนหน้าจอ

ลบและติดตั้ง macOS อีกครั้ง

ถ้าคุณลบและติดตั้ง macOS อีกครั้ง ทุกสิ่งทุกอย่าง (บัญชีผู้ใช้ของคุณ การตั้งค่าเครือข่าย และไฟล์และโฟลเดอร์ทั้งหมดของคุณ) จะถูกลบ ก่อนที่คุณจะลบ macOS ให้สำรองข้อมูลไฟล์ใดๆ ที่คุณต้องการเก็บไว้โดยคัดลอกไฟล์เหล่านั้นไปยังดิสก์โวลุ่มอื่น แล้วจดการตั้งค่าเครือข่ายของคุณจากการตั้งค่าเครือข่าย เพื่อทำให้การเชื่อมต่ออีกครั้งหลังจากติดตั้ง macOS อีกครั้งง่ายยิ่งขึ้น

หมายเหตุ:คุณอาจต้องการจดขั้นตอนเหล่านี้ก่อนที่จะปิดหน้าต่างนี้หรือออกจาก Safari

  1. เลือก Safari > ออกจาก Safari
  2. ตรวจสอบให้แน่ใจว่าคุณเชื่อมต่ออินเทอร์เน็ตอยู่ (หากต้องการเชื่อมต่อกับเครือข่ายไร้สาย ให้ทำตามขั้นตอนใน “เชื่อมต่ออินเทอร์เน็ต” ซึ่งปรากฏก่อนหน้าในเอกสารนี้)
  3. เลือกยูทิลิตี้ดิสก์ แล้วคลิกดำเนินการต่อ
  4. เลือกดิสก์โวลุ่มของคุณในรายการทางซ้าย แล้วคลิกลบ
  5. ป้อนชื่อสำหรับดิสก์โวลุ่มของคุณ เลือก Mac OS แบบขยาย จากเมนูป๊อปอัพรูปแบบ แล้วคลิกลบ

    หลังจากลบดิสก์โวลุ่มเรียบร้อยแล้ว ให้เลือกยูทิลิตี้ดิสก์ > ออกจากยูทิลิตี้ดิสก์ แล้วเลือกติดตั้ง macOS อีกครั้ง คลิกดำเนินการต่อ จากนั้นทำตามขั้นตอนในตัวติดตั้ง macOS เพื่อติดตั้ง macOS อีกครั้ง
