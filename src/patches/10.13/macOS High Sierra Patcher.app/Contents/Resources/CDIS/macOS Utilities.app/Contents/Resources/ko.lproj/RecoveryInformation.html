복구 정보

이 도큐멘트에는 컴퓨터 볼륨을 복구하는 방법, 사용자의 파일 및 사용자 설정을 그대로 유지하며 macOS를 다시 설치하는 방법, Time Machine을 통해 컴퓨터 시스템을 복원하는 방법 및 macOS를 지우고 다시 설치하는 방법에 대한 정보가 포함되어 있습니다.

추가 도움말은 https://www.apple.com/kr/support 사이트로 이동하십시오.

인터넷 연결하기

인터넷에 연결해야 하는 경우 무선 또는 유선 네트워크 연결을 사용할 수 있습니다. 무선 네트워크에 연결하려면,

  1. 메뉴 막대에서 Wi-Fi 상태 아이콘(화면 오른쪽 상단에 있음)을 클릭한 다음 네트워크를 선택하십시오.
  2. 필요한 경우 네트워크 암호를 입력하십시오.

폐쇄형 네트워크에 연결하려면 다른 네트워크에 연결을 선택하십시오. 네트워크 이름을 입력하고 필요한 경우에는 네트워크 암호도 입력하십시오.

볼륨 복구하기

컴퓨터에 문제가 있거나 컴퓨터를 시작한 후에 macOS 유틸리티 패널이 나타나는 경우 컴퓨터의 내부 저장 기기 및 볼륨을 복구해야 할 수 있습니다.

참고:이 윈도우를 닫거나 Safari를 종료하기 전에 이 지침을 메모해 두십시오.

  1. Safari > Safari 종료를 선택한 다음 macOS 유틸리티 패널에서 디스크 유틸리티를 선택한 다음 계속을 클릭하십시오.
  2. 왼쪽에 있는 목록에서 내부 저장 기기를 선택한 다음 검사/복구를 클릭하십시오.
  3. 왼쪽 목록에서 시동 볼륨을 선택한 다음 검사/복구를 클릭하십시오.
  4. 실행을 클릭하십시오.

디스크 유틸리티가 사용자의 디스크를 복구할 수 없는 경우 가능한 많은 정보를 백업한 후에 이 도큐멘트 후반부에 있는 ‘macOS 지우고 재설치하기’에 있는 지침을 따르십시오.

macOS 다시 설치하기

일부의 경우 macOS를 다시 설치해야 할 수 있으며 사용자의 파일 및 사용자 설정을 그대로 유지한 상태로 macOS를 다시 설치할 수 있습니다.

참고:이 윈도우를 닫거나 Safari를 종료하기 전에 이 지침을 메모해 두십시오.

  1. Safari > Safari 종료를 선택하십시오.
  2. 인터넷에 연결되어 있는지 확인하십시오(무선 네트워크에 연결하려면 이 도큐멘트 초반부에 있는 ‘인터넷 연결하기’의 지침을 따르십시오).
  3. macOS 다시 설치를 선택한 다음 계속을 클릭하십시오.
  4. 볼륨을 선택하는 패널에서 현재의 macOS 볼륨을 선택하십시오(대부분의 경우 사용 가능한 디스크는 하나뿐임).
  5. 사용자화를 클릭하여 옵션 소프트웨어를 선택 또는 선택 해제하십시오.
  6. 설치를 클릭하십시오.

Time Machine을 사용하여 시스템 복원하기

macOS를 지우고 다시 설치할 때 이전에 Time Machine 백업을 생성한 적이 있는 경우 복원 유틸리티를 사용하여 컴퓨터에 있는 모든 항목을 이전 상태로 복원하십시오.

오직 Time Machine 백업만을 사용하여 백업의 소스인 컴퓨터에 정보를 복원하십시오. 정보를 새로운 컴퓨터로 전송하려면 마이그레이션 지원을 사용하십시오(Launchpad의 기타 폴더에 있음).

참고:이 윈도우를 닫거나 Safari를 종료하기 전에 이 지침을 메모해 두십시오.

  1. Safari > Safari 종료를 선택하십시오.
  2. 사용자의 백업이 Time Capsule에 있을 경우 인터넷에 연결되어 있는지 확인하십시오(무선 네트워크에 연결하려면 이 도큐멘트 초반부에 있는 ‘인터넷 연결하기’의 지침을 따르십시오)
  3. Time Machine 백업에서 복원을 선택한 후에 계속을 클릭하십시오.
  4. 백업 소스를 선택하고 해당 Time Machine 백업이 포함된 볼륨을 선택한 다음 화면 상의 지침을 따르십시오.

macOS 지우고 다시 설치하기

macOS를 지우고 다시 설치하면 모든 항목(사용자 계정, 네트워크 설정, 사용자의 모든 파일 및 폴더)이 삭제됩니다. OS X을 지우기 전에 보관하려는 모든 파일을 다른 볼륨으로 복사하여 백업하십시오. 네트워크 환경설정에 있는 네트워크 설정 정보를 메모해 두면 macOS를 다시 설치한 후에 네트워크에 더 쉽게 다시 연결할 수 있습니다.

참고:이 윈도우를 닫거나 Safari를 종료하기 전에 이 지침을 메모해 두십시오.

  1. Safari > Safari 종료를 선택하십시오.
  2. 인터넷에 연결되어 있는지 확인하십시오(무선 네트워크에 연결하려면 이 도큐멘트 초반부에 있는 ‘인터넷 연결하기’의 지침을 따르십시오).
  3. 디스크 유틸리티를 선택한 다음 계속을 클릭하십시오.
  4. 왼쪽에 있는 목록에서 볼륨을 선택한 다음 지우기를 클릭하십시오.
  5. 볼륨 이름을 입력하고 포맷 팝업 메뉴에서 Mac OS 확장을 선택한 다음 지우기를 클릭하십시오. 볼륨이 지워지면 디스크 유틸리티 > 디스크 유틸리티 종료를 선택하십시오. macOS 다시 설치를 선택하고 계속을 클릭한 다음 macOS 설치 프로그램의 지침에 따라 macOS를 다시 설치하십시오.
