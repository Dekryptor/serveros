Informations de récupération

Ce document comporte des informations sur la réparation du volume de votre ordinateur, sur la réinstallation de macOS tout en gardant vos fichiers et réglages utilisateur intacts, sur l’utilisation de Time Machine pour restaurer le système de votre ordinateur, ainsi que des informations pour effacer et réinstaller macOS.

Pour obtenir de l’aide pour Utilitaire de disque, rendez-vous sur https://www.apple.com/fr/support.

Se connecter à Internet

Vous devez accéder à un réseau avec ou sans fil pour pouvoir vous connecter à Internet. Pour vous connecter à un réseau sans fil :

  1. Cliquez sur l’icône d’état du Wi-Fi dans la barre des menus (coin supérieur droit de l’écran), puis choisissez un réseau.
  2. Si nécessaire, saisissez le mot de passe du réseau.

Pour accéder à un réseau fermé, choisissez Se connecter à un autre réseau. Saisissez le nom du réseau et, si nécessaire, le mot de passe du réseau.

Réparer votre volume

Si votre ordinateur rencontre des problèmes ou si la sous-fenêtre Utilitaires macOS s’ouvre au démarrage, il se peut que vous ayez besoin de réparer le volume et l’appareil de stockage interne de votre ordinateur.

Remarque : il est conseillé de noter ces instructions avant de fermer cette fenêtre ou de quitter Safari.

  1. Choisissez Safari > Quitter Safari, sélectionnez Utilitaire de disque dans la sous-fenêtre Utilitaires macOS, puis cliquez sur Continuer.
  2. Dans la liste de gauche, sélectionnez l’appareil de stockage interne, puis cliquez sur S.O.S.
  3. Dans la liste de gauche, sélectionnez le volume de démarrage, puis cliquez sur S.O.S.
  4. Cliquez sur Exécuter.

Si Utilitaire de disque ne parvient pas à réparer votre disque, sauvegardez autant de données que possible, puis suivez les instructions de la section « Effacer et réinstaller macOS » de ce document.

Réinstaller macOS

Dans certains cas, il peut être nécessaire de réinstaller macOS. Cette réinstallation peut s’effectuer en gardant intacts vos fichiers et réglages utilisateur.

Remarque : il est conseillé de noter ces instructions avant de fermer cette fenêtre ou de quitter Safari.

  1. Choisissez Safari > Quitter Safari.
  2. Vérifiez que vous êtes connecté à Internet. (Pour vous connecter à un réseau sans fil, suivez les instructions de la section « Se connecter à Internet » de ce document.)
  3. Sélectionnez Réinstaller macOS, puis cliquez sur Continuer.
  4. Dans la sous-fenêtre permettant de sélectionner un volume, sélectionnez votre volume macOS actuel (dans la plupart des cas, il s’agit du seul volume disponible).
  5. Cliquez sur Personnaliser pour sélectionner ou désélectionner les logiciels facultatifs.
  6. Cliquez sur Installer.

Utiliser Time Machine pour restaurer votre système

Si vous effacez puis réinstallez macOS, et que vous aviez précédemment créé une sauvegarde Time Machine, utilisez l’utilitaire de restauration pour rétablir l’état de votre ordinateur avant la réinstallation.

N’utilisez votre sauvegarde Time Machine que pour restaurer des données sur l’ordinateur qui est la source de cette sauvegarde. Si vous souhaitez transférer des données sur un nouvel ordinateur, utilisez Assistant migration (dans le dossier Autre dans Launchpad).

Remarque : il est conseillé de noter ces instructions avant de fermer cette fenêtre ou de quitter Safari.

  1. Choisissez Safari > Quitter Safari.
    2. Si votre sauvegarde se trouve sur une Time Capsule, vérifiez que vous êtes connecté à Internet. (Pour vous connecter à un réseau sans fil, suivez les instructions de la section « Se connecter à Internet » de ce document.)
  2. Sélectionnez Restaurer à partir d’une sauvegarde Time Machine, puis cliquez sur Continuer.
  3. Sélectionnez une sauvegarde, sélectionnez le volume contenant la sauvegarde Time Machine, puis suivez les instructions qui s’affichent à l’écran.

Effacer et réinstaller macOS

Si vous effacez puis réinstallez macOS, toutes vos données (comptes utilisateur, réglages réseau, tous les fichiers et dossiers) sont supprimées. Avant d’effacer macOS, sauvegardez tous les fichiers que vous voulez conserver en les copiant sur un autre volume. Notez les réglages réseau définis dans les préférences Réseau ; cela vous aidera à vous reconnecter plus facilement après avoir réinstallé macOS.

Remarque : il est conseillé de noter ces instructions avant de fermer cette fenêtre ou de quitter Safari.

  1. Choisissez Safari > Quitter Safari.
  2. Vérifiez que vous êtes connecté à Internet. (Pour vous connecter à un réseau sans fil, suivez les instructions de la section « Se connecter à Internet » de ce document.)
  3. Sélectionnez Utilitaire de disque, puis cliquez sur Continuer.
  4. Sélectionnez votre volume dans la liste de gauche, puis cliquez sur Effacer.
  5. Nommez votre volume, choisissez Mac OS étendu dans le menu local Format, puis cliquez sur Effacer.

    Une fois le volume effacé, choisissez Utilitaire de disque > Quitter Utilitaire de disque. Sélectionnez Réinstaller macOS, cliquez sur Continuer, puis suivez les instructions du programme d’installation de macOS pour réinstaller macOS.
